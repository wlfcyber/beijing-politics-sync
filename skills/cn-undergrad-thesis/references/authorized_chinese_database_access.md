# 授权中文数据库访问规则

## 适用场景

用户提供本人或学校授权的 CNKI、万方、维普、超星、学校图书馆、人大图书馆代理等入口，并希望在论文工作流中读取文献。

## 可做

- 在用户授权和登录后的浏览器会话中检索、筛选、阅读可见题录、摘要和全文。
- 复用用户长期保持登录的浏览器会话，用于当前论文任务的检索和阅读。
- 处理用户导出的题录、参考文献、RIS、BibTeX、EndNote、NoteExpress、CAJ、PDF、截图或复制文本。
- 记录每条文献的作者、题名、期刊/来源、年份、数据库、链接或数据库路径、摘要、关键词、研究对象、方法、核心观点、局限和核验状态。
- 在全文可见且访问许可允许时，选择性阅读与论文相关的段落，提取论证、方法、发现、限制和页码/位置。
- 生成检索式、检索日志、文献矩阵、证据台账、引用核验表。

## 不可做

- 不绕过登录、SSO、验证码、付费墙、下载限制、访问控制或机构规则。
- 不自动填写或猜测账号密码。
- 不查看、导出、复制或保存 cookies、local storage、浏览器密码、会话文件或账号凭据。
- 不批量下载、镜像、抓取无关文献或建立超出论文任务范围的数据库。
- 不把只看到题录/摘要的文献写成“全文已读”。
- 不根据标题或摘要编造文献结论、数据、页码或引用原文。
- 不长篇复制受版权保护的正文；正式写作以概括、转述和短引用为主。

## 浏览器流程

1. 优先使用用户已经登录好的 Chrome/浏览器标签页；如果没有打开，再打开用户提供的授权入口。
2. 如果出现登录、验证码、统一身份认证、会话过期、权限确认或下载确认，停止并让用户本人完成。
3. 根据课程名或选题生成 3 组检索式：
   - 核心概念检索式。
   - 研究对象检索式。
   - 方法或场景检索式。
4. 先读题录和摘要，筛出候选文献。
5. 对候选文献标注：
   - `题录可见`
   - `摘要可见`
   - `全文已读`
   - `PDF/user_exported`
   - `待用户导出`
6. 对正式进入论文正文的文献，必须进入文献矩阵和证据台账。
7. 如果全文不可见或不可下载，保留为候选文献，并提示用户导出题录或上传 PDF。

## 长期登录会话规则

- 可以把知网、人大代理或学校图书馆页面长期保持在用户自己的 Chrome 里。
- 后续论文任务可优先接管这个已登录标签页进行可见页面操作。
- 登录状态属于用户账号环境，不属于 skill 或代理；不得保存、复制、导出或分析登录凭据。
- 会话过期时，由用户重新登录；代理不得尝试绕过或自动处理身份验证。
- 每次检索都要服务于当前论文任务，不得因为会话长期可用而扩大到无关批量抓取。

## 写作规则

- `题录可见`：只能用于判断是否值得继续找，不能写成实质性结论。
- `摘要可见`：可用于初步综述和候选矩阵，正式提交前最好核验全文。
- `全文已读`：可进入正式文献综述，但仍需记录页码或章节位置。
- `PDF/user_exported`：可作为正式证据层，但要检查题录字段。
- `待用户导出`：不能进入最终正文的确定性引用。
