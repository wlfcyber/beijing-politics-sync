# 中文本科论文表达修订规则

## 原则

修订表达时只改变写法，不改变证据边界。不能为了显得自然而增加新事实、新文献、新数据或新结论。

## 常见空话替换

| 空泛表达 | 更好的方向 |
| --- | --- |
| 随着时代的发展 | 指明具体政策、技术、制度或社会变化 |
| 具有重要意义 | 说明对哪个问题、对象或研究脉络有何作用 |
| 众所周知 | 删除，或给出来源 |
| 综上所述可以看出 | 直接总结前文可支持的发现 |
| 相关研究较少 | 改为“在当前已检索文献范围内，较少发现……” |

## 修订流程

1. 标出没有来源支撑的判断。
2. 把抽象词换成具体对象、时间、场景、方法或文献。
3. 调整句长，避免每句都用同一结构。
4. 保留必要的学术语气，不把论文改成散文或公众号文。
5. 输出修订后文本，并列出不能确定的事实。

## 禁止

- 不承诺“降低检测率”或“绕过 AI 检测”。
- 不删除必要引用。
- 不把“待核验”改成确定事实。
- 不添加用户没有提供的访谈、问卷、数据或导师意见。
