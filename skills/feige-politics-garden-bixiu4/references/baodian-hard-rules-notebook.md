# 飞哥必修四宝典硬性要求记事本

每次重跑、修订、生成 Word 或接手 ClaudeCode/Codex 输出前，必须先读本文件。任何新要求只能追加，不能覆盖旧要求。

## 总目标

- 产物是给高三学生看的《2026北京高考政治哲学宝典---三年模拟全触发全链条》，不是工作日志、审计报告或逐题流水账。
- 内容必须按飞哥的原理方法论框架组织，不按试卷逐题排列。
- 依据只能来自三年北京区级模拟题、答案、评分细则、阅卷/评标/讲评等可靠来源；旧稿只能定位和审计，不能当证据。
- 每套卷至少要有 3 个可用触发点；不足 3 个必须在审计表里明示，不准假装完成。

## 框架颗粒度

所有大框架都必须分出具体原理方法论节点，不能只拆唯物论，后面混成大桶。

### 唯物论

- 物质决定意识，意识对物质具有能动作用
- 一切从实际出发 / 实事求是 / 主观与客观具体的历史的统一
- 主观能动性 / 意识的能动作用
- 尊重客观规律与发挥主观能动性相结合
- 规律的客观性

### 辩证法

- 联系的普遍性 / 联系的观点（总）
- 整体与部分 / 系统观念 / 系统优化
- 发展的观点 / 发展的普遍性
- 量变与质变 / 适度原则
- 事物发展是前进性与曲折性的统一
- 辩证否定 / 守正创新
- 矛盾就是对立统一
- 矛盾的普遍性
- 矛盾的特殊性 / 具体问题具体分析
- 矛盾的普遍性和特殊性
- 两点论与重点论
- 内因与外因

### 认识论

- 实践与认识（总）
- 实践是认识的基础
- 认识对实践的反作用
- 认识发展原理
- 真理观

### 历史唯物主义

- 社会存在与社会意识
- 社会发展的两大基本规律和基本矛盾
- 改革 / 改革的实质
- 人民群众

### 价值观 / 人生观

- 价值观的导向作用
- 价值判断与价值选择
- 实现人生价值

### 文化线

文化线若放入同一产物，必须单独标为必修四文化线，且拆成文化功能、优秀传统文化传承与双创、民族精神/文化自信、社会主义核心价值观等节点，不能混入哲学主框架。

## 单条内容顺序

每一条必须按学生看到题时的思维顺序写：

1. 材料触发点：至少一个完整句子，指出材料中哪句话、哪个词、哪个关系、哪个主体行动触发该原理。
2. 设问：主观题必须抄全设问，不能只写题号或半截设问。
3. 为什么能想到：用知识逻辑解释材料和原理的关系，不要写“因为细则提到了所以能放进来”。
4. 答案落点：必须写成能回答本题设问的答案句或答案方向，不能写“要回应设问”“服务设问”这种空话。

## 排序

- 同一节点内，主观题优先于客观题。
- 同一类型内，区县顺序为：海淀、西城、东城、朝阳、丰台，再到其他区。
- 不是按题目流水账排列，而是每个题进入它真正触发的原理方法论节点。

## 学生版与审计版分离

- 学生最终版不得出现来源路径、line id、file id、slide、OCR、debug、日志、英文内部编号、`/Users/...` 等无效信息。
- 来源、细则摘录、OCR 状态、证据边界、漏项扫描、模型分歧都放审计文件。
- 漫画题要尽量把漫画图嵌进 Word，不能只文字描述。
- Word 首页只留标题和大署名“飞哥正志讲堂”；第二页留前言空位；不要工作日志附录。

## 开工前自检

- 先读本记事本，再看生成脚本和当前成品目录。
- 修一个新要求时，要回头检查旧要求是否仍满足：框架颗粒度、顺序、设问、答案落点、排序、干净版式、漫画、覆盖表。
- 抽检必须包含用户指出过的硬样本：`2026海淀一模 第16题` 必须出现在 `物质决定意识，意识对物质具有能动作用` 节点下，并且答案落点必须说明人工智能时代客观技术变革决定人文学科价值被重新认识，正确意识又能引导技术服务文明进步。
