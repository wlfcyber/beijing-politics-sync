# 飞哥必修四宝典硬性要求记事本

每次重跑、修订、生成 Word 或接手 ClaudeCode/Codex 输出前，必须先读本文件。任何新要求只能追加，不能覆盖旧要求。

## 总目标

- 产物是给高三学生看的《2026北京高考政治哲学宝典---三年模拟全触发全链条》，不是工作日志、审计报告或逐题流水账。
- 内容必须按飞哥的原理方法论框架组织，不按试卷逐题排列。
- 依据只能来自三年北京区级模拟题、答案、评分细则、阅卷/评标/讲评等可靠来源；旧稿只能定位和审计，不能当证据。
- 每套卷至少要有 3 个可用触发点；不足 3 个必须在审计表里明示，不准假装完成。

## 框架颗粒度

所有大框架都必须分出具体原理方法论节点，不能只拆唯物论，后面混成大桶。

### 唯物论

- 物质决定意识，意识对物质具有能动作用
- 一切从实际出发 / 实事求是 / 主观与客观具体的历史的统一
- 主观能动性 / 意识的能动作用
- 尊重客观规律与发挥主观能动性相结合
- 规律的客观性

### 辩证法

- 联系的普遍性 / 联系的观点（总）
- 联系的客观性
- 根据固有联系建立新的具体联系
- 联系的多样性
- 整体与部分
- 系统观念 / 系统优化
- 发展的观点 / 发展的普遍性
- 量变与质变 / 适度原则
- 事物发展是前进性与曲折性的统一
- 辩证否定 / 守正创新
- 矛盾就是对立统一
- 矛盾的普遍性
- 矛盾的特殊性 / 具体问题具体分析
- 矛盾的普遍性和特殊性
- 两点论与重点论
- 内因与外因

### 认识论

- 实践与认识（总）
- 实践是认识的基础
- 认识对实践的反作用
- 认识发展原理
- 真理观

### 历史唯物主义

- 社会存在与社会意识
- 社会发展的两大基本规律和基本矛盾
- 改革 / 改革的实质
- 人民群众

### 价值观 / 人生观

- 价值观的导向作用
- 价值判断与价值选择
- 实现人生价值

### 文化线

文化线若放入同一产物，必须单独标为必修四文化线，且拆成文化功能、优秀传统文化传承与双创、民族精神/文化自信、社会主义核心价值观等节点，不能混入哲学主框架。

## 单条内容顺序

每一条必须按学生看到题时的思维顺序写：

1. 材料触发点：至少一个完整句子，指出材料中哪句话、哪个词、哪个关系、哪个主体行动触发该原理。
2. 设问：主观题必须抄全设问，但设问栏只能放真正的题目问法；材料正文不得混入设问栏，材料只放在“材料触发点”里。若原始字段写成“材料与设问”，必须切出最后的完整问句；评价型设问可连同“有人说/有人认为/这一观点”等被评价对象一起保留。
3. 为什么能想到：用知识逻辑解释材料和原理的关系，不要写“因为细则提到了所以能放进来”。
4. 答案落点：必须写成能回答本题设问的答案句或答案方向，不能写“要回应设问”“服务设问”这种空话。

### 复合节点与细则子任务规则

- 复合标题不能自动展开成整套原理。比如 `物质决定意识，意识对物质具有能动作用` 只是框架节点名；单条正文必须按细则实际支持的半句来写。细则只支持“物质决定意识/必要性”时，不得补写“意识反作用于物质/意识反作用”。
- `为什么能想到` 和 `答案落点` 必须服从细则的子任务位置。若细则把某原理放在“必要性”里，解释链就要写“为什么客观现实决定必须这样做”；不能泛泛写原理本身，更不能把其他未出现原理塞进答案。
- 同一题出现多个明确原理时，要按框架分别入栏。不能因为一条已经进了 `物质决定意识`，就漏掉同题在 `主观能动性 / 意识的能动作用` 下的独立触发链。
- 用户人工核对到更完整细则时，若本地缓存只抽到简版角度清单，要把本地缓存标为“不完整细分”，按用户指出的完整细则口径修正，并在审计文件里注明边界。

## 排序

- 同一节点内，主观题优先于客观题。
- 同一类型内，区县顺序为：海淀、西城、东城、朝阳、丰台，再到其他区。
- 不是按题目流水账排列，而是每个题进入它真正触发的原理方法论节点。

## 学生版与审计版分离

- 学生最终版不得出现来源路径、line id、file id、slide、OCR、debug、日志、英文内部编号、`/Users/...` 等无效信息。
- 来源、细则摘录、OCR 状态、证据边界、漏项扫描、模型分歧都放审计文件。
- 漫画题要尽量把漫画图嵌进 Word，不能只文字描述。
- Word 首页只留标题和大署名“飞哥正志讲堂”；第二页留前言空位；不要工作日志附录。

## 开工前自检

- 先读本记事本，再看生成脚本和当前成品目录。
- 修一个新要求时，要回头检查旧要求是否仍满足：框架颗粒度、顺序、设问、答案落点、排序、干净版式、漫画、覆盖表。
- 抽检必须包含用户指出过的硬样本：`2026海淀一模 第16题` 必须出现在 `物质决定意识，意识对物质具有能动作用` 节点下；该节点只写“物质决定意识/必要性”，说明人工智能时代客观技术变革和技术局限决定人文学科不可替代，不得写“正确意识引导技术”等意识反作用内容。
- 抽检必须包含用户指出过的硬样本：`2024东城一模 第18题第（3）问` 的传统产业与未来产业关系必须进入 `辩证否定 / 守正创新`；若设问限定《逻辑与思维》，学生版可作为同名原理迁移链呈现，审计版必须注明模块边界。
- 最终学生版不得出现 `yes`、`pass`、`filled`、`correct_option_chain` 等状态字段作为答案落点；这类词一旦进入正文，视为生成失败。
- 最终学生版不得出现 `评标`、`参考答案`、`答案写`、`答案/补充`、`答案核`、`可从……角度作答` 等审计/答案来源话术；这些只能留在审计文件，不能冒充材料触发点、为什么能想到或答案落点。
- 最终学生版必须抽检 `2026海淀一模 第16题`：`设问` 只能是“有人说，人文学科在人工智能时代具有不可替代的价值。结合材料，运用《哲学与文化》知识，谈谈对这一观点的认识。”，不得把前面整段 AI 与人文学科材料抄进设问栏。
- 最终学生版必须抽检 `2026海淀一模 第16题` 的细则口径：在 `物质决定意识，意识对物质具有能动作用` 节点下，只写“物质决定意识/必要性”，不得出现“意识反作用于物质”；同题还必须单独出现在 `主观能动性 / 意识的能动作用` 节点下。
