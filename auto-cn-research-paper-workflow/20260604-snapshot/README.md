# Auto CN Research Paper Workflow

这是一次中文人文社科论文自动科研流的工作流镜像，记录从主题收窄、人大图书馆/知网授权检索、文献矩阵、优秀论文 craft 学习、论证骨架、初稿写作、审计闸口到外部评审的试运行成果。

当前试运行主题：基层数字治理中的形式主义。

当前论文题目：可见绩效的生产：基层数字治理何以滑向形式主义——基于既有案例的机制性比较分析。

## 目录

- `docs/`：工作流规格、工程化验收、知网和 Chrome 接入记录。
- `runs/试运行_数字形式主义/`：本次试运行的选题、检索、文献矩阵、范式提取、论证骨架、初稿、审计、外部评审和复盘文档。
- `skills/auto-cn-research-paper/`：当前使用的 Codex skill 说明、脚本和浏览器准入参考。
- `UPLOAD_SCOPE.md`：本镜像的上传范围和排除规则。

## 使用原则

本仓库只保存工作流、审计记录、脚本和论文草稿，不保存知网/RUC 下载的原始全文、转换文件、全文抽取文本、浏览器截图、登录状态或认证信息。

最终完成必须同时满足：材料层通过、文本质量层通过、页码层通过、人大图书馆/知网 hands-free 闸口通过、Claude 网页评审通过、GPT-5.5 Pro 真实评审通过，并且总闸口矩阵显示 `final_user_goal_ready=yes`。
