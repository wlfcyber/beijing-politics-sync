# 00 飞哥旧框架风格 DNA（学习样本：选必一国政经框架+三年题链 + 必修四哲学宝典 + 选必三思维零基础课稿）

旧框架文件仅供学习风格，不作为选必二法律题的证据来源；本轮不继承旧的题链、分类或结论。

## 一、口径写法

- 开头先讲“怎么用”：只看试卷原题 + 细则/评标/讲评；不把参考答案当细则；每题统一压成 `材料场景 → 细则抓手 → 回填框架`。
- 用回填、抓手、稳定收口这类带工艺感的词，不用"全面分析""科学论证"这类报告腔。

## 二、模块层

- 模块清单先列“高频主线”小标题，每条一句，短。
- 每条用关键术语而不是讲解：`贸易投资自由化便利化`、`两个市场两种资源`、`共商共建共享`。
- 飞哥习惯把抽象概念拍成口诀型链条：`高水平对外开放 → 更高层次开放型经济 → 双循环`。

## 三、题链层

- 每题一段，三件套：
  1. 材料一句话场景。
  2. 细则抓手分一/二/三层，逐层短句。
  3. 回填框架定位。
- 写法示例：
  > `2024朝阳一模 Q21`：材料是"如何推动中国经济在变局中开新局"。细则抓手分两步：先概括经济长期向好但外部环境严峻、风险挑战增多，再从总体国家安全观、推动新型国际关系、构建新发展格局、推进高水平对外开放等角度展开。回填时放在"理论 → 统筹发展和安全"，并兼跨"经济全球化""中国"。

## 四、高频回灌层

- 模块结尾必有 `高频重复点回灌`：把这一模块反复考的词组单独提出来。
- 必有 `当前可直接反复用的答题链`：把整张试卷可以打包成的链条单独列：
  `便利往来/规则对接/平台建设 → 要素流动与资源配置 → 贸易投资自由化便利化 → 高水平对外开放 → 开放型经济`

## 五、收尾层

- 最后必有 `暂挂起` / `待补` 段，把没找到细则或边界不清的题目单独写明原因，不混入主链。
- 这种坦白本身就是飞哥风格 —— 宁可挂起，不硬塞。

## 六、语气

- 短句、判断句、动词起头。
- 不要 AI 报告语：不要"综上所述"、"我们可以得出"。
- 直接给学生用："直接做成三大模板题"、"放在 X 模块"、"考场起手就写"。
- 飞哥喜欢插入 1-2 句带裁断感的话："这个不是结尾空话"、"这个已经形成固定链条"、"中国题常见收束方式不是单写'中国很重要'，而是写……"。

## 七、本轮选必二需要补的两层

旧框架（选必一线）侧重 `正向触发`。本轮选必二要在它的风格基础上补两层：

1. **反向筛**：每个触发点要加"看到 X 不要写 Y"的对照。
   - 法院判决 ≠ 公正司法（必修三）
   - 平台用工 ≠ 自动认定劳动关系（要有三从属性）
   - 消费者 ≠ 一律三倍赔偿（要有欺诈）
2. **飞哥想说**：每个触发点末尾留一句裁断 —— 这道题学生最容易写偏的地方在哪，哪条用得最稳，哪条别乱套。

## 八、术语黑名单（学生正文绝不出现）

corpus、atom、evidence、rubric、PASS、reference_only、page、slide、R_、M_、评分细则原文、参考答案原文、阅卷总结原文。

学生看到的应当是被翻译过的"细则抓手""高频得分句"，而不是直接粘贴的细则编号或评分点。
