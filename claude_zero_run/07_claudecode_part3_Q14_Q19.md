# 07 下篇 全题题链（飞哥版）· part3 Q14–Q19

## Q14 2025西城期末 第19题

- 题源："小爱同学"智能语音唤醒词案——陈某在不同商品类别上抢先注册相关商标、与 Y 公司合作在运动手表/闹钟上使用并发布宣传，M 公司诉构成不正当竞争；法院判反不正当竞争法保护、陈某赔 120 万、Y 公司对其中 25 万承担连带责任。
- 设问入口：S01 问诉求/裁判理由（运用法律知识分析法院的判决）。
- 命中框架：知识产权·混淆型不正当竞争路径（反不正当竞争法）+ 意义层。
- 材料触发：知识产权 / 不正当竞争 / 商业混淆。
  1. "小爱同学"作为有一定影响力的唤醒词、人工智能引擎及智能音箱商品名称 → 受反不正当竞争法保护 → 法律依据：反不正当竞争法对有一定影响商品名称的保护。
  2. 陈某在他人已使用并有一定影响的情况下抢先注册、与 Y 公司合作发布引人误解的商业宣传 → 构成混淆 + 虚假宣传 → 属不正当竞争行为、违背诚实信用原则与商业道德。
  3. 法院判停止侵权、陈某赔 120 万、Y 公司连带 25 万 → 合理确定各方利益、规制滥用权利 → 保护科技创新型企业品牌商誉、促进新业态发展、为数字经济发展营造良好的法治化营商环境。
- 答案骨架：确权（唤醒词受反不正当竞争法保护）→ 行为定性（抢注+引人误解的商业宣传+混淆+虚假宣传）→ 违反原则（诚实信用、商业道德）→ 责任承担（停止侵权+赔偿+连带）→ 收口意义层（品牌商誉+新业态+法治化营商环境）。
- 反向筛：F04 别把唤醒词案写成"商标侵权/著作权侵权"，要锁反不正当竞争法路径；F08 别铺反不正当竞争法条文；要写出"诚实信用原则/商业道德"才有原则分。
- 飞哥想说：唤醒词、APP 名称、软件图标这类"有一定影响的标识"都走反不正当竞争法，记牢"混淆 + 虚假宣传 + 诚实信用原则"这一组关键词。

## Q15 2025顺义一模 第19题（第2问）

- 题源：案例1高某与刘某邻居纠纷、刘某门口堆杂物垃圾+微信群辱骂、高某诉至法院、经法院调解握手言和；案例2甲公司有较高知名度、乙公司将名称变更为与甲相似、市监局裁决侵权、乙不服起诉被驳回。
- 设问入口：S03 问意义/价值（阐释定分止争的价值）。
- 命中框架：纠纷解决·多元化解机制 + 意义四层。
- 材料触发：纠纷解决 / 调解 / 行政裁决 / 诉讼 + 物权相邻 + 反不正当竞争。
  1. 案例1邻里堆物纠纷、经法院调解和好 → 多元纠纷解决机制·调解优先 → 调解高效化解矛盾、维护邻里和谐、增强公众对法律的信任与遵守意识。
  2. 案例2市监局裁决+诉讼维持 → 行政裁决+司法监督衔接 → 保护企业合法权益、规范市场主体行为、维护公平市场竞争秩序。
  3. 两案合并 → 民事主体行使民事权利不得损害他人合法权益、公民应有维权意识又应选择适当途径 → 化解社会矛盾、促进社会和谐、维护社会秩序、提升社会治理水平。
- 答案骨架：对个人价值（维护合法权益+增强法律信任）→ 对企业价值（保护合法权益+规范市场主体+维护公平竞争）→ 对社会价值（化解矛盾+促进和谐+维护秩序+提升治理）→ 收口社会主义核心价值观（文明、和谐）。
- 反向筛：F05 价值要分三层（个人-企业-社会），堆同义词不重复给分；F08 别铺民法典；案例1别只说"调解"而不写"调解的价值"。
- 飞哥想说：定分止争的"分"是权利边界，"止争"是化解矛盾；这道题就用"个人-企业-社会"三档阶梯，再收一句"理性表达+依法维权"就稳了。

## Q16 2026东城一模 第18题

- 题源：案例一某科技公司三创始人专利权归属纠纷、合议庭居中斡旋促成和解；案例二乙公司侵甲公司植物新品种权、适用惩罚性赔偿 300 万；案例三丙公司恶意诉讼诉讼请求反复变更、被驳回并明确谴责。
- 设问入口：S05 问"如何守护"（法院如何以司法之力守护"向新力"）。
- 命中框架：知识产权·司法守护三动作（调解 + 惩罚性赔偿 + 驳回恶意诉讼）。
- 材料触发：知识产权 / 调解 / 惩罚性赔偿 / 恶意诉讼。
  1. 案例一合议庭未"一判了之"而是居中斡旋、促成当事人和解 → 诉讼调解 → 推动纠纷实质性化解、效率高、节约司法资源、让创新主体回归创新创业。
  2. 案例二乙公司侵植物新品种权、覆盖范围广、判令停止侵害并适用惩罚性赔偿 300 万 → 严惩主观恶性强、社会危害大的知产侵权 → 增加侵权成本、保护当事人合法权益、维护市场秩序与诚信原则。
  3. 案例三丙公司恶意起诉时机选在丁公司上市关键期、诉讼请求反复无常缺乏依据、法院驳回并谴责 → 警示规范作用 → 维护诉讼秩序、节约司法资源、彰显权利义务相统一与诚信原则。
- 答案骨架：案例一调解（化解纠纷+回归创新）→ 案例二惩罚性赔偿（提高侵权成本+保护合法权益+市场秩序）→ 案例三驳回谴责（规制恶意诉讼+维护诉讼秩序）→ 总收口（公正司法、以事实为依据法律为准绳、保护知识产权就是保护创新、营造法治化营商环境）。
- 反向筛：F08 案例二别写成"商业诋毁/不正当竞争"，是"知产侵权 + 惩罚性赔偿"；案例三别写成"商业诋毁"，是"恶意诉讼/滥用诉讼权利"；F09 不要直接甩"公正司法严格执法"，要先有具体动作。
- 飞哥想说：三案对应三动作——"调、惩、辨"是这一类题的固定结构；每个动作都要带"谁的权益 + 哪条秩序 + 一句法治环境"。

## Q17 2026朝阳一模 第18题

- 题源："调、惩、辨"三维实践——【以"调"解纷】特邀第三方调解化解生物医药专利权归属纠纷；【以"惩"亮剑】对故意侵植物新品种权/商业秘密适用惩罚性赔偿（玉米品种 5300 余万、玻璃机技术秘密 3.8 亿）；【以"辩"正风】驳回并谴责某日化公司恶意诉讼。
- 设问入口：S05 问"如何守护"（法院如何依法保护知识产权以护航新质生产力发展）。
- 命中框架：知识产权·司法守护三动作（调解 + 惩罚性赔偿 + 规制恶意诉讼）。
- 材料触发：知识产权 / 民法典惩罚性赔偿 / 诚信与公平原则。
  1. 特邀第三方调解、一揽子化解关联诉讼、让科研人员安心回归科研 → 高效化解知识产权纠纷 → 维护当事人合法权益、激发创新积极性、提高诉讼效率、降低诉讼成本。
  2. 对故意侵植物新品种权/商业秘密适用民法典惩罚性赔偿、判赔 5300 余万和 3.8 亿 → 提高侵权违法成本 → 有效遏制再次侵权、保护创新主体核心利益、维护良好的创新生态和公平竞争市场环境。
  3. 依据民法公平、诚信原则、谴责某日化公司不诚信诉讼行为 → 规制权利滥用 → 保障诉讼秩序与经营秩序、避免"假维权"拖垮"真创新"、营造诚信守法的创新生态。
- 答案骨架：调（特邀第三方调解+诉讼效率+创新驱动）→ 惩（惩罚性赔偿+提高违法成本+创新生态+市场秩序）→ 辩（驳回谴责恶意诉讼+诚信公平原则+诉讼秩序）→ 总收口（整合司法手段+保护知识产权+兼顾社会公共利益+为新质生产力发展提供稳定可预期的法治环境）。
- 反向筛：F04 别把所有侵权写成"不正当竞争"；F08 别铺民法典条文；F09 别甩四句口号当结尾。
- 飞哥想说：这道题和东城一模 Q18 几乎是孪生题，结构一样——三动作 + 三价值 + 一收口；记牢"假维权拖垮真创新"这种带画面感的句子，加分点就是它。

## Q18 2026石景山一模 第18题

- 题源：表格题——案件一刑事公诉（朱某、宁某生产销售假冒泡泡玛特玩具、人民检察院提起公诉）；案件二行政诉讼（某公司不服区人社局工伤认定+区政府行政复议、起诉要求撤销）；案件三民事侵权（业主卢某开窗时窗户脱落砸张某车辆）。
- 设问入口：S04 完成下表（举证责任的分配方式及理由）。
- 命中框架：纠纷解决·三类诉讼举证规则（刑事公诉、行政诉讼、民事举证及倒置）。
- 材料触发：举证责任分配 + 程序性规则。
  1. 案件一公诉案件、由人民检察院承担被告人有罪的举证责任 → 国家机关承担证明责任、保障控辩双方公平对抗 → 这也是无罪推定原则的必然要求、有利于保障人权。
  2. 案件二原告对行政行为不服、被告区人社局及区政府应当提供作出该行政行为的证据和所依据的规范性文件 → 行政诉讼核心问题是行政行为是否合法、行政相对人在举证能力上无法与行政机关抗衡 → 由行政机关承担合法性证明责任。
  3. 案件三业主卢某开窗时窗户脱落砸车 → 原告张某需对车辆损害事实+损害与窗户坠落因果关系举证、被告卢某作为建筑物使用人需对自己没有过错承担举证责任 → 民事一般"谁主张谁举证"、建筑物坠落造成他人损害适用过错推定、关于过错的举证责任倒置。
- 答案骨架：案件二理由——行政诉讼+行政行为合法性+举证能力差距；案件三方式——张某证损害与因果关系+卢某证无过错；案件三理由——民事"谁主张谁举证"+建筑物坠落过错推定+举证责任倒置。
- 反向筛：F08 别把举证规则写成法考要件；不要把"谁主张谁举证"套到刑事和行政诉讼里；行政诉讼别写成"民事谁主张谁举证"。
- 飞哥想说：举证题的三档要分清——刑事公诉看检察院、行政诉讼看行政机关、民事看"谁主张谁举证 + 特殊情形倒置"；"过错推定"和"举证责任倒置"是这道题的两个抓手术语。

## Q19 2026西城一模 第17题

- 题源：民法典物权编节选——第 286 条业主应遵守法律法规及管理规约、相关行为应符合节约资源、保护生态环境的要求；业主大会或业主委员会对弃置垃圾、排放污染物或噪声等损害他人合法权益行为有权请求停止侵害、排除妨碍、消除危险、恢复原状、赔偿损失；第 294 条不动产权利人不得违反国家规定弃置固体废物、排放大气污染物、水污染物、土壤污染物、噪声、光辐射、电磁辐射等有害物质；第 326 条用益物权人行使权利应遵守法律有关保护和合理开发利用资源、保护生态环境的规定。
- 设问入口：S05 问"如何推动"（民法典物权编上述规定如何推动国家绿色发展）。
- 命中框架：物权·绿色原则 + 法治在绿色发展中的作用。
- 材料触发：物权 / 相邻关系 / 绿色原则。
  1. 民法典物权编上述规定贯彻民法绿色原则 → 为绿色发展提供法律依据/法律保障 → 使生态环境保护有法可依。
  2. 第 286 条业主行为应符合节约资源、保护生态环境的要求 + 第 294 条禁止弃置固体废物、排放污染物 + 第 326 条用益物权人应遵守生态环境保护规定 → 合理划定权利界限、强化法定义务、明确法律责任与权利救济 → 权利义务相统一、强化救济、协调各方利益关系。
  3. 业主大会有权请求停止侵害、排除妨碍、消除危险、恢复原状、赔偿损失 → 平衡个人利益和生态公共利益 → 促进资源开发利用绿色转型、推动可持续发展、节约资源保护环境。
- 答案骨架：法律依据（贯彻绿色原则+为绿色发展提供法治保障）→ 作用路径（划定权利界限+强化法定义务+明确责任与救济+结合 286/294/326 条具体内容）→ 意义价值（平衡个人利益与公共利益+绿色转型+可持续发展）。
- 反向筛：F07 别把物权编写成纯"所有权之争"；F10 别套双循环/宏观调控；不要忘把"民法绿色原则"四个字直接写出来。
- 飞哥想说：这是民法典"绿色原则"的标准考题，三段式"法律依据 + 作用路径 + 意义价值"是阅卷固定阶梯；"权利与义务相统一""权利救济""可持续发展"这三个词要一起出，缺一个掉一档。
