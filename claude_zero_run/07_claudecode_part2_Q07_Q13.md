# 07 下篇 全题题链（飞哥版）· part2 Q07–Q13

## Q07 2025丰台一模 第19题

- 题源：原告方近40名高管和技术人员离职入职被告方，利用原单位技术信息申请12件实用新型专利；最高院适用2倍惩罚性赔偿，判赔6.4亿余元，入选保护科技创新典型案例。
- 设问入口：S03 问意义/价值。
- 命中框架：知识产权·商业秘密路径（反不正当竞争 + 惩罚性赔偿）；意义四层。
- 材料触发：知识产权 / 商业秘密 / 不正当竞争。
  1. 被告利用在原单位接触的技术信息申请专利 → 不正当竞争 → 维护当事人合法权益、平衡双方权利与义务、严惩侵权行为。
  2. 适用2倍惩罚性赔偿、判赔6.4亿创新高 → 提高侵权成本 → 规范市场竞争秩序、营造公平市场环境、以法治力量鼓励创新。
  3. 案件入选典型案例 → 司法导向作用 → 为同类案件提供范例、提升审判质量与效率、增强司法公信力、推动法治中国建设。
- 答案骨架：企业层（维护合法权益+严惩侵权）→ 创新层（规范竞争+鼓励创新）→ 司法层（典型范例+司法公信力）→ 法治层（法治信仰+依法办事）。
- 反向筛：F04 商业秘密走反不正当竞争路，别只挂"专利侵权"；F05 不要堆"公平正义"裸口号；不要写"优化营商环境"这种非细则承认的表达，要写"公平市场环境/法治化营商环境"。
- 飞哥想说：这道题的稳分动作是"主体分层"——企业-创新-司法-法治四级阶梯走完，6 分以上稳；写到法治中国得先有铺垫，不能裸落。

## Q08 2025房山一模 第19题

- 题源：与丰台一模同案——A集团技术秘密被B集团挖人取走、申请12件实用新型专利推出电动汽车；法院适用2倍惩罚性赔偿判赔6.4亿余元。
- 设问入口：S03 问意义/价值（"真创新"受"真保护"的法治价值）。
- 命中框架：知识产权·商业秘密路径 + 法治价值四层。
- 材料触发：知识产权 / 商业秘密 / 不正当竞争。
  1. B集团挖人窃取技术秘密 → 不正当竞争行为定性 → 法院判决打击违法行为、促使企业依法诚信经营、维护公平竞争环境、促进市场经济健康发展。
  2. 法院适用高额惩罚性赔偿 → 提高侵权成本 → 强化知识产权保护、激励研发投入、激发创新活力与科技进步。
  3. 离职高管利用原单位技术信息 → 劳动者权利义务相统一 → 劳动者负有保守商业秘密义务、严守职业道德。
- 答案骨架：定性"不正当竞争"→ 打击违法+诚信经营+市场秩序 → 惩罚性赔偿+激发创新 → 劳动者守密义务 → 收口公正司法、社会主义核心价值观。
- 反向筛：F04 别把"商业秘密侵权"全写成"专利侵权"；F05 价值要分层；写知识时单纯堆"有利于"不结合材料只能拿一半分。
- 飞哥想说：这题和丰台同源，但多了"劳动者保密义务"这条暗线，写到劳动者那一层就能比别人多一分。

## Q09 2025昌平二模 第20题（第2问）

- 题源：表格题——李某用 AI 绘图软件制作图片发自媒体、刘某抹去水印当配图，法院支持李某诉请；另一案小王信用卡透支催缴不还。
- 设问入口：S04 完成裁判理由。
- 命中框架：知识产权·著作权路径 + 合同·诚信履行。
- 材料触发：人格权·AI 著作权 + 合同·诚信原则。
  1. AI 绘图图片具有"智力成果"和"独创性"要件 → 受著作权法保护、李某享有著作权 → 著作权人限于自然人/法人/非法人组织，AI 不能成为著作权主体。
  2. 刘某未经许可使用并抹去水印 → 侵害信息网络传播权和署名权 → 属侵权行为、应承担侵权责任。
  3. 信用卡协议系双方真实意思表示、合法有效 → 透支后经催缴仍不还款构成违约 → 违背诚信原则和全面履行原则、应承担违约责任。
- 答案骨架：先写法律依据（著作权法）→ 认定独创性 + 著作权归属 → 行为定性（侵害信息网络传播权/署名权）→ 责任承担。信用卡案：合同有效 → 违约 → 违背诚信与全面履行 → 承担违约责任。
- 反向筛：F08 别铺著作权法全文；AI 不能写成著作权主体；信用卡那条别只写"违法"，要落"违约责任"。
- 飞哥想说：AI 作品的两个钥匙词是"智力成果"和"独创性"，缺一个就不受保护；信用卡那条记住"诚信原则 + 全面履行原则"这套餐。

## Q10 2025朝阳一模 第19题

- 题源：王某通过 A 机票代销平台买票，平台搭售 10 元外卖红包无法拒绝、不能清楚知悉支出细节；王某诉请退还 310 元并三倍赔偿，法院支持。
- 设问入口：S01 问诉求/裁判理由。
- 命中框架：合同·欺诈 + 消费者权益保护法·三倍赔偿。
- 材料触发：合同欺诈 / 消费者欺诈 / 退一赔三。
  1. 王某通过平台购票、平台已出票 → 要约+承诺、合同成立 → 但合同成立不等于生效。
  2. A 公司擅自搭售 10 元红包 + 王某不能清楚知悉费用支出细节 + 无法拒绝 → 主观欺诈故意 + 客观欺诈行为 + 致违背真实意思 → 王某可请求人民法院撤销合同。
  3. A 公司提供服务有欺诈行为 → 依照消费者权益保护法、增加赔偿三倍、不足五百的为五百 → 侵犯消费者知情权、公平交易权、自主选择权。
- 答案骨架：合同成立 + 不等于生效 → 欺诈三要件踩齐 → 援引民法典请求撤销 → 援引消费者权益保护法支持三倍赔偿 → 收口意义层（维护消费者合法权益、规范电商平台经营、优化网络消费环境、促进数字经济健康发展）。
- 反向筛：F03 别一看"消费"就上三倍，必须先证欺诈；不要写"消费者权益法/消费者保护法"漏字；"看到网络就写隐私权"是低级错。
- 飞哥想说：这道题两个诉讼请求都要回应——退还 310 元走"撤销合同"，三倍赔偿走"消费者权益保护法"。两条路径都要踩齐，缺一条就掉一档。

## Q11 2025朝阳二模 第20题（第1问）

- 题源：刘先生在入户门安装智能可视门铃摄像头（人脸识别、180度广角、自动人体侦测、云存储）；孙女士认为可能拍到自家日常出行规律及动态信息，要求拆除。
- 设问入口：S02 问评析/分析案例。
- 命中框架：人格权·隐私权 + 物权·共有部分 + 相邻关系四原则。
- 材料触发：人格权·隐私 + 物权·相邻 + 民法典 271 条共有部分。
  1. 摄像头能拍到孙女士家日常进出信息、与财产安全和私人生活习惯高度关联 → 属个人隐私范畴/具有隐私性质的人格利益 → 刘先生侵犯孙女士隐私权、应承担侵权责任。
  2. 楼道属双方专有部分以外的共有部分 → 双方享有共有与共同管理的权利 → 在共有部分设备使用应有边界。
  3. 刘先生虽有权保障自身财产安全、但安装设备范围与功能已超出必要限度、未尽妥善注意义务 → 主观有过错 → 相邻权利人应按"有利生产、方便生活、团结互助、公平合理"原则处理相邻关系。
- 答案骨架：定性"侵犯隐私权 + 应承担侵权责任"→ 确权（共有部分共有共管）→ 事实分析（日常出行规律等动态信息）→ 主观过错（超出合理限度+未尽注意义务）→ 收口相邻关系四原则。
- 反向筛：F07 别把相邻关系写成"侵犯所有权/用益物权"；隐私权字别写错（侵/稳）；事实部分不能只写"拍到家门口"，要写"日常出行规律等动态信息"才得分。
- 飞哥想说：这道题三条线齐踩——侵权 + 共有部分 + 相邻关系四原则，谁缺一线就掉一档；调解员建议那一小问要写"拆除设备 + 清除已拍信息"两件齐。

## Q12 2025海淀期中 第21题（第1问）

- 题源：1950年婚姻法"由男方清偿"vs民法典"夫妻共同债务共同偿还/协议清偿/法院判决"；小海认为1950年规定不合理、民法典才符合良法要求。
- 设问入口：S02 问评析。
- 命中框架：法治·良法标准（符合国情和实际、符合社会发展需求）+ 权利与义务相对应。
- 材料触发：婚继·夫妻共同债务 + 良法之治。
  1. 新中国成立初期尚未完全建立夫妻共同财产制度、男方经济地位更强 → 根据权利与义务相对应原则、由男方清偿具有合理性 → 有利于保护当事各方利益。
  2. 时代发展、夫妻经济状况变化 → 具备"共同偿还""协议清偿"的条件 → 民法典规定符合现实需要。
  3. 良法应符合国情和实际、符合社会发展需求 → 婚姻法与民法典的规定都符合良法要求 → 不能用今天的标准否定 1950 年的规定。
- 答案骨架：表态"小海观点片面"→ 列 1950 年规定的合理性（权利与义务相对应+国情）→ 列民法典进步性（现实条件变化+共同清偿）→ 收口良法标准（符合国情和实际+符合社会发展需求+两部法律都是良法）。
- 反向筛：F08 别只判对错；评析题必须正反两面 + 法律历史条件；不要只写民法典进步、不写 1950 年合理性。
- 飞哥想说：评析题最稳的开场就是"片面"二字；这道题的得分密码是"良法要看时代条件"，把权利与义务相对应原则写进去就是亮点。

## Q13 2025石景山一模 第20题

- 题源：表格题——案例一集成电路研发委托合同违约、案例二"好省"APP被"超好省"软件名称图标混淆。
- 设问入口：S04 完成下表（裁判理由 + 对发展新质生产力的保障作用）。
- 命中框架：合同·违约责任 + 知识产权·混淆型不正当竞争。
- 材料触发：合同违约 + 反不正当竞争·混淆。
  1. 案例一：被告超期且未提交达标研发成果 → 依据民法典、当事人不履行合同义务或履行不符合约定的、守约方可要求承担违约责任 → 被告构成违约、应返还研发费用、支付违约金。
  2. 案例一保障作用 → 维护守约方合法权益、激励从业者秉持契约精神和敬业态度、提升创新研发能力 → 促进集成电路产业扩容提质、为新质生产力发展提供重要支撑。
  3. 案例二："好省"软件经长期宣传有影响力、"超好省"名称图标内容相近造成公众混淆 → 依据反不正当竞争法、构成不正当竞争 → 应承担停止侵害、赔偿损失的侵权责任。
- 答案骨架：①裁判理由格——民法典违约责任链条 + 守约方救济；②保障作用格——契约精神 + 创新研发 + 产业升级 + 新质生产力。案例二保障作用——依法打击不正当竞争、促进公平竞争、维护市场秩序、增强市场活力、营造创新环境、护航软件产业高质量发展。
- 反向筛：F04 软件名称图标混淆走反不正当竞争路、别全部挂"著作权侵权"；F08 别铺民法典条文；写"对新质生产力的保障作用"时不能只甩"促进新质生产力"一句空话。
- 飞哥想说：表格题里"对发展新质生产力的保障作用"这一格是必拿分点，要把"打击侵权 + 创新环境 + 产业升级"三件套写齐；契约精神和敬业态度是这道题的高频得分句。
