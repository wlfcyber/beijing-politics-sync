命中框架：问裁决理由 + 平台用工三从属性 + 意义价值

材料触发：合同成立、履行、违约、格式条款；平台用工、三从属性、新就业形态；调解、仲裁、诉讼、举证责任

1. 平台派单并用制度奖惩王某 → 人格从属性 → 劳动者受平台管理，不能只看合同名称
