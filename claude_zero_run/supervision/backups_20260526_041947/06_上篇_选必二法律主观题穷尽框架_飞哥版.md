# 上篇：选必二法律主观题穷尽框架

## 1. 先导：这类题到底考什么

选必二法律主观题，不是背法条，不是法考，也不是必修三。

它考的是一件事：把生活纠纷翻译成法律得分点。

材料里有人、事、冲突、请求、结果。你要做四步：先定题型，再翻材料，再写法律，最后收价值。

飞哥想说：法律题不是“我觉得谁有理”。考场只认“材料事实 → 法律关系 → 责任或价值”。

## 2. 总法门

### 定题型

看到什么：问理由、问诉求、问裁判理由、问完成下表、问评析、问意义。

要干什么：先判断它要你写“为什么判”，还是“怎么评价”，还是“有什么价值”。

别写什么：不要一上来背民法典，不要把法院题写成公正司法两句口号。

飞哥想说：设问是方向盘，材料是油门，法律是刹车。方向错了，写得越多越远。

### 翻材料

看到什么：合同、平台、消费者、知识产权、继承、相邻、举证、调解。

要干什么：把生活话翻成法律话。

别写什么：不要抄“他很生气”“影响很大”，要翻成“合同违约”“名誉受损”“隐私被侵害”“权利边界被突破”。

飞哥想说：材料不是拿来抄的，是拿来翻译的。

### 写法律

看到什么：一方有请求，法院或仲裁支持，或者让你补裁判理由。

要干什么：写法律关系、法定规则、责任承担。

别写什么：不要堆完整法条，不要把所有题都写成合同、侵权、程序三件套。

飞哥想说：短法条，强材料，准结论。

### 收价值

看到什么：问意义、价值、典型案例、新质生产力、数字经济、养老、绿色发展。

要干什么：从个案往外推：保护谁、规范谁、形成什么秩序。

别写什么：不要裸写公平正义、核心价值观、法治中国。

飞哥想说：意义题不是喊口号，是把本案的规则效果说清楚。

## 3. 设问分诊穷尽

### 问理由 / 诉求 / 判决结果

看到：说明法院为什么支持、为什么裁决、为什么判决。

第一步：先找谁请求什么，再找请求靠什么事实成立。

答案骨架：事实成立 → 法律定性 → 法定责任 → 支持请求。

反向筛：不要只说“维护合法权益”；不要把请求写丢。

来源题：2024朝阳二模__Q17、2025朝阳一模__Q19、2026石景山一模__Q18。

飞哥想说：这类题最稳的句式是“因为材料中……，所以法律上……，因此应当……”。

### 问裁判理由 / 补表

看到：完成裁判理由、以案说法、填写民法原则、填写保障作用。

第一步：先看表头，每格只答表头那一层。

答案骨架：材料事实 → 法律翻译 → 表头要求的话。

反向筛：不要把意义写进裁判理由格，不要把法律依据写成空价值。

来源题：2025东城一模__Q19、2025东城二模__Q19、2025石景山一模__Q20、2025昌平二模__Q20。

飞哥想说：补表题的表头就是小设问，别自由发挥。

### 问评析

看到：评析、谈认识、观点是否合理。

第一步：先表态，通常是“片面”“部分合理”。

答案骨架：合理处 → 不足处 → 正确规则。

反向筛：不要只判对错；不要漏材料里的历史条件、权利义务条件。

来源题：2025海淀期中__Q21、2024西城一模__Q18。

飞哥想说：评析题不喜欢绝对句，喜欢分条件裁断。

### 问意义 / 价值

看到：社会价值、法治价值、典型意义、护航新质生产力、守护向新力。

第一步：先把本案保护的对象找出来。

答案骨架：保护当事人 → 规范行为人 → 稳定行业秩序 → 形成法治价值。

反向筛：意义不等于“有利于公平正义”单句。

来源题：2024海淀二模__Q19、2025丰台一模__Q19、2025房山一模__Q19、2026东城一模__Q18、2026朝阳一模__Q18。

飞哥想说：价值题要有阶梯，不要一脚跳到法治中国。

### 问调解 / 维权 / 路径

看到：调解、定分止争、解决纠纷、举证责任、诉讼路径。

第一步：先分纠纷类型，再分解决机制。

答案骨架：纠纷类型 → 调解/仲裁/诉讼/举证 → 实体规则 → 效果。

反向筛：不要所有纠纷都直接“起诉”；不要把行政举证和民事举证混写。

来源题：2025顺义一模__Q19、2026石景山一模__Q18、2025东城一模__Q19。

飞哥想说：程序不是套话，是谁来证明、用什么机制解决。

## 4. 材料触发穷尽

### 合同成立 / 履约 / 违约

看到：签订合同、按约定、未按期、成果不达标、违约金。

翻成法律话：合同成立有效，一方未按约履行，应承担违约责任。

高频句：当事人应当按照约定全面履行自己的义务；履行不符合约定的，应承担继续履行、赔偿损失、支付违约金等责任。

别写：不要随便写合同无效；不要把违约写成侵权。

来源题：2025石景山一模__Q20、2025东城一模__Q19。

飞哥想说：合同题先看有没有约定，再看有没有不履行。

### 格式条款 / 平台搭售 / 隐藏费用

看到：服务条款、小字提示、预先扣款、搭售、不能拒绝。

翻成法律话：经营者负提示说明义务；排除消费者主要权利或加重责任的条款不能当然有效。

高频句：格式条款未充分提示说明，不能作为免除责任的依据。

别写：不要把所有条款都叫霸王条款。

来源题：2025东城二模__Q19、2025朝阳一模__Q19。

飞哥想说：格式条款题的核心不是字小，是消费者有没有真正知道和选择。

### 消费者欺诈 / 知情权 / 三倍赔偿

看到：消费者、经营者、虚假或隐瞒、误导购买、违背真实意思。

翻成法律话：经营者存在欺诈，消费者可请求撤销并主张惩罚性赔偿。

高频句：经营者提供商品或服务有欺诈行为的，应按消费者要求增加赔偿。

别写：消费者不等于三倍赔，必须先证明欺诈。

来源题：2025朝阳一模__Q19。

飞哥想说：退一赔三不是万能钥匙，欺诈才是钥匙孔。

### 平台用工 / 三从属性

看到：平台派单、奖惩制度、数据和价格由平台掌握、以平台名义服务。

翻成法律话：人格、经济、组织从属性明显，可认定事实劳动关系。

高频句：不能仅看合同名称，应根据实际用工管理和从属性判断劳动关系。

别写：平台不等于劳动关系，灵活就业也不等于无劳动保护。

来源题：2024朝阳二模__Q17。

飞哥想说：三从属性就是这类题的三颗钉子。

### 知识产权 / 不正当竞争

看到：技术秘密、商业标识、唤醒词、AI图片、软件名称图标、惩罚性赔偿。

翻成法律话：先分权利类型，再定侵权或不正当竞争，再写责任和价值。

高频句：依法保护知识产权，规制不正当竞争，提高违法成本，维护公平竞争和创新环境。

别写：不要把所有知识产权都写成不正当竞争；著作权、商业秘密、混淆行为要分开。

来源题：2025丰台一模__Q19、2025房山一模__Q19、2025西城期末__Q19、2026朝阳一模__Q18。

飞哥想说：知识产权题的第一刀是分类型。

### 人格权 / 隐私 / 名誉 / 个人信息

看到：无事实依据辱骂诋毁、摄像头拍出行规律、抹去署名。

翻成法律话：名誉权、隐私权、署名权分别处理。

高频句：行为造成社会评价降低，构成名誉权侵害；拍摄日常出行规律，涉及隐私利益。

别写：名誉权不等于隐私权；“拍到门口”不够，要写日常出行规律等动态信息。

来源题：2025东城二模__Q19、2025朝阳二模__Q20、2025昌平二模__Q20。

飞哥想说：人格权题最怕权利名称叫错。

### 多主体过错 / 举证责任

看到：谁主张谁举证、行政诉讼、建筑物脱落、使用人无过错。

翻成法律话：先分诉讼类型，再分证明责任。

高频句：行政机关对行政行为合法性承担证明责任；民事诉讼一般谁主张谁举证，特殊情形可过错推定。

别写：不要把行政诉讼和民事诉讼混用。

来源题：2026石景山一模__Q18。

飞哥想说：举证题不是背规则，是看谁离证据更近。

### 相邻关系 / 物权边界 / 绿色原则

看到：邻居、楼道、摄像头、加装电梯、垃圾排放、绿色发展、物权编。

翻成法律话：相邻权利人应按有利生产、方便生活、团结互助、公平合理处理关系；物权行使不能突破生态和他人权益边界。

高频句：权利行使应有边界，不能损害他人合法权益和社会公共利益。

别写：不要把相邻关系写成单纯所有权之争。

来源题：2025朝阳二模__Q20、2026西城一模__Q17。

飞哥想说：所有权也有边界，边界就是民法的温柔和硬度。

### 继承 / 赡养 / 监护

看到：老人、子女、照护、遗产、遗赠扶养协议、精神慰藉。

翻成法律话：先看协议和义务履行，再看遗产归属和赡养内容。

高频句：成年子女对父母负有赡养义务，赡养包括经济供养、生活照料和精神慰藉。

别写：不要把遗赠扶养协议直接写成法定继承。

来源题：2024海淀二模__Q19、2024西城一模__Q18、2025东城一模__Q19。

飞哥想说：家庭法律题有情，但得分靠规则。

### 纠纷解决 / 调解 / 仲裁 / 诉讼

看到：调解、仲裁、诉至法院、行政裁决、举证责任。

翻成法律话：不同机制解决不同纠纷，各有功能。

高频句：调解有利于高效化解矛盾，仲裁体现专业和便捷，诉讼具有权威性和终局性。

别写：不要把调解、和解、仲裁、诉讼混成一个词。

来源题：2025顺义一模__Q19、2025东城一模__Q19、2026石景山一模__Q18。

飞哥想说：机制题要说清“为什么用它”。

## 5. 反向筛查穷尽

法院 ≠ 公正司法。要写法院在本案中调解、判决、驳回、支持、分配责任的具体动作。

法治 ≠ 全面依法治国。选必二先写生活法律关系，再收一句法治价值。

企业 ≠ 经济生活。企业题先看合同、竞争、劳动、消费者、知识产权。

平台 ≠ 劳动关系。三从属性齐了，才敢认定事实劳动关系。

消费者 ≠ 三倍赔偿。有欺诈，才写退赔。

创新 ≠ 只写保护创新。要写保护什么权利、打击什么行为、稳定什么秩序。

意义 ≠ 裸价值。意义要按对象分层。

多主体 ≠ 一锅炖。先分主体，再分行为，再分责任。

案例 ≠ 法考。高考要短法条、强材料、准结论。

## 6. 高频得分语言

### 诉求支持类

最推荐背：材料中……已经符合……的法律要件，因此法院支持……请求。

什么时候用：问法院为什么支持、为什么判。

什么时候别用：问意义价值时，不要只停在诉求。

飞哥想说：支持类题一定要落到“支持什么”。

### 裁判理由类

最推荐背：先定法律关系，再接材料事实，最后写责任承担。

什么时候用：补裁判理由、分析判决。

什么时候别用：补表格的“典型意义”格。

飞哥想说：裁判理由不是感受，是理由链。

### 评析行为类

最推荐背：该观点看到了一面，但忽视了……，应结合……全面判断。

什么时候用：评析观点、谈认识。

什么时候别用：题目只让写理由时不要强行两面。

飞哥想说：评析题喜欢“有条件地裁断”。

### 合同类

最推荐背：依法成立的合同，对当事人具有法律约束力；一方不履行或履行不符合约定，应承担违约责任。

什么时候用：合同、研发、培训协议、装修。

什么时候别用：人格权、相邻关系、举证题。

飞哥想说：合同先看约定，再看履行。

### 劳动类

最推荐背：判断劳动关系不能只看合同名称，应看人格、经济、组织从属性。

什么时候用：平台用工、新就业形态。

什么时候别用：普通承揽、临时合作没有管理事实时。

飞哥想说：三性不全，别急着认劳动关系。

### 消费者类

最推荐背：经营者提供服务有欺诈行为，侵害消费者知情权和自主选择权，应承担退还及惩罚性赔偿责任。

什么时候用：隐形搭售、虚假诱导、不能拒绝。

什么时候别用：普通质量瑕疵、没有欺诈事实。

飞哥想说：三倍赔偿前先写欺诈。

### 人格权类

最推荐背：行为造成他人社会评价降低，构成名誉权侵害；拍摄日常出行规律等私人信息，涉及隐私权益。

什么时候用：自媒体诋毁、摄像头、个人动态信息。

什么时候别用：单纯财产纠纷。

飞哥想说：权利名称写错，后面再漂亮也危险。

### 知识产权 / 不正当竞争类

最推荐背：依法保护知识产权，打击不正当竞争，提高违法成本，维护公平市场秩序和创新生态。

什么时候用：技术秘密、唤醒词、AI作品、软件混淆、植物新品种。

什么时候别用：普通合同违约题。

飞哥想说：先分类型，后写保护。

### 纠纷解决类

最推荐背：不同纠纷可通过调解、仲裁、诉讼等机制解决，依法定分止争，维护合法权益和社会秩序。

什么时候用：调解、仲裁、举证、行政裁决。

什么时候别用：实体法理由已经足够，且设问没有程序入口。

飞哥想说：程序分不是赠品，是题目给你的路标。

### 意义价值类

最推荐背：该案保护了……合法权益，规范了……行为，维护了……秩序，形成……法治导向。

什么时候用：意义、价值、典型案例。

什么时候别用：只问具体判决理由时，价值放最后一句即可。

飞哥想说：意义题用“保护谁、规范谁、影响谁”三问自检。

## 7. 最短记忆版

1. 法律题不是背法条，是翻译纠纷。
2. 先定题型，再翻材料，再写法律，再收价值。
3. 问理由：事实 → 法律 → 责任。
4. 问评析：合理处 → 不足处 → 正确规则。
5. 问意义：保护谁 → 规范谁 → 稳定什么秩序。
6. 补表题：表头让写什么，就只写什么。
7. 合同看约定和履行。
8. 平台看三从属性。
9. 消费者先证欺诈，再谈三倍赔。
10. 知识产权先分类型。
11. 人格权先叫准权利名称。
12. 相邻关系看权利边界。
13. 继承先找协议和义务履行。
14. 举证先分诉讼类型。
15. 任何价值句，都要回到本案。
