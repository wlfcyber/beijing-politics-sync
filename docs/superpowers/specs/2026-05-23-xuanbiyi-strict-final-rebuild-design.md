# 选必一严格最终版重建设计

## 背景

用户要求把选必一《当代国际政治与经济》主观题宝典做成最终成品版。范围为桌面上的 `2024各区模拟题`、`2025各区模拟题`、`2026各区模拟题`，只处理主观题。入主表采用严格证据口径：必须能定位正式评分细则、评标、阅卷细则或讲评材料中的选必一采分点；普通参考答案不能冒充主链。

当前阶段稿已经完成一轮反合并修正：学生主表暂为 314 条题例、103 个核心答题点，12 个无法安全拆分的合并题例已移出主表并进入回源重做队列。2026 二模题目层面尚未闭合，至少顺义 Q20 必补，丰台 Q22、房山 Q21、丰台 Q20 需逐题定位细则并裁决。

## 成功标准

- 三年模拟题目录下所有主观题都有清单记录，包含试卷来源、细则来源、题号、模块判断、入库状态。
- 所有入主表题例都有可核验 `细则位置`，且 `术语` 来自评分源原词。
- 题例层面禁止多题合并；同一核心只在框架层合并，题例必须保持单题展示。
- 缺口题和12个残留合并题例必须走 `Codex独立厚稿 -> ClaudeCode独立厚稿 -> GPT Pro主融合 -> Claude Opus 4.7 Adaptive复审 -> Codex证据回查落盘`。
- 最终交付包含学生厚版 Markdown/Word、考前导航版 Markdown/Word、全量覆盖清单、纳入/未纳入清单、细则定位表、模型证据链、Governor 审计。

## 流程

1. 全量扫描：枚举三年模拟题目录的所有文件，提取 Word/PDF/PPT/旧 doc 文本，保留源文件和提取日志。
2. 主观题定位：按题号和主观题结构识别第16题以后或试卷中实际主观题，建立题目清单。
3. 细则定位：为每道主观题匹配答案、细则、评标、阅卷总结、讲评 PPT/PDF/OCR 文本；找不到时标注为“细则未定位”，继续保留检索证据。
4. 选必一筛选：按材料、设问、评分源术语判断是否含选必一采分点，排除选择题和其他模块。
5. 覆盖对照：将全量题目清单与当前宝典、BATCH_014、12个残留合并题例、2026二模缺口逐题比对。
6. 四线生产：对缺口题建源包，分别生成 Codex 厚稿和 ClaudeCode 厚稿，再提交 GPT Pro 主融合，最后由 Claude Opus 4.7 Adaptive 复审。
7. 合入与验收：证据回查后合入终稿，重建 Word，执行反合并、覆盖、归桶、字段、来源等级和导航同步检查。

## 约束

- 不得因为一时找不到细则就判定题目无细则；只能先写“细则未定位”并扩大检索路径。
- 普通参考答案最多进入降级附录或待裁决区，不能进入严格主表。
- 经济全球化节点必须按表述可替代性合并，不得把开放型世界经济、开放型经济、规则制定、多边贸易、贸易自由化等宽泛压成一类。
- `独立自主和平外交政策`、`和平共处五项原则`、中国倡议、中国担当归入 `中国`；`新型国际关系`、全球治理方向、国际关系民主化归入 `政治多极化`。
- 最终声明必须区分“材料已覆盖”“题目已闭合”“严格最终版已通过”三种状态。
