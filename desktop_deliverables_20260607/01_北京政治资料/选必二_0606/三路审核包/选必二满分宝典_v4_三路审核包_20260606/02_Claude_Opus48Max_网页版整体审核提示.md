# Claude Opus 4.8 Max 网页版整体审核提示

请审核随包的 `选必二法律与生活_满分宝典_v4_0606_精简截图校准版.pdf` 或 DOCX。

## 审核定位

这不是逐题回源审核，逐题回源由 Claude cowork 应用端负责。你负责整体结构、学生可读性、语言干净度、逻辑顺序、是否符合“简单、全面、高效”的满分宝典目标。

## 请重点判断

1. 五环框架是否便于学生复习：知识仓是否够短，例题是否跟框架自然对应。
2. 是否还有废话、包装话术、口号、无依据的“必背句”、自拟聪明栏目。
3. 材料、设问、答题点的顺序是否高效；有原题图时是否截图优先且文字不妨碍阅读。
4. 答题点是否看起来像“细则可给分的点”，而不是泛泛答案。
5. 全文是否过长、重复、拥挤；哪些内容可以删而不损失得分。
6. 黑白简洁版式是否足够稳定；是否需要少量分隔线、目录或页码改进。

## 输出格式

### 总体结论

通过 / 有条件通过 / 不通过

### 结构问题

列出阻断或明显影响使用的问题。

### 语言问题

列出需要删除或改写的废话、口号和不必要表述。

### 效率改进

列出能让学生更快使用的具体调整。

### 最终建议

是否可以作为候选交给 GPT-5.5 Pro 终审。
