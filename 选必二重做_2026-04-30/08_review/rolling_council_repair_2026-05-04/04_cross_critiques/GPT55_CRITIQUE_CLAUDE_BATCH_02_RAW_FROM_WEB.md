## 1. 我同意 Claude 的哪些框架变化，为什么

### 1.1 同意继续强化法律小问入框闸门

Claude 指出题12多案例选择题、题14的12348热线题存在关键词触发误入框风险，这个批评应采纳。题12更像多案例综合判断，题14更像政府公共服务和法治社会建设类表达，不能只因出现物权、婚姻、继承、合同等词就进入某一法律域。具体是否剔除或重归位，**需本地核验**。

Codex 应把入框闸门写成可执行规则：

1. 是否有具体民事主体之间的权利义务关系。
2. 是否要求运用《法律与生活》作法律判断。
3. 是否有责任、效力、归属、救济、权利边界等法律结果。
4. 混合模块只抽法律小问，非法律小问不进入法律框架。

### 1.2 同意物权题需要恢复有限识别动作

Claude 认为用益物权与担保物权、使用价值与交换价值不能完全删除，这个修正有道理。Batch02 题卡中多道选择题确实围绕抵押、质押、登记、交付、使用价值、交换价值设干扰项，说明物权题需要轻量识别卡。具体是否为稳定采分点，**需本地核验**。

但这只能恢复为**识别动作**，不能恢复成完整物权法体系。学生版建议写成：

> 用益看使用，担保看交换；动产重交付，不动产重登记；抵押、质押常考优先受偿或对抗问题。

### 1.3 同意继承位序需要学生小卡

Claude 提出的协议优先于遗嘱、遗嘱优先于法定继承，可以作为继承题的重要识别卡。Batch02 中遗赠扶养协议和遗嘱冲突的题卡确实支持这一方向，具体能否称为通用口诀，**需本地核验**。

更稳的学生版表达：

> 继承先看协议，再看遗嘱，最后看法定继承；同时核身份、扶养事实、丧失继承权。

### 1.4 同意相邻关系要从普通侵权或赔偿题中拆出来

Claude 对题20的压力测试很重要。通州加装电梯题的核心不应写成赔偿题，主要落点应是相邻关系处理、必要便利、程序合法、事实是否造成损害、停止妨害、邻里和谐价值。具体细则采分结构，**需本地核验**。

建议学生版相邻关系卡：

> 相邻不动产，必要利用，尽量避免损害，按有利生产、方便生活、团结互助、公平合理处理；结果落到提供便利、停止妨害、排除妨碍或合理协调。

### 1.5 同意题9评分摘录污染必须进入流程硬步骤

Claude 指出题9评分摘录混入外交内容，这类异常不应 case by case 临时处理，应列入 Codex 流程硬步骤。Batch01 和 Batch02 都出现了类似污染风险，是否真实错配，**需本地核验**。

建议 Codex 加一条证据闸门：

> formal_or_scoring_source 摘录必须与题号、设问、分值、材料主题逐项对齐；若出现其他模块、其他题号或无关评分点，先隔离，不推动主干。

### 1.6 同意选择题不强求三问全套

Claude 说选择题不需要完整三问，这一点应采纳。选择题更适合“定主体、定关系、排错项、落答案”，不宜写成完整主观题链。具体各题错项类型，**需本地核验**。

学生版可写成：

> 选择题只做小四步：看主体，看关系，看规则，看错位。

### 1.7 同意命题路径模板必须继续重写

Claude 批评 Batch02 选择题命题路径仍是统一模板，这一点应直接吸收。多案例综合题、物权登记题、相邻关系题、继承位序题、家庭财产题的命题机制不同，不能统一写成题干事实到题肢关键词再排除错位。

## 2. 我反对或降级 Claude 的哪些变化，为什么

### 2.1 反对用 Batch02 直接撤回交易与财产、人身与家庭五域方案

Claude 建议撤回交易与财产、人身与家庭，把学生五域改为合同与消费、物权与相邻、家庭与继承、劳动与职业、第五域待定。这个建议不宜直接采纳。Batch02只覆盖物权、相邻、家庭、继承，不能用本批次重画全量五域，尤其不能让第五域待定。项目要求框架要能覆盖所有题，不能因本批样本把人格权、侵权、知识产权、竞争秩序、程序救济暂时悬空。

更稳的融合方案是：

| 学生前台五域  | 二级识别卡               | 与原五域兼容方式                               |
| ------- | ------------------- | -------------------------------------- |
| 交易与财产   | 合同消费卡；物权相邻卡         | 原合同消费者劳动中的合同消费回流此域；原物权相邻继承家庭中的物权相邻回流此域 |
| 人身与家庭   | 人格侵权卡；家庭继承卡         | 原人格权侵权回流此域；原物权相邻继承家庭中的家庭继承回流此域         |
| 劳动与职业   | 劳动关系卡；解除补偿卡；新就业形态卡  | 原合同消费者劳动中的劳动部分单独成域                     |
| 创新与竞争   | 知识产权卡；不正当竞争卡        | 原知识产权不正当竞争保留                           |
| 救济与公共利益 | 程序救济卡；生态公益卡；公共法律服务卡 | 原纠纷解决生态公益与新业态回流此域                      |

这样既承认 Batch02 中物权与家庭的采分动作不同，又避免把五域拆成七域或让第五域空缺。学生负担也更可控：背五个域，做题时选二级卡。

### 2.2 降级物权三要件作为硬主干

Claude 把物权三要件设立、变动、对抗列为物权域硬识别主干。这个方向有用，但表达过法考化。高中学生不宜系统背设立、变动、对抗三层框架，尤其还涉及登记生效、登记对抗、交付生效、善意取得等细分。具体是否稳定采分，**需本地核验**。

建议降级为物权选择题识别卡：

> 看登记或交付是否完成；看能否对抗第三人；看抵押、质押是否只产生担保权；看所有权是否转移。

### 2.3 反对把协议大于遗嘱大于法定作为继承域唯一硬识别

这条口诀适合遗赠扶养协议与遗嘱冲突题，但不能覆盖全部继承题。Batch02 题卡还涉及法定继承位序、遗嘱有效性、继承权丧失、继承人以外扶养较多人适当分得遗产等内容，具体权重**需本地核验**。

建议改为三张继承小卡：

1. **先后卡：** 协议，遗嘱，法定。
2. **身份卡：** 第一顺序、第二顺序、是否有扶养事实。
3. **排除卡：** 遗弃、虐待、伪造遗嘱等是否导致丧失继承权。

### 2.4 降级相邻关系四原则作为唯一答题骨架

相邻关系四原则很重要，但不能称为唯一骨架。题13暖气管路更像业主共有设施和擅自处分，题20加装电梯才更典型地触发相邻关系原则。具体归属和采分，**需本地核验**。

相邻关系题应按场景分流：

| 场景       | 主抓手                      |
| -------- | ------------------------ |
| 共用设施被截断  | 共有财产、无权擅自处置、恢复或排除妨害      |
| 加装电梯影响采光 | 程序合法、事实影响、必要便利、停止妨害、邻里和谐 |
| 管线、通行、排水 | 必要利用、尽量避免损害、合理便利         |

### 2.5 反对把一核表达进意义题首句作为学生版硬动作

Claude 把一核表达进意义题首句列为 accept。这个过满。题20细则中出现以事实为根据、以法律为准绳，能证明该题可以这样写，不能推出所有意义题都要首句套一核。是否多题稳定给分，**需本地核验**。

建议定位为：

> 教师版保留为高分句；学生版提示在法官判决、裁判理由、司法意义题中优先使用。不能要求每一道意义题都机械开头。

### 2.6 降级必须写法律具体名称为全局硬规则

Claude 提到题20写按照法律不给分，必须写民法典，这对该题很关键；Batch01也出现消费者权益保护法字样要求。但这仍不能推出所有 formal 题都要求写法律全称。具体是否为全局评分习惯，**需本地核验**。

建议写成分层规则：

1. 题面或细则给出具体法律名称时，优先写全称。
2. 裁判依据题、法条解释题、表格补全题优先写具体法律名称。
3. 选择题讲解不强制学生背法律全称。
4. 未见细则明确时，不把具体法律名称作为主干采分前提。

### 2.7 降级物权第四步默认归属加排除妨害

物权题的第四步不能只写归属和排除妨害。Batch02 题卡还涉及质权、抵押权、优先受偿、不得对抗善意第三人、土地经营权流转等。具体分布，**需本地核验**。

建议第四步写成：

> 物权题第四步按题型落到归属、对抗、优先受偿、共有处置、排除妨害或提供便利。

## 3. 哪些建议会增加学生负担、过度法考化、过度宏观法治化

### 3.1 会增加学生负担的建议

1. **五域改成合同、物权、家庭、劳动、第五域待定。**
   这会让学生以为后续还要新增人格侵权、知识产权、程序救济等域，最终变成六域、七域甚至八域。更稳做法是五域固定，二级卡分化。

2. **每个物权题都套设立、变动、对抗。**
   选择题可用，主观题不一定需要。若强制所有物权题都套，学生会被登记生效、登记对抗、交付、善意第三人等概念拖住。

3. **继承域只背协议大于遗嘱大于法定。**
   学生可能漏掉第一顺序、第二顺序、丧失继承权、扶养较多人适当分得遗产。

4. **把具体法律名称要求扩大到所有题。**
   这会把主干训练从规则适用变成法律名称背诵。具体名称应在裁判依据题、法条解释题、细则明确题中强化。

5. **每道意义题首句都套一核。**
   有用但容易模板化。学生可能写出形式正确但内容空的答案。

### 3.2 会过度法考化的建议

1. **用益物权、担保物权、所有权体系化讲解。**
   保留识别动作，拒绝体系化背诵。

2. **抵押、质押、留置完整分类。**
   高中题只需识别抵押与质押的基本差异和优先受偿，不需要完整担保物权法体系。

3. **善意取得、登记对抗、登记生效全套铺开。**
   题内出现再讲，不能作为前台主干。

4. **民法典条文编号背诵。**
   可以写民法典规定，通常不要求学生记具体条号，除非本地细则明确，**需本地核验**。

5. **继承编系统化。**
   学生版保留身份、协议、遗嘱、法定、扶养、丧失继承权即可。

### 3.3 会过度宏观法治化的建议

1. **把2026西城绿色原则题直接归入生态公益大口号。**
   该题更稳的入口是民法典物权编绿色原则、权利边界、法定义务、责任救济，最后上升到生态公共利益。是否跨到生态公益域，**需本地核验**。

2. **把题14的12348热线直接纳入法律生活框架。**
   若题肢主要考政府公共服务、法治社会建设、善政等表达，应先过入框闸门。是否入框，**需本地核验**。

3. **相邻关系题开头写和谐邻里、友善价值。**
   题20应先写事实、民法典、相邻关系、停止妨害，再写邻里和谐和核心价值观。价值不能替代法律判断。

4. **遗赠扶养协议题直接写尊老敬老。**
   应先写协议有效、扶养义务履行、财产归属，再写养老助老和社会价值。

## 4. 哪些建议必须等待 formal_or_scoring_source / official_answer_key 本地核验

   倾向综合民法判断题，逐项定关系；是否从物权域移出，**需本地核验**。

2. **题14的12348热线题是否进入《法律与生活》法律框架。**
   倾向先触发入框闸门；是否剔除或归政治模块，**需本地核验**。

3. **题9评分摘录是否真实污染。**
   题卡中出现外交内容，需回本地细则逐分值对齐，**需本地核验**。

   建议双标签处理：物权编绿色原则为规则入口，生态公共利益为价值出口；最终归位**需本地核验**。

5. **2026通州加装电梯题是否明确不写赔偿或补偿。**
   从题卡看应落停止妨害和相邻关系理由，但是否有补偿采分，**需本地核验**。

6. **物权三要件是否为本地通用框架。**
   设立、变动、对抗能否进学生小卡，**需本地核验**。

7. **用益与担保、使用价值与交换价值是否多题稳定采分。**
   倾向保留识别动作，是否进入学生版，**需本地核验**。

8. **协议大于遗嘱大于法定是否可作为继承硬口诀。**
   倾向作为小卡，不作唯一骨架；是否多题稳定采分，**需本地核验**。

9. **相邻关系四原则是否每题都必须写全。**
   题20可能明确要求，其他相邻题是否如此，**需本地核验**。

10. **一核表达是否多题作为开头采分点。**
    题20出现不能直接推广，**需本地核验**。

11. **法律具体名称要求是否为全局硬规则。**
    民法典、消费者权益保护法、反不正当竞争法等是否必须写全，**需本地核验**。

12. **五域拆并是否与本地教师讲法兼容。**
    尤其是物权与相邻、家庭与继承是否独立上墙，还是进入二级卡，**需本地核验**。
