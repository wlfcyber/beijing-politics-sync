## 2. 从学生可学性看，"主观题主干 / 选择题识别池"分层是否必要、是否表达清楚

### 2.1 必要性：**强必要**

5 批次累计审议中我和 GPT 都曾隐含触及这一问题但都没明确分层。本批增量第一次正面解决：

**5 批次中暴露的"识别池被误当主干"信号**：
- Batch01：上诉期 10 日 / 15 日陷阱（4 批次累计 6 题）— 学生不需要背天数，但识别动作必备
- Batch02：用益物权 vs 担保物权识别 — 系统背诵 reject，识别动作保留
- Batch03：归责原则三分（一般过错 / 过错推定 / 无过错）— 同处理方式
- Batch04：著作权具体权能（信息网络传播权 / 复制权 / 署名权）— 同处理方式
- Batch05：诉讼程序节点（一审 / 二审 / 终审 / 监督）— 同处理方式

**5 批次共有 5 处用"区分系统背诵 vs 识别动作"原则处理**，但都是分散应对。12:45 版把这 5 处统一升级为"主观题主干 vs 选择题识别池"双层结构，是结构性的关键收束。

如果不做这一分层，学生会面临两类失败：
- 主观题：把识别池细节（如发表权保护期、二审终审天数）写进答案，反而失去关系-规则-结论-意义骨架分
- 选择题：用主观题大模板（关系-规则-结论-意义）解选择题，找不到错项陷阱

### 2.2 表达清晰度：**部分清晰，2 处需修**

**清晰部分**：
- "主观题主干"判定标准（"是否在主观题中反复要求学生写出关系、规则、结论、意义"）— 标准明确可操作
- "选择题识别池"具体内容列举（5 大类）— 学生可见

**不够清晰的 2 处**：

**问题 1：分层不是绝对的，但表达过于绝对。**
12:45 版"选择题识别池只用于排错训练，不得反推为主观题大模板"是**过强的二元表达**。实际上：
- 大部分识别池内容确实只在选择题考（如二审终审天数、AI 作品独创性的具体讨论）
- 但少数识别池内容在主观题中**也作为踩点出现**，只是不作为主干展开

例如：
- "司法确认使调解协议具有强制执行效力"在 Batch05 题 5、22 主观题细则中作为 1 分踩点出现 — 这是识别池内容但主观题也踩点
- "举证责任倒置"在 Batch03 题肢和 Batch05 题 21 主观题细则中都出现 — 识别池但主观题在程序题中作为踩点

**修订建议**：分层增加第三状态——"**识别池常驻 + 主观题踩点**"（识别池内容如果在主观题中作为单一句采分点出现，可写但不作为段落展开）。

**问题 2："主观题主干"和"选择题识别池"两词是 Codex 内部语言，不是学生语言。**
学生看到"识别池"会困惑。课堂表达应转换为：
- 主观题主干 → "**写答案要展开的部分**"
- 选择题识别池 → "**只在选择题里识别错项的部分**"

### 3.2 需再细化的 1 类

**"侵权细节（运动伤害公平分担、新闻报道公共利益抗辩、个人信息单独同意、特殊侵权责任错项）"**

这一类需细化：
- **"运动伤害公平分担"**（Batch05 题 3、4）— 确实只在选择题考，恰当降级
- **"新闻报道公共利益抗辩"**（Batch05 题 9）— 确实只在选择题考，恰当降级
- **"特殊侵权责任错项"**（动物饲养 / 高空抛物 / 公共场所 / 产品责任）— **不应全部降级**

理由：Batch03 题 16（2026 丰台一模 20 公共场所安全保障义务）是**主观题且涉及特殊侵权责任**，细则明确给"无过错 / 合理限度 / 自我注意义务"采分。这类内容在主观题中作为侵权域子卡（"安全保障义务卡"）考过。

**修订建议**：把"特殊侵权责任错项"从识别池中**进一步拆分**：
- 动物饲养、高空抛物、产品责任 → 识别池（确实只在选择题考）
- 公共场所安全保障义务 → **主观题主干**（属于侵权域"安全保障义务卡"）
- "见义勇为公平责任"（Batch03 题 15）— Batch03 cross-critique 已提作为侵权域特殊规则卡，应在主观题主干层

`需 Codex 本地核验：上述细分是否与本地教师讲法兼容`

### 3.3 12:45 版未列入识别池但应考虑加入的 2 项

**A. "诉讼时效具体期间"** — Batch05 题 1（2024 东城一模 19）涉及诉讼时效原因题，但题面没要求学生背具体年限。具体期间应在识别池。

**B. "民事行为能力具体年龄分段"**（无民事行为能力 / 限制 / 完全的具体岁数）— Batch05 题 16（西城二模 18 未成年打赏）、Batch01 题 25（西城期末 12 先用后付）涉及限制民事行为能力，但识别"是否限制"是主观题主干，识别"几岁分段"是选择题识别池。

**修订建议**：识别池补入这 2 类，但表述限定在"具体年限 / 具体岁数"层面，不影响"诉讼时效原则"和"民事行为能力分类原则"作为主观题主干。

## 4. 哪些内容虽然选择题常见，但主观题也可能考，不能降级过头

12:45 版有 **3 处过度降级风险**，必须保留为主观题主干：

### 4.1 程序细节中的"调解 / 仲裁 / 诉讼三选一"——**主观题核心，不降级**

12:45 版把"司法确认"放进识别池。司法确认本身（即"经人民法院依法确认有效的调解协议具有强制执行效力"）确实只在选择题考具体程序，但**多元纠纷解决机制本身（调解 / 仲裁 / 诉讼如何区分、和解为何不同）是主观题主干**：
- Batch05 题 5（石景山 18 法官指导）— 主观题考多元纠纷解决
- Batch05 题 17（顺义一模 19 如何解决纠纷）— 主观题考多元纠纷解决
- Batch05 题 22（门头沟 18 公益诉讼）— 主观题踩点"和解、调解、诉讼"功能各有侧重 + 教材原文采分

**修订建议**：明确区分——
- "多元纠纷解决机制（和解 / 调解 / 仲裁 / 诉讼及其法律效果）"→ 主观题主干（救济与公共利益域子卡）
- "司法确认的具体程序节点"→ 识别池
- "辩护人 vs 诉讼代理人具体身份区分"→ 识别池

### 4.2 知识产权细节中的"AI 作品独创性"——**主观题主干，不降级**

12:45 版把"AI 作品独创性"放进识别池。这是**重大降级失误**。
- Batch04 至少 4 道主观题（题 2、7、8）核心采分点是"独创性 + 智力投入"作为作品认定的主观题骨架
- Batch04 题 7（昌平二模 20）细则明确"具备智力成果和独创性要件，受到著作权法保护"是主观题踩点

**修订建议**：把"AI 作品独创性"**从识别池移到主观题主干**（创新与竞争域著作权子卡核心采分动作）。识别池只保留"AI 作品著作权登记的具体程序"等真正只在选择题考的点。

### 4.3 侵权细节中的"个人信息单独同意"——**双层处理**

12:45 版把"个人信息单独同意"放进识别池。Batch03 题 14（门头沟 3）确实是选择题。但 Batch04 题 11（东城一模 11 AI 直播未标注）等 AI 题中"消费者知情权 + 个人信息保护"作为侵权 + 消费双标签，主观题也有可能考。

**修订建议**：双层处理——
- "个人信息单独同意的具体规则"→ 识别池
- "个人信息作为人格权益类型 + 处理者义务"→ 主观题主干（人格权侵权域内 8 类具体人格权之一）

### 4.4 总体原则补强

5 批次审议累计共识：**"识别动作 vs 系统背诵"分层**。12:45 版的"主观题主干 vs 识别池"是这一原则的最终表达，但需要明确**部分内容跨越两层**：

| 类型 | 主观题主干层 | 识别池层 |
|---|---|---|
| 民事行为能力 | 分类原则 + 限制行为能力效力规则 | 具体岁数分段 |
| 诉讼时效 | 时效设立目的 + 不适用情形原则 | 具体年限 |
| 物权变动 | 登记 / 交付的法律效果 | 具体登记机关、登记程序 |
| 著作权 | 独创性 + 智力投入 + 主体识别 | 登记误区、保护期具体年限、法定许可情形 |
| 多元纠纷 | 调解 / 仲裁 / 诉讼区分 + 法律效果 | 司法确认具体程序、辩护人 vs 诉讼代理人 |
| 侵权 | 三要件 + 归责原则 + 具体人格权类型 | 特殊侵权类型具体细则、举证倒置具体情形 |

## 5. 五域命名与主观/选择计数呈现是否会误导学生

### 5.1 五域命名问题：**5 域中 2 域命名仍有问题，必须修**

12:45 版仍使用 5 批次审议过程中已被论证不合适的命名：
- "**合同消费者劳动**" — 这是 5 批次审议第一批次就被 GPT 论证应拆为"合同与消费 + 劳动与职业"两域的命名，**12:45 版退回原始命名**
- "**物权相邻继承家庭**" — 这是 Batch02 cross-critique 论证应拆为"物权与相邻 + 家庭与继承"两域的命名

5 批次累计共识的五域学生版命名应为：
1. 合同与消费
2. 物权与相邻
3. 家庭与继承
4. 人身人格与侵权（采纳 Batch03 GPT 命名）
5. 救济与公共利益（采纳 Batch05 GPT 命名）

**但这是 5 域结构，与原始 5 域结构（"合同消费者劳动 / 物权相邻继承家庭 / 人格权侵权 / 知识产权不正当竞争 / 纠纷解决生态公益与新业态"）不兼容**——12:45 版保留原始 5 域命名，把"劳动"放在第一域内、"创新与竞争"作第四域，这是 **5 批次审议未达成的妥协**。

**回到关键问题**：原始五域是用户硬约束。是否应改名？

**我的建议**：
- 维持原始 5 域结构（兼容用户硬约束）
- 但学生版课堂表达**简化为短名**，避免"合同消费者劳动"这种 6 字串：
  - "合同消费者劳动" → 学生版课堂称"**合同与劳动域**"或"**交易与就业域**"（具体待 Codex 本地核验本地教师讲法）
  - "物权相邻继承家庭" → 学生版课堂称"**财产与家庭域**"
  - "人格权侵权" → 学生版课堂称"**人身侵权域**"
  - "知识产权不正当竞争" → 学生版课堂称"**创新与竞争域**"（采纳 Batch04 GPT 命名）
  - "纠纷解决生态公益与新业态" → 学生版课堂称"**救济与公共利益域**"（采纳 Batch05 GPT 命名）

`需 Codex 本地核验：本地教师讲法对域名的接受度`

### 5.2 主观/选择计数呈现：**有 1 处误导风险**

12:45 版呈现："1. 合同消费者劳动：38 道，主观 16，选择 22。"

**误导风险**：学生看到"38 道"会认为这是"题量第一大域必须重点准备"。但**主观题占比才是主观题答题准备的核心信号**：
- 合同消费者劳动：主观 16 / 38 = 42%
- 物权相邻继承家庭：主观 5 / 22 = 23%
- 人格权侵权：主观 6 / 16 = 38%
- 知识产权不正当竞争：主观 8 / 19 = 42%
- 救济与公共利益：主观 8 / 15 = **53%**

按主观题占比：救济与公共利益 > 合同消费者劳动 ≈ 知识产权不正当竞争 > 人格权侵权 > 物权相邻继承家庭。

**修订建议**：在分布表后加一句**主观题占比**导向语：
> 主观题答题准备应优先按主观题占比排序：救济与公共利益（53%）> 合同消费者劳动 / 创新与竞争（42%）> 人格权侵权（38%）> 物权相邻继承家庭（23%）。

或者直接把主观题占比加入分布表，让学生看清"哪个域更可能在主观题里出现"。

### 5.3 五域内是否需要呈现"主观题主干 / 选择题识别池"双轨

12:45 版在框架层声明分层规则，但**五域内具体内容**没有明确双轨呈现。建议每域用以下结构：

```
合同与消费域（合同消费者劳动域）
├── 主观题主干：合同效力 / 违约责任 / 消费者权益
└── 选择题识别池：生鲜灯 / 先用后付 / 价格规则 / 直播带货
```

这样学生一眼能看出哪些是写答案用、哪些是排错用。
