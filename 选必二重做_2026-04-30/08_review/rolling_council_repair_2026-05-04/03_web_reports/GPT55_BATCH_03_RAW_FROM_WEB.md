# Batch03 人格权侵权框架委员会意见

## 1. 总判定

**verdict：PASSAFTERFIX。**

Batch03 可以支持**人格权侵权作为独立前台域候选**，但不能按当前题卡原样 PASS。原因是：本批确实出现肖像、隐私、个人信息、身体权、健康权、安全保障义务、侵权责任、受益人适当补偿等稳定材料；同时，当前题卡存在明显的**关键词归域污染**，把若干合同消费者、物权相邻、知识产权竞争、劳动职业、综合民法题误塞进人格权侵权域。若不修正，学生会把“出现赔偿、侵权、法院”误学成“人格权侵权”，这会直接破坏一核中的“具体法律关系”判断。Batch03 题包显示本批为 17 题，其中主观题 7、选择题 10，证据结构为 official_answer_key=9、formal_or_scoring_source=8；这些只能支持本批候选判断，所有具体题域、答案、采分点仍**需本地核验**。

**不能直接 PASS 的主要风险：**

1. **错归域风险高。**
   共享单车禁停区更像共有部分管理、排除妨碍、相邻或物权秩序；牙膏虚假宣传与三倍赔偿更像合同消费者；真创新真保护更像知识产权和不正当竞争；竞业限制更像劳动与职业；摄影作品著作权更像创新与竞争。这些都不能只因出现“侵权、赔偿、法院”就归入人格权侵权，具体归属**需本地核验**。

2. **综合题拆分不足。**
   2025 东城二模第19题表格含合同效力、身体/健康权、格式条款；2025 朝阳期末第20题同时有数据不正当竞争和摇摆桥人身损害；这类题应按小问、案例、栏目拆分，不能整题粗归人格权侵权，具体拆分**需本地核验**。

3. **评分摘录污染严重。**
   2025 海淀期末第20题、2025 海淀期末第21题的评分摘录中出现与设问明显不匹配的内容，如思维方法、土地权利、乡村振兴、无过错责任立法精神等。此类 formal_or_scoring_source 摘录必须先做题号、设问、分值、材料主题对齐，未核验前不能推动主干。

4. **题型标签误贴。**
   多个主观题被贴为表格补全题或程序救济题，但设问实际是案例分析、裁判依据、意义认识或表格双栏。题型误贴会导致学生答题动作错位，具体题型**需本地核验**。

**Codex 三个立即动作：**

1. 对 Batch03 全部 17 题执行**法律小问入框闸门 + 题域重判**。
2. 对 8 个 formal_or_scoring_source 执行**摘录对齐检测**，污染样本先隔离。
3. 将人格权侵权域主干压缩为**人格权识别、侵权轻量链、安全保障义务边界、合同违约与侵权竞合、价值后置**五张卡。

## 2. 本批次支持的高频主干

### 2.1 具体人格权识别卡

**候选主干：肖像权、隐私权、个人信息权益、名誉权、身体权、健康权等具体人格利益识别。需本地核验。**

本批可见样本包括：

1. 未成年人照片发布，涉及肖像、网络发布、未成年人保护、侵权责任。
2. 摄像头拍摄日常出行动态，涉及隐私权、共有部分、相邻关系共同出现。
3. 敏感个人信息、未成年人个人信息处理，涉及个人信息保护。
4. 文身、身体健康损害等场景，涉及身体权、健康权。

学生版应避免泛写“侵犯人格权”。更稳写法是：

> 先判断是哪一种人格利益，再说明行为如何侵害该利益，最后落到停止侵害、赔礼道歉、赔偿损失、删除信息等责任方式。

### 2.2 侵权责任轻量判断链

**候选主干：行为、损害、因果、过错或法定特别责任、责任方式。需本地核验。**

本批选择题与主观题多次出现过错、损害、因果、举证责任、赔偿责任。比如出租车事故、公共场所摔倒、动物园管理责任、高空抛物、污染等题干或题肢，都在测试“谁承担侵权责任、依据是什么、是否有免责或边界”。这些足以支持轻量侵权链，但不能扩张成法考式归责体系。

学生版建议：

> 侵权题只问五件事：谁的行为，伤了谁的什么权利，有无损害，行为和损害有没有因果，有无过错或法律特别规定，最后承担什么责任。

### 2.3 经营者、公共场所安全保障义务及边界

**候选主干：安全保障义务不是无限责任，需结合合理限度、证据、受害人注意义务判断。需本地核验。**

2025 丰台期末第12题涉及景区观景台湿滑、旅游合同与安全保障义务；2026 丰台一模第20题涉及公共场所经营者、管理者是否尽到合理限度内的安全保障义务，细则摘录强调“合理限度”“无过错”“完全民事行为能力人自我注意义务”等。

学生版应固定边界表达：

> 公共场所经营者、管理者负有安全保障义务，但该义务有合理限度；若证据显示其无过错，且损害主要由受害人自身未尽注意义务造成，就不当然承担赔偿责任。需本地核验。

### 2.4 合同违约与侵权竞合识别

**候选主干：同一事实可能同时触发合同责任和侵权责任，但要分别说明关系和结果。需本地核验。**

Batch03 中出租车运输、小王乘车受伤、景区游玩摔伤、牙膏虚假宣传、摇摆桥项目受伤等题，都可能同时出现合同关系和侵权责任。学生容易把“有合同”误判为只能违约，或把“有人身损害”误判为只能侵权。

学生版可设一张竞合卡：

> 先看是否存在合同关系，再看是否侵害绝对权或人身权；合同关系不排除侵权责任，人身损害也可能同时有合同履行义务问题。具体选择哪条责任线，按设问和细则处理，需本地核验。

### 2.5 见义勇为、受益人适当补偿

**候选主干：保护他人民事权益导致自己受损时，可由受益人给予适当补偿；价值必须后置。需本地核验。**

2025 顺义一模第11题显示，卫先生保护古阿姨权益而受伤，法院判令受益人适当补偿。该题的价值点是用法治力量保护见义勇为和知恩图报，但不能把古阿姨直接说成侵权人，也不能把补偿误写成赔偿责任。

学生版应写：

> 不是所有给钱都叫赔偿。侵权人承担赔偿；受益人可给予适当补偿。该处理有利于鼓励见义勇为、弘扬传统美德。需本地核验。

### 2.6 价值后置

**候选主干：意义题先回本案，再写权利保护、秩序维护、价值导向。需本地核验。**

本批价值题常出现网络空间治理、媒体公信力、公平竞争、创新保护、公共场所安全责任边界、社会主义核心价值观等表达。价值不能开头悬空写。学生版应固定顺序：

> 本案处理结果，保护了谁的什么权利，规范了谁的什么行为，维护了什么秩序，最后才写价值。

## 4. 五域归位是否合理，是否要拆/并/改名

### 4.1 结论

Batch03 支持“人格权侵权”独立成域，但域名建议改为：

**人身人格与侵权责任。**

理由是：Batch03 的真主干不只人格权，还包括身体健康损害、公共场所安全保障义务、见义勇为受益人补偿、一般侵权责任边界。若只叫“人格权侵权”，学生会误以为所有侵权都指肖像、名誉、隐私等人格权。若叫“人身人格与侵权责任”，更能覆盖本批真实场景。该域名是否与本地教师讲法兼容，**需本地核验**。

### 4.2 建议前台五域继续固定为五个

不建议因 Batch03 把五域拆成更多域。建议前台五域为：

1. **交易与财产**
   合同、消费、物权、担保、相邻、债务财产。

2. **人身人格与家庭**或拆为**人身人格与侵权责任**、**家庭继承**？
   这里需要谨慎。Batch03 支持人格侵权独立动作，Batch02 支持家庭继承独立动作。若严格五域，推荐采用：
   **人身人格与家庭**作为大域，下设人格侵权卡、家庭继承卡。
   若本地学生负担允许，也可把人格侵权和家庭继承拆成两个前台域，但这样会挤压程序或创新域，需本地核验。

3. **劳动与职业**
   劳动关系、解除、补偿、竞业限制、新就业形态。

4. **创新与竞争**
   著作权、技术秘密、数据权益、不正当竞争、虚拟数字人、软件标识。

5. **救济与公共利益**
   诉讼、仲裁、调解、执行、公益、公共法律服务、生态公共利益。

但从 Batch01、Batch02、Batch03 连续表现看，更稳的学生版是：

| 前台五域    | 二级卡                 |
| ------- | ------------------- |
| 交易与财产   | 合同消费卡、物权相邻卡         |
| 人身与家庭   | 人格侵权卡、家庭继承卡         |
| 劳动与职业   | 劳动关系卡、解除补偿卡、竞业限制卡   |
| 创新与竞争   | 著作权卡、不正当竞争卡、技术秘密数据卡 |
| 救济与公共利益 | 程序救济卡、执行救济卡、生态公益卡   |

这样能保留五域负担上限，同时避免把人格侵权、家庭继承、物权相邻、IP竞争混成一锅。

### 4.3 八域后台继续保留

Batch03 再次证明后台八域有必要，因为题目跨域、混合、并列案例较多。后台可以保留：

1. 合同交易与消费者保护
2. 财产权、物权与相邻关系
3. 人格权与侵权责任
4. 婚姻家庭与继承扶养
5. 劳动就业与职业边界
6. 知识产权与竞争秩序
7. 纠纷解决与程序救济
8. 生态公益、新业态与法治价值

学生不背八域，Codex 用八域做索引和核验。

## 5. 一核二线三问四步如何修订

### 5.1 一核

保留原一核：

> 以事实为根据、以法律为准绳，通过具体法律关系中的规则适用实现定分止争。

Batch03 要强化“一核前第0步”：

> 先判是否为法律小问，再判具体法律关系。

因为本批大量错归来自关键词触发。看到侵权、赔偿、法院，只能提示可能是法律题，不能直接决定题域。

### 5.2 二线

建议继续改写为：

1. **权利保护与权利边界。**
   Batch03 强支持。人格权保护、经营者安全保障义务边界、受害人自我注意义务、网络表达权边界，都属于这一线，需本地核验。

2. **规则适用与价值校准。**
   Batch03 也支持。见义勇为、网络空间治理、创新保护、公共场所责任边界，都需要价值后置。但不能用“公正司法”“核心价值观”替代法律判断。

不建议删除价值线。价值线应作为后置校准，不作答案开头。

### 5.3 三问

三问保留，但按题型触发：

1. **选择题：** 判哪项，凭什么排错，通常不写意义。
2. **人格权识别题：** 侵害什么权利，凭什么，承担什么责任。
3. **安全保障义务题：** 是否尽到合理限度，凭什么，是否承担责任。
4. **竞合题：** 合同关系和侵权关系分别是什么，最终按设问落责任。
5. **意义题：** 本案怎么处理，依据什么，意义是什么。

### 5.4 四步

四步继续保留，但第四步按域和题型分流：

1. 定主体关系。
   例：乘客与出租车司机、游客与景区、网络发布者与被拍摄人、经营者与消费者、平台与用户。

2. 找争点事实。
   例：未经同意发布照片、摄像头拍摄出行规律、经营场所是否有湿滑风险、工作人员是否过度摇晃、是否虚假宣传。

3. 对规则条件。
   例：人格权保护、隐私信息、个人信息单独同意、安全保障义务、过错与因果、合同欺诈、不正当竞争。

4. 落法律结果。
   Batch03 应分为：

   * 人格权题：停止侵害、删除、赔礼道歉、赔偿。
   * 身体健康损害题：赔偿医疗费等损失。
   * 安全保障义务题：承担或不承担侵权责任。
   * 受益人补偿题：适当补偿，不能写成侵权赔偿。
   * IP或竞争题：停止侵权、消除影响、赔偿、规范竞争秩序。
   * 价值题：本案处理、权利保护、秩序维护、价值导向。
