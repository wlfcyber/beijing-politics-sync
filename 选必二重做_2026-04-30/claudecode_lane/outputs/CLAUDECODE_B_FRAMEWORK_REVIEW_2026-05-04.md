## 四、命题路径反推（按域抽样）

- 合同交易与消费者保护（25 题）：高频路径 = 案例事实 → 民法/消费者权益保护法/食品安全法规则 → 效力/赔偿/惩罚性赔偿落点 → 价值后置（公平、诚信、消费环境）。命题人偏好"超出生活消费范围"等限缩条件作为评分门槛。
- 纠纷解决与程序救济（20 题）：高频路径 = 案件 → 多元解纷渠道（调解/仲裁/诉讼）→ 司法确认/强制执行 → 价值后置（多元化、效率与公平）。
- 人格权与侵权责任（17 题）：路径 = 名誉/肖像/隐私事实 → 侵权要件 → 停止侵害/赔礼道歉/赔偿 → 个人尊严与公共秩序。
- 知识产权与竞争秩序（15 题）：路径 = 创新成果/标识 → 著作权/商标/反不正当竞争规则 → 停止侵权/赔偿/责令改正 → 创新驱动与市场秩序。
- 劳动就业与职业边界（13 题）：路径 = 劳动关系事实 → 劳动法/劳动合同法/工伤保险条例 → 经济补偿/工伤待遇/劳动仲裁 → 劳动者保护与企业用工规范。
- 婚姻家庭与继承扶养（11 题）：路径 = 家庭事实 → 民法典婚姻家庭/继承编 → 抚养、扶养、遗嘱效力、遗产归属 → 家庭伦理与代际责任。
- 财产权、物权与相邻关系（10 题）：路径 = 物权事实/登记 → 物权法定、所有权/用益/担保 → 归属确认/登记/相邻容忍 → 财产秩序。
