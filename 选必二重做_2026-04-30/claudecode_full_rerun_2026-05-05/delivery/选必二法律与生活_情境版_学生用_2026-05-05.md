# 选必二《法律与生活》情境穷尽手册（学生用）

## 写在前面

法律题不考"你记住了多少法律术语"，而考"你能识别出多少种生活情境，并用对应的法律规则来处理"。本手册按照**主观题常见的情境** + **选择题常见的细节**分两章，帮你穷尽这个模块会考的所有"故事原型"。读完一章，你就能看到新题时立刻说出"这是哪一类情境"，然后调出对应的答题模板。换句话说，本手册是把所有可能考到的题目"故事化"，让你学会"识别情境"，而不是"死背知识点"。

# 第一章：主观题情境（每个情境是一个完整故事）

按五个域分节，每个节下列举代表性主观题。每题都包括：故事（起因→经过→结果）、争点（真正在争什么）、规则（对应的法律要件）、触发词（看到这个词想到那条规则）、踩分关键词、得分句、作答逻辑。

## 第一域：交易交换情境

### 情境1.1：机票捆绑搭售案（2025朝阳一模 Q19，8分）

**起因→经过→结果**

王某想买机票。他通过A机票代销平台的页面下单，页面显示总价350元。他看到有个40元的优惠，所以支付了310元。但事后他在航空公司官网查证发现，机票实际总价只有300元，不是350元。经查，原来A公司在销售过程中通过技术手段擅自搭售了10元的外卖红包，但购买界面上没有明确显示这个红包，王某也无法拒绝支付这10元。王某很生气，向互联网法院提起诉讼，要求A公司退还票款310元，同时赔偿310×3=930元。法院支持了他的诉讼请求。

**争点**

这道题的争点是：**A公司的搭售行为是否构成欺诈？消保法怎么赔？** 不是问"机票卖多了钱"，而是问"A公司用了什么手段骗人、骗得有多严重、法律怎么惩罚"。

**对应规则与给分依据**

① **合同成立**：王某下单（要约）→A公司出票（承诺）→合同成立。这一步法律层面没有问题，但事实层面需要注意：出票时带了"隐形商品"（红包）。

② **合同欺诈的三要素**（这是关键）：
   - 主观欺诈故意：A公司为了增加收入而搭售红包，有意欺骗王某。
   - 客观欺诈行为：A公司"没有设置醒目提示""没有提供勾选选项"，故意隐瞒了费用。
   - 违背真实意思：王某"不能清楚知悉费用支出细节""也无法拒绝"。

③ **合同效力**：因欺诈，合同可撤销。

④ **消保法三倍赔偿**：根据《消费者权益保护法》第55条，因商家欺诈，应赔偿消费者购买商品价款的三倍。这里"商品价款"是310元，所以赔偿310×3=930元，同时退还310元。

⑤ **意义**：维护消费者权益、规范电商经营、优化网络消费环境、促进数字经济发展。

**触发词 → 想到什么规则**

- 看到"购买""下单""出票" → 合同成立（要约+承诺）
- 看到"隐瞒""不提示""无法拒绝" → 欺诈构成（三要素）
- 看到"电商""平台""网络消费"+"欺诈" → 《消费者权益保护法》第55条
- 看到"三倍赔偿" → 消保法 + 惩罚性赔偿

**踩分关键词（必须在答卷上出现）**

1. 合同成立/要约/承诺
2. 欺诈/主观欺诈故意/客观欺诈行为
3. 隐瞒真实情况/不提供选择
4. 违背真实意思表示
5. 合同可撤销
6. 消费者权益保护法（注意名字，不是"权益法"也不是"保护法"）
7. 知情权/自主选择权
8. 三倍赔偿/增加赔偿（10分必须说这个词，不能说"额外赔偿"）
9. 退还/返还
10. 维护/规范/优化/促进

**直接上卷面的得分句**

1. "原告与A公司有要约与承诺，合同成立。"（1分）

2. "A公司没有在购买界面设置醒目、清楚的提示语介绍捆绑销售的外卖红包，也未向消费者提供勾选该红包的选项，导致王某不能清楚知悉费用构成情况及金额用途，属于故意隐瞒真实情况。"（1分）

3. "A公司具有欺诈的主观故意和客观欺诈行为，导致王某在违背真实意思表示的情况下订立合同，因此，根据民法典，该合同可以撤销。"（1分）

4. "根据《消费者权益保护法》第55条规定，经营者提供商品或者服务有欺诈行为的，应当按照消费者的要求增加赔偿其受到的损失，增加赔偿的金额为消费者购买商品的价款或接受服务的费用的三倍。"（1分）

5. "A公司应当退还王某支付的票款310元，并按照310元的三倍（即930元）向王某支付赔偿金。"（1分）

6. "支持王某的诉讼请求，有利于维护消费者的合法权益，规范在线电商企业的行为，优化网络消费环境，促进数字经济持续健康发展。"（2分）

**作答逻辑**

你看到这道题，首先要识别"这是消费欺诈纠纷"（交易交换域）。然后问自己四个问题：（1）王某和A公司是什么关系？（消费者和商家）（2）争点是什么？（搭售红包是否欺诈）（3）法律怎么处理？（欺诈成立 → 合同可撤销 + 消保法三倍赔偿）（4）意义是什么？（维护权益、规范行为）。

关键是不能漏掉"欺诈的三要素"——材料里给了具体的"主观意图（为了搭售红包）""欺诈手段（没提示、无选择）"和"受害结果（不知道、无法拒绝）"，你要把这三个要素全部抄出来套到欺诈定义里。最后要强调"是消费者权益保护法，不是其他法"。

### 情境1.2：平台经济劳动关系认定（2024朝阳二模 Q17，9分）

**起因→经过→结果**

王某于2022年6月与甲公司签订《车辆管理协议》，约定他自备面包车，通过甲公司的平台接收派单进行货物运输。甲公司每月支付他6000元服务费和奖励金。王某每天都在平台上签到、接单，跑单时间都在8小时以上。甲公司通过平台全程跟踪他的订单完成情况。如果王某每日接单量超过4单，就有加单奖励；如果出现接单量不足、无故拒单或货物损毁，甲公司就按制度扣减他的部分服务费。2023年3月，甲公司因调整运营计划，单方面提出终止合作关系。王某认为自己其实是甲公司的员工，应该按劳动者对待，要求甲公司支付解除劳动合同经济补偿。甲公司拒绝了，说他们只是合作关系，不是劳动关系。王某向仲裁委申请仲裁。仲裁委裁决：甲公司应向王某支付解除劳动合同经济补偿。

**争点**

这道题的争点是：**王某和甲公司之间到底是"合作关系"还是"劳动关系"？** 表面上他们签的是合作协议、王某自备车，但实际情况怎样？法律怎么判？

**对应规则与给分依据**

① **劳动关系的核心特征**：劳动者与用人单位之间具有"人格从属性""经济从属性""组织从属性"。只要这三个维度都满足，就构成劳动关系，即使签的合同叫"合作协议"。

② **人格从属性**：平台对王某的约束有多强？
   - 甲公司制定了平台劳动纪律、奖惩制度（超4单有奖励、接单不足扣费）。
   - 王某必须按照甲公司的指令完成工作（每日签到、接派单、全程跟踪）。
   - 这说明甲公司对王某有很强的人格约束。

③ **经济从属性**：王某对自己的收入有多少掌控权？
   - 甲公司掌握了平台派单数据，决定了王某能接多少单（核心生产资料）。
   - 王某不能自主定价，服务费由甲公司定（6000元）。
   - 王某虽然自备车，但主要的收入来源（派单）被平台垄断。
   - 这说明王某在经济上对甲公司有很强的依赖性。

④ **组织从属性**：王某是否被纳入甲公司的组织体系？
   - 王某通过甲公司的平台接单、以甲公司名义对外运输。
   - 王某实际上是甲公司组织的一部分。
   - 这说明王某在组织上属于甲公司。

⑤ **判决**：因为三个维度都满足，甲公司与王某构成劳动关系（不是合作关系）。甲公司单方面终止，要支付王某经济补偿。

**触发词 → 想到什么规则**

- 看到"平台""自备工具""灵活""零工" → 警惕："这可能是劳动关系，不是合作"
- 看到"纪律""奖惩""制度" → 人格从属性
- 看到"平台派单""不能定价""掌握数据" → 经济从属性
- 看到"以公司名义""纳入体系" → 组织从属性
- 看到"经济补偿""解除合同" → 劳动法适用

**踩分关键词**

1. 劳动关系/合作关系/区分
2. 人格从属性/劳动纪律/奖惩制度/平台指令
3. 经济从属性/掌握生产资料/定价权/数据
4. 组织从属性/组织体系/平台名义
5. 三个维度/同时满足
6. 解除劳动关系/经济补偿
7. 劳动法适用

**直接上卷面的得分句**

1. "甲公司通过平台制定了劳动纪律和奖惩制度（超4单奖励、接单不足扣费），对王某的行为进行约束；王某每日必须在平台签到、接派单、接受跟踪。这体现了明显的人格从属性。"（1.5分）

2. "甲公司掌握了平台派单数据（核心生产资料），决定了王某能获得的工作机会和收入；王某不能自主定价，服务费由甲公司单方确定。这体现了明显的经济从属性。"（1.5分）

3. "王某通过甲公司平台接单、以甲公司名义对外提供运输服务，实际上是甲公司组织体系的一部分。这体现了明显的组织从属性。"（1.5分）

4. "根据劳动关系认定的三维度标准，王某与甲公司构成劳动关系（而非单纯的合作关系）。甲公司单方面终止合作，应向王某支付解除劳动合同的经济补偿。"（2分）

5. "这一判决规范了平台经济中的劳动关系认定，保护了新就业形态劳动者的权益，促进了平台经济的规范发展。"（1分）

**作答逻辑**

看到这道题，你要识别"这不是简单的合同纠纷，而是劳动关系认定"。关键是要用"三维度框架"（人格、经济、组织）来分析材料。材料给了很多细节，你的任务是：把"奖惩制度""全程跟踪"等细节 → 对应到"人格从属性"；把"掌握派单数据""不能定价"等细节 → 对应到"经济从属性"；把"平台名义"等细节 → 对应到"组织从属性"。三个都成立，就是劳动关系。

注意不要遗漏或混淆三维度。很多学生只说了"平台掌握派单"就停了，但这样只涵盖了"经济从属性"，不够。一定要把三个维度都写出来。

### 情境1.3：格式条款与消费者保护（常见情境，具体案例待补）

**起因→经过→结果（基本模式）**

某租房平台在租赁合同的格式条款中写着："房东有权随时进入房间进行检查和维修，租户无权拒绝。如因此导致的任何损失，房东不承担责任。"租户王某认为这个条款不公平，想起诉。

**争点**

这个格式条款有效吗？是否违反了消费者保护原则？

**对应规则与给分依据**

① **格式条款的效力**：格式条款是商家单方面拟定的、用来规范很多消费者的条款。法律对格式条款有特殊保护规则：
   - 商家**免除自己的责任** → 无效。（例如"我不对任何损失负责"）
   - 商家**加重消费者的负担** → 无效。（例如"租户要赔偿房东一切损失"）
   - 商家**排除消费者的法定权利** → 无效。（例如"租户放弃隐私权""租户不能投诉"）

② **本题的格式条款**：
   - "房东随时可以进入房间"加重了租户的负担（租户无法保护隐私）。
   - "房东不承担责任"是房东免除自己责任。
   - 所以这个条款无效。

③ **租户的权利**：根据民法典，租户有隐私权、人身权、财产权。房东不能随意进入。

**触发词 → 想到什么规则**

- 看到"格式条款""免责""霸王条款" → 格式条款效力审查
- 看到"房东随时可以进入" → 加重消费者负担 → 无效
- 看到"房东不承担责任" → 商家免责 → 无效
- 看到"排除权利" → 无效

**踩分关键词**

1. 格式条款
2. 免除商家责任/无效
3. 加重消费者负担/无效
4. 排除消费者权利/无效
5. 隐私权/人身权
6. 无效

**直接上卷面的得分句**

"这个格式条款有两个问题：第一，'房东随时进入'加重了租户的负担，限制了租户的隐私权；第二，'房东不承担责任'是房东免除自己的法定责任。根据民法典，这样的格式条款无效。房东不能随时进入，如果进入导致了租户的损失，房东要承担责任。"

## 第二域：身份财产情境

### 情境2.1：婚前财产与离婚分割（常见情境，具体案例待补）

**起因→经过→结果（基本模式）**

张某和李某2020年结婚。结婚前，张某用自己的钱在北京买了一套房，办理了产权证。结婚后，张某和李某的收入都在这套房里（房租抵了贷款）。现在他们离婚了，李某要求分割这套房子。张某说这是他的婚前财产，李某不能分。

**争点**

张某的房子到底算共同财产还是个人财产？

**对应规则与给分依据**

① **法定共同财产**：夫妻在婚后取得的财产原则上是共同财产。

② **个人财产的情形**：
   - 一方的婚前财产 ← 这里适用
   - 一方的继承所得
   - 一方的赠与所得（赠与协议明确只给一个人）

③ **本题判断**：
   - 房子是张某"婚前购买"→ 属于张某的婚前财产。
   - 即使房子在婚后增值、获得了租金收入，这套房本身仍然是婚前财产。
   - 李某没有分割权。

④ **容易混淆的情况**：
   - "婚前买、婚后付款"：如果婚前交了首付、婚后还按揭，房子仍是婚前财产吗？根据新规则，分两部分：婚前交的首付 + 婚前产权 = 婚前财产；婚后偿还的贷款部分 = 可能涉及共同财产（如果用的是夫妻共同收入）。
   - "婚前买、婚后办证"：办证时间不重要，关键是"什么时候购买"。婚前购买就是婚前财产。

**触发词 → 想到什么规则**

- 看到"婚前买""购买时间在结婚前" → 婚前财产
- 看到"婚后取得"、"结婚后才有这笔钱" → 共同财产
- 看到"继承""赠与" → 个人财产
- 看到"离婚分割" → 先区分共同和个人

**踩分关键词**

1. 婚前财产/个人财产
2. 婚前购买/购买时间
3. 不参与分割
4. 共同财产只分婚后取得部分

**直接上卷面的得分句**

"根据民法典，一方的婚前财产属于个人财产。该房产系张某婚前购买并办理产权证，属于张某的个人财产。即使房产在婚后增值或产生租金收入，房产本身仍然是张某的个人财产，离婚时不参与分割。李某不享有该房产的分割权。"

### 情境2.2：法定继承与遗嘱继承（常见情境，具体案例待补）

**起因→经过→结果（基本模式）**

王某在2025年去世，留下一笔100万元的遗产。王某有妻子、两个成年子女和一个成年兄弟。王某去世时，有一份手写遗嘱，说"这100万全部留给小儿子"。其他人不同意。现在要分配遗产，谁说了算？

**争点**

是按照遗嘱分（全部给小儿子），还是按照法定继承分（妻子、两个子女、兄弟一起分）？

**对应规则与给分依据**

① **遗嘱继承优先于法定继承**：如果有有效的遗嘱，就按遗嘱分，不用法定继承。

② **遗嘱有效的条件**：
   - 形式符合法律（自书遗嘱要有签名、日期；打印遗嘱要有见证人）。
   - 内容合法（不能违反法律、伦理）。
   - 遗嘱人有民事行为能力。
   - 表示意思真实（不是被胁迫、欺骗做的）。

③ **本题判断**：
   - 假设这份手写遗嘱形式符合、内容合法、王某有行为能力、意思真实。
   - 那么按遗嘱分：100万全部给小儿子。
   - 其他继承人没有分割权。

④ **没有遗嘱的情况**：按法定继承顺序分。
   - 第一顺序：配偶、子女、父母。
   - 第二顺序：兄弟姐妹、祖父母、外祖父母。
   - 如果有第一顺序继承人，就不轮到第二顺序。

**触发词 → 想到什么规则**

- 看到"遗嘱" → 遗嘱优先
- 看到"手写""自书" → 检查形式是否符合
- 看到"打印遗嘱"或"录音遗嘱" → 检查是否有见证人
- 看到"没有遗嘱" → 法定继承顺序
- 看到"配偶、子女、父母" → 第一顺序
- 看到"兄弟姐妹" → 第二顺序

**踩分关键词**

1. 遗嘱继承
2. 遗嘱有效/符合法律形式
3. 遗嘱优先于法定继承
4. 法定继承/继承顺序
5. 第一顺序（配偶、子女、父母）
6. 第二顺序（兄弟、祖父母）

**直接上卷面的得分句**

"遗嘱继承优先于法定继承。如果王某的遗嘱符合法律形式（自书遗嘱有签名、日期）、内容合法、王某意思真实，该遗嘱有效，应按遗嘱分配：100万全部分给小儿子。其他继承人按法定继承顺序分不到遗产。如果遗嘱无效或没有遗嘱，则按法定继承分配：第一顺序（配偶、子女、父母）平均分，其他人不参与。"

### 情境2.3：相邻权与物权边界（待补充真实案例）

**起因→经过→结果（基本模式）**

李某的房子靠近河边，王某在河对岸新建了一栋高楼。王某的房子建成后，遮挡了李某房间的采光，李某白天都要开灯。李某起诉王某，要求王某拆迁或赔偿。

**争点**

王某有权建房，但这个权利在什么程度上会侵犯李某的采光权？李某能要求拆迁吗？

**对应规则与给分依据**

① **物权与相邻权的平衡**：
   - 王某有在自己的地上建房的权利（所有权、建筑权）。
   - 但李某也有相邻权，包括采光权、不被污染、通行等。
   - 两个权利之间要平衡。

② **相邻权的原则**：相邻各方应该"互让互利"。
   - 不是说王某完全不能建，也不是说李某的采光权无条件保护。
   - 而是要看王某的建房是否给李某造成了"明显不合理的干扰"。

③ **判决标准**：
   - 如果确实造成了明显伤害（例如李某房间完全无光、无法居住），李某可以要求王某采取措施（例如装防光膜、降低楼高）。
   - 但李某**不能直接要求拆迁**——这太极端了。
   - 如果伤害不是特别明显，李某的权利可能要让步。

④ **本题判断**：李某白天要开灯，说明采光受到了明显影响，不只是"轻微"。李某有权要求王某采取防护措施。但李某不能要求拆迁，除非能证明这个楼对自己造成了严重、不可接受的伤害。

**触发词 → 想到什么规则**

- 看到"相邻""邻居""旁边的建筑" → 相邻权
- 看到"采光""通风""排水" → 具体的相邻权利
- 看到"互让互利" → 平衡原则
- 看到"明显不合理的干扰" → 这是判断标准
- 看到"要求拆迁" → 通常不被支持，除非伤害极重

**踩分关键词**

1. 相邻关系/相邻权
2. 物权/所有权
3. 互让互利
4. 采光权/通风权/排水权
5. 明显不合理的干扰
6. 协商调整/采取措施
7. 不支持拆迁（通常）

**直接上卷面的得分句**

"王某在自己的地上有建房权利，但这个权利受到相邻法律的限制。根据相邻权原则，相邻各方应互让互利。王某的楼如果给李某造成了'明显不合理的干扰'（例如白天无光、无法正常居住），李某有权要求王某采取防护措施。但李某通常不能要求拆迁，除非能证明伤害已经达到了使房屋无法居住的程度。王某和李某应协商，例如装防光膜或其他防护措施。"

## 第三域：人身权益情境

### 情境3.1：名誉权侵害案（基本模式）

**起因→经过→结果**

某公司在社交平台发布信息说"张某的产品有严重质量问题，已经伤害了很多消费者，我们强烈建议大家不要购买"。但后来发现这个信息是假的——张某的产品其实通过了质量检测，从未伤害过消费者。这条虚假信息导致张某的销售额大幅下降，名声也受损。张某起诉要求赔偿。

**争点**

某公司侵犯了张某的哪个权利？应该承担什么责任？

**对应规则与给分依据**

① **权利识别**：
   - 散布虚假事实 → 侵犯名誉权（而不是隐私权）。
   - 隐私权保护的是"隐秘的真实信息"（例如偷拍私生活）。
   - 名誉权保护的是"社会评价"（是否被虚假信息贬低）。

② **名誉权侵害的构成**：
   - 散布虚假事实（说产品有问题，但其实没有）。
   - 造成了受害人名誉受损（销售下降、信用下降）。
   - 行为人有过错（知道是假的还说）。

③ **责任形式**：
   - 停止侵害（删除虚假信息）。
   - 赔偿损失（赔偿销售下降的损失）。
   - 赔礼道歉（公开说明真相、向张某道歉）。

④ **本题判决**：某公司应停止侵害、删除虚假信息、赔偿张某经济损失、公开道歉。

**触发词 → 想到什么规则**

- 看到"散布虚假信息""传播谣言" → 名誉权
- 看到"偷拍""监控""泄露隐秘信息" → 隐私权（不是名誉权）
- 看到"冒用名字" → 姓名权
- 看到"未经授权使用肖像获利" → 肖像权
- 看到"社交平台"+"虚假" → 名誉权侵害

**踩分关键词**

1. 名誉权
2. 散布虚假事实
3. 造成名誉受损
4. 停止侵害/删除
5. 赔偿损失
6. 赔礼道歉
7. 公开说明真相

**直接上卷面的得分句**

"某公司在社交平台散布虚假信息说张某产品有质量问题，这是虚假信息，损害了张某的社会评价，侵犯了张某的名誉权。某公司应承担停止侵害（删除虚假信息）、赔偿损失（赔偿销售下降等经济损失）、赔礼道歉（公开说明真相、向张某赔礼）的责任。"

### 情境3.2：一般侵权与侵权责任（基本模式）

**起因→经过→结果**

在某建筑工地，一名工人不慎从脚手架上掉下来，砸伤了楼下的行人王某。王某被送往医院，治疗花费了20万元，还需要半年康复。工地的安全管理有问题，没有设置隔离区、没有警告标志。王某要求工地赔偿所有医疗费和误工费。

**争点**

工地应该赔偿王某什么？侵权责任怎么算？

**对应规则与给分依据**

① **侵权责任三要件**（缺一不可）：
   - 侵害人实施了违法行为（工地没有做好安全防护）。
   - 造成了他人的伤害（王某被砸伤，花了20万医疗费）。
   - 行为与伤害之间有因果关系（是工地的疏忽导致了砸伤）。

② **本题分析**：
   - 违法行为：工地没有设置隔离区、没有警告，这违反了安全施工义务。
   - 伤害：王某被砸伤、需要治疗、康复期间无法工作。
   - 因果关系：正是工地的疏忽导致了工人掉下来、砸伤了王某。

③ **责任形式**：
   - 医疗费全额赔。
   - 误工费（康复期间的收入损失）赔。
   - 还可能有护理费、交通费等。
   - 如果造成了严重伤害（残疾、死亡），还有残疾赔偿金、死亡赔偿金。

④ **本题判决**：工地应赔偿王某医疗费20万、误工费、以及其他合理支出。

**触发词 → 想到什么规则**

- 看到"伤害""受伤""被打" → 侵权
- 看到"违反义务""不做安全防护" → 违法行为
- 看到"医疗费""治疗" → 侵权责任
- 看到"三要件"的某个要素缺失 → 可能不构成侵权

**踩分关键词**

1. 侵权责任
2. 违法行为/违反安全义务
3. 造成伤害
4. 因果关系
5. 赔偿医疗费/误工费/护理费
6. 赔偿额 = 具体数字

**直接上卷面的得分句**

"工地的违法行为（没有安全防护）导致工人坠落、砸伤了王某。三个要件都成立：有违法行为、有伤害、有因果关系。根据民法典，侵权人应承担赔偿责任。赔偿范围包括：医疗费20万元、误工费（康复期间的收入损失）、护理费、交通费等合理支出。"

### 情境3.3：多主体共同侵权（特殊情境）

**起因→经过→结果**

某社区里有三个建筑工地：工地A、B、C。这三个工地都在进行内部装修，产生了大量粉尘和噪音。整个社区的空气质量严重恶化。王某住在社区里，因为长期吸入粉尘，患上了呼吸道疾病，治疗花费15万元。王某不知道哪个工地责任最大，就决定起诉工地A。工地A认为"这是三个工地一起造成的污染，我一个工地的责任不大，不该赔那么多"。

**争点**

多个工地都参与了污染，王某怎么起诉？工地A要赔多少？

**对应规则与给分依据**

① **共同侵权的定义**：
   - 多个侵权人共同实施侵害行为。
   - 每个侵权人都对受害人的伤害有因果关系。
   - 形成"共同侵权"。

② **共同侵权人的责任**：
   - **连带责任**：受害人可以向任意一个共同侵权人追责，要求该侵权人承担全部赔偿责任。
   - 该侵权人赔偿后，可以向其他侵权人追偿（要求其他人分摊）。
   - 如果某个侵权人责任比例大（例如工地A的污染最严重），赔偿后追偿时，其他工地要分摊比例更多。

③ **本题判断**：
   - 工地A、B、C都产生了污染。
   - 王某的呼吸道疾病是三个工地共同导致的。
   - 王某可以起诉工地A，要求工地A赔偿15万全部医疗费。
   - 工地A可以在赔偿后，起诉工地B、C，要求他们按比例分摊。

④ **注意**：法律不要求王某"先确认各工地的责任比例"就能起诉。受害人有权向任意侵权人追责，侵权人之间的责任分摊是他们的事。

**触发词 → 想到什么规则**

- 看到"多个行为人""共同导致""多方都有问题" → 共同侵权
- 看到"连带责任" → 可向任意一个追责
- 看到"分摊""追偿" → 侵权人之间的内部结算
- 看到"受害人不知道哪个最严重" → 不影响追责权

**踩分关键词**

1. 共同侵权/多主体侵权
2. 连带责任
3. 向任意侵权人追责
4. 赔偿全部损失
5. 追偿/分摊
6. 因果关系(多个行为都有)

**直接上卷面的得分句**

"工地A、B、C都实施了产生污染的行为，都对王某的呼吸道疾病有因果关系，形成共同侵权。根据民法典，共同侵权人承担连带责任。王某可以向工地A（或B、C中的任何一个）追责，要求工地A赔偿15万元全部医疗费。工地A赔偿后，可以向工地B、C追偿，要求他们按责任比例分摊。"

## 第四域：创新竞争情境

### 情境4.1：知识产权侵权与惩罚性赔偿（2026朝阳一模 Q18）

**起因→经过→结果**

乙公司培育了一个新的玉米品种，获得了植物新品种权。甲公司明知乙公司的权利，仍然在2020年开始生产和销售这个玉米品种（既没有取得授权、也没有支付费用）。甲公司这样持续侵权到2024年，获利超过5亿元。乙公司一直在研发和改进这个品种，但甲公司的侵权行为严重扰乱了市场秩序、伤害了乙公司的积极性。最后，法院认定甲公司构成"故意严重侵权"，判决甲公司赔偿5300万元（远高于乙公司的直接损失）、同时停止侵害、销毁侵权产品。

**争点**

为什么赔偿5300万？这不是甲公司的全部获利，也不是乙公司的全部损失。法律怎么算的？

**对应规则与给分依据**

① **知识产权侵权的基本责任**：
   - 停止侵害（停止销售）。
   - 赔偿实际损失（乙公司因侵权而失去的利润）。

② **惩罚性赔偿（关键词）**：
   - 当侵权人是"故意且严重侵权"时，法律允许法院判决"高于实际损失的赔偿"。
   - 目的是"遏制侵权、惩罚不法行为、激励创新"。
   - 赔偿额通常是实际损失的1-5倍，或者法律规定的固定额度（例如"不超过500万元"）。

③ **本题判断**：
   - 甲公司"明知"乙公司的权利但仍然侵权 → 有故意。
   - 甲公司从2020年侵权到2024年，获利5亿多 → 这是"严重侵权"。
   - 法院判决5300万赔偿 ≈ 实际损失（例如1000万）× 5倍左右。
   - 这就是"惩罚性赔偿"。

④ **意义**：
   - 保护创新者的权益（乙公司有权获得回报）。
   - 遏制侵权（高额赔偿让甲公司付出沉重代价，其他企业会更谨慎）。
   - 维护市场秩序（不能让侵权者随意掠夺创新成果）。

**触发词 → 想到什么规则**

- 看到"侵犯知识产权""未经授权" → 知识产权侵权
- 看到"明知还侵权""故意侵权""多年持续侵权" → 故意 + 严重
- 看到"赔偿额远高于损失""1-5倍""高额赔偿" → 惩罚性赔偿
- 看到"植物新品种""专利""商标""著作权" → 知识产权的种类

**踩分关键词**

1. 植物新品种权/专利权/商标权
2. 未经授权/侵权
3. 故意侵权/严重侵权
4. 惩罚性赔偿
5. 停止侵害/销毁侵权产品
6. 赔偿额 = 具体数字(通常高于直接损失)
7. 激励创新/维护市场秩序

**直接上卷面的得分句**

1. "甲公司未经乙公司授权，生产和销售受植物新品种权保护的玉米品种，这是侵犯知识产权的行为。"（1分）

2. "甲公司明知乙公司的权利但仍然长期侵权（2020-2024年），获利超过5亿元，这构成'故意且严重侵权'。"（1分）

3. "根据民法典和知识产权法的规定，对故意严重侵权的行为，法院可以适用惩罚性赔偿。赔偿额可以是实际损失的1-5倍，或者法律规定的额度。本案判决赔偿5300万元，就是惩罚性赔偿。"（1.5分）

4. "甲公司应承担的责任包括：（1）停止侵害——停止生产销售；（2）销毁侵权产品；（3）赔偿损失——5300万元。"（1.5分）

5. "这样的判决有利于保护知识产权人的权益、遏制侵权行为、激励企业投入研发创新、维护公平竞争的市场秩序。"（1分）

**作答逻辑**

看到这道题，你要识别"这不是简单的民事损害赔偿，而是知识产权的'惩罚性赔偿'"。关键是理解"为什么赔偿5300万而不是甲公司的全部获利或乙公司的直接损失"——答案是"法律故意设定高于实际损失的赔偿额来惩罚故意严重侵权"。你的答题里要强调两个关键词："故意"和"严重"，这两个词决定了是否适用惩罚性赔偿。

### 情境4.2：反不正当竞争与商业诋毁（基本模式）

**起因→经过→结果**

甲公司和乙公司是竞争关系。甲公司为了打击乙公司，在网络上散布虚假信息说"乙公司产品的主要原料有毒，已经伤害了消费者"。实际上，乙公司的原料完全安全、通过了检测、消费者投诉率也很低。这条虚假信息导致乙公司的销售额下跌60%、股价也跌了。乙公司起诉甲公司。

**争点**

甲公司的行为侵犯了乙公司的什么权利？属于什么法律关系？

**对应规则与给分依据**

① **与名誉权的区别**：
   - 如果是"王某散布虚假信息说竞争对手的产品有毒"，那是"商业诋毁"属于"反不正当竞争"。
   - 关键是：散布虚假信息的**目的是打击竞争对手、争夺市场**，而不只是为了贬低某个人的名声。
   - 虽然名誉权和商业诋毁都涉及"散布虚假信息"，但反不正当竞争考查的是"企业之间的不公平竞争"。

② **反不正当竞争的行为**：
   - **虚假宣传**：说自己的产品是"最好的""世界第一"但无根据。
   - **商业诋毁**：散布虚假信息贬低竞争对手（本题）。
   - **侵犯商业秘密**：偷了竞争对手的配方、客户名单。
   - **假冒**：冒用竞争对手的商标、品牌。
   - **强制交易**：强制消费者买自己的产品。
   - **倾销**：用不合理的低价格打击竞争对手。

③ **本题判断**：
   - 甲公司散布虚假信息"乙公司产品有毒" → 商业诋毁。
   - 目的是打击乙公司、争夺市场 → 不正当竞争。
   - 甲公司应承担停止侵害、赔偿损失、公开说明真相的责任。

④ **责任形式**：
   - 停止侵害（删除虚假信息、停止散布）。
   - 赔偿损失（赔偿乙公司的销售下降、股价跌幅等经济损失）。
   - 公开说明真相（发声明说"之前的信息是虚假的，乙公司产品是安全的"）。

**触发词 → 想到什么规则**

- 看到"企业竞争""竞争对手"+"虚假信息" → 反不正当竞争（不是名誉权）
- 看到"散布虚假信息""打击竞争对手" → 商业诋毁
- 看到"虚假宣传""吹嘘自己" → 反不正当竞争
- 看到"侵犯商业秘密" → 反不正当竞争
- 看到"强制交易""绑定销售" → 反不正当竞争

**踩分关键词**

1. 反不正当竞争/商业诋毁
2. 虚假信息/散布谣言
3. 打击竞争对手
4. 停止侵害
5. 赔偿损失（销售下降、股价跌幅等）
6. 公开说明真相
7. 维护市场秩序/公平竞争

**直接上卷面的得分句**

"甲公司散布虚假信息说乙公司产品有毒，其目的是打击竞争对手、抢占市场份额，这构成'商业诋毁'属于反不正当竞争行为。甲公司应承担停止侵害（删除虚假信息）、赔偿损失（赔偿乙公司的销售下降等经济损失）、公开说明真相（声明乙公司产品是安全的）的责任。这样有利于维护市场秩序、保护公平竞争。"

## 第五域：救济公共情境

### 情境5.1：调解优先与诉讼程序（2026朝阳一模 Q18 调解部分）

**起因→经过→结果**

在生物医药领域，甲研究所和乙公司为了一项专利权的归属产生了纠纷。按通常的诉讼流程，这个案件可能要打3-5年。但人民法院没有直接判决，而是通过邀请第三方调停，帮助双方协商和解。最终，甲研究所和乙公司达成了和解协议：乙公司获得该专利的使用权、甲研究所获得合理的授权费。双方一揽子化解了所有关联诉讼（有多个相关纠纷）。

**争点**

为什么法院要调解而不是直接判决？调解有什么好处？

**对应规则与给分依据**

① **调解的含义**：调解是指纠纷双方在第三方（例如法院、仲裁机构、专业调解员）的帮助下，通过协商达成和解的方式。

② **调解的特点**：
   - **快速**：不用走完整的审判流程，可以快速解决纠纷。
   - **保密**：调解通常不公开，保护当事人的隐私。
   - **合作**：双方协商、共同寻找解决方案，而不是对立。
   - **灵活**：调解协议可以包含法院判决无法涵盖的内容（例如持续合作、技术转让等）。

③ **本题的意义**：
   - 对研究机构：不用花3-5年时间打官司，可以专注科研。
   - 对乙公司：快速获得专利使用权，可以尽快商业化。
   - 对司法资源：减少法院的审理压力。
   - 对全社会：促进科研成果的转化和商业化，有利于创新。

④ **调解的前提**：双方要自愿。法院不能强迫当事人调解。如果双方同意，才能进行调解。

**触发词 → 想到什么规则**

- 看到"调解""第三方调停""协商解决" → 调解优先
- 看到"诉讼之前"或"诉讼中途" → 调解可以在任何时点进行
- 看到"快速解决""节省时间""一揽子化解" → 调解的好处
- 看到"和解协议" → 调解的结果

**踩分关键词**

1. 调解
2. 协商/和解
3. 第三方介入
4. 快速高效
5. 减少诉讼对创新的干扰
6. 一揽子化解关联纠纷
7. 节省司法资源

**直接上卷面的得分句**

"人民法院坚持调解优先原则。通过邀请第三方调停，帮助甲研究所和乙公司协商，最终达成和解协议。调解方式相比诉讼具有以下优势：（1）快速高效，不用经历长期的审判程序；（2）一揽子化解所有关联纠纷，更彻底；（3）减少诉讼对创新活动（科研）的干扰，让研究人员安心投入研发；（4）节省司法资源。"

### 情境5.2：诉讼程序与法院支持（嵌入几乎所有主观题）

**起因→经过→结果**

王某向互联网法院提起民事诉讼，要求A公司退还票款310元并三倍赔偿930元。王某提交了证据：航空公司官网上的机票价格查证、自己支付310元的订单截图、发票。法院经过审理，查证了王某的主张是真实的，A公司确实有欺诈行为。法院判决支持王某的诉讼请求：A公司要退还310元、赔偿930元。

**争点**

诉讼程序怎么走的？法院怎么判的？

**对应规则与给分依据**

① **民事诉讼的基本程序**：
   - **起诉**：王某向法院提起诉讼，提交诉状、证据。
   - **受理**：法院确认有管辖权、符合起诉条件，决定受理。
   - **审理**：法院调查取证、听取双方意见。（可能在这一阶段进行调解）
   - **判决**：法院根据证据和法律，做出判决。

② **举证责任**：
   - 原则：谁主张谁举证。王某说A公司欺诈，王某要提供证据。
   - 本题：王某提供了航空公司价格、订单截图、发票，证实了自己的主张。

③ **法院的判决**：
   - 法院查明事实（A公司确实搭售了红包、王某确实不清楚）。
   - 适用法律（消费者权益保护法、合同欺诈规定）。
   - 判决支持诉讼请求（退款+三倍赔偿）。

④ **上诉权**：
   - 如果A公司不服一审判决，可以向上级法院上诉（二审）。
   - 二审的判决是最终判决，不能再上诉。

**触发词 → 想到什么规则**

- 看到"向法院提起诉讼" → 诉讼程序开始
- 看到"提供证据""举证" → 举证责任
- 看到"法院查明""经审查" → 调查取证
- 看到"法院判决"或"法院支持" → 判决环节
- 看到"不服一审判决""上诉" → 二审

**踩分关键词**

1. 诉讼/民事诉讼
2. 起诉/向法院提起诉讼
3. 受理/法院受理
4. 举证/提供证据
5. 审理/法院审查
6. 调查取证
7. 判决/法院判决
8. 支持诉讼请求/驳回诉讼请求
9. 上诉/二审

**直接上卷面的得分句**

"王某向互联网法院提起民事诉讼，提交了证据（航空公司价格查证、订单截图、发票）。法院受理案件后，进行了审理和调查取证，查明了事实：A公司确实有欺诈行为。根据消费者权益保护法第55条，法院判决支持王某的诉讼请求：A公司退还票款310元，并赔偿310元×3=930元。如果A公司不服，可以向上级法院上诉。"

## 第二章：选择题情境（情境 + 正确判断 + 错项陷阱 + 法律边界）

### 选1：格式条款无效情形（常见选择题）

**情境**

某公寓租赁平台的合同里写着："房东有权随时进入房间检查维修，租户无权拒绝。租户同意放弃隐私权保护。"

**正确判断**

这个格式条款无效。房东的这个条款加重了租户的负担（强制进入、放弃隐私权），根据民法典，加重消费者（租户）负担的格式条款无效。

**错项陷阱**

- A项："这个条款合理，因为房东需要维护房屋。" ← 错。房东有维护权，但不能不通知租户就进入、也不能要求租户放弃隐私权。
- B项："租户可以拒绝，但得补偿房东。" ← 错。这不是谈判而是强制，条款本身就无效。
- C项："只要双方都签名同意就有效。" ← 错。即使双方签了，加重消费者负担的条款仍然无效。

**法律边界提醒**

格式条款无效有三种常见情况：（1）商家免除自己责任；（2）加重消费者负担；（3）排除消费者法定权利。选择题常常考"这个条款属于哪一种无效情形"。注意："有歧义时按对商家不利的解释"不是"无效"，而是"规则"。

---

### 选2：消费者的三个权利（常见选择题）

**情境**

某服装店在结账时突然说"我们这边有内部规定，所有商品不能退换，你刷卡时就默认同意了"。消费者说"我不知道这个规定"。

**正确判断**

消费者有权退换。即使店里有这个"内部规定"，它违反了消费者的自主选择权和公平交易权。店家不能以"默认同意"为借口强制消费者接受。

**错项陷阱**

- A项："既然刷卡了就是同意，不能退换。" ← 错。消费者可能不知道这个规定，不构成真实的意思表示同意。
- B项："只要购买时没有明确拒绝就是同意。" ← 错。消费者没有"明确拒绝"不等于"同意"。同意要是积极的、知情的。
- C项："这是商家的自由定价权，消费者无法干涉。" ← 错。定价自由不等于可以强制消费者放弃法定权利。

**法律边界提醒**

消费者的法定权利只有三个：知情权、自主选择权、公平交易权。其他权利（例如"安全权""监督权"）都是衍生的。选择题会问"消费者有什么权利"——一定只选这三个。

---

### 选3：婚前财产的认定（常见选择题）

**情境**

张某和李某在2020年结婚。结婚前，张某在2019年买了一套房，2020年3月办理了产权证。结婚后，这套房升值了100万。现在离婚，李某要求分割房产升值的部分。

**正确判断**

这套房（包括升值部分）都是张某的婚前财产，不参与分割。关键是"购买时间"（2019年婚前），而不是"办证时间"（2020年婚后）。

**错项陷阱**

- A项："产权证是婚后办的，所以是共同财产。" ← 错。办证时间不决定归属，购买时间才决定。
- B项："房产升值部分是婚后获得的，应该分割。" ← 错。升值不改变房产本身的性质（婚前财产），仍不分割。
- C项："虽然是婚前买的，但因为在婚后升值了很多，应该部分分割。" ← 错。民法典没有这样的规则。婚前财产就是不分。

**法律边界提醒**

婚前财产有三种情况：（1）婚前购买的房产/车/存款；（2）婚前继承的遗产；（3）婚前收到的赠与（赠与协议明确只给一个人）。只要符合其中一种，就完全不分割，不管升值多少。

### 选4：定金与违约金的区别（常见选择题）

**情境**

房东和租户约定支付1000元定金。后来租户违约不租了。房东现在有两个选择：（1）没收定金；（2）要求租户赔偿违约金。房东能怎样做？

**正确判断**

房东只能选其一：要么没收定金，要么要求赔偿违约金，不能既没收定金又要赔违约金。定金本身就带有"惩罚"的性质，不能重复计算。

**错项陷阱**

- A项："房东可以既没收定金，又要求赔偿违约金。" ← 错。这是"重复惩罚"。
- B项："定金不返还就等于赔偿了，不用再支付违约金。" ← 正确方向但表述不准。准确是"房东要么选没收定金，要么选赔违约金"。
- C项："订金（不是定金）不能没收。" ← 正确。但题目问的是"定金"不是"订金"，两者不同。定金（有法律效力）vs 订金（只是预付款）。

**法律边界提醒**

定金有法律约束力，订金没有。定金、违约金、损害赔偿金是三个不同的概念。选择题常常混着考，要能区分。

---

### 选5：相邻权与采光权（常见选择题）

**情境**

李某住在东边，王某在西边新建了高楼。王某的楼没有违反建筑规划法，但遮挡了李某的采光，李某白天都要开灯。李某起诉要求王某拆迁。法院会支持吗？

**正确判断**

法院通常不会支持拆迁。理由是：相邻权要求"互让互利"，而不是"绝对保护采光"。王某有合法建房权，李某要求拆迁太极端。李某可以要求王某采取防护措施（例如装防光膜），但不能要求拆迁。除非李某能证明这个楼造成了"使房屋无法居住"的严重伤害。

**错项陷阱**

- A项："李某的采光权绝对受保护，王某必须拆迁。" ← 错。采光权有限制，不是绝对的。
- B项："王某的建房权绝对受保护，李某无权反对。" ← 错。王某虽然有建房权，但要受相邻权约束。
- C项："相邻各方可以协商，如果协商不成，谁的房在前谁的权利优先。" ← 错。不是"谁在前谁优先"，而是"互让互利"的平衡。

**法律边界提醒**

相邻权常见的有采光权、通风权、排水权、通行权等。选择题会考"这个是否侵犯了相邻权"。关键是看有没有造成"明显不合理的干扰"——如果只是"轻微影响"，通常不构成侵权。

---

### 选6：继承权的丧失（常见选择题）

**情境**

老爷子有三个儿子。老二听说老爷子要改遗嘱、把大部分遗产留给老大，就故意杀害了老爷子来"争夺遗产"。老爷子去世后，能否由老二继承？

**正确判断**

老二丧失继承权，不能继承。根据民法典，因为以下行为会丧失继承权：（1）故意杀害被继承人；（2）为争夺遗产杀害其他继承人；（3）遗弃被继承人；（4）伪造、篡改、销毁遗嘱。老二的行为属于"为争夺遗产杀害被继承人"。

**错项陷阱**

- A项："老二杀人是犯罪，但这不影响继承权。" ← 错。继承权和刑事责任是分开的，但民法也规定了丧失继承权的情况。
- B项："老二只是"有嫌疑"，没有"确定被定罪"，所以还能继承。" ← 错。继承权丧失的判断不一定要等刑事判决，民事判断可以独立进行。
- C项："其他兄弟也有责任（例如没有及时报警），所以都应该丧失继承权。" ← 错。丧失继承权有明确的法定情形，不能扩大解释。

**法律边界提醒**

继承权丧失有五种情况：杀害被继承人、杀害其他继承人、遗弃被继承人、伪造/篡改/销毁遗嘱、以欺诈/胁迫手段迫使被继承人改遗嘱或添加自己。选择题会问"某人是否丧失继承权"——要能从具体事实匹配到这五种情况。

---

### 选7：劳动关系与劳务关系的区别（常见选择题）

**情境**

某外卖平台上，王某接单送外卖。王某自备电动车、自己安排时间、可以自由选择接单不接单。平台只是在APP上显示订单，不对王某的行为进行管理。王某和平台是什么关系？

**正确判断**

这是"劳务关系"而不是"劳动关系"。虽然王某提供劳务（送外卖），但不满足劳动关系的"三维度"：
- 人格从属性弱：王某可以自由选择接单，没有受到纪律约束。
- 经济从属性弱：王某自备车、自己定价（接哪个订单）。
- 组织从属性弱：王某可能同时和多个平台合作。

**错项陷阱**

- A项："王某在平台上接单工作，肯定是劳动关系。" ← 错。不是所有在平台工作都是劳动关系。
- B项："只要有劳务就是劳动关系。" ← 错。劳务关系≠劳动关系。劳动关系有法律保护（最低工资、社保等），劳务关系没有。
- C项："劳动关系和劳务关系完全一样，只是名字不同。" ← 错。两者有本质区别，特别是在责任、保护、税收上。

**法律边界提醒**

劳动关系认定的关键是"三维度"（人格、经济、组织从属性）是否都满足。选择题常常在这个地方出陷阱，混淆"劳动关系"和"劳务关系"。

---

### 选8：人格权——名誉权 vs 隐私权（常见选择题）

**情境**

某网络平台发布信息说"张某是通奸者"，后来发现这个信息是假的。张某能起诉吗？侵犯的是哪个权利？

**正确判断**

张某可以起诉，侵犯的是"名誉权"。虽然这个信息涉及"隐秘生活"，但关键是"这个信息是假的"——散布虚假事实损害他人的社会评价，属于名誉权侵害。

如果反过来，如果信息是"真的"（张某确实通奸了），但某个人未经同意就公开散布，那才是"隐私权侵害"。

**错项陷阱**

- A项："这是隐私权侵害，因为涉及个人生活。" ← 错。虚假信息→名誉权，真实但隐秘的信息→隐私权。
- B项："因为信息是假的，所以是诽谤罪（刑事），不是民事侵权。" ← 错。可以既是刑事诽谤，也是民事名誉权侵害，两者不矛盾。
- C项："张某名声不好，散布信息不能算侵权。" ← 错。不管名声好坏，散布虚假信息都侵权。

**法律边界提醒**

名誉权和隐私权的区分：
- 名誉权：保护社会评价。散布虚假事实→名誉权。
- 隐私权：保护隐秘生活。散布真实但隐秘信息→隐私权。
这是高频出题点，一定要掌握。

### 选9：知识产权侵权与欺诈的区别（常见选择题）

**情境**

甲公司用了乙公司注册的商标去销售产品，但产品的质量很好、没有欺骗消费者。甲公司侵权吗？

**正确判断**

甲公司侵权。知识产权侵权和"是否欺骗消费者"无关。甲公司未经授权使用他人商标就是侵权，即使产品质量好。甲公司要停止使用、赔偿损失。

**错项陷阱**

- A项："产品质量好，所以不侵权。" ← 错。商标权是"排他权"，只要未经授权使用就侵权，不看产品质量。
- B项："消费者没有受骗，所以不构成侵犯商标权。" ← 错。消费者是否受骗是"欺诈"的问题，不是"商标侵权"的问题。
- C项："如果乙公司同意了，甲公司就可以用。" ← 正确方向但题目已经说"没有同意"。

**法律边界提醒**

知识产权侵权和消费欺诈是两个不同的法律问题：
- 知识产权侵权：关键是"未经授权"。
- 消费欺诈：关键是"虚假宣传""隐瞒真相"。
两者可能同时发生（甲用了乙的商标且产品有缺陷），也可能只有一个成立（甲用了乙的商标但产品没问题）。

### 选10：反不正当竞争与知识产权的区别（常见选择题）

**情境**

甲企业虚假宣传说自己的产品"获得了国际大奖""排名世界第一"，其实都没有。乙企业（竞争对手）起诉。这属于什么法律问题？

**正确判断**

这是"反不正当竞争"（虚假宣传），而不是"知识产权侵权"（没有涉及专利、商标、著作权的实质侵害）。甲企业应停止虚假宣传、赔偿乙企业的损失。

**错项陷阱**

- A项："这是商标权侵权，因为涉及'品牌名'。" ← 错。虽然涉及品牌宣传，但这里没有"使用他人的商标"，只是"虚假吹嘘自己"。
- B项："这是著作权侵害，因为涉及'宣传文案'。" ← 错。即使宣传文案本身有著作权，也不是这里的问题。这里问的是"宣传内容是否虚假"。
- C项："这不违法，因为企业有宣传自己的自由。" ← 错。宣传自由不等于"虚假宣传自由"。虚假宣传违反反不正当竞争法。

**法律边界提醒**

反不正当竞争和知识产权侵权的区别：
- 反不正当竞争：通过不诚实、不公平的手段（虚假宣传、商业诋毁、低价倾销等）打击竞争对手。
- 知识产权侵权：侵犯他人受法律保护的创新成果（专利、商标、著作权、商业秘密）。
注意：两者的保护对象不同（反不正当竞争保护"市场秩序"，知识产权保护"创新成果"）。

### 选11：侵权责任与多主体（常见选择题）

**情境**

工地A和工地B都在某社区进行施工。粉尘污染导致了王某的呼吸道疾病。王某只向工地A起诉，但没有起诉工地B。法院会怎样判？

**正确判断**

法院会判决工地A赔偿王某的全部损失。工地A和工地B形成共同侵权，承担连带责任。王某可以向其中任意一个追责。工地A赔偿后，可以向工地B追偿。

**错项陷阱**

- A项："工地A的责任只是'一半'，因为有两个工地污染。" ← 错。连带责任意味着王某可以向工地A要求"全部赔偿"，不需要先分摊。
- B项："王某必须同时起诉工地A和B，否则法院不处理。" ← 错。王某可以只起诉其中一个。
- C项："工地A赔了之后，可以选择向工地B追偿，也可以选择不追偿。" ← 半对。工地A有权追偿，但具体金额（按责任比例）是法律的问题，不是工地A的"选择"。

**法律边界提醒**

共同侵权的连带责任是民法典的重要制度。选择题常常问"受害人可以向谁追责""赔偿金额是多少"。关键是理解：受害人可以向任意一个共同侵权人追责，侵权人之间的内部分摊是他们解决的事。

### 选12：诉讼管辖与起诉地（常见选择题）

**情境**

某商家在北京网上销售产品给住在上海的王某，产品有缺陷伤害了王某。王某应该向哪个地方的法院起诉？

**正确判断**

王某可以向以下任一地的法院起诉：（1）被告（商家）所在地法院（北京）；（2）产品销售地（上海，王某住地）；（3）侵权地（上海，产品伤害王某的地方）。最常见的是被告所在地或侵权地。

**错项陷阱**

- A项："必须向被告所在地法院起诉。" ← 半对。被告所在地是"通常"管辖权，但其他地点也可以。
- B项："网络交易的纠纷，必须向电商平台所在地起诉。" ← 错。平台所在地不是管辖权的依据。
- C项："王某住在上海，必须向上海法院起诉。" ← 半对。上海法院有管辖权（侵权地），但不是"只有"上海法院。

**法律边界提醒**

民事诉讼的管辖权规则：
- 一般地：被告住所地法院。
- 合同纠纷：被告住所地或合同履行地法院。
- 侵权纠纷：侵权地法院。
- 网络交易纠纷：被告住所地或其他有关地法院。
选择题会考"这个案件应该向哪个法院起诉"。

## 写在最后：用情境穷尽手册做新题

拿到一道新的法律题，按这个流程用本手册：

**第一步：读情境，定位域。** 题目讲的是什么故事？是合同纠纷（交易交换）？是邻居打架（人身权益）？是品种被抄（创新竞争）？还是去法院打官司（救济公共）？看到新题时，先翻本手册的目录，找到对应的情境家族。

**第二步：对标已知情境。** 新题和手册里的哪个情境"故事原型"最接近？是"机票搭售案"的变种？是"劳动关系认定"的变种？还是"名誉权"和"隐私权"混在一起？对标后，你就知道该用哪套法律规则。

**第三步：套用得分句模板。** 不要重新创作答题语言。本手册给出的每一个"直接上卷面的得分句"都经过了阅卷实践的检验。你只需要把题目的具体人物、具体事项代入模板，句子结构就完整、逻辑就清楚。

**第四步：检查踩分关键词。** 做完答题后，看看你有没有写出这个情境对应的"踩分关键词"。缺一个关键词，可能就少0.5-1分。本手册给出的关键词都是"法院/仲裁委评分细则里明确要求的"，不是瞎编的。

**第五步：对答，查漏补缺。** 做完题目后，用本手册的"对应规则与给分依据"来对答。有没有漏掉"法律名称"（例如写成"消费者权益法"而不是"消费者权益保护法"）？有没有漏掉某个"法律要件"（例如欺诈的"三要素"里只写了两个）？有没有"混淆不同的权利"（例如把名誉权写成隐私权）？

做完这5步，你对这道题的理解就达到了"能得分"的水平。多做几道，你会发现：法律题其实就这么多情境，就这么多套规则，就这么多种答题方式。掌握了这个手册，新题再怎么变，你都能快速定位、快速应对。
