# 框架进化日志（v1 → v2）

跑完 60 套卷子、汇总 99 主观题 + 78 选择题选必二真实数据后，对 v1 框架的反思与进化。
**进化原则：所有改动必须有题号证据**。

## 1. v1 框架要点回顾
- 一核：以事实为根据、以法律为准绳，定分止争
- 二线：权利保护与权利边界 / 法治规则与德治价值
- 三问：判什么/怎么处理 / 凭什么 / 有什么意义
- 四步：定主体关系 → 找争点事实 → 套法定要件 → 落责任/效力/归属/救济/价值
- 五域：交易交换 / 身份财产 / 人身权益 / 创新竞争 / 救济公共

- "未分类"15 题需要二轮人工审。

## 3. v1 → v2 五条结构改动

### 改动 1：救济公共域 不再是平行第 5 域，改为嵌入层"程序责任落点"

**v1**：五域并列，第 5 域 = 救济公共。
**v2**：救济公共**降为嵌入层**，每个主观题"四步法"的第四步必到这层；五域只剩 4 个独立域。

**证据**：99 题中只有 4 题"主轴是诉讼程序"（如举证责任、公益诉讼），其余 24 次诉讼/调解/仲裁出现都是其他域的最后一步嵌入。
**学生收益**：不再让学生以为"救济公共"是个独立知识块；改为"每道题最后必须写程序、责任、价值"的固定动作。

### 改动 2：交易交换域内部重排序——合同效力四档作为头号节点

**v1**：交易交换域下平铺合同、消保、格式条款、劳动。
**v2**：交易交换域内部按"出现频次 + 答题动作链"重排：
1. **合同效力四档**：成立 / 生效 / 可撤销 / 无效（含欺诈三要素）/ 解除（违约）→ 头号主干
2. **消保三权 + 三倍赔偿**：与合同欺诈紧耦合（10/43 合同题里有消保耦合）
3. **格式条款无效三情形**：免除商家责任 / 加重消费者负担 / 排除法定权利（选择题专属为主）
4. **劳动关系三从属性**：人格 / 经济 / 组织（平台经济新趋势，2024朝阳二模 Q17、2024海淀一模 Q20、2026东城期末 Q21）

**证据**：合同 43 题里至少 22 题以"合同效力判断"为核心争点。
**学生收益**：把 v1 的"合同→消保→格式→劳动"四节并列，改成"先效力判断，再消保叠加"的有序工作流。

### 改动 3：人身权益域 拆出 "侵权三要件 + 多主体连带" 独立节点

**v1**：人身权益域下平铺人格权、一般侵权、特殊侵权。
**v2**：以"侵权三要件 + 多主体连带"为头号节点（最高频考核点），人格权降为次节点。

**证据**：
- 一般侵权 + 多主体侵权 = 8 题（2025昌平二模 Q21 多主体污染，2025房山一模 Q20 侵权）
- 人格权独立题 1 题（2024海淀二模 Q21）
**学生收益**：突出"违法行为 + 损害后果 + 因果关系"三要件，附带"多主体连带"特别规则。

### 改动 4：创新竞争域 强调"惩罚性赔偿"与"调惩辨"答题模式

**v1**：知产 + 反不竞，节点笼统。
**v2**：创新竞争域增设两个专题节点：
1. **知产惩罚性赔偿三要件**：故意 + 严重 + 1-5 倍赔偿（2026朝阳一模 Q18 植物新品种权、2025丰台一模 Q19）
2. **调惩辨三维**：调解优先 / 惩罚高额 / 辨明边界（2026朝阳一模 Q18，新趋势）

**证据**：2026 年开始出现"调惩辨"复合考点（调解 + 惩罚 + 辨别），不是单纯"侵权赔偿"。
**学生收益**：抓住知产题的新趋势，避免只会答"侵权停止 + 赔偿"老套路。

### 改动 5：五域→4 主干 + 1 嵌入 + 1 选择题专属

**v2 总图**：
```
一核：以事实为根据、以法律为准绳，定分止争

二线：权利保护与权利边界 / 法治规则与德治价值

三问：判什么 / 怎么处理 / 凭什么 / 有什么意义

四步：
  Step 1 定主体关系
  Step 2 找争点事实
  Step 3 套法定要件
  Step 4 落责任/效力/归属/救济/价值（→ 程序责任落点 嵌入层）

四主干（按频次降序）：
  ① 交易交换域 — 合同效力四档（头号）/ 消保三权 / 格式条款 / 劳动三从属性
  ② 身份财产域 — 婚姻共同 vs 个人财产 / 法定 vs 遗嘱继承 / 物权与相邻五子项
  ③ 人身权益域 — 侵权三要件（头号）/ 多主体连带 / 人格权识别
  ④ 创新竞争域 — 知产惩罚性赔偿（头号）/ 调惩辨 / 反不竞商业诋毁

程序责任落点 嵌入层：
  调解 → 仲裁 → 诉讼 → 举证 → 执行 → 公益（每题最后一步必到，但不独立成域）

选择题专属边界：
  限制民事行为能力 / 定金订金违约金 / 格式条款细节 / 商标权范围 / 著作权合理使用 / 商业秘密三要件 / 诉讼时效 / 管辖权 / 一审二审再审
```

## 5. v2 框架"主干性"自检

按"考频高 + 进入细则深 + 学生最容易丢分"三标准，主干层应当包含：
1. **合同效力四档（含欺诈三要素）+ 消保三倍赔偿** —— 22 题考过，绝对主干
2. **侵权三要件 + 多主体连带** —— 10 题考过，第二主干
3. **婚姻共同 vs 个人财产** —— 6 题考过，第三主干
4. **法定 vs 遗嘱继承** —— 4 题考过，第三主干
5. **劳动关系三从属性** —— 6 题考过，第三主干（新趋势）
6. **知产惩罚性赔偿** —— 6 题考过，第三主干

次主干（每个 2-4 题）：
- 物权与相邻五子项
- 反不正当竞争（含商业诋毁、虚假宣传）
- 人格权侵害与责任
- 格式条款无效三情形（主要选择题，但主观题也有触及）

边界节点（学生为新题预留容器）：
- 公益诉讼（生态污染、消费者群体）
- 仲裁前置
- 网络消费的特殊规则（实名、平台先行赔付）

## 6. v2 框架"教研员 PPT"对齐自检

韩金阳教研员 PPT 要求 8 对相统一 + 三种载体 + 防误用。v2 框架对应：
- **学理性 ↔ 政治性**：v2 主干是"具体法律制度"（学理性），意义层用本模块语言（政治性）
- **知识性 ↔ 价值性**：v2 每题都"知识层（要件） + 价值层（意义）"分两层
- **主导性 ↔ 主体性**：v2 给学生"看到什么材料 → 想到什么规则 → 写出什么得分句"工作流（学生主体性）
- **三类载体**（案例 / 法条 / 法律现象）：v2 在"高频载体"小节按这三类各列代表题
- **防法考化**：v2 不写"主体过多、关系过多"，只保留高考考过的法律关系
- **防必修3化**：v2 意义层只用本模块语言（合法权益、合同自由、契约精神、民事主体平等、市场秩序、消费者保护、知识产权强国），不跳到"全面依法治国"
