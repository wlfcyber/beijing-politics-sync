# 2024 北京一模/二模 选必二《法律与生活》主观题补漏

### 【2024东城一模】Q19
- 案情材料：案件1：郭某2018年为李某制作电表箱，李某出具欠据约定2019年2月1日前还清，郭某从未催要，2024年3月催讨被拒后起诉，因诉讼时效届满败诉。案件2：冯某与母亲程某约定每月支付赡养费，2019年1月起停付，程某2024年起诉追讨5年费用，法院依民法典第196条判决冯某照付。
- 设问：运用《法律与生活》知识，谈谈法律规定诉讼时效并对其不适用的情形加以规定的原因。（单设问）
- 法律领域：民事诉讼时效 + 婚姻家庭赡养
- 答案踩分点：①督促当事人积极主张权利，推动纠纷解决；②节约司法资源；③赡养问题事关老年人基本生存权利、成年子女有赡养义务；④请求支付赡养费不适用诉讼时效保护老年人合法权益；⑤彰显公序良俗、人文关怀，构建和谐家庭。
- 命题路径反推：命题人想培养"权利义务对应+程序与实体并重"的法律思维。选两组对比鲜明的案例（普通债权 vs 赡养费），让考生看到诉讼时效既"督促行权"又"有边界"。设问路径是"为什么设时效—为什么有例外"的双层追问，引导学生跳出"法条背诵"，理解立法目的之间的平衡。踩分点奖励同时讲到"效率价值"（督促、节约）与"公序良俗价值"（孝老、扶弱）的双向论证。

### 【2024朝阳一模】Q19
- 案情材料：镜头一：A公司与B公司签订购销合同，A公司依约供货后B公司590万货款逾期未付，法院判其支付货款及违约金。镜头二：某购物中心为商户建立"码上诚信"二维码，消费者扫码可查商户信用信息，市民因此放心消费。
- 设问：运用《法律与生活》知识，分析诚信原则对促进社会主义市场经济健康发展的积极作用。（单设问）
- 法律领域：民法基本原则（诚信原则） + 合同 + 消费者权益
- 答案踩分点：①社会主义市场经济本质上是法治经济，民法典将诚信原则确立为基本原则；②合同当事人按约定全面履行义务，规范民事主体行为、维护合法权利；③司法机关依诚信原则定分止争、维护公平正义和市场秩序；④经营者诚信经营保护消费者权益，平衡经营者与消费者关系；⑤守法诚信经营者扩大市场，增进社会整体福祉。
- 命题路径反推：命题人在做"价值观入法"的能力培养——把社会主义核心价值观（诚信）通过民法典第七条具象化到合同履行和消费者保护。选两个镜头是为了让学生分别从"司法救济端"（合同违约判决）和"市场预防端"（信用公示）双向理解诚信。设问只问"积极作用"，引导学生从"主体行为—司法适用—市场秩序—社会福祉"四层递进展开，奖励能讲出"诚信不仅约束双方还规范市场整体"的层次感。

### 【2024海淀一模】Q19
- 案情材料：甲公司打造超写实虚拟数字人Ada并发布两段宣传/动捕视频。乙公司未经许可将上述视频用于自己的社交账号，替换片头片尾标识、加注其注册商标，并将其他虚拟数字人名称作为视频标题。甲公司起诉，法院判决乙公司消除影响并赔偿损失。
- 设问：运用《法律与生活》知识，谈谈对本案判决的认识。（单设问）
- 法律领域：知识产权（著作权） + 反不正当竞争
- 答案踩分点：①民事主体的著作权受法律保护，甲公司创作Ada享有著作权；②乙公司未经许可使用相关视频，故意替换标识、加注注册商标，构成著作权侵权；③乙公司虚假和引人误解的商业宣传，属于不正当竞争；④法院依法判赔有利于激发创新活力；⑤维护市场秩序。
- 命题路径反推：命题人借"虚拟数字人首例案"这种新型客体，考查学生能否把"传统著作权"原理迁移到新业态。案例的精巧之处在于乙公司同时触犯著作权和反不正当竞争两条线，引导学生做"侵权类型识别"。设问"对判决的认识"是开放型，要求学生把法律规定、行为定性、判决结果、社会效果一气呵成。奖励学生主动指出"激发创新+维护秩序"的双重价值定位。

### 【东城二模】Q19
- 案情材料：甲公司从乙公司购入电视，约定3月10日发货后由快递仓库代储至3月20日（仓储费由乙承担）。3月19日甲提出延长至3月30日，乙回复"可以"。3月30日甲收货并垫付全部仓储费，要求"尾款扣除仓储费"，乙拒绝承担3月21日及以后费用，甲拒付尾款。双方欲诉讼解决。
- 设问：（1）选择一方代理并撰写起诉状（原告/被告/诉讼请求/事实与理由）；（2）为避免类似问题，我们应该______。
- 法律领域：合同（变更/履行/违约）+ 民事诉讼文书
- 答案踩分点：①诉讼请求2分（判令承担全部仓储费用、承担诉讼费）；②事实陈述要客观+法律术语2分；③理由要有"违背诚信原则/单方变更合同"或"全面履行原则/公平原则"，构成违约应担责；④（2）合同内容应清晰明确（明确双方权利义务、避免模糊意思表示）；⑤合同缺少重要条款时双方应平等协商补充。
- 命题路径反推：这是一道罕见的"程序性法律文书"题——让学生真正动笔写起诉状。命题人想培养"代理人立场+法律术语+事实/法理分离"的实操能力。案例故意设计"乙回复'可以'但未明确费用承担"的歧义，让学生体会合同条款不严密带来的纠纷成本。设问（2）回到"如何防患未然"的合同意识训练。

### 【丰台一模】Q17
- 案情材料：甲某搭乙某车出行，未听乙某提醒系安全带；雨天路滑乙某撞灯杆致甲某受伤住院，乙主动垫付部分医疗费。甲诉请乙承担全部赔偿（医疗费、误工费、护理费等），乙以"好意同乘、并无过错"抗辩。法院综合考量好意同乘情境、事故事实、乙主动担责行为，酌情减轻乙赔偿责任。资料卡引民法典第1217条好意同乘条款。
- 设问：运用《法律与生活》知识，谈谈法院酌情减轻被告赔偿责任的法理依据和现实意义。（单设问）
- 法律领域：侵权责任（机动车交通事故 + 好意同乘）
- 答案踩分点：①法院判决以事实为根据、以法律为准绳；②驾驶人与搭乘人形成好意同乘关系，驾驶人对搭乘人生命财产安全负有保障义务，乙发生事故侵犯甲的生命权、健康权应担责；③乙存在过错但非故意或重大过失，甲诉请全部赔偿有违公平原则；④依民法典和好意同乘规定酌情减轻赔偿；⑤维护双方合法权益、促进社会公平、维持人际关系和谐、倡导绿色出行。
- 命题路径反推：命题人以"好意同乘"这一典型情谊行为为切口，考查"无过错责任 vs 过错责任 vs 公平原则"的精细切分。资料卡直接给出民法典第1217条，意在让学生学会"案情—法条—判决"三对一映射。设问把"法理依据"与"现实意义"分开问。奖励能用"故意或重大过失"这一专业表达精准锁定免责例外条款的学生。

### 【丰台二模】Q17
- 案情材料：民法典确立绿色原则（民事活动应有利于节约资源、保护生态环境）、保护知识产权、规定环境侵权责任。案例一：某公司发明专利从源头解决污水污泥分离不彻底等关键技术问题。案例二：甲公司长期超标排放废水废气损害环境公共利益，乙环境研究所提起环境民事公益诉讼，法院判甲立即停止侵权、消除危险、赔偿环境损失。
- 设问：运用《法律与生活》知识，结合上述案例分析民法典的相关规定对保护生态环境的作用。（单设问）
- 法律领域：民法基本原则（绿色原则）+ 知识产权 + 环境侵权 + 公益诉讼
- 答案踩分点：①民法典绿色原则为生态环境保护提供基本遵循；②保护知识产权鼓励技术创新，为生态保护提供技术支撑；③污染、破坏环境应承担侵权责任；④合理确定民事主体权利义务，既保护合法权益又惩罚违法行为；⑤平衡个人利益和公共利益、合理预防损害、维护良好生态环境。
- 命题路径反推：命题人用"专利 + 公益诉讼"两个案例，串联起民法典三类条款（绿色原则—知识产权—环境侵权），考查学生看到"民法典体系性保护生态"的能力。案例一是"激励正向创新"，案例二是"惩罚负向破坏"，正反对照让学生理解民法典是"奖罚并举"的制度组合。

### 【海淀二模】Q19
- 案情材料：某社区居委会与曹某签订协议，约定居委会照看关心曹某、给予基本生活费、免费看病、节日慰问、养老至寿终；曹某动产和不动产寿终后产权移交居委会。居委会按约履行扶养义务。曹某去世后四个子女要求继承遗产与居委会产生争议。法院认定属有效遗赠扶养协议，判决曹某遗产归居委会所有。
- 设问：运用《法律与生活》知识，分析本案判决的社会价值。（单设问）
- 法律领域：继承（遗赠扶养协议）+ 婚姻家庭（赡养）
- 答案踩分点：①法院确认居委会与曹某签订的遗赠扶养协议有效；②认定居委会赡养行为，肯定居委会养老送终所起作用；③不仅实现个案公平正义，更倡导全社会积极助力养老；④促进"不尽孝者少分或不分遗产"司法理念深入人心；⑤弘扬尊老敬老爱老助老传统美德。
- 命题路径反推：命题人选了一个非常温暖但又锋利的案例——"扶养老人的居委会战胜了不尽孝的子女"。这考查学生对"遗赠扶养协议"这一非典型继承方式的辨识，更考查"法律如何引导道德"的深层理解。

### 【石景山一模】Q17
- 案情材料：北京市首例碳排放权交易纠纷案。2021年12月，乙公司就采购碳排放配额发布比选公告，甲公司向其报价（内容具体明确，承诺未履约则乙可另购同等配额、差价由甲补足），乙比选后确认甲中标并送达中标通知。其后甲明确不再履行，乙另与第三方以更高价签约购入，差价289万余元。乙诉请甲支付差价款及利息。一审、终审均判甲支付。
- 设问：运用法律知识，结合案情，说明甲乙两公司合同成立的理由，并从民法基本原则角度阐述该司法判决的意义。（双设问，合并问）
- 法律领域：合同（要约/承诺/合同成立）+ 民法基本原则（诚信、公平、绿色）
- 答案踩分点：①甲公司向乙公司报价、内容具体确定，属于要约；②乙公司确认中标、送达中标通知，属于承诺；③中标通知书送达甲公司，双方合同成立；④判决引导碳排放配额交易的民事主体遵循诚信、公平、绿色原则；⑤推动企业转型升级、兼顾经济效益和社会效益；⑥维护我国碳排放权交易市场正常秩序、推动要素市场化改革、落实"双碳"国家重大决策。
- 命题路径反推：命题人选"首例碳排放权交易纠纷"，把"双碳"国家战略与民法合同制度焊在一起。考查"要约—承诺—合同成立"的标准三段判定，又延伸到"民法基本原则如何为新型市场赋能"的高阶思考。

### 【石景山一模】Q18(2)
- 案情材料：石景山法院推出"一米法庭"项目化解基层纠纷。2023年7月特大暴雨后，业主李先生家因房屋漏雨与物业公司起争议——物业未在气象预警后加强巡检、加固房顶、做好防水；物业以"天灾免责"抗辩；调解员与业主调解达成的协议被业主认为不算数，业主准备直接起诉。社区调解员老刘通过"一米法庭"求助法官指导。
- 设问：假如你是法官，请结合材料，运用法治知识，为老刘提供专业指导。（单设问）
- 法律领域：合同（物业服务合同+违约责任）+ 不可抗力 + 调解协议司法确认
- 答案踩分点：①"天灾"不等于不可抗力，物业"天灾免责"说法不成立；②不可抗力指不能预见、不能避免且不能克服的现象；③暴雨前已有气象预警，物业理应预见并采取预防措施，不能免责；④业主与物业存在物业服务合同关系，物业未全面履行服务义务构成违约，应担违约责任；⑤诉前可优先非诉解决；⑥经人民调解委员会调解达成的协议，双方可申请司法确认，确认有效后具强制执行效力。
- 命题路径反推：命题人构造了一个让学生"扮演法官"的真实场景——既要"回怼"物业的错误抗辩（不可抗力的法定要件），又要"安抚"业主的过度反应（调解协议同样有效力）。考查多元纠纷解决机制下"诉与非诉、调解与司法确认"的关系把握。

### 【西城一模】Q18
- 案情材料：郭某与刘某夫妻，刘某2022年去世，育有四子女，郭某年已九十由小女儿照顾。郭某每月有退休金五千余元。郭某起诉另外三子支付赡养费：大儿子在外地，二儿子肢体残疾，三儿子认为郭某收入足够无需子女付费。一审认为郭某不存在生活困难驳回。郭某上诉。二审期间三子前往探望。二审虽不支持赡养费但向三子发出《督促履行义务告知书》要求常回家看看、多沟通交流。某班据此组织模拟庭审。
- 设问：（1）镜头一上诉人申请书记员小袁回避的适当理由（1分）；（2）镜头二中"我是被上诉人郭某三儿子的辩护人"有一处明显错误，找出并改正（1分）；（3）运用法治知识，谈谈对二审法院向三子发出《督促履行义务告知书》的认识（6分）。
- 法律领域：婚姻家庭（赡养义务/精神赡养）+ 民事诉讼（回避制度、诉讼代理人）
- 答案踩分点：①回避理由（任选一种）：书记员小袁与本案当事人有近亲属或利害关系或其他可能影响公正审理的关系；②错误："辩护人"应改为"诉讼代理人"（民事案件用代理人，刑事案件才用辩护人）；③成年子女对父母有赡养义务，包括经济供养、生活照料、精神慰藉、照顾特殊需要；④三个儿子都是成年人应履行精神慰藉义务；⑤《督促履行义务告知书》保护郭某合法权益、引导孝老敬亲、传承中华传统美德；⑥发挥法律对道德建设的促进作用。
- 命题路径反推：命题人精心设计了"模拟庭审"的真实情境，把"回避制度""刑民称谓区别""精神赡养""法律对道德的引导作用"四个易错点串成一道题。第（1）（2）问是程序细节考查，专门挑学生最容易混淆的"辩护人/代理人"和"回避适用情形"。第（3）问聚焦"虽不判赔偿但发督促书"这一柔性司法形式。

### 【西城二模】Q16
- 案情材料：食品安全法第148条第2款规定生产或经营不符合食品安全标准食品，消费者除赔偿损失外可主张价款十倍或损失三倍的惩罚性赔偿金。市民甲首单购买"黄芪薏米饼干"30盒后，确认饼干不符合食品安全标准，但在两个多月内多次向同一商家加购同款饼干共200盒、18.4千克。法院认为：首单30盒未超合理消费需要应支持惩罚性赔偿；加购行为综合考虑保质期、消费习惯等已超正常生活消费需要，加购部分的惩罚性赔偿不予支持。
- 设问：运用《法律与生活》知识，分析人民法院判决的意义。（单设问）
- 法律领域：消费者权益 + 民法基本原则（禁止权利滥用）
- 答案踩分点：①经营者应依法、诚信经营，保护消费者合法权益；②支持首单惩罚性赔偿打击违法生产经营、维护消费者合法权益；③民事主体行使民事权利不能超过正当界限，不得滥用民事权利损害他人合法权益；④加购行为超出正常生活消费需要，不予支持有利于引导消费者诚信、理性维权；⑤体现保证食品安全和维护生产经营秩序两种价值取向的平衡。
- 命题路径反推：命题人选了一个"职业打假式加购"的边缘案例，考查学生能否看到惩罚性赔偿制度的"双向边界"——一方面打击违法经营者，另一方面防止消费者权利滥用。这是对"权利—义务—边界"三重思考的高阶训练。

### 【顺义思政二模】Q17
- 案情材料：某校学习小组围绕"财产制度助力经济社会发展"开展议题探究，搜集整理财产权保护、产权制度、各类所有制经济财产权依法保护等资料。
- 设问：结合材料，运用所学知识，说明我国财产制度改革如何助力经济社会发展。（单设问，跨模块综合）
- 法律领域：物权（平等保护）+ 财产权 + 知识产权
- 答案踩分点：①财产制度规范民事主体的财产关系，坚持物权平等保护原则；②确定财产归属、增强人民群众财产财富安全感，稳定市场主体投资交易预期；③建立财产权保护长效机制，保护财产权就是保护劳动、保护发明创造、保护和发展生产力；④贯彻创新驱动发展战略，增强各类经济主体创新创业动力；⑤完善产权制度，激活社会主义市场经济内生动力。
- 命题路径反推：命题人借"议题式探究"把选必二的财产权制度（物权平等保护、知识产权保护）和经济与社会的产权制度改革焊接起来，考查跨模块"看到一个法律制度的经济社会效用"的整合能力。
