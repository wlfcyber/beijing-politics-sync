# 北京 2025 届《法律与生活》试题切片结构化摘要

> 范围：2025 届各区一模 / 二模 / 期中 / 期末，仅 _法律与生活_ 板块。
> 用途：建框架、情境库、备考底料。
> 说明：少数题目本身命题载体跨学科（如 "破立"、"网络暴力治理"），只摘其法律生活面。
> 已剔除：纯哲学/文化/逻辑/国际政治题，以及不含法律生活考点的非法律题。

---

## 一、主观题（按套卷顺序）

### 1. 丰台一模 · Q19（8 分） · 知识产权侵权裁判意义

- **案情**：知识产权侵权典型案例，法院判决严惩侵权、保护创新。
- **设问**：分析本案判决的现实价值（意义类）。
- **法律领域**：知识产权 / 侵权责任 / 司法保护。
- **答案踩分点**：① 维护当事人合法权益、严惩侵权 → 保护知识产权；② 规范市场竞争秩序，营造公平市场环境，以法治力量鼓励创新；③ 作为典型案例提供范例 → 提升审判质量、增强司法公信力；④ 引导公民依法办事，增强法治信仰，推动法治中国建设。
- **命题路径反推**：命题人想培养"以司法案例反推法治价值"的能力——从一个典型判决中读出对当事人、对市场、对司法、对社会四层意义。选知产案例是因为它能同时承载 "保护创新（产业政策）"+"市场公平（经济法）"+"法治信仰（社会层面）" 三条线。设问采用最典型的"判决意义"路径：让学生分四层（个体→市场→司法→社会）展开。细则奖励主体明确的递进式表述，惩罚单点重复。

### 2. 丰台期末 · Q19 · 劳动合同纠纷裁判理由

- **套卷题号**：丰台期末 Q19。
- **案情**：尹某入职购物中心后违反规章制度、消极怠工、无故旷工；公司多次提醒未果遂解除劳动合同；尹某起诉要求继续履行；法院驳回。
- **设问**：运用《法律与生活》知识，阐释法院的裁判理由（原因类）。
- **法律领域**：劳动法 / 劳动合同法 / 劳动者权利义务 / 劳动争议解决。
- **答案踩分点**：① 劳动者应完成劳动任务、遵守劳动纪律和职业道德（《劳动法》《劳动合同法》依据）；② 尹某未履行义务，违反法律规定，不符合敬业等社会主义核心价值观；③ 公司解除合同符合《劳动合同法》第 39 条 "严重违反规章制度" 情形；④ 法院驳回诉请，保护用人单位合理用工自主权；⑤ 引导劳动者坚持权利义务相统一，构建合理健康的劳动关系。
- **命题路径反推**：命题人想培养"用法条 + 价值观双重标尺给法院判决归因"的能力。选劳动合同纠纷是因它最容易让学生情感倾向劳动者，但本题反而要求学生站在法院视角支持企业——训练去情绪化、回到法条本身的素养。设问"裁判理由"是最锁定路径的设问形式，学生必须先定位法条再扣事实再升华价值。细则奖励 "法条—事实—价值" 三步走，特别是对 "用工自主权" 和 "权利义务相统一" 的精确表述。

### 3. 房山一模 · Q10 · AI 生成图片著作权

- **案情**：李某用 AI 软件输入提示词生成图片，发布到社交平台；刘某未经许可将该图作为原创诗歌配图、并截去署名水印。
- **设问**：判断李某享有何种权利、刘某的行为定性。
- **法律领域**：著作权 / 信息网络传播权 / 署名权 / 侵权责任。
- **答案踩分点**：① AI 生成图片若有 "智力成果 + 独创性" 要件，可受著作权法保护；② 李某通过提示词生成并发表 → 享有著作权；③ 刘某未经许可使用、抹去水印 → 侵犯信息网络传播权与署名权，承担侵权责任；④ 不构成违约（双方无合同关系，是侵权而非违约）。
- **命题路径反推**：命题人抓住 AI 创作热点，要培养学生 "新业态—旧法理" 嫁接能力。AI 图片是 2024 年北京互联网法院首例落槌案的现实素材，命题人选它一举解决 "时事性 + 新业态 + 著作权细分权能" 三重需求。设问让学生在 "违约 vs 侵权" 二选一的边界上做切割，逼出对著作权各项权能（署名权、信息网络传播权）的精确识别。细则奖励 "权能拆分" 和 "侵权而非违约" 的辨析。

### 4. 房山一模 · Q19（约 9 分） · 知识产权惩罚性赔偿

- **案情**：A 集团掌握技术秘密；B 集团无技术积累却短期内推出同系列电动汽车，涉嫌侵害技术秘密；法院认定侵权并适用 2 倍惩罚性赔偿，判赔 6.4 亿余元；并提及我国知识产权侵权诉讼判赔数额创历史新高。
- **设问**：运用 "依法治国、尊重知识产权、公平竞争" 知识，从意义角度说明 "真创新受到真保护" 的价值（说明类）。
- **法律领域**：知识产权 / 商业秘密 / 反不正当竞争 / 惩罚性赔偿 / 严格执法公正司法。
- **答案踩分点**：① 激励创新积极性——高额赔偿与惩罚性赔偿保障创新者合法权益与合理回报，激发社会创新动力；② 维护市场公平竞争——严惩以不正当手段获取技术秘密的行为，保障市场规则公平；③ 彰显法治权威——严格执法和公正司法增强法律威慑力，提升公众对法治的信任。
- **命题路径反推**：命题人借大额惩罚性赔偿这个"司法重锤"载体，让学生把 "依法治国—知识产权—公平竞争" 三块教材知识打通。选 6.4 亿这种巨额数字是为了让学生直观感受"惩罚性"力度，不是单纯赔偿。设问"意义角度"导向三层：创新者—市场—法治，命题人希望学生不再写 "维护合法权益" 这类老套表达，而能上升到 "新质生产力 / 法治社会建设"。细则奖励对 "惩罚性 ≠ 补偿性" 的精确区分。

### 5. 昌平二模 · Q15 · 商事仲裁 vs 劳动仲裁的边界

- **案情**：案例 1：贸易公司与制造企业签设备采购合同（约定仲裁解决），出现质量问题；案例 2：张某与公司签劳动合同，公司单方面降薪、不付加班费，张某申请劳动仲裁。
- **设问**：对两案的认识正确的是（综合判断）。
- **法律领域**：商事仲裁 / 劳动仲裁 / 仲裁与诉讼衔接。
- **答案踩分点**：① 平等主体之间的合同纠纷可提交商事仲裁；② 商事仲裁实行 "或裁或审"，劳动仲裁实行 "仲裁前置"——二者不可混淆；③ 劳动争议仲裁不予受理时，劳动者可向法院起诉；④ 商事仲裁经协议约定后排除诉讼，劳动仲裁是诉讼的法定前置程序。
- **命题路径反推**：命题人特意把 "商事仲裁" 和 "劳动仲裁" 并排放，逼出学生对两套仲裁体系差异的精细记忆。选这两个对照案例是因为它们在教材里都讲、但学生最容易混淆的就是 "前置 / 不前置"、"协议 / 法定" 这两组对立。设问表面问 "认识"，实质考四条边界。细则奖励 "或裁或审 vs 仲裁前置" 的清晰对比。

### 6. 昌平二模 · Q20（6 分） · AI 著作权 + 信用卡违约

- **案情**：① AI 生成图片的著作权归属与侵权——李某用人工智能软件制作图片，刘某未经许可作配图使用并抹去水印；② 信用卡违约——小王签订《信用卡领用相关手续》，透支后经催缴仍不还款。
- **设问**：分析两个案例中的法律关系与责任承担（综合分析类）。
- **法律领域**：著作权 / 信息网络传播权 / 署名权 + 合同 / 违约责任 / 诚信原则 / 全面履行原则。
- **答案踩分点**：① AI 不能成为著作权主体，著作权主体限于自然人、法人或非法人组织；② 李某的图具备 "智力成果 + 独创性" → 受保护；③ 刘某抹去水印作配图 → 侵害信息网络传播权和署名权，承担侵权责任；④ 信用卡领用手续是双方真实意思表示，合法有效；⑤ 小王透支后不还款 → 违反诚信原则和全面履行原则，承担违约责任。
- **命题路径反推**：命题人把 "新型创作侵权" 和 "传统金融违约" 拼到一题里，是想训练学生在不同民事法律关系（侵权 vs 合同）之间做切换。选 AI 图片 + 信用卡是因为前者是 2024 年热点（一定要考新），后者是日常生活（每个家庭都有）。设问 "分析法律关系" 要求学生先定性（侵权 / 违约）再扣条文。细则奖励对 "AI 不是著作权主体" 这个反常识结论的精准把握，以及 "诚信 + 全面履行" 的合同基本原则表述。

### 7. 朝阳一模 · Q19（8 分） · 平台捆绑销售欺诈

- **案情**：A 公司（在线文旅平台）未在购买界面设置醒目提示语、也未提供勾选选项，将"外卖红包"捆绑销售给王某；王某不清楚自己付款金额构成；起诉至法院。
- **设问**：分析本案中平台的行为定性、王某可获得的赔偿、判决的现实意义。
- **法律领域**：消费者权益保护 / 平台经营者义务 / 欺诈认定 / 惩罚性赔偿（三倍）。
- **答案踩分点**：① 平台未醒目提示、未提供选项、隐瞒真实情况 → 构成欺诈；② 依据《消费者权益保护法》第 55 条，按王某要求赔偿损失，并按支付票款三倍增加赔偿；③ 支持王某诉请，维护消费者合法权益；④ 规范在线文旅企业行为，优化网络消费环境，促进数字经济持续健康发展。
- **命题路径反推**：命题人用 "捆绑销售 / 默认勾选" 这一新经济顽疾，让学生把消保法第 55 条 "三倍赔偿" 与现实场景对应起来。选 "在线文旅平台" 是因为这是高考生及家庭都有体感的消费场景，且 2024 年消费维权舆情中默认勾选是热点。设问采用 "定性—责任—意义" 三段式，最后一段引导学生看见司法判决对市场秩序的反向规训作用。细则奖励对 "三倍" 这个具体数字的精确记忆和对 "故意隐瞒真实情况 = 欺诈" 的法理推论。

### 8. 朝阳期末 · Q20（10 分） · 侵权责任综合论述（开放短文）

- **案情**：切片只保留了评分量规——要求运用 "侵权责任、过错责任" 等知识，从 "法律依据、法律事实、法律责任" 三个层面作答。
- **设问**：综合论述类（短文 / 论述）。
- **法律领域**：侵权责任 / 过错责任 / 法律推理三段式（依据—事实—责任）。
- **答案踩分点**（按等级量规反推）：水平 4 要 "观点鲜明、紧扣问题、知识准确、全面展开、逻辑严密、条理清晰"；学生需在论述中显式呈现 "法律依据条款 → 案件事实归纳 → 法律责任承担" 三段，避免罗列。
- **命题路径反推**：这是 2025 届本批卷子里少见的开放性论述题——命题人用评分量规而非答题点，引导学生进行 "法律人式思考" 训练，即模仿判决书的 "依据—事实—责任" 结构。选侵权领域是因为它最容易切分三段。细则奖励的是 "结构化表达"，而非堆砌知识点。备考时应训练学生先列三段标题再填内容。

### 9. 海淀期末 · Q20（6 分） · 民法典侵权责任编司法解释适用

- **案情**：最高法《关于适用 < 民法典 > 侵权责任编的解释（一）》部分条款：① 第 7 条——未成年子女致害父母共同担责；② 第 19 条——产品缺陷致买受人财产损害（含缺陷产品本身损害）由生产者 / 销售者赔偿；③ 第 23 条——禁止饲养的烈性犬致害，饲养人 / 管理人不得主张减免责。
- **设问**：参照示例（监护人责任），完成 "产品责任" 与 "饲养烈性犬责任" 的分析（填表）。
- **法律领域**：民法典 / 侵权责任 / 监护人责任 / 产品责任 / 饲养动物致害 / 无过错责任。
- **答案踩分点**：① 生产者产品责任是无过错侵权责任——不论有无过错，损害他人民事权益即应担责；解释明确缺陷产品自身损害纳入产品责任赔偿范围；② 禁止饲养的烈性犬等危险动物致害，饲养人 / 管理人承担无过错责任，无任何免责 / 减责事由；③ 体现 "最严格的无过错责任" 立法精神，保障人民群众生命财产安全。
- **命题路径反推**：命题人借最高法司法解释新规切入，让学生在 "示例" 框架下做迁移练习——示例已搭好脚手架（条文功能 → 立法目的 → 社会效果），学生只需把另两条填进去。选 "产品责任 + 饲养动物" 是因为它们是无过错责任最典型的两类，且日常常见。细则奖励对 "无过错责任 / 过错责任" 二分法的清醒，以及对 "最严格" 这种立法表态的复述。

### 10. 海淀期末 · Q21（8 分） · 竞业限制双案对照

- **案情**：案例 1：李某是 A 公司高级管理人员，签保密 + 竞业限制协议；离职次日入职同业 B 公司任同岗位；A 公司诉讼，法院支持竞业限制并判赔；案例 2：刘某是餐饮公司冷菜厨师，公司认为其掌握菜品秘方，签竞业限制；刘某离职后到其他餐馆做冷菜，餐饮公司起诉，法院认定协议无效。
- **设问**：说明法律规定 "竞业限制" 并对其加以限制的原因。
- **法律领域**：劳动合同法 / 竞业限制 / 商业秘密 / 用人单位经济补偿义务 / 自由择业权。
- **答案踩分点**：① 竞业限制是用人单位通过协议对劳动者自由择业的合理限制 → 保护知识产权、维护市场公平竞争、推动社会创新；② 李某作为负有保密义务的劳动者，泄露商业秘密违反协议 → 担违约责任；③ 若不对竞业限制加以限制，会侵犯劳动者基本劳动权利、阻碍人才合理流动；④ 刘某是普通员工，非负有保密义务的人员，餐饮公司与其签的竞业限制协议违反法律规定，无效；⑤ 法律既"规定"又"限制"，旨在依法平衡劳动者权益与企业商业秘密保护，营造公平公正市场环境。
- **命题路径反推**：命题人选两个反向案例（一支持一无效）做对照，让学生看见同一法律工具在不同主体（高管 vs 普工）上的差异化适用——这是 "法律平衡多元利益" 最经典的训练场景。设问 "为什么规定 + 为什么限制" 是双向论述结构，逼学生同时论证 "立" 与 "破"，避免单向偏袒。细则奖励 "主体限定（高级管理人员、高级技术人员、其他负有保密义务人员）" 这个法条细节。

### 11. 石景山一模 · Q20（8 分） · 软件研发违约 + 不正当竞争

- **案情**：被告软件公司开发时间超过履行期限且最终未提交符合合同约定功能的技术成果；并存在不正当竞争情形。
- **设问**：分析法院判决的法律依据与判决意义。
- **法律领域**：合同 / 违约责任 / 守约方救济 / 反不正当竞争 / 新质生产力护航。
- **答案踩分点**：① 当事人不履行合同义务或履行不符合约定，守约方可要求违约方承担违约责任；② 被告超期且最终未交付符合约定功能的技术成果 → 构成违约；③ 应承担返还研发费用、支付违约金等责任；④ 依法打击不正当竞争，促进公平竞争，维护市场秩序，营造创新环境，护航软件产业高质量发展，培育新质生产力。
- **命题路径反推**：命题人用软件研发场景同时考违约与不正当竞争，是因为这两者在数字经济中常常并发。选 "软件 / 技术成果" 载体能把"新质生产力"自然嵌进答案。设问要求学生在 "违约 + 反不正当竞争" 两条法律线上分别落点，然后向上收束到产业政策意义。细则奖励 "返还研发费用 + 支付违约金" 这种具体救济措施的列举，以及 "新质生产力" 的时政语汇。

### 12. 石景山一模 · Q21（9 分） · "破立" 主题大综合（含违约、不正当竞争）

- **案情**：以 "破与立" 为主题——案例一为研发合同违约（保护守约方、激励契约精神）；案例二为软件名称、图标相似构成不正当竞争。整题串联 "新中国成立—改革开放—新时代" 三阶段的破立逻辑，并落到当前 "立新发展格局、立新质生产力、立公正高效权威的司法制度" 等具体方向。
- **设问**：阐述党在进一步全面深化改革中统筹破立关系，对全面建成社会主义现代化强国的重要意义（综合论述）。
- **法律领域**（仅就法律生活面）：违约责任 / 反不正当竞争 / 经营者义务 / 司法对市场秩序的塑造。
- **答案踩分点**（法律生活面）：① 违约方未交付合规技术成果 → 担违约责任；② 软件名称图标相似易致消费者混淆 → 构成不正当竞争，担停止侵害、赔偿损失等侵权责任；③ 司法判决保护契约精神、激励创新、营造公平市场。
- **命题路径反推**：这是一道跨模块综合题，但法律生活在其中承担 "立公正高效权威的司法制度" 的实证支撑——命题人用两个判决证明 "司法制度的立" 是真实可触的。选这两个案例意在显示 "破：破除违约和侵权" 与 "立：立信用、立创新" 的辩证。细则奖励学生能在大主题下用 "微观法律案例" 印证 "宏观制度建设"，体现微观—中观—宏观三层视野。

### 13. 西城一模 · Q19（8 分） · "唤醒词" 反不正当竞争

- **案情**：M 公司 2017 年推出 "小爱同学" 智能语音设备，截至 2020 年 12 月月活 8670 万、累计唤醒约 495 亿次；陈某 2017—2020 年间在不同商品类别上抢先大量申请注册 "小爱同学" 商标，并发函要求 M 公司停止使用，同时与深圳 Y 公司合作在运动手表、闹钟等商品上使用 "小爱同学"，发布相关宣传文章；M 公司诉之；法院判陈某、Y 公司停止侵权，陈某赔偿 120 万元，Y 公司对其中 25 万承担连带责任。
- **设问**：" 唤醒词" 亦受法律保护，请运用法律知识分析法院的判决。
- **法律领域**：反不正当竞争 / 商标抢注 / 混淆与虚假宣传 / 诚实信用原则 / 数字经济法治化营商环境。
- **答案踩分点**：① 陈某明知他人已使用 "小爱同学" 唤醒词且有一定影响，仍抢先注册 → 违背诚实信用原则；② 发布引人误解的商业宣传信息 → 构成混淆及虚假宣传，属不正当竞争；③ 法院判决合理确定各方利益，规制了滥用权利的行为；④ 保护了科技创新型企业的品牌商誉；⑤ 有利于促进新业态发展，为数字经济营造良好的法治化营商环境。
- **命题路径反推**：命题人选 "唤醒词" 这种典型新型客体，让学生看到反不正当竞争法对教材未列举类型的兜底功能。设问让学生在 "新客体—老规则" 之间做适配——本题没有现成法条对应 "唤醒词"，必须从 "有一定影响" 这个反不正当竞争法表述中找到落点。细则奖励对 "诚实信用原则 + 滥用权利 + 法治化营商环境" 这条价值链条的连贯论述。

### 14. 门头沟一模 · Q20 · 司法 "以事实为根据，以法律为准绳"（填表）

- **案情**：① 案例一——A 享有 "我不是胖虎" 系列美术作品的复制权、改编权、信息网络传播权；B、C 公司未经许可在微博账号发布 "橘猫" 类似图，A 起诉；② 案例二——孙某入职甲公司，简历写明 7 年 4 段工作经历，实际曾在 17 家公司任职；甲公司以未如实告知为由试用期辞退；孙某起诉要求违法解除赔偿金。
- **设问**：参照案例一格式，完成案例二的 "裁判结果 / 裁判理由 / 现实意义" 填空。
- **法律领域**：① 著作权（复制权、改编权、信息网络传播权）/ 侵权责任；② 劳动合同 / 入职诚信义务 / 用人单位单方解除权 / 试用期解除条件。
- **答案踩分点**（推理）：① 裁判结果：驳回孙某诉求，支持公司试用期解除；② 裁判理由：劳动者应如实说明与劳动合同直接相关的基本情况，孙某隐瞒真实工作经历、违反诚信原则；公司有权依据《劳动合同法》第 39 条 "试用期间不符合录用条件" 解除合同；③ 现实意义：维护用人单位合理用工自主权，引导劳动者诚信履约，构建合理健康的劳动关系。
- **命题路径反推**：命题人巧妙用 "脚手架题型"——案例一已经搭好答题模板，学生只需把案例二填进去。选 "简历造假" 是因为这个场景对高考生（即将入职场）有强提醒意义，且涉及试用期这一实务热点。设问 "结果—理由—意义" 三段式与丰台期末 Q19、朝阳期末 Q20 完全同构，是 2025 届本批卷子的高频结构。细则奖励 "诚信原则 + 试用期不符合录用条件" 这两个具体抓手。

## 二、选择题（按套卷顺序）

### 1. 丰台一模 · Q10 · 平台骑手公益诉讼

- **题干**：检察院发现某外卖平台长期忽视骑手工伤保险缴纳义务（37 名骑手索赔无门），提起公益诉讼，要求补缴社保并公开道歉。
- **选项核心法律点**：
  - A：公司应缴社保 + 平等自愿订立合同（"平等自愿" 在此处用错——劳动合同遵循平等自愿协商一致，但社保是法定义务，非协商）。
  - B：人民检察院通过公益诉讼，可纠正损害公共利益的行为（正确表述）。
  - C：人民法院在 "执法" 过程中要公开透明（错——法院是 "审判"，不是 "执法"）。
  - D：公益诉讼是劳动争议的 "必经程序"（错——劳动争议的必经程序是劳动仲裁，不是公益诉讼）。
- **法律细节边界**：检察公益诉讼的功能定位（公共利益）vs 劳动争议处理路径（仲裁前置）；法院 "审判" vs 行政机关 "执法" 用语区分。

### 2. 丰台二模 · Q10 · 调解优势

- **题干**：金融法院审理乙公司诉甲公司还款案，甲表达调解意愿；法官顾及裁判会导致甲在谈商业项目搁置、员工失业，决定调解；经 10 轮磋商，约定两年半还清。
- **选项核心法律点**：
  - ①：诉讼是最有效的解决方式（错——本案恰恰说明调解更优）。
  - ②：依 "谁主张谁举证"，甲公司应举证其未恶意违约（错——主张方是乙，举证责任在乙）。
  - ③：成功调解 → 各方共赢，激发市场主体活力（正确）。
  - ④：诉讼调解化解金融纠纷，传递司法温度（正确）。
- **法律细节边界**：举证责任分配 "谁主张谁举证" 的方向；调解 vs 诉讼的功能取舍；司法 "刚性裁判 vs 柔性调解" 的成本权衡。

### 3. 丰台二模 · Q11 · 农村土地经营权用益物权

- **题干**：农村集体经济组织将 100 亩集体所有土地经营权流转给 M 公司，期 20 年，用途耕地；一年后政府因修路征收部分土地。
- **选项核心法律点**：
  - ①：M 公司可在土地上修建民宿（错——耕地不得改变用途）。
  - ②：M 公司取得经营权属于用益物权（正确——土地经营权是法定用益物权）。
  - ③：农村集体经济组织具有法人资格（正确——《民法典》总则编规定）。
  - ④：农村土地被征收有权获得合理赔偿（错——本题语境下被征收主体应是农村集体）。
- **法律细节边界**：用益物权类型（土地承包经营权、土地经营权、宅基地使用权、建设用地使用权）；农村集体经济组织法人地位；耕地用途管制。

### 4. 丰台期末 · Q11 · 肖像权（互联网司法保护）

- **题干**：李某未经同意公布他人照片，侵犯肖像权；法院判决体现互联网司法对人格权的保护。
- **选项核心法律点**：
  - ①：李某的发布属于行使权利且属正常行为（错——未经同意公布即侵权）。
  - ②：判决属于"对未成年人肖像权的特殊保护"（错——本案是一般保护，非特殊保护）。
  - ③：权利义务统一，李某发布信息时也需守宪法和法律（正确）。
  - ④：判决彰显互联网司法重视人格权保护，对网络空间法治化治理具示范意义（正确）。
- **法律细节边界**：肖像权的一般保护 vs 未成年人特殊保护；权利义务统一原则在网络场景下的具体适用。

### 5. 丰台期末 · Q12 · 旅游合同 + 安全保障义务

- **题干**：王某购票进景区游玩，在湿滑观景台摔倒受伤；景区是否担责。
- **选项核心法律点**：
  - ①：王某购票进景区，旅游合同依法成立（要约 + 承诺，承诺到达即生效）—— 正确。
  - ②：王某承担违约责任（错——王某无违约行为）。
  - ③：王某自行承担全部责任（错——景区也有安全保障义务）。
  - ④：景区未尽安全保障义务 → 承担侵权责任（《民法典》第 1198 条）—— 正确。
- **法律细节边界**：要约 / 承诺规则（旅游合同成立时点）；公共场所经营者的安全保障义务；游客自身注意义务的减责作用（不是免责）。

### 6. 延庆一模 · Q10 · "搭便车" 不正当竞争

- **选项核心法律点**：
  - A：丙构成不正当竞争，担违约责任（错——是侵权 / 不正当竞争责任，不是违约）。
  - B：本案事实清楚，原告无需举证（错——原告仍需举证证明权利和侵权事实）。
  - C：丙的企业名称经工商核准 → 法院应不予受理（错——核准不构成抗辩，法院应受理审理）。
  - D：法院应以司法裁判树立法律边界，护航新质生产力发展（正确）。
- **法律细节边界**：不正当竞争 vs 违约的区分；商标权和企业名称权冲突；行政核准与民事侵权认定的关系。

### 7. 延庆一模 · Q11 · 新业态劳动关系认定

- **题干**：小何入职某网络科技公司从事外卖配送，签 "承揽配送协议"，未签书面劳动合同；公司配统一服装、记考勤；小何取餐途中摔伤。
- **选项核心法律点**：
  - A：基于意思自治签承揽协议 → 承揽合同关系（错——形式不能掩盖实质，存在劳动管理则为劳动关系）。
  - B：公司对小何有劳动管理 → 形成事实劳动关系（正确）。
  - C：未签书面合同就无法确定权利义务（错——可凭事实劳动关系主张）。
  - D：新就业形态突破劳动关系范畴，应由 "平台经济法" 调整（错——该法不存在，新就业形态仍受劳动法及配套规则规制）。
- **法律细节边界**：事实劳动关系的认定要素（人格 / 经济 / 组织从属性）；新就业形态劳动者权益保障的法律渊源。

### 9. 房山一模 · Q11 · 相邻权与房屋改造合法边界

- **选项核心法律点**：
  - ①：高先生改造未违反自愿原则（错——自愿原则前提是不违法，违反强制性规定即违法）。
  - ②：行使权利要注意界限（不得违反法律和公序良俗）—— 正确。
  - ③：调解是解决纠纷的优先选择（错——题干没体现调解）。
  - ④：邻居处理相邻权纠纷应秉持友善态度（正确）。
- **法律细节边界**：业主行使所有权的合法边界；相邻关系的处理原则（友善 / 公平 / 不违反强制性规定）；自愿原则的适用前提。

### 10. 朝阳一模 · Q11 · 行政公益诉讼

- **题干**：区检察院发现古桥（市级保护文物）有坍塌风险；建议区文旅局整改未果；提起行政公益诉讼。
- **选项核心法律点**：
  - A：检察院 "无偿提供代理法律服务"（错——检察院是公益诉讼起诉主体，非代理人）。
  - B：本案适用 "审判监督程序"，一经判决即生效（错——审判监督程序是再审程序，本案是一审）。
  - C：区文旅局在行政公益诉讼中始终处于被告地位（说法过于绝对，仅在行政机关不依法履职时才作被告）。
  - D：检察院当诉则诉，体现以检察职能维护社会公共利益的担当（正确）。
- **法律细节边界**：行政公益诉讼的提起前置程序（检察建议 → 不整改 → 诉讼）；公益诉讼起诉主体资格；审判程序（普通程序 / 简易程序 / 审判监督程序）的区别。

### 11. 海淀期中 · Q10 · 网络暴力信息治理（法规推断）

- **题干**：《网络暴力信息治理规定》第 24 条——网络信息服务提供者发现用户面临网暴风险时的提示、保护、报告义务。
- **选项核心法律点**：
  - ①：整治网暴需多方主体共同参与（正确）。
  - ②：法规关注特殊群体（未成年人、老年人、残疾人）权益，彰显社会公平正义（正确）。
  - ③：用户遭网暴可向 "网络信息服务提供者" 提起诉讼（错——诉讼对象应是侵权人，不是平台；平台承担的是协助 / 提示等义务）。
  - ④：用户人身、财产受到严重损害，网络信息服务提供者必然承担责任（错——条文是 "可能造成严重后果" 时启动保护机制，非自动担责）。
- **法律细节边界**：网络平台 / 网络信息服务提供者的义务类型；侵权诉讼的被告确定；法规中 "应当报告" vs "应当承担责任" 的语义差异。

### 12. 海淀期末 · Q9 · 越界春笋的天然孳息归属

- **题干**：村民甲种植春笋越界长到村民乙的承包地里；《民法典》第 321 条 "天然孳息由所有权人取得；既有所有权人又有用益物权人的，由用益物权人取得"。
- **选项核心法律点**：
  - A：归村集体（林地所有权人）所有（错——本案有用益物权人）。
  - B：归村民乙所有（正确——乙是越界后所在地的用益物权人，依条文获得天然孳息）。
  - C：归种植者甲所有（错——条文以所有权人 / 用益物权人为准，而非种植行为人）。
  - D：基于绿色原则（错——条文依据是天然孳息归属规则，非绿色原则）。
- **法律细节边界**：天然孳息 vs 法定孳息；所有权 / 用益物权人对孳息的权利顺位；土地承包经营权的孳息收益权能。

### 13. 海淀期末 · Q10 · 合同 + 定金 + 不可抗力 + 合同成立

- **题干**：幸福石油公司与开心公司签汽油买卖合同；约定 1 日内付 10 万 "保金"；甲方股东抢公章致管理失控未能交货。
- **选项核心法律点**：
  - A：乙方可要求双倍返还保证金共 20 万（题干用 "保金"，定性是否定金 / 保证金需注意；若定金则适用定金罚则）。
  - B：甲方可主张不可抗力免责（错——股东抢公章是内部治理问题，不是不可抗力）。
  - C：经乙方同意，甲方可不按合同约定时间交货（合同变更需双方合意，但延期供货影响合同履行——选项表述若不严谨需推敲）。
  - D：双方未约定货物质量标准 → 合同不成立（错——可依国家标准 / 行业标准 / 通常标准补充，不影响合同成立）。
- **法律细节边界**：定金罚则（双倍返还）vs 保证金的区分；不可抗力的法定要件（不能预见、不能避免、不能克服）；合同成立 vs 合同生效；合同必备条款 vs 可补充条款。

### 14. 海淀期末 · Q11 · 客运合同承运人责任 + 法律援助 + 上诉期

- **题干**：赵某乘坐客运公司客车追尾受伤；《民法典》第 823 条规定承运人对运输过程中旅客伤亡承担赔偿责任（旅客自身健康原因或故意 / 重大过失除外）。
- **选项核心法律点**：
  - ①：客运公司能证明无过错即免责（错——承运人责任是无过错责任 + 法定免责事由限定）。
  - ②：因家庭经济困难无法承担诉讼费用，可申请法律援助（正确）。
  - ③：不服一审判决可在判决后 15 日内上诉（错——民事案件上诉期是判决书送达之日起 15 日）。
  - ④：法律给承运人分配更多注意义务以平衡各方利益、合理预防损害（正确）。
- **法律细节边界**：承运人责任的归责原则（无过错）；民事 / 刑事上诉期差异；法律援助的申请条件。

### 15. 海淀期末 · Q12 · 法定继承 + 抚恤金 + 仲裁边界

- **题干**：张大叔与王阿姨再婚未生育；唯一孩子是王阿姨与前夫所生小马（已成年，多年未尽赡养义务）；王阿姨去世，留 200 万房产（婚后购）和 8 万抚恤金。
- **选项核心法律点**：
  - 甲：房产 100 万（夫妻共同财产 1/2）属王阿姨遗产（正确——夫妻共同财产中王阿姨份额是其遗产）。
  - 乙：未留遗嘱即适用法定继承分配房产 + 抚恤金（错——抚恤金不是遗产，不进入继承）。
  - 丙：张大叔与小马都是第一顺位继承人，应平均分配（错——小马未尽赡养义务可少分或不分；且有法定酌定情节）。
  - 丁：可选择仲裁解决争议（错——婚姻 / 收养 / 监护 / 扶养 / 继承类纠纷不能仲裁）。
- **法律细节边界**：夫妻共同财产分割与遗产边界；抚恤金的法律性质（不是遗产，是对家属的慰问）；法定继承顺序与酌定调整规则；仲裁的适用范围排除（婚姻家庭纠纷不可仲裁）。

### 16. 海淀期末 · Q14 · 著作权（蜡染图案借鉴）

- **题干**：食品公司在产品包装上擅自使用洪某画作中具独创性的蜡染花鸟图案；公司辩称图案来自传统蜡染。
- **选项核心法律点**：
  - ①：洪某发表权等著作人身权不受保护期限制（正确——人身权无期限限制；财产权有期限）。
  - ②：洪某画作借鉴传统图案的事实由被告举证（正确——被告主张抗辩事实由被告举证）。
  - ③：未到登记机关登记 → 不能主张著作权（错——著作权自创作完成之日起自动产生，登记非必要）。
  - ④：双方可在法院主持下达成调解协议（正确——诉讼调解是法定纠纷解决方式）。
- **法律细节边界**：著作人身权 vs 著作财产权的保护期；著作权产生方式（自动取得，登记非必要）；诉讼调解的法定地位；举证责任分配规则。

### 17. 海淀期末 · Q15 · 探店视频 / 虚假宣传

- **题干**：探店视频混杂真实体验与广告营销；消费者难以区分。
- **选项核心法律点**：
  - A：虚假宣传探店视频侵犯消费者自主选择权（A 选项考的是消费者权利精确归类——8 项权利中 "知情权" 是首要被侵害的）。
  - B：发布广告类探店视频构成不正当竞争、侵犯其他经营者权益（错——合规广告本身合法，关键在 "虚假" 而非 "广告" 性质）。
  - C：法院可主动启动诉讼程序整治（错——法院 "不告不理"，被动启动）。
  - D：短视频平台推动制定行业内容管理公约 → 提高社会治理法治化水平（正确——平台自治 + 行业公约是社会治理重要补充）。
- **法律细节边界**：消费者八项权利的精确区分（特别是知情权 vs 自主选择权）；法院 "不告不理" 原则；社会治理多元主体（政府、平台、行业、公众）。

### 18. 西城一模 · Q9 · 行政处罚不予执行裁定

- **题干**：水果店虚假宣传被行政执法机关罚款 5 万；店主未缴纳；机关向法院申请强制执行；法院以过罚相当原则、违法行为轻微及时改正、未造成危害后果为由，裁定不准予强制执行。
- **选项核心法律点**：
  - A：及时化解了公民和政府之间的 "民事" 纠纷（错——这是行政纠纷，非民事）。
  - B：通过 "行政复议" 维护社会公平正义（错——本案是司法审查，不是行政复议）。
  - C：有效规范行政执法行为，依法保障行政相对人权益（正确——这是司法对行政权力的监督功能）。
  - D：行政执法机关应严格文明执法，尽可能减少自由裁量权（错——自由裁量权不能被消灭，关键是规范行使）。
- **法律细节边界**：行政复议 vs 行政诉讼；过罚相当原则；行政强制执行的司法审查；行政自由裁量权的合理空间。

### 19. 西城一模 · Q10 · 网络公众号转载 / 著作权 / 肖像权 / 隐私权

- **题干**：学生在网络平台注册公众号，发布学科活动、时政述评、学习成果等。
- **选项核心法律点**：
  - A：网上公开图片资源都属于公共财富，可拿来用（错——网络发布不等于放弃著作权）。
  - B：注明作者名字即合法（错——还需取得授权或符合合理使用 / 法定许可）。
  - C：汇编 / 改编他人作品付出劳动 → 直接刊登合法（错——需经原著作权人许可）。
  - D：发布他人拍摄的他人照片 → 存在著作权、肖像权、隐私权三重侵权风险（正确——拍摄者享有著作权，被拍摄者享有肖像权 + 隐私权）。
- **法律细节边界**：转载 / 引用 / 汇编 / 改编的授权要求差异；著作权与肖像权 / 隐私权的竞合；合理使用与法定许可的边界。

### 20. 西城一模 · Q11 · 委托合同 + 饲养动物致害 + 未成年人诉讼

- **题干**：小阳外出旅游，委托宠物店主小飞每天遛狗；小飞遛狗时狗将 6 岁小丁咬伤。
- **选项核心法律点**：
  - A：小丁父母只能向狗主人小阳赔偿（错——可向饲养人 / 管理人主张，小飞此时是管理人）。
  - B：小阳与小飞间存在委托合同；小飞过错则构成违约（正确——但需注意：小丁求偿是侵权，小阳与小飞之间是合同关系）。
  - C：小丁未成年起诉必须委托律师（错——未成年人由法定代理人代为诉讼，律师非必要）。
  - D：小丁是无民事行为能力人，即使有过错小飞也不可主张减轻（错——无民事行为能力人不存在 "过错" 概念，但饲养动物致害归责仍有规则）。
- **法律细节边界**：饲养动物致害的责任主体（饲养人 / 管理人）；委托合同的违约责任 vs 第三人侵权责任；未成年人诉讼能力（由法定代理人代理）。

### 21. 西城期末 · Q12 · "先用后付" 网络消费

- **题干**：网络购物 "先用后付"（无需付款下单、确认收货后付款），存在 "莫名开通"、"被自动勾选"、"开通易关闭难"、家里小孩 "误点" 等问题。
- **选项核心法律点**：
  - ①：消费者确认收货后再付款，"先用后付" 本质是合同关系（正确——双方意思表示一致即成立）。
  - ②："莫名开通" 该支付方式 → 侵犯消费者知情权和隐私权（错——主要侵犯知情权和自主选择权，而非隐私权）。
  - ③：平台应完善服务协议，重要条款应当以显著方式提醒消费者注意（正确——格式条款的提示义务）。
  - ④：小孩误点没关系，未满 18 周岁的民事行为都是可撤销的（错——未满 8 周岁是无民事行为能力人，行为无效；8—18 是限制民事行为能力人，与其年龄智力相适应的行为有效）。
- **法律细节边界**：电子合同的成立与生效；格式条款的提示和说明义务；自然人民事行为能力的三段划分（无 / 限制 / 完全）。

### 22. 西城期末 · Q13 · 侵权责任四情形分析

- **题干**：四种侵权情形的归责判断。
- **选项核心法律点**：
  - A：游客喂猴子被咬，应由 **动物园举证已尽管理职责**，否则担责（正确——饲养动物致害举证责任倒置）。
  - B：12 岁学生跳绳被绊倒，校方不论有无过错都担责（错——8 周岁以下校方过错推定；8 周岁以上学生需举证校方有过错）。
  - C：经营者排放氯化氰致土壤污染，由起诉村民证明因果关系（错——环境污染侵权由污染者承担因果关系不存在的举证责任，举证责任倒置）。
  - D：路人被楼上抛掷物砸中，住户能证明自己家中无人 → 仍需担责（错——能证明自己不是侵权人即免责）。
- **法律细节边界**：四种特殊侵权的举证规则——饲养动物（饲养人举证）、教育机构（年龄段不同规则不同）、环境污染（污染者举证因果关系不存在）、高空抛物（住户可证明自己不是侵权人免责）。

### 23. 门头沟一模 · Q3 · 网络数据安全 / 个人信息处理

- **题干**：《网络数据安全管理条例》第 22 条——网络数据处理者基于个人同意处理个人信息时的规则（不得超范围、敏感个人信息单独同意、不满 14 岁须监护人同意等）。
- **选项核心法律点**：
  - ①：网络数据处理者应增强法治观念，依法处理个人信息（正确）。
  - ②：处理个人信息都应取得父母 / 监护人同意（错——只针对不满 14 岁未成年人）。
  - ③：需完善法律实施机制，使法律法规得到有效实施（正确）。
  - ④：被侵权个人依法行使 "国家权力" 维权（错——公民行使的是 "权利"，不是 "国家权力"）。
- **法律细节边界**：个人信息处理的同意规则（一般同意 / 单独同意 / 监护人同意 / 书面同意）；法律实施机制 vs 法律制定；公民权利 vs 国家权力（基本宪法概念）。

### 24. 门头沟一模 · Q12 · 邻里养狗 + 摄像头 + 隐私权 + 饲养动物侵权

- **题干**：孙某楼道养狗引邻居不满；翟某装可视门铃，于某装室内临窗及楼道摄像头。
- **选项核心法律点**：
  - ①：孙某饲养的宠物伤人，不论有无过错均担侵权责任（饲养动物致害的无过错责任原则，但分一般动物 / 烈性犬 / 动物园 / 饲养人放弃管理等不同规则）。
  - ②：装可视门铃 / 摄像头需经全体业主同意（错——属个人行为，需注意不侵害他人隐私即可，无须全体同意）。
  - ③：孙某楼道养狗有过错；翟某、于某装设备属维护自身合法权益（部分正确——但摄像头若拍到他人住宅 / 楼道私密区域可能侵犯隐私）。
  - ④：孙某可向法院起诉，请求停止侵害、排除妨碍（正确——若摄像头等行为侵犯其权益）。
- **法律细节边界**：饲养动物致害的归责原则；可视门铃 / 摄像头与他人隐私权 / 个人信息保护的边界；停止侵害、排除妨碍的请求权基础。

### 25. 门头沟一模 · Q13 · 多元纠纷解决方式合规性

- **题干**：纠纷解决方式（仲裁、调解、刑事诉讼、劳动争议）。
- **选项核心法律点**：
  - ①：商事仲裁与诉讼之间，当事人只能选其一（正确——或裁或审）。
  - ②：人民调解协议具有 "强制执行" 效力（错——需经司法确认才有强制执行效力）。
  - ③：刑事诉讼中犯罪嫌疑人 / 被告人有委托 "诉讼代理人" 的权利（错——刑诉中应是 "辩护人"，民诉才用 "诉讼代理人"）。
  - ④：劳动争议可通过劳动行政部门维权（正确）。
- **法律细节边界**：仲裁 / 调解 / 诉讼三大解决方式的衔接；调解协议的 "司法确认 → 强制执行" 路径；刑事 "辩护人" vs 民事 "诉讼代理人" 的术语区分。

## 四、命题趋势速读（基于 2025 届整体）

1. **"判决意义" 三段式高频出现**：法律依据 / 法律事实 / 法律责任 + 现实意义 → 这是 2025 届最稳定的主观题答题模板，建议学生按此模板背训。
2. **新业态高度集中**：AI 生成图片著作权、外卖骑手劳动关系、平台捆绑销售、"先用后付"、数字消费维权、智能音箱唤醒词商标 → 命题人偏好 "新场景 + 老法理"。
3. **侵权责任题量最大**：饲养动物（含烈性犬）、产品责任、安全保障义务、网络侵权（肖像权 / 著作权 / 隐私权）、相邻权、监护人责任、教育机构责任、高空抛物——基本覆盖《民法典》侵权责任编全图。
4. **劳动法热度上升**：试用期解除、竞业限制、新业态用工认定、劳动仲裁前置、违法解除赔偿——朝阳、海淀、丰台、门头沟、延庆均出题。
5. **公益诉讼成新增长点**：检察公益诉讼（民事 + 行政）、"当诉则诉"、新就业形态权益保护——朝阳一模、丰台一模、海淀（古桥文物）多次出现。
6. **仲裁 / 调解 / 诉讼衔接是稳定考点**：商事仲裁 vs 劳动仲裁、调解协议司法确认、或裁或审 vs 仲裁前置——昌平二模、门头沟一模、丰台二模均涉及。
7. **答题动作奖励**：① 主体明确（"法院 / 公司 / 劳动者 / 消费者" 表述精确）；② 法条编号或核心要件复述；③ 上升到 "新质生产力 / 法治社会 / 数字经济" 这类时政语汇收束。
8. **不寻常发现**：① 同一情境（AI 图片侵权 / "小爱同学" 唤醒词 / 探店视频 / 烈性犬）在多套卷子中重复出现，命题人在 2024—2025 年最高法、北京互联网法院的典型案例库中高频取材；② "脚手架填表题"（朝阳期末 Q20、海淀期末 Q20、门头沟一模 Q20）成为新题型，命题人主动给学生答题模板；③ "新质生产力"、"数字经济"、"法治化营商环境" 三组时政语汇在主观题答案末尾几乎成 "标配收束句"，备考时应有意识训练。
