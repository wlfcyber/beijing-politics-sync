## 序言：这本框架要解决的 7 个真实痛点

读这本之前你可能在哪里卡住：
1. 看到一段民事案例，第一秒不知道它属于哪个法律领域。
2. 知道是合同/侵权题，但不知道答案应该写几段、每段写什么。

本框架七大部分对应解决这七件事。**不要跳读**——序言之外每一部分都要从头看完，因为后面表格依赖前面的术语。

## 第一部分：顶层主干（最重要的两段式答题结构）

整本《法律与生活》的命题底层逻辑只有一句话：

> **法律模块既要考"学理性、知识性"，又要紧扣"政治性、价值性"。**

把这句话翻译成你能上手的答题动作，就是**主观题答题双段结构**：

### 1.1 主观题答题双段结构（一切主观题都套这个模板）

```
第一段：知识链段 = 关系判断 + 争点识别 + 规则套入 + 效力/责任/救济落点
        （回答"凭法律怎么判""凭什么"）

第二段：功能落点段 = 该法律规则如何推动 [法治建设 / 社会经济发展 / 市场秩序 / 新业态规范]
        （回答"为什么这么判有意义"）
```

**第一段的最小完整结构**（按顺序写，缺一不可）：

| 动作 | 写什么 | 例 |
|---|---|---|
| ① 定主体关系 | 写清楚是 A 与 B 之间的什么关系（合同 / 侵权 / 婚姻家庭 / 雇佣 / 物权 / 行政 / 诉讼） | "甲乙双方系买卖合同关系" |
| ② 找争点事实 | 用一句话点出真正的争议焦点 | "争议焦点在于：是否构成欺诈" |
| ③ 套规则要件 | 把要件 1/2/3 与材料事实逐一比对 | "构成欺诈需满足：故意+虚假陈述+消费者陷于错误认识；本案……三要件齐备" |
| ④ 落效力/责任/救济 | 写出这道题的实体法律后果（**3 句强制句式，下面有完整模板**） | 见下方"判定句 3 句模板" |

**第④步"判定句 3 句模板"**（每道主观题第一段末尾**必须**写够这三句，少一句扣分）：

```
第 1 句（关系成立/效力定性）：故 [甲乙之间的 X 法律关系] [成立/不成立 / 合同 X 效力]。
第 2 句（责任归属）：[X 当事人] 应承担 [Y 责任名称——违约/侵权/连带等]。
第 3 句（救济落点）：[Z 当事人] 享有 [W 救济——继续履行/损害赔偿/停止侵害/惩罚性赔偿等具体救济方式 2-3 项] 请求权。
```

**真题套用例**（2026 朝阳期末 Q17(1) 直播带货）：
> "故陈某（消费者）与张某（经营者）之间形成的买卖合同有效。张某未按约定交付符合材质的产品构成违约且构成欺诈。陈某享有按"假一赔十"约定主张赔偿的请求权，依据**消费者权益保护法**亦可主张退一赔三的法定惩罚性赔偿。"

**第二段的标准句式**（只有 4 个落点，挑 1-2 个写）：

> "该规则的实施，通过 [落点 1]，进而 [落点 2]，对 [社会经济发展 / 法治建设 / 市场秩序 / 新业态规范] 起到 [具体作用]。"

四个落点是：
- **法治建设**：维护法律权威 / 推进良法善治 / 提升司法公信
- **社会经济发展**：保障交易安全 / 平衡多元利益 / 优化营商环境
- **市场秩序**：规范经营行为 / 制止不正当竞争 / 保护诚信交易
- **新业态规范**：明确新业态合规边界 / 保护新业态主体权益 / 引导技术向善

⚠️ **绝对不要写**："全面依法治国""中国特色社会主义法治体系""社会主义法治精神""国家治理现代化""社会主义核心价值观"——这些是必修三的话，写了等于没写。

### 1.2 选择题考查重点

选择题考的是**制度本身的内容乃至细节**——具体的概念、要件、程序、期限、救济方式、责任划分。

**两条永不可越的边界**：

- **不法考化**：选项不会出"过多主体、过多法律关系、疑难法学问题、罕见特殊侵权、超出选必二教材"的内容。如果某选项让你觉得"这是法考题吧"，那它通常是干扰项。
- **不必修三化**：选项也不会停在"价值取向、法治意义"层面而不涉及具体制度。如果某选项纯粹喊价值口号，往往是干扰项。

### 1.3 两个强制标签（每道题做完都要打）

每道题做完后给自己打两个分：
- **学理性 vs 政治性强度**（这题更考具体法律知识，还是更考价值导向？）
- **必修三化风险**（如果写飞了会丢哪几分？）

打这两个标签的目的是让你在 30 天冲刺里知道自己的弱项是"知识精确度"还是"价值话语精确度"，分别针对性补。

## 第一部分续：民事法律关系三要素（开题第一刀）

很多新题（如 2026 房山一模 Q17 AI 幻觉案）要求学生用"民事法律关系"知识辨析。**在动用任何主线前，先做这个最基础的判别**：

**民事法律关系 = 主体 + 客体 + 内容**

| 要素 | 内涵 | 例子 |
|---|---|---|
| **主体** | 谁在和谁发生这段关系 | 自然人 / 法人 / 非法人组织。**AI 不是民事主体**——AI 自动生成的"承诺"不产生意思表示效力 |
| **客体** | 这段关系指向什么 | 物（动产/不动产）、行为（劳务、服务）、智力成果（作品、专利、商标、商业秘密）、人身利益（生命、身体、肖像、隐私、个人信息） |
| **内容** | 主体之间的权利和义务 | 一方的请求权 / 另一方的义务；权利与义务相对应、相统一 |

**民事主体能力分级**（直接影响行为效力）：

| 主体类型 | 行为能力 | 行为效力规则 |
|---|---|---|
| 8 周岁以下 | 无民事行为能力人 | 实施的民事法律行为**无效**，由法定代理人代为实施 |
| 8 周岁至 18 周岁 | 限制民事行为能力人 | 与年龄、智力、精神健康相适应的行为有效；其余须经法定代理人同意/追认（**典型陷阱**：13 岁孩子文身——未追认则合同无效） |
| 18 周岁以上 | 完全民事行为能力人 | 行为有效 |
| 法人 / 非法人组织 | 通过法定代表人 / 负责人行使 | 越权行为外部相对人善意时仍对法人有效 |

**为什么这一节重要**：2026 年命题趋势是把"AI 主体""未成年人合同""法人代表行为"等具体情境包装成考题。学生必须先用"主体—客体—内容"切开案情，才能进入下面的七大主线。

## 第二部分：从材料到答案的四步动作（一核 / 二线 / 三问 / 四步 / 五域 落地版）

这套口诀是答题动作记忆口诀，**不是分类工具**。看到题不要先想"这是五域里哪一域"，要先按下面四步走完一遍。

### 2.1 一核：以事实为根据，以法律为准绳，定分止争

每道题最终都要回到这一核——**用法律解决具体的争议**。如果你写的答案没把"事实+规则+处理"三者闭环，多半丢分。

### 2.2 二线（永不混淆）

| 线 | 内容 | 答题侧重 |
|---|---|---|
| 第一线 | 权利保护 ↔ 权利边界 | 既要保护权利人，也要明确权利不能滥用（恶意诉讼、滥用诉权、相邻关系限制） |
| 第二线 | 法治规则 ↔ 德治价值 | 答完规则要回扣"诚信""公平""和谐""向善"等价值（这才是选必二的"价值"，不是必修三的"法治精神"） |

### 2.3 三问（任何材料都先答这三个）

| 问 | 答什么 | 例 |
|---|---|---|
| 判什么 / 怎么处理 | 法律关系类型 + 处理结果 | 合同有效/无效/可撤销；侵权成立/不成立；婚姻有效/无效 |
| 凭什么 | 法律要件比对 + 法条名称（必须写全名） | 民法典第 X 条；消费者权益保护法 |
| 有什么意义 | 制度功能落点（四向指南针） | 维护交易安全 / 平衡多元利益 / 推动新业态规范 |

### 2.4 四步（写主观题第一段时按顺序走）

```
第一步：定主体关系   → 写"X 与 Y 之间是什么关系"
第二步：找争点事实   → 写"争议焦点在于……"
第三步：套要件规则   → 写"根据民法典第 X 条 / XX法第 X 条，构成 X 需满足……"
第四步：落责任效力   → 写"故 X 应承担 Y 责任 / 合同效力为 Z / Y 享有 W 救济"
```

走完这四步，第一段（知识链段）就完成了。

### 2.5 五域（情境二级标签——只用来识别题型，不用来组织答案）

| 域 | 划分依据（必须解释） | 典型情境 |
|---|---|---|
| 交易交换域 | 共同处理"承诺、履行、信息不对称、权益保护"——一切以合意/对价为核心的关系 | 合同、消费者、电子商务、买卖租赁 |
| 身份财产域 | 共同处理"身份关系派生的财产分配与扶助"——基于亲属/家庭/继承的财产规则 | 婚姻、继承、夫妻共同财产、遗赠扶养 |
| 人身权益域 | 共同处理"人格自身的不可让渡性"——隐私、名誉、肖像、个人信息 | 肖像权、声音权益、个人信息保护、AIGC 数字人 |
| 创新竞争域 | 共同处理"无形财产的识别、保护、不正当竞争边界" | 著作权、专利、商标、商业秘密、不正当竞争 |
| 救济公共域 | 共同处理"国家/集体力量介入私人争议的程序与边界" | 民事诉讼、行政诉讼、调解、仲裁、公益诉讼 |

**边界归位规则**（学生最容易卡的 3 个边界情境）：

1. **网购+平台责任** → 优先归"交易交换域"（合同/消费者主线），平台责任作为附带分析。归"创新竞争域"只在涉及不正当竞争或商业秘密时才用。
2. **婚姻彩礼返还+共同债务** → 归"身份财产域"。如果题问的是"应该走哪种诉讼程序"才归"救济公共域"。
3. **AIGC 侵权+数字人格权** → 优先归"人身权益域"（肖像权/声音权益），新业态合规义务作为附加层。归"创新竞争域"只在涉及著作权权属争议时才用。

⚠️ **不要把"消费者"和"侵权"硬合并**——两者要件链不同，主线不同，混合答会丢"关系判断"分。

## 第三部分：七大主线（学生第一遍学的入口）

这七大主线是从近三年北京各区一模/二模/期末/期中里**实际考过的命题模式**总结的。每条主线下都给：核心规则 → 高频情境 → 看到什么材料想到什么 → 标准得分句模板 → 反例（必修三化 / 法考化各一例）。

### 主线 1 — 合同（交易交换域核心）

**核心规则**：
- 合同效力四态：有效 / 无效 / 可撤销 / 效力待定
- 有效要件：主体适格 + 意思表示真实 + 内容形式合法 + 不违反公序良俗
- 履行原则：全面履行 + 诚实信用
- 违约救济：继续履行 / 采取补救措施 / 赔偿损失 / 支付违约金 / 定金罚则
- 解除规则：法定解除 / 约定解除 / 协议解除

**高频情境**：
1. **直播带货虚假宣传**（朝阳期末 Q17(1) "假一赔十"案，海淀一模 Q18 案件 2）：
   - 看到 → 经营者承诺/广告/直播话术与实物不符
   - 写得分句 → "直播间承诺构成买卖合同的内容，对经营者具有约束力；经营者交付的货物不符合合同约定，应当按约定承担赔偿责任。约定赔偿标准虽高于法定标准，依意思自治原则应当履行。"
   - 反例（必修三化）：~~"全面推进消费领域依法治国"~~ → 改为"维护诚信交易，规范新业态经营秩序"
   - 反例（法考化）：~~"经营者构成商业欺诈罪"~~ → 改为"经营者构成欺诈，消费者有权主张退一赔三"

2. **合同+侵权竞合**（东城期末 Q18 产品瑕疵致害案）：
   - 看到 → 合同未全面履行同时造成人身/财产损害
   - 写得分句 → "甲乙系买卖合同关系。乙未交付符合约定的产品构成违约，应承担违约责任；同时产品瑕疵造成人身损害构成侵权，应承担侵权责任。两种责任竞合，受害人可择一主张。"
   - 反例：直接写"刑事责任"——本案是民事，没有刑事

3. **合同形式自由 vs 必须书面**（海淀一模 Q10 收购口头约定）：
   - 看到 → 题目讨论"未书面=无效？"
   - 写得分句 → "买卖合同采取何种形式由当事人约定，原则上不影响合同效力。除法律另有规定（如不动产买卖、融资租赁等）外，口头形式有效。"
   - 反例：~~"未签书面则合同未成立"~~（错——形式自由原则）

**主线 1 给分硬卡口**：
- 写"合同有效"必须同时点出"主体适格 + 意思表示真实 + 不违法不违公序良俗"至少 2 项要件，不能只下结论
- 涉及"约定优先于法定"的题，必须出现"意思自治"四个字
- "继续履行""采取补救措施""赔偿损失""违约金""定金"是 5 种合同救济方式，写到救济段时尽量列至少 2 种

### 主线 2 — 侵权（一般 + 七类特殊）

**核心规则**：
- 一般侵权四要件：违法行为 + 损害事实 + 因果关系 + 主观过错
- 归责原则三种：过错责任 / 过错推定责任 / 无过错责任
- 救济方式六种：停止侵害 / 排除妨碍 / 消除危险 / 赔礼道歉 / 消除影响恢复名誉 / 赔偿损失（含精神损害+惩罚性赔偿）

**七类特殊侵权**（每类对应不同归责原则）：

| 类型 | 归责 | 责任主体 | 抗辩事由 |
|---|---|---|---|
| 产品责任 | 无过错（生产者）/ 过错（销售者） | 生产者+销售者连带 | 未投入流通 / 投入时科技水平不能发现 |
| 高度危险作业 | 无过错 | 经营者 | 受害人故意 / 不可抗力 |
| 饲养动物 | 无过错（一般动物）/ 严格无过错（烈性犬等禁养） | 饲养人/管理人 | 受害人故意（一般）/ 仅"重大过失"对烈性犬 |
| 网络侵权 | 过错 | 网络用户 + 网络服务提供者（"通知-删除"规则） | 平台已尽合理注意义务 |
| 环境/生态 | 无过错 | 污染者 | 法律规定的免责事由 |
| 建筑物悬挂物脱落 | 过错推定 | 所有人/管理人/使用人 | 证明自己无过错 |
| 监护人责任 | 无过错（自己责任） | 监护人 | 不能完全免责，仅能减轻 |

**高频情境**：

1. **AIGC 数字人侵权**（东城一模 Q11，房山一模 Q10）：
   - 看到 → 未经同意采集人脸/声音生成数字人 + 未标注"AI 生成"
   - 写得分句 → "未经赵某同意采集其面部特征和声音用于商业，侵害肖像权和声音权益（声音权益参照肖像权保护）。未标注'AI 生成'违反新业态标识合规义务。受害人可主张停止侵害、删除数据、赔礼道歉、赔偿损失。"
   - 反例（错误救济用词）：~~"恢复原状"~~（人格权侵权救济不写"恢复原状"，要写"消除影响"或"赔礼道歉"）
   - 反例（必修三化）：~~"维护社会主义核心价值观"~~ → 改为"推动技术向善 / 规范新业态合规边界"

2. **建筑物悬挂物脱落**（石景山一模 Q18 窗户脱落致车损）：
   - 看到 → 高空坠物/悬挂物脱落致损
   - 写得分句 → "窗户脱落致车损属于建筑物悬挂物脱落侵权，适用过错推定责任。受害人需举证损害事实、损害程度及因果关系；所有人/管理人需举证自己无过错或法定免责事由。"
   - 反例：~~"无过错责任"~~（错——是过错推定，不是无过错）

3. **安全保障义务边界**（丰台一模 Q20 餐厅外摔倒案）：
   - 看到 → 经营者场所内顾客自伤起诉经营者
   - 写得分句 → "经营者负有安全保障义务，但义务有边界。本案台阶安全无异常、无积水积雪等隐患，经营者已尽合理义务；受害人未尽自身注意义务（低头看手机），应自行承担后果。判决体现过错责任原则与个人责任意识的平衡。"
   - 反例（"和稀泥"思维）：~~"经营者应基于公平原则承担部分责任"~~（错——无过错就是无过错，不是和稀泥）

**主线 2 给分硬卡口**：
- 一般侵权答题必须出现"违法行为/损害事实/因果关系/过错"四要件中至少 3 项，不能只说"侵权成立"
- 救济用词必须精准：肖像权用"停止侵害+消除影响+赔礼道歉"，**不要写"恢复原状"**
- 提到"惩罚性赔偿"必须说明"法定情形"（故意 / 情节严重 / 欺诈 / 重大事故 / 烈性犬等），不能泛泛说"该赔偿"
- 写"举证责任"时必须分清"谁主张谁举证"（一般侵权）和"过错推定"（建筑物悬挂物脱落、产品责任、医疗）的差别

### 主线 3 — 物权（所有权 / 用益物权 / 担保物权）

**核心规则**：
- 物权三类：所有权（占有/使用/收益/处分四权能）、用益物权（土地承包经营权/宅基地使用权/建设用地使用权/居住权/地役权）、担保物权（抵押权/质权/留置权）
- 物权变动：动产交付为生效要件 / 不动产登记为生效要件（少数登记对抗）
- 善意取得：受让人善意 + 合理对价 + 已登记或交付 → 取得所有权
- 相邻关系：通行/排水/采光/通风/相邻不动产利用便利
- 绿色原则：物权行使须节约资源、保护环境

**高频情境**：

1. **宅基地+婚后建房+夫妻共同债务**（东城一模 Q10）：
   - 看到 → 宅基地使用权人是一方，婚后共同建房 + 一方借款用于装修
   - 写得分句 → "宅基地使用权属于用益物权，受集体经济组织成员资格限制，不得擅自处分。婚后在宅基地上建造的房屋通过建造取得所有权，无需登记，属夫妻共同财产。借款用于家庭日常生活（装修），属于夫妻共同债务。"
   - 反例：~~"宅基地建房需先登记才能取得所有权"~~（错——建造取得不需登记）

2. **抵押 vs 质押客体辨别**（朝阳一模 Q10）：
   - 看到 → 题目把动产、不动产、权利混罗列问能否抵押/质押
   - 写得分句 → "抵押客体包括建筑物、建设用地使用权、生产设备等动产/不动产；质押客体包括动产（动产质押）和权利（权利质押，如汇票、股权、存单、知识产权中的财产权）。股票应设立质押（权利质押）而非抵押。质权人在债务到期未清偿时不得擅自占有使用质押物，须依法处置（拍卖、变卖、折价）。"
   - 反例（法考化）：纠缠"流质条款效力"——超纲

3. **物权 + 绿色原则**（西城一模 Q17）：
   - 看到 → 题目讨论物权行使的环境义务边界
   - 写得分句 → "民法典将绿色原则贯穿物权编全过程：民事主体享有物权的同时必须履行节约资源、保护环境的义务。法律合理界定不动产、用益物权、相邻关系等权利行使边界，协调多元利益，推动绿色生产方式转型。"
   - 反例：~~"维护社会主义生态文明建设"~~（必修三化） → 改为"实现个人权利与公共生态利益的平衡"

**主线 3 给分硬卡口**：
- "用益物权"必须写"使用价值"，不要写"所有权"
- 涉及宅基地的题，**不要忘记**"集体经济组织成员资格限制""不得擅自处分"
- 善意取得三要件（善意+合理对价+已登记/交付）必须三件套全列，不能少一件

### 主线 4 — 婚姻继承（身份财产域核心）

**核心规则**：
- 夫妻财产制：法定共有 + 约定优先（约定可以全部归各自）
- 共同债务认定：① 夫妻共同意思表示 ② 用于家庭日常生活 ③ 用于共同生产经营
- 离婚财产分割：照顾子女、女方和无过错方 + 家务劳动补偿（民法典第 1088 条）
- 继承顺序：**遗赠扶养协议 > 遗嘱继承/遗赠 > 法定继承**
- 法定继承顺位：第一顺位（配偶/子女/父母）→ 第二顺位（兄弟姐妹/祖父母/外祖父母）

**高频情境**：

1. **遗赠扶养协议 vs 遗嘱**（通州期末 Q12）：
   - 看到 → 立遗嘱在前 + 后签遗赠扶养协议 + 受扶养人已履行义务
   - 写得分句 → "遗赠扶养协议是双务有偿合同，效力优先于遗嘱。受扶养人已履行扶养义务的，有权主张协议约定的财产归属。"
   - 反例：~~"遗嘱在先效力优先"~~（错——遗赠扶养协议优先）

2. **家务劳动补偿**（海淀一模 Q11）：
   - 看到 → 一方承担主要家务、另一方收益高 + 离婚分割
   - 写得分句 → "夫妻双方一方承担较多家庭义务的，离婚时有权请求另一方给予补偿（民法典第 1088 条）。该制度体现夫妻平等的核心原则，承认家务劳动的价值。"
   - 反例：~~"举证责任倒置"~~（错——是"谁主张谁举证"）

3. **婚后收入归属 + 著作权人身权独立**（海淀一模 Q11）：
   - 看到 → 婚内创作小说+版权费 + 离婚分割
   - 写得分句 → "婚姻关系存续期间所得的稿酬等知识产权财产收益属于夫妻共同财产，应当均分。但著作权的人身权（署名权、修改权、保护作品完整权）专属于作者本人，不属于夫妻共同财产。"
   - 反例：~~"著作权全部归作者个人"~~（错——财产权部分共有）

**主线 4 给分硬卡口**：
- 写"夫妻共同债务"必须给出认定标准（共同意思表示 / 家庭日常生活 / 共同生产经营）至少 1 项
- 涉及遗嘱的题，**不要漏**"遗赠扶养协议优先"这一隐藏顺位
- "家务补偿"要点出民法典第 1088 条 + "夫妻平等"价值

### 主线 5 — 知识产权 + 新业态（创新竞争域核心）

**核心规则**：
- 著作权：自动取得（独创性即获保护）/ 包含人身权（署名权等）+ 财产权（复制权 / 发行权 / 改编权 / 信息网络传播权等 17 种）
- 专利权：发明 / 实用新型 / 外观设计 / 必须申请审查授权 / 期限分别为 20/10/15 年
- 商标权：注册取得（少数知名未注册保护）/ 期限 10 年可续展
- 商业秘密：非公知 + 商业价值 + 保密措施 → 区别于专利（公开换保护）
- 惩罚性赔偿：故意 + 情节严重 → 1 倍至 5 倍（知识产权侵权）
- 新业态合规：直播带货标识 / AIGC 标识 / 平台主体责任

**高频情境**：

1. **司法守护"向新力" 三案模板**（朝阳一模 Q18，东城一模 Q18）：
   - 看到 → 三案分别涉及调解、惩罚性赔偿、规制滥诉
   - 写得分句标准模板：
     - 调解案 → "通过诉讼调解化解纠纷，推动纠纷实质性化解，节约司法资源，减少诉讼对创新的干扰。"
     - 惩罚性赔偿案 → "对故意、情节严重的知识产权侵权适用惩罚性赔偿，大幅增加侵权成本，发挥警示规范作用，维护市场秩序。"
     - 规制滥诉案 → "驳回并谴责恶意诉讼，惩治滥用诉讼程序的行为，维护诉讼司法秩序，体现诚信原则与权利义务统一。"
     - 总说 → "公正司法，以事实为根据，以法律为准绳，发挥司法裁判导向作用。保护知识产权就是保护创新，提升创新活力。"
   - **写"换包装销售商品"案——绝对不要写**：~~"不正当竞争""搭便车""消费者知情权"~~（题意是侵权，不是混淆行为）
   - **写"恶意诉讼"案——绝对不要写**：~~"商业诋毁""不正当竞争"~~（题意是程序滥用，不是实体侵权）
   - 同一短语在不同段落不重复给分——三段必须用差异化表达

2. **著作权独立于物权**（通州期末 Q13，东城期末 Q10D）：
   - 看到 → 物的赠与/出售 vs 作品著作权
   - 写得分句 → "赠与/出售物本身转移所有权（物权），但不转移著作权。著作权基于作品独创性自动产生，与作者年龄无关。受赠人取得物的所有权后，未经著作权人许可不得擅自展览、汇编、出版（侵犯展览权、汇编权）。"
   - 反例：~~"未满 18 周岁不享有著作权"~~（错——著作权与年龄无关）

3. **商业秘密 vs 专利**（东城期末 Q10B）：
   - 看到 → 非公知技术信息被破译/披露
   - 写得分句 → "非公知技术信息属于商业秘密，与专利权不同：专利权以公开换取保护，商业秘密以保密获得保护。破译并销售他人商业秘密构成不正当竞争，应承担停止侵害、赔偿损失等责任。"

**主线 5 给分硬卡口**：
- 写"惩罚性赔偿"必须说明"故意+情节严重"的法定门槛，不能滥用
- 知识产权侵权救济不要写"恢复原状"，要写"停止侵害+赔偿损失（含惩罚性）"
- 与"创新""向新力"挂钩的题，最后必须有"保护知识产权就是保护创新"或同义表达
- "商业秘密""专利权""商标权""著作权"四者不能混用——商业秘密属"权益"非"权"

### 主线 6 — 消费者 + 劳动（交易交换域 + 救济公共域交叉）

**核心规则（消费者）**：
- 八项权利：安全权 / 知情权 / 自主选择权 / 公平交易权 / 求偿权 / 结社权 / 获知识权 / 受尊重权 + 个人信息保护
- 经营者三义务：保障安全 / 真实告知 / 不得欺诈
- 惩罚性赔偿：欺诈 → 三倍 / 食品药品 → 十倍 / 一般缺陷致死亡或健康严重损害 → 二倍以下
- 法律名必须写全：**消费者权益保护法**——**不要缩写为"消法""保护法"**

**核心规则（劳动）**：
- 劳动合同必须书面（建立劳动关系起一个月内）/ 未签可主张二倍工资
- 试用期上限：合同 1 年内试用期 ≤ 1 个月；1-3 年 ≤ 2 个月；3 年以上或无固定期限 ≤ 6 个月
- 解除：协商一致 / 劳动者单方解除（提前 30 日通知）/ 用人单位单方解除（法定情形）
- 经济补偿 vs 赔偿金：合法解除给经济补偿；违法解除给赔偿金（= 经济补偿×2）
- 劳动争议处理：协商 → 调解 → 仲裁（前置）→ 诉讼

**高频情境**：

1. **直播带货虚假宣传 + 三倍/十倍**（朝阳期末 Q17(1)，海淀一模 Q18 案件 2）：
   - 看到 → 经营者明知产品不符合宣传仍销售
   - 写得分句 → "经营者明知产品不符合宣传仍销售构成欺诈。**消费者权益保护法**规定欺诈情形下消费者可主张三倍惩罚性赔偿；食品药品另规定十倍赔偿。本案约定'假一赔十'已构成合同内容，依意思自治原则应当履行。"
   - **法律名硬卡口**：写"消费者权益保护法"必须写全，缩写不给分

2. **违法解除劳动合同**（东城期末 Q11）：
   - 看到 → 用人单位以"旷工""绩效不合格"等单方解雇 + 程序违法/事实不实
   - 写得分句 → "口头承诺与书面合同条款发生冲突时，事实劳动关系仍受保护。用人单位以未充分查明的事实解除劳动合同构成违法解除，应当支付赔偿金（经济补偿×2）。劳动者可申请劳动仲裁，对仲裁不服 15 日内向法院起诉。"
   - 反例：~~"提起行政复议"~~（错——劳动争议是仲裁→诉讼，不是行政复议）

3. **个人信息保护 + 经营者义务**（海淀一模 Q18 案件 1）：
   - 看到 → 商家公开消费者监控录像/聊天信息报复差评
   - 写得分句 → "监控录像、聊天信息属于个人隐私和个人信息。经营者负有保护消费者个人信息的法定义务。公开传播构成隐私权与个人信息权益侵害，受害人可主张停止侵害、赔礼道歉、赔偿损失（含精神损害赔偿）。"

**主线 6 给分硬卡口**：
- "消费者权益保护法"必须写全名——写"消法""保护法"或"消费法"**直接不给分**
- "经济补偿"和"赔偿金"是两个不同概念，违法解除是赔偿金（×2），不要混
- 劳动争议必须先仲裁后诉讼（仲裁前置），不能跳过仲裁直接起诉

### 主线 7 — 程序救济（救济公共域核心）

**核心规则**：
- 民事诉讼 vs 行政诉讼：被告差异（私主体 vs 行政机关）/ 举证责任差异（谁主张谁举证 vs 行政机关举证倒置）
- 仲裁：协议优先 / 一裁终局
- 调解：人民调解 / 司法调解 / 行政调解 / 行业调解
- 公益诉讼：民事公益（污染、消费、英烈、未成年保护）+ 行政公益（食品药品、生态、国有资产、未成年人）
- 检察建议：非诉化解机制
- 上诉期：判决书送达之日起 15 日内（民事）/ 15 日内（行政）/ 10 日内（裁定）

**高频情境**：

1. **举证责任分配**（石景山一模 Q18）：
   - 看到 → 民事侵权 + 行政诉讼并列，问举证责任如何分配
   - 写得分句 → "行政诉讼实行举证责任倒置：被告（行政机关）应提供作出该行政行为的证据和依据的规范性文件。这一倒置目的是督促依法行政、保障行政相对人合法权益。民事侵权一般实行'谁主张谁举证'：原告需举证损害事实、损害程度及因果关系；特殊侵权（如建筑物悬挂物脱落）适用过错推定，由所有人/管理人/使用人举证自己无过错。"
   - 反例：~~"民事都实行举证责任倒置"~~（错——一般侵权是"谁主张谁举证"）

2. **行政公益诉讼**（丰台期末 Q18）：
   - 看到 → 检察院发现行政机关不依法履职 + 立案 + 检察建议
   - 写得分句 → "检察机关立案行政公益诉讼并发出检察建议，督促行政机关依法履职，是法律监督职能的体现。该制度维护国家利益和社会公共利益，弥补一般诉讼'原告缺位'的问题，推动法律实施落地。"
   - 反例：~~"维护社会主义法治体系"~~（必修三化） → 改为"维护国家利益和社会公共利益 / 推动严格执法"

3. **多元解纷**（朝阳一模 Q18 第一段，东城一模 Q18 第一段）：
   - 看到 → 法院调解促成和解

### 表 6 — 踩分边界禁区（高级反例库）

1. **不重复给分**：同一个意思在不同段落用不同表达写两次也只算一次。比如朝阳一模 Q18 三段答案不能都用"司法公正"，要用"调解优先 / 严格保护 / 规制滥用"差异化表达。
2. **材料触发完整性**：题目材料给了 N 个条件，必须 N 个全应答。比如海淀一模 Q18 给了案件 1+案件 2，不能只答一案。
3. **案例独立性**：三个案例的答案不能互相借词，每个案例必须用自己的关键词。
4. **设问主体精准**：题目问"法院如何"就答"法院做了什么+依据什么"，问"经营者怎么做"就答"经营者义务+应做+不该做"，**不要把主体写错**。
   **常见主体错位反例**：
   - 问"**法院**如何守护知识产权"→ 写"国家/政府/我们"等大主语都不给分。必须开头"法院通过……"。
   - 问"**经营者**应如何"→ 写"消费者应……"或"国家应……"是反向回答。
   - 问"**检察院**如何监督行政"→ 写"政府部门/纪委/法院"是错位主体。
   - 问"**未成年人保护机构**应如何"→ 写"学校/家长"是漏掉法定主体。
   - 问"**当事人**应如何维权"→ 写"法律应……"是把行动主体倒错。
5. **细节精准胜过宏观**：法律名要写全 / 要件要列出 / 救济要具体 / 法条号能写就写。学生最容易因为"为了简洁"丢这些分。
6. **判定句强制句式**：每段第一句都要是"构成 X 责任 / 合同 X 效力 / 享有 X 权利 / 应承担 X"——不能上来就罗列法条而无判定。
7. **权利不得滥用**（v5 新增）：民事主体行使民事权利不能超过正当界限。**消费者**首单受食品安全惩罚性赔偿保护，但加购大量同款超出正常生活消费需要的部分**不予支持**（西城二模 Q16）；**起诉权**也不能滥用，恶意诉讼/反复变更诉求/利用程序权利打击对手将被驳回并谴责（朝阳一模 Q18 案 3、东城一模 Q18 案 3）。把"权利保护"和"权利边界"双向写入答案才完整。

## 第五部分：选择题专属（17 个二级类陷阱模式总览）

选择题必须分开学。**主观题踩分关键词不要带到选择题分析里来**——选择题考"制度细节"，要动脑子分辨概念。

下面 17 类陷阱按出现频率排序，**每类模板都给典型考点 + 常见错项陷阱 + 正确判断方向**。

### 类 1 — 抵押 vs 质押客体辨别（朝阳一模 Q10、东城期末 Q10A）
- **典型考点**：股票/汇票/股权/专利/动产/不动产能否抵押或质押
- **错项陷阱**：把"质押客体（股票/汇票/权利）"写成"抵押客体"
- **判断方向**：动产抵押或权利质押才对；动产质押需交付质物；不动产抵押需登记

### 类 2 — 著作权 vs 物权独立（通州期末 Q13、东城期末 Q10D、朝阳一模 Q10）
- **典型考点**：物的赠与/出售 vs 作品著作权
- **错项陷阱**：以为转移物的所有权同时转移著作权
- **判断方向**：物权与著作权独立；著作权基于独创性自动产生，与年龄无关

### 类 3 — 商业秘密 vs 专利权 vs 不正当竞争（东城期末 Q10B）
- **典型考点**：非公知技术信息被破译/披露
- **错项陷阱**：把"破译商业秘密"写成"侵犯专利权"
- **判断方向**：商业秘密以保密换保护，专利以公开换保护；二者维度不同

### 类 4 — 失信被执行人 vs 个人破产试点（房山一模 Q11）
- **典型考点**：法院创新司法实践、平衡债权人和债务人利益
- **错项陷阱**：把"失信名单"写成"违约责任承担方式"
- **判断方向**：失信名单是执行措施；保留生活必需品是执行豁免（保障生存权）；信用修复是激励机制

### 类 5 — 一般侵权举证 vs 过错推定 vs 无过错（房山一模 Q10、海淀一模 Q11）
- **典型考点**：肖像权/名誉权侵权举证、建筑物悬挂物脱落、产品责任
- **错项陷阱**：把"一般侵权"说成"举证责任倒置"
- **判断方向**：一般侵权"谁主张谁举证"；过错推定（建筑物悬挂物、医疗等）；无过错责任（产品/高度危险/环境/饲养动物）

### 类 6 — 网络侵权"通知-删除"+ 平台责任（房山一模 Q10）
- **典型考点**：直播主播+平台共同责任、AIGC 标识合规
- **错项陷阱**：以为主播享有著作权可对抗肖像权
- **判断方向**：著作权不能对抗人格权；平台收到通知后未及时删除承担连带责任

### 类 7 — 婚姻家庭财产 + 家务补偿（海淀一模 Q11）
- **典型考点**：婚后稿酬归属、家务劳动补偿、举证责任
- **错项陷阱**：以为家务补偿适用举证责任倒置
- **判断方向**：婚后所得知识产权财产收益属共同财产；著作权人身权不属共同财产；家务补偿是"谁主张谁举证"

### 类 8 — 宅基地+夫妻共同债务+用益物权处分（东城一模 Q10）
- **典型考点**：婚后建房归属、用于装修的借款定性、宅基地处分
- **错项陷阱**：以为婚后建房需登记取得；以为用益物权可自由处分
- **判断方向**：建造取得不需登记；家庭日常生活借款是共同债务；宅基地受成员资格限制不得擅自处分

### 类 9 — AIGC + 数字人格权（东城一模 Q11、房山一模 Q10）
- **典型考点**：未经同意采集人脸/声音生成数字人、未标注"AI 生成"
- **错项陷阱**：把救济写成"恢复原状"；以为未标注 AI 是著作权问题
- **判断方向**：声音权益参照肖像权保护；标识违规属合规义务违反；救济用"停止侵害+消除影响+赔礼道歉"

### 类 10 — 遗赠扶养协议 vs 遗嘱（通州期末 Q12）
- **典型考点**：受扶养人履行扶养义务后的财产归属
- **错项陷阱**：以为遗嘱在先效力优先
- **判断方向**：**遗赠扶养协议 > 遗嘱继承/遗赠 > 法定继承**

### 类 11 — 合同形式自由 vs 不可抗力 vs 商标驳回救济（海淀一模 Q10）
- **典型考点**：买卖合同口头形式效力、误伤的归责、商标驳回救济渠道
- **错项陷阱**：以为口头合同无效；以为过失致损是不可抗力；以为商标驳回可申请行政复议
- **判断方向**：合同形式自由（买卖合同口头有效）；过失致损不是不可抗力；商标驳回应申请商标评审复审

### 类 12 — 劳动关系/合同/争议处理（东城期末 Q11）
- **典型考点**：未签书面合同、口头承诺、违法解除、争议处理
- **错项陷阱**：把"劳动争议"写成"行政复议"；把"违法解除赔偿金"写成"经济补偿"
- **判断方向**：事实劳动关系受保护；违法解除给赔偿金（×2）；争议处理是仲裁→诉讼

### 类 13 — 隐私权 vs 个人信息 vs 经营者义务（海淀一模 Q18）
- **典型考点**：商家公开消费者监控/聊天信息、虚假宣传
- **错项陷阱**：把"经营者义务"和"消费者权利"反向理解
- **判断方向**：经营者负有保护消费者个人信息的法定义务；侵犯隐私权救济包括精神损害赔偿

### 类 14 — 安全保障义务边界（丰台一模 Q20）
- **典型考点**：经营者场所内顾客自伤
- **错项陷阱**："和稀泥"思维（让经营者承担部分公平责任）
- **判断方向**：过错责任为原则；无过错就是无过错；个人责任意识

### 类 15 — 行政诉讼举证倒置 vs 民事举证一般规则（石景山一模 Q18）
- **典型考点**：被告是行政机关 vs 被告是私主体
- **错项陷阱**：把民事侵权说成"举证责任倒置"
- **判断方向**：行政诉讼倒置（被告举证）；民事一般"谁主张谁举证"；过错推定是民事特殊情形

### 类 16 — 行政公益诉讼 + 检察建议（丰台期末 Q18）
- **典型考点**：检察院发现行政机关不依法履职
- **错项陷阱**：把"行政公益诉讼"说成"民事公益诉讼"或"行政复议"
- **判断方向**：检察建议是非诉化解；公益诉讼维护国家利益和社会公共利益

### 类 17 — 物权 + 绿色原则 + 相邻关系（西城一模 Q17）
- **典型考点**：物权行使的环境义务、相邻不动产利用
- **错项陷阱**：把"绿色原则"写成"社会主义生态文明建设"
- **判断方向**：绿色原则贯穿物权行使全过程；权利与义务边界由法律合理界定

## 第六部分：写得分句的 7 个万能模板

直接套，套对就给分：

### 模板 1 — 合同效力判定句
> "甲乙之间形成 X 合同关系（买卖/承揽/租赁/赠与）。该合同主体适格、意思表示真实、内容形式合法、不违公序良俗，**合同有效/可撤销/无效**，理由是 [要件比对]。"

**真题套用演示**（2026 东城期末 Q18 第一问"分析合同效力"）：
> "陈某与刘某之间形成买卖合同关系。该合同主体适格、双方意思表示真实、内容合法、不违公序良俗，**合同有效**。理由是：双方就标的、价款、履行方式等核心条款达成合意，刘某收受陈某价款已实际履行交付义务，无重大误解、欺诈、胁迫等可撤销事由。"

### 模板 2 — 侵权成立判定句
> "[行为人] 实施了 [违法行为]，造成 [损害事实]，二者具有因果关系，行为人 [主观过错存在/过错推定/无过错亦应担责]，故 **侵权成立**。"

**真题套用演示**（2026 东城一模 Q11 AIGC 数字人侵权）：
> "某独资企业实施了未经林某同意采集其面部特征和声音用于商业的违法行为，造成林某肖像权与声音权益受损的损害事实，二者具有直接因果关系，企业作为商业主体明知未取得授权仍使用并未标注'AI 生成'，主观过错存在，故**肖像权与声音权益侵权成立**。"

### 模板 3 — 责任承担句
> "[当事人] 应承担 [违约责任/侵权责任]，具体形式包括 [继续履行/停止侵害/赔偿损失/赔礼道歉/惩罚性赔偿等]（挑 2-3 项写）。"

**真题套用演示**（2026 朝阳期末 Q17(1) "假一赔十"）：
> "张某（经营者）应同时承担**违约责任**与**侵权责任**：违约责任表现为按"假一赔十"约定向陈某支付十倍赔偿；侵权责任表现为依**消费者权益保护法**向陈某承担退一赔三的法定惩罚性赔偿。陈某可择一主张，亦可在两项中取较高者。"

### 模板 4 — 程序选择句
> "本案争议属于 [民事/行政/劳动] 争议，处理途径为 [协商/调解/仲裁/诉讼]。其中 [劳动争议先仲裁后诉讼 / 行政诉讼可申请行政复议或直接起诉 / 民事可直接起诉或调解]。"

### 模板 5 — 举证责任句
> "本案适用 ['谁主张谁举证'/过错推定/举证责任倒置]。具体为：[原告需举证 X，被告需举证 Y]。这一规则 [合理分配诉讼风险/督促依法行政/平衡当事人地位]。"

### 模板 6 — 功能落点句（功能落点段万能开头）
> "该规则的实施，通过 [落点 1：具体法律动作]，进而 [落点 2：制度功能]，对 [社会经济发展/法治建设/市场秩序/新业态规范] 起到 [具体作用]。"

### 模板 7 — 综合性命题路径句（用在长答最后总结）
> "本案体现了 [法律领域] 制度的精细化与场景化适用。通过 [具体处理]，平衡了 [当事人 A 的权利保护] 与 [当事人 B 的合理期待]，引导社会形成 [诚信交易/合理预期/责任意识]，推动 [创新活动/营商环境/纠纷化解] 向更高水平迈进。"

## 第八部分 — 补充制度（一二模高频但易漏的 12 个情境）

补到 v4 七大主线之外的真实命题——这些是 2024-2026 一二模反复出现但学生最容易卡住的：

### 8.1 民事法律关系三要素之延伸：限制行为能力人合同（13 岁文身案）

**情境**：13 周岁未成年人（限制民事行为能力人）单独与经营者签合同接受文身/纹身/穿耳/重大消费——经营者收费完成。父母诉请清洗赔偿。

**判定句模板**：
> "限制民事行为能力人实施与年龄、智力、精神健康状况不相适应的民事法律行为，须经法定代理人同意或追认。本案文身/纹身行为非与年龄相适应，未获追认，合同无效。经营者构成对未成年人身体权、健康权的侵害，应承担清洗、赔偿损失等侵权责任。"

**反例**：~~"未成年人也是民事主体，合同有效"~~ —— 错，主体≠完全行为能力。

### 8.2 合同：格式条款效力 + 提示说明义务

**情境**：经营者使用 App 预订、电商单方面"到店支付""不退不换"等格式条款。消费者质疑。

**判定句模板**：
> "格式条款是当事人为重复使用而预先拟定、未与对方协商的条款。提供格式条款一方应采取合理方式提示对方注意免除或减轻其责任、加重对方责任、限制对方主要权利的条款，并按对方要求予以说明。未尽提示说明义务的，对方可主张该条款不成为合同内容（不发生效力）。"

**反例**：~~"格式条款只要写在合同里就有效"~~ —— 错。

### 8.3 合同：要约 vs 承诺 vs 合同成立（碳排放权交易首例案）

**情境**：报价（内容具体确定）+ 中标通知 → 合同成立。一方不履行另购损失差价主张。

**判定句模板**：
> "甲方报价内容具体确定，构成要约；乙方中标通知送达甲方，构成承诺。承诺到达要约人时合同成立。一方不履行构成违约，应赔偿守约方的实际损失（含差价 + 利息）。"

### 8.4 合同：诉讼时效（普通债权 3 年 vs 不适用情形）

**情境**：欠款多年未催讨，债权人起诉时被告以诉讼时效抗辩；或赡养费欠付多年，被赡养人起诉。

**判定句模板（普通债权）**：
> "向人民法院请求保护民事权利的诉讼时效期间为三年。自权利人知道或应当知道权利受损害以及义务人之日起计算。最长不超过二十年。期间届满，债务人取得抗辩权，权利人丧失胜诉权。"

**判定句模板（不适用情形）**：
> "下列请求权不适用诉讼时效：①支付存款本息；②兑付国债、金融债券或向不特定对象发行的企业债券；③基于投资关系产生的缴付出资请求权；④**支付赡养费、抚养费、扶养费**；⑤其他法律规定的不适用情形。本案郭某诉冯某追讨赡养费，**不受诉讼时效限制**。"

**功能落点**：
> "诉讼时效一方面督促当事人积极行使权利，节约司法资源；另一方面对赡养费等基本生存权利保留例外，彰显公序良俗与人文关怀。"

### 8.5 合同：物业服务合同 + 不可抗力抗辩失败

**情境**：暴雨致房屋漏水，物业以"天灾免责"抗辩。

**判定句模板**：
> "不可抗力指不能预见、不能避免且不能克服的情形——三要件齐备方可免责。本案暴雨前已有气象预警，物业**理应预见**并采取预防措施，不构成不可抗力。物业未全面履行物业服务合同的安全保障义务，构成违约，应承担违约责任。经人民调解委员会调解达成的协议，双方可申请司法确认，确认有效后具有强制执行效力。"

### 8.6 侵权：好意同乘减责

**情境**：搭便车人未受邀但搭车，途中受伤，搭车人主张全额赔偿。

**判定句模板**：
> "驾驶人与搭乘人形成好意同乘关系，驾驶人对搭乘人生命财产安全负有合理的注意义务。本案驾驶人有过错（如雨天疏忽）但**非故意或重大过失**，主动垫付医疗费体现善意。依民法典第 1217 条好意同乘规则，**酌情减轻**驾驶人赔偿责任。这一规则既保护受伤一方权益，又鼓励善意助人、维护人际和谐、倡导绿色出行。"

**反例**：~~"无偿同乘所以驾驶人完全免责"~~ —— 错（仅减责，不免责）。

### 8.7 婚姻继承：遗产管理人制度（民法典新增）

**情境**：被继承人去世，继承人之间或继承人与受遗赠人之间无法协商遗产管理，需由民政部门或法院指定遗产管理人。

**判定句模板**：
> "继承开始后，继承人均不能管理或拒绝管理遗产时，应申请人民法院指定遗产管理人；继承人无人或全部放弃继承时由被继承人生前住所地民政部门或村民委员会担任。遗产管理人依法清理遗产、防止遗产损毁、按法律规定与遗嘱处理遗产、向继承人报告。"

**功能落点**：
> "遗产管理人制度填补了民法典前继承空白，依法落实遗产管理、避免遗产纠纷扩大、维护继承公平正义。"

### 8.8 知识产权：商标抢注与老字号在先权利

**情境**：他人将老字号或在先未注册商标抢注，老字号权利人主张撤销/无效宣告。

**判定句模板**：
> "民事主体取得权利受诚信、公平、守法和公序良俗原则约束。**抢注他人在先使用且具有一定影响的商标违反诚信原则**，老字号或在先权利人可向商标行政部门主张商标无效宣告。法院或行政机关支持在先权利保护，旨在维护正常市场秩序，引导经营者守法诚信经营，弘扬商业道德。"

### 8.9 劳动：平等就业权 / 反就业歧视

**情境**：企业以性别、户籍、外貌等差别对待求职者，求职者起诉。

**判定句模板**：
> "劳动者享有平等就业权，**不因民族、种族、性别、宗教信仰等受歧视**。本案企业以'仅招男性'为由拒绝面试，构成基于性别的就业歧视，违反公平、平等原则。求职者可主张赔礼道歉、精神抚慰金、诉讼费等民事救济。"

### 8.10 劳动：竞业限制 + 配偶间接竞业

**情境**：高管离职后其配偶在竞争企业任投资人/高管，原公司主张违约金；劳动争议仲裁委裁决支付违约金。

**判定句模板**：
> "竞业限制协议合法有效。劳动者离职后两年内不得从事与原单位生产或经营同类产品、相关业务的竞争活动。本案劳动者配偶在离职后成为业务竞争公司投资人，且夫妻关系存续期间所得为夫妻共同财产，存在利益一致性，关联公司代缴社保进一步印证利益关联。劳动者通过配偶间接从事竞业活动，构成违反竞业限制约定，应支付违约金。"

**功能落点**：引导劳动者权利义务相统一，平衡劳动者择业自由与用人单位商业秘密保护，构建和谐劳动关系。

### 8.11 劳动：集体合同 / 集体协商

**情境**：企业与工会代表签订集体合同涉及工资、工时、休息休假、保险福利等。劳动者权益保护与集体合同约定冲突。

**判定句模板**：
> "工会或职工代表通过与企业平等协商签订集体合同，对全体劳动者具有约束力。劳动合同条款不得低于集体合同标准。集体合同制度通过集体协商平衡劳动者与用人单位的力量对比，是构建和谐劳动关系的重要制度安排。"

### 8.12 程序：诉讼代理人 vs 辩护人 + 起诉状写作 + 司法建议

**8.12-A 诉讼代理人 vs 辩护人**（程序称谓硬卡口）：
- **民事案件 / 行政案件**：当事人委托的人称为"**诉讼代理人**"。
- **刑事案件**：被告人委托的人称为"**辩护人**"；被害人/自诉人委托的人称为"诉讼代理人"。
- ⚠️ **写错就丢分**：模拟庭审题里"我是被告人 X 的辩护人"——民事案件应改为"诉讼代理人"。

**8.12-B 起诉状要件**（程序文书写作题）：
> "起诉状应包含：①原告（姓名/名称、身份信息、地址）；②被告（同上）；③诉讼请求（具体明确，含赔偿金额、其他承担方式、诉讼费由被告承担）；④事实与理由（按时间顺序客观陈述事实，并指出对方违反何法律、何规定，承担何责任）；⑤起诉法院、起诉日期、原告签名/盖章。"

**8.12-C 司法建议作为软性治理工具**：
> "司法建议是法院在审判过程中针对所发现的问题向相关单位发出的非诉化解机制。它与判决书不同，**不具有强制执行效力**，但具有指引和规范功能。司法建议在'红线'与'空间'之间为新业态、新技术发展划定合规边界，体现司法的能动作用。"
