# ROUND 2 — 愤怒高三生事前冷读：框架必须回答的最低问题清单

**模拟身份**：北京高三学生，聪明、肯学，但选必二《法律与生活》一无所知。距高考 30 天。已经被市面上的"宝典""一本通"骗过两次（"全是大词，关上书就忘"）。

**冷读规则**：
- 只读教研员韩金阳总纲（4 页 PPT）+ 飞哥小本本（含 4-30 / 5-03 / 5-04 全部追加）。
- **不读** B 线任何产物、不读任何已有框架、不读旧选必二材料。
- 直接列：要让我看完两份 Word 后能上考场写新题，文档必须正面回答的问题清单。

---

## A. "看到材料 → 想到规则" 类问题（共 12 条）

A1. 看到一段民事案例，第一眼怎么判断它是哪个领域？（合同 / 侵权 / 物权 / 婚姻继承 / 知产 / 劳动 / 消费者 / 诉讼）
A2. "民事关系"和"行政关系""刑事关系"在题面里有什么硬触发词区分？（写错领域全题清零）
A3. 出现"夫妻共同""父母子女""继承""遗嘱"——分别会触发哪几条具体规则？
A4. 出现"合同"——是缔约前 / 履行中 / 违约 / 解除？四阶段触发的规则不同。
A5. 出现"侵权"——是一般侵权还是特殊侵权（产品 / 高度危险 / 饲养动物 / 网络 / 环境）？特殊侵权直接归责原则不同。
A6. 出现"消费者"——是经营者欺诈、商品质量、服务争议、还是个人信息？四种触发的具体法不同。
A7. 出现"知识产权"——是著作权 / 专利 / 商标 / 商业秘密 / 不正当竞争？侵权要件四不同。
A8. 出现"劳动"——是劳动合同 / 劳动报酬 / 工伤 / 解雇？争议处理途径不同。
A9. 出现"诉讼""调解""仲裁""人民调解"——非诉与诉讼的边界 + 民事诉讼 vs 行政诉讼。
A10. 看到"未成年""限制行为能力""无行为能力"——主体能力如何影响行为效力？
A11. 看到"AIGC""数字人""算法""平台"——新业态侵权 / 数据 / 算法 责任怎么落？
A12. 看到"情节严重""恶意""惩罚性赔偿"——什么情形可主张惩罚性赔偿？倍数？

## B. "想到规则 → 写出得分句" 类问题（共 10 条）

B1. 答主观题第一段叫"知识链段"——它的**最小完整结构**是什么？(要件→规则→效力？还是关系→争点→规则→责任→救济？给出一个能套的模板)
B2. 答主观题第二段叫"功能落点段"——它**不是**写"全面依法治国/法治精神"，那它**是**什么？给出 4 个具体的落点话语模板（法治建设 / 社会经济发展 / 市场秩序 / 新业态规范）。
B3. 知识链段里"要件"指的是什么？给一个民法典 X 条原文 + "要件 1/2/3"分解的样例。
B4. "效力"和"责任"在答案里如何区分？什么情况只写效力（合同有效/无效/可撤销/效力待定），什么情况要写责任（违约/侵权/缔约过失）？
B5. "救济"段——具体救济措施有哪些？（停止侵害、排除妨碍、消除影响、恢复名誉、赔礼道歉、赔偿损失、惩罚性赔偿、返还财产、恢复原状、继续履行、采取补救措施、违约金、定金 → 哪些是合同的、哪些是侵权的）
B6. 题目问"为什么"——必须先答什么，再答什么？（先把法律关系/争点说清楚，再上规则，再落价值，不能上来就口号）
B7. 题目问"意义"——四个落点话语怎么组装？给出一个标准句式：" XX 法律规则的实施，通过 [落点 1]，进而 [落点 2]，对 [社会经济发展 / 法治建设] 起到 [具体作用]。"
B8. 题目问"启示""建议"——主体怎么判（学生 / 经营者 / 平台 / 司法机关 / 立法者）？不同主体话语不一样。
B9. 短答（4-6 分）和长答（10-12 分）字数 / 段落 / 落点数量怎么区分？
B10. 何时必须分点写（1/2/3/分号 / 项），何时连贯写（叙述+评价）？

## C. "硬卡口 / 反例" 类问题（共 8 条）

C1. "消费者权益保护法"必须写全名——只写"消费者法"或"保护法"不给分。还有哪些**法律名精确卡口**？（民法典 vs 民法、电子商务法 vs 电商法、知识产权法 ≠ 著作权法/专利法/商标法）请列至少 6 条。
C2. **法律概念精确卡口**：商业诋毁 vs 滥用诉讼权利 vs 不正当竞争 ——具体怎么区分（题目陷阱里这三个最容易混）？
C3. **责任 / 秩序定位精确卡口**：司法秩序 ≠ 社会秩序 ≠ 市场秩序 ——分别什么时候用？
C4. "不必修三化"具体长什么样？请给 5 个**学生最容易写成必修三的反例段**（把它们标红，告诉学生不要写）：
   - "全面推进依法治国"
   - "弘扬社会主义法治精神"
   - "建设社会主义法治国家"
   - "推进国家治理现代化"
   - "维护社会公平正义"
   ……反例后面紧跟"应该改成什么"。
C5. "不法考化"具体长什么样？给 3 个**超纲的法律细节反例**（学生别钻牛角尖去查的内容）。
C6. "不重复给分"——同一句话重复套两次不能多得分，那同一规则用**不同表达**写两次能得分吗？（举例）
C7. "材料三个条件必须写全"——这是哪一道题的硬卡口？这个原则的迁移是什么？（凡是题目材料给了 N 个条件，必须 N 个全应答）
C8. 常见 FAIL 模式：抄材料原句 / 只写法条原文不结合材料 / 只判断不论证 / 价值口号脱离案例。每种模式给一个反例 + 改写。

## D. "选择题必须分得清楚" 类问题（共 6 条）

D1. 选择题考"制度细节"——给一个"细节级"考点的样例（比如：法定继承人顺位、合同解除权行使期限、专利保护年限、商标注册审查程序）。
D2. 选择题易混肢辨——给一个**完整的 4 选项题**，逐个分析 ABCD 为什么对、为什么错（不只是给答案）。
D3. 选择题超纲反例——出现这种选项要警觉（疑难法学问题、罕见特殊侵权、超出选必二教材的细节）。
D4. 选择题不必修三化反例——选项滑成"价值口号"通常是干扰项（举例）。
D5. 选择题里"程序"类（仲裁程序、诉讼时效、举证责任）和"实体"类（要件、责任、效力）出题方式有什么不同？
D6. 选择题里"特殊侵权"5 类（产品、高度危险、饲养动物、网络、环境/生态）+ 监护人责任、用人单位责任 ——能不能一张表对照"归责原则 / 抗辩事由 / 责任主体"？

## E. "情境分类必须穷尽且不重不漏" 类问题（共 6 条）

E1. 主观题情境二级分类有多少类？类间是不是互斥？我看到一个新题，能不能 30 秒内归类？
E2. 选择题情境二级分类有多少类？同上。
E3. 五域（交易交换 / 身份财产 / 人身权益 / 创新竞争 / 救济公共）划分依据是什么？为什么"消费者"和"合同"放同域而不是和"侵权"放同域？
E4. 边界情境：消费者+网购+平台责任 —— 这种到底归"交易交换"还是"创新竞争"？文档要给出归位规则。
E5. 边界情境：婚姻彩礼返还 + 共同债务 —— 归"身份财产"还是"救济"？
E6. 边界情境：AIGC 侵权 + 数字人格权 —— 归"人身权益"还是"创新竞争"？

## 最低硬伤红线（任意一条命中 → 直接 FAIL，不进入计数）

- L1. 任何一段话学生读完会想"这是废话/正确的废话/政治课原话"。
- L2. 任何一个法律名缩写（如"消法""保护法""电商法"）出现在示范答案里。
- L3. 任何一段示范答案没说"先判断什么 → 再套什么规则 → 再写什么效力/责任/救济 → 再上什么落点"，而是直接堆术语或直接喊口号。
- L4. 五域里两个域的边界没说清楚（边界情境没归位规则）。
- L5. 选择题节点里出现主观题踩分关键词混排。
- L6. 任何一段引用了"评标""阅卷总结""讲评 PPT"等内部词。
- L7. 任何一段引用了 Codex / Claude / Governor / Confucius / pipeline / PASS 等内部词。
