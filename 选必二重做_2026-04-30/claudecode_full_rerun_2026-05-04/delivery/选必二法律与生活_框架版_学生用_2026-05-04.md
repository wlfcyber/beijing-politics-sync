# 写在前面

这本书要解决的不是法律细节背诵，而是**一组判断和处理民事争议的稳定动作**。整本书无论考什么情境，最后都在回答同一组问题。把这个动作链装进脑子里，你看到一个新案例，就能像剥洋葱一样把它处理完。

整本书的总目录只用一句话：**以事实为根据，以法律为准绳，定分止争。**

下面这套结构（一核 · 二线 · 三问 · 四步 · 五域），是我们从北京近三年区模拟卷的真实评分细则倒推出来的，用它套题就够了。

---

# 一核：以事实为根据，以法律为准绳，定分止争

这十二个字是答任何选必二大题的总指挥。看到题，第一动作是问自己：

- 我有没有先把"这件事到底发生了什么"写清楚？（事实）
- 我有没有把"哪条法律规则适用"找对？（法律）
- 我有没有让争议得到一个明确的结果——支持/不支持、撤销/不撤销、有效/无效、谁赔谁不赔？（定分止争）

很多同学失分，不是因为不懂法，而是只写了法没写事，或者只写了事没写法，或者全篇喊"维护社会公平正义"没有真正"定分止争"。

---

# 二线：贯穿全书的两条主线

整本书的所有内容，都被这两条主线串起来。每一道题至少落在其中一条上。

## 二线之一：权利保护与权利边界

民法的灵魂是"权利"。但权利不是"我想干啥就干啥"，每一项权利都同时存在一条边界：边界由别人的同等权利、由公共利益、由法律的强制性规定共同划出。

- 看到"权利"，必须想到"是哪种权利、谁的权利"。
- 看到"我有权"，必须想到"边界在哪里、超过会怎样"。

## 二线之二：法治规则与德治价值

民法既是规则，也是价值。诚信原则、公平原则、公序良俗这些不是空话，它们是规则不够时的兜底线。当你被问"为什么这样判""有什么意义"时，沿着这条线往上抬，就有的写。

但记住反例：意义层不能只喊"维护社会公平正义""推进法治社会"。**它必须先回扣本案处理结果，再往上抬一级。**

# 三问：碰到任何主观题，都要回答这三件事

| 问 | 你要写什么 |
| --- | --- |
| 判什么 / 怎么处理 | 法院应当支持谁、不支持谁；合同有效还是可撤销；责任谁承担；归属归谁；程序怎么走。 |
| 凭什么 | 写出适用的法律名称（要写全：不能把"消费者权益保护法"简写成"保护法"）+ 法律规则的关键词 + 把材料里的事实对应进去。 |
| 有什么意义 | 先把本案处理对当事人/这一类关系的意义讲一句，再上升到行业、市场秩序、社会治理、法治价值。**先回扣，再上升。** |

# 四步：处理任何案例的标准动作

这四步不是答题模板，是民法人在脑子里跑的固定流程。把它内化掉，你写出来的答案才像"判决"。

## 第一步：定主体关系

先问自己：本案有哪几方当事人？他们之间是什么关系？

- 是不是合同当事人？（卖方—买方、平台—消费者、用人单位—劳动者）
- 是不是所有权关系？（所有权人—占有人；相邻不动产权利人）
- 是不是侵权关系？（加害人—受害人）
- 是不是身份关系？（夫妻、父母子女、继承人）

主体定错，后面都是错的。

## 第二步：找争点事实

把材料逐条扫一遍，挑出"双方争什么"的那几句话。**评分细则常常显式要求"3 个事实条件必须写全"才得分**——这意味着你不能漏抄关键事实。

阅卷常见反面：只写法不写事，或事实和理由混在一起。所以这一步要单独写一段。

## 第三步：套法定要件 / 规则条件 / 程序要求

- 实体题：把法律规则的"要件"列清楚，再让事实和要件一一对应。
- 程序题：写明哪个程序（起诉、调解、仲裁、执行）、各程序的关键要素（诉讼请求、事实与理由、证据等）。

## 第四步：落责任 / 效力 / 归属 / 救济 / 价值

最后写"结果"：违约责任承担谁；合同是有效、无效、可撤销、效力待定；财产归属于谁；法院支持/不支持；意义在哪里。

不能跳过这一步，跳过了就不叫"定分止争"。

# 五域：所有法律题归位的五个领域

这五个领域是按"法律关系的客体和保护逻辑"分的，不是按教材目录抄的。每个领域的内部都有共同原理。

## 域一：交易交换域

**领域内涵**：合同 + 消费者 + 格式条款 + 劳务承诺与履行。
**为什么并在一起**：它们共同处理"交易关系中的承诺—履行—信息不对称—权益保护"。消费者法和合同法不是两本独立的书，而是合同法在消费者一端的强化版（针对信息不对称做了倾斜保护）。劳动关系也属于"约定型交易"，只是劳动法对劳动者做了倾斜保护。

### 主干 1：合同生命周期

合同有它的"一生"：成立 → 生效 → 履行 → 变更 → 终止。每个节点都有可能出问题。

#### 节点 1.1 成立（要约 / 承诺 / 意思表示一致）

**看到什么材料**：双方有没有出价、有没有同意。
**想到什么规则**：要约 + 承诺 = 合同成立。
**解决什么争点**：有没有合同？
**写出什么得分句**：原告通过……平台购买……，有要约和承诺，合同成立。

⚠ 反例：不要把"要约邀请"和"要约"混淆。商家在网页上挂出商品价格，多数情况下是"要约邀请"或"要约"看具体语境；课本里通常以网店挂出标价为要约。

#### 节点 1.2 变更（默示续约 / 交易习惯）

**看到什么材料**：一方提出修改条款，另一方只回了"好的""同意"，没把所有细节再约定一遍。
**想到什么规则**：默示方式可以续约或变更合同，交易习惯可以补充合同未明示的内容。
**解决什么争点**：原合同条款是否还适用？谁拒绝继续履行就违约？
**写出什么得分句**：……以默示方式延续此前的交易习惯，续签协议；……拒绝承担费用，并未与对方协商一致，属于单方变更合同行为，违反全面履行原则，构成违约。

#### 节点 1.3 全面履行 + 诚信原则 + 公平原则

**看到什么材料**：一方部分不履行；一方擅自变更条件；一方拖延或额外加重对方负担。
**想到什么规则**：全面履行原则 / 诚信原则 / 公平原则。
**解决什么争点**：违约行为如何定性。
**写出什么得分句**：……应按原合同内容全面履行，单方拒绝违背诚信原则；构成违约，应承担违约责任。

⚠ 反例：在一道"合同条款建议题"中，"诚信履行""全面履行""意思表示一致"都不能直接当建议写——它们是履行原则，不是合同设计建议。该写的是"合同内容应清晰明确"。

### 主干 2：合同效力瑕疵

合同成立 ≠ 生效。生效之后还可能因为欺诈、胁迫、显失公平等被撤销或确认无效。

#### 节点 2.1 欺诈 → 可撤销

**看到什么材料**：一方故意隐瞒或捏造（搭售而不显示、隐瞒重大瑕疵、虚假承诺），让对方在不真实意思下做出决定。
**想到什么规则**：欺诈的一方主观有故意，客观实施了欺诈行为，使另一方违背真实意思表示订立合同 → 受欺诈方可以请求撤销合同。
**解决什么争点**：合同是否可被撤销。
**写出什么得分句**：A 公司主观上具备欺诈故意，客观上实施了欺诈行为；……（把材料里 3 条事实写齐：搭售/界面看不清/无法拒绝）；使……在违背真实意思表示的情况下订立合同；根据民法典规定，可请求人民法院撤销合同。

⚠ 反例：把"欺诈"写成"诈骗"——诈骗是刑法概念。把"侵犯了……"写成"侵犯隐私权"——这是张冠李戴。

#### 节点 2.2 格式条款（待补题源）

> 此节点目前没有由本线独立读到强证据细则，暂作"框架槽位"保留。等后续真实题源补齐后再写细则化得分句。

### 主干 3：消费者权益（与合同欺诈并轨）

合同主体是消费者时，消保法替合同法做了"倾斜保护增量"。看到经营者以服务/商品交易方式损害消费者，必须双轨同时启动。

#### 节点 3.1 三大权利

**看到什么材料**：消费者无法清晰知悉费用 / 不能拒绝某些项目 / 被搭售。
**想到什么规则**：知情权（必须清晰知悉）、公平交易权（不能被搭售或不公平条款）、自主选择权（必须能选择是否购买）。
**写出什么得分句**：A 公司提供服务有欺诈行为，侵犯了消费者的知情权、公平交易权和自主选择权。

⚠ 反例：不要写"消费者同意权""消费者隐私权"——前者不是法定权利，后者不是消保法的核心三权。

#### 节点 3.2 三倍赔偿（惩罚性赔偿）

**看到什么材料**：经营者欺诈消费者。
**想到什么规则**：根据《消费者权益保护法》（**法律名必须写全，写"保护法"不给分**），经营者提供商品或服务有欺诈行为的，应按消费者要求的价款的三倍赔偿，最低 500 元。
**写出什么得分句**：根据《消费者权益保护法》，因 A 公司提供服务有欺诈行为，法院应支持……要求三倍赔偿的诉讼请求。

### 主干 4：劳务交易（劳动关系）

劳动关系是"约定型交易+倾斜保护"。看到劳动者—用人单位—政府三角，要按"主体—权利或义务—理论依据"的格式各写一条。

#### 节点 4.1 劳动者的主要权利

**核心权利**：取得劳动报酬权、休息休假权、接受职业技能培训的权利、依法享有社会保险和福利。
**对应义务（劳动者一侧）**：提升劳动职业技能（义务），权利与义务相一致。
**得分句**：劳动者应当不断提高职业技能，提升职业技能是劳动者的义务，权利与义务相辅相成、不可分割。

#### 节点 4.2 用人单位的主要义务

- 守法经营 / 诚信经营 → 保护劳动者合法权益（不拖欠工资、不非法延长工时）。
- 全面履行劳动合同。
- 为劳动者提供职业技能培训。

#### 节点 4.3 主体不能错配

⚠ 反例：政府"完善劳动关系认定标准"是错的（应该由立法机关或政府授权的劳动行政部门）；企业"制定劳动关系认定标准"是错的；"司法机关提供法律援助"也错配（法律援助是司法行政或法律援助机构）。

## 域二：身份财产域

**领域内涵**：物权（所有权 + 用益物权 + 担保物权）、相邻关系、婚姻、家庭、继承、遗赠扶养。
**为什么并在一起**：它们共同处理"基于身份/物的归属与排他/分配"——核心是"谁享有、谁排除他人、怎么分"。

> 本生产线 B 在本会话独立读到的强证据，主要落在"所有权识别 / 用益物权 / 相邻关系 / 不动产物权变动"这四个选择题节点。婚家继的细则未由本线在本会话独立读到，列为框架槽位。

### 主干 1：所有权及其权能（选择题高频识别点）

- 所有权 = 占有 + 使用 + 收益 + 处分（四项权能）。
- 看到"赃物被警察扣押→所有权能否对抗善意取得"：涉及**所有权归属**。
- 看到"房产支付 80% 后又被卖给他人→不动产物权变动以登记为准"：涉及**所有权归属**。

### 主干 2：用益物权（不是所有权）

- 农村土地承包经营权属于用益物权，土地所有权一直在集体。承包人去世后承包经营权能否继承，是**用益物权归属**问题，不是"土地所有权归属"问题。

### 主干 3：相邻关系（不是侵权，不是所有权）

- "光污染对 15 米外住宅"是相邻权侵害，处理路径是排除妨害+赔偿；定性上不是"所有权归属争议"。

### 主干 4：婚家继（待补题源）

> 婚姻成立条件、夫妻共同财产 vs 个人财产、法定继承 vs 遗嘱继承 vs 遗赠扶养协议——这三大节点是选必二教材的核心，但本会话尚未由本线独立读到细则化得分句。后续二轮回到题源补齐后再写。

## 域三：人身权益域

**领域内涵**：民事行为能力、人格权、一般侵权、特殊侵权。
**为什么并在一起**：它们共同处理"对自然人人身/人格的损害与救济"——核心是"自然人作为权利主体，他的人身和人格如何被尊重和保护"。

### 主干 1：民事行为能力（选择题高频）

- 限制民事行为能力人（8 周岁以上未成年人 / 不能完全辨认自己行为的成年人）实施法律行为，原则上需法定代理人代理或事后追认。
- 两类例外：① **纯获利益**的法律行为；② 与其**智力、精神健康状况相适应**的法律行为（如小额日常消费）。
- 选择题命题人最爱设的陷阱是"让你以为它当然有效或当然无效"——其实是"效力待定"。

### 主干 2：人格权（待补题源）

> 生命健康权、姓名权、肖像权、名誉权、隐私权和个人信息保护——这是当下命题热点（特别是 AI、平台、大数据时代），但本会话尚未由本线独立读到细则化得分句。

### 主干 3：一般侵权 / 特殊侵权（待补题源）

> 一般侵权遵循过错责任，特殊侵权（产品责任/饲养动物/高空抛物/学校责任）有专门规则。本会话尚未由本线独立读到细则化得分句。

## 域四：创新竞争域

**领域内涵**：知识产权（专利、商标、著作权）+ 反不正当竞争。
**为什么并在一起**：它们共同处理"以智力成果/竞争行为为客体的法益与边界"——保护的是"无形的成果"和"市场上的诚实竞争"。

### 主干 1：实用新型专利（选择题高频）

- 实用新型专利由国家知识产权局授权。
- **法定保护期**届满后，自动进入公共领域，任何人都可以免费使用。
- ⚠ 反例：① 把专利和商业秘密混为一谈（"保密就一直保护"是商业秘密的逻辑，不是专利）；② 把实用新型和外观设计混淆（"美观效果"是外观设计）；③ 误以为"权利人授权就行"，忽略了保护期届满后的公共领域。

### 主干 2：著作权 / 商标权（待补题源）

### 主干 3：反不正当竞争——反例提醒

⚠ 在合同欺诈题中，把"欺诈消费者"错写成"反不正当竞争"是常见失分点。反不正当竞争法规制的是经营者之间的不正当竞争行为（如商业贿赂、虚假宣传、商业诋毁、混淆商品来源等），不是消费者—经营者之间的合同欺诈。

## 域五：救济公共域

**领域内涵**：调解 + 仲裁 + 诉讼 + 举证 + 执行 + 公益保护 + 生态责任。
**为什么并在一起**：它们共同处理"争议进入程序后的处理路径与公共法益保护"——前面四个域是"实体判断"，本域是"如何让判断落地"。

### 主干 1：起诉状结构（程序高频）

写起诉状必须有四块，缺一不可：

| 模块 | 内容 |
| --- | --- |
| 原告 / 被告 | 你代理谁？写"原告：某某"。错把当事人写反，整个题就跑偏。 |
| 诉讼请求 | 你要法院做什么？例如"判令被告承担全部仓储费用 / 继续履行"和"被告承担本案诉讼费用"。**必须是原告对被告主张的利益，不是反过来。** |
| 事实 | 把案件经过按时间和事件抄清楚。**评分细则要求"时间—约定—默示续约—费用纠纷"等关键事实必须写全，缺一不给分。** |
| 理由 | 用法律语言（不是日常语言）解释为什么对方该承担：违背诚信原则 / 单方变更合同 / 违反全面履行原则 / 构成违约 / 承担违约责任。**评分细则鼓励写两个理由（1 个 2 分，2 个 3 分）。** |

⚠ 反例：① 事实和理由混在一起，没有法律语言；② 不分两个诉讼请求，只论一个；③ 立场不统一（一会儿站原告，一会儿站被告）。

### 主干 2：诉讼请求识别（一题多请求）

如果一道题有两个或更多诉讼请求（如"退还票款"+"三倍赔偿"），必须**逐一**对应法律依据：第一项靠合同欺诈 + 撤销，第二项靠消保法三倍赔偿。**漏一项扣对应分。**

### 主干 3：司法说理与意义层级

法院"为什么支持"类题，答题顺序必须是：

1. **事实**：复述材料中决定本案的关键事实（用法律语言）。
2. **法理**：援引法律名称 + 规则要件 + 把要件套到事实上。
3. **结论**：说出法院应当如何处理（支持哪几项请求、是否撤销合同等）。
4. **意义层**：先回扣本案当事人 / 这一类关系，再上升到行业秩序、社会治理、法治价值。

⚠ 反例：① 全篇喊"维护社会公平正义"没回扣本案；② 把"意义"写在阐述理由之前，导致裁判理由像缺了主干；③ 用"可从……角度作答"这种空话占位（这是评卷术语，不能直接进答案）。

### 主干 4：调解 / 仲裁 / 举证 / 执行 / 公益（待补题源）

> 这些节点是选必二第四单元的主体内容，但本会话尚未由本线独立读到细则化得分句。后续二轮回到题源补齐后再写。

# 选择题专属框架

> 选择题考的是"边界识别"和"基础知识精确度"，**不**与主观题的踩分句混排。

## 选择题判断流程（四步）

1. **审题肢排错误**：先看每个选项里有没有把法律名称、权利名称、机关写错。
2. **审设问明指向**：题目问"涉及……的是""……做法的依据是""……启示我们"——指向不同，正确选项不同。
3. **审材料抓关键**：把材料里"决定法律性质"的关键事实圈出来。
4. **再审题肢定选项**：用关键事实匹配每个选项的法律论断。

## 选择题专属边界识别清单

### 边界一：民事行为能力的"两类例外"

- 限制民事行为能力人 → 原则上需法代代理或追认。
- 例外 1：纯获利益的法律行为（如接受赠与）独立有效。
- 例外 2：与其智力、精神健康状况相适应的法律行为（如买文具）独立有效。
- ⚠ 命题陷阱：把"效力待定"当成"无效"或"有效"。

### 边界二：所有权 vs 用益物权 vs 相邻关系 vs 物权变动登记

- 所有权 = 占有/使用/收益/处分 4 项权能合一。
- 用益物权（土地承包、宅基地使用、地役权等）≠ 所有权。
- 相邻关系（光污染、通风、采光、排水）→ 排除妨害+赔偿，不是所有权归属。
- 不动产物权变动以登记为生效要件，未登记前所有权未转移。

### 边界三：实用新型 vs 外观设计 vs 商业秘密

- 实用新型专利保护"产品的形状、构造或者其结合所提出的适于实用的新的技术方案"。
- 外观设计专利保护"产品的形状、图案、色彩或其结合的富有美感的新设计"。
- 商业秘密以保密换保护，没有法定期限；专利以公开换保护，期限届满进入公共领域。

### 边界四：消保法的"消费者—经营者"关系

- 消保法的适用对象是消费者—经营者之间因生活消费产生的关系。
- 经营者—经营者之间的不正当竞争 → 走反不正当竞争法。
- 注意：把消费者欺诈写成"反不正当竞争"是错配主体关系。

> 本生产线 B 已锁定的选择题情境主要来自 2024 朝阳一模讲评（限制民事行为能力人、所有权归属、实用新型专利）。其他区一模/二模/期末的选择题情境将在第二轮独立扫描后补入本节点。

# 答题前的最后自检（凡是大题动笔前默念一遍）

1. 我有没有先写出"主体—关系"（原告/被告，劳动者/用人单位等）？
2. 材料里"决定本案"的事实，我有没有抄全？（评分细则常常显式要求 3 个事实条件全才得分）
3. 我有没有写出适用的法律名称的全名？（不能写"保护法"，不能写"民法"，要写《消费者权益保护法》《民法典》《劳动法》等）
4. 我有没有把要件套到事实上、得出"违约/可撤销/支持/不支持"的明确结论？
5. 如果有多个诉讼请求或多个争点，我是不是逐一对应论证了，没有漏？
6. 意义层我是不是先回扣本案再上升一级，没有空喊口号？
7. 我有没有写错权利（隐私权 ≠ 知情权；同意权 ≠ 自主选择权；诈骗 ≠ 欺诈）？
8. 我有没有错配主体（让政府制定细则、让司法机关执法）？
