## B01-Q1：2024 朝阳二模 第17题

- 证据等级：formal_or_scoring_source
- 设问：结合材料，说明仲裁委员会作出上述裁决的理由，并分析该典型案例的意义。
- 法律关系：平台企业与新就业形态劳动者之间是否构成事实劳动关系。
- 争点事实：合同写成“合作关系”，但平台派单、签到、奖惩、费用结算、业务组成等事实显示从属性。
- 法律规则/要件：劳动关系认定核心看人格从属性、经济从属性、组织从属性。
- 处理结果：仲裁支持劳动者解除劳动合同经济补偿请求。
- 细则奖励动作：三个从属性各自结合材料给分，最后归纳存在劳动关系；意义按法律、道德、经济维度分层。
- 命题路径：
  - 命题目标：用事实和要件识别新就业形态劳动关系，培养“以事实为根据、以法律为准绳”的法治思维。
  - 载体选择：平台经济典型案例 + 资料包，适合考“新现象中识别旧规则”。
  - 设问路径：从事实到要件，再到裁决理由和案例意义。
  - 细则落点：要件匹配是理由分，价值分层是意义分。
- 框架归位：合同消费者劳动 -> 劳动关系认定 -> 要件匹配 -> 责任/程序/意义。
- 对框架的推动：应把“法律关系判断 + 法定要件匹配”升为主干高频层。

## B01-Q2：2024 海淀一模 第19题

- 证据等级：formal_or_scoring_source（纠正：预处理表曾误标 reference_answer）
- 设问：运用《法律与生活》知识，谈谈对本案判决的认识。
- 法律关系：著作权人与未经许可使用者/竞争者之间的侵权与不正当竞争关系。
- 争点事实：虚拟数字人相关视频是否受著作权保护；乙公司替换标识、添加营销信息和商标是否构成侵权及不正当竞争。
- 处理结果：法院判令乙公司消除影响并赔偿经济损失。
- 细则奖励动作：著作权保护 -> 未经许可使用构成侵权 -> 虚假/引人误解商业宣传构成不正当竞争 -> 保护创新和市场秩序；细则另有层次化等级描述。
- 命题路径：
  - 命题目标：在新技术场景中识别权利保护边界和市场竞争秩序。
  - 载体选择：虚拟数字人 / AI 技术案例，贴合新质生产力和知识产权保护。
  - 设问路径：从新现象到权利保护，再到市场秩序意义。
  - 细则落点：已找到本地细则层次表，可作为 formal/scoring 证据；但“虚拟数字人/AI”仍只是单题载体，不能据此升为高频主干。
- 框架归位：知识产权不正当竞争 -> 权利保护与边界 -> 责任/价值收束。
- 对框架的推动：可支持“知识产权/不正当竞争”进入正式覆盖索引；但“新技术/虚拟数字人”仍只作开放容器，不能凭单题升主干。

## B01-Q3：2024 海淀二模 第19题

- 证据等级：formal_or_scoring_source
- 设问：运用《法律与生活》知识，分析本案判决的社会价值。
- 法律关系：遗赠扶养协议当事人与法定继承人之间的遗产归属争议。
- 争点事实：遗赠扶养协议是否有效，居委会是否履行扶养义务，子女继承主张是否应被支持。
- 法律规则/要件：有效遗赠扶养协议 + 扶养义务履行 -> 可按协议处理遗产。
- 裁判结果：确认协议有效，判定房屋、股权、现金、存款归居委会所有。
- 细则奖励动作：确认居委会提供协议和赡养行为；个案公平正义；倡导社会助力养老；弘扬尊老敬老传统美德和社会主义核心价值观。
- 命题路径：
  - 命题目标：用继承/扶养制度说明法律裁判如何实现公平正义和德治价值。
  - 载体选择：养老与遗赠扶养协议案例，天然连接民法制度和社会价值。
  - 设问路径：从具体争议到裁判确认，再到社会价值。
  - 细则落点：不是复杂继承份额计算，而是“法律确认 + 价值收束”。
- 框架归位：物权相邻继承家庭 -> 继承/扶养 -> 裁判结果 -> 社会价值。
- 对框架的推动：应设置“具体法律关系锚定后的价值收束”主干，但禁止脱离案件直接写宏观价值。
