# 第一册 思维宝典

## 一、科学思维

### 1. 客观性

- 触发链：调查 / 数据 / 真实场景 / 蹲点观察 / 立足实际 → 科学思维 → 客观性。
- 答案句模板：科学思维追求认识的客观性。<本题材料事实(从老人真实生活场景出发/蹲点观察/抽样全面真实准确)> 表明该<主体>立足实际、依据客观事实展开，避免主观臆断。
- 易错：把蹲点观察等于经验主义；把材料中的‘数据’默认为‘量变积累’而漏掉客观性这一科学思维特征。

挂题：
- **2026顺义一模 Q19(2)** ｜ 材料：企业蹲点观察老人 “安全照护/健康提醒/情感陪伴” 真实生活习惯。｜ 答：科学思维追求认识的客观性，企业从老人真实生活场景与实际需求出发，深入调研、蹲点观察，立足实际设计产品功能。
- **2024海淀二模 Q17(1)** ｜ 材料：国家时间利用调查抽样全面、统计真实准确。｜ 答：科学思维追求认识的客观性，调查抽样全面、统计真实准确，确保结论建立在事实基础上。【挂载带 BLK-001 标注，Codex 回源核验】
- **2026海淀一模 Q17(2) 角度一** ｜ 材料：调研活动从实际出发设计选项。｜ 答：科学思维追求认识的客观性，要求调研从居民真实需求出发、按事实而非主观想象设计选项。

### 2. 预见性

- 触发链：趋势研判 / 未来布局 / 提前规划 → 科学思维 → 预见性 ；或重叠到超前思维节点。
- 答案句模板：科学思维结果具有预见性。<本题材料事实(企业研判老龄化与具身智能趋势/某市预测消费趋势)> 表明该<主体>把握发展趋势，预见<潜在空间/市场潜力>。
- 易错：把市场潜力研判默认成 “创新思维”；把 “趋势” 写成 “发展观点” 忘掉预见性。

挂题：
- **2026顺义一模 Q19(2)** ｜ 材料：研判老龄化、具身智能趋势，预测机器人在养老服务中应用空间。｜ 答：科学思维结果具有预见性，企业追踪具身智能与人口老龄化发展趋势，预测具身机器人在养老服务中具备广阔的应用空间和持续增长的市场潜力。

### 3. 可检验性

- 触发链：反复测试 / 多轮迭代 / 验证修正 → 科学思维 → 可检验性。
- 答案句模板：科学思维结果具有可检验性。<本题材料事实(多轮测试/用户反馈/数据比对)> 表明该<主体>用实践与试验验证结论并优化方案。
- 易错：把 “多轮迭代” 默认成 “辩证否定”；忘了科学思维的可检验性。

挂题：
- **2026顺义一模 Q19(2)** ｜ 材料：经过多轮应用测试、打磨与迭代。｜ 答：科学思维结果具有可检验性，企业产品反复测试、多轮迭代，以用户试用与实践效果不断优化调整。
- **2026海淀一模 Q17(2) 角度一** ｜ 材料：调研结束后通过实地测量、对比不同人群回答或与相关部门数据比对。｜ 答：调研结论需经过实地测量、跨群体对比与权威数据比对，体现科学思维结果的可检验性。

### 4. 探索性 / 方法更新（来源支持时）

- 触发链：思路新、方法新、调查工具升级 → 科学思维 → 探索性 / 方法更新。
- 挂题：**2024海淀二模 Q17(1)**【BLK-001 待 Codex 回源核验】。

### 5. 整体安排（来源支持时）

- 触发链：分阶段、有步骤、任务明确 → 科学思维 → 整体安排。
- 挂题：**2024海淀二模 Q17(1)**【BLK-001 待 Codex 回源核验】。

### 6. 超前思维（科学思维子节点）

- 触发链：未来布局 / 提前规划 / 前瞻研判 → 超前思维 → 调查研究 / 矛盾分析 / 推理与想象。
- 易错：把超前思维写成 “未来思维 / 前瞻思维” 等非术语。
- 挂题：
  - **2026海淀期末 Q20(2)**：超前思维三件套（调查研究 + 矛盾分析 + 推理与想象）。
  - **2026朝阳期中 Q21(2)**：超前思维（预测消费趋势从工业城市转网红旅游）作为创新思维下属。

## 二、辩证思维

### 1. 分析与综合 / 整体性 / 系统优化

- 触发链：多主体 / 多环节 / 协同治理 / 一盘棋 → 辩证思维 → 分析与综合 / 整体性。
- 易错：只答 “分析” 或 “综合” 中的一边（最高 1 分）。
- 挂题：
  - **2025海淀二模 Q20** ｜ 共建共享、做大蛋糕分好蛋糕、主体/领域/权利全面 → 分析与综合 / 整体性 / 系统优化。
  - **2026朝阳期中 Q20** ｜ 分析机遇因素与风险因素，再综合考量。

### 2. 质量互变 / 动态性

- 触发链：循序渐进 / 分阶段 / 由低到高 → 辩证思维 → 质量互变（也可写动态性）。
- 易错：写成 “量变就是好的”，丢失飞跃；与动态性混在一处不区分。
- 挂题：
  - **2025海淀二模 Q20** ｜ 共富循序渐进、先富帮后富。
  - **2026朝阳期中 Q20** ｜ 科研持续投入实现技术突破质变、循序渐进化解外部冲击。

### 3. 动态性 / 发展观

- 触发链：环境不断变化 / 顺应科技革命趋势 → 辩证思维 → 动态性。
- 挂题：
  - **2026朝阳期中 Q20** ｜ 顺应科技革命动态趋势，把潜在机遇变现实动力。

### 4. 辩证否定

- 触发链：扬弃 / 否定之否定 / 保留与克服 / 不是平均主义但仍是共富 → 辩证思维 → 辩证否定。
- 挂题：
  - **2025海淀二模 Q20** ｜ 共同富裕 “不是整齐划一的平均主义” 是对旧富裕观的扬弃。
  - **2026朝阳期中 Q20** ｜ 矛盾分析法可替换为辩证否定观。

### 5. 矛盾分析法（辩证思维节点）

- 触发链：机遇与挑战 / 危与机 / 对立统一 → 辩证思维 → 矛盾分析法。
- 挂题：
  - **2026朝阳期中 Q20** ｜ 机遇与挑战对立统一。
  - **2026海淀一模 Q17(2) 角度二** ｜ 对比不同群体回答差异 → 矛盾分析法。

### 6. 全面观点

- 触发链：综合多法/多角度 → 辩证思维 → 全面观点。
- 挂题：
  - **2026海淀一模 Q17(2) 角度二** ｜ 综合运用观察 / 访谈 / 文献 / 问卷四法。

# 第二册 推理宝典

## 一、形式逻辑基本规律

### 1. 同一律

- 口令：同一思维过程中概念与判断的外延必须保持同一；否则犯偷换概念 / 偷换论题。
- 挂题：**2026西城一模 Q19(3) 丙**｜“村民会议” 与 “村委会” 偷换概念，违反同一律。

### 2. 矛盾律

- 口令：相互矛盾或反对的判断不能同真，必有一假。联言判断要求所有联言肢都真。
- 挂题：
  - **2026东城期末 Q17(2) 主张1** ｜ 完整保留 vs 拆除改造 联言不能同真 → 违矛盾律。
  - **2026东城期末 Q7** ｜ 四老师发言矛盾律真假定位。
  - **2024朝阳一模 Q6 C 选项** ｜ 全称肯定与特称否定矛盾不能同真。

### 3. 排中律

- 口令：相互矛盾的判断不能同假，必有一真。
- 挂题：**2024朝阳一模 Q6 B 选项**（正例）。

### 4. 概念划分规则

- 口令：每次划分按同一标准 / 不交叉 / 不越级。
- 挂题：
  - **2024朝阳一模 Q6 A 选项** ｜ 越级划分。
  - **2026海淀一模 Q17(1)** ｜ 划分标准不一致。

### 5. 选言判断完整性

- 口令：选言肢必须穷尽；遗漏即犯逻辑错误。
- 挂题：**2026海淀一模 Q17(1)** ｜ 选项遗漏其他有选择价值的情况。

### 6. 对当关系 — 反对关系 vs 矛盾关系

- 口令：反对关系可同假不能同真；矛盾关系必有一真一假。
- 挂题：**2026西城一模 Q19(3) 乙**｜全称肯定 vs 全称否定 = 反对关系（正例）。

## 二、演绎推理 — 三段论

### 1. 演绎保真双条件

- 口令：① 前提真实；② 结构合规（三段三概念、中项至少周延一次、不大/小项扩大、不双否）。
- 挂题：**2026顺义一模 Q19(1)**｜量子科技推理：前提 “所有具备前瞻/战略/颠覆特点的产业都是未来产业” 不真，结论不保真。

### 2. 三项三概念 / 四概念错误

- 挂题：
  - **2024朝阳一模 Q6 D 选项** ｜ “老舍的作品” 外延不一致 → 四概念。
  - **2026西城一模 Q19(3) 甲** ｜ “公共利益” 外延被窄化 → 四概念。

### 3. 中项周延规则

- 口令：中项至少周延一次。
- 挂题：
  - **2026东城期末 Q17(2) 主张3**｜中项 “该模式制定的” 大小前提皆不周延。
  - **2026东城期末 Q6 选择题** ｜ 补大前提保真型，正确答 B；其他选项犯中项不周或大项不当扩大。

### 4. 大项 / 小项不当扩大

- 口令：前提中不周延的项在结论中也不得周延。
- 挂题：
  - **2026丰台一模 Q18(2) 乙** ｜ 大项 “带动居民消费” 不当扩大。
  - **2024朝阳二模 Q7**（旧硬样本：小项不当扩大）【BLK-002】。
  - **2025顺义一模 Q7**（旧硬样本：大项不当扩大 + 谬误名称纠错）【BLK-003】。

### 1. 充分条件假言推理（若 p 则 q）

- 无效式：肯后肯前、否前否后。
- 挂题：

### 2. 必要条件假言推理（只有 p 才 q）

- 无效式：肯前肯后、否后否前。
- 挂题：
