## 如何使用本稿

先从材料动作进，不要先背题号。看到“蹲点观察、真实需求、测试迭代”，先翻科学思维三性；看到“共建共享、渐进推进、扬弃旧偏差”，先翻辩证思维复合题；看到“跨界组合、冷资源转热经济、多方案再聚焦”，先翻创新思维；看到“具体经验抽出理念再回到完整图景”，先翻思维抽象与思维具体。

再用知识点反查。复习科学思维客观性时，同时看 Q0001 和 Q0011；复习创新思维三新时，不能只看 Q0003，还要看 Q0011 的 3 分创新模块；复习辩证整体性时，也要回看 Q0011 的 2 分辩证模块。

选择题按“选项为什么诱人、为什么错”读。Q0004 重点排“类比/系统整合”误判，Q0017 重点排“新机制=创新思维”的过度推断。本稿仍是 review draft，未经过 GPT Pro 真实外审，不能直接当终稿发学生。

## 一、科学思维

### 1. 三性同题：客观性、预见性、可检验性

样本：`Q0001 2026顺义一模 Q19(2)`

触发口诀：

- 看到“真实生活场景、蹲点观察、从实际需求出发” -> 客观性。
- 看到“趋势、未来应用空间、市场潜力、提前预判” -> 预见性。
- 看到“测试、试用、实践效果、迭代优化” -> 可检验性。

答题模板：

科学思维追求认识的客观性，材料中的主体从真实场景和实际需求出发开展调研、设计方案；科学思维结果具有预见性，主体根据发展趋势对未来应用空间作出预测；科学思维结果具有可检验性，主体通过测试、试用、实践反馈不断修正方案。

常见误判：

- 把趋势预测写成创新思维。
- 把反复迭代写成辩证否定，漏掉“可检验性”。
- 只写“科学思维具有客观性”，不把材料动作落到每一层。

### 2. 科学思维总帽下的三模块复合题

样本：`Q0011 2024海淀二模 Q17(1)`

分值结构：

- 科学思维角度 2 分：客观性，或结果具有预见性。
- 创新思维角度 3 分：三新，或发散思维、聚合思维、超前思维。
- 辩证思维角度 2 分：整体性，或分析与综合、整体部分、系统优化、矛盾分析法。

① 材料动作：

- “全面、真实、准确了解” -> 客观性。
- “调查思路新、方法新、范围对象内容更丰富、创造性解决问题” -> 创新思维三新/发散聚合/超前。
- “总体安排、分阶段实施、任务明确” -> 辩证思维整体性/分析与综合。

② 为什么能联想到这些方法：

设问虽然写“体现科学思维”，但正式细则不是单角度科学思维。阅卷把材料分成三组动作：真实准确的调查目的对应科学思维；新思路、新方法、新工具和创造性解决问题对应创新思维；分阶段、有整体安排对应辩证思维。学生答题时要跟随细则的材料分组，不要把所有动作硬塞回“科学思维”一个帽子下。

③ 卷面答案句：

此次时间利用调查是为了全面、真实、准确了解我国居民时间利用情况，追求认识的客观性；与前两次调查相比，调查思路新、方法新，调查范围、对象、内容更加丰富，并创造性地解决问题，体现创新思维；调查工作具有整体性，分阶段实施、任务明确，体现辩证思维的整体性和分析综合。

④ 易错与边界：

本题不能再按“科学思维单角度”处理。学生最容易犯四个写作错误：第一，只写客观性、预见性、可检验性等科学思维三性，会漏掉创新和辩证两层；第二，写“调查更科学”但不写“思路新、方法新、范围对象内容更丰富、创造性解决问题”，会漏掉创新思维 3 分模块；第三，写“有步骤推进”但不点整体性、分阶段实施、任务明确，会漏掉辩证思维 2 分模块；第四，把整体性随便替换成不贴材料的辩证术语，会削弱材料对应。原先“不要扩写成三模块并列”的说法已被 Claude 外审 V0 和正式细则共同否定。

## 二、辩证思维

### 1. 共同富裕复合题

样本：`Q0002 2025海淀二模 Q20`

触发口诀：

- “主体全面、领域全面、权利全面、共建共享、做大蛋糕+分好蛋糕” -> 分析与综合 / 整体性 / 系统优化。
- “循序渐进、分阶段、有步骤、从低级到高级、从不均衡到均衡” -> 质量互变 / 动态性。
- “不是平均主义、不是少数人富裕、继承与发展、保留与克服” -> 辩证否定。

答题模板：

推进共同富裕要运用分析与综合，统筹共建与共享、做大蛋糕与分好蛋糕、各主体各领域各权益的全面共享；要遵循质量互变规律，在分阶段、循序渐进的量的积累中推动共享水平提升；要坚持辩证否定，克服平均主义和两极分化，保留共同富裕的价值指向，实现扬弃式发展。

阅卷提醒：

只答“分析”或只答“综合”，不能吃满分析与综合这一层。矛盾分析法可以给分但不是本题最优主链。

### 2. 机遇与挑战复合题

样本：`Q0014 2026朝阳期中 Q20`

触发口诀：

- “机遇与挑战并存、风险中有契机” -> 矛盾分析法 / 辩证否定。
- “分析促成新机遇的因素和动荡变革期的风险，再综合考量” -> 分析与综合 / 整体性。
- “持续投入、脚踏实地积累、循序渐进化解冲击” -> 质量互变。
- “科技革命趋势、国际环境变化、不确定中应变” -> 动态性。

答题模板：

面对机遇与挑战，要坚持矛盾分析法，在机遇中看到风险、在挑战中挖掘契机；要运用分析与综合，把新机遇和风险因素分开分析后整体谋划；要遵循质量互变规律，通过持续投入和阶段性推进实现突破；要用动态性思维看问题，顺应科技革命和国际环境变化，把潜在机遇转化为现实发展动力。

### 3. 复合题指针：科学总帽下的辩证模块

样本：`Q0011 2024海淀二模 Q17(1)`，完整分析见 §一.2。

本道题虽然设问写“体现科学思维”，但细则给了辩证思维角度 2 分：调查工作具有整体性，分阶段实施，并且每个阶段任务明确。学生从辩证思维章节复习时，必须知道这类题不一定在题干中明说“辩证思维”，要看细则和材料动作是否把整体安排、分阶段实施、分析与综合单独给分。

## 三、创新思维

### 1. 文旅经济复合题

样本：`Q0003 2026朝阳期中 Q21(2)`

触发口诀：

- “预判消费趋势、城市发展理念转变” -> 超前思维。
- “冰雪+音乐+文化叙事、跨界联结” -> 联想思维。
- “冷资源转热经济、静态建筑转动态场景” -> 逆向思维 / 转换性思考。
- “多角度挖掘，再聚焦融合模式街区” -> 发散思维与聚合思维。
- “新思路、新方法、结果新、沉浸式复合空间” -> 三新。

满分安全线：

只写“三新+简单材料”不够稳。细则明确，满分需要三新之外至少两个创新解决问题的方法。本道题的主链应写“超前、联想、逆向/转换、发散聚合”，三新作统摄或补充。

### 2. 复合题指针：科学总帽下的创新模块

样本：`Q0011 2024海淀二模 Q17(1)`，完整分析见 §一.2。

本道题的创新思维角度给 3 分：调查思路新、方法新，调查范围、对象、内容更加丰富，并且创造性解决问题。可用三新，也可用发散思维、聚合思维或超前思维。学生从创新思维章节复习时，要把这题当作“设问总帽不等于给分小帽”的典型例子。

### 3. 联想思维设计题：迁移与想象

样本：`Q0019 2026朝阳一模 Q17(1) 后半`

触发口诀：

- “运用联想思维的具体方法提出设计方案” -> 先点方法，再落方案。
- “把博物馆、铁路、历史、科技元素做成文创产品” -> 迁移。
- “围绕铁路文化构造新产品形态、使用场景、视觉符号” -> 想象。

答题安全线：

方法和方案必须一致，且方案要含中国铁路相关元素。只写抽象方法、不写具体文创产品，或者只写产品、不点迁移/想象，都不能吃满分。

## 四、超前思维

### 1. 城市书房建设运营题

样本：`Q0021 2026海淀期末 Q20(2)`

触发口诀：

- “广泛深入了解居民需求、阅读内容、服务需求” -> 调查研究。
- “建设和运营成本与居民需求之间的矛盾” -> 矛盾分析方法。
- “结合人口结构变化，对未来功能进行规划” -> 推理和想象。

答题模板：

城市书房建设要运用超前思维的方法，先通过调查研究把握居民真实阅读和服务需求；再运用超前思维的矛盾分析方法，前瞻性识别建设运营成本与居民需求之间的潜在张力；还要通过推理和想象，结合人口结构和阅读方式变化，对未来功能作出前瞻性规划。

常见误判：

- 把“城市书房”写成文化生活或公共服务，不落选必三方法。
- 只写“要超前规划”，不写调查研究、矛盾分析、推理和想象。
- 将“矛盾分析”误并入必修四哲学答法或普通辩证思维答法，漏掉本题要求的超前思维方法链。

## 五、调研改进题：多方法触发

### 1. 问卷和调研方案改进

样本：`Q0023 2026海淀一模 Q17(2)`

触发口诀：

- “从实际出发、了解真实问题和需求” -> 科学思维客观性。
- “实地测量、数据比对、实践检验” -> 科学思维可检验性。
- “提高不完全归纳可靠程度、寻找因果联系、求异法” -> 逻辑思维规则。
- “观察、访谈、文献、问卷综合使用” -> 辩证思维全面观点。
- “先广泛收集意见，再聚焦核心问题” -> 发散思维与聚合思维。

答题策略：

本题不是让学生把四类思维全写一遍，而是任选两个角度，每个角度都要做到“具体建议 + 对应知识”。最稳写法是先抓材料中的调研问题，再选择两条最容易匹配的知识链。

卷面示范：

- 科学思维角度：调研方案应从真实存在的问题和需求出发设计选项，并通过实地测量、不同人群回答对比或相关部门数据比对，检验调研结论是否可靠。
- 逻辑规则角度：提高不完全归纳推理结论的可靠程度，要扩大样本、完善选项，并在认识对象与有关现象之间寻找因果联系，避免由片面样本推出片面结论。

## 六、思维抽象与思维具体

### 1. 选择题顺序陷阱

样本：`Q0004 2026通州期末 Q11`

触发口诀：

- 材料先给具体经验对象或杂多现象 -> 感性具体。
- 从现象中抽出核心理念、概念、结构 -> 思维抽象。
- 再回到包含多层规定的整体对象 -> 思维具体。

本道题：

大运河“河—合—和”理念不是类比推理，也不是简单“感性到理性”。它是从运河具体经验出发，抽出“河/合/和”的理念结构，再回到文化之河、生态之河、发展之河、民生之河的整体诠释，所以选 C。

选项原文与逐项解释：

- A. 实现了对运河文化从感性认识到理性认识的迁移和飞跃。错在只停到“感性到理性”，没有体现“再回到多层规定的思维具体”。
- B. 展现了从分析到综合、从普遍一特殊一普遍的思维进阶。错在把理念诠释误挂为分析综合和普遍特殊往返，不能解释“河—合—和”抽象后再具体展开。
- C. 体现了从感性具体到思维抽象再到思维具体的发展过程。正确，符合从大运河具体形态抽出理念，再回到文化之河、生态之河、发展之河、民生之河的链条。
- D. 呈现了对“水脉一文脉一人脉”进行类比推理与系统整合。错在材料不是由相似性推出新结论的类比推理，也不能只用系统整合替代思维抽象到思维具体。“水脉、文脉、人脉”是平行隐喻和理念展开，不构成“由两类对象在若干属性相似推出其他属性相似”的类比推理结构。

选择题排雷：

- 见“融合、整合”不一定就是系统优化。
- 见“由此及彼”才考虑类比；本题没有用相似性推出新结论。
- 见“理念诠释过程”要优先检查是不是“感性具体 -> 思维抽象 -> 思维具体”。

## 七、选择题边界：治理事实不等于思维方法

样本：`Q0017 2026通州期末 Q9`

触发口诀：

- 材料说“打通数据壁垒、一站式结算、资源整合、形成新机制” -> 系统化、数字化治理模式。
- 只有材料出现“多方向展开、多方案发散再收束”，才考虑发散/聚合。
- 只有材料要求“如何创新性解决问题”，才优先调创新思维；单纯新机制不自动等于创新思维。

本道题：

正确选 D。B、C 的陷阱是把现实治理流程误挂成“创新思维”“发散思维”。这类题的安全策略是先判断材料有没有明确的思维动作；没有，就按治理事实本身选择。

选项原文与逐项解释：

- A. 实现医保与商保数据共享，完善我国基本医疗保障体系。错在材料是基本医保与商业健康保险的结算协同，不是单纯完善基本医疗保障体系。
- B. 发挥创新思维保障民生，优先构建数据共享和服务标准。错在把数据打通和结算机制建设误挂成创新思维。材料虽有新机制，但没有多方案展开的发散、反向转换的逆向、跨对象移用的联想，也没有未来趋势预判的超前思维动作。
- C. 通过发散思维拓展医疗数据的应用场景，转化服务要素。错在材料强调“一站式”结算路径和资源整合，不是多方向展开方案的发散思维。
- D. 运用系统化、数字化的模式整合保障资源，形成新机制。正确，抓住“打通数据壁垒”“一站式结算”“数据多跑路，群众少跑腿”的治理机制。
