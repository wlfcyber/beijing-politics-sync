# Framework Skeleton

This is not student final text. It is the framework that both lanes must fill with source-locked questions.

## 思维宝典

### 科学思维

- 客观性：调查、数据、真实场景、事实校验。
- 预见性：趋势研判、未来布局、提前规划。
- 可检验性：测试、迭代、实践验证、反馈修正。
- 来源支持的探索性/方法更新/整体安排：只在细则明确支持时进入。

### 辩证思维

- 整体性：多主体、多环节、全局协同、系统推进。
- 动态性：阶段推进、过程变化、趋势把握。
- 分析与综合：先拆因素、再合整体。
- 质量互变：积累、渐进、阶段性突破。
- 辩证否定：保留合理、克服旧弊、推动升级。
- 矛盾分析法：只在题面和细则均指向选必三辩证思维方法时进入；必修四边界单列。

### 创新思维

- 三新：思路新、方法新、结果新。
- 超前思维：提前预测、提前布局。
- 联想思维：跨对象、跨领域联结。
- 迁移与想象：旧经验/旧资源迁入新场景。
- 发散与聚合：多方案铺开，再聚焦收束。
- 逆向思维：方向、状态、关系反过来想。

### 思维抽象与思维具体

- 固定触发链：杂多现象 -> 抽出核心概念 -> 回到完整整体图景。
- 重点防错：不要误挂为系统整合、分析综合、联想、类比。

## 推理宝典

### 三段论

- 中项至少周延一次。
- 前提中不周延的项，结论中不得周延。
- 大项不当扩大、小项不当扩大、中项不周延、四概念等分开归类。

### 充分条件假言推理

- 有效式：肯前肯后；否后否前。
- 无效式：否前否后；肯后肯前。

### 必要条件假言推理

- 有效式：否前否后；肯后肯前。
- 无效式：肯前肯后；否后否前。

### 选言推理

- 区分相容/不相容选言。
- 明确否定肯定式、肯定否定式适用边界。

### 归纳推理

- 完全归纳、不完全归纳、求同法、求异法、共变法、剩余法。
- 必须说明结论可靠程度和前提覆盖。

### 类比推理

- 相同/相似属性、结论或然性、提高可靠程度的条件。

### 形式逻辑基本规律

- 同一律、矛盾律、排中律。
- 只进入推理宝典，不进入思维方法主链。
