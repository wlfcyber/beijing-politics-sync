## 旧框架的真实结构

旧课件不是单纯按教材章节讲授，而是按“考情引入 -> 法条和意义总判断 -> 民事法律关系/合同/知识产权/侵权 -> 家庭继承 -> 企业场景 -> 多元纠纷解决 -> 题例批注”展开。

核心入口在 p05-p06：这本书考“法条基础 + 法治意义”，学好要做到法条知识扎实、意义角度全面。这个判断是旧框架最值得保留的部分。

法治意义库在 p07-p08：经济、社会、权利、秩序、利益平衡、法治、公平正义、社会主义核心价值观、传统美德、德治法治结合等角度都已有积累。

知识线从 p09 起展开：民事法律关系三要素、权利体系、合同成立/生效/履行、知识产权、合同变更和违约责任、侵权构成和归责、家庭继承、劳动消费经营、多元纠纷解决。

题例和批注主要集中在 p14-p16、p22-p23、p25-p26、p30-p31、p35。这些页面展示了教师讲题时圈主体、圈行为、圈损害、圈因果关系、圈程序角色、圈意义词的习惯，是新版框架转成“材料触发链”的主要依据。

## 旧框架的有效部分

1. 总入口准确：高考难点确实不只是背法条，而是把法条判断和法治意义结合起来。
2. 民事法律关系三要素可迁移：主体、客体、内容能帮助学生先定位“谁与谁、围绕什么、有什么权利义务”。
3. 侵权部分相对成熟：权利、要件、归责、举证、责任方式、边界都有明确链条。
4. 意义角度丰富：既有经济社会实际意义，也有权利、秩序、平衡、法治价值等上位意义。
5. 疑难点意识强：肖像权/名誉权、个人信息原则、合同成立/生效/履行、过错推定/无过错、合理使用/法定许可、辩护人/诉讼代理人等都已进入课堂。

## 旧框架的问题

1. 层级混在一起：教材知识、题型方法、法治意义、评分细则样式、课堂批注放在同一层。学生容易背下条目，但不知道新材料中先启动哪一步。
2. “企业”不是稳定一级分类：劳动者、消费者、经营者之间竞争分别对应不同法律关系，单用“企业”归类会遮蔽请求路径。
3. 主观题缺少统一动作：旧课件有大量圈点，但没有稳定写成“材料事实 -> 法律关系 -> 规则要件 -> 事实对应 -> 责任/救济 -> 意义”。
4. 法治意义有口号化风险：保护权益、维护公平正义、弘扬核心价值等如果不绑定材料事实，容易写成万能套话。
5. 程序法放得太靠后：起诉状、诉讼请求、诉讼代理人/辩护人、调解/仲裁/诉讼经常和实体题同时出现，不能只作为最后一章孤立记忆。
6. 一材多点没有登记机制：虚拟数字人、食品安全、滑梯案、社区纠纷等材料都跨知识产权/侵权/竞争/程序/意义，旧框架没有稳定的交叉索引。
7. 答案和细则来源边界不清：课件里有“细则”样式，但本轮不能把它们升级为官方评分细则。

## 改造原则

新版不推翻旧框架，而是重排旧框架。

保留：法条基础 + 法治意义、民事法律关系三要素、侵权责任链、意义角度库、疑难点积累。

重排：以教材四单元作明线，以“事实 -> 关系 -> 规则 -> 后果 -> 意义”作解题暗线。

新增：一材多点触发表、跨模块索引、选择题错项入口、事实触发式意义库、证据标签。

## 最关键的改造

旧框架问：“这道题属于哪个知识点？”

新版框架问：

> 谁与谁，基于什么法律关系，围绕什么权利义务，请求什么责任或救济，为什么这种处理有法治意义？

这个问题比单纯按章节归类更适合北京主观题，也能解释为什么同一材料往往不止一个采分方向。
