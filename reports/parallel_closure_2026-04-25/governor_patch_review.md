# 2026-04-25 并行收口 Governor 补丁复核矩阵

## 一、验收门槛

本矩阵只做最终验收前的监管判定，不替代 `governor_board.md` 的最终 PASS，也不改写 artifacts、题源清单、ledger、governor、current-state 或劳动者报告。

### 1. 套卷级闭环最低门槛

任一套卷从 `待补证据` 转为闭环，必须同时满足以下五项：

| 门槛 | 最低要求 | 不足时的判定 |
|---|---|---|
| 题面与答案源 | 第1-15题必须有可核题面、选项和可靠客观答案表；扫描件必须已有渲染/读图/OCR记录 | 只能记为客观答案补证或待核，不能入库错肢 |
| 选择题错肢线 | `choice_question_processing_ledger.md` 有对应批次或套卷记录；错肢库有来源套卷+题号；只入可复用、稳定、纯知识或高频错项 | 不能仅凭答案字母或“已处理”字样转闭环 |
| 正确项回填线 | 选择题中稳定的必修四/文化正确项已回填哲学/文化框架；非必修四题有明确模块边界说明 | 错肢库已做但正确项未回填时，不得签套卷级闭环 |
| 主观题细则线 | 主观题使用正式细则、阅卷报告、评标PPT/文档、用户确认可作评分口径的文件；材料-知识触发链写明来源题号 | 普通参考答案、等级描述、答案解析不能升格为评分细则 |
| Governor一致性 | pending、STEP_02缺口清单、ledger、哲学框架、错肢库、governor对同一套卷的状态和边界一致 | 任一核心文件仍显示阻塞时，只能写“待验收收口”，不能写最终 PASS |

### 2. 16个验收型条目的总判定

这16套不是新增生产项，而是“既有证据已基本齐备、缺套卷级验收收口”的项目。可进入验收收口，但不得由劳动者C/D用一句“已处理”直接转闭环；必须逐套列出选择题、主观题、正确项回填、边界说明和 governor 无禁项结论。

其中需要特别拆账的条目：

- `2025东城二模、2025昌平二模、2025朝阳二模、2025西城二模`：选择题证据在 `2025二模选择题补充（首轮）` 批次中，验收报告必须拆成套卷级证据，不能只引用批次名。
- `2026东城一模`：选择题证据在 `2026一模选择题首轮` 中，验收报告必须明确该套已做必修四选择筛查、正确项回填和非必修四边界。

### 3. 文化线对“待补证据清零”的影响

文化线会阻止“全体系待补证据清零”或“全部试卷已穷尽 PASS”。`governor_board.md` 记录的文化线是 `partial pass`，仍需最终 acceptance sync。

可区分两种口径：

| 清零口径 | 文化线是否阻止 | 可用表述 |
|---|---|---|
| 必修四哲学题源总表内部清零 | 不直接阻止，但必须注明“仅限哲学线” | “哲学线待补证据可进入清零验收” |
| 哲学+文化+全部试卷最终穷尽 | 阻止 | 文化线未最终同步前，不得写“全量最终 PASS” |

## 四、劳动者C/D报告复核清单模板

劳动者C/D报告必须按套卷逐项填写，监管复核只接受具体证据，不接受“已完成”“已处理”“见前文”式总括。

### A. 基础信息

| 检查项 | C/D报告填写要求 | 监管判定 |
|---|---|---|
| 套卷名称 | 年份、区、考试批次，与 pending 行完全一致 | 一致 / 不一致 |
| 处理范围 | 第1-15题、主观题号、排除题号分别列出 | 完整 / 缺项 |
| 使用源文件 | 试卷、答案表、细则/评标/阅卷报告的文件名或已记录证据路径 | 可追溯 / 不可追溯 |
| 产物位置 | ledger批次名、错肢库小节、哲学/文化框架条目、governor记录 | 齐备 / 不齐 |

### B. 选择题复核

| 检查项 | 通过条件 | 失败信号 |
|---|---|---|
| 答案源 | 有可靠答案表且与题面/选项同步核查 | 只有答案字母、无题面核验 |
| 错肢入库 | 每条有来源套卷+题号、错误类型、为什么错、规范说法 | 机械搬运所有选项、无复用价值、无题号 |
| 正确项回填 | 稳定必修四/文化正确项已回填框架 | 错肢库有记录但框架无正确项或边界说明 |
| 模块边界 | 非必修四题逐题说明政治、经济、法律、逻辑、国政经等边界 | 用“无关”概括整套或整段 |
| 重复检查 | 新批次不重复既有同题同错项 | 同一错肢重复入库或换标题重复 |

### C. 主观题复核

| 检查项 | 通过条件 | 失败信号 |
|---|---|---|
| 细则权威性 | 正式细则、阅卷报告、评标PPT/docx、用户确认评分口径 | 普通参考答案、答案解析、等级描述被当细则 |
| 材料-知识链 | 每条写材料信息、触发知识、逻辑链、来源题号 | 只列知识点，不解释材料如何触发 |
| 多点触发 | 一材多点按框架分条落位 | 只写一个知识点，漏掉同材多点 |
| 跨模块边界 | 逻辑、法律、经济、国政经等不硬并入必修四 | 为求闭环强行迁入 |
| 文化线 | 文化题同步到文化框架或明确等待文化总审 | 哲学线 PASS 被扩大成全线 PASS |

### D. Governor复核结论格式

每套卷只能给以下三类结论：

| 结论 | 使用条件 |
|---|---|
| `可转闭环-仅待状态同步` | 五项门槛均满足；只是题源清单/STEP_02/governor未改状态 |
| `可转闭环-需批次拆账` | 证据在批次级记录中；C/D已逐套拆出选择题、主观题、正确项回填和边界 |
| `不得转闭环-缺最低产物` | 缺题面核验、缺错肢批次、缺正确项回填、缺正式主观细则、缺文化总审任一项 |

| 禁止项 | 具体规则 |
|---|---|
| 参考答案升格 | 普通参考答案、答案解析、等级描述只能作参考，不得写成评分细则、评标细则或采分点依据 |
| 答案字母反推 | 任何新增错肢必须基于题面、选项、答案三者同步核查；不得由 `1B 2C...` 直接编错因 |
| 重复入库 | 已有同源同题同类错肢不得换标题、换批次重复写入；已闭环套卷只能抽查，不得重开生产 |
| 缺正确项回填 | 选择题中稳定必修四/文化正确项必须回填框架；没有回填或边界说明时，不得只凭错肢库签闭环 |
| 缺细则硬迁主观 | 主观题无正式细则时，不得把参考答案包装成“材料-知识触发链” |
| 批次替代套卷 | 2025二模首轮、2026一模首轮等批次记录不能直接替代套卷级验收表 |
| 文化线偷换 | 文化线 partial pass 不能被写成全体系 PASS；哲学线清零必须注明口径 |

## 六、最终可签 PASS 的条件

最终 PASS 只能在以下条件同时满足时签署：

1. 16个验收型条目均有套卷级验收记录；其中批次级记录已逐套拆账，选择题、主观题、正确项回填、模块边界和禁项检查齐备。
2. `2024门头沟一模` 具备题面核验、选择题错肢批次、正确项回填/边界、主观正式细则或明确边界；不能只保留客观答案源补证。
3. `2024顺义二模` 完成第1-15题选择题整套处理，并完成16（2）-21逐题细则/模块边界总审；第16（1）已入三链只能算局部证据。
4. 文化线完成最终 acceptance sync；若文化线仍是 partial pass，只能签哲学线局部验收，不能签全体系清零。
5. `pending_18_rows.md`、STEP_02缺口清单、`choice_question_processing_ledger.md`、哲学框架、错肢库、`governor_board.md` 对状态、题号和边界无冲突。
6. 无参考答案升格、无答案字母反推、无重复入库、无稳定正确项漏回填、无扫描件未读图/OCR即跳过。
7. governor 明确记录最终 PASS 的适用范围；若仍有任一边界或文化线阻塞，结论必须保留为 NOT PASS 或 partial pass。
