# 必修四《哲学与文化》大框架（Claude Code 重跑版）

适配北京高考。本框架以飞哥 2026 班课五大模块为骨架，结合 2024、2025、2026 三年北京各区一模、二模、期中、期末套卷的评分细则、阅卷报告和阅卷讲评，把"材料如何触发哲学原理"翻译成可在考场上直接照搬的答题路径。

> 本版的工作前提：每条触发都对得上一份可信给分依据（评分细则 / 阅卷报告 / 阅卷总结 / 用户已确认参考答案），来源套卷与题号在每条结论后注明，逻辑链是"为什么这条材料触发这个原理"，不是教学口号。证据边界与未纳入说明见末章。

---

## 一、使用说明

1. **不是题海，是答题路径库。** 你拿到一段陌生材料，先问"这段话写的是什么、要做什么"，再去匹配下面的原理词条，最后把材料层与原理层用逻辑链对接。
2. **优先级**：主观题给分点 > 选择题正确项触发 > 课本原理。本框架记录的都是"细则上写过的角度"，避免你写一堆课本原理却踩不到给分点。
3. **不要硬塞**。一道题如果细则只给了三条，就只写三条；不要为了凑字数把所有原理都拉一遍。
4. **跨模块边界**。涉及《逻辑与思维》（分析与综合、动态性思维、辩证否定的思维方法）、《文化与哲学》中的文化部分时，本框架只在题链中标注边界，不把跨模块原理硬并为必修四哲学。
5. **三类禁用区**。不再使用"可替代""反向筛查""教学提醒"三个标签。所有提示直接写到答题路径里，避免混淆。

## 二、框架总览（一图看懂五大模块）

| 模块 | 核心问题 | 高频考点（北京题目证实可触发） |
|---|---|---|
| 唯物论 | 客观和主观谁决定谁 | 物质决定意识；意识能动作用；一切从实际出发；尊重规律与发挥主观能动性相结合；规律的客观性 |
| 辩证法 | 事物之间是什么关系，事物如何变化，事物为什么会向前发展 | 联系（普遍性、客观性、多样性、整体与部分、系统观念）；发展（发展观点、量变质变、前进性曲折性、辩证否定/守正创新）；矛盾（对立统一、普遍性、特殊性、主次矛盾、矛盾主次方面、两点论重点论） |
| 认识论 | 实践和认识谁先谁后，认识为什么不断变化 | 实践是认识的基础；实践决定认识；认识具有反复性、无限性、上升性；真理的客观性、具体性、条件性 |
| 历史唯物主义 | 历史是怎么前进的，谁在推动 | 社会存在决定社会意识、社会意识反作用；社会基本矛盾运动；改革；人民群众是历史主体 |
| 价值观与人生价值 | 人为什么活着，怎么活才有价值 | 价值观导向作用；价值判断与价值选择；在劳动和奉献中创造价值 / 在个人和社会的统一中实现价值 / 在砥砺自我中走向成功 |

> 北京高考主观题最常出现的"哲学说明类"组合：联系 + 矛盾 + 发展 + 价值观；"应如何"组合：实事求是 + 尊重规律与发挥主观能动性 + 群众观点 + 系统观念。

## 三、唯物论

### 3.1 物质决定意识，意识对物质具有能动作用

**核心判断**：评估对象是不是从客观存在出发被认识、被反映、被改造。

- **当材料里出现 → 触发本原理**
  - 出现"基于现实条件 / 基于客观实际 / 立足时代 / 立足国情 / 客观规律 / 客观需要 / 现实情况 / 真实情况 / 不能脱离实际"等表达；
  - 出现"对……的认识 / 看法 / 评价 / 反思 / 总结"——认识是对客观存在的反映；
  - 出现"作品打动人心 / 政策符合实际 / 治理取得成效"等结果，再回看其前置条件。

- **典型给分材料（已对评分点）**
  1. 2025 北京东城高三二模 第16题：人工智能浪潮基于"科技水平、社会需求等客观实际"。阅卷报告把"物质决定意识"列为给分点。
  2. 2026 北京西城高三（上）期末 第16题（2）：把"真实"界定为"客观事物及其规律的本来面目"。用户补充评分页明确该角度可得分。
  3. 2024 北京朝阳高三二模 第16题（2）：阅卷总结明确"物质意识"为可选知识；人工智能既源于客观实践，又反过来影响人的生产生活。

- **答题路径**
  - 第一步：把材料中的客观条件、规律、需求、问题列出来。
  - 第二步：写"……（结论 / 判断 / 政策 / 表达）正是基于以上客观实际形成的"。
  - 第三步：回扣设问，强调主观认识必须与客观实际相统一。

### 3.2 一切从实际出发 / 实事求是 / 主观与客观具体的历史的统一

**核心判断**：方案、政策、措施、答案、文化创作是否依据具体条件而不是凭主观想象。

- **当材料里出现 → 触发**
  - "因地制宜 / 立足时代之基 / 立足实际 / 一手 / 真情实感 / 调查研究 / 把握真情况"；
  - "不一哄而上 / 不脱离实际 / 不凭主观想象 / 不照搬模板"；
  - 具体处境与统一目标交替出现的对比结构。

- **典型给分材料（已对评分点）**
  1. 2025 北京丰台高三（上）期末 第16题（`丰台期末细则.pptx`）：评分标准把"数从哪里来"对应到实践、实际、规律、发挥主观能动性。
  2. 2025 北京朝阳高三（上）期末 第16题（`朝阳期末评标.pdf`）：文化交流既要符合本民族文化发展实际，也要兼顾交流对象需求。
  3. 2026 北京海淀高三一模 第8题：政绩观要求"深入基层一线倾听不同声音、掌握实情"。
  4. 2024 北京东城高三二模 第16题（`阅卷总结/16题/16题二模阅卷总结.docx`、`16题评分细则.pptx`）：桑基鱼塘"立足实践、尊重规律 / 一切从实际出发"为哲学层面给分点。
  5. 2025 北京门头沟高三一模 第21题（2）（`门头沟一模细则.doc`）：评分细则明确允许从"主观与客观、认识与实践具体的历史的统一"作答。

- **答题路径**
  - 第一步：把材料中的"具体条件 / 现实问题 / 真实情况"列出来。
  - 第二步：写"做法立足上述客观情况，不脱离实际，不凭主观想象"。
  - 第三步：在"应如何"题中加上"做到主观与客观、认识与实践具体的历史的统一"。

### 3.3 主观能动性 / 意识的能动作用

**核心判断**：成果是不是经过人的主动构思、创造、设计、转化才形成的。

- **当材料里出现 → 触发**
  - "精心打造 / 精雕细琢 / 设计 / 改造 / 转化 / 创新 / 主动谋划"；
  - "认识规律后做事 / 把对……的认识转化为……行动"；
  - 文艺创作、技术研发、文化传播、产品开发等场景。

- **典型给分材料（已对评分点）**
  1. 2025 北京朝阳高三一模 第16题（用户补充评分截图）：《哪吒之魔童闹海》创作团队对技术、细节、关键戏份的主动把握和持续打磨。
  2. 2025 北京海淀高三（上）期末 第16题（用户确认 PPT 给分口径）：中医药新时尚靠主体对传统资源、消费需求、产品形式的主动选择与转化。
  3. 2025 北京东城高三（上）期末 第16题（用户补充讲评 PDF 答案细则）：把对器物价值的认识转化为搜集、保护、研究等行动。
  4. 2025 北京丰台高三（上）期末 第16题（`丰台期末细则.pptx`）：把对"数"的认识转化为做计划、定方案、国家治理。
  5. 2026 北京海淀高三（上）期末 第16题（评分标准）：数字原住民必须"主动培养独立思考、表达训练、判断能力"，否则技术便利反过来弱化能力。

- **答题路径**
  - 第一步：找出材料中的"主动行为词"——构思、设计、组织、推动、改造、打磨。
  - 第二步：写"主体（团队 / 国家 / 人民 / 创作者）发挥主观能动性，把……转化为……"。
  - 第三步：把成果落回"满足……需求 / 实现……目标 / 推动……发展"。

### 3.4 尊重客观规律与发挥主观能动性相结合

**核心判断**：题目是不是同时强调"按规律办事"和"主动作为"。

- **当材料里出现 → 触发**
  - "顺应规律 + 主动作为"；
  - "尊重自然规律 + 生态治理 / 修复 / 转型"；
  - "把握规律 + 创造条件 / 引导 / 规范"。

- **典型给分材料（已对评分点）**
  1. 2025 北京东城高三二模 第16题：勇立潮头与人工智能共创未来——既顺应人工智能发展规律，又主动引导。
  2. 2025 北京朝阳高三（上）期末 第16题：评分细则提出"文化发展有其自身规律 + 正确发挥主观能动性 + 共建美美与共的文明百花园"。
  3. 2025 北京延庆高三一模 第4题：长江"共抓大保护、不搞大开发 + 修复 + 统筹各方力量"。
  4. 2024 北京东城高三二模 第16题：桑基鱼塘"遵循人与自然规律 + 保护利用 + 产业融合 + 文化研学"。
  5. 2025 北京西城高三二模 第16题（1）（`讨论定稿-答案细则 -25.5西城高三政治二模-1.docx`）：把握生态规律、动物需求、场地条件后主动改造水井。

- **答题路径**
  - 第一步：先写"……规律 / 客观条件不可违背"。
  - 第二步：再写"在尊重规律基础上，主体主动……"（动词必须来自材料）。
  - 第三步：落到"实现……目标 / 取得……成效"。

### 3.5 规律的客观性

**核心判断**：是否强调规律不以人的意志为转移、违规律就受惩罚。

- **当材料里出现 → 触发**
  - "客观存在的规律 / 自身规律 / 内在规律"；
  - "违背规律就……出错 / 失败 / 受惩"；
  - 文化传承、生态治理、经济运行的"自身规律"。

- **典型给分材料（已对评分点）**
  1. 2025 北京丰台高三（上）期末 第16题：毛泽东"不懂得数量界限就不能不犯错误"。
  2. 2025 北京朝阳高三（上）期末 第16题：文化发展、文化传承、文化交流"有其自身规律"。
  3. 2026 北京朝阳高三（上）期末 第16题（`2026朝阳期末细则.pdf`扫描页读图核定）：戏曲"保留艺术精髓、赋予时代内涵"是按规律转化。

## 四、辩证法

### 4.1 联系

#### 4.1.1 联系的普遍性 / 联系的观点（总）

**核心判断**：材料是否把多个对象、多个主体、多种要素放在同一关系网络里相互作用。

- **典型给分材料**
  1. 2025 北京东城高三二模 第4题：流失国宝回归靠"外交 + 法律 + 国际合作 + 文化认同"多主体、多路径联动。
  2. 2025 北京朝阳高三（上）期末 第16题：评分细则要求处理"中国文化与外来文化、中国文化与世界文化"的关系。
  3. 2025 北京朝阳高三（上）期末 第22题：评分细则要求"两个结合"——理论创新要在多重对象的结合中展开。
  4. 2026 北京东城高三一模 第2题：隆福寺街区把"老字号 + 文创 + 新饮品 + 潮流市集 + 非遗体验 + 机器人科技"放进同一关系网络。
  5. 2026 北京海淀（上）期末 第15题：北京旅游预约平台把"交通 + 住宿 + 餐饮 + 景点 + 体验式场景"组织进同一服务链。

- **答题路径**：写"事物是普遍联系的 → 材料中 X 与 Y 与 Z 不是孤立存在 → 它们的协同 / 联动 / 相互作用形成……"。

#### 4.1.2 联系的客观性 / 根据固有联系建立新的具体联系

**核心判断**：是否先发现客观存在的联系，再据此建立新的具体联系。

- **典型给分材料**
  1. 2025 北京西城高三二模 第16题（1）：先看到"井 + 水 + 动物"之间客观存在的联系，再设计台阶、雨水池建立新的生态联系。
  2. 2025 北京海淀高三二模 第9题：把握工业资源与消费场景、文化传播、教育体验之间的客观联系，再"工业旅游+"。
  3. 2026 北京东城高三（上）期末 第4题：通过实验发现核糖核酸与氨基酸之间客观存在的化学关联。

- **答题路径**：写"联系是客观的，不能凭主观臆造 → 主体先把握……之间固有联系 → 据此建立新的具体联系"。

#### 4.1.3 联系的多样性

**核心判断**：是否强调多种条件、多种方式、多种路径并列发挥作用。

- **典型给分材料**
  1. 2025 北京丰台高三二模 第4题：无人机施肥 + 5G 农业 + 自动驾驶拖拉机，并列展示多技术路径。
  2. 2026 北京海淀高三一模 第5题：数字游民 + 远程工作 + 乡村生态 + 文创业态。
  3. 2025 北京朝阳高三二模 第3题：现场教学 + 线上课程 + 实景研学 + 校园宣讲 + 志愿服务多种红色教育形式。
  4. 2026 北京朝阳高三（上）期中 第4题：全球治理"主权平等 + 国际法治 + 多边主义 + 以人为本 + 行动导向"五大理念。

- **答题路径**：写"联系是多样的，存在直接 / 间接、内部 / 外部、本质 / 非本质等不同方式 → 材料展示了 X / Y / Z 多条件多路径协同 → 以此推动……"。

#### 4.1.4 整体与部分 / 系统优化（系统观念）

**核心判断**：是否要求"立足整体、统筹局部、协同增效"。

- **典型给分材料**
  1. 2025 北京丰台高三（上）期末 第16题：评分标准把"数当如何定"指向"系统优化、整体部分"。
  2. 2025 北京朝阳高三（上）期末 第22题：细则要求"坚持系统观念，整体性推进党和国家各项事业"。
  3. 2026 北京海淀高三一模 第4题："统筹长城文化带、大运河文化带、西山永定河文化带"传承发展。
  4. 2025 北京朝阳高三（上）期末 第9题：韧性城市建设按圈层（首都功能核区 / 中心城区 / 副中心 / 两轴 / 多点 / 生态涵养区）协同推进。
  5. 2024 北京东城高三二模 第16题：桑、蚕、鱼、塘组成相互联系系统，"立足整体 / 系统优化"为细则给分点。
  6. 2024 北京丰台高三二模 第13题：一个人站起来看清比赛——局部行为叠加导致整体功能受损。

- **答题路径**：写"整体与部分相互依存又相互区别 → 材料中要素 A、B、C 共同支撑整体目标 → 必须立足整体把握部分 / 统筹推进 / 系统优化"。

### 4.2 发展

#### 4.2.1 发展的观点 / 发展的普遍性

**核心判断**：对象是否处在持续生成、转化、扩展中，是否把对象写成过程而不是状态。

- **典型给分材料**
  1. 2025 北京东城高三二模 第16题：人工智能"横空出世 + 浪潮滚滚 + 不断拓展到科技、产业、治理、医疗、教育、家庭"。
  2. 2025 北京东城高三（上）期末 第16题：何尊 → "中国"一词产生与发展 → 中华文明发展历程。
  3. 2025 北京海淀高三（上）期末 第16题：传统中医药 → 石斛曲奇、养生面包、中药咖啡等现代消费产品。
  4. 2026 北京西城高三（上）期末 第21题：五年规划具有连续性又随发展变化不断优化。
  5. 2024 北京东城高三二模 第16题：桑基鱼塘"两千多岁仍未老"。

- **答题路径**：写"事物是变化发展的 → 材料中 X 经历 X1 → X2 → X3 不断生成新形态 / 新功能 → 用发展眼光看……"。

#### 4.2.2 量变与质变 / 适度原则

**核心判断**：是否强调量的积累或数量界限。

- **典型给分材料**
  1. 2025 北京海淀高三二模 第16题：知识储备的不断积累是创造力的前提。
  2. 2025 北京海淀高三（上）期末 第17题（2）：永动机"反复探索的量的积累 → 能量守恒定律的质的突破"。
  3. 2025 北京丰台高三（上）期末 第16题：质量都表现为一定的数量、注意"度"。
  4. 2025 北京延庆高三一模 第2题：课间 15 分钟反对"急功近利、拔苗助长、孩子时间填得过满"——把握适度。
  5. 2025 北京朝阳高三一模 第1题：革命、建设、改革"经历从量的积累到质的飞跃"。

- **答题路径**：写"事物发展从量变开始 → 材料中持续投入、持续积累 → 达到关节点引起质变 → 形成更高层次结果"。

#### 4.2.3 前进性与曲折性

**核心判断**：是否同时呈现"事物总体向前 + 道路曲折"。

- **典型给分材料**
  1. 2025 北京海淀高三（上）期末 第17题（2）：永动机研究反复失败但推动科学认识前进。
  2. 2025 北京西城高三（上）期末 第18题：新技术"不可避免存在弱点和不完善"，对其认识要有过程。

- **答题路径**：写"事物发展是前进性与曲折性的统一 → 材料中存在反复、失败、修正 → 但总体趋势仍是向上的"。

#### 4.2.4 辩证否定 / 守正创新

**核心判断**：是否既保留合理内核，又克服旧形态中不合时宜的部分。

- **典型给分材料**
  1. 2026 北京朝阳高三（上）期末 第16题（`2026朝阳期末细则.pdf`扫描页核定）：传统戏曲"剔除与当代社会脱节的部分，保留艺术精髓，并赋予新的时代内涵"——细则把"辩证否定观"列为 4 分给分点。
  2. 2025 北京海淀高三（上）期末 第16题：中医药新时尚"守正不守旧、尊古不复古"。
  3. 2025 北京朝阳高三（上）期末 第22题：理论创新"守正 + 创新"。
  4. 2024 北京东城高三二模 第16题：桑基鱼塘"保留生态循环和农业智慧 + 更新功能形态"。
  5. 2025 北京东城高三一模 第16题：现代中国年保留团圆、祈愿、喜庆精神核心，融入现代科技与休闲。
  6. 2026 北京朝阳高三一模 第3题：春节科普"非遗变脸 + 机器人乐队 + 科普实验"同台呈现。
  7. 2025 北京西城高三一模 第2题：中华民族现代文明"传承 + 创新创造"。

- **答题路径**：写"辩证否定的实质是扬弃 → 材料中既不否定全部、也不照单全收 → 保留……合理内核 + 克服……不合时宜部分 + 形成……新的表现 → 实现守正创新"。

### 4.3 矛盾

#### 4.3.1 矛盾就是对立统一 / 一分为二 / 全面观点

**核心判断**：是否要求看到同一对象内部既相互区别又相互联系的两方面。

- **典型给分材料**
  1. 2025 北京东城高三二模 第16题：人工智能"机遇 + 风险"两面共存。
  2. 2026 北京海淀高三（上）期末 第16题：网络技术"红利 + 弱化能力"。
  3. 2025 北京西城高三（上）期末 第18题：技术恐惧"抵制技术进步 + 倒逼制度规范完善"。
  4. 2026 北京朝阳高三（上）期中 第18题：人工智能情绪价值"长处 + 缺陷弊端"。
  5. 2025 北京东城高三（上）期末 第4题：汉字笔画"主次、长短、留白差异 + 整体气韵相生"。
  6. 2025 北京海淀高三二模 第12题：耐心资本"短期与长期、风险与收益"对立统一。

- **答题路径**：写"矛盾就是对立统一 → 材料中 X 与 Y 既相互区别又相互联系 → 要坚持一分为二，全面看……"。

#### 4.3.2 矛盾的特殊性 / 具体问题具体分析

**核心判断**：是否要求按对象自身特点 / 阶段 / 方面分别处理。

- **典型给分材料**
  1. 2024 北京西城高三二模 第7题：北京老城街巷"因地制宜，知晓每处街巷矛盾特殊性"。
  2. 2025 北京海淀高三二模 第17题（1）：调查方法"问卷法、访谈法、观察法、文献法"分别适用不同情境。
  3. 2025 北京朝阳高三（上）期末 第7题："删繁就简三秋树，领异标新二月花"——独特风格、突出个性。
  4. 2024 北京丰台高三二模 第18题（2）：评标在"其他角度论证"中允许从"具体问题具体分析"得分。
  5. 2024 北京海淀高三二模 第4题："指尖上的形式主义"治理要回到基层真实负担。

- **答题路径**：写"矛盾具有特殊性 → 不同对象 / 阶段 / 方面有不同矛盾 → 必须具体问题具体分析 → 不能套用同一模板"。

#### 4.3.3 矛盾的普遍性与特殊性 / 主要矛盾和次要矛盾 / 矛盾的主要方面和次要方面 / 两点论与重点论

**核心判断**：是否要求"既看主要又看次要 / 抓重点同时不忽视一般"。

- **典型给分材料**
  1. 2024 北京东城高三二模 第3题："解剖麻雀"——从特殊性中把握普遍性，以点的突破带动面的提升。
  2. 2024 北京东城高三二模 第11题：糖类有不同特点 → 区分对待，体现特殊性 + 价值选择。
  3. 2024 北京海淀高三二模 第17题（1）：细则明确"矛盾分析法 / 具体问题具体分析"为给分点之一。
  4. 2024 北京丰台高三二模 第20题：8 分等级题，材料把"发展与安全"作为对立统一的两方面。

- **答题路径**：写"矛盾具有普遍性也具有特殊性，二者辩证统一 → 材料中 A 是事物的主要方面（或主要矛盾），起决定作用 → 同时不能忽视次要方面（或次要矛盾） → 坚持两点论和重点论的统一"。

#### 4.3.4 内因与外因

**核心判断**：是否要求"内因起决定作用、外因通过内因起作用"。

- **典型材料定位（已对应评分点）**
  - 在涉及"自身条件 + 外部环境"协同作用题目（如长江生态治理 / 区域转型）中，可与"尊重客观规律与发挥主观能动性相结合"配套使用，但不强行单列。
  - 当评分细则明确给到"内外因"时再写；细则没给的不要硬塞。

## 五、认识论

### 5.1 实践是认识的基础 / 实践决定认识

- **答题路径**：写"实践是认识的来源、动力、目的、检验标准 → 材料中实践 X 推动认识 Y → 认识又反过来指导实践"。

### 5.2 认识发展原理（认识的反复性、无限性、上升性）

  2. 2025 北京海淀高三（上）期末 第17题（2）：永动机"反复探索 → 突破 → 进入更深科学认识"。

- **答题路径**：写"认识具有反复性、无限性和上升性 → 材料中主体在 X 实践 → Y 实践 → Z 实践中不断深化对……的认识 → 与时俱进、开拓创新"。

## 六、历史唯物主义

### 6.1 社会存在与社会意识

**核心判断**：是否把社会意识写成对社会存在的反映，并强调先进社会意识反作用于社会存在。

- **典型给分材料**
  1. 2026 北京海淀高三一模 第16题：细则给"社会存在与社会意识"为可作角度。
  2. 2026 北京丰台高三期末 第1题：经典歌曲"反作用于社会实践"，凝聚奋进力量。
  3. 2024 北京朝阳高三期中 第16题（`2024.11期中政治朝阳评标docx.docx`、`阅卷细则总.rtf`）：把"社会存在与社会意识"列为给分。

- **答题路径**：写"社会存在决定社会意识 / 社会意识反作用于社会存在 → 材料中现实条件 X 决定 Y 类社会意识形态 → 先进社会意识 Y 又反过来推动 X"。

### 6.2 社会基本矛盾运动 / 社会发展的两大基本规律

**核心判断**：是否涉及生产力与生产关系、经济基础与上层建筑的适应关系。

- **典型给分材料**
  1. 2024 北京东城高三二模 第18题（2）：阅卷细则"上层建筑适应经济基础的发展"——平台用工新形态推动劳动法律规范完善。
  2. 2024 北京海淀高三二模 第21题：政府工作报告"新质生产力 + 新型举国体制"指向生产力 → 制度组织方式调整。
  3. 2024 北京朝阳高三期中 第16题：评标列出社会基本矛盾 / 基本规律。

- **答题路径**：写"生产关系 / 上层建筑 一定要适应 生产力 / 经济基础 → 材料中 X 变化 → Y 必须随之改革 → 形成新的适应"。

### 6.3 改革

**核心判断**：是否把改革写成社会主义制度的自我完善与发展。

- **典型给分材料**
  1. 2025 北京朝阳高三一模 第1题：革命、建设、改革"从量的积累到质的飞跃"——改革作为历史方位。

- **答题路径**：写"改革是社会主义制度的自我完善与发展 → 材料中 X 制度调整不是推翻原制度，而是在保持社会主义性质前提下解决矛盾"。

### 6.4 人民群众是历史的创造者

**核心判断**：是否突出"人民"作为主体——历史 / 价值 / 实践的主体。

- **典型给分材料**
  1. 2024 北京西城高三二模 第1题：人民至上 + 现实的人。
  2. 2024 北京海淀高三二模 第1题：《共产党宣言》"绝大多数人的、为绝大多数人谋利益"。
  3. 2024 北京丰台高三二模 第21题：评标把"个人与国家紧密联系 + 局部推动整体"列为得分点；个人的奋斗汇聚成中国整体步伐。
  4. 2024 北京东城高三二模 第16题：阅卷细则中"人民群众"为价值层给分点。
  5. 2024 北京朝阳高三期中 第16题：评标"人民群众"为给分点。

- **答题路径**：写"人民群众是历史的创造者，是社会物质财富、精神财富的创造者，是社会变革的决定力量 → 材料中 X 来自人民、依靠人民、为了人民 → 必须坚持群众观点和群众路线"。

## 七、价值观与人生价值

### 7.1 价值观的导向作用

**核心判断**：是否要求树立正确价值观、用正确价值观引导行为。

- **典型给分材料**
  1. 2025 北京东城高三二模 第16题：阅卷报告"价值观的导向作用"为给分点。

  3. 2024 北京海淀高三二模 第16题：分项细则"重要性层 + 价值观 + 共识共鸣 + 社会主义核心价值观 + 个人与社会国家人民立场"。

- **答题路径**：写"价值观对人们认识世界和改造世界的活动具有导向作用 → 树立正确价值观 → 引导……的判断与选择"。

### 7.2 价值判断与价值选择

**核心判断**：是否要求"自觉站在最广大人民立场上 / 遵循社会发展客观规律"。

- **典型给分材料**
  1. 2024 北京海淀高三二模 第1题：站在最广大人民根本利益。
  2. 2024 北京东城高三二模 第11题：基于健康目标作出消费选择。
  3. 2024 北京西城高三二模 第1题：人民立场 / 人民群众。
  4. 2026 北京延庆高三一模 第4题：社区在多种价值判断中选择"保留 + 更新"方案。

- **答题路径**：写"价值判断和价值选择因人而异 → 必须遵循社会发展客观规律 + 自觉站在最广大人民立场上"。

### 7.3 实现人生价值

**核心判断**：是否突出"在劳动和奉献中创造价值 / 在个人与社会的统一中实现 / 在砥砺自我中走向成功"。

- **典型给分材料**

  2. 2026 北京房山高三一模 第2题：人的价值"只能在社会中实现、必须正确处理个人与社会关系"。
  3. 2024 北京丰台高三二模 第21题：每一个人在不同岗位上持续奋斗，汇聚成国家发展整体步伐。

- **答题路径**：写"人的价值在于创造价值，在劳动和奉献中实现 → 在个人与社会的统一中实现 → 在砥砺自我中走向成功 → 材料中主体 X 把自身能力投入 Y 服务于社会与人民 → 实现人生价值"。

## 八、综合题型：材料触发多个哲学点的处理

### 8.1 "是怎样做到的 / 是如何"题（以人工智能、文化传承、生态治理类为代表）

**通用配方**（北京阅卷线已多次验证）：
1. 唯物论：从实际出发 / 物质决定意识；
2. 辩证法：联系（系统观念）+ 矛盾（全面观点 / 具体问题具体分析）+ 发展（守正创新）；
3. 认识论：实践推动认识、认识反复无限上升性；
4. 历史唯物主义：人民群众或社会基本矛盾；
5. 价值观：导向作用 + 价值选择立场。

**典型套题**：
- 2025 北京东城高三二模 第16题（人工智能）：物质决定意识 + 价值观 + 矛盾全面观点 + 实践基础 + 认识反复无限上升性。
- 2024 北京东城高三二模 第16题（桑基鱼塘）：实事求是 + 尊重规律与发挥主观能动性 + 联系系统 + 发展观点 + 辩证否定 + 人民群众 + 价值观导向。
- 2025 北京朝阳高三（上）期末 第16题（文化对话）：实事求是 + 联系 + 发展 + 辩证否定 + 价值观 + 规律的客观性。
- 2026 北京西城高三（上）期末 第16题（1）（真实）：实践基础 + 实践检验真理 + 价值观导向 + 群众观点。

> 写法提醒：评分细则给几个角度，就组织几条；不要凑齐五大模块。

### 8.2 "应如何 / 如何做"题

**通用配方**：
1. 主体识别（党 / 政府 / 企业 / 学校 / 个人 / 国际社会）；
2. 顺应规律 → 主动作为；
3. 实事求是 + 系统观念；
4. 站在人民立场，依靠人民群众；
5. 用正确价值观引导。

**典型套题**：
- 2025 北京东城高三二模 第16题（"应如何"）：尊重规律 + 主观能动性 + 全面观点 + 实践 + 认识发展。
- 2026 北京海淀高三一模 第16题（评分细则给"发展观点 + 社会存在意识 + 价值观 + 文化功能"）。

### 8.3 等级评分题（8 分大题）

**判分逻辑**：水平 1（1-2 分） → 写出某一个角度但浅；水平 4（7-8 分） → 三个及以上角度阐释完整、紧扣材料、有逻辑链。

**典型套题**：
- 2024 北京丰台高三二模 第20题：联系 + 矛盾 + 认识发展三个角度论证统筹发展和安全。
- 2025 北京海淀高三二模 第16题：联系 + 量变质变 + 辩证否定 + 系统观念 + 认识能动反映。

> 等级题不要写一长段；要分点、分层、有清晰的角度切换。

## 九、高频失分点与审题提醒

1. **把材料当成原理。** 写哲学题不是复述材料，而是把材料层与原理层用逻辑链对接。先抽原理，再回扣材料。
2. **凭主观想象凑原理。** 评分细则只给三个角度时，第四个角度即便课本上写得通也不给分。要按细则口径写。
3. **混淆"原理"和"方法论"。** 唯物论里"物质决定意识"是原理，"一切从实际出发"是方法论；辩证法里"联系是普遍的"是原理，"用联系的观点看问题"是方法论。考场上原理和方法论要成对出现。
4. **跨模块原理硬塞。** 出现"分析与综合 / 演绎与归纳 / 辩证思维 / 创新思维 / 发散思维 / 聚合思维"时，那是《逻辑与思维》（选必三），不是必修四哲学。
5. **以参考答案当评分细则。** 普通参考答案（"参考答案 / 高三模拟答案"）不是评分细则；细则要看"评分标准 / 评分细则 / 阅卷报告 / 阅卷讲评"。本框架内的所有给分点都来自评分细则或用户已确认可用的源。
6. **忽视"应如何"题的主体识别。** "应如何"题第一步是从材料中找出"谁来做"——党、政府、人民、企业、个人。主体不同，措施口径不同。
7. **盲目堆砌价值观。** 写价值观要落到"立场 + 标准"两层；只写"价值观重要"不得分。
8. **辩证否定写成完全否定。** 辩证否定 = 扬弃 = 既肯定又否定。漏掉"保留合理内核"或"克服不合时宜"任意一层都会被扣分。
