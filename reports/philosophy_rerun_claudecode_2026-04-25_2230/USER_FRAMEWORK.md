## 五大模块

### 一、唯物论
- 物质决定意识，意识对物质具有能动作用
- 一切从实际出发 / 实事求是 / 主观与客观具体的历史的统一
- 主观能动性 / 意识的能动作用
- 尊重客观规律与发挥主观能动性相结合
- 规律的客观性

### 二、辩证法
- 联系：联系的普遍性、联系的客观性、联系的多样性、整体与部分、系统观念、系统优化
- 发展：发展的观点 / 发展的普遍性、量变与质变、适度原则、前进性与曲折性、辩证否定 / 守正创新
- 矛盾：对立统一、矛盾的普遍性、矛盾的特殊性、矛盾普遍性和特殊性、主次矛盾、矛盾主次方面、两点论重点论、内外因

### 三、认识论
- 实践与认识 / 认识对实践的反作用
- 认识发展原理（认识具有反复性、无限性、上升性，追求真理是过程）
- 真理观（真理的客观性、具体性、条件性、与谬误相伴而行）

### 四、历史唯物主义
- 社会存在与社会意识
- 社会发展的两大基本规律和基本矛盾
- 改革 / 改革的实质
- 人民群众（社会历史的主体、群众观点与群众路线）

### 五、价值观 / 人生观
- 价值观的导向作用
- 价值判断与价值选择
- 实现人生价值（在劳动和奉献中创造价值；在个人和社会的统一中实现价值；在砥砺自我中走向成功）

## 框架使用规则（用户已固化）
- 触发记录统一四项：来源套卷与题号 / 材料信息 / 触发的原理或方法论 / 逻辑链。
- 选择题正确项触发与主观题触发都纳入；选择题错肢另立错肢库。
- 不允许新增 `可替代`、`反向筛查`、`教学提醒` 三类节。
- 跨模块原理（如《逻辑与思维》的分析与综合、动态性思维、辩证否定的思维方法）只在原题逻辑链中标注跨模块边界，不并为必修四哲学原理。
- 同一题若稳定触发多个哲学点，分条归入对应模块，但每条都要写出可解释的逻辑链。
- 当题目所给评分细则只触发了部分哲学点，其余只能写"具有触发可能但细则未给分"，不能伪造给分。
