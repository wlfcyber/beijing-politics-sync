# User Framework

## Locked Top-Level Buckets

1. 时代背景
2. 理论
3. 经济全球化
4. 政治多极化
5. 中国
6. 联合国

## User's Current Product Shape

- 不是普通答案库，而是从评分细则、评标、阅卷总结中积累“能得分的术语和链条”。
- 六大要素下面放术语/框架容器，再把各题插入对应术语下。
- 每个术语/答题点后面标注出现频次；详细频次卷子列表可另做统计文档。
- 每条必须同时保留：术语、完整设问、细则位置、来源、材料触发、答案句。
- 材料触发要解释“为什么这道题的材料关系触发这个点”，不能写成材料复述。
- 答案句要像考生卷面句：术语 + 本题事实 + 因果/作用/结论。

## Branch Placement Rules

- 时代背景：和平与发展、经济全球化/世界多极化深入发展、逆全球化、霸权主义、强权政治、单边主义、贸易保护主义、治理赤字等背景。
- 理论：国家利益、国家安全、共同利益、国际竞争实质、综合国力、合作共赢等解释性理论。
- 经济全球化：贸易投资自由化便利化、要素跨国流动、两个市场两种资源、国际分工协作、推动经济全球化朝开放包容普惠平衡共赢方向发展。
- 政治多极化：新型国际关系、共商共建共享全球治理观、国际关系民主化、反霸权强权、构建人类命运共同体等国际秩序/治理方向。
- 中国：政策、智慧、责任、地位等中国方案/中国行动/中国担当。
- 联合国：联合国地位作用、《联合国宪章》宗旨原则、多边主义、以联合国为核心的国际体系、中国与联合国关系。

## Boundary Rules

- 必修二术语如扩大国际市场、推进高水平对外开放、开放发展理念、新发展理念、高质量发展等，不能硬塞进选必一主表。
- 选必二、选必三、政治与法治、必修四术语不能进入选必一主链。
- 参考答案中的结果句不是自动术语，必须回到评分细则原词判断。
