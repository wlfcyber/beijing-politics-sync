# User Questions

## Non-Blocking Default Assumptions

- 默认按“从源头重新核”跑选必一；旧 v12 成果只作为硬规则和纠错参照，不直接复制内容。
- 默认核心交付是主观题细则术语学生版 Markdown + DOCX，而不是选择题宝典。
- 默认保留六大要素结构，不重开新顶层框架。

## Questions For User If They Want To Override

1. 是否要求像必修四那样完全“从0不借旧文件”？如果是，旧 v12 成果也只读小本本和用户纠错，不读旧正文。
2. 是否要把选必一选择题也纳入最终文档？当前 branch skill 默认只做主观题，选择题只在用户明确要求时处理。
3. 最终文档是否沿用“飞哥正志讲堂”水印、目录、页码和浅蓝标题风格？默认沿用文档流水线里的稳妥版。
