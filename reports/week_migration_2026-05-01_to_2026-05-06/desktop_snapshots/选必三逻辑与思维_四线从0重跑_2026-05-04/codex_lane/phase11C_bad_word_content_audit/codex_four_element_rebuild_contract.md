# Codex Four-Element Rebuild Contract

## Required Student Entry Shape

### 材料触发点

Must identify concrete signals a student can circle:

- subject/action: who did what;
- relation: coordination, contrast, transformation, sequence, conflict, trend, verification;
- signal-to-method bridge: why this signal points toward a thinking method.

Bad: `材料强调科学思维。`

Good: `企业先蹲点了解老年人照护、健康提醒、情感陪伴等真实需求，再围绕这些需求反复测试产品。`

### 设问

Must be the actual question prompt. If the prompt is not available, the row is blocked until source is reopened.

Bad: `本题要求结合材料说明其体现的思维方法。`

Good: `结合材料，运用科学思维的知识，说明该企业研发具身机器人的过程给我们的启示。`

### 为什么能想到

Must explain the path:

`材料信号 -> 总帽子 -> 小方法 -> 为什么不是别的方法`

Example:

`从老人真实需求出发，说明认识对象不是凭主观想象设定，触发科学思维的客观性；研判老龄化趋势和产品未来场景，触发预见性；反复测试和迭代，说明方案要接受实践检验，触发可检验性。`

### 答案落点

Must be a direct answer-sheet sentence:

`思维方法/术语 + 本题材料事实 + 因果/作用/结论`

Bad:

`卷面要把材料中的具体动作写进方法里：从真实需求出发写客观性，跟踪趋势写预见性。`

Good:

`科学思维具有客观性、预见性和可检验性；该企业从老年人真实照护需求出发研发产品，研判老龄化趋势和具身智能应用前景，并通过反复测试迭代检验方案，使产品研发更符合实际需要和未来发展方向。`

## Multi-Node Rule

If one source question is mounted under several framework nodes, do not duplicate the same four elements.

For example, `2025东城期末 Q18(2)` can split:

- Under `联想思维`: focus on `飞天飘带 / 火箭尾焰 / 航天服造型` connection.
- Under `发散思维与聚合思维`: focus on `行走、攀爬、驾车、科考、防护、防眩光` multiple needs and final design方案.
- Under `超前思维`: only include if source supports future task/environment anticipation strongly enough; otherwise quarantine or mark as weak.

## Forbidden Phrases

Any future student entry containing these phrases should be rejected unless it is inside an audit report:

- `本题要求结合材料说明其体现的思维方法`
- `卷面要把`
- `先写`
- `要写`
- `本题需要`
- `设问要求`
- `采分点`
- `做选择题时先圈`
