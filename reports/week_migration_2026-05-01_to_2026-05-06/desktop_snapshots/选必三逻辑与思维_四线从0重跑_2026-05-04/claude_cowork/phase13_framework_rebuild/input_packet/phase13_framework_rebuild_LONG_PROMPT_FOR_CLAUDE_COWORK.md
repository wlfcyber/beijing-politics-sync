### 1. 产物不是知识点摘要，而是学生能迁移的宝典

哲学宝典的成功结构是：

`大框架 -> 原理/方法论节点 -> 模拟题条目`

每个模拟题条目的固定表达是：

- `【材料触发点】`
- `【设问】`
- `【为什么能想到】`

选择题也要保留类似链条：

- `【材料触发点】` 或 `【题干信号】`
- `【设问】`，含完整选项或完整组合项
- `【为什么能想到】`
- `【答案落点】`，含正确项、错项陷阱

# 选必三《逻辑与思维》思维与推理全触发宝典（Phase13 Cowork 框架重构候选）

## 使用方法

用短段告诉学生：

- 先看材料动作；
- 再判思维类型或推理题型；
- 再落到小方法/规则；
- 最后写成答案句或选择题排错动作。

不能写成冗长说明书。

## 第一部分：思维方法框架

### 一、科学思维

#### 1. 客观性

节点必须先写固定分析流程：

- `【材料怎么看】` 看调查、数据、事实、实际情况、真实需求、客观条件、规律约束。
- `【该写哪个方法】` 科学思维的客观性。
- `【为什么触发】` 因为科学思维要求从客观实际出发，如实反映对象，而不是凭主观想象。
- `【答案句怎么落】` 示例句要能直接上卷面。
- `【易错项怎么避】` 区分客观性与哲学“一切从实际出发”、与纯材料事实描述。

然后挂对应模拟题条目。每题用：

- `【材料触发点】`
- `【设问】`
- `【为什么能想到】`
- `【答案落点】`
- `【考场动作】`
- `【同类题】`

#### 2. 预见性

必须挂趋势研判、未来规划、老龄化、市场潜力、发展趋势、提前布局类题。

注意：不是所有“未来”都叫超前思维；科学思维预见性强调基于规律和事实作出合理判断。

#### 3. 可检验性

必须挂测试、迭代、验证、实践检验、效果反馈类题。

#### 4. 探索性与方法更新（仅题源支持）

仅当题源支持时使用。尤其注意：

`2024 海淀二模第17题第(1)问`

本轮回源结论是 `SCIENCE_ONLY_SOURCE_SUPPORTED`：

- 原题只问“此次时间利用调查是如何体现科学思维的”；
- 不能把设问改成科学思维、创新思维、辩证思维三模块并列；
- 但答案源支持“客观性、探索性/方法更新、整体安排”等科学思维设问下的具体落点；
- clean 版要说明这些是本题细则支持的具体落点，不是所有科学思维题万能模板。

#### 5. 整体安排（仅题源支持）

仅作为科学调查题的来源支持点，不要泛化为所有科学思维固定特征。

### 二、辩证思维

#### 1. 整体性

看多主体、多环节、系统协同、一盘棋、整体推进、统筹安排。

#### 2. 动态性

看阶段变化、发展进程、当前与长远、问题趋势变化、动态调整。

#### 3. 分析与综合

看先分解要素、方面、层次、环节，再综合成整体方案或整体认识。

#### 4. 质量互变

看数量积累、渐进推进、适度原则、量变到质变、度的把握。

#### 5. 辩证否定

看旧资源、旧模式、旧产业、传统元素的继承、改造、创新发展。

注意：不要把所有“继承借鉴”都机械写成辩证否定，选择题中要区分材料是否真考这个点。

### 三、创新思维

#### 1. 思路新、方法新、结果新

必须把“三新”拆开写：

- 思路新：看问题角度、联结对象、设计方向变化；
- 方法新：看技术路径、组织方式、采集方式、表达方式变化；
- 结果新：看形成新方案、新产品、新效果、新形态。

尤其注意：

`2025 东城期末第18题第(2)问`

登月服题建议拆成：

- 思路新：把传统文化形象、火箭形象与登月服设计联结；
- 方法新：围绕行走、攀爬、驾车、科考、防护、防眩光等任务需求发散并聚合；
- 结果新：形成兼具文化表达、任务适配和技术防护的登月服方案。

#### 2. 联想思维

看由一物想到另一物、跨领域形象联结、迁移和想象、借已有经验打开新设计。

#### 3. 发散思维与聚合思维

看多方案、多角度、多任务需求展开，再围绕目标筛选整合。

#### 4. 逆向思维

看反过来想、改变通常方向、从相反角度解决问题。

#### 5. 超前思维

看基于趋势和规律进行前瞻性布局、提前谋划、创造条件。

注意：

`2025 丰台期末第7题`

只能作为边界陷阱或哲学唯物论伪装，不能挂成超前思维正例。

#### 6. 改变条件 / 建立新的具体联系

看主动创造条件、调整关系、建立新联系、让原本分散的要素发生新组合。

### 四、认识发展历程

#### 1. 感性具体

看对对象外在、生动、丰富表象的直接把握。

#### 2. 思维抽象

看从感性材料中抽取本质、属性、关系、规律。

#### 3. 思维具体

看把多个抽象规定综合起来，形成对事物整体、本质、具体联系的把握。

注意：

`2026 通州期末第11题`

感性具体、思维抽象、思维具体链条可用；错项要压成学生能辨的版本：

- A 错在把认识历程误判为迁移；
- B 错在把认识历程误判为普遍和特殊的循环；
- D 错在把理念诠释误判为类比推理。

### 五、形象思维与抽象思维

按题源支持挂载，不要空写概念。重点区分形象表达、概念判断、抽象推理。

### 六、系统观念与材料事实边界

这里可以放系统化、数字化、治理整合类选择题，但要明确哪些是：

- 正向系统观念；
- 材料事实；
- 选择题陷阱；
- 不能迁移成主观题模板。

尤其注意：

`2026 通州期末第9题`

只能作为选择题陷阱 / 数字化治理材料事实与选必三方法区分，不能挂成主观题可迁移方法正例，也不能挂成发散聚合正例。

### 七、边界陷阱与易混选择题

必须单独设置，不要把边界陷阱塞进正向节点。

至少包含：

## 第二部分：推理题型框架

推理部分也不能按地区时间流水账。必须按题型树组织：

### 一、概念与定义 / 概念外延关系

固定流程：

- `【材料怎么看】` 看概念、种属、定义、外延关系。
- `【该用哪个规则】` 下定义规则、外延关系判断、概念划分规则。
- `【为什么触发】` 题目要求判断概念是否清楚、外延是否相容、定义是否犯错。
- `【解题动作】` 找属概念、找种差、看外延包含/交叉/全异。
- `【易错项怎么避】`

然后挂全部同类题。

### 二、判断类型识别

挂联言、选言、假言、性质判断等识别题。

### 三、联言判断与联言推理

规则口令：

- 联言判断全真才真，一假即假；
- 合成式、分解式；
- 不能把部分真当整体真。

### 四、相容选言推理

规则口令：

- 否定一支可以肯定另一支；
- 肯定一支不能否定另一支。

### 五、不相容选言推理

规则口令：

- 肯定一支可以否定另一支；
- 否定一支可以肯定另一支。

### 六、充分条件假言推理

规则口令：

- 肯定前件有效；
- 否定后件有效；
- 肯定后件无效；
- 否定前件无效。

硬锁样本：

- `2024 朝阳一模第20题第(1)问` 只挂充分条件假言推理。
- 表达要写清 P/Q：
  - P = 该地区古代没有广泛生长竹子；
  - Q = 不可能有大量炭化竹节；
  - 已知大量炭化竹节存在，就是否定 Q；
  - 所以否定 P，推出并非没有广泛生长竹子。
  - 不是“看到后件真后倒推前件”，而是否定后件式。

- `2025 西城二模第16题第(2)问` 只挂充分条件假言推理，肯定后件无效。

### 七、必要条件假言推理

规则口令：

- 否定前件有效；
- 肯定后件有效；
- 肯定前件无效；
- 否定后件无效。

硬锁样本：
