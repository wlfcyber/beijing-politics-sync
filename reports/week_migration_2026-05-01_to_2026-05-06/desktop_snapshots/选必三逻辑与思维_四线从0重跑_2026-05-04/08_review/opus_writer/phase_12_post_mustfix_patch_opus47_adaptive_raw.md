### MF3. `Q-2025海淀二模-20` 答案落点必须分句重写

> 角度池写法：可优先选"整体性 + 动态性"或"分析与综合 + 质量互变"等组合，核心是两角度写透。考场上优先选择材料最顺的两个角度，每个角度都写成"方法名 + 共享层次 + 推进共同富裕作用"；不要把所有角度堆成必答清单。

### MF5. `Q-2024朝阳一模-20-1` 必须给学生留下"考场口令"
现稿用 P/Q 形式抽象表达否定后件式，弱学生很难凭它在新材料上复用。要求在【答案落点】之后追加：

> 考场动作：先把题干"如果……就……"改写成"如果 P 那么 Q"；遇到"现在 Q 不真"就推"P 不真"，遇到"现在 Q 真"不能推 P 真。本题用的是"否定后件式 → 否定前件"，不是"肯定后件式"。

否则它对弱学生来说是"看懂了今天，下一道相似题还是不会"。

### MF6. `Q-2025顺义一模-7` 教学表达需补一行"四步走"
事实锁定无误，但当前讲解仍偏审计话，弱学生看完不一定能在新题上复用"先按结论定大小项、再判周延"的口令。要求在【答案落点】末尾追加：

> 考场动作（三段论谬误命名通用四步）：① 先看结论，找出谓项=大项、主项=小项；② 在结论与前提中各自标周延；③ 凡是"前提中没周延 → 结论里却周延"的项，就是"该项不当扩大"；④ 命名时务必带项的角色（大项 / 小项 / 中项），不要只写"不当扩大"。

并在【易错陷阱】下补：

> A 项给出的"小项不当扩大"是命名陷阱。"青年"在结论里是谓项，因此是大项不是小项；该题真实错误是大项不当扩大。

### MF7. `Q-2026丰台一模-18-2` 体例必须并入【】框

## should_fix（教学体验上强烈建议改，但不阻塞）

### SF1. 体例统一
- 主观题统一沿用【】块，字段顺序固定：`【材料触发点】→【设问】→【为什么能想到】→【答案落点】→【易错陷阱】→【考场动作】→【同类题索引】`。
- 选择题统一沿用：`【完整选项】→【题干信号】→【正确项】→【错项陷阱】→【考场动作】（可选）→【同类题】`。
- bullet 行式条目（如丰台一模 18(2)）和 `- 字段:` 行式条目（如海淀二模 20）统一回收。

### SF2. 学生稿状态横幅必须删
- `Status: REVIEW_ONLY_NO_WORD_NO_FINAL`、`Status: REBUILT_AFTER_MUST_FIX_CONTENT_REVIEW_ONLY` 等头部状态行只供审稿链使用，final student clean 必须剥离。
- HTML qid 注释 `<!-- question_id: ...; phase12_decision: ...; source_pool: ... -->` 同样不得进入学生稿。

### SF3. 错项陷阱口径统一
- 选择题错项陷阱当前混用"……过度 / 过头 / 拔高 / 偏离 / 不准 / 范围太宽"等多种判语。建议统一成 4 类标签：`绝对化 / 范围错位 / 方向倒置 / 类别错配`，括号内再说人话。例：

  - `绝对化（"就能""一定""完全消除"）`
  - `范围错位（"所有 / 全部 / 仅有"超出材料）`
  - `方向倒置（肯定后件、否定前件、属种顺序写反）`
  - `类别错配（把联想说成类比、把感性具体说成思维抽象等）`

  这样学生在不同年份不同区的同类陷阱之间能形成"陷阱字典"。

### SF4. 推理类题应统一给出"规则口诀 + 错位风险"两行
现已有该模式的样板：丰台一模 18(2)、丰台期末 18(1)。其他推理题（朝阳一模 20(1)/20(2)、通州期末 19(2)、西城二模 16(2)、海淀期末 8 等）建议补齐：

- 规则口诀：一句压缩判断方法（"前真后必真，否后必否前"等）。
- 错位风险：本题最容易错挂的相邻规则（必要条件 vs 充分条件、中项不周延 vs 大项不当扩大、归纳 vs 类比）。

### SF5. 边界陷阱条目必须更醒目
- `2025 丰台期末第7题`（漫画"把握当下"）已正确锁为边界陷阱，但条目内部仍写"本节点是边界陷阱"。建议改写为面向学生的红字提醒：

  > 提醒：本题不是选必三思维方法的正例。看到选项 B 出现"超前思维"不要选；漫画重心是"把握当下 / 从实际出发"，是哲学唯物论方向。

- `2025 海淀二模第13题`、`2025 海淀期末第8题`、`2026 朝阳期中第12题`、`2025 顺义一模第6题` 等"看似考逻辑、实际兼考哲学常识"的条目，可以在【正确项】前加一行"题型识别："说明它属于哪一类（选必三正考 / 哲学+逻辑混合 / 跨模块辨析），让学生知道该用哪一套答题模板。

### SF6. `2025 西城二模第6题`（属种关系：洛阳红/牡丹）排序
现稿在【错项陷阱】中说 D"《红楼梦》是四大名著之一，更接近个体与集合成员关系，不如 C 的种属关系稳定"——这种"不如 C 稳定"的措辞容易让学生误以为 D 也对。建议换成硬判：

> D 是个体与集合的成员关系，不是种属关系，故排除；C 谷雨与节气是稳定的种属关系，与"洛阳红 / 牡丹"同型，故选 C。

### SF7. `2026 通州期末第10题` D 项措辞
现稿："④'要么放弃继承，要么接受继承'并不构成相容选言判断，表达更接近不相容选择"。措辞较模糊。建议改硬：

> ④ 是不相容选言（两支不可同真且不可同假），不是相容选言。学生答题时只要看到"要么……要么……"+"两支互斥不可同真"就判不相容。

### SF8. `2024 朝阳二模第7题` 学生层注解
A 项 manual_lock 已正确锁为"小项不当扩大"。建议在【错项陷阱】里多写一句让学生记住命名口令：

> A 项：先按结论"娱乐工具都是思想政治教育的工具"找大小项 → "娱乐工具"是结论主项=小项；它在前提中作"自媒体也是娱乐工具"的谓项，肯定判断谓项不周延；却在结论"娱乐工具都是……"中周延 → 这是小项不当扩大，不要错记成中项不周延。

这条与 Q-2025顺义一模-7 的"大项不当扩大"形成对照对，弱学生应该看完两条之后能区分大小项。

### SF9. `2025 海淀期末第8题` D 选项学生层
现稿对 D 的解释"要么 P 要么 Q 为真，说明 P、Q 中恰有一个为真，因此或者 P 或者 Q 也一定为真"是对的，但弱学生需要更具象的口诀：

> 不相容选言为真 → 恰好一支真一支假；既然有一支真，"或者……或者……"的相容选言（至少一支真）当然成立。这是"强条件推弱条件"，方向不能反过来。

### SF10. `2025 海淀二模第13题` C 项三段论展示
C 是"单一制 → 法国 → 法国中央政府享有最高权力"。建议在【正确项】下追加"周延快查"：大前提"单一制国家中央政府享有最高权力"——主项周延、谓项不周延；小前提"法国是单一制国家"——主项周延；结论"法国中央政府享有最高权力"——主项周延、谓项不周延。结构无周延错。这能让学生学会用"周延快查表"判合规。

## final_clean_build_advice（学生 clean index 构建口令）

不授权立即构建 final / Word / PDF。下列口令仅供 final clean 阶段照办。

### 1. 状态与注释剥离
- 删除每份文件首部的 `Status:` 行。
- 删除每条 `<!-- question_id: ...; phase12_decision: ...; source_pool: ... -->` HTML 注释；question_id 如需保留请在条目末尾以 `(题号: Q-XXXX)` 形式以人话出现。
- 删除条目右侧的 `manual_lock:`、`phase06_logical_form_locked:`、`external_forced_check`、`source_locked_D`、`SCIENCE_ONLY_SOURCE_SUPPORTED`、`scientific_thinking_primary` 等英文审计尾注；学生稿只留 `[正文正例] / [选择题陷阱] / [边界陷阱] / [辅助挂载]` 四种中文标签。

### 2. NEEDS_* 节点替代命名（四线统一）
- `NEEDS_TYPE_CONFIRMATION` → `推理题型 · 辅助挂载（本题不作典型推理正例，仅供同类题检索）`
- `NEEDS_METHOD_CONFIRMATION` → `思维方法 · 辅助挂载（本题不作典型方法正例，仅供同类题检索）`
- 节点下条目右尾的 `no_phase06_locked_reasoning_row` / `no_manual_positive_mount_after_MUST_FIX_CONTENT; kept out of positive nodes` 全部去掉，仅留 `[辅助挂载]` 标签。
- 节点首行加一句学生提醒：
  > 提示：本节点条目并不是该索引意义上的典型正例。要看典型正例，请回到上方主节点；本节点条目仅在做完同类题归类时作为辅助检索。

### 3. 选择题条目"完整选项"补齐
对 MF1 列表中的所有选择题条目，必须从原题或卷面图复原 `【完整选项】` 字段并写入正文。这一步可由 ClaudeCode 辅助批量做，但每一条都需人工审核选项与现稿"错项陷阱"叙述对得上。

### 4. 主观题三件套补齐
对 MF4 列表中的所有主观题，补齐 `【易错陷阱】`、`【考场动作】`、`【同类题索引】`。其中【考场动作】不是套话，必须落到"考场上的下一步动作"，比如：

- 充分条件假言推理类：写出"如果 P 那么 Q"模板，并标"否后必否前 / 后真不能倒推前"。
- 必要条件假言推理类：写出"只有 P 才 Q"模板，并标"无前必无后 / 有后能推前"。
- 三段论类：标"先看结论定大小项 → 标周延 → 命名错误时带项的角色"。
- 创新思维类：标"先抄一遍方法名 → 立刻贴材料动作 → 不能只写帽子"。
- 辩证思维类：标"角度池中按材料挑两个写透，不堆必答清单"。

### 5. 体例统一
全册主观题字段顺序统一为：

```
【材料触发点】→【设问】→【为什么能想到】→【答案落点】→【易错陷阱】→【考场动作】→【同类题索引】
```

全册选择题字段顺序统一为：

```
【完整选项】→【题干信号】→【正确项】→【错项陷阱】→【考场动作】（可选）→【同类题】
```

bullet 行式（如丰台一模 18(2)、海淀二模 20）一律收编。

### 6. 锁定型条目教学注解前置
final clean 时，下列 manual_lock 条目要在【正确项】或【答案落点】前明确写出"本题答案 / 正确项已锁定"：

- `Q-2025顺义一模-7`：选 A；真实错误=大项不当扩大；A 项把它误称为小项不当扩大。
- `Q-2024朝阳二模-7`：A 项错误是小项不当扩大（不是中项不周延）。
- `Q-2026丰台一模-18-2`：甲=必要条件假言推理肯定后件式（成立）；乙=三段论大项不当扩大（不成立）。
- `Q-2024朝阳一模-20-1`：充分条件假言推理 · 否定后件式。
- `Q-2024朝阳一模-20-2`：必要条件假言推理 · 肯定后件式。
- `Q-2025西城二模-16-2`：充分条件假言推理 · 后件真不能推前件真。
- `Q-2026通州期末-19-2`：推理①充分条件肯定前件（成立）；推理②必要条件肯定前件（不成立）。
- `Q-2024海淀二模-17-1`：原题只问"科学思维"，不可拆为科学+创新+辩证三模块并列。

### 7. 边界陷阱与混类条目可视化
- `Q-2025丰台期末-7`：在条目顶端加红字"⚠ 本题是边界陷阱，不是选必三思维方法正例"。
- `Q-2026通州期末-9`：在条目顶端加"⚠ 本题考的是材料事实理解，不是创新思维 / 系统思维主观题方法正例"。

### 8. 双索引"考场用法"前言
final clean 时，两个索引文件首页都应有一段面向学生的简介，告诉学生：

- 推理索引按"判断类型 + 推理形式"分节；每节里 `[正文正例]` 是必背、`[选择题陷阱]` 是反例对照、`[辅助挂载]` 是同类题检索而不是正例。
