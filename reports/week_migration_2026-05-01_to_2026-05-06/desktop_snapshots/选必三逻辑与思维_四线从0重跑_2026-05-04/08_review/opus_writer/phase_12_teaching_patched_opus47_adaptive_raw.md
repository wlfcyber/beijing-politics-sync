## small_patch（不阻塞 gate，但建议在转 Governor 之前收一遍）

### SP1. 主观题"最小三件套"模板化告警
Codex 用以下三段样板句批量填充 12+ 条主观题（题号节选见下）：

模板 A（思维方法类，命中节点最多）：
- `【考场动作】先圈材料中的动作、关系和变化，再对应到小方法，最后写成"方法名 + 本题材料事实 + 作用结论"的卷面句。`
- `【易错陷阱】不要只写科学思维、辩证思维或创新思维总帽子；必须落到本题材料触发的小方法和具体作用。`
- `【同类题索引】同类题见思维方法索引中本题对应的正例或辅助节点。`

模板 B（推理类）：
- `【考场动作】先把题干改写成清楚的逻辑形式，判断属于哪一种推理或判断规则，再写"规则名 + 有效式/错误式 + 本题结论"。`
- `【易错陷阱】不要只写结论对错；必须说明前提、逻辑形式和有效/无效原因，否则容易把推理规则和材料事实混在一起。`
- `【同类题索引】同类题见推理题型索引中本题对应的题型节点。`

模板 C（交叉）：
- `【考场动作】先分清本题主考思维方法还是推理规则；若两者同时出现，按设问顺序分别写"材料动作/逻辑形式 + 方法或规则 + 结论"。`

命中模板填充的主观题（部分）：
`Q-2025海淀期末-17-1`（永动机）、`Q-2025海淀期末-18`（图书馆）、`Q-2024海淀二模-17-2`（认识发展历程）、`Q-2025西城二模-16-2`、`Q-2025西城二模-16-3`、`Q-2024西城一模-19-2`、`Q-2024西城一模-19-3`、`Q-2024西城一模-19-5`、`Q-2026东城一模-19-4`、`Q-2026朝阳期中-20`、`Q-2026朝阳期中-21-2`、`Q-2024朝阳一模-20-1`（除已加的考场口令外，三件套是模板）、`Q-2024朝阳一模-20-2`、`Q-2024朝阳二模-19-1`、`Q-2024朝阳二模-19-2`、`Q-2026顺义一模-19-1`、`Q-2026顺义一模-19-2`、`Q-2026通州期末-19-2`、`Q-2025顺义一模-17-1`。

问题：
- 模板满足"最小三件套字段存在"的硬 gate；但同一句话出现在 12+ 条不同题目下，对学生而言失去针对性，与本套书"每题只讲这道题怎么读"的卖点反向。
- 部分原稿中已有的具体同类题号清单（如 `2025海淀二模-20` 早期就列出"2024 朝阳一模第9题；2024 朝阳二模第19题第(1)问；…"等）在模板化填充的条目里被替换为"见索引"的 meta 指针。这等于把"同类题清单"功能下放到索引文件，对纸质阅读和考前快速翻阅不友好。

建议：在 final clean 之前（不晚于 Governor 闸口前）做一次"反模板"补丁：

- 对每条命中模板的主观题，至少把【易错陷阱】写成"本题材料独有"的一两条具体陷阱（例：永动机条目应写"不能写成创新思维或辩证否定，只考客观性"；图书馆条目应写"不能只写'有创意'，逆向 / 联想必须配回'人找书→书找人'与'赤印意象迁建筑'两个动作"；朝阳一模20(1) 应写"不能把否定后件式误写成肯定后件式倒推前件"）。
- 对【考场动作】，至少把题型对应的"动作三步"写到题面级别（例：朝阳一模20(2) 必要条件类应写"先把题干改写成'只有 P 才 Q' → 看到 Q 真就能推 P 真 → 否则不能"）。
- 对【同类题索引】，能列具体题号的，列出来；只在确实没有同类题号时才用 meta 指针。可优先从 `phase12_reasoning_typology_index_TEACHING_PATCHED_REVIEW_ONLY.md` 与 `phase12_thinking_method_index_TEACHING_PATCHED_REVIEW_ONLY.md` 的同节点条目里抄题号。

### SP3. `Q-2024朝阳一模-20-1` 字段顺序与重复
该条目当前字段顺序是：

```
材料触发点 → 设问 → 为什么能想到 → 考场口令 → 答案落点 → 考场动作 → 易错陷阱 → 同类题索引
```

`【考场口令】`（具体、有针对性，是上一轮 MF5 的核心修复）与 `【考场动作】`（模板句）含义重叠，且口令在前、动作在后，反而让模板句覆盖了具体口令的视觉地位。

建议：

- 把 `【考场口令】` 的内容并入 `【考场动作】` 字段（保留具体口诀），删掉模板版【考场动作】，避免一题两段动作。
- 让 `【易错陷阱】` 写成"否后必否前；不能把本题写成肯定后件式 P→Q→……"这一题相关版本，删模板。

同样的字段重复也出现在 `Q-2025顺义一模-7`（`【四步口令】` + 答案落点同时承担"动作 + 命名"职能），但顺义一模7 没有再叠模板【考场动作】，所以可不动；只看朝阳一模20(1)。
