## 二、结构性失败原因（对四要件逐项分析）

### 1. `【设问】` 假化：占 100%

`bad_word_four_element_failure_matrix.csv` 中 181 条全部使用三类模板假设问之一，没有一条是真实题目原文。这违反 `xuanbisan-hard-rules-notebook.md` 第十二补一「`设问` 必须是真实题目设问句；禁止用 `本题要求结合材料说明其体现的思维方法` 之类模板句冒充」。

后果：

- 学生看到 `【设问】` 仍不知道考题真正在问什么，也无法据此识别细则评分位（例如「分析」「评价」「提出做法」「补全推理」「指出错误」等关键动作词被全部抹平）。
- 同题进入不同节点时，由于设问栏永远一样，学生看不到不同节点应当回应的不同题目要点。
- 同书内三类模板分别对应三种题型，等于把「主观思维」「选择思维」「推理」三类全部用模板假冒过去，没有一条经过题源回溯。

### 2. `【答案落点】` 制作说明化：占 89%（161/181）

矩阵中 `INSUFFICIENT_meta_instruction = 161`：

- 思维主观题：`卷面要把材料中的具体动作写进方法里：……` 接一段以「动作 → 写××」为骨架的写作指令，而不是直接写出考生答题纸上的句子。
- 思维选择题：`……。 因此做选择题时先圈材料动作，再看选项是否把方法、主体、关系或作用说准。`，把答题落点替换成做题口令。
- 推理：`先翻译逻辑结构，再套本题型最小规则，最后判断能否必然推出。`，跳过本题真实结论。

这违反 `xuanbisan-hard-rules-notebook.md` 第七、第十二补一：
> `答案落点` 必须是学生能直接写在答题纸上的答案句，结构是 `思维方法/术语 + 本题材料事实 + 因果/作用/结论`。
> 禁止 `卷面要把...`、`先写...`、`要写...`、`本题需要...`、`设问要求...`、`采分点...`。

后果：考生看完整页四要件，不知道这道题答题纸上要写什么，只看到「写法指引」。

### 3. `【为什么能想到】` 与 `【答案落点】` 高度同义重复

矩阵中绝大多数条目里 `【为什么能想到】` 写完一段「方法名 + 写作步骤」之后，`【答案落点】` 直接复制其后半句，前面再加 `卷面要把材料中的具体动作写进方法里：`。例如：

> 【为什么能想到】 ……一看到调查商品样本 → 查标准 → 补方法 → 形成提案，就先写获取客观材料，再写对资料做分析综合，最后写据此判断预测并形成可执行方案。
> 【答案落点】 卷面要把材料中的具体动作写进方法里：一看到调查商品样本 → 查标准 → 补方法 → 形成提案，就先写获取客观材料，再写对资料做分析综合，最后写据此判断预测并形成可执行方案。

这等于把同一句话写两遍，没有形成「先解释思路 → 再写卷面句」的递进关系。

### 4. 同题多节点复制粘贴：占 63%（114/181）

矩阵 `node_specific=NO_multi_node_copy` 计 114。多数典型例：

- `2024丰台一模 Q19(2)` 在「科学思维三特征」「调查研究与研究设计题」「分析与综合 / 整体性」「发散思维与聚合思维」「超前思维」等节点下重复出现，四要件文字完全一致，没有按节点重新切问/重写答案落点。
- `2025门头沟一模 Q21(1)`、`2025海淀二模 Q20`、`2026丰台期末 Q20`、`2025东城期末 Q18(2)`、`2026朝阳期中 Q21(2)` 等复合题在多个节点下都是同一段文字。
- 这违反第十二补一：「同一道题多节点挂载时，每个节点都要针对该节点重写四要件；不能把同一套'科学思维 + 辩证思维 + 创新思维'的总表复制到不同框架节点下面。」

后果：

- 学生进入「分析与综合」节点时看到的不是「这道题在分析与综合上要怎么写答案句」，而是和「科学思维」节点完全相同的全方法清单，节点的教学价值被掏空。
- Word 体量被人为放大（181 条里有 2-3 倍冗余），但教学密度反而被稀释。

### 5. 复合题方法清单化，没有逐点对应

矩阵中大量复合题（如 `辩证否定 + 整体性/动态性 + 联想/发散 + 超前思维 + 科学思维`）只罗列方法名再贴写作指令，没有「材料信号 → 总帽子 → 小方法 → 答案句」逐点拆分。这违反第十二补一：「复杂题必须拆出 `材料信号 -> 总帽子 -> 小方法 -> 答案句` 的逐点对应；只列多个方法名属于空心条目。」

### 6. 选择题四要件没有错项分析与陷阱类型

矩阵中 29 条选择题：

- `【设问】` 全员模板假化。
- `【答案落点】` 全员收尾 `因此做选择题时先圈材料动作...`。
- 没有给出题干信号、四个选项、正确选项理由、诱人错项理由、陷阱类型这五件事。
- 这违反 `xuanbisan-hard-rules-notebook.md` 第八「选择题硬规则」与第十二补一「选择题四要件不能只写'圈材料动作再看选项'；必须呈现题干信号、正确项理由、诱人错项理由和陷阱类型」。

### 7. 推理部分混入「Q相关」假题号 + 模板设问

矩阵中 51 条推理条目：

- `【设问】` 全部模板假化。
- 有 `2026顺义一模 Q相关`、`2025朝阳期末 Q相关`、`2026海淀期中 Q相关` 等 `Q相关` 字样直接进入 Word；这些条目实际上来源不固定、题号未确认，应被归入 `quarantine_until_source`。
- `【答案落点】` 大量为 `先翻译逻辑结构，再套本题型最小规则，最后判断能否必然推出。`，没有写出本题真正的有效式 / 错误式 / 结论。

### 8. 学生稿与审计稿混用语言

矩阵抽样发现部分条目仍带「高频陷阱」「分析步骤：①②③」字段，本是审计/工作日志写法（哲学宝典、phase11A 草案严格分离）。坏 Word 把这种字段直接搬入学生稿，违反 `xuanbisan-hard-rules-notebook.md` 第十一「学生版与审计版分离」原则。

## 四、为什么这是「内容失败」而不是「格式问题」

- 哲学宝典对照展示的不是排版差距，而是「设问真不真、答案落点是不是卷面句、节点是不是真在解释为什么进这个节点」。
- 排版 / 字号 / TOC 都不能补救假设问。即使把 `.docx` 重新美化、加目录加水印，学生在每页看到的仍然是「本题要求结合材料说明其体现的思维方法 + 卷面要把材料中的具体动作写进方法里」。
- 所以：本卷必须先重建 Markdown 内容，再做内容审，才能进 Word。
