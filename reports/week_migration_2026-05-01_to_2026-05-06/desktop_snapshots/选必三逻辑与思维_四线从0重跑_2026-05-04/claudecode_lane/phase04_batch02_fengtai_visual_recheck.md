Let P = 开展全市性志愿服务活动; Q = 使用标识. Given P→Q is TRUE.

**Options**:
- ① 不开展全市性志愿服务活动，就必然不使用标识 → ¬P→¬Q: **denial of antecedent, invalid** ✗
- ② 不使用标识，就不是开展全市性志愿服务活动 → ¬Q→¬P: **modus tollens, valid** ✓
- ③ 有些使用标识的活动是全市性志愿服务活动 → ∃x(Qx ∧ Px): **valid** (P is non-empty in reality; since ∃P and P→Q, then ∃(P∧Q), meaning some Q-members are also P-members) ✓
- ④ 使用标识，就是开展全市性志愿服务活动 → Q→P: **affirming consequent, invalid** ✗

**Answer C = ②③** — confirmed from answer key.

**Lane B note on ③**: The existential inference holds because P (全市性志愿服务活动) is realistically non-empty; the question context establishes the standard exists and is in use.
