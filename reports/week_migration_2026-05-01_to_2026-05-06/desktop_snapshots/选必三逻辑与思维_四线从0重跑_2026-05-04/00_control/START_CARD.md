# Start Card

- 用户原始启动语: “内容质量没法比，严格按照之前的四线工作流程进行，不要偷懒。”
- 本轮书目: 选必三《逻辑与思维》
- 本轮范围: 思维部分 + 推理部分
- 本轮模式: 四线从0重跑，目标是哲学宝典同等质量的最终学生文档
- 从0/续跑声明: 不继承旧结论；旧稿只用于定位源材料和暴露遗漏风险
- 用户给定考题框架: 思维部分像哲学一样按每个思维/方法组织；推理部分先分类题型，再把所有题放到对应题型下面
- 默认证据范围: 本机 2024-2026 北京区级模拟题、试卷、答案、评分细则、讲评、阅卷/评标材料；用户上传的思维/推理框架 PDF 作为框架学习材料
- 特殊禁区: 2026石景山期末仍按用户既定规则排除，除非用户提供新评分来源；普通参考答案不得冒充正式细则
- 交付要求: Markdown + Word + 可行 PDF/截图 + FINAL_ACCEPTANCE_REPORT；Word 必须用 Microsoft Word 更新目录/页码并保存验收
- 待确认但不阻塞项: 最终可先合并成一本“思维部分+推理部分”学生版；若用户后续要求，也可拆成两本分册
