# 审稿意见

## 总体判断

框架是站得住的：思维部分的 `材料动作 → 总帽子 → 小方法 → 触发逻辑 → 答案句` 五步链条确实做到了"按哲学题处理"，推理部分按题型族群拆分也清楚。但有几处会直接误导学生的硬伤必须改，因此本稿不能直接出。

最终判定：**CONDITIONAL PASS**（按下列必改清单修订后可发学生）。

## 必改清单（按原文标题定位）

### 1. `三、创新思维 → 1. 创新思维"三新"`
**问题**：把课本原话"思路具有多向性，步骤具有跨越性，结果具有独特性"直接等同于"思路新、方法新、结果新"。这是错的。
- "步骤的跨越性" ≠ "方法新"。
- 学生背了"三新"上考场，遇到考"跨越性"的题会失分。

**改法**：保留课本三特征作为可写原理，把"三新"降级为口诀；明确写"三新是记忆口令，不能替换答题术语"。

### 2. `三、创新思维 → 5. 超前思维`
**问题**：原文写"方法题要写三招"，但列了四个（调查研究、矛盾分析、推理、想象）。这是结构性矛盾。
- 课本口径是 **矛盾分析法 + 推理与想象** 两条主方法，调查研究是基础/前提，不是并列的"招"。

**改法**：要么改成"两招加一基础"，要么写"三招：矛盾分析、推理、想象"并把"调查研究"剥离到前置基础句中。这一处必须统一，不能含糊。

### 3. `二、辩证思维 → 2. 动态性、质量互变` 与 `5. 动态性专项`
**问题**：同一章下出现两个"动态性"小节，材料信号高度重叠（都讲"持续展开""不断变化"），学生无法判断该套哪一个。

**改法**：把 #2 收窄为"质量互变规律"专门方法，标题改成"质量互变（量变质变）"；把 #5 改成纯"动态性"特征。两者一个是方法，一个是特征，必须切开。

### 4. 跨章节歧义：`一、科学思维 → 2. 结果具有预见性` 与 `三、创新思维 → 5. 超前思维`
**问题**：两处材料信号几乎一样（都写"预测趋势""前瞻布局""提前研判"），但答题方向完全不同——一个落"科学思维特征"，一个落"创新思维方法"。学生看完会两边都不敢套。

**改法**：在两处任一节末尾加一段 **辨析口令**，例如：
- 单纯讲"做出准确预测/方向判断"→ 科学思维·预见性
- 讲"用矛盾分析+推理想象去研判未来不确定性"→ 超前思维

这是本稿最隐蔽但最致命的盲区，必加。

### 5. `感性具体、思维抽象、思维具体` 节末"选择题样板"
**问题**：以"2026 通州期末第 11 题"的具体题源给出原题与正解 C。我无法核验本地题源，按你的要求也不应替你确认。如果该题号、答案不是你亲手对过原卷，发给学生后被反查到题源对不上，整本稿的可信度会塌。

**改法**：要么补一句"题号待校"，要么改为"仿编例题"。原文模式"来源例题"的出处属性必须能被你本人 1:1 复核，否则降为"示例题"。

### 6. `推理部分 → 七、求异法`
**问题**：定性写成"不完全归纳推理中的求异法"。课本里 **求异法** 是"探求因果联系的方法"四法之一（求同、求异、共变、剩余），与归纳推理是关联但不是从属。学生若按"求异法 = 不完全归纳的一种"答非选择题，会被扣点。

**改法**：标题/定义里改写为"求异法（探求因果联系的方法之一，常配合不完全归纳使用）"。

## 建议改（非阻断，但建议一并处理）

- **`三、选言判断与选言推理`**：补一行最致命错点——"把'或者'当'要么'后用肯定否定式"在高考改卷里几乎每年都踩，建议把这个错例独立成行加粗。
- **`八、周延判断与换质换位`**：目前只有口诀没有最小例子，学生看完不会落笔。每条口令后加一个一行例子（如"所有 S 是 P → 所有 S 不是非 P"）。
- **`九、逻辑三律`**：同上，三律每条配一句最短违反例（"刚才说的'人'和现在说的'人'不是一个人 → 偷换概念 → 违反同一律"）即可。
- **`一、三段论` 解题步骤**：第 3-6 步建议合并成一个"四查表"，否则学生临场来不及一条条核。

## 回答你提的四个具体问题

1. **思维部分是否真像哲学题一样从材料动作触发方法？** —— 结构上是的，但有两处 **信号重叠区**（预见性 vs 超前思维；动态性两节）会让"材料动作触发"失灵，必须加辨析口令。
2. **推理部分题型族群是否清楚？** —— 主干清楚（三段论 / 假言 / 选言 / 联言 / 归纳 / 类比 / 求异 / 周延换质换位 / 三律），但求异法的归属定位有误，周延和三律两节过薄。
3. **术语错误**：创新思维"三新"等同三特征（错）、超前思维"三招"列了四个（自相矛盾）、求异法归属（不准确）。这三处是真错。
4. **必须改的段落**：上方"必改清单"的 6 处，最优先级是 1、2、4 三项，因为它们会直接造成学生在大题里写错答题术语。
