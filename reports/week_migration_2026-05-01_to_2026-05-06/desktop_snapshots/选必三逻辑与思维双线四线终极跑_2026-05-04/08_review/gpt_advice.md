## 3. codex_A_assignment

Codex A 应承担证据锁定主线，任务重点是把候选表变成可进入教研成果的证据条目。

### 3.1 核心任务

Codex A 不应直接写学生稿。它应产出四类文件：

1. **Evidence Lock Table：证据锁定表**
2. **Rejected Candidate Log：剔除候选日志**
3. **Ambiguity Queue：待人工复核歧义队列**
4. **Cluster Matrix：题型族群归并矩阵**

### 3.2 候选到证据锁定的流程

每条候选必须经过以下流程：

1. **候选定位**

   * 候选 ID
   * 原始命中词
   * 本地来源路径
   * 年份、区、阶段、套卷信息
   * 题号或页码
   * 来源类型：试卷、答案、评分细则、讲评、旧稿、缓存

2. **题源配对**

   * 题面
   * 选项或设问
   * 参考答案
   * 评分细则或正式讲评
   * 必要时补充答案表
   * 主观题必须配齐设问、材料、答案、细则
   * 选择题必须配齐题干、选项、正确答案、解析或答案依据

3. **正负行识别**

   * 正向给分点
   * 错答提醒
   * 不得分说明
   * 模块混淆说明
   * 讲评中的排除项
   * 仅用于定位的旧稿行

4. **框架标签判定**

   * 主标签：只能一个
   * 副标签：可选，但必须说明理由
   * 所属线：思维线或推理线
   * 题型族群：例如假言推理、思维抽象与思维具体、整体性等
   * 归类证据：必须引用本地题面或细则中的具体材料动作

5. **规则与材料动作绑定**

   * 识别入口：学生看到什么材料动作应启动该方法或规则
   * 规则表述：本题实际考查的逻辑规则或思维方法
   * 常见设错：从本题选项、错答、讲评中提取
   * 答案句骨架：仅做教师内部草案，不能直接作为最终学生稿

6. **状态标记**

   * candidate_only：仅候选，不能使用
   * source_paired：题面与答案已配对
   * scoring_verified：评分细则或正式讲评已核验
   * evidence_locked：可进入教学化加工
   * rejected_false_positive：误命中
   * rejected_old_draft_only：只有旧稿定位，无正式证据
   * unresolved_missing_pair：缺题面、答案或细则
   * unresolved_module_conflict：模块边界不清

### 3.3 Evidence Lock Table 建议字段

每条 evidence_locked 至少包含：

1. lock_id
2. candidate_id
3. 年份
4. 区
5. 阶段
6. 题号
7. 题型：选择题或主观题
8. 来源路径
9. 来源类型
10. 题面摘录，本地核验用
11. 设问或选项摘录，本地核验用
12. 答案或评分细则摘录，本地核验用
13. 主标签
14. 副标签
15. 题型族群
16. 识别入口
17. 核心规则或原理
18. 常见设错
19. 模块边界说明
20. 证据等级
21. 锁定状态
22. 复核人或复核轮次
23. 备注

### 3.4 Codex A 的特别压力测试

Codex A 必须重点清洗以下误命中：

1. 普通语言中的或者、要么、只要、只有、如果。
2. 评分细则中的错答、不得分、不给分、混淆为、误写为。
3. 讲评中的排除选项，被误当成正向知识点。
4. 必修四哲学表述，被误归为选必三思维方法。
5. 一般方法论表述，被误归为具体推理规则。
6. 旧稿中的归类，被误当成本轮证据结论。

## 6. highest_risks

按严重程度排序如下。

### 1. 候选索引被误当成证据

这是最高风险。候选命中只能说明某处文本可能相关，不能说明题源、答案、细则和归类成立。

**控制办法：** 所有最终条目必须有 evidence_locked ID。

### 2. 题面、答案、评分细则错配

同一年、同一区、同阶段可能存在多份文件、答案版、讲评版。错配会直接污染最终结论。

**控制办法：** 每题必须锁定来源路径、题号、页码或文本位置，并保留配对证据。

### 3. 负面评分行被误识别为正向给分点

错答、不得分、混淆模块一类语句会反向污染学生稿。

**控制办法：** 所有细则行必须标记 positive_scoring、negative_warning、module_confusion、irrelevant_context。

### 4. 普通语言误判为逻辑推理题

或者、如果、只有、要么等词既可能是自然语言，也可能是真正逻辑规则考点。

**控制办法：** 只有当题面或解析明确考查判断类型、推理有效性、规则使用时，才能归入推理题型族群。

### 5. 必修四哲学与选必三思维串线

整体、联系、发展、矛盾、辩证否定等表述容易跨模块混用。

**控制办法：** 每条思维线证据必须写清：该题为什么归入选必三逻辑与思维，以及与必修四哲学表述的边界。

### 6. 推理规则讲成经验技巧

假言、选言、三段论、换质换位都有严格规则。若只讲关键词，学生会在变式题中失效。

**控制办法：** 每个推理族群必须保留规则、有效式、无效式、设错方式。

### 7. 思维方法讲成抽象口号

科学思维、辩证思维、创新思维若只列名词，学生无法从材料识别。

**控制办法：** 每个方法必须绑定材料动作，例如抽取核心概念、综合成整体图景、逆向设想、发散方案等。

### 8. 低频题型过度概括

某些题群样本少，若强行总结为北京高考稳定规律，会造成过拟合。

**控制办法：** 区分高频稳定规则、低频补充规则、仅见个案。

### 9. 旧选必三稿污染证据链

旧稿可帮助定位，但不能替代正式题源、答案、细则。

**控制办法：** 旧稿来源一律不得进入证据等级，只能标为定位线索。

### 10. 教学化后丢失可追溯性

学生稿语言顺滑后，容易看不出每个结论来自哪条证据。

**控制办法：** 教师版或内部附录保留 evidence_locked ID 映射。

## 8. acceptance_checks_before_final

Markdown、Word、PDF 交付前必须通过以下检查。

### 8.1 证据完整性检查

1. 每个例题、变式、规则归纳都有 evidence_locked ID。
2. 每个 evidence_locked 条目都有题面、答案、评分细则或正式讲评配对。
3. 每个主观题都有设问与评分细则配对。
4. 每个选择题都有题干、选项、答案与解析依据。
5. 所有旧稿来源均未作为证据进入最终结论。

### 8.2 归类一致性检查

1. 每题只能有一个主标签。
2. 副标签必须说明用途，不能替代主标签。
3. 思维线与推理线不得混写。
4. 必修四哲学、选必三思维、一般系统观念必须有边界说明。
5. A 与 B 对同一题的冲突必须处理完毕，不能带冲突进入 Opus。

### 8.3 误命中清洗检查

必须专项检索并复核：

1. 或者
2. 如果
3. 只要
4. 只有
5. 要么
6. 可能
7. 必然
8. 错答
9. 不得分
10. 不能得分
11. 混淆
12. 误写为

这些词所在行不能自动判为正向知识点。

### 8.4 推理规则检查

1. 假言推理有效式与无效式无误。
2. 选言推理区分相容与不相容。
3. 联言推理区分分解式与合成式。
4. 三段论检查中项周延、前提否定、结论周延范围。
5. 换质换位检查主谓项位置、联项变化、谓项否定、周延限制。
6. 归纳与类比统一标注为或然推理，除非教材或本地细则有特殊表述。

### 8.5 思维方法检查

1. 每个思维方法都有材料动作触发条件。
2. 每个答案句都能回扣材料，而非只背原理。
3. 科学思维预见性与超前思维有区分。
4. 整体性、动态性、辩证否定未直接搬用必修四哲学模板。
5. 思维抽象与思维具体样本必须出现三段链条：杂多现象、核心概念、完整整体图景。

### 8.6 教学可用性检查

1. 学生能按识别入口启动解题。
2. 每节有规则、例题、变式、避坑提示。
3. 口令不替代规则。
4. 答案句自然，避免机械堆术语。
5. 题型族群结构清晰，不能按年份流水排列。

### 8.7 交付格式检查

1. Markdown 标题层级稳定。
2. Word 目录、页码、表格不乱。
3. PDF 渲染无断表、错页、乱码。
4. 所有内部证据 ID 与附录一致。
5. 学生版和教师版内容边界清楚。
6. 2026 石景山期末默认排除，除非已有用户提供的新评分细则并完成单独锁定。

## 9. specific_warnings

以下话术、归类、规则写法最容易误导学生，应在 A、B、Opus 三线中共同设为红线。

### 9.1 关于选言推理

危险写法：

1. 看到或者就是选言推理。
2. 只要材料有二选一，就是不相容选言。
3. 肯定一个选言支，就能否定另一个选言支。

正确处理：

1. 先判是否真在考选言判断或选言推理。
2. 再判相容选言或不相容选言。
3. 相容选言中，肯定一个选言支通常不能推出否定另一个选言支。
4. 不相容选言才可使用肯定否定式和否定肯定式。

### 9.2 关于假言推理

危险写法：

1. 看到如果就固定判充分条件。
2. 否定前件可以否定后件。
3. 肯定后件可以肯定前件。
4. 只有和只要不加区分。

正确处理：

1. 先把前件、后件标出来。
2. 判断充分条件、必要条件、充要条件。
3. 充分条件假言推理中，肯定前件、否定后件是有效方向。
4. 必要条件假言推理中，否定前件、肯定后件是有效方向。
5. 任何倒置条件的结论都要回到规则核查。

### 9.3 关于三段论

危险写法：

1. 只看三个概念，不查中项。
2. 只看结论顺眼，不查周延。
3. 忽略两个否定前提、两个特称前提等规则风险。
4. 把材料中的普通归类句直接当三段论。

正确处理：

1. 标出大项、小项、中项。
2. 检查中项是否至少周延一次。
3. 检查前提中不周延的项在结论中是否被扩大。
4. 检查否定前提与特称前提带来的限制。
5. 再判结论是否必然推出。

### 9.4 关于换质换位

危险写法：

1. 换质只理解为换个说法。
2. 换位只理解为主谓颠倒。
3. 不查周延限制。
4. 把换质、换位、换质位混成一个技巧。

正确处理：

1. 换质要改变联项，并把谓项换成其矛盾概念。
2. 换位要交换主项和谓项，并遵守周延限制。
3. 换质位是连续操作，应逐步检查。

### 9.5 关于联言推理

危险写法：

1. 有几个条件并列就一定是联言推理。
2. 知道一个支判断成立，就推出整个联言判断成立。
3. 把部分成立写成全部成立。

正确处理：

1. 联言判断为真，要求各联言支都真。
2. 从全体联言判断可推出任一支判断。
3. 从若干支判断推出联言判断，必须确保全部支判断都成立。

### 9.6 关于归纳推理和类比推理

危险写法：

1. 归纳推出必然结论。
2. 类比推出必然结论。
3. 只要相似就能类比。
4. 只要样本多，归纳就可靠。

正确处理：

1. 归纳和类比通常属于或然推理。
2. 归纳要看样本代表性、数量、范围、是否存在反例。
3. 类比要看相同属性与推出属性之间是否相关。
4. 差异属性越关键，类比可靠性越低。

### 9.7 关于逻辑三律

危险写法：

1. 矛盾律和排中律混用。
2. 任何前后变化都判违反同一律。
3. 把概念偷换全部写成自相矛盾。

正确处理：

1. 同一律重点看同一思维过程中概念和判断是否保持确定。
2. 矛盾律重点看两个互相矛盾或反对的判断不能同真。
3. 排中律重点看两个矛盾判断不能同假。
4. 概念偷换优先回到同一律或概念外延内涵变化核验。

### 9.8 关于整体性与动态性

危险写法：

1. 看到整体、系统、全局就判辩证思维整体性。
2. 看到变化就判动态性。
3. 直接套必修四整体与部分原理。

正确处理：

1. 整体性要看是否从整体、结构、层次、功能关系中把握对象。
2. 动态性要看是否把对象放在变化发展过程、阶段转换、趋势演化中把握。
3. 若题源细则指向必修四哲学，应按本地证据处理，不能强归入选必三。

### 9.9 关于分析与综合

危险写法：

1. 分析等于列举。
2. 综合等于总结。
3. 看到分点论述就判分析与综合。

正确处理：

1. 分析强调把整体分解为要素、部分、层次、方面。
2. 综合强调在分析基础上把握对象整体联系和本质。
3. 要核验材料是否存在分解再整合的动作。

### 9.10 关于质量互变与适度原则

危险写法：

1. 任何数量变化都判量变质变。
2. 任何把握分寸都判适度原则。
3. 适度原则写成折中主义。

正确处理：

1. 质量互变要看量的积累、度的范围、关键节点、质的变化。
2. 适度原则要看是否在度的范围内保持事物性质稳定。
3. 材料若强调突破临界点或防止过度，才适合进入该框架。

### 9.11 关于辩证否定

危险写法：

1. 辩证否定写成全盘抛弃。
2. 辩证否定写成外部力量消灭旧事物。
3. 只要有创新就判辩证否定。

正确处理：

1. 重点看自我否定、扬弃、保留合理因素、克服旧因素。
2. 要把否定与发展联系起来。
3. 创新思维题不能自动归为辩证否定。

### 9.12 关于思维抽象与思维具体

危险写法：

1. 把抽象理解为脱离现实。
2. 把具体理解为举例子。
3. 看到概念就判思维抽象。
4. 看到全面分析就判思维具体。

正确处理：

1. 感性具体对应杂多现象的直观呈现。
2. 思维抽象对应抽取本质属性、核心概念、关键规定。
3. 思维具体对应把多方面规定综合为完整整体图景。
4. 本轮硬样本必须优先核验三段链条是否完整。

### 9.13 关于创新思维

危险写法：

1. 联想、发散、聚合、逆向、超前混用。
2. 看到新办法就判创新思维。
3. 看到预测就判超前思维。
4. 把超前思维和科学思维预见性合并讲。

正确处理：

1. 联想看对象间的关联迁移。
2. 发散看多方向、多方案展开。
3. 聚合看多信息、多路径向一个目标收束。
4. 逆向看反向思考、反向操作、反向设问。
5. 超前思维看立足现实根据，对未来趋势进行预测和规划。
6. 科学思维预见性与超前思维要根据题源细则和材料动作区分。
