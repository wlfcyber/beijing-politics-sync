# User Framework

本文件吸收用户上传 PDF，并作为本轮学生稿框架。PDF OCR 有少量错字，关键结构以页面渲染和文本抽取共同校正。

## 一、总口径

- 选必三《逻辑与思维》分两块：`思维部分`与`推理部分`。
- `思维部分`和哲学题完全同构：每一个思维、方法都按哲学“原理和方法论”的方式处理。
- 学生不先背定义，先看材料动作：材料在调查、预测、统筹、转化、联结、反向、抽象，分别触发不同思维方法。
- 本轮最终教学稿要能让学生完成：`材料动作 -> 总帽子 -> 小方法 -> 触发逻辑 -> 答案句`。

## 二、思维部分框架

### 1. 逻辑与思维总框架

- `逻辑`
- `科学思维`
- `辩证思维`
- `创新思维`
- `P.思维`

### 2. P.思维基础

- 思维特征：`间接性`、`概括性`、`能动性`。
- 实践关系：立足实践，在实践中产生，反作用于实践。
- 抽象思维：
  - 基本单元是概念；
  - 运行方式是推导；
  - 表达具有严谨性。
- 形象思维：
  - 基本单元是感性形象；
  - 具有形象性、想象性、情感性。

### 3. 科学思维

- 三个稳定特征：
  - 追求认识的客观性；
  - 结果具有预见性；
  - 结果具有可检验性。
- 课堂触发：
  - 调查、数据、样本、实验、验证、迭代 -> 客观性/可检验性；
  - 趋势研判、未来布局、提前规划 -> 预见性/超前思维。

### 4. 辩证思维

- 总口径：用联系、发展、矛盾的观点看待问题和思考问题；本质和核心是矛盾分析法。
- 高频小方法：
  - 整体性；
  - 动态性；
  - 分析与综合；
  - 质量互变与适度原则；
  - 辩证否定；
  - 思维抽象与思维具体。
- 分析与综合：
  - 分析是综合的基础；
  - 综合是分析的先导；
  - 二者对立统一、相辅相成。
  - 分析方法：定性分析、定量分析、功能分析、因果分析、矛盾分析法。
  - 综合方法：结构综合、功能综合。
- 质量互变：
  - 量变是质变的必要前提；
  - 质变是量变的必然结果；
  - 适度原则强调得中、把握分寸，不是折中主义。
- 辩证否定：
  - 是联系的环节；
  - 是发展的环节；
  - 实质是扬弃；
  - 方法论上要求树立创新意识。
- 认识发展历程：
  - 感性具体 -> 思维抽象 -> 思维具体；
  - 思维抽象环节含分离、提纯、简略化、理想化。

### 5. 创新思维

- 特点与“三新”：
  - 思路具有多向性；
  - 步骤具有跨越性；
  - 结果具有独特性；
  - 思路新、方法新、结果新。
- 联想思维：
  - 跨越的联结性；
  - 非逻辑制约的畅想性；
  - 方法：迁移、想象。
- 发散思维与聚合思维：
  - 发散方法：检核表法、信息交合法、头脑风暴法；
  - 聚合思维用于把多种方向收束成较优方案；
  - 发散与聚合是创新思维的两翼。
- 逆向思维：
  - 对事物结构顺序的已有认识进行反向思考；
  - 对已有事物存在状态和认识作转换性思考；
  - 对已有事物功能的认识作反向思考；
  - 对已有事物之间因果关系的认识作交换性思考。
- 超前思维：
  - 性质：探索性、预测性、不确定性；
  - 方法：注重调查研究、运用矛盾分析法、运用推理和想象。

## 三、推理部分框架

### 1. 总体结构

- 学习要求：
  - 熟练掌握逻辑推理规则；
  - 运用框架思维解决“思维是何问题”。
- 推理分为：
  - 必然推理：演绎推理；
  - 或然推理：不完全归纳推理、类比推理等；
  - 完全归纳推理；
  - 假言、选言、联言、三段论等规则型题。

### 2. 逻辑三律

- 同一律：A 是 A；常见错误是偷换概念、偷换论题。
- 矛盾律：A 不是非 A；常见错误是自相矛盾。
- 排中律：A 或者非 A；常见错误是“两不可”。

### 3. 概念

- 定义；
- 划分；
- 外延关系；
- 不相容关系；
- 矛盾关系；
- 反对关系。

### 4. 九种推理/题型族群

- 演绎推理：前提真实、结构正确。
- 换质推理：
  - 两变两不变；
  - 不变：主项和量项；
  - 变：联项和谓项。
- 换位推理：
  - 联项不变；
  - 主谓交换；
  - 前提中不周延的项换位后也不能周延。
- 周延判断：
  - 主项看量项：全称、单称周延，特称不周延；
  - 谓项看联项：否定周延，肯定不周延。
- 三段论推理：
  - 只能有三个不同的项；
  - 中项至少周延一次；
  - 前提中不周延的项在结论中不得周延；
  - 两个否定前提不能必然推出结论。
  - 常见错误：四概念、中项不周延、大项/小项不当扩大、双否前提。
- 联言推理：
  - 合成；
  - 分解。
- 选言推理：
  - 相容选言与不相容选言；
  - 区分标准：`或者` vs `要么`；
  - 相容选言：否定肯定式；
  - 不相容选言：否定肯定式、肯定否定式。
- 假言推理：
  - 充分条件：肯前必肯后，否后必否前；
  - 必要条件：否前必否后，肯后必肯前；
  - 充要条件：四种有效式都可用；
  - 连接词：
    - 充分条件：如果……那么……、只要……就……；
    - 必要条件：只有……才……、要想……必须……、除非……否则……。
- 归纳推理：
  - 完全归纳推理；
  - 不完全归纳推理；
  - 方法：求同法、求异法、共变法、求同求异并用法、剩余法。
- 类比推理：
  - 类比根据越多越好；
  - 相同属性越接近本质属性，结论可靠程度越高；
  - 前提中确认的属性不应与结论属性相互排斥。

## 四、用户硬样本与永久注意

- `2026通州期末 Q11`：思维抽象与思维具体，口令是`杂多现象 -> 抽出核心概念 -> 回到完整整体图景`。
- 推理题不能混入思维主链；思维题不能被形式逻辑规则偷换。
- 推理最终稿必须把同类题放一起，不按年份流水账。
