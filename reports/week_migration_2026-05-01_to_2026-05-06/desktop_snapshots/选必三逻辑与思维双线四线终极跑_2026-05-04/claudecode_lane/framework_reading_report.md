## 一、思维部分：如何按哲学触发链处理

### 1. 处理逻辑

思维部分与必修四哲学完全同构。处理逻辑不是先背定义、再对号入座，而是：

**材料动作 → 识别总帽子 → 定位小方法 → 写出触发逻辑 → 落点答案句**

- **总帽子**只有三个：科学思维、辩证思维、创新思维（外加P.思维基础）
- **小方法**是给分点，必须落实到框架具体节点
- **触发逻辑**是考场可用的迁移路径，不是后台说明

### 2. 稳定触发链（B线确认版）

| 材料动作信号 | 总帽子 | 小方法落点 |
|---|---|---|
| 调查/数据/样本/实验/验证/迭代 | 科学思维 | 客观性/可检验性 |
| 趋势研判/未来布局/提前规划/预判 | 科学思维或创新思维 | 预见性/超前思维 |
| 多主体/多环节/协同治理/一盘棋 | 辩证思维 | 整体性/分析与综合 |
| 当前与长远/由低到高/渐进推进 | 辩证思维 | 动态性/质量互变 |
| 旧产业/旧资源/旧模式升级 | 辩证思维 | 辩证否定 |
| 跨界组合/文旅融合/文创设计/老资源新场景 | 创新思维 | 联想/迁移和想象 |
| 多方案铺开再收束 | 创新思维 | 发散思维→聚合思维 |
| 把原方向/位置/状态/流程反过来 | 创新思维 | 逆向思维 |
| 杂多现象→抽出核心概念→回到完整整体图景 | 辩证思维 | 思维抽象与思维具体 |

### 3. 边界裁定原则

- 纯形式逻辑/推理规则（三段论、假言、选言、周延判断）不进思维主链
- 同题前半是逻辑推理、后半是思维方法应用：**只收后半**
- "辩证思维"与必修四辩证法高度重叠，必须用**设问模块、评分来源、答题术语三者共同裁定**
- 必修四哲学题即使出现"辩证""整体""创新"也不自动并入
- 2026石景山期末全模块排除（无可用评分细则）

### 4. 主观题条目结构（B线确认）

按硬规则记事本第五条执行：

1. 思维方法/给分点（细则原词或来源支持的精确方法）
2. 完整设问（仅题目要求学生作答部分，不含材料正文）
3. 来源（年份、区、考试阶段、题号）
4. 材料动作（学生圈得出的关键信号）
5. 总帽子
6. 小方法
7. 触发逻辑（材料事实与思维方法之间的关系）
8. 答案句（考生答题纸能直接落笔的一句话）
9. 框架落点
10. 证据等级

## 二、推理部分：如何按题型族群处理

### 1. 处理逻辑

推理部分**不按年份流水账**，同类推理类型放一起。每个族群对应一个解题协议：

**识别入口 → 规则 → 常见错误 → 解题步骤 → 例题变式 → 避坑口令**

### 2. 九个族群的处理方式

**族群1：三段论**
- 规则：①只有三个不同的项；②中项至少周延一次；③前提中不周延的项在结论中不得周延；④两否定前提不能必然推出结论
- 常见错误：四概念、中项不周延、大项不当扩大、小项不当扩大、双否前提
- 主观题型：补全三段论（给大前提/小前提/结论其一，补另两项）
- 客观题型：找出逻辑分析错误的选项

**族群2：假言推理**
- 充分条件（如果…那么…/只要…就…）：肯前必肯后，否后必否前
- 必要条件（只有…才…/要想…必须…/除非…否则…）：否前必否后，肯后必肯前
- 充要条件：四种有效式均可用
- 常见错误：充分条件下否前/肯后，必要条件下肯前/否后

**族群3：选言推理**
- 相容选言（或者）：否定肯定式（否定一个选言支→肯定另一个）
- 不相容选言（要么）：否定肯定式+肯定否定式
- 常见错误：混淆相容/不相容，对相容选言用肯定否定式

**族群4：联言推理**
- 合成：单个肢为真→联言为真
- 分解：联言为真→任何一个肢为真

**族群5：三大逻辑律**
- 同一律（A是A）：违反→偷换概念/偷换论题
- 矛盾律（A不是非A）：违反→自相矛盾
- 排中律（A或者非A）：违反→"两不可"骑墙

**族群6：换质推理**
- 两变（联项+谓项）两不变（主项+量项）

**族群7：换位推理**
- 联项不变，主谓互换；前提中不周延的项换位后也不能周延

**族群8：周延判断**
- 主项看量项：全称/单称周延，特称不周延
- 谓项看联项：否定周延，肯定不周延

**族群9：归纳与类比**
- 不完全归纳→或然推理：求同法、求异法、共变法、求同求异并用法、剩余法
- 类比推理：类比根据越多越好，共同属性越接近本质属性结论越可靠，确认属性不应与结论属性相斥

## 三、B线吸收的关键边界规则

1. **石景山期末全排除**：2026/2025/2024石景山期末，除非用户提供新细则来源
2. **思维抽象与思维具体专项**：2026通州期末Q11是永久硬样本；口令固定为"杂多现象→抽出核心概念→回到完整整体图景"；"合/融于一身/整体"不等于系统整合/分析综合/联想
3. **证据等级分类**：A-formal（正式细则/评标/阅卷总结）> A-support（讲评PPT明确给分口径）> B-choice-signal（选择题信号）> C-boundary（边界记录）> missing（缺证据）
4. **答案句硬规则**：思维方法+本题材料事实+因果/作用/结论，不能只喊口号
5. **旧稿地位**：只作为定位索引，不作为证据。凡标`old_xuanbisan_index_only`的条目在B线矩阵中统一标`index_only→needs_recheck`
