## 题型族群初筛

### 1. 三段论

**识别入口**

- 题干出现“编写一个合乎规则的三段论”“下列三段论分析错误/正确”“中项/大项/小项/周延/四概念”。
- 设问要求判断某推理是否符合三段论规则，或要求补全大前提/小前提/结论。

**高价值候选**

- `2025西城期末`：要求以“工业固废是放错了地方的资源”为小前提，编写合乎规则的三段论；候选中出现“符合三段论基本规则 4分”。
- `2025延庆一模 Q5`：围绕四位同学的三段论推理，选项涉及小项不当扩大、中项不周延、大项不当扩大、四概念。
- `2025顺义一模 Q7`：三段论逻辑分析题，覆盖大项不当扩大、双否前提、四概念、中项不周延。
- `2025东城期末 Q13`：央企民用产品材料，判断三段论违反相同逻辑规则，候选显示中项不周延。
- `2026东城期末 Q1`、`2026`未补套卷名 Q1/Q2/Q3：候选涉及“乌毛蕨形成的镧独居石”为小项、不犯四概念、中项必须周延等规则，价值高但套卷名需补齐。

**常见设错**

- 四概念：同一词在前后语境中偷换含义，例如“物质”既指哲学范畴又指具体物品。
- 中项不周延：中项在两个前提中都没有至少周延一次。
- 大项/小项不当扩大：前提中不周延的项到了结论中变周延。
- 两个否定前提：不能必然推出结论。
- 把三段论误判为换位、假言、类比、归纳。

**解题模板**

1. 先圈三项：结论主项是小项，结论谓项是大项，两个前提共同出现的是中项。
2. 查是否只有三个不同项；同词不同义先判四概念。
3. 查中项是否至少周延一次。
4. 查结论中的大项、小项是否在前提中已经周延。
5. 查两个前提是否都是否定。
6. 卷面表达：`这个推理属于三段论。中项“X”在两个前提中都不周延/大项“Y”在前提中不周延而在结论中周延，因此违反……规则，结论不能必然得出。`

**回源核验优先**

- `2025西城期末` 对应三段论主观题：核题面、正式细则、分值。
- `2025延庆一模 Q5`：核四位同学原推理和答案。
- `2025顺义一模 Q7`：核选项原文和答案。
- `2025东城期末 Q13`：核“违反相同逻辑规则”的题面和答案。
- `2026东城期末 Q1`、`2026`未补套卷名 Q1/Q2/Q3：先补全套卷名，再核原题。

### 2. 假言推理

**识别入口**

- 连接词：`如果……那么……`、`只要……就……`、`只有……才……`、`要想……必须……`、`除非……否则……`。
- 题干要求判断充分条件、必要条件、充要条件，或要求说明推理结构是否有效。

**高价值候选**

- `2024朝阳一模 Q20`：讲评候选显示“以上推理属于充分条件假言推理”“推理类型 1分”“理由 2分”“替代：肯定前件/前真后必真”。
- `2025顺义一模 Q17`：杭州科创基金材料，甲为充分条件假言判断，乙为必要条件假言判断，丙为相容选言判断，适合做综合判断类型题。
- `2026海淀期末 Q20`、`2026`未补套卷名 Q20：候选显示“充分条件假言推理，错误；肯定后件，结论肯定前件，是错误结构”，疑似高价值主观题。
- `2025丰台期末 Q18`：候选显示必要条件假言判断和联言判断保真条件，适合判断类型与保真条件合练。
- `2025丰台期末`研学题：充分条件假言推理不能由肯定后件推出肯定前件。
- `2025东城二模`、`2026丰台一模`：候选显示学生把“充分假言推理”、充分/必要条件混淆，是设错素材。

**常见设错**

- 充分条件误用：肯定后件推出肯定前件；否定前件推出否定后件。
- 必要条件误用：肯定前件推出肯定后件；否定后件推出否定前件。
- 把“只有……才……”误当“只要……就……”。
- 把判断类型和推理类型混写，如“必要条件假言判断推理”。
- 看到“必然联系”就机械套假言，不看条件方向。

**解题模板**

1. 翻译条件：充分条件写成 `P -> Q`；必要条件写成 `Q -> P`，或“只有P才Q”等价于 `Q -> P`。
2. 判断有效式：充分条件只保留“肯前肯后、否后否前”；必要条件只保留“否前否后、肯后肯前”；充要条件四式都可用。
3. 对照题干实际推理，看它肯定/否定的是前件还是后件。
4. 卷面表达：`该推理属于充分条件假言推理。充分条件假言推理有效式是肯定前件式和否定后件式。本题由“Q真”推出“P真”，属于肯定后件，不能必然推出结论。`

**回源核验优先**

- `2024朝阳一模 Q20`：核讲评是否可作为 A-support，是否有正式细则。
- `2025顺义一模 Q17`：核甲乙丙三判断题面和采分。
- `2026海淀期末 Q20`/`2026`未补套卷名 Q20：核套卷名、题面、细则三步采分。
- `2025丰台期末 Q18`：核必要条件假言判断与联言判断保真条件的正式来源。

### 3. 选言推理

**识别入口**

- 相容选言：`或者P，或者Q`，至少一真即可。
- 不相容选言：`要么P，要么Q`，有且只有一真。
- 题干要求判断“否定一部分能否肯定另一部分”“肯定一部分能否否定另一部分”。

**高价值候选**

- `2025顺义一模 Q17`：丙为相容选言判断，和假言判断同题综合，适合作为判断类型合练。
- `2025海淀期末 Q9`：候选显示“如果‘要么P要么Q’这个不相容选言判断为真，那么‘或者P或者Q’这个相容选言判断也为真”。
- `2025房山一模 Q6`：候选出现“要么去颐和园，要么去古北水镇”，适合不相容选言训练。
- `2026海淀一模 Q17`、`2026`未补套卷名 Q17：候选显示“划分标准不一；选言判断遗漏其他有选择价值的情况”，可与概念划分合练。
- `2026朝阳期末 Q6`：AI 幻觉材料中“要么事实性幻觉，要么忠实性幻觉”，适合核查是否为穷尽且不相容。

**常见设错**

- 把相容选言当不相容选言：肯定一个选言支后错误否定另一个。
- 把不相容选言当相容选言：只否定一个不敢推出另一个，或忘记肯定否定式。
- 选言支不穷尽：遗漏其他有选择价值的情况。
- 选言支交叉：不是同一划分标准，导致选言判断不清。
- 只因句子有“或者”就判选言推理，实际可能只是普通并列或法律条文列举。

**解题模板**

1. 先判连接词和语义：`或者`通常相容，`要么`通常不相容，但最终看能否同真。
2. 相容选言只用否定肯定式：否定一支，才能肯定剩余至少一支；肯定一支不能否定另一支。
3. 不相容选言两式都可用：否定一支肯定另一支；肯定一支否定另一支。
4. 检查选言支是否同一标准、是否穷尽。
5. 卷面表达：`该判断是不相容选言判断，P和Q不能同真。题中已经肯定P，因此可以否定Q；若只是相容选言，则不能这样推出。`

**回源核验优先**

- `2025顺义一模 Q17`
- `2025海淀期末 Q9`
- `2025房山一模 Q6`
- `2026海淀一模 Q17`
- `2026朝阳期末 Q6`

### 4. 归纳推理

**识别入口**

- 从个别/部分样本推出一般结论。
- 题干出现“完全归纳”“不完全归纳”“科学归纳”“求同法”“求异法”“共变法”“剩余法”。
- 材料常见实验、样本比较、变量控制、因果探求。

**高价值候选**

- `2024东城一模`：北极熊毛衣材料，候选显示“不完全归纳推理的求异法”，并与类比推理同题。
- `2024西城二模`/`东城二模 Q18`：高粱 AT1 基因材料，候选显示归纳推理、不完全归纳、科学归纳、共变法/求异法。
- `2024海淀一模 Q18/19`：候选显示“不完全归纳推理”。
- `2024朝阳期中 Q18`：楚王推理属于不完全归纳，强调前提只涉及部分对象，属于或然推理。
- `2025朝阳二模 Q17/16`：要求识别同学所用推理类型并谈如何更好发挥该推理类型功能，候选指向不完全归纳。
- `2026朝阳一模 Q1`、`2026西城期末 Q20`：候选显示不完全归纳、求同/求异/共变等因果方法。

**常见设错**

- 把不完全归纳误写成完全归纳。
- 只写“归纳推理”，不写样本到一般结论的推理过程。
- 把类比推理、归纳推理混淆：类比是对象之间相似迁移，归纳是从个别到一般。
- 因果方法混淆：求同、求异、共变、剩余法不分。
- 把“或然推理”写成“必然推出”，夸大可靠性。

**解题模板**

1. 判方向：多个个别事实/样本 -> 一般结论，为归纳。
2. 判是否穷尽对象：穷尽为完全归纳；未穷尽为不完全归纳。
3. 若是实验因果题，再判方法：同中求同、异中求异、变量同步变化为共变，排除已知原因后剩余为剩余法。
4. 卷面表达：`该推理属于不完全归纳推理。研究者考察若干样本，由这些样本都具有X属性，推出某类对象一般具有X属性；该推理结论具有或然性，需要扩大样本、控制变量、反复验证来提高可靠程度。`

**回源核验优先**

- `2024东城一模`北极熊毛衣题：核题号和细则。
- `2024西城二模`/`东城二模 Q18`高粱 AT1 基因题：核套卷归属、题面与细则。
- `2024朝阳期中 Q18`
- `2025朝阳二模 Q17/16`
- `2026朝阳一模 Q1`
- `2026西城期末 Q20`

### 5. 类比推理

**识别入口**

- 材料中有“拿甲比乙”“模仿结构/功能”“由一个对象迁移到另一个对象”。
- 题干要求说明两个对象在相似属性基础上推出另一相似属性。

**高价值候选**

- `2024东城一模`北极熊毛衣材料：候选显示“运用类比推理的方法，模仿北极熊毛的结构和功能”。
- `东城二模 Q18`高粱 AT1 与水稻 GS3：候选显示“相似属性进行类比推理”。
- `2024朝阳期中 Q18`：晏子推理属于类比推理，根据两个或两类对象相同或相似属性推出其他属性相似。
- `2024朝阳二模 Q19`：候选显示“动态性、类比”，需回源判断是思维部分还是推理部分。
- `2026东城期末 Q1/Q2/Q3`：候选涉及类比推理、认识和利用规律等复合材料，需严守推理/哲学边界。

**常见设错**

- 看到“像”“借鉴”“经验迁移”就判类比，但材料其实是在联想思维或迁移想象。
- 相同属性太少、太表层，结论可靠性不足。
- 前提中已有属性与结论属性相互排斥。
- 把类比推理误写成归纳推理或发散思维。
- 类比题只写概念，不写“哪两个对象、哪些相似属性、推出什么属性”。

**解题模板**

1. 找两个对象：甲对象、乙对象。
2. 找已知相似属性：结构、功能、条件、机制、结果。
3. 找迁移结论：由甲的某属性推乙可能也有某属性。
4. 评价可靠性：相似根据越多、越接近本质，可靠性越高；若有排斥属性，可靠性下降。
5. 卷面表达：`该推理属于类比推理。研究者根据甲和乙在A、B等属性上相似，推测乙也可能具有甲的C属性。要提高结论可靠性，应增加相似根据，重点比较本质属性，并排除与结论相冲突的属性。`

**回源核验优先**

- `2024东城一模`北极熊毛衣题
- `东城二模 Q18`高粱/水稻基因题
- `2024朝阳期中 Q18`
- `2024朝阳二模 Q19`
- `2026东城期末 Q1/Q2/Q3`

### 6. 联言推理

**识别入口**

- 连接词：`并且`、`既……又……`、`不仅……而且……`、`同时`。
- 题干要求判断联言判断真假，或从一个联言判断推出各联言支。

**高价值候选**

- `2024朝阳二模 Q19/20`：候选显示“该判断是联言判断；当且仅当组成它的各个联言支都是真的，这个联言判断才是真的”。
- `2025丰台期末 Q18`：必要条件假言判断与联言判断保真条件同题，适合复合判断训练。
- `2025朝阳一模 Q3/Q4`：候选显示学生术语不准，如“充分条件联言判断”“选言充分条件推理”。
- `2026西城一模 Q20`、`2026`未补套卷名 Q20：候选显示“联言判断：同真则真/反对关系/材料分析”。
- `2026东城期末`：候选显示“既要完整保留历史街巷的一砖一瓦，也要对无法适应现代需要的部分进行必要拆除和改造”为联言判断，需判断两个联言支能否同真。

**常见设错**

- 把联言判断写成联言推理，判断和推理不分。
- 以为只要一个联言支为真，整个联言判断就真。
- 忽略联言支之间可能存在反对或矛盾关系。
- 把“既……又……”和选言、假言混淆。

**解题模板**

1. 拆联言支：P 并且 Q。
2. 判真假条件：P、Q都真，联言判断才真；任一为假，整体为假。
3. 联言推理合成式：P真、Q真 -> P并且Q真。
4. 联言推理分解式：P并且Q真 -> P真；P并且Q真 -> Q真。
5. 卷面表达：`该判断是联言判断。只有各个联言支都为真，整个联言判断才为真。题中P与Q不能同时成立/其中Q不成立，因此该联言判断为假。`

**回源核验优先**

- `2024朝阳二模 Q19/20`
- `2025丰台期末 Q18`
- `2025朝阳一模 Q3/Q4`
- `2026西城一模 Q20`
- `2026东城期末`历史街巷联言判断题

### 7. 逻辑三律/概念

**识别入口**

- 同一律：A 是 A，常见错误为偷换概念、偷换论题。
- 矛盾律：A 不是非 A，常见错误为自相矛盾。
- 排中律：A 或非 A，常见错误为对矛盾判断“两不可”。
- 概念题：定义、划分、外延关系、矛盾关系、反对关系、不相容关系。

**高价值候选**

- `2024朝阳一模 Q6`：讲评候选覆盖越级划分、排中律、矛盾律、四概念，适合作为概念与逻辑规律综合选择题。
- `2026西城一模 Q20`、`2026`未补套卷名 Q20：候选显示丙错误，偷换“村民会议/村委会”概念，违背同一律。
- `2026海淀一模 Q17`：候选显示划分标准不一、选言判断遗漏其他有选择价值的情况。
- `2026朝阳期末 Q6`：AI 幻觉材料，候选涉及矛盾律、选言判断、概念外延，需回源核选择题答案。
- `2025东城期末 Q15`：北京中轴线材料，候选涉及概念外延、关系判断、周延，适合概念关系训练。

**常见设错**

- 同一律：同一个词前后含义变了，或论题被悄悄换掉。
- 矛盾律：同时断定互相矛盾的命题都真。
- 排中律：对两个矛盾判断同时否定。
- 定义过宽/过窄、循环定义、比喻代定义。
- 划分标准不一、越级划分、子项不全、子项相容。
- 矛盾关系与反对关系混淆：矛盾关系不能同真也不能同假，反对关系不能同真但可以同假。

**解题模板**

1. 先看题目问“逻辑规律”还是“概念规则”。
2. 若是三律：找同一时间、同一方面、同一对象下是否偷换、自相矛盾或两不可。
3. 若是概念：先定外延关系，再看定义/划分规则。
4. 卷面表达：`该说法违背同一律。前提中的“X”指……，结论中的“X”变成……，同一概念在同一推理中含义不一致，属于偷换概念。`

- `2026西城一模 Q20`
- `2026海淀一模 Q17`
- `2026朝阳期末 Q6`
- `2025东城期末 Q15`

### 8. 换质换位

**识别入口**

- 换质：不改变主项和量项，改变联项，并把谓项换成其矛盾概念。
- 换位：不改变联项，交换主项和谓项，且前提中不周延的项换位后不得周延。
- 题干出现“换质位”“换质推理”“换位推理”“特称否定不能换位”等。

**高价值候选**

- `2026东城一模 Q7`：候选显示“可通过换质位得到……”。
- `2025丰台二模 Q14`：候选显示“从‘有的人不是说谎者’这一前提，能够通过换位推理必然得出结论”，适合特称否定不能直接换位。
- `2025房山一模`：候选显示“很多人没去过古北水镇”属于特称否定判断，不能直接进行换位推理。
- `2024海淀二模 Q7`：候选显示“经换位推理可得出……”，需核对是否为正确/错误选项。
- `2026丰台一模 Q1/Q2/Q3`：候选显示换质位、充分必要假言、三段论等混淆，是错因素材。
- `2025海淀一模`：候选仅显示“换质推理”，需回源确认题号与价值。

**常见设错**

- 换质时只加“不”，没有同时改变联项和谓项。
- 换位时把前提中不周延的项换到结论后变周延。
- 特称否定判断直接换位。
- 把换质、换位、换质位顺序混乱。
- 把换位误判为三段论或假言推理。

**解题模板**

1. 判断原命题类型：全称肯定、全称否定、特称肯定、特称否定。
2. 换质：量项、主项不变；肯定变否定/否定变肯定；谓项换成矛盾概念。
3. 换位：联项不变；主谓交换；检查周延，前提中不周延的项换位后不得周延。
4. 换质位：先换质再换位，逐步检查，不跳步。
5. 卷面表达：`该换位不能成立。原判断中“X”是不周延的，换位后“X”成为否定判断的谓项而周延，违反前提中不周延的项换位后也不能周延的规则。`

- `2025丰台二模 Q14`
- `2025房山一模`换位题
- `2024海淀二模 Q7`
- `2026丰台一模 Q1/Q2/Q3`
- `2025海淀一模`换质推理候选

## 可直接沉淀为学生稿的总模板

### A. 选择题通用四步

1. 先判题型：三段论、假言、选言、联言、归纳、类比、换质换位、三律/概念。
2. 再翻译结构：把自然语言翻成项、前后件、选言支、联言支、样本与结论、相似对象。
3. 套最小规则：只套该题型的有效式或禁忌规则。
4. 查常见陷阱：条件方向、周延、偷换概念、相容/不相容、归纳/类比、判断/推理混淆。

### B. 主观题通用卷面句

- `推理类型句`：`该推理属于……推理。`
- `规则句`：`……推理的有效规则是……，不能……。`
- `材料结构句`：`题中由……推出……，实际是……。`
- `结论判断句`：`因此该推理结构正确/错误，结论能/不能必然得出。`

### C. 最容易混淆的三组

- `归纳 vs 类比`：归纳是多个个别推出一般；类比是甲对象迁移到乙对象。
- `充分 vs 必要`：充分看“有它就够”；必要看“没它不行”。
- `相容选言 vs 不相容选言`：相容可同真，不相容只能一真。
