# 选必一按题训练稿 + 桥接表 审稿意见

声明:本次审稿只看教学结构、可读性、迁移性、卷面表达。所有"术语是否真实存在""分值/必答口径""得分点边界"由 Codex 据本地证据裁定,我不替它判。

---

## A. 总判定

**这一版已经显著优于纯六桶术语表,可以作为学生训练稿的内部预览版投入小范围试读,但还不能作为对外学生终稿。**

理由是结构上三件事已经做对了:每道题独占一节、按题视图与六桶桥接表分工、每条核心点合并后保留全称不压扁。学生第一次看到这个版本,能直接练整题答题卡而不是只背术语。

但还有四类问题影响"学生真能迁移":
- 部分题的"答题卡版答案"仍带老师讲义口吻,不像考生在卷面写的句子;
- 部分题的"材料抓手"仍只列关键词,没把"看到 X → 想到 Y"的桥写出来;
- 部分核心点的禁用条件(慎用提醒)写得过弱,学生看完不知道边界在哪;
- 个别题(尤其综合题与开放短文题)与选必一的接口写得太软,学生会把它当万能句,反而破坏前 7 道题建立的迁移纪律。

下面 must_fix_transfer 给出可以直接动手的位置。

---

## B. 最该保留的优点

下面这些做法在下一轮返修里**不要动**:

**B1. 按题节内部的四段式骨架**(审题结论 / 材料抓手 / 命中核心点 / 答题卡版答案 / 慎用提醒)
这五块的命名学生看得懂、对应自己做题的五个动作:读题、抓材料、调术语、写卷面、避陷阱。这个骨架是这一稿最大的可读性资产。

**B2. "审题结论"先告诉学生题目问什么、不问什么**
朝阳期中 Q17 的"不是让你泛泛谈人工智能意义"、通州 Q20 的"不能只背中国方案一个万能句"、朝阳一模 Q20 的"不能只写大国担当"——这种"否定式审题"对高三学生极其有用,直接挡掉常见跑题。要保留并扩到所有题。

**B3. "命中核心点"按六桶分组列出**
学生在按题视图里就能看到这道题横跨哪几个桶,不需要回查桥接表才意识到这是跨桶题。朝阳一模 Q20 的五桶覆盖一目了然。

**B4. 桥接表"使用提醒"列**
这一列是六桶视图的灵魂——它把"什么时候可以用 / 什么时候不能用"压成一句话。多数提醒写得到位,例如"共同利益解释为什么能合作,正确义利观解释中国怎样合作,两者不要合并"。这种边界提示要继续做。

**B5. 几条关键的合并纪律**
- "和平与发展仍是时代主题"是核心点、"中国做法符合……"在表述积累——做对了。
- "推动经济全球化朝着更加开放、包容、普惠、平衡、共赢的方向发展"五个词没被压扁——做对了。
- "新型国际关系"作核心,"合作共赢/互利共赢/共享发展成果"在表述积累——做对了。
- "人类命运共同体"作核心,"外交思想"作变式——做对了。

## C. must_fix_transfer(必改的迁移性问题)

### C1. 朝阳期中 Q17 的"两个市场两种资源"在按题视图丢了

按题视图的"命中核心点"里,经济全球化桶只列了"充分利用国内国际两个市场、两种资源"一句,但答题卡版答案的第一段"也要坚持开放合作,主动参与国际分工和全球科技交流"并没有把这个术语原话写进去。学生抄完答案卡背一遍,会发现卡上看不到"两个市场、两种资源"这八个字。结果是核心点和卷面句脱节。

**改法**:答题卡第一段末尾补一句术语原话,例如"在自主可控的同时主动参与国际分工和经济全球化,联通国内国际两个市场、两种资源"。术语原话出现在卷面,学生才知道考场上这个点该怎么落到笔上。

需要本地证据核验:这句术语原话在朝阳期中 Q17 细则里是否对应得分点。

### C2. 海淀期中 Q16(2) 的两段答案之间逻辑断裂

现在的写法是:"如果材料进一步呈现贸易摩擦……企业可以……"。"如果"这个词出现在学生答题卡里很奇怪——学生在考场写的不是条件句,他要么写要么不写。

**改法**:把"如果材料进一步呈现"这一行从答题卡里挪到"慎用提醒"中,作为判断条件;答题卡只保留两段两手都要写的并列结构,第二段开头改成"针对海外可能出现的贸易摩擦或规则争端,企业还应在政府和行业组织支持下……"。

把"判断条件"和"卷面句"分开,是这一版最该养成的纪律——慎用提醒写"什么时候用",答题卡写"用上后是怎么说的",两件事不能在同一段里讨论。

### C3. 朝阳一模 Q20 答题卡有四段,但第四段是"再升华",学生会以为这道题正常作答就要写四段

第四段"中国的发展和合作顺应和平与发展的时代主题,坚持共商共建共享,推动建设合作共赢的新型国际关系……"读起来像把前三段已经命中的术语再扫一遍,再加上"和平与发展"补一笔。这种结构在老师讲义里合理,因为老师要把所有得分点都展示出来;但放进学生答题卡,学生会模仿"凡跨桶题就写四段、最后一段总扫一遍"。

**改法**:两个方案二选一,由 Codex 据本地证据决定:
- 方案一:把第四段拆开,把"和平与发展"并入第一段(能力来源段),把"共商共建共享/合作共赢"并入第三段(开放发展段),整道题压成三段;
- 方案二:保留第四段但明确标注为"得分点扫尾段(选写)",并在慎用提醒里写明:"若前三段已自然带出和平与发展、共商共建共享,可不再单列扫尾段。"

需要本地证据核验:第四段对应的得分点在前三段是否已经落到、是否还需要独立成段。

### C4. 通州期末 Q20 第三段"彰显担当"中"中国提出全球治理公共产品"读起来不像考生话

"中国提出全球治理公共产品"这种动宾搭配学生不会自己写,他会写成"中国把全球治理倡议作为重要的国际公共产品贡献给世界",或者更白一点"中国向世界提供……"。当前版本把这一句压得太短,学生背下来不会用,换一道题就接不上。

**改法**:第三段改写为"中国把全球治理倡议作为重要的国际公共产品贡献给世界,坚持正确义利观,兼顾各方合理利益,贡献中国智慧中国方案,推动构建人类命运共同体,展现负责任大国担当。"——动作主语清楚,术语原话保留。

### C5. 海淀二模 Q21"中国需要联合国、联合国需要中国"答题卡分四段,逻辑上其实是 2+2

现在第一段写"联合国是……三大支柱",第二段写"联合国为中国提供平台",第三段写"中国是创始会员国……",第四段写"中国维护联合国体系"。学生看到四段会以为是四个独立要点,但实际上是"中国需要联合国(段一段二)+ 联合国需要中国(段三段四)"两侧两段。

**改法**:在每一侧前补一行小标题或起始句,让两侧的边界对学生显性可见。例如第一段开头改成"一方面,中国需要联合国。",第三段开头改成"另一方面,联合国也需要中国。"。这种显性连接词在双向关系题里是学生考场必备,要从范文里就教会。

### C6. 西城期末 Q20 第二段一口气列了七个理念名,学生抄不下来也用不出来

"……顺应和平、发展、合作、共赢的时代潮流,回应各国共同利益,也体现人类命运共同体理念、全人类共同价值、正确义利观、共商共建共享、真正的多边主义和互利共赢要求。"
这一句里"人类命运共同体理念 / 全人类共同价值 / 正确义利观 / 共商共建共享 / 真正的多边主义 / 互利共赢"六个并列名词,学生看了第一反应是"这都背?"。考生答题卡里不会出现六词并列。

**改法**:挑两到三个最直接对接气候治理材料的理念写进卷面,其余进表述积累。例如卷面只保留"人类命运共同体理念、正确义利观和真正的多边主义",其他四个收进桥接表的"中国全球治理理念与价值取向"那一行。在慎用提醒里写明:"理念词最多写三个,选与材料最对接的;不要写满。"

需要本地证据核验:西城 Q20 细则中这六个理念是各自分项给分还是同一得分点的并列变式。

### C7. 海淀期末 Q22(愚公移山短文)与选必一的接口太软,会反向破坏迁移纪律

短文题本身性质就开放,不是问题。问题是这一节的写法让学生有两种危险解读:
- 把"中国智慧中国方案 + 人类命运共同体"当成万能结尾;
- 反过来,看到任何"团结奋斗""迎难而上"的材料就直接套选必一。

**改法**:本节增加一个明确的"前置条件"段落,例如"只有在主题任务里出现'共同挑战 / 全球治理 / 共同发展 / 中国与世界'信号时,才用选必一这两个核心点;否则按主题任务的本来要求作答。"现在的"慎用提醒"写了类似话但太靠后,学生看到答题卡示例时已经模仿过去了。前置条件要放在审题结论之前,作为本节的开场白。

### C8. 东城一模 Q16 在选必一框里只剩两个核心点,但答题卡却展开成两段完整作答,容易被误读为标准模板

这道题在按题视图里只命中"中国智慧中国方案 + 人类命运共同体 + 新型国际关系"三个点,本身得分锚点就少。现在的答题卡两段读起来很顺,但学生看完会以为"哦,综合题用选必一就这两段套话"。

**改法**:两个动作一起做:
- 在审题结论里加一句:"本题主线在文化/哲学模块,选必一只能贡献局部表达,不要把这两段当成完整答案。"
- 把答题卡前面加一行明确标签:"【选必一可用片段】",而不是"答题卡版答案"。**让学生从标题就知道这不是一道完整答题卡,只是一段可迁移的话**。

下一轮返修里,可以考虑把东城 Q16 整体降级为"选必一迁移练习"专栏的一例,而不是和朝阳期中 Q17 这种正题平起平坐。

### C9. "材料抓手"没有统一格式,有的写"想到",有的写"先想到/再想到",有的写"不是,而是"

按题视图九道题翻下来:朝阳期中 Q17 用"想到 X";海淀期中 Q16(2) 用"先想到 / 再想到";顺义一模 Q20 全用"想到";通州 Q20 也用"想到";朝阳一模 Q20 用"想到";海淀二模 Q21 用"想到";西城 Q20 用"想到";东城 Q16 用"想到";海淀期末 Q22 用"想到"。

唯一不统一的是海淀期中 Q16(2) 的"先 / 再"和东城 Q20 隐含的边界提示。

**改法**:全部题统一用三段式格式:
- "看到材料里的 <事实/关键词>";
- "→ 想到 <核心点>";
- 必要时第三行写"(慎用条件:……)"。

这个格式要在所有题里一致,因为学生学的不是题,是这个三段式动作。

需要本地证据核验:无,纯排版。

### C10. 桥接表"中国"桶下"贡献中国智慧、中国方案、中国力量"命中 7 道题,但没有按题型分类

"中国智慧中国方案"在通州 Q20(全球治理)、朝阳一模 Q20(全球发展)、海淀二模 Q21(联合国)、西城 Q20(气候治理)、东城 Q16(文明)、海淀期末 Q22(短文)六类不同语境下都出现,使用提醒只写了一句"要分题改写"。这一句太弱,学生会把它当成"凡国际题就写中国智慧中国方案"。

**改法**:在该行"使用提醒"里把六类语境的具体改写口径列出,例如:
- 全球治理类:贡献全球治理体系改革的中国方案;
- 全球发展类:贡献开放合作的中国方案;
- 联合国类:贡献支持联合国和多边主义的中国方案;
- 气候治理类:贡献绿色低碳的中国方案;
- 文明类:贡献文明交流互鉴的中国方案。

需要本地证据核验:上述五种语境化措辞在本地细则里是否各自存在对应得分点。

## D. 文本润色 / 结构建议

**D1. 答题卡的段间连接词建议统一一组**

学生看完九道题会发现连接词五花八门。建议每道题答题卡内部固定一组段间连接词:
- 三段式题:"首先 / 其次 / 同时(或更进一步)";
- 双向关系题:"一方面 / 另一方面";
- 关系处理题:"要处理好 X 关系 / 要处理好 Y 关系 / 还要处理好 Z 关系"——这一组朝阳期中 Q17 已经做对,可以作为模板。

学生背的不是连接词本身,是"看到这种题用这组连接词"。

**D2. "慎用提醒"建议拆成两类:**"使用条件"+"易错提醒"

现在的慎用提醒混着两类内容:有的是"什么时候用"(共商共建共享只在通州主归全球治理),有的是"用了别错"(开放包容普惠平衡共赢不要压扁)。学生看到一段话里两类内容混在一起,不知道哪句是触发条件、哪句是表达纪律。

建议拆成两行:
- **使用条件**:写"什么样的材料/设问出现时才用";
- **易错提醒**:写"用了之后不要写错的细节"。

**D3. 答题卡末尾建议统一带一句"回扣设问"句**

现在大部分题的最后一段已经在回扣了,但不显性。建议在每道题答题卡的最后单独加一句"回扣式收尾",并在格式上标出来。学生学到的不是某一句话,而是"凡是答题卡最后一定要回扣题目动词"这个动作。

**D4. 桥接表"表述积累"列建议加分号统一,核心点列建议禁止出现分号**

现在桥接表里有的核心点用分号("当前国际竞争的实质是以经济和科技实力为基础的综合国力较量;坚持创新驱动战略"),有的不用。学生看到分号会以为是两个核心点。

建议:
- 核心点列:一行只能放一个核心,不能用分号串两个;
- 表述积累列:用分号或顿号串多个变式都可以,但要全表统一。

需要本地证据核验:"综合国力较量 + 创新驱动战略"在本地细则里是一个得分点还是两个分项。

**D5. 综合题/开放题在按题视图里建议加底纹或不同前缀**

东城 Q16、海淀期末 Q22、东城 Q20 这三道边界题和朝阳期中 Q17、通州 Q20 这种正题不在一个性质上。建议在节标题前加前缀(例如"【边界题】"或"【迁移练习】"),让学生在目录上一眼就能区分。

不区分的话,学生会把所有十一道题当成同等地位的范题去模仿,反而把边界题里的妥协写法学成主线。

## E. 下一轮返修优先级(最该改的 5 个具体位置)

按"对学生迁移性的破坏程度"从高到低排,前 5:

**E1. 海淀期末 Q22(愚公移山)和东城一模 Q16 重新定位为"选必一迁移练习",而不是和正题并列的"按题训练稿"成员**

这两道题如果保持现在的位置和写法,会反向教学生"短文/综合题里中国智慧中国方案 + 人类命运共同体可以无脑写"。这是最该先动的位置——它影响的不是单题质量,是整个迁移纪律。

具体动作:加节前缀"【选必一迁移片段】",在审题结论前加前置条件段,把答题卡标题从"答题卡版答案"改为"选必一可用片段"。

需要本地证据核验:这两道题在本地证据系统里是否本来就标注为综合题/边界题。

**E2. 朝阳一模 Q20 第四段处理**(对应 C3)

这是九道正题里唯一在结构上会让学生误学"四段式套路"的题。改成三段或明确标"扫尾段(选写)",影响所有跨桶题的范式。

需要本地证据核验:第四段对应得分点的独立性。

**E3. 西城期末 Q20 第二段的六词并列(对应 C6)+ 桥接表"中国全球治理理念与价值取向"行的使用提醒细化**

理念词六个并列堆在卷面里,是"老师讲义口吻最严重的一处"。这一处不改,学生会复制这种堆叠到其他题里。

具体动作:卷面减到三个理念,其余收进表述积累;桥接表使用提醒明确"理念词最多写三个,选与材料最对接的"。

需要本地证据核验:六个理念在本地细则里是否各自分项给分。

**E4. 海淀期中 Q16(2) 的"如果……则……"句式从答题卡挪到慎用提醒(对应 C2)**

这一处单独看影响不大,但它示范了一个最坏的模式——把"判断条件"和"卷面句"写在同一段里。这个模式如果不在第一道短题里掐掉,后面所有题都会沾染。

需要本地证据核验:无,纯结构调整。

**E5. 桥接表"中国"桶下"贡献中国智慧中国方案中国力量"行的语境化使用提醒(对应 C10)**

这条核心点命中 7 道题,是全表覆盖最广的一条。它的使用提醒现在只有一句"要分题改写",这句话不教学生怎么改。这一行不细化,学生在对外讲义里会把"中国智慧中国方案"当万能结尾,反而抹平六桶分类的努力。

具体动作:在使用提醒里按题型给出五种语境化措辞(全球治理/全球发展/联合国/气候治理/文明)。

需要本地证据核验:五种语境化措辞在本地细则里是否各自有对应得分点。
