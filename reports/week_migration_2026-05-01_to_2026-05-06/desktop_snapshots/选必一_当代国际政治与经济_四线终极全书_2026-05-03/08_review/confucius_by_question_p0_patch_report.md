# Confucius artifact-only 学会性检查报告

verdict: PASS_WITH_FIXES

检查范围：

- `07_student_doc/by_question_view_draft_20260503.md`
- `07_student_doc/six_bucket_to_question_crosswalk_draft.md`

检查限制：只按学生可见稿判断，不回看源证据、评分细则、worker、fusion、review 或后台日志；未改学生正文。

## 总判断

这版已经能让零基础学生大体完成“先按题审题，再按六桶复盘”的学习动作：多数题目都有审题结论、材料抓手、核心点、答题卡版答案和慎用提醒；“中国智慧中国方案 / 人类命运共同体 / 和平与发展”也有多处防万能句提醒。

但还不能直接判 PASS，因为个别位置会让学生分不清“主线必写”与“可选扫尾”，尤其是朝阳一模 Q20、和平与发展、共商共建共享和新型国际关系这些高频万能化风险点。

## 学生会卡住的位置

1. 2026朝阳一模 Q20：`和平与发展` 在按题稿里被列入“命中核心点”，但答案中又写成“可选扫尾”。
   - 位置：`by_question_view_draft_20260503.md` 第155-171行；`six_bucket_to_question_crosswalk_draft.md` 第17行。
   - 学生卡点：桥接表把朝阳一模 Q20 归入“和平与发展仍是时代主题”的命中题目，按题稿又把它列为时代背景核心点；但答题卡答案第171行说“若题目要求进一步升华”才用。学生会不确定考试时这点是必写主线，还是最后有余力才写。
   - 建议补丁方向：朝阳一模 Q20 中把“和平与发展”明确标为“可选升华 / 弱触发”，或在桥接表该行对朝阳一模 Q20 加括注“可选扫尾，不作前三段主干”。

2. 2026朝阳一模 Q20：`共商共建共享` 的桶位与主次不够稳定。
   - 位置：`by_question_view_draft_20260503.md` 第155-171行；`six_bucket_to_question_crosswalk_draft.md` 第51行、第61-64行。
   - 学生卡点：按题稿第161行把“共商共建共享”放在“中国”核心点中，第171行又放进“可选扫尾”；桥接表第51行强调通州全球治理方向不要和朝阳开放合作语境硬合并。学生可能误以为朝阳一模必须写共商共建共享，或把它挂到通州同一个全球治理主桶。
   - 建议补丁方向：朝阳一模 Q20 明确写“共商共建共享为可选升华，不替代经济全球化和中国贡献主线”；桥接表保留防混提醒。

3. 2026顺义一模 Q20：`中国方案 / 人类命运共同体` 的使用边界还可以再钉死。
   - 位置：`by_question_view_draft_20260503.md` 第81-100行；`six_bucket_to_question_crosswalk_draft.md` 第61-63行。
   - 学生卡点：按题稿第83行由“中国经验走出去、本土化创新、技术合作”直接想到“中国方案、正确义利观、人类命运共同体”，第92行也列为中国核心点；但顺义这题最稳的起点其实是共同利益、南南合作、互利共赢和全球化方向。零基础学生可能一看到“中国经验走出去”就先写中国方案/人类命运共同体，弱化“深层逻辑”里合作为什么成立。
   - 建议补丁方向：在顺义慎用提醒后补一句区分：“本题先写共同利益和合作共赢，中国方案/人类命运共同体只能接在科技小院让全球南方受益之后，不能开头当万能帽子。”

4. 2025海淀期中 Q16(2)：`贸易摩擦` 的使用条件已经提醒，但桥接表主语动作仍可能让学生混。
   - 位置：`by_question_view_draft_20260503.md` 第46-69行；`six_bucket_to_question_crosswalk_draft.md` 第43行。
   - 学生卡点：按题稿第65行写“企业还应在行业组织协调和政府支持下依法运用国际规则和相关机制”，桥接表第43行又写“政府依法运用国际组织规则和争端解决机制”。学生可能不清楚企业、行业组织、政府分别能做什么，进而把“企业利用国际组织权利”写得过满。
   - 建议补丁方向：把动作拆成学生能背的三主体句：企业合规维权，行业组织协调预警，政府运用国际组织规则和争端解决机制并参与全球经济治理。

5. 六桶桥接表：`推动建设合作共赢的新型国际关系` 行覆盖题目偏多，主干/可选标签需要更直观。
   - 位置：`six_bucket_to_question_crosswalk_draft.md` 第51-53行；`by_question_view_draft_20260503.md` 第224-229行。
   - 学生卡点：桥接表第53行把朝阳一模、顺义一模、海淀二模都放入同一核心点，虽然使用提醒说海淀二模只在“中国为联合国贡献什么”一侧作可选表达，但零基础学生复盘时容易把海淀二模联合国题误挂到政治多极化主干。
   - 建议补丁方向：桥接表“命中题目”里直接标注“2025海淀二模 Q21（可选贡献侧）”，避免学生误当主干。

## 五项能力判断

1. 知道每道题问什么：基本 PASS。每题“审题结论”清楚，拓展迁移区也说明不是完整答案模板。
2. 知道材料信号怎么触发术语：PASS_WITH_FIXES。多数触发链清楚；顺义和朝阳一模仍需防止从材料信号直接跳到万能帽子。
3. 知道哪些点是主线，哪些是可选/慎用：PASS_WITH_FIXES。这是当前主要短板，朝阳一模 Q20 和海淀二模 Q21 的可选标签还不够醒目。
4. 不把中国智慧中国方案、人类命运共同体、和平与发展写成万能句：PASS_WITH_FIXES。桥接表已有强提醒，但按题稿局部仍把这些词列入核心点，需补主次边界。
5. 能用六桶桥接表复盘而不误挂桶：PASS_WITH_FIXES。六桶结构可用；需要修正“可选贡献侧”“可选扫尾”“弱触发”的显示方式，防止误挂政治多极化或时代背景。
