### F-07

**位置**: 第 24 题 2026朝阳一模 Q20 整题汇总卷面答案,第 939-946 行(共 7 句,前置"第一层~第七层"标号);第 23 题朝阳期中 Q17 的整题汇总,第 873-880 行(共 8 句,无显式分层标号)
**问题**: 朝阳一模 Q20 已采用"第一层看时代和能力 / 第二层看开放联通 / 第三层看发展共享 / 第四层看规则和治理 / 第五层看合作关系 / 第六层看民生项目 / 第七层作价值收束"七层标号,这种**七层式整题汇总**虽然解决了上一轮 D-06 的"11 句平行陈述"问题,但产生了新副作用:
1. **七层超过考场可控范围**——高三政治大题答题卡可写空间通常对应 3-5 个层次,七层学生考场写不下,且"层"这个字本身在政治答题卡上不会作为连接词出现。
2. **"第 X 层看 Y"的句首格式不像考生卷面**——考生写"首先""其次""第三""此外",不写"第一层看时代和能力"。这是教学骨架,不是考生句式。

朝阳期中 Q17 的整题汇总反向相反——8 句之间**完全没有显式分层标号**,既不"第一层"也不"首先",学生抄下来 8 句平行,不知道这些得分点哪些是同一层并列、哪些是不同层递进。两道高密度题在分层标记上风格冲突。
**为什么影响学生**: 这两道是全稿最高密度的范题,学生最有可能拿来当作迁移模板。一边教学生写"第一层看 X / 第二层看 Y",一边在另一题里完全不显式分层,学生学到的是混乱的两套规则。考场答题卡的真实分层连接词("首先 / 其次 / 一方面 / 另一方面 / 一是 / 二是")反而没建立起来。
**具体改法**: 全稿(尤其这两道)的整题汇总按题型统一分层连接词:
- **关系处理类**(朝阳期中 Q17):用"第一,处理好 X 关系…… / 第二,处理好 Y 关系…… / 第三,处理好 Z 关系……"。同关系内部用"一方面…另一方面…"。
- **跨桶高密度题**(朝阳一模 Q20):压到 4-5 层,用"首先 / 其次 / 同时 / 此外 / 最后"。把当前的"七层"合并为"能力来源(包含时代背景 + 综合国力 1-2 句) / 开放发展(包含两个市场 + 要素流动 + 贸易投资便利化 + 五词方向 2-3 句) / 合作关系(包含共商共建共享 + 新型国际关系 1-2 句) / 大国担当(包含南南合作 + 命运共同体 + 中国智慧中国方案 1-2 句)"。
- **双向论证题**(海淀二模 Q21):用"一方面 / 另一方面"分两侧。
- **三层显性题**(通州 Q20):保留三层"正逢其时 / 指引方向 / 彰显担当",每层下用"一是 / 二是"。

不要求改答案内容,只统一连接词模板。
**是否需 Codex 本地核验**: 朝阳一模 Q20 合并到 4-5 层后是否仍能覆盖该题本地细则下的全部得分点。

### F-08

**位置**: 第 3 题 2025海淀期中 Q21(2) 整题汇总,第 128-135 行(共 7 句,无"变 / 不变"分栏)
**问题**: 该题完整设问明确"分析新中国外交的'变'与'不变'",`设问触发(本题独有)` 已正确写"按'变'和'不变'分栏:变侧写时代背景、国力和外交实践的发展,不变侧写基本立场、宗旨、目标和准则。" 但 `整题汇总卷面答案` 7 句话之间**没有显式分栏标号**,学生抄下来 7 句平行陈述。实际内容是:
- 第 1-3 句属于"变":时代背景变 / 综合国力变 / 时代主题变;
- 第 4-6 句属于"不变":国家性质不变 / 独立自主立场不变 / 宗旨目标准则不变;
- 第 7 句习近平外交思想又是"变"侧的指导思想。

学生看不出来这是 3+3+1 结构,可能写成 7 句平行,失去双栏对照感。这与 F-07 是同一类问题在不同题的体现,但海淀期中 Q21(2) 的 `设问触发(本题独有)` 自己已经写好了"变 / 不变分栏"的指令,卷面层却没落下来——这是设问触发与整题汇总的闭环断裂。
**为什么影响学生**: "变与不变"是高三政治高频对照型题(地理、历史、政治都常考),学生最需要从这一题学到的是"双栏分写"的卷面动作。当前整题汇总把这种动作隐藏了。
**具体改法**: 该整题汇总改写为显式双栏:
- **变(随时代和国力发展而变)**:① 时代背景的变…… ② 综合国力的变…… ③ 习近平外交思想为新时代中国特色大国外交提供根本遵循……
- **不变(基本立场、宗旨、目标和准则不变)**:① 始终服务于人民民主专政的国家性质…… ② 始终坚持独立自主的基本立场…… ③ 始终贯彻维护世界和平、促进共同发展的宗旨,坚持和平共处五项原则……

每栏 3 条,显式分栏标号。同样动作可推广到任何"变与不变""为什么需要与如何做""中国需要 X 与 X 也需要中国"的对照题。
**是否需 Codex 本地核验**: 不需要,纯卷面分层重排。

## 4. nice_to_have(不阻断终稿的润色建议,6 项)

1. **`设问触发(题型通用)` 字段全表共出现 7-8 种通用句**(分析说明类、追问做法类、追问意义类、追问原因类、写作类等),34 处使用通用模板。建议在讲义"使用路线"那一节前增加一个"题型通用触发对照表",学生背过一遍后,正文里 `设问触发(题型通用)` 这一行可以缩为题型标签(如"【题型:分析说明类】"),不再每题复读完整通用句,可读性会显著提升。

2. **桥接表(六桶术语积累复盘)在每个核心点行末尾列了"例题来源"**,但例题数超过 8 题的核心点(如"建设开放型世界经济"12 题、"经济全球化方向五词"12 题、"两个市场两种资源"11 题、"贸易投资自由化便利化"11 题、"共商共建共享"12 题、"国际关系民主化"11 题、"人类命运共同体"12+ 题)末尾用"等"省略。建议这些"等"展开为完整列表,因为高密度核心点的命中题数本身就是教学信息(命中越多越是高频考点),省略反而削弱了术语优先级判断。

3. **第 16 题东城期末 Q20 的 `路径分层` 字段是新增的范例字段**,只在这一题下出现。建议把这种"路径分层"字段在所有"设问自带分层信号"的题里推广(通州 Q20"三层"、朝阳期中 Q17"三组关系"、海淀期中 Q21(2)"变与不变"、海淀二模 Q21"双向"),使它成为模板字段,而不是单题独有。

4. **"和平、发展、合作、共赢是时代潮流"** 这一表述在朝阳一模 Q20 第 9 条目积累中出现(997 行),但该核心点是"国际发展合作、南南合作"。这个完整短语本身更接近"和平与发展仍是时代主题"族的扩展,放在"国际发展合作"核心积累里语义偏移。建议这个完整短语挂到"和平与发展仍是时代主题"核心积累里作为承接短语;"国际发展合作"核心只保留与南南合作直接相关的术语。

5. **慎用与跨模块章中"2025海淀一模 Q21(2)" 的题目锚点/设问** 写为"我国外贸为何能连上两个万亿级台阶"——这道题问的明显是经济主属,但目前没有写明"主属哪个模块"。建议慎用章每条都加一个"主属模块"标签,避免学生误以为这些都是选必一可写题。

6. **"白话提醒" 字段在 8 处出现**(和平与发展、五词方向、共商共建共享、新型国际关系、正确义利观、治理赤字逆全球化等),已经形成相对稳定的"边界守门员"机制。建议在使用路线那一节明确"白话提醒"是什么、什么时候用,让学生知道这一字段是给他们的,不是装饰。

## 5. final_teacher_judgement

**这份讲义已经达到"框架可发"的程度,但还差最后一轮字段清洗才能"学生可发"——8 处硬伤集中在元说明泄漏(F-01、F-02)、单字孤词积累(F-03)、积累字段重复粘贴(F-04)、按题与桥接表不一致(F-05)、模块归属错位(F-06)、整题汇总分层标号风格不统一(F-07、F-08),这些不是结构性返工,而是落地最后一公里的精修。修完这 8 项,讲义即可作为正式学生终稿分发。**
