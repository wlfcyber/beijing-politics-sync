# A. 总判定

**NEEDS_FIX**

这版已经比五题闭环稿更接近可用的术语迁移表，六桶意识明显增强，核心教材术语也保留得更完整。但它目前仍偏向 **教师后台术语库**，还没有达到学生外部稿标准。

最大问题有四个：

1. **多题同类项合并后，只给一个答案句**，容易让学生把 A 题材料答案套到 B 题。
2. **部分核心名信息量不足**，标题短，答案句长，学生复习时抓不住最高信息量原词。
3. **仍保留细则位置、必答、分值角度等后台痕迹**，像中间流程文件。
4. **缺整题卷面答案和按题视图**，学生只能背术语卡，难以形成完整答题结构。

# C. should_fix_transfer

## 1. 必须改成按题视图＋六桶索引

现在只有六桶视图，学生看完知道有哪些术语，但不知道一整道题该怎么写。

建议主文档结构改为：

| 层级        | 学生看到什么                                            |
| --------- | ------------------------------------------------- |
| 第一部分：按题视图 | 每道题的审题、材料层、术语对应、整题答案                              |
| 第二部分：六桶索引 | 时代背景、理论、经济全球化、政治多极化、中国、联合国                        |
| 第三部分：易混辨析 | 共同利益 vs 正确义利观；人类命运共同体 vs 中国智慧中国方案；联合国框架 vs 一般国际组织 |
| 第四部分：禁用条件 | 哪些材料信号不足时慎用该术语                                    |

六桶索引适合复习，按题视图适合迁移。学生版不能只保留索引。

## 2. 每道高频题必须有整题卷面答案

现在每个术语有答案句，但没有整题答案。学生会出现两个问题：

一是不会排序，不知道哪个点先写。
二是不会取舍，把所有术语堆进一段。

建议每道题补四件东西：

1. **审题结论**：题型、主体、知识范围、作答角度。
2. **材料分层**：材料一、材料二、材料三分别触发什么。
3. **答案骨架**：写几段，每段承担什么功能。
4. **答题卡版答案**：每点 40 到 70 字，术语＋材料＋结论。

## 3. 学生版应去掉后台痕迹

细则位置、必答、分值、DOCX 段落、PPT 页码等信息适合留在教师证据版，不适合进入学生版。

学生版建议改成：

| 当前字段    | 学生版改法       |
| ------- | ----------- |
| 细则位置    | 例题依据        |
| 来源      | 例题          |
| 必答      | 高优先级        |
| 第几段、第几页 | 删除，留给教师版    |
| 2分、1分   | 删除或改成高频得分方向 |
| 答案句     | 卷面表达        |

## 4. 每个高危术语补禁用条件

目前有少量辨析提醒，但还不够。建议至少给这些术语加禁用条件：

| 术语          | 禁用或慎用提醒                                   |
| ----------- | ----------------------------------------- |
| 和平与发展仍是时代主题 | 题目只问企业经营、国内政策、单一经济措施时，不要强行开时代主题。          |
| 经济全球化正确方向   | 不能只写普惠、平衡、共赢，核心表述要保留开放、包容、普惠、平衡、共赢。       |
| 两个市场、两种资源   | 没有国内国际联动、开放合作、国际分工、市场资源配置时慎用。             |
| 国际组织赋予的权利   | 一般海外法律纠纷不能直接上升到国际组织机制。                    |
| 新型国际关系      | 没有国家间合作、互利共赢、共同发展空间时慎用。                   |
| 人类命运共同体     | 不能作为所有国际题的万能结尾，需有全球共同利益、公共产品、共同发展或共同安全指向。 |
| 联合国宪章       | 没有联合国、宪章、国际法治、主权平等、安理会等材料信号时慎用。           |

## 5. 材料触发要拆成原词和推导

现在材料触发写得较长，有时像教师讲解。建议拆成两列：

| 材料原词                | 推导               |
| ------------------- | ---------------- |
| 治理赤字、规则法治缺位、有效治理期待  | 全球治理倡议正逢其时       |
| 自贸协定、免签、人才流、商品流、资金流 | 高水平开放、贸易投资自由化便利化 |
| 全球南方、农业增产、培养本土人才    | 共同利益、南南合作、普惠平衡共赢 |
| 联合国宪章、主权平等、国际法治     | 联合国框架、宪章宗旨和原则    |
| 公共产品、中国方案、大国担当      | 中国智慧中国方案、人类命运共同体 |

学生迁移靠材料原词，不能只靠教师解释。

## 6. 同类项合并要有三档

现在有些合并仍偏粗。建议给 Codex 设三档：

| 合并档   | 使用规则                 |
| ----- | -------------------- |
| 可硬合并  | 教材语义同源，材料触发相近，卷面句可共用 |
| 可交叉索引 | 术语相关，但题型、材料层、主体不同    |
| 禁止合并  | 合并后会改变主体、知识范围或得分逻辑   |

例如：

经济全球化正确方向和普惠平衡共赢，可以交叉索引，但核心标题必须保留完整五词。
共同利益和正确义利观，只能辨析，不能合并。
联合国宪章和一般国际组织机制，不能合并。
新型国际关系和人类命运共同体，可以相邻呈现，但不能互相替代。

# D. 可保留优点

1. **经济全球化正确方向保留了完整五词**：开放、包容、普惠、平衡、共赢，这点比旧稿安全。
2. **正确义利观已经放入中国桶**，没有继续错挂政治多极化。
3. **人类命运共同体进入中国桶**，方向正确，符合中国外交属性。
4. **联合国桶没有泛化到所有国际组织**，并提醒国际组织不等于联合国，这点应保留。
5. **共同利益和正确义利观有辨析意识**，能防止学生把合作基础和中国合作方式混成一个点。
6. **材料触发普遍比旧稿更具体**，多数条目能看到材料信号和术语之间的对应。
7. **卷面答案句的语言比流程稿更接近学生可写表达**，但仍需要分题改写。

# E. 给 Codex 的返修优先级

## P0：先修硬内容

1. 核对并修正联合国三大支柱表述。
2. 处理 2025海淀期中 Q16(2) 的贸易摩擦、知识产权、国际组织机制边界。
3. 处理时代主题条目中海淀期中 Q21(2) 的来源断链。
4. 给新型国际关系加中国外交属性或交叉桶标注。
5. 把三大关系从理论桶中拆到关系型设问专用模板。

## P1：再修合并方式

1. 所有多题合并条目，补分题卷面句。
2. 每个合并条目写清楚：哪些可以共用，哪些只能交叉索引。
3. 标出主干点、替换点、结尾点，避免学生堆术语。

## P2：改学生版结构

1. 主体改成按题视图。
2. 六桶降为索引或复习表。
3. 每道题补整题卷面答案。
4. 删除或转移细则位置、分值、DOCX/PPT 页码、必答等后台信息。

## P3：做迁移测试

用两类题测试：

1. 同源题：通州、朝阳、顺义这些已入表题，检验学生能否复现完整答案。
2. 近似新题：只给材料和设问，不给术语，检验学生能否从材料触发六桶术语。

**最终返修口径：这版可以作为教师后台术语库继续打磨，暂不适合作为学生终稿；下一步必须转成按题视图＋六桶索引，并补整题卷面答案。**
