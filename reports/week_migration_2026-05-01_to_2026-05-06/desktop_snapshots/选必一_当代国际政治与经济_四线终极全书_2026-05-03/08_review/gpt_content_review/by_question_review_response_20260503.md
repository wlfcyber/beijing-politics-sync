# C should_fix_transfer

## 1. 按题视图和六桶桥接表需要双向校准

当前最大迁移风险是：**按题视图没有写的点，桥接表却列为命中；桥接表列为命中的点，学生回到题目找不到。**

优先校准这些：

| 术语            | 当前问题                    | 处理                 |
| ------------- | ----------------------- | ------------------ |
| 和平与发展仍是时代主题   | 顺义题在桥接表命中，但按题答案没有出现     | 补入按题答案或从桥接表降级为可选背景 |
| 新型国际关系        | 海淀二模在桥接表命中，但按题视图没有出现    | 补为可选点或删除           |
| 参与全球经济治理和规则制定 | 主挂政治多极化，按题语境更像经济全球化规则风险 | 调整主桶               |
| 中国全球治理理念与价值取向 | 合并过宽                    | 拆主干和备选             |

## 2. 每题应标主干点、可选点、慎用点

现在命中核心点列得比较全，但学生不知道哪些必须先写，哪些只是替换或补充。

建议每题改成：

| 层级  | 示例                |
| --- | ----------------- |
| 主干点 | 本题最稳的 2 到 3 个得分方向 |
| 可选点 | 材料支持时可以补写         |
| 慎用点 | 容易套错、证据不足时不要写     |

例如 2026朝阳一模 Q20 可以这样分：

| 层级  | 内容                                |
| --- | --------------------------------- |
| 主干点 | 科技创新与综合国力；共享发展经验与中国方案；高水平开放与经济全球化 |
| 可选点 | 新型国际关系；和平与发展时代主题；共商共建共享           |
| 慎用点 | 不要把所有中国贡献都写成人类命运共同体一个结尾句          |

## 3. 答题卡版答案仍有几处不像答题卡

主要问题是句子过长、术语过密、材料对应偏泛。

重点压缩这些题：

| 题目           | 问题                                  | 建议                             |
| ------------ | ----------------------------------- | ------------------------------ |
| 2026朝阳期中 Q17 | 第三段术语堆叠：正确义利观、中国智慧中国方案、人类命运共同体都在一句里 | 保留发展中国家需求和能力建设，再择两个中国外交术语      |
| 2026朝阳一模 Q20 | 四段都很完整，但最后一段有泛化结尾感                  | 标明第四段为可选升华，考场时间不足时可删           |
| 2026西城期末 Q20 | 第二、三段术语密度过高                         | 拆成全球问题、国际承诺、中国实践三个清晰采分点        |
| 2024东城一模 Q16 | 民族的发展不是封闭的自我完成，偏议论文感                | 改成更政治答题卡式：中华文明创造为人类现代化探索提供中国智慧 |

## 4. 材料抓手建议拆成材料原词和教材推导

当前材料抓手已经比上一版清楚，但复杂题仍建议用双列表，学生更容易迁移。

例如：

| 材料原词           | 教材推导               |
| -------------- | ------------------ |
| 治理赤字、规则法治缺位    | 全球治理体系改革、公正合理国际秩序  |
| 主权平等、国际法治、多边主义 | 联合国宪章宗旨和原则、国际关系民主化 |
| 全球南方能力建设、智能鸿沟  | 正确义利观、中国方案、人类命运共同体 |
| NDC、低碳转型、巴黎协定  | 履行国际义务、全球气候治理、中国责任 |
| 商标权、知识产权、败诉反诉  | 企业合规、知识产权保护、国际规则风险 |

## 5. 拓展题要和主线题分层

2024东城一模 Q16、2024东城一模 Q20、2025海淀期末 Q22 可以保留，但需要换位置。

建议结构：

1. 第一部分：选必一主线题
2. 第二部分：联合国与全球治理专项题
3. 第三部分：综合题中的选必一迁移
4. 第四部分：短文题可选表达

这样学生不会把开放短文、综合哲学题、经济模块题学成固定选必一模板。

# D 可保留优点

1. **按题视图已经成型。**
   每题都有审题结论、材料抓手、命中核心点、答题卡版答案、慎用提醒，比纯六桶术语表更适合学生训练。

2. **整题答案补上了。**
   学生不再只看到术语卡，能够看到一整道题如何分层成文。

3. **Q16(2) 已经修正了最危险的贸易摩擦泛化倾向。**
   当前稿明确提醒看到商标权、知识产权时先写合规和知识产权风险，这个方向应保留。

4. **经济全球化正确方向保留完整五词。**
   开放、包容、普惠、平衡、共赢没有被压缩成泛泛的正确方向，适合学生背诵和迁移。

5. **联合国三大支柱已修正为和平与安全、发展、人权。**
   这是上一轮的重要硬伤，当前稿处理正确。

6. **中国外交术语桶位整体比上一版安全。**
   人类命运共同体、正确义利观、中国智慧中国方案主要进入中国桶，并配有慎用提醒。

7. **桥接表有实际价值。**
   它能让学生从一道题回到六桶，也能从一个核心点回看多道题，方向正确。

8. **慎用提醒质量明显提高。**
   如共同利益和正确义利观的区分、联合国和一般国际组织的区分、经济全球化五词保留，都是对学生迁移有帮助的提醒。

# E 给 Codex 的返修优先级

## P0：进入学生预览版前必须修

1. 校准桥接表和按题视图不一致项：顺义题的时代主题，海淀二模的新型国际关系。
2. 重修 2025海淀期中 Q16(2)：主体分工、主桶归属、企业经营层是否漏点。
3. 重修 2025海淀期中 Q21(2)：外交变的历史阶段表达，避免用当前时代背景统摄全程。
4. 重修 2026西城期末 Q20：删减术语堆叠，核对联合国核心作用的材料信号。
5. 将 2024东城一模 Q16/Q20、2025海淀期末 Q22 移入拓展迁移区，不和主线题同级。
6. 把中国全球治理理念与价值取向拆成主干点和备选点，不能作为术语大包直接给学生背。

## P1：提高学生迁移能力

1. 每题增加主干点、可选点、慎用点。
2. 材料抓手改成材料原词和教材推导双列表。
3. 每题标出考场可删句，避免答案过长。
4. 对多桶组合题增加作答顺序，例如先能力，再贡献，再开放，再升华。
5. 桥接表增加主桶、交叉桶、禁止误挂桶。

## P2：语言和版式优化

1. 把过长句压成 40 到 70 字左右的采分句。
2. 降低宏大表述密度，每段最多保留 1 到 2 个核心政治术语。
3. 把教师提醒转成学生能直接执行的判断句。
4. 保留教学预览版标识，但去掉后续校对等制作流程感强的说明，正式学生版中不出现。

**最终口径：这版已经从教师术语库升级为按题训练雏形，但仍有 P0 内容风险；修完 P0 后，可以进入学生预览版。**
