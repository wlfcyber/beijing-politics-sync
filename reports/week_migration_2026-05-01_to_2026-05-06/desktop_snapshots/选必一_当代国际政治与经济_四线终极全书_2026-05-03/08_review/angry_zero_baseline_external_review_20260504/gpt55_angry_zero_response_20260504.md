这份讲义已经有完整闭环，但零基础学生读起来仍像在啃一套巨大索引。
最大问题不是缺知识点，而是优先级不够清楚，学生不知道先写哪三句、哪些只作备选。
还有少量框架落点和答案句功能不一致，足以导致误套，正式给学生前要修。

## 2. 最危险的 8 个误套风险

| 风险点           | 学生会怎么错用                           | 建议防错                                                 |
| ------------- | --------------------------------- | ---------------------------------------------------- |
| 和平与发展仍是时代主题   | 任何国际题都先写时代主题，导致开头空泛。              | 加硬规则：只有设问问时代背景、世界大势、倡议正逢其时、外交变迁、全球合作必要性时优先写。         |
| 经济全球化五词       | 学生只背开放、包容、普惠、平衡、共赢，不会接材料。         | 保留五词，同时给三种材料触发：市场开放对应开放，最不发达国家对应普惠平衡，互利合作对应共赢。       |
| 新型国际关系        | 学生看到合作就写新型国际关系。                   | 加硬规则：要有国家间关系、国际合作方式、互利共赢、非零和竞争等信号。企业经营、开放政策、技术标准题慎用。 |
| 共商共建共享的全球治理观  | 学生看到共享发展成果就写全球治理观。                | 区分：全球治理体系改革写全球治理观，普通开放合作写共享发展或互利共赢。                  |
| 人类命运共同体       | 学生当万能结尾句，任何中国贡献题都写。               | 加选择规则：共同挑战、共同前途、全球公共产品、共同安全或共同发展明确出现时用。              |
| 中国智慧/中国方案     | 学生写成空口号，缺少材料事实。                   | 每次必须接一个事实：技术、经验、倡议、制度方案、国际公共产品。没有事实不写。               |
| 联合国核心作用       | 学生把一般国际组织、WTO、国际规则、巴黎协定都当联合国核心作用。 | 已经较安全，但还要在学生版写清：联合国、宪章、安理会、联合国体系、巴黎协定框架等信号明确时才写。     |
| 正确义利观 vs 共同利益 | 学生把合作为什么成立和中国怎样合作合并成一个点。          | 继续保留现有辨析，并在每道相关题里用一句话提醒：共同利益解释合作基础，正确义利观解释中国合作方式。    |

## 3. 哪些地方太密、太像索引

最密的是三类。

第一类是 **答题点自身积累**。它现在像教师词库，很多条目是同义词并排，例如贸易投资自由化便利化相关表达反复出现。学生很容易以为这些都要背，结果写卷面时堆碎片。建议改名为 **可替换表达**，每条最多保留 3 到 5 个，删除重复低信息短语。

第二类是 **长题的条目拆解**。2026朝阳一模 Q20、2026朝阳期中 Q17、2026西城期末 Q20 这几题需要加固定栏：主干必写 3 到 4 点、可选补充、时间不足删句。现在只有少数题有类似提示，覆盖不够。

第三类是 **六桶总框架索引**。索引适合复盘，不适合开篇压给零基础学生。建议开头先放一张极简路线图，然后再进入按题训练，六桶索引后移。

## 6. 示范文本

### A. 开头使用路线，建议改成这样

这份讲义不要从头到尾硬背。
每道题先做三步：

1. **先判题型**：问原因，就找能力来源和合作基础；问措施，就找主体和路径；问意义，就分中国、合作方、世界秩序；问评价，就先亮明性质再分析影响。
2. **再抓材料信号**：看到自贸协定、免签、人才流，就想经济全球化；看到治理赤字、主权平等、国际法治，就想全球治理改革；看到联合国、宪章、安理会，就想联合国桶；看到中国经验、公共产品、全球南方，就想中国智慧中国方案。
3. **最后写卷面句**：每点按教材术语 + 材料事实 + 作用结论写。时间不够时，只写主干必写点，不抄完整词库。

### B. 长题样例，建议用 2026朝阳一模 Q20 改成这样

**题目核心**：中国为什么能为全球发展注入稳定性和正能量。

**考场主干必写 4 点**

1. **科技能力**
   当前国际竞争的实质是以经济和科技实力为基础的综合国力较量。中国坚持创新驱动，AI、大模型、创新药等科技成果增强综合国力，为全球产业链供应链稳定提供支撑。

2. **开放联通**
   中国依托超大规模市场，扩大自贸朋友圈，推动免签和人员、商品、资金流动，联动国内国际两个市场两种资源，推动贸易投资自由化便利化。

3. **发展合作**
   中国通过鲁班工坊、菌草技术和现代化经验共享，帮助发展中国家培养人才、改善民生、增强内生发展动力，体现正确义利观和互利共赢。

4. **中国方案**
   中国分享发展成果，贡献中国智慧中国方案，推动经济全球化朝更加开放、包容、普惠、平衡、共赢方向发展，推动构建人类命运共同体。

**可选补充**
若分值高，可补和平与发展仍是时代主题、新型国际关系、共商共建共享。

**时间不足删句**
删掉重复的时代背景句和重复的开放合作句，保留科技能力、开放联通、发展合作、中国方案四层。

### C. 六桶复盘条目，建议改成这样

**核心点：推动经济全球化朝着更加开放、包容、普惠、平衡、共赢方向发展**

**什么时候用**
材料出现中国扩大开放、自贸协定、一带一路、最不发达国家零关税、全球南方受益、发展成果共享、开放合作反对保护主义时使用。

**先背这一句**
中国通过扩大开放和互利合作，推动经济全球化朝着更加开放、包容、普惠、平衡、共赢方向发展，使发展成果更多更公平惠及各国。

**材料怎么接**
自贸协定、免签、市场开放，对应开放。
零关税、全球南方、发展中国家受益，对应普惠和平衡。
互利合作、共享发展机遇，对应共赢。

**不要这样用**
只写经济全球化正确方向几个字。
材料只讲企业内部经营、国内产业升级、个人奋斗时，不要强行套五词。

## 7. 总结

这份讲义已经能覆盖知识点，也有按题闭环，但对零基础学生仍然太重。最大短板是 **优先级不足、词库过密、个别链条错挂、长题缺删句规则**。修完上述 MUST_FIX 和 SHOULD_FIX 后，它才更像学生能拿来直接练卷面答案的讲义。
