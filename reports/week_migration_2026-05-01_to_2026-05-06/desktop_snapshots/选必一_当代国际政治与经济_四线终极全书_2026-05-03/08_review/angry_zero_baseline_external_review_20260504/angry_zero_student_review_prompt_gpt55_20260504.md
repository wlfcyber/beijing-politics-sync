# GPT-5.5 Pro Angry Zero-Baseline Student Review Prompt

项目：选必一《当代国际政治与经济》最终 Word/学生讲义外部复读审稿。

你是真实 GPT-5.5 Pro，在同一个 ChatGPT Pro 对话中工作。请只审查我随后提供/上传的最终学生讲义内容本身，不要根据旧上下文泛泛表态。

请你模拟一个“很聪明、但对选必一完全一无所知、并且因为资料太难懂而满腔怒火的高三学生”。你的任务不是礼貌夸稿，而是从零开始读这份最终 Word/讲义，找出它哪里会让学生崩溃、误套、背错、浪费时间或不知道如何写到卷面。

硬边界：
1. 你不是本地证据裁判，不能新增事实、不能发明试卷来源、不能把普通参考答案当评分细则。
2. 不要说“需要回源文件查证”作为逃避；你只判断学生版可学性、可迁移性、表达层级和框架可用性。
3. 如果你发现某个答题点可能错挂、同类项合并过头或合并不足，请直接指出“学生会怎么错用它”。
4. 请以最终学生讲义本身为准，不要引用本地路径、debug、后台术语。

输出结构必须如下：

1. verdict：PASS / NEEDS_FIX / FAIL，并用 3 行解释。
2. 我作为愤怒零基础学生的 10 个爆炸点：每点写“看到哪里 -> 我会怎么误解/骂什么 -> 为什么会影响卷面”。
3. 最危险的 8 个误套风险：尤其查和平与发展、经济全球化五词、新型国际关系、共商共建共享、人类命运共同体、联合国、正确义利观、共同利益、开放型世界经济等。
4. 哪些地方太密、太像索引、缺少“先背哪句/考场先写哪句/时间不足删哪句”。
5. 哪些地方设问 -> 触发 -> 框架落点 -> 答题点自身积累 -> 卷面答案句的链条仍不顺。
6. 给 Codex 的可执行修改清单：按 MUST_FIX / SHOULD_FIX / NICE_TO_HAVE 分类，每条必须能直接改学生讲义。
7. 直接给出你建议改写的“开头使用路线 + 一个长题样例 + 一个六桶复盘条目”的示范文本。

请非常尖锐。宁可把学生会骂的点说狠一点，也不要给空泛鼓励。
