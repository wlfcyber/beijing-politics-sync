## Hard Rules Converted To Checks

1. Main student document uses six buckets: 时代背景、理论、经济全球化、政治多极化、中国、联合国.
2. Every core term entry keeps: 术语、完整设问、细则位置、来源、材料触发、答案句, plus 表述积累 for merged variants.
3. `术语` must preserve scoring-rule/marking wording; invented summaries are containers only, not terms.
4. Same scoring core merges before expression accumulation. Merge must not delete information. Core name preserves highest-information wording.
5. Economic-globalization direction core must preserve `开放、包容、普惠、平衡、共赢`; vague `经济全球化正确方向` is only an expression variant.
6. 时代主题/和平与发展/中国做法符合时代主题 belong under one core, with variants accumulated.
7. 新型国际关系/合作共赢 variants belong under one core when the answer function is the same.
8. Material trigger must explain prompt relation + material relation + why the term is triggered; it cannot be material paraphrase.
9. Answer sentence must be answer-sheet prose: term + material fact + causal/effect/result chain.
10. Forbidden student wording: local path, debug, audit, model chat, `评标`, `细则要求`, `采分点`, `要落到`, `材料中`, `设问要求`, `本题需要`, `v7`, `v8`.
11. General/summary and subpoint layers must both be checked; do not keep only the total layer when scoring has 分说.
12. 2026通州期末 Q20 must keep the user-confirmed six-point structure.
13. 2026朝阳期中 Q17 must keep 总说 and all 分说, with module boundary exclusions recorded separately.
14. 2026石景山期末 is excluded for all modules unless user provides new usable scoring rules.
15. Ordinary reference answers cannot be promoted to scoring rubrics; mark them P1/reference-only unless user-confirmed or supplemented by true marking evidence.
16. Mixed-book terms are excluded or boundary-noted; do not force 必修二、选必二、选必三、政治与法治、必修四 into the xuanbiyi main chain.
