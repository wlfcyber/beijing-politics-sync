## Student-Facing Core Unit

Each merged core point should show:

- 核心采分点：highest-information scoring wording.
- 表述积累：same-core variants from different questions.
- 来源题目：suite/question list.
- 完整设问：full prompt for each source question.
- 细则位置：scoring/rubric/marking position and evidence level.
- 材料触发：why the material relation triggers the term.
- 答案句：answer-sheet sentence.

## Merge Principle

Same textbook core, same scoring function, or same answer role becomes one core point. Differences are preserved as expression accumulation, not lost and not split into fake rows.
