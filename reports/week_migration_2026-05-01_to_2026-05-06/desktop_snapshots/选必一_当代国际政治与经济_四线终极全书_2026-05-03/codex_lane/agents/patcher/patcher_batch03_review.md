## 2. 桶位裁定

### 中国理念同槽

裁定：采用总控预融合方案，主桶为 `中国`，交叉提示 `政治多极化 / 联合国`；worker 主挂 `政治多极化` 需要改。

理由：

- 该槽细则原词是“中国理念：命运共同体 / 全人类共同价值 / 正确义利观 / 共商共建共享 / 践行真正的多边主义 / 坚持互利共赢”，评分功能是在解释中国为什么参与全球气候治理，主答题动作是中国全球治理理念与价值取向。
- `命运共同体`、`正确义利观`、`全人类共同价值` 明显是中国外交价值/主张；`真正的多边主义`、`共商共建共享`可交叉政治多极化或联合国，但不能把整槽主挂政治多极化。
- 这一个槽只得 1 分，是同槽替代表述，不可拆成 HMC、正确义利观、共商共建共享、多边主义、互利共赢多个独立核心或多个频次。

### 联合国核心作用

裁定：`维护联合国的核心作用 / 完善全球治理体系` 主桶必须是 `联合国`。

- worker 和预融合均保留了联合国核心作用，方向正确。
- 后续不得泛化成普通 `多边主义`、`全球治理` 或 `国际组织作用`；学生稿里也要保留“联合国核心作用”这一高信息表达。

### 全球可持续发展 / 清洁世界

裁定：不宜主挂政治多极化；应作为中国气候治理实践效果/结果语言处理，教师核查层保留 P0 效果槽。

- worker 将其放入政治多极化偏宽。
- 总控预融合放入中国并标为 result expression 更稳，但学生主表中不宜把它升级成高频核心，除非后续多题细则反复出现。
