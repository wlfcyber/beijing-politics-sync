## 补丁检查标准

### A. 一材料多答题点

判断单位不是“材料段落”或“表格一行”，而是评分位置。

- 若同一材料信号对应多个评分位置、多个分值层、总说/分说、或不同六桶，应拆成多个可核查点。
- 每个点必须保留 `术语 / 完整设问 / 细则位置 / 来源 / 材料触发 / 答案句`。
- 同一材料可以重复出现在多个点的 `材料触发` 中，但不能把多个评分点压成一个“综合答案层次”。
- 总说不单独赋分时，必须在 `细则位置` 说明证据层级；总说单独赋分时，必须作为独立点保留。

强制复查样本：

- `2026通州期末 Q20`：用户复核六点必须全收，人类命运共同体不得错归为四大全球倡议。
- `2026朝阳期中 Q17`：三组关系各有总说/分说，不能只保留三条总关系。
- `2025海淀期中 Q16(2)`：贸易摩擦、国际组织赋权、全球经济治理和规则制定要从图片表格细则核清。
- `2026朝阳一模 Q20`：科技能力、中国担当、开放机制、合作方向等层次不能因同在一段答案中被合成一个大点。

### B. 同核心误拆

合并看采分功能和教材核心，不只看字面。

- `时代主题 / 和平与发展 / 和平与发展仍是时代主题 / 中国做法符合和平与发展时代主题`：归为 `和平与发展仍是时代主题`，其他写入表述积累。
- `推动经济全球化朝着更加开放、包容、普惠、平衡、共赢方向发展 / 经济全球化正确方向 / 普惠、平衡、共赢`：归为最高信息量原词。
- `合作共赢的新型国际关系 / 推动建设合作共赢的新型国际关系 / 合作共赢、互利共赢、共享发展成果`：同一采分功能时归并。
- 同一题同一评分位置的并列原词不得拆成多条；例如“中国作为联合国创始会员国 / 安理会常任理事国；世界上最大的发展中国家 / 国际政治经济格局中的重要力量”若在同一 `中国的地位` 点位中，应是一条合并术语。

### C. 粗暴合并丢最高信息量

合并后的核心名必须保留最高信息量的评分原词。

- 禁止把 `推动经济全球化朝着更加开放、包容、普惠、平衡、共赢方向发展` 压成 `经济全球化正确方向`。
- 禁止把 `当前国际竞争的实质是以经济和科技实力为基础的综合国力较量；坚持创新驱动战略` 压成 `国际竞争实质与创新驱动` 后当作术语。
- 禁止把 `充分利用国际组织赋予的权利；积极参与全球经济治理和规则制定` 压成 `全球经济治理规则` 后当作术语。
- 可以设置短标题或容器，但 `术语：` 和六桶核心采分点必须尽量贴近细则原词。

### D. 六桶框架归位

默认六桶：`时代背景 / 理论 / 经济全球化 / 政治多极化 / 中国 / 联合国`。

- `时代背景`：和平与发展、经济全球化/世界多极化作为背景、霸权主义/强权政治/逆全球化、治理赤字等世界形势。
- `理论`：国家利益、共同利益、国际竞争实质、综合国力、合作成立的理论理由。
- `经济全球化`：国际经济循环、两个市场两种资源、贸易投资便利化、要素跨国流动、经济全球化正确方向。
- `政治多极化`：全球治理观、国际新秩序、国际关系民主化、多边主义、新型国际关系等国际秩序和治理方向。
- `中国`：中国智慧中国方案、大国担当、人类命运共同体、正确义利观、兼顾他国合理关切、中国地位。
- `联合国`：联合国地位作用、《联合国宪章》宗旨原则、以联合国为核心的国际体系、联合国与中国关系。

边界提醒：`推进高水平对外开放 / 扩大国际市场 / 落实开放发展理念 / 高质量发展 / 现代化产业体系` 等若评分源明确属于必修二，不进选必一主链，只能做边界说明或答题链补充。
