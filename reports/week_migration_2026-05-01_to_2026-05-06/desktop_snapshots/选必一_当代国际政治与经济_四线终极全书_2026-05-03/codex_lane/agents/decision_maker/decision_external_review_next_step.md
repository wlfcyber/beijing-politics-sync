# Decision: External Review Next Step

Run: 选必一《当代国际政治与经济》四线终极全书 2026-05-03

Role: Codex A / Decision Maker

Scope: 只写下一步执行方案；不改学生稿、不改桥接表、不覆盖其他角色输出。

## 总裁决

当前版本不能宣布完成，不能进入对外学生终稿、Word/PDF、coverage closed 或 FINAL_ACCEPTANCE。二审共同结论是：按题训练稿和六桶桥接表方向正确，但仍存在 P0 内容风险。下一轮必须先做 P0 修复，再做 P1 迁移增强，最后做 P2 语言版式优化。

执行主线：`证据核验队列 -> 学生稿结构修补队列 -> Patcher 复查 -> Governor 门禁`。不要让任何一个角色停在“等别人全做完”状态；证据核验和纯结构语言修补可以并行。

## P0 必修项

### P0-1 校准按题视图与六桶桥接表不一致

任务：

- 顺义一模 Q20：桥接表列入 `和平与发展仍是时代主题`，但按题答案没有出现。必须裁定是补入按题答案，还是从桥接表降为可选背景/表述积累。
- 海淀二模 Q21：桥接表列入 `推动建设合作共赢的新型国际关系`，但按题视图没有作为核心点。必须裁定是补为可选点，还是从桥接表移除。

需要先本地证据核验：是。要查对应 scoring atoms / merge register / source evidence 后才能改核心命中关系。

可直接结构改：否。桥接关系不能凭外审建议直接改。

负责顺序：证据核验者先给 `confirmed / downgrade_to_optional / remove_from_crosswalk`；Patcher 再改任务单；Governor 最后判是否仍可进 preview。

### P0-2 重修 2025海淀期中 Q16(2)

任务：

- 主体分工拆清：企业层、行业组织层、政府层。
- 桶位修正：`利用国际组织赋予的权利，积极参与全球经济治理和规则制定` 主桶应优先判为经济全球化/全球经济治理规则风险，不要主挂政治多极化；不要放联合国桶。
- 核验是否漏企业经营层：目标市场、成本压力、品牌定位、本土化经营、供应链等。
- 把“如果材料进一步呈现……”从卷面句逻辑中移出，改为使用条件/慎用提醒。

需要先本地证据核验：

- 主体分工和企业经营层是否为评分/参考可用点，需要核验材料三、图片评分材料、参考答案边界。
- 桶位与核心术语归属需要核验融合表/merge register。

可直接结构/语言改：

- “如果……”条件句从答题卡挪到慎用提醒，这是结构语言修补，可直接改。

负责顺序：证据核验者先查漏点和桶位；结构劳动者可并行准备“条件句移出卷面”的改稿任务；Patcher 合并成一条返修指令。

### P0-3 重修 2025海淀期中 Q21(2)

任务：

- “变”的表述要阶段化，不能用“和平与发展成为时代主题”统摄新中国外交全程。
- “和平与发展仍是时代主题”与“中国外交宗旨：维护世界和平、促进共同发展”继续分开。
- 图片备注未分类内容不得刷成独立核心。

需要先本地证据核验：是。必须回到 image8 表格、Batch01/merge register 中对“世界之变 / 中国之变 / 不变”的分层。

可直接结构/语言改：阶段化表达的文字框架可以先拟，但不得落稿为事实结论，等证据核验确认后执行。

负责顺序：证据核验者先输出阶段分层；Patcher 再判哪些是主干点、哪些是备注/表述积累；Governor 看是否消除历史阶段错位。

### P0-4 重修 2026西城期末 Q20

任务：

- 第二段理念词不能堆满六个：`人类命运共同体 / 全人类共同价值 / 正确义利观 / 共商共建共享 / 真正的多边主义 / 互利共赢` 应择最贴材料的 2-3 个进卷面，其余入表述积累。
- 核对 `维护联合国在国际事务中的核心作用` 的材料信号；若材料只足以支撑巴黎协定/NDC/联合国气候框架，就改为更窄表达。
- 继续保留角度2 `4选3`、同槽变体不拆频、绿色发展只作实践支撑的边界。

需要先本地证据核验：

- 六个理念是否同一得分槽，必须核验 Batch03 细则结构。
- 联合国核心作用是否有足够材料信号，必须核验题面、细则和 evidence card。

可直接结构/语言改：

- 降低术语密度、把答案拆成“全球问题 / 国际承诺 / 中国实践”三点，是结构语言修补，可直接准备。

负责顺序：证据核验者确认同槽和 UN 信号；结构劳动者并行准备降密度版本；Patcher 防止把同槽变体拆频。

### P0-5 综合题/开放题降级为迁移区

任务：

- 2024东城一模 Q16、2024东城一模 Q20、2025海淀期末 Q22 不和主线题完全同级。
- 标成 `选必一迁移练习 / 选必一可用片段 / 边界题`。
- Q22 在审题结论前加前置条件：只有出现“中国与世界 / 共同挑战 / 全球治理 / 共同发展”等信号时才用选必一角度。
- 东城 Q16 的“答题卡版答案”应改为“选必一可用片段”，避免误认为完整答案模板。

需要先本地证据核验：

- 这些题在证据系统中是否为 P2、综合题、短文题或跨模块题，需要核验 coverage 和 evidence labels。

可直接结构/语言改：

- 栏目降级、标题前缀、前置条件、答题卡标题调整，均可直接执行。

负责顺序：结构劳动者先拟栏目移动方案；证据核验者确认标签；Patcher 查是否误删可用选必一角度。

### P0-6 拆开“中国全球治理理念与价值取向”大包

任务：

- 主干点与备选点分开：主干优先 `人类命运共同体 / 真正的多边主义`，备选按材料选择 `正确义利观 / 共商共建共享 / 全人类共同价值 / 互利共赢`。
- 桥接表里 `贡献中国智慧、中国方案、中国力量` 命中多题，必须按语境写使用提醒：全球治理、全球发展、联合国、气候治理、文明/综合题不能用同一句万能结尾。

需要先本地证据核验：

- 各语境化措辞是否在对应细则中有支撑，需要核验 merge register 和各题 scoring atoms。

可直接结构/语言改：

- “主干点 / 备选点 / 慎用点”的表格结构可直接改。

负责顺序：证据核验者列出各题可支撑的中国智慧语境；结构劳动者把大包改成分层表；Patcher 查是否把同一 scoring slot 拆成多个频次。

需要先本地证据核验：若判断某段是否独立得分，需要核验；纯“考场时间不足可删”的提示可直接改。

执行：先对 P0 冲突项补，其他行后置。

## P2 语言和版式项

### P2-1 压缩过长采分句

需要先本地证据核验：不需要，除非压缩时删掉评分术语原词。

规则：每段最多保留 1-2 个核心政治术语；术语原词不压扁。

### P2-2 统一段间连接词

需要先本地证据核验：不需要。

规则：

- 三段式：首先 / 其次 / 同时或更进一步。
- 双向关系：一方面 / 另一方面。
- 关系处理题：要处理好 X / 要处理好 Y / 还要处理好 Z。

### P2-3 慎用提醒拆成“使用条件 / 易错提醒”

需要先本地证据核验：不需要。若使用条件涉及评分边界，再交 Patcher。

### P2-4 桥接表排版统一

需要先本地证据核验：部分需要。核心点列是否能拆成单核心，需要核验 scoring point 是否同一分项；分号、顿号统一本身可直接改。

## 防止 Codex 停在半路

1. P0 拆成两个并行队列：
   - `Evidence Queue`：顺义时代主题、海淀二模新型国际关系、Q16(2) 主体/桶位/漏点、Q21(2) 阶段化、西城 Q20 同槽与 UN 信号、综合题证据标签、中国智慧语境。
   - `Direct Edit Queue`：条件句移出卷面、综合题迁移区标题、材料抓手格式、答案降密度草案、主干/可选/慎用表格框架。
2. 每个 P0 项必须写状态：`needs_evidence`、`direct_structural_fix`、`ready_for_patcher`、`blocked_with_reason`。禁止只写“待处理”。
3. 如果证据核验未完成，结构劳动者仍可先做不改变事实判断的框架修补。
4. 如果某题证据有争议，该题先标 `preview_recheck_needed`，其他题继续推进。
5. 每完成两个 P0 项，就叫醒 Patcher 做增量复查，不等六项全完。
6. Patcher 每次复查后立即叫醒 Governor 做小门禁，门禁只判“能否进入学生预览下一版”，不判最终。
7. 自动化检测者持续检查：学生稿是否出现终稿/闭环/FINAL_ACCEPTANCE；P2/P1/P0 标签是否被混写；后台语是否泄漏。
