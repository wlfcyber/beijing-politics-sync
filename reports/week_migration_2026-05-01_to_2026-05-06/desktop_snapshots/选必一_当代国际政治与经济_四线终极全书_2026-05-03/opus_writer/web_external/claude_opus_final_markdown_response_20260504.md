# 选必一全书最终 Markdown 候选稿 审稿意见

声明:本次只审教学结构、可读性、迁移性、卷面表达。所有"术语真假/分值/必答口径/同核心是否真的同核心"由 Codex 据本地证据裁定,我不替它判。

---

## verdict

**NEEDS_FIX**

理由短版:六桶骨架、设问触发、材料触发、答案句变体、表述积累这条主链已经搭起来了——这是一份可以继续打磨的成品候选稿,不是要推倒重写的草稿。但有四类问题在当前形态下会让高三学生学错或学不会,必须修完才能 PASS:
1. 同一核心点下出现多份 `**答题点自身积累**` 但内部同义表述堆叠(典型如"和平与发展"族里塞了 14 个变体);
2. 多处 `框架落点` 与该核心点的真实归属错位(典型如东城一模 Q16 这道哲学文化题在本稿被标到 `中国 -> 推动构建人类命运共同体` 和 `政治多极化 -> 新型国际关系`,但其完整设问明确写"运用《哲学与文化》知识");
3. `**答案句变体**` 大量没有标点、用空格断句,卷面读起来不像考生写的;
4. `**慎用提醒**` 现在统一用一句"本题可作表述积累或开放题训练,不能当固定逐点模板"复读,失去了边界提醒的作用。

下面分项展开。

## must_fix_teaching_text(教学文本必改)

### MFT-1. 答案句标点缺失,断句靠空格——这是全稿最系统性的问题

抽样:

> 中国创新成果共享、鲁班工坊和菌草技术合作、中国式现代化经验交流以及扩大开放,符合和平与发展的时代主题,因而能为全球发展注入稳定性和正能量。

这一句格式正常。但全稿超过半数的 `答案句变体` 是这种形态:

> 面对发展鸿沟和安全赤字 各国利益交汇 命运交织 休戚与共 和平与发展仍是时代主题 中国答案顺应这一时代背景

> 中国通过劝和促谈、推动气候合作和提出全球人工智能治理倡议 为解决全球性问题贡献中国智慧和中国方案

> 中国坚持正确义利观 积极践行真正的多边主义 坚持合作共赢 致力于扩大同各国利益的汇合点

学生考场写卷面时不会用空格断句,他要么写顿号,要么写逗号,要么写分号。空格断句的句子学生抄下来背一遍,会以为这是某种"AI 草稿格式",自己写卷面时不知道该怎么落标点,最终要么全句不断,要么乱断。

**改法**:全稿统一为标点中文句式。每个 `答案句变体` 必须以"。"结尾,内部按"理论句 → 材料对接 → 回扣设问"三段式用"。"或"; "划分,层内并列用"、"或","。空格只在术语原话里出现(如"开放、包容、普惠、平衡、共赢"已经用顿号,这是对的)。

需要本地证据核验:无,纯文字规范。

### MFT-2. `答题点自身积累` 字段把同义变体塞成长清单,失去"积累"功能

抽样,"和平与发展仍是时代主题"那行的 `答题点自身积累`:

> 和平与发展仍是时代主题;时代主题;和平与发展成为时代主题;顺应各国人民愿望;经济全球化深入发展;中国的做法符合时代主题和平与发展;时代主题:和平和发展;全人类命运与共;国与国之间利益交汇命运交织休戚与共;和平与发展;和平与发展是当今时代主题;世界发展大势;服务国家发展大局;中非战略性共识符合和平与发展时代主题。

14 个分号串起来。学生看到这一行的反应只有两种:全部背、全不背。这一行作为"积累"的设计意图(让学生看到这个核心在不同题里以哪几种相近表述出现)被自身长度吞掉了。

更严重的是,里面混了三类不同性质的内容:
- 真正的同义变体("和平与发展""时代主题:和平和发展""和平与发展是当今时代主题");
- 派生短语("顺应各国人民愿望""世界发展大势""服务国家发展大局");
- 题目专属对接句("中非战略性共识符合和平与发展时代主题")。

这三类必须分开,学生才知道哪些是术语本体的稳定变体、哪些是某道题专属的承接表述。

**改法**:`答题点自身积累` 这一行拆成三个子字段:
- **核心同义变体**(2-4 条,术语本体的稳定改写);
- **常见承接短语**(3-5 条,在卷面里把术语接到材料的过渡词);
- **题目专属表述**(若有,标注题号,提示"此说法只在 X 题语境里能直接得分")。

需要本地证据核验:三类内容的边界(尤其"题目专属表述"是否真的只在该题成立)。

### MFT-3. `框架落点` 字段把综合题/哲学文化题归到选必一桶,会让学生学错迁移规则

最严重的两处:

**(a)** 2024东城一模 Q16,完整设问明确写"运用《哲学与文化》知识"。本稿把这道题的两条答案分别归到:
- `政治多极化 -> 推动建设相互尊重、公平正义、合作共赢的新型国际关系`;
- `中国 -> 推动构建人类命运共同体`。

这道题学生考场上调用的是哲学文化模块的"民族与世界、文明交流互鉴"。把它的两条派生句归进选必一的桶,会让学生看到"民族对世界做事 = 新型国际关系 + 人类命运共同体"这种错误对应——下次他在选必一题里看到"民族""文明",就反向去套这两个核心点。

**(b)** 2025海淀期末 Q22 是开放短文题,完整设问"自选角度、自拟题目,围绕愚公移山精神写一篇 200 字左右短文"。本稿把它归到 `中国 -> 推动构建人类命运共同体`,`答题点自身积累` 还专门加了一行"综合短文可选知识:人类命运共同体;中国智慧中国方案"。

这等于鼓励学生看到任何团结奋斗的开放题就往人类命运共同体上靠。

**改法**:在六桶之外开第七节"边界题与跨模块迁移",收东城 Q16、海淀期末 Q22、丰台一模 Q20(用户已在慎用与排除里点了它)、东城一模 Q20(经济模块为主)。这一节明确标"非选必一主线题,以下表达只能作为局部迁移材料,不进入桶内核心点统计"。原桶内不再放这些题的派生条目。

需要本地证据核验:每道题在本地证据系统里的"主属模块"标签。

### MFT-4. `**慎用提醒**` 字段在全稿里被复读成同一句,失效

抽样,本稿出现 `本题可作表述积累或开放题训练,不能当固定逐点模板。` 这句话(及其轻微变体)的位置粗数有 30+ 处。学生读到第三次就会自动跳过。

更关键的是,这句话本身没有教学生"什么时候才不能用",只是笼统说"别照抄"。学生需要的是:看到 X 信号才用、看到 Y 信号别用。

**改法**:`慎用提醒` 字段必须按"使用条件 + 易错提醒"两行写,且每条都要给出可识别的材料/设问信号。例:

不要写:"本题可作表述积累或开放题训练,不能当固定逐点模板。"

要写:"使用条件:材料明确出现自贸协定/免签/双向流动等开放合作信号时使用。 易错提醒:不要把'两个市场两种资源'压缩成'国内国际市场';五个词一起写或不写,不能只挑两个。"

如果某条目实在没有特殊使用条件,就删掉这个字段,而不是用一句无信息的话占位。

需要本地证据核验:每条慎用提醒里"使用条件"的判定信号是否与本地细则口径一致。

### MFT-5. `设问触发` 字段被压成五句话循环,起不到教审题的作用

全稿的 `设问触发` 实际上只有以下五种文字:

1. "先看题目追问的是原因、措施、意义还是评价,再决定把材料事实接到哪一个教材核心。"(对应"理解/阐释"题)
2. "追问'为什么能成立/为什么要这样做',先找合作、国家利益、时代背景或能力来源,再把行动接到对应教材术语。"(对应"为什么"题)
3. "追问'如何做到',按措施、机制、结果三步审题:做法先转成机制,再写对中国或世界的作用。"(对应"如何"题)
4. "追问'有什么意义/体现什么担当',先分清对象:对中国、对合作方、对世界秩序或全球治理分别作答。"(对应"意义/担当"题)
5. "写作类题要先确定中心论点,再把术语变成论证段,避免把选必一词语堆成清单。"(对应"短评/小论文"题)

这五句的内容不错,但所有同类型题共用同一句话,等于学生背完这五句后,任何具体题里都看不到"这道题的设问到底在追问什么"。

**改法**:`设问触发` 字段拆成两层:
- **题型通用句**(就是上面五句之一,简短保留);
- **本题独有触发**(一行,具体到这道题的关键动词或对象——例如通州 Q20 的"指引方向""彰显担当"、朝阳一模 Q20 的"中国为什么能"、海淀二模 Q21 的"中国需要联合国/联合国也需要中国"双向)。

学生通过通用句知道题型,通过独有触发知道该怎么落到本题。

需要本地证据核验:无,纯结构调整。

### MFT-6. `答案句变体` 实际是"模板答案",但命名让学生误以为是"备选写法"

`答案句变体` 这个字段名容易让学生理解为"这条得分点有好几种说法,任选一种"。但本稿里大多数情况下,该字段只给一个句子,没有真正的"变体"。

少数有多句的(如经济全球化"开放、包容、普惠、平衡、共赢方向发展"在朝阳一模/顺义一模合并条目里给出共用变体),也没有把不同语境标出来,学生看到两个并列变体不知道该套哪个。

**改法**:字段名改为 `**答题卡示范句**`(单题)或 `**答题卡示范句(分语境)**`(多题合并条目)。多题合并的条目,变体下用括号标出对应题号,例如:

```
**答题卡示范句**:
- (朝阳一模 Q20 / 全球发展语境):中国通过……,推动经济全球化朝着更加开放、包容、普惠、平衡、共赢的方向发展。
- (顺义一模 Q20 / 南南合作语境):科技小院推动……,让发展成果更普惠、更平衡、更共赢,体现经济全球化方向。
```

需要本地证据核验:多题合并条目下的语境分配是否与本地细则一致。

## should_fix_transfer(影响迁移性的中等问题)

### SFT-1. 同一核心点下条目过多,缺少"主条目 + 衍生"的层级

最典型:`经济全球化 -> 建设开放型世界经济,参与全球经济治理和规则制定` 这一核心点下挂了 12 个条目(从东城一模到门头沟一模),每个条目下又各自有 `答题点自身积累`。学生从第一条读到第十二条会发现这十二条说的其实是三件事:
- 制度型开放(对接国际规则);
- 利用国际组织权利(企业出海/反补贴);
- 提升发展中国家话语权。

但本稿没有任何子层结构把这三件事区分,十二条平铺。

**改法**:核心点下若条目超过 5 条,应在核心点行下加一段 `**子分类(适用情境)**`,列出 2-3 个子情境(每个对应若干题),让学生扫一眼就知道这个核心点在不同题里走的是哪条具体路径。

需要本地证据核验:子情境的划分是否切合本地细则的得分点分组。

### SFT-2. "新型国际关系"的核心名在条目里被反复改写,违反"核心名不压扁"硬规则

核心标题写的是"推动建设相互尊重、公平正义、合作共赢的新型国际关系",但条目里出现:
- "推动建设合作共赢的新型国际关系"(朝阳一模 Q20、顺义一模 Q20)——丢了"相互尊重、公平正义";
- "推进构建新型国际关系(国际关系民主化)"(东城一模 Q16)——把不同核心点压在一起;
- "推动建设以合作共赢为核心的新型国际关系"(朝阳二模 Q21)。

`搭配提醒` 行已经写了"本核心自身要保留'相互尊重、公平正义、合作共赢'",但条目正文不遵守。

**改法**:每个条目的 `**答题卡示范句**` 里至少出现一次完整术语原话"相互尊重、公平正义、合作共赢的新型国际关系"。简称只能出现在 `答题点自身积累` 的"承接短语"子层。

需要本地证据核验:本地细则中"合作共赢的新型国际关系"是否构成独立得分点(若是,需要在搭配提醒里说明何时可用简称)。

### SFT-3. "共商共建共享"核心点下混入了其他角度,搭配提醒不够强

`共商共建共享的全球治理观` 这个核心点下挂着这样一条:

> "传播中华文化、增强文明互鉴,体现正确义利观、共商共建共享原则和负责任大国形象"

这一条同时命中正确义利观、共商共建共享、负责任大国三个核心,但被整体归到"共商共建共享"桶下。学生看到会以为"传播中华文化"这件事的主要落点是共商共建共享,而不是义利观或大国形象。

**改法**:同一条得分点同时命中多个核心时,应在主归核心下出现一次,在其他相关核心下用 `**交叉引用**` 字段(只一行,标"本条同时见于 X 核心")提示,而不是把整条复制过去,也不是隐藏归属。

需要本地证据核验:本地细则该条得分点的"主要落点"。

### SFT-4. 通州期末 Q20 的"三层结构"在本稿里被打散,学生看不到原题的层

通州 Q20 完整设问明确分三层:正逢其时、指引方向、彰显担当。这是这道题最值得学生学的"设问自带分层信号"的范例。

但在本稿里,通州 Q20 的得分点散在六个核心点下(时代背景/和平与发展、政治多极化/共商共建共享、政治多极化/国际新秩序、中国/中国智慧中国方案、中国/正确义利观、联合国/宪章宗旨原则)。学生从六个不同位置看到通州 Q20 的局部条目,但永远拼不回"正逢其时(时代背景层)+ 指引方向(秩序层)+ 彰显担当(中国担当层)"这个三层骨架。

**改法**:在每道横跨三个及以上桶的"分层显性题"末尾,补一份 `**整题三层骨架**` 视图(只在该题第一次出现的位置一次性给出),把三层各自命中哪些核心点列出。或者,在使用方法那一节后面,补一个"按题索引"附录,每道题一行,列出该题的层数 + 各层命中核心点 + 跳转锚点。

需要本地证据核验:每道分层显性题的"层数"和"各层得分点归属"。

### SFT-5. "搭配提醒"行只出现在三个核心点下,大多数核心点没有

全稿只有以下三处用了 `**搭配提醒**`:
- 维护国家利益并兼顾他国合理关切;
- 共商共建共享的全球治理观;
- 推动建设相互尊重、公平正义、合作共赢的新型国际关系。

但还有更多核心点同样需要搭配提醒,典型如:
- "推动构建人类命运共同体"(命中题最多,最容易被滥用);
- "贡献中国智慧、中国方案、中国力量"(也命中很多题,且语境分化大);
- "推动经济全球化朝着更加开放、包容、普惠、平衡、共赢方向发展"(五词的写满与否要明确)。

**改法**:命中 5 题及以上的核心点必须强制配 `搭配提醒`,内容包括"何时可单独使用 / 何时必须与其他核心点搭配 / 易混核心点辨析"。

需要本地证据核验:无,纯结构补充。

### SFT-6. 部分核心点的命名本身就是冗余堆词,违反"核心名不压扁也不堆叠"

举例:
- 中国桶下:"中国作为负责任大国积极参与全球治理并推动全球南方现代化"——这是把"负责任大国 + 全球治理 + 全球南方现代化"三件事拼成一个核心名;
- 中国桶下:"坚定维护本国利益,坚持互利共赢、和平谈判并推动贸易自由便利"——四件事拼成一个核心名;
- 中国桶下:"改善民生、促进就业和经济社会发展,提升当地民众获得感"——这是某道题的具体得分点表述,被升格为核心点;
- 联合国桶下:"联合国对中国的对外开放和现代化事业作出了独特贡献,成为中国开展多边合作的主要舞台"——同样是某题专属的具体得分点表述被升格为核心。

核心点应该是高度浓缩的术语单元,不应是某道题的整句答案。本稿这种"题目专属答案 = 一个核心点"的处理,会让学生在六桶速览里看到一堆又长又具体的词条,失去六桶作为索引的简洁性。

**改法**:核心名长度建议控制在 20 字以内,超出的应拆成"核心名(短) + 表述积累(长)"两层。例如"改善民生、促进就业和经济社会发展……"应拆为:
- 核心名:`改善发展中国家民生`
- 表述积累:"促进就业和经济社会发展、提升当地民众获得感、中老铁路、安居村、200 口井、中国园林"

需要本地证据核验:核心名拆分后是否仍能覆盖本地细则的所有得分点。

## best_parts_to_keep(下一轮不要动的优点)

**B1. 六桶骨架在全稿严格执行**——时代背景、理论、经济全球化、政治多极化、中国、联合国六桶的边界在大多数核心点上是清楚的(虽然有 SFT-3 类交叉问题,但骨架本身没塌)。这是这一稿最大的资产。

**B2. `完整设问` 字段保留题号 + 设问原文**——学生能看到原题文字。这一点比之前预览版只列题号更友好。

**B3. 同题在多核心点下出现时,允许 `完整设问` 行用分号串联多题号 + 多设问原文**——例如朝阳一模 Q20 + 顺义一模 Q20 在多个核心点下用同一条目合并出现,设问行同时给两题原文。这种合并写法保留了"两题在该核心点上得分共享"的信息,优于强行拆成两条。

**B4. 已经把"五词不压扁""人类命运共同体不滥用""共同利益与正确义利观不合并"几个关键合并纪律明文写出**——`搭配提醒` 字段虽然只出现在三处,但出现的三处写得到位。

**B5. `慎用与排除` 那一节**——明确列出 2026 石景山期末不纳入、2026 丰台期末 Q20 只有题面暂不进入、混合模块题只收选必一直接给分部分。这种"什么不收"比"什么收"更稀缺,要保留。

**B6. 朝阳一模 Q20、顺义一模 Q20、通州期末 Q20 这三道高密度跨桶题各自的桶覆盖完整**——这三道题在本稿里几乎在每个相关桶下都出现,信息覆盖度高,只要解决 SFT-4 的"层结构"问题,就是这一稿最适合作学生范题的三道。
