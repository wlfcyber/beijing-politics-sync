# 给 Codex 的教学文本建议(Phase 01 / batch01 学生化方向)

声明:以下全部是教学结构与可读性建议,不替本地证据线裁决任何事实。所有"是否成立"的结论须由 Codex 据本地文件复核。

---

## 1. 首批 section_batch 的排版:六桶视图与按题视图并存

**结论:同一份讲义里挂两套目录,共用同一组核心点条目,不重复正文。**

具体排法建议如下:

**A. 主体正文按"题"排(按题视图为正文)**
- 每道题独占一节(`§ 2026朝阳一模 Q20`、`§ 2026朝阳期中 Q17` …)。
- 一道题的内部顺序固定为四块:`完整设问 → 命中核心点清单(按六桶分组列出)→ 各核心点详细条目 → 整题汇总卷面答案`。
- 完整设问一道题只出现一次,放在该题节首;后面引用同一题时只写题号,不再粘原文。
- "整题汇总卷面答案"是把这道题命中的所有核心点串成的一段考生作答样例,这是学生练答题卡时真正会照着背的目标形态。

**B. 六桶视图作为索引,不是正文复制**
- 在讲义前部或附录,做六张桶索引页:`时代背景索引`、`理论索引`、`经济全球化索引`、`政治多极化索引`、`中国索引`、`联合国索引`。
- 每张索引页只列"该桶下的核心点 + 命中题号 + 跳转锚点",不重复粘正文。
- 学生背术语时走桶索引,练整题时走按题正文,两条路径指向的是同一份核心点条目。

**为什么要这样:**学生考场面对的是题不是桶。把正文按题排,他读完一节就能直接练一道题;六桶作为索引,保留你"按桶背术语"的复习路径,但避免同一个核心点在六桶和按题里被复制两遍、学生看到第三次就开始跳读的问题。

需要本地证据核验:每个核心点在六桶下的归属是否唯一(同一核心点是否会在两个桶里都被列出)。

## 2. 每个核心点的呈现模板:"材料触发 → 为什么想到 → 答案句"

batch01 草案现在的结构是 `材料触发 → 答案句`,中间缺一步"为什么从这条材料能想到这个核心点"。这一步是迁移性的关键——学生没有这一步,换一道题他就不会触发同一个术语。

**建议统一模板(每个核心点条目固定四行):**

```
【核心点】<核心点全称,不压扁>
【完整设问】首次出现时给原文;再次出现时写"见 §<题号>"
【细则位置】<分值 / 必答或任答 / 简短引用>     ← 由 Codex 据本地文件填
【来源】<题号>

【材料触发】(只写材料里的事实,不带分析口吻)
  - 材料关键句/关键事实:"……"
  - 材料另一关键句/关键事实:"……"

【为什么想到这个核心点】(把材料事实和核心点之间的"桥"显式写出来)
  - 看到材料里的 X,联想到课本里 Y 这个判断
  - 设问问的是"为什么/如何/怎样",决定了要从 <能力来源 / 做法 / 方向> 哪个层面取核心点

【答案句】(单条核心点对应的卷面句,理论先行 + 材料对接 + 回扣设问)
  - 理论句:<不变的术语原话>
  - 材料对接:<把材料里的具体做法接回术语>
  - 回扣设问:<最后一小句把答案勾回题目问的那个动词>

【表述积累】(同核心下的可替换说法、并列得分点、变式)
  - 如"中国做法符合和平与发展时代主题"、"契合和平与发展的世界大势"等
```

**两点强调:**
- "为什么想到"这一行,batch01 当前是用"触发 X"这种作者旁白,学生看完知道你怎么想的、自己换题还是不会想。改成"看到 X,联想到 Y"的明示桥,学生才能内化。
- "答案句"必须是三段式(理论句/材料对接/回扣设问)分行写,不要压成一长句。学生抄三行能看懂结构,抄一长句只能死记。

需要本地证据核验:`细则位置` 行的分值、必答/任答、引用文字。

## 3. 同类项合并:核心名不压扁,变式进表述积累

合并的总原则:**一个核心点 = 一行不可改的术语原话 + 一组可替换的表述变式。** 核心名永远是课本/细则上的全称,变式收进 `表述积累`。

下面四个高频族,按这个原则各自怎么处理:

**族 A:"和平与发展仍是时代主题"**
- 核心点(唯一):**和平与发展仍是时代主题**
- 表述积累(变式,不另起核心):
  - "中国(的做法)符合和平与发展时代主题"
  - "顺应和平与发展的世界大势"
  - "契合和平与发展的时代主题,直面治理赤字……"
- 学生用法说明:核心句一律先写"和平与发展仍是时代主题",再用变式去对接材料(中国做法/南南合作做法/倡议做法)。

**族 B:经济全球化方向"开放、包容、普惠、平衡、共赢"**
- 核心点(唯一,不可压扁为"正确方向"):**推动经济全球化朝着更加开放、包容、普惠、平衡、共赢的方向发展**
- 表述积累:
  - "经济全球化方向:普惠、平衡、共赢"(命题人侧重子集时的写法)
  - "让发展成果更普惠、更平衡、更共赢"(对接材料时的口语化承接)
- 强调:写答案句时,五个形容词原则上全列;只有命题材料明确侧重三项时,才用子集变式,且要在表述积累里标注"侧重式写法"。

**族 C:新型国际关系**
- 核心点(唯一):**推动建设(以)合作共赢(为核心)的新型国际关系**
- 表述积累:
  - "合作共赢"
  - "互利共赢"
  - "共享发展成果"
- 强调:这三个变式不是三个核心点。合并时核心名保留"新型国际关系"四个字,三个变式作为对接材料的承接词使用。batch01 目前把"新型国际关系:合作共赢/互利共赢/共享发展成果"写成一个并列长术语,合并时要把"新型国际关系"提到核心位、其余三个降到表述积累。

**族 D:人类命运共同体**
- 核心点(唯一):**推动构建人类命运共同体**
- 表述积累:
  - "习近平外交思想的核心理念"
  - "外交思想:构建人类命运共同体"
  - "中国主动担当 / 大国担当"在材料对接里的承接句
- 强调:"外交思想"和"人类命运共同体"不是两个核心点。当题目是外交语境(如顺义 Q20 的中国方案/南南合作),用"外交思想"做引子,落点仍回到"推动构建人类命运共同体"。

**统一的合并动作清单(给 Codex):**
1. 在每个核心点条目顶端只放一行术语全称,作为核心名。
2. 把所有同义/近义/侧重子集变式收进 `表述积累`,逐条标注"何时用"。
3. 不允许出现"核心点 A / 核心点 B"这种斜杠并列做核心名;斜杠后的内容要么拆成独立核心,要么降到表述积累。
4. 合并后,核心点的"出现次数"统计要按合并后口径重算(原来标 `出现1次` 的若干条,合并后可能是 `出现3次`)。

需要本地证据核验:每一族变式在本地细则里到底是同核心(应合并)还是分项给分(必须保留为独立核心)。这一步是合并能不能动手的前提。

## 4. 零基础学生最容易误解的表达,改成高三能背会用的话

下面这些表达在 batch01 草案中反复出现,学生读起来要么不知道在说什么,要么知道但不会用:

**(1)"触发 X / 对应触发 Y"**
- 误解点:学生以为"触发"是一个术语,会问"老师触发是哪一课的"。
- 改法:换成"看到材料里的 ……,就要想到课本里 ……"。"想到"是学生自己也会用的词,"触发"是命题分析者的词。

**(2)"实质是"、"指向"、"契合"**
- 误解点:这三个词学生不会主动用,看到时也分不清差别。
- 改法:在条目内部分工——"实质是"只用在"国际竞争的实质是综合国力较量"这种课本原句里,其他位置一律换成"说明""体现""属于"。"指向"在学生作答里改成"说的就是 / 落点在"。"契合"作为承接词学生可以用,但条目说明里要给一个示范句:"中国做法契合 X = 中国做法对应的就是 X"。

**(3)"两个市场、两种资源"、"双循环"、"高水平对外开放"**
- 误解点:这是术语,学生背得下来,但不知道写在卷面哪一句里。
- 改法:在 `表述积累` 里给一个固定承接句模板,例如:"中国通过 <材料里的具体做法>,联通国内国际两个市场两种资源,服务双循环新发展格局。"——这是一个填空式句子,学生考场套着写。

**(4)"百年变局"、"治理赤字"、"逆全球化抬头"**
- 误解点:学生认得字,但不知道这三个词组合起来是在说"现在世界很乱"。
- 改法:第一次出现时,在条目里附一行"白话注":百年变局 = 国际格局正在大变;治理赤字 = 该有的治理没跟上;逆全球化抬头 = 有国家在搞贸易保护、退群。这一行只在第一次出现的条目里给,后续不重复。

**(5)"共商共建共享"**
- 误解点:学生会背六个字,但不会用动词。
- 改法:在 `表述积累` 里给三个动词版示范——"共同讨论 / 共同建设 / 共同分享",写卷面时三个动词替换原词,句子才像"考生写的话"。

**(6)"正确义利观"**
- 误解点:学生不知道"义"和"利"具体指什么,容易写成"中国讲义气"。
- 改法:在第一次出现的条目里加一行白话:"正确义利观 = 在国际合作中既讲共同利益、又重国际道义,优先帮助发展中国家。"

**(7)条目里的"前者触发 / 后者触发"句式**
- 误解点:这是命题分析者在拆题,不是学生在答题。
- 改法:`材料触发` 段拆成 `材料事实 1 / 材料事实 2` 两条点列,`为什么想到` 段单独写"事实 1 → 核心点 A / 事实 2 → 核心点 B"。学生看到的是因果对照,不是论文式分析。

需要本地证据核验:上述白话注涉及的术语含义边界(例如"正确义利观"在本课本表述中的官方定义),由 Codex 据本地教材/细则核对后再定稿。

## 5. batch01 六个样本的"学生化排序建议"

排序的依据是教学梯度:**结构最清晰、桶覆盖最少、设问最直白的题先讲;桶覆盖最广、设问最绕的题最后讲。**

不替 Codex 判断分值/证据强度,只从学生学习曲线给出顺序:

**第 1 题:2026朝阳期中 Q17(三层式范题)**
- 理由:这道题的得分结构本身就是显性三层(自力更生与对外开放 / 发展与安全 / 中国与世界发展),材料里三个同学甲乙丙各自对应一层,学生第一次看就能学到"题目自带分层信号 → 答案就分层写"。这是教"按层组织答案"的最佳起点题。

**第 2 题:2025海淀期中 Q16(2)(单点深练)**
- 理由:命中的得分点窄、设问短(给中国咖啡企业出海提建议),学生练完上一题学会分层之后,这道题用来训练"单点也要写完整三段式"——理论 + 材料对接 + 回扣设问,练句子不练结构。

**第 3 题:2026顺义一模 Q20(理论 → 材料 → 结论 模板题)**
- 理由:科技小院的材料具体可感(种地、培养人才、降成本),学生容易代入;同时这道题命中"共同利益—经济全球化方向—新型国际关系—人类命运共同体"这条主线,是练"理论先行,材料对接"的标准模板题。

**第 4 题:2026通州期末 Q20(分句对应分点训练)**
- 理由:设问三段(正逢其时 / 指引方向 / 彰显担当)和答案三段一一对应,学生在这里学的是"设问里有几个动词短语,答案就要写几层"。比朝阳 Q17 的层数多一点(三层 + 各层并列点),适合中段练。

**第 5 题:2026朝阳一模 Q20(跨桶组合的高难度整合题)**
- 理由:这道题的得分点横跨五个桶,学生需要先掌握前四题才有能力做"跨桶组合"。放在第五位,正好用来检验前面学到的"按题重组"是否到位。

**第 6 题:2025海淀期中 Q21(2)、2024东城一模 Q16 / Q20**
- 理由:这三道(或两道,取决于 Codex 在 batch01 里实际入选哪些)我没有在用户给的草案文档里看到正文,只在 batch 名单里出现。建议放在最后讲——既因为它们在草案里成熟度最低,也因为前五题打牢之后再回头看这几道,学生会用前面学到的模板自己拆。

**两点附加说明:**
- 这个顺序只是"教学讲解顺序",不等于"在讲义里的物理顺序"。讲义可以仍按时间倒序或按桶,但课堂展开顺序按上面排。
- 跨桶题(朝阳一模 Q20)无论如何都要放在分层题(朝阳期中 Q17)之后,这一相对顺序不要颠倒。

需要本地证据核验:batch01 实际入选的 6 道题是否就是上面列出的这几道,以及每道题在本地草案里的成熟度等级。

## 6. 给 Codex 的返修清单

每一项都标注"需要本地证据核验",不替 Codex 裁决事实。

**【结构类】**

1. 把正文从"按桶排"改为"按题排",六桶降为索引页;同一核心点条目只写一次,通过锚点在两个视图里复用。需要本地证据核验:核心点在多桶之间是否真的唯一归属。

2. 每道题节末补 `整题汇总卷面答案` 块,把该题命中的所有核心点的"答案句"按设问层次串成一段考生作答样例。需要本地证据核验:汇总后的字数是否在该题分值对应的合理作答字数区间内。

3. 每个核心点条目按 `核心点 / 完整设问 / 细则位置 / 来源 / 材料触发 / 为什么想到 / 答案句 / 表述积累` 八字段固定排版;`完整设问` 同题再次出现时只引用题号,不复制原文。需要本地证据核验:`细则位置` 字段的分值与必答/任答标记。

**【合并类】**

4. "和平与发展仍是时代主题" 合并为单一核心点;"中国做法符合……""顺应世界大势"等收入 `表述积累`。需要本地证据核验:本地细则中这些变式是否同分项还是分项独立给分。

5. "推动经济全球化朝着更加开放、包容、普惠、平衡、共赢的方向发展" 保留全称为核心名,不得简称为"经济全球化正确方向";子集变式("普惠/平衡/共赢")降为表述积累。需要本地证据核验:细则中是否存在以子集形式独立给分的情况。

6. "推动建设合作共赢的新型国际关系" 保留全称为核心名,"合作共赢/互利共赢/共享发展成果"降为表述积累。需要本地证据核验:三个变式在细则中是否分项给分。

7. "推动构建人类命运共同体" 保留全称为核心名,"外交思想:构建人类命运共同体""习近平外交思想的核心理念"降为表述积累。需要本地证据核验:外交思想表述在细则中是否构成独立得分点。

8. 合并完成后,所有核心点的"出现次数"按合并后口径重算,并删除目前每个术语后面 `(出现1次)` 的括注;若保留考频,改为"高/中/低频"或"近 N 年命中 X 次"。需要本地证据核验:合并后口径下的实际命中频次。

**【表达类】**

9. 全文将"触发 X""对应触发 Y"句式替换为"看到材料里的……,就要想到……"。需要本地证据核验:无,纯文字风格。

10. 在术语首次出现的条目里附一行"白话注",覆盖至少:百年变局、治理赤字、逆全球化抬头、共商共建共享、正确义利观、两个市场两种资源、双循环。需要本地证据核验:白话注的措辞是否与本地教材的官方定义一致。

11. 每条 `答案句` 改为三行式:理论句 / 材料对接 / 回扣设问,各占一行,不压缩成单句。需要本地证据核验:无,纯排版。

**【清理类】**

12. 学生版正文中删除所有路径、PPTX 页号、DOCX 段号、worker batch 编号、`draft/not_promoted` 标识、`(出现1次)` 频次括注、模型对话语气("触发""即可看出""作者认为");页号段号挪到讲义末"出处索引"附录,仅供老师/复核者查。需要本地证据核验:挪入附录后,出处索引与本地源文件页码段号是否完全一致。

13. 学生版任何位置不得出现"Codex / Claude / GPT / commander / Phase / batch / audit"等后台制作语。需要本地证据核验:无,合规检查。

**【排序类】**

14. 教学讲解顺序按第 5 节给出的六题序展开;讲义物理排序可与之不同,但跨桶题(朝阳一模 Q20)必须排在分层题(朝阳期中 Q17)之后。需要本地证据核验:batch01 实际入选题与上述假设是否一致。

**【边界类】**

15. 学生版当前阶段标注为 `教学预览版 / 非终稿`,不进入学生终稿分发渠道,直至 commander 解除条件 GO 的前置(11 个高优先题证据锁定 + 术语合并完成 + 学生迁移验证)。需要本地证据核验:三项前置条件的当前完成度。

---

以上为教学结构与可读性方向的建议,所有涉及"是否成立""分值多少""变式是否同分项"的判断,均待 Codex 据本地文件复核后定稿。
