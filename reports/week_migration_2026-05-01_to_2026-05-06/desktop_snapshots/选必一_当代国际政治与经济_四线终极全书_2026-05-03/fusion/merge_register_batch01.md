# Batch 01 Merge Register

Status: `draft_not_promoted`

Purpose: keep same-core merge decisions explicit before any student-facing section or frequency count is generated.

## Locked Merge Principles

1. Same scoring core must merge before expression accumulation.
2. The merged core name must preserve the highest-information scoring wording.
3. Same literal words do not guarantee merge; scoring function and material trigger must match.
4. Different evidence levels can live under one core only if each source keeps its evidence label.
5. Student-facing final cannot show evidence labels or paths, but the teacher/audit layer must preserve them.
