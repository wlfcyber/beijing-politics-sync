# Batch04C 东城同类项合并登记

## 新增合并方向

- `全球倡议系统推动构建人类命运共同体`
  - 来源：2026东城期末 Q20 四大全球倡议；2024东城二模 Q20 三大倡议。
  - 保留差异：四大全球倡议是发展、安全、文明、治理四维，并有4+2+1结构；三大倡议是发展、安全、文明三项，每项路径1分+效果1分，总论1分。
  - 处理：合并到同一父核心下做表述积累，不能拆成“四大全球倡议”和“三大倡议”两个孤立频次点。
  - 四大全球倡议子槽：全球发展倡议以发展促繁荣；全球安全倡议以安全保稳定；全球文明倡议以文明增互信；全球治理倡议以治理求公正；四大全球倡议相互促进、有机统一。
  - 三大倡议子槽：全球发展倡议通过合作援助开辟发展合作新空间、提供新机遇、注入新动力、增强发展中国家内生动力、繁荣世界经济；全球安全倡议通过对话协商和平解决国际争端、遵守联合国宪章宗旨和原则、以共赢思维应对安全挑战、推动持久和平普遍安全；全球文明倡议通过交流互鉴、传承创新、民心相通，建设开放包容的世界。
  - 频次边界：同一父核心下区分总目标、倡议分项、系统推动、实践层和现实意义；不得把所有含“人类命运共同体”的句子合成一个空泛频次。

- `同球共济 / 人类命运共同体 / 共同利益`
  - 来源：2025东城二模 Q20。
  - 处理：`同球共济`作为材料表述；核心仍是人类命运共同体和共同利益。共同利益不能泛化为所有国际合作题的万能帽。

- `推动经济全球化朝着更加开放、包容、普惠、平衡、共赢方向发展`
  - 来源：2025东城一模 Q20。
  - 处理：并入完整五词核心，保留中拉合作、自贸协定升级、共建“一带一路”等题型用法。

- `贸易保护主义 / 逆全球化`
  - 来源：2025东城期末 Q20。
  - 处理：并入贸易摩擦和全球化受阻核心；与西城汽车贸易壁垒题同类，但本题侧重短文认识类。

## 边界

- `2026东城一模 Q19(3)`：仅边界观察。制度型开放、国际高标准经贸规则、国际科创合作可记录；未来产业、高水平对外开放、产业链韧性、高质量发展不直接计入选必一主链。
