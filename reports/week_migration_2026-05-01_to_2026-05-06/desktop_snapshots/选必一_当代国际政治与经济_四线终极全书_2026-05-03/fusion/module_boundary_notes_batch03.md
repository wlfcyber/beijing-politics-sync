# Batch 03 Module Boundary Notes

Scope: 2026西城期末 Q20.

## Boundary Decisions

1. `坚持绿色发展 / 新发展理念，有为政府 + 有效市场`
   - Evidence: P0 formal scoring PDF visual.
   - Use: keep in teacher evidence and the question-specific answer structure.
   - Boundary: not a reusable 选必一 core term; do not put it in the six-bucket student core as a high-frequency term.

2. `促进全球可持续发展 / 建设清洁世界`
   - Evidence: P0 formal scoring PDF visual.
   - Use: result language for China's climate-governance practice.
   - Boundary: not an independent high-frequency core unless later P0 rubrics repeat it as a reusable scoring core.

3. `命运共同体 / 全人类共同价值 / 正确义利观 / 共商共建共享 / 践行真正的多边主义 / 坚持互利共赢`
   - Evidence: one P0 same-slot scoring item.
   - Main bucket: `中国`.
   - Cross-reference: `政治多极化 / 联合国`.
   - Boundary: count once; do not split into six frequencies.

4. `维护联合国的核心作用 / 完善全球治理体系`
   - Evidence: P0 formal scoring PDF visual.
   - Main bucket: `联合国`.
   - Boundary: do not generalize to all international organizations or generic multilateralism.

## Student-Document Rule

If this question enters a student-facing section, use an按题三段 structure:

- 中国实践是什么：建设者/引领者/负责任大国 + 绿色低碳实践.
- 为什么参与：时代潮流、共同利益、中国理念、国际义务中按设问和材料取舍.
- 实践效果：可持续发展/清洁世界、联合国核心作用/全球治理体系、中国智慧/中国力量.

Do not expose P0/P3, visual recheck, PDF pages, source ledger, worker, fusion, Governor, 4选3, or frequency-control language in student prose.
