# 2026 北京高考政治 选必二《法律与生活》试题切片 结构化摘要

## 一、主观题部分

### 【1】2026 东城期末 Q18（11分） —— 合同+侵权竞合
- **案情材料**：陈某与刘某签订合同，刘某交付的产品存在瑕疵，造成陈某人身伤害与财产损失，双方就赔偿和处理方式产生争议。
- **设问**：(1) 分析合同效力与责任承担；(2) 提出维权建议（固定证据、明确诉求、争议解决方式）。
- **法律领域**：合同（违约）+ 侵权责任 + 违约责任竞合 + 纠纷多元解决。
- **答案踩分点**：
  1. "主体适格、意思表示真实、形式与内容合法 → 合同有效"
  2. "未全面履行合同 → 构成违约 → 承担违约责任"
  3. "产品瑕疵造成人身/财产损害 → 构成侵权 → 承担侵权责任"
  4. "固定证据、明确诉求、争议解决方式"三段
  5. "以事实为根据，以法律为准绳"
- **命题路径反推**：
  1. 训练"违约/侵权竞合"判别——这是《法律与生活》最经典的二元归因。
  2. 选购物纠纷做载体，能同时挂"合同有效要件"+"侵权四要件"两条线，逼学生分别检索。
  3. 设问"分析责任 + 维权路径"两段式，先实体后程序，模仿真实律师工作流。
  4. 给分点奖励"判别动作"——必须明确写"构成 X 责任"的判定句，而非泛罗法条。
  5. 第二问把"权利意识"落到操作层（取证→诉求→选途径），呼应教材"依法维权"主线。

### 【2】2026 朝阳期末 Q17(1)（7分） —— 网络消费 / "假一赔十"承诺
- **案情材料**：直播间将大叶紫檀手串宣称为小叶紫檀，承诺"保真、假一赔十"。侯某购买后发现材质不符，诉至法院要求十倍赔偿，法院支持。
- **设问**：说明法院判决的依据。
- **法律领域**：合同（要约/承诺/约定优先）+ 消费者权益 + 经营者义务。
- **答案踩分点**：
  1. "假一赔十"承诺构成买卖合同的内容，对张某具有约束力
  2. 张某交付的货物不符合合同约定（材质偏差影响价值）
  3. 约定赔偿标准虽高于法定，但应当履行——意思自治原则
  4. 经营者应履行如实告知义务，不得欺诈
  5. 消费者享有公平交易权与知情权
- **命题路径反推**：
  1. 命题人要分离"法定赔偿"与"约定赔偿"两个层面——很多学生只会条件反射套"假一赔三"。
  2. 选直播带货载体，紧扣 2025-2026 年监管热点，让学生在新业态中识别老制度。
  3. "判决依据"是法理路径还原，奖励"合同约束力"+"约定优先"两个判别动作。
  4. 给分点引导学生区分"承诺即约束"和"消费者倾斜保护"两条理由，避免单线作答。

### 【3】2026 海淀一模 Q18(3)（7分） —— 网络消费两案对比
- **案情材料**：
  - 案件1：商家被差评后在社交平台公开顾客的包厢监控录像、聊天记录，顾客起诉。
  - 案件2：经营者宣称理疗产品可"治疗糖尿病并发症"，孙某购买无效后诉请退款+三倍惩罚性赔偿，法院支持。
- **设问**：填表写出案件1裁判结果、案件1裁判理由、案件2裁判理由。
- **法律领域**：人格权（隐私 / 个人信息）+ 消费者欺诈 + 惩罚性赔偿。
- **答案踩分点**：
  1. 案件1判决：商家侵权成立，停止侵害、赔礼道歉、赔偿损失（含精神损害）
  2. 案件1理由：监控录像、聊天信息属个人隐私/个人信息；公开传播侵害隐私权与个人信息权益；经营者负有保护消费者个人信息的法定义务
  3. 案件2理由：经营者作虚假宣传构成欺诈；消费者权益保护法规定欺诈情形下三倍惩罚性赔偿
  4. 强化"经营者义务"对应"消费者权利"两端
- **命题路径反推**：
  1. 用"两案对照表"逼学生在同一情境下分别调用"人格权侵权"和"消费者权益"两条知识链。
  2. 案件1难点是识别"差评-报复"中商家的角色错位——经营者反过来侵权，打破学生定式思维。
  3. 案件2考"惩罚性赔偿适用条件"——必须有"欺诈"认定才能三倍。
  4. 表格化作答训练司法话语习惯，奖励"裁判结果+理由"分写规范。

### 【4】2026 朝阳一模 Q18 —— 知识产权 / 司法守护"向新力"
- **案情材料**：法院针对专利权归属纠纷邀第三方调停促成和解；对故意侵害知识产权适用惩罚性赔偿；对恶意诉讼快速审结并予否定。
- **设问**：说明法院如何以司法之力守护"向新力"（措施类）。
- **法律领域**：知识产权（专利权）+ 侵权责任（惩罚性赔偿）+ 纠纷多元解决（调解/诉讼）+ 司法公正。
- **答案踩分点**：
  1. 调解优先，促进合作共赢——多元解决减少诉讼对创新的干扰
  2. 严格保护，加大惩戒力度——惩罚性赔偿大幅提高侵权违法成本
  3. 规制滥用，维护创新环境——依法规制恶意诉讼、权利滥用、不诚信诉讼
  4. 公正司法、以事实为依据法律为准绳
  5. 保护知识产权就是保护创新
- **命题路径反推**：
  1. 训练"措施类"主观题三段式：行为+依据+效果。
  2. 三案分别对应"调解-惩罚-规制"，逼学生跨制度调取知识。
  3. 暗扣"新质生产力"国策，让法律与经济发展叙事对接。
  4. 给分点要求"调解优先 / 惩罚性赔偿 / 规制滥用"三条都要踩，训练全景思维。

### 【5】2026 东城一模 Q18（8分） —— 知识产权三案 / 司法守护"向新力"【含命题给分硬卡口】
- **案情材料**（与朝阳一模 Q18 同型同向）：
  - 案例一：科技公司三创始人因专利权归属纠纷长期诉讼，合议庭居中斡旋促成和解。
  - 案例二：乙公司擅自将甲公司植物新品种稻种换包装销售，法院判停止侵害+惩罚性赔偿 300 万。
  - 案例三：丙公司为打击上市期的丁公司以专利侵权为由恶意诉讼且诉求反复，法院驳回并谴责。
- **设问**：法院是如何以司法之力守护"向新力"的？
- **法律领域**：知产侵权 + 惩罚性赔偿 + 调解 + 滥用诉权规制 + 司法公正。
- **得分硬卡口**（命题给分点原句）：
  - 案例一：必须出现"诉讼调解"（替代"调解"，1分）+ "推动纠纷实质性化解 / 以和为贵 / 和谐价值观 / 法德结合 / 效率高 / 节约司法资源"任一（1分）
  - 案例二：必须写"惩罚性赔偿 / 严惩（主观恶性强、社会危害大）知识产权侵权"+"增加侵权成本"（1分）；落点写"警示规范作用 / 保护当事人合法权益 / 市场秩序 / 诚信原则 / 公平竞争 / 公平正义"任一（1分）
  - 案例二硬卡口：**写"搭便车""不正当竞争""消费者知情权"不给分**（题意是侵权，不是混淆行为）
  - 案例三：必须写"驳回并谴责恶意诉讼 / 法院惩治滥用程序权利的行为"（1分）+"诚信原则 / 权利行使界限 / 权利义务统一 / 维护诉讼司法秩序 / 节约司法资源"任一（1分；与案例一不重复给）
  - 案例三硬卡口：**写"商业诋毁""不正当竞争"不给分**
  - 总说必踩：法院维度——"公正司法 / 以事实为依据，以法律为准绳 / 司法裁判导向作用"（1分）；创新维度——"保护知识产权就是保护创新 / 提升创新活力"（1分）
  - 满分结构：3 段案例分析（每段 2 分）+ 2 句总说（2 分），共 8 分
- **命题路径反推**：
  1. 用"调解 / 严惩 / 驳回滥诉"三案排布，训练学生在不同司法工具间精准匹配——一案一对应不能错位。
  2. 案例二专门埋"换包装销售"陷阱，引诱学生答"不正当竞争 / 搭便车"，给分点直接堵死，奖励准确识别"侵害植物新品种权"。
  3. 案例三埋"上市关键期+反复变更诉求"两个信号词，训练学生从程序滥用而非实体侵权角度切入，给分点直接禁掉"商业诋毁"。
  4. 总说必踩"司法依据 + 创新意义"两根支柱，训练答题闭环。
  5. "与案例一不重复给"明文规则，意在防止同一短语反复套用骗分，逼学生差异化作答。

### 【6】2026 丰台一模 Q20（8分） —— 安全保障义务 / 餐厅外摔倒
- **案情材料**：郭某餐后边低头看手机边走台阶，自行摔倒受伤，无积水积雪等异常。郭某起诉餐厅与物业要求承担侵权责任，法院判驳回全部诉讼请求。
- **设问**：阐明法院作出该判决的法理依据和现实意义。
- **法律领域**：侵权责任（安全保障义务）+ 民法典第 1198 条 + 过错责任原则。
- **答案踩分点**：
  1. 经营者、管理者负有安全保障义务（法定义务的范围与界限）
  2. 侵权责任以过错为构成要件——本案被告无过错（无积水积雪冰冻、台阶安全无异常）
  3. 损害的发生主要因原告自身未尽注意义务（低头看手机），构成自我过错
  4. 民法典第 1198 条规定"未尽到安全保障义务"才担责——经营者义务有边界
  5. 现实意义：明确义务边界 / 引导个人对自身安全负责 / 防止"和稀泥"裁判 / 鼓励经营者正常营业
- **命题路径反推**：
  1. 纠正"摔倒就找经营者"的本能——训练"过错+因果关系"的精确判别。
  2. 给原文法条+案情+判决全套材料，让学生练"以法解案"的引证技巧。
  3. 双层设问："法理依据"考实体法适用，"现实意义"考价值导向，奖励有"边界意识"的答案。
  4. 给分点重视"自身过错"——这是 2026 年新现的反向判例，对应"个人责任回归"的司法风向。

### 【7】2026 石景山一模 Q18 —— 举证责任分配（行政诉讼 vs 民事诉讼）
- **案情材料**：
  - 案例二：原告就行政机关具体行政行为提起行政诉讼。
  - 案例三：张某车辆受损（窗户脱落），向卢某主张侵权赔偿。
- **设问**：分析两案的举证责任分配方式及理由（表格作答）。
- **法律领域**：行政诉讼举证（被告负担）+ 民事诉讼举证（"谁主张谁举证"+ 过错推定）。
- **答案踩分点**：
  1. 行政诉讼举证责任倒置：被告（行政机关）应提供作出该行政行为的证据和所依据的规范性文件
  2. 倒置目的：督促依法行政、保障行政相对人合法权益、体现权责统一
  3. 民事侵权"谁主张谁举证"——原告张某需举证损害事实、损害程度及因果关系
  4. 同时适用过错推定（建筑物悬挂物脱落类）——卢某需举证自己无过错或法定免责事由
  5. 制度功能：合理分配诉讼风险、平衡当事人地位
- **命题路径反推**：
  1. 用对照表强迫学生区分两类诉讼的举证规则——这是高频混淆点。
  2. 用"窗户脱落致车损"特定情境，命中"建筑物搁置物悬挂物脱落"侵权——既考"谁主张谁举证"，又考"过错推定"叠加。
  3. 给分点要求"原告举证什么 / 被告举证什么"分写到具体动作层，训练精细化作答。
  4. "举证责任倒置"作锚点，让学生一眼分辨行政与民事场景的程序差异。

### 【8】2026 西城一模 Q17（8分） —— 物权 + 绿色原则
- **案情材料**：民法典将"绿色原则"贯穿物权编（节约资源、保护环境义务），涉及不动产、用益物权、相邻关系。
- **设问**：阐述法条体现的法律精神/治理意义。
- **法律领域**：物权 + 用益物权 + 相邻关系 + 民法基本原则（绿色原则）。
- **答案踩分点**：
  1. 绿色原则贯穿物权行使全过程——民事主体享权同时必须履行节约资源保护环境义务
  2. 从源头禁止违法排污，实现个人权利行使与公共生态利益的平衡
  3. 法律合理界定不动产、用益物权、相邻关系等权利行使边界，协调多元利益关系
  4. 兼顾资源利用效率与生态环境保护，推动绿色生产方式转型
  5. 以良法夯实绿色发展制度根基，构建基层自治与行政监管协同发力的治理体系
- **命题路径反推**：
  1. 把"民法典绿色原则"和"物权制度"勾连，让学生把宏大叙事落到具体物权行使行为上。
  2. 设问指向"权利边界 + 多元利益协调"，奖励能用"权利-义务统一"框架展开的答案。
  3. 给分点强调"良法善治+协同治理+权利救济"三件套，训练制度功能完整链条的写作。

### 【9】2026 丰台期末 Q18（8分） —— 行政公益诉讼 / 未成年人观影优惠
- **案情材料**：检察院发现部分影院未在网络与现场购票场所明确未成年人优惠政策，且把"未成年人"窄化为身高 1.3 米/1.2 米以下。检察院依法对区文旅局立案行政公益诉讼并发检察建议；文旅局督促整改落实。
- **设问**：阐述该检察公益诉讼案例的积极意义。
- **法律领域**：行政公益诉讼 + 检察建议 + 未成年人保护法 + 依法行政。
- **答案踩分点**：
  1. 维护未成年人合法权益（特殊群体保护）
  2. 监督行政机关依法履职，推动严格执法
  3. 检察机关履行法律监督职能 → 维护国家利益和社会公共利益
  4. 推动法律实施落地，弥补一般诉讼的"原告缺位"
  5. 增强法治观念，引导经营者守法
- **命题路径反推**：
  1. 选"行政公益诉讼"（2026 年新增热点），让学生掌握"四大公益诉讼"中的行政分支。
  2. 用"身高代替年龄"的真实争议作切入，训练学生识别"行政不作为/违法作为"的微观表现。
  3. "积极意义"奖励多角度——保护特殊群体、监督依法行政、维护公益、推动治理。
  4. 突出"检察建议→整改"的非诉化解机制，弱化对抗性、强化协同性。

## 二、选择题部分

### 【1】2026 东城期末 Q10 —— 多法律点综合案例
- **题干情境**：表格列出 4 个生活案例，要求选出"分析正确"项。
- **4 个选项**：
  - A：李某用银行承兑汇票为借款设质押但未办理登记 → 称张某无优先受偿权 / 票据质押的设立与优先受偿
  - B：乙公司破译甲公司"门禁系统电路解码装置"非公知技术信息销售 → 称侵犯专利权 / 商业秘密 vs 专利权区分（非公知信息属商业秘密）
  - C：AI 软件强制升级捆绑安装 → 称侵犯知情权且违反自愿原则 / 捆绑安装的法律定性
  - D：王某将画作赠与张某后张某擅自展览汇编出版 → 称张某有权拒绝返还但需承担侵权责任 / 物权（所有权）vs 著作权（展览权、汇编权）的分离
- **法律细节**：商业秘密与专利权边界；著作权独立于物权（赠与转移所有权但著作权未转移）；票据质权设立要件；消费者自主选择权。

### 【2】2026 东城期末 Q11 —— 劳动合同 / 违法解除
- **题干情境**：小王入职未签书面劳动合同，上司口头承诺"灵活工作制"，公司后推行打卡，以"多次旷工"为由单方解雇。
- **4 个选项**：
  - A：未签书面合同 → 劳动关系未成立无须担责 / 错——事实劳动关系受保护
  - B：口头承诺有效，公司应给经济补偿 / 错——违法解除是赔偿金而非经济补偿
  - C：公司侵犯劳动者合法权益，构成违法解除劳动合同 / 正确
  - D：可向劳动保障监察部门投诉，提起行政复议 / 错——劳动争议是仲裁→诉讼，非行政复议
- **法律细节**：事实劳动关系；违法解除（赔偿金 = 经济补偿×2）；劳动争议处理途径（仲裁前置）；口头承诺举证。

### 【3】2026 房山一模 Q10、Q11
- **Q10 题干情境**：探店博主直播时赵某明确拒绝出镜后仍被清晰拍摄并实时传播。
- **Q10 选项**：
  - ①科技向善、自由不能逾越法律红线 / 正确
  - ②主播对画面享有著作权可公开发表 / 错——著作权不能对抗肖像权、隐私权
  - ③赵某有权要求删除画面、停止传播、赔礼道歉 / 正确（人格权请求权）
  - ④博主需对其没有过错承担举证责任 / 错——一般侵权"谁主张谁举证"
- **Q10 法律细节**：肖像权/隐私权/个人信息保护与著作权冲突；人格权请求权（停止侵害+消除影响+赔礼道歉）；举证责任一般规则。

- **Q11 题干情境**：李某被列入失信名单，法院试点个人债务集中清偿，为其保留生活必需的面包车与手机；按期还清后解除惩戒并出具《信用修复证明书》。
- **Q11 选项**：
  - A：失信名单是补救措施的违约责任承担方式 / 错——失信名单是执行措施
  - B：保留面包车手机是保护用益物权 / 错——保护的是基本生活必需（生存权）
  - C：李某遵守自愿原则依方案清偿 / 错——执行方案非纯自愿
  - D：法院创新司法实践，平衡债权人和债务人利益关系 / 正确
- **Q11 法律细节**：个人债务集中清偿试点（类个人破产）；失信被执行人制度；信用修复机制；执行豁免（生活必需品保留）。

### 【4】2026 朝阳一模 Q10 —— 财产权综合（抵押质押 + 著作权）
- **题干情境**：学生讨论"依法有效保护财产权"，4 项观点判正误。
- **4 个选项**：
  - ①房屋、汽车、电脑、股票、债权、股权均可抵押和质押 / 错——抵押 vs 质押客体边界（如股票质押而非抵押）
  - ②未经甲同意将其小说拍成科幻电影 → 侵犯改编权 / 正确（小说→电影属改编）
  - ③丙画廊未经丁许可将丁画作印在宣传页 → 侵犯复制权 / 正确
  - ④王某专利权质押银行，到期未还银行可占有使用专利 / 错——质权人不得擅自占有使用，需依法处置（拍卖/变卖/折价）
- **法律细节**：抵押 vs 质押客体；著作权各项财产权（复制、改编、发行、信息网络传播）；质权实现方式。

### 【5】2026 海淀一模 Q10、Q11
- **Q10 题干情境**：李某返乡创业经营行为分析。
- **Q10 选项**：
  - A：与王某签协议承租其承包地用于苹果种植 → 王某有权依法流转土地承包经营权 / 正确（"三权分置"）
  - B：无人机喷农药"误伤"邻田 → 因不可抗力可免赔 / 错——是过失致损非不可抗力
  - C：与收购商口头约定收购条款 → 未采取书面形式部分无效 / 错——买卖合同口头形式有效
  - D：以相似图案申请商标被驳回 → 可申请行政复议 / 错（应商标评审复审）
- **Q10 法律细节**：土地承包经营权流转（三权分置）；不可抗力 vs 一般过错；合同形式自由；商标驳回救济渠道。

- **Q11 题干情境**：姜某婚后创作小说获 20 万元版权费，妻子苏某承担主要家务，离婚时法院判 20 万均分并判姜某给苏某家务补偿。
- **Q11 选项**：
  - A：小说著作权及 20 万版权费均为夫妻共同所有 / 错——著作权人身权部分属姜某个人
  - B：苏某举证家务承担属举证责任倒置 / 错——是"谁主张谁举证"
  - C：苏某获家务补偿符合夫妻平等核心 / 正确
  - D：姜某不服可在判决生效后 15 日内上诉 / 错——上诉期是"判决书送达之日起 15 日内"
- **Q11 法律细节**：婚姻家庭财产制；著作权人身权与财产权分离；家务劳动补偿（民法典第 1088 条）；上诉期限起算点。

### 【6】2026 东城一模 Q10 —— 宅基地+夫妻债务+宅基地处分
- **题干情境**：张三婚前申请宅基地，婚后夫妻在该地建房；张三私下向王五借款用于装修；夫妻商议出租房屋。
- **4 个选项**：
  - A：宅基地由张三申请 → 房屋属其个人财产，需登记取得所有权 / 错——婚后建房属共同财产，建造取得所有权无需登记
  - B：李四不知情 → 借款属个人债务，李四无需偿还 / 错——用于家庭装修属夫妻共同债务
  - C：宅基地权益对应物的使用价值，张三与李四无权处分该地块 / 正确（用益物权限制处分性）
  - D：出租所获财产性收益属于再分配 / 错（属初次分配）
- **法律细节**：宅基地使用权（用益物权）的限制处分性；夫妻共同债务认定（家庭日常生活需要标准）；建造取得 vs 登记取得。

### 【7】2026 东城一模 Q11 —— AIGC 数字人 / 人格权益
- **题干情境**：某独资企业用 AIGC 采集博主林某公开短视频中的人脸+声音生成数字人用于商业直播，未标注"AI 生成"；林某要求停止使用、删除复刻数据并赔偿。
- **4 个选项**：
  - A：该企业作为营利法人，其收入应缴企业所得税 / 错——独资企业缴个人所得税
  - B：侵犯林某肖像权和声音权益，应停止侵害、恢复原状 / 错——救济方式中"恢复原状"用词不当
  - C：未标注"AI 生成"虽未侵犯著作权但属违法行为 / 正确——违反《直播电商监督管理办法》第 37 条标识义务
  - D：企业提供自身不存在过错的证据属举证责任倒置 / 错——肖像权侵权"谁主张谁举证"
- **法律细节**：声音权益参照适用肖像权保护（民法典第 1023 条）；AIGC 标识合规义务；侵权救济方式精准用词；独资企业税收性质。

### 【8】2026 通州期末 Q12 —— 遗嘱 vs 遗赠扶养协议
- **题干情境**：刘某先立遗嘱将房产由其子继承，后又与保姆签《遗赠扶养协议》约定房产归保姆。保姆履行扶养义务，刘某去世后房产归属起争议。
- **4 个选项**：
  - ①遗赠扶养协议为双务合同，效力优先于遗嘱 / 正确（民法典 1158 条）
  - ②保姆已履行扶养义务，有权主张房产所有权 / 正确
  - ③遗嘱效力优于遗赠扶养协议 / 错
  - ④遗嘱与协议均因冲突而无效 / 错
- **法律细节**：继承优先顺序（遗赠扶养协议 > 遗嘱继承/遗赠 > 法定继承）；遗赠扶养协议双务有偿性。

### 【9】2026 通州期末 Q13 —— 未成年人著作权 + 法定许可 + 举证
- **题干情境**：17 周岁张某拍摄古建筑照片投稿杂志社，次年发现照片被某出版社用作图书封面，未署名未付酬。
- **4 个选项**：
  - A：未满 18 周岁 → 照片不属于著作权法保护的作品 / 错——著作权基于"作品"独创性产生，与作者年龄无关
  - B：可适用"法定许可"但应补付报酬 / 错——图书封面使用不属于法定许可情形
  - C：本案不属于举证责任倒置情形，应由张某就侵权事实承担举证责任 / 正确
  - D：著作权侵权须先经版权管理部门处理 / 错——可直接起诉
- **法律细节**：著作权产生与年龄无关；法定许可的具体情形（教科书、报刊转载等有限场景）；著作权纠纷不实行行政前置；举证责任一般规则。

## 三、命题趋势小结

1. **侵权 / 违约竞合 + 多元解决**仍是高频主线（东城期末 Q18、朝阳期末 Q17(1)、丰台一模 Q20）。
2. **知识产权（含惩罚性赔偿、调解、滥诉规制）**强势上升，朝阳一模与东城一模均设大题，"司法守护向新力"成为标准化命题模板。
3. **新业态侵权**（AIGC 数字人、直播带货、探店拍摄）成为情境主战场——肖像权、声音权益、个人信息、虚假宣传四个考点反复轮转。
4. **举证责任规则**（行政诉讼倒置 / 民事"谁主张谁举证" / 过错推定）是 2026 年的高难精细化考点（石景山一模 Q18 专门设题）。
5. **婚姻家庭财产+继承**（家务补偿 / 著作权归属 / 遗赠扶养协议优先）回归选择题考察重点。
6. **行政公益诉讼+检察建议**作为新制度持续被命题（丰台期末 Q18）。
7. **物权（特别是宅基地、用益物权、绿色原则物权）**保持稳定输出（西城一模 Q17、东城一模 Q10）。
8. 2026 东城一模 Q18 的命题给分点呈现明显**"反套路化"取向**——明确禁止用"不正当竞争""搭便车""商业诋毁"等近义词替代精准法律概念，倒逼学生用准词。
