# 2024 北京高考政治《法律与生活》模拟题切片摘要

> 范围：北京 2024 届一模/二模/期中切片中与《法律与生活》直接相关的题。
> 用途：建框架 + 情境库 + 学生备考文档的源头素材。
> 注意：题号以原卷为准；同一切片文件可能涵盖多道相邻题，本摘要按"法律相关题"逐题列出。

## 一、主观题（共 4 题）

### 主观题 1 ｜ 朝阳二模 Q17（9 分）：新就业形态劳动关系认定

- **案情材料**：王某 2022 年与甲公司签订《车辆管理协议》，名义为"合作关系"，自备面包车经平台接单送货；公司每月支付包月服务费 6000 元加奖励，平台每日签到、跟踪订单，王某日跑单 8 小时以上、超额奖励、拒单或货损扣费。2023 年甲公司提前终止合作，王某主张已构成劳动关系并申请仲裁，仲裁委裁决甲公司支付解除劳动合同经济补偿。
- **设问**：结合材料，说明仲裁委员会作出上述裁决的理由，并分析该典型案例的意义。
- **法律领域**：劳动法（劳动关系认定）+ 仲裁 + 新就业形态侵权/权益保护。
- **踩分点关键短语**：
  1. 人格从属性（劳动纪律、奖惩办法适用，按平台指令完成任务）；
  2. 经济从属性（平台掌握生产资料/数据信息，劳动者无服务定价权）；
  3. 组织从属性（被纳入平台组织体系，以平台名义对外提供服务）；
  4. 名为"合作"实为劳动关系，应受劳动法保护，享有解除经济补偿；
  5. 典型案例的意义：统一裁审尺度、压实平台责任、保障新就业形态劳动者权益、引导平台合规用工。
- **命题路径反推**：
  1. 命题人锁定"新就业形态"这一时代热点，目的是让学生跨越"形式合同名称"去识别"实质法律关系"；
  2. 选这个载体是因为外卖/网约车/货运司机贴近学生生活经验，且能直接对应国家近年来对平台用工的政策表态；
  3. 设问采用"裁决理由 + 案例意义"双层结构，前者考调用规则要素的能力（三个从属性），后者考从个案到制度功能的升华能力；
  4. 给出资料包（劳社部通知 + 三性指标）等于把"赋分细则"前置——学生必须从材料中抠出每一个从属性的事实对应；
  5. 细则奖励的动作：能"由名入实"（不被"合作关系"字面迷惑）、能逐项对接事实与三性、能写出对劳动者权益及行业规范的双重价值；
  6. 这是一道把"劳动合同"从一般用工拓展到平台用工的标志题，是 2024 年法律与生活最具命题学价值的题之一。

### 主观题 2 ｜ 海淀二模 Q19（6 分）：遗赠扶养协议纠纷案宣判

- **案情材料**：某社区居委会与曹某签订协议，约定居委会结对照看、提供生活费、免费就医、节日慰问，养老至寿终；曹某去世后其动产与不动产产权移交居委会。居委会按约履行义务。曹某去世后，其四个子女主张继承遗产并与居委会发生争议，居委会诉至法院。法院认定该协议属有效遗赠扶养协议，判决案涉房屋、股权、现金、银行存款归居委会所有。
- **设问**：结合材料，运用《法律与生活》知识，分析本案判决的社会价值。
- **法律领域**：继承法（遗赠扶养协议）+ 民事权利的处分自由 + 基层组织的法律地位。
- **踩分点关键短语**：
  1. 遗赠扶养协议优先于遗嘱继承和法定继承，是被继承人对身后财产作出的合法处分；
  2. 协议双方权利义务对等（一方扶养至寿终、一方移交财产），属意思自治范围内的有效约定；
  3. 判决支持居委会取得财产 = 鼓励组织参与养老、回应人口老龄化与失独/空巢老人现实需求；
  4. 通过司法裁判明确"非自然人主体亦可作为扶养人"，拓宽养老主体；
  5. 弘扬契约精神与诚信履约文化，引导基层社区共同体承担社会责任。
- **命题路径反推**：
  1. 命题人选择"居委会作为扶养人"这一非典型主体，意在让学生跳出"亲属继承"的常识窗口，理解遗赠扶养协议的开放性；
  2. 选这个载体是因为老龄化是北京命题的稳定背景，且子女与居委会的争议自然引出"协议优先于法定继承"的规则；
  3. 设问指向"社会价值"而非"法律理由"——这是命题人故意从"裁判结果是否合规"上移到"司法裁判的社会功能"；
  4. 细则奖励的动作：先点出协议性质与效力位阶、再扩展到养老制度的社会价值、最后落到契约精神/法治信念；
  5. 这道题是"继承"模块在 2024 年最具典型性的设问，配合的是《民法典》倡导的多元养老支持。

### 主观题 3 ｜ 丰台二模 Q17（8 分）：民法典守护生态 + 知识产权双案例

- **案情材料**：
  - 案例一：某公司发明专利技术，攻克农村污水分离关键工艺，方法简单、净化效率显著提升，已在多地推广。
  - 案例二：甲公司长期超标排放废水废气，损害环境公共利益与周边居民生活；乙环境研究所提起环境民事公益诉讼，法院判决甲公司停止侵权、消除危险、赔偿环境损失。
- **设问**：结合材料，说明民法典在守护碧水蓝天、保护生态环境中所发挥的作用。（题干给出民法典条文：节约资源保护生态原则 + 知识产权依法享有 + 因污染环境/破坏生态承担侵权责任）
- **法律领域**：民法典基本原则（绿色原则）+ 知识产权 + 侵权责任（环境侵权）+ 民事公益诉讼。
- **踩分点关键短语**：
  1. 绿色原则确立民事主体在民事活动中保护生态、节约资源的行为准则；
  2. 保护知识产权 = 鼓励绿色技术创新（案例一），把环保科研成果转化为可复制可推广的解决方案；
  3. 环境污染破坏生态的侵权人承担侵权责任 = 划定行为底线，让违法者付出代价（案例二）；
  4. 通过民事公益诉讼制度，把"环境公共利益"纳入可诉范围，让环境损害得到救济；
  5. 民法典通过"激励 + 约束"双向机制（鼓励绿色创新 + 追究环境侵权），共同护航生态文明建设。
- **命题路径反推**：
  1. 命题人把"绿色原则 + 知识产权 + 环境侵权"打包成一道题，意在让学生看懂民法典是体系化工具而不是孤立条文；
  2. 双案例并列设计是关键巧思——一正一反、一鼓励一惩戒，对应同一立法目的；
  3. 设问用"作用"而非"分析谁对谁错"，逼学生从规则适用上升到立法目的；
  4. 细则奖励的动作：能区分两个案例对应不同条款、能把绿色原则讲成贯穿性原则、能升华到生态文明法治保障；
  5. 这道题是 2024 年民法典与"双碳"政策对接的代表题，也是"知识产权"模块少有的带情境亮相。

### 主观题 4 ｜ 西城二模 Q16（8 分）：食品安全惩罚性赔偿案

- **案情材料**：市民甲首单购买"黄芪薏米饼干"30 盒，发现不符合食品安全标准；其后两个多月内多次向同一商家加购同款饼干 200 盒（共 18.4 千克）。法院引用《食品安全法》第 148 条第 2 款关于"价款十倍或损失三倍"的惩罚性赔偿规定，支持甲对首单饼干的惩罚性赔偿请求，但不支持加购部分（认定加购超出正常生活消费需要）。
- **设问**：结合材料，运用《法律与生活》的知识，分析人民法院判决的意义。
- **法律领域**：消费者权益保护（惩罚性赔偿）+ 民事责任承担方式 + 经营者义务 + 权利不得滥用。
- **踩分点关键短语**：
  1. 惩罚性赔偿是消费者维权利器：支持首单赔偿，倒逼经营者履行食品安全义务、保护消费者人身财产安全；
  2. 经营者承担生产经营不符合食品安全标准食品的法律责任，体现"过罚相当"；
  3. 权利行使应符合诚实信用与禁止权利滥用原则——加购 200 盒已超出正常消费需要，不能借维权牟利；
  4. 区别支持/不支持的判决方式，平衡了"严格保护消费者"与"防止职业打假式营利套利"两端；
  5. 通过个案确立裁判指引，统一同类案件的尺度，引导诚信消费 + 合规经营 + 理性维权。
- **命题路径反推**：
  1. 命题人选这个案子是因为它把"惩罚性赔偿"和"权利滥用边界"放在同一案中，特别考验学生的双向辩证；
  2. 选食品案而不是普通买卖，是为了凸显《食品安全法》十倍/三倍这一加重责任规则的特殊性；
  3. 设问只问"判决意义"，不问"法律依据"，是因为依据已写在题干（148 条 2 款）；学生要做的是从规则升华到价值；
  4. 细则奖励的动作：分别讲到"支持首单"和"不支持加购"两条裁判逻辑，且都能落到立法精神；
  5. 这道题示范了 2024 年北京命题的风向——让学生在"保护谁"和"约束谁"之间作出有层次的法律论证，反对单边思维。

## 二、选择题（共 17 题）

### 东城一模 Q9 ｜ 物权确认与变动

- **题干情境**：张某因房屋产权纠纷向法院提起物权确认之诉，法院判决房屋归张某所有。
- **选项核心法律点**：
  - A：尽快办理房产登记，以免影响物权变动 → 错；
  - B：他物权对应使用价值 → 干扰；
  - C：涉案房产与占地法律关系内容一致 → 干扰；
  - D：张某对房屋的物权自判决宣告时生效 → 正确（生效法律文书的非基于法律行为的物权变动）。
- **法律细节**：基于法律文书的物权变动自文书生效时发生，不以登记为生效要件（登记仅影响处分）；区分自物权（所有权）与他物权（用益物权、担保物权）。

### 东城一模 Q10 ｜ 遗嘱效力与酌定继承

- **题干情境**：高某夫妇由侄子高某翔照顾，立"遗嘱"将全部遗产由高某翔继承，但高某翔本人不知情；其独生子高某亮反对，请求确认遗嘱无效。题干旁附《民法典》第 1131 条（继承人以外依靠被继承人扶养/对被继承人扶养较多的人可分给适当遗产）。
- **选项核心法律点**：
  - A：该"遗嘱"实为遗赠扶养协议 → 错（缺协议合意，且高某翔不知情）；
  - B：高某翔因照顾被继承人，可继承适当的遗产 → 正确（依 1131 条酌定分得适当遗产）；
  - C：遗嘱无效，全部按法定继承 → 表述偏颇；
  - D：高某翔尽主要赡养义务，视为第一顺位继承人 → 错（不能改变继承顺位）。
- **法律细节**：遗赠扶养协议须双方合意；非继承人扶养较多者可"分给适当遗产"，不进入继承人顺位。

### 东城一模 Q11 ｜ 民事公益诉讼 + 古树保护

- **题干情境**：陆某州等人盗割毁坏国家二级保护野生植物闽楠，检察院提起刑事附带民事公益诉讼，法院发出全国首份《古树救治令》并发出司法建议。
- **选项核心法律点**：
  - ①司法与行政同频共振、各发挥古树保护职能 → 对；
  - ②坚持能动司法、探索司法保护与生态修复结合的新途径 → 对；
  - ③毁坏古树适用无过错责任原则、人民检察院依法提起公诉 → 错（毁坏野生植物适用过错责任；这里是公益诉讼非普通公诉）；
  - ④政府对法院负责、司法建议对其有约束力 → 错（司法建议无强制约束力；政府对人大负责）。
- **法律细节**：刑事附带民事公益诉讼制度；归责原则区分（环境污染适用无过错责任，但破坏生态/盗伐适用过错责任）；司法建议的性质（建议而非命令）。

### 朝阳二模 Q11 ｜ 遗嘱继承与法定继承顺位

- **题干情境**：小李父亲生前立遗嘱将房产由小李继承；小李母亲名下房产、外祖父母去世后的房产份额涉及法定继承。
- **选项核心法律点**：
  - ①父亲名下房产份额按遗嘱由小李全部继承 → 对；
  - ②无需取得舅舅同意即可取得房产全部产权 → 错（涉夫妻共同财产份额、母亲份额另算）；
  - ③母亲去世后名下房产份额按法定继承一般应当均等分配 → 对；
  - ④外祖父母去世后小李以第二顺序继承人身份继承 → 错（代位继承属第一顺序）。
- **法律细节**：遗嘱继承覆盖范围只限被继承人个人份额；法定继承同一顺序均等原则；代位继承的顺位归属。

### 朝阳二模 Q12 ｜ 民事诉讼程序

- **题干情境**：羽毛球比赛中刘某被张某挥拍击伤右眼，刘某起诉要求赔偿；一审判决驳回，二审维持原判。
- **选项核心法律点**：
  - A：双方都有委托辩护人、申请回避和上诉的权利 → 错（民诉是诉讼代理人，不是辩护人；二审已是终审，无上诉权）；
  - B：自诉人刘某可向法院追究张某刑事责任 → 错（本案为民事侵权，非自诉案件）；
  - C：本案为解决平等主体之间民事权利义务纠纷的民事诉讼 → 正确；
  - D：刘某不服二审，可请求启动审判监督程序或向上一级法院上诉 → 错（二审终审，只能审判监督）。
- **法律细节**：民事诉讼与刑事自诉的区分；两审终审制；辩护人 vs 诉讼代理人；审判监督程序作为终审后的特殊救济。

### 海淀二模 Q9 ｜ 内地与香港民商事判决互认

- **题干情境**：2024 年 1 月《关于内地与香港特别行政区法院相互认可和执行民商事案件判决的安排》生效，含"重复诉讼不予受理"与"复议/上诉"条款。
- **选项核心法律点**：
  - ①扩大两地法院管辖权 → 错（是判决互认，非管辖扩张）；
  - ②消除两地法律制度、司法理念差异 → 错（不可能也非目的）；
  - ③坚持主权统一原则，彰显"一国两制"制度优势 → 正确；
  - ④有效避免或减少当事人重复诉讼，节约司法资源 → 正确（对应第二十三条）。
- **法律细节**：跨法域司法协助；一事不再理在区际司法协助中的体现。

### 海淀二模 Q10 ｜ 个人信息与隐私权

- **题干情境**：赵某开发可通过新闻链接获取他人实时定位的程序并嵌入 App，用户付费向被定位者发送链接以获取位置信息。
- **选项核心法律点**：
  - ①敏感个人信息泄露可能危害自然人人身与财产权益 → 正确（行踪信息属敏感个人信息）；
  - ②未经同意向他人提供个人信息应承担违约责任 → 错（应是侵权责任，非违约）；
  - ③个人信息与隐私权密切相关，不得泄露或非法提供 → 正确；
  - ④大数据时代个人信息保护应适用隐私权法律规定 → 错（个人信息保护与隐私权有交集但不等同，已有专门法）。
- **法律细节**：敏感个人信息（含行踪轨迹）规则；个人信息权益与隐私权的区分与衔接；侵害个人信息的民事责任性质。

### 海淀二模 Q11 ｜ 业主共有权与物权救济

- **题干情境**：题干引用《民法典》第 274 条（建筑区划内道路、其他公共场所、公用设施和物业服务用房属业主共有），讨论共享单车扎堆停放妨碍通行问题。
- **选项核心法律点**：
  - ①业主对共有部分享有共有和共同管理权利 → 正确；
  - ②有权要求排除妨碍、恢复正常通行 → 正确（物权请求权）；
  - ③应用程序运营方仅提供链接服务、无需承担侵权责任 → 错（须依注意义务/避风港规则具体判断）；
  - ④可向业委会或物业投诉，强制禁止共享单车进入小区 → 错（无权"强制禁止"，越权）。
- **法律细节**：建筑物区分所有权（专有部分 + 共有部分 + 共同管理权）；物权请求权（排除妨碍、恢复原状）；平台/运营方注意义务边界。

### 西城一模 Q6 ｜ 担保物权（质押）

- **题干情境**：王某向赵某借 50 万元，将唯一机动车质押给赵某；又向钱某借 20 万元未还，钱某起诉要求将该机动车折价赔偿。
- **选项核心法律点**：
  - ①本案涉及债权关系，客体是机动车 → 错（债权客体是行为/给付，不是物）；
  - ②本案涉及担保物权，对应机动车的交换价值 → 正确；
  - ③将机动车交予赵某，赵某拥有机动车所有权 → 错（质押不转移所有权）；
  - ④赵某占有担保物，享有质权，可依法优先受偿 → 正确。
- **法律细节**：动产质权以转移占有为成立要件；担保物权对应交换价值（区别于用益物权对应使用价值）；优先受偿权对抗一般债权人。

### 西城一模 Q7 ｜ 经营者义务 + 消费者权益（"生鲜灯"）

- **题干情境**：商家用"生鲜灯"使生鲜食品色泽诱人，市场监管总局规定不得使用此类照明设施误导消费者。
- **选项核心法律点**：
  - ①经营者应依法、诚信经营，保障消费者合法权益 → 正确；
  - ②消费者发现被误导可依法维权 → 正确；
  - ③属于不正当竞争，是搭他人商业信誉的便车 → 错（不是仿冒/搭便车，而是误导消费者）；
  - ④利用信息不对称侵害消费者自主选择权 → 干扰（核心是知情权而非自主选择权，此选项常被误判）。
- **法律细节**：经营者义务（真实信息提供）；消费者知情权与自主选择权区分；不正当竞争行为的具体类型（混淆、虚假宣传等）。

### 西城一模 Q8 ｜ 纠纷解决方式

- **题干情境**：李某将空调外机安装在张某卧室窗外墙体，噪音和气流严重影响张某休息，多次协商无果。
- **选项核心法律点**：
  - ①直接拆除李某空调外机 → 错（私力救济越界）；
  - ②向相关仲裁机关申请商事仲裁 → 错（邻里纠纷非商事仲裁范围）；
  - ③向人民调解委员会申请调解 → 正确；
  - ④到人民法院起诉 → 正确。
- **法律细节**：纠纷解决方式的适用范围（自力救济边界、商事仲裁的协议前提与商事属性、人民调解的非强制性、诉讼作为最终救济）。

### 西城二模 Q11 ｜ 行政公益诉讼程序

- **题干情境**：H 省第一公立小学旧址（省级文保单位）面临损毁，J 县文旅局未尽监管职责；J 县检察院提起公益诉讼，法院判令文旅局继续履职。
- **选项核心法律点**：
  - ①J 县文旅局应提供证据证明其行为合法性 → 正确（行政诉讼举证责任在被告行政机关）；
  - ②法庭辩论和宣告判决是整个诉讼过程中最重要的环节 → 错（无"最重要"绝对化表述）；
  - ③应严格按民事诉讼法规定的程序进行审理 → 错（行政公益诉讼应适用行政诉讼法）；
  - ④文旅局可在一审判决书下达后 15 日内提起上诉 → 正确（行政诉讼上诉期 15 日）。
- **法律细节**：行政诉讼"举证责任倒置"由被告承担；行政公益诉讼的程序法依据；行政诉讼上诉期与民事诉讼上诉期（15 日）。

### 西城二模 Q12 ｜ 夫妻共同财产/共同债务 + 继承

- **题干情境**：王某 2018 年个人做生意借款 3 万元未还；2020 年与赵某结婚；2021 年赵某遗嘱继承父亲指定给她的房子；现感情破裂协议离婚。王某主张炒股本金 10 万元（工资奖金）的 5 万元收益是"私房钱"。
- **选项核心法律点**：
  - A：王某所欠 3 万元是其个人债务，非夫妻共同债务 → 正确（婚前债务，且未用于共同生活）；
  - B：炒股本金属王某个人财产，5 万元收益属夫妻共同财产 → 错（10 万元是婚后工资奖金 → 已属共同财产，5 万元收益亦共同）；
  - C：婚姻关系存续期间继承的房子属共同财产 → 错（遗嘱明确指定给赵某一方的属个人财产）；
  - D：所有财产相加平均分配 → 错（不区分个人/共同财产，违法）。
- **法律细节**：婚前个人债务的归属；夫妻共同财产的范围（工资奖金、生产经营收益）；遗嘱明确指定给一方的财产为个人财产；离婚财产分配的"先识别、后分配"原则。

### 西城二模 Q13 ｜ 人民调解（"枫桥经验"）

- **题干情境**："小事不出村、大事不出镇、矛盾不上交"的"枫桥经验"作为人民调解的典型代表。
- **选项核心法律点**：
  - ①尊重当事人权利，只收取最基本的费用 → 错（人民调解不收费）；
  - ②是党领导下解决纠纷行之有效的基层社会治理机制和方法 → 正确；
  - ③依靠人民群众正确处理人民内部矛盾 → 正确；
  - ④由人民法院主持调解、可使当事人达成调解协议 → 错（人民调解由人民调解委员会主持，非法院）。
- **法律细节**：人民调解的免费性、群众性、自愿性；调解主体（人民调解委员会）；调解协议的效力。

### 顺义二模 Q10 ｜ 直播带货合同纠纷

- **题干情境**：经营者吴某与某文化传媒公司签订推广合同，约定平均每日增加营销额数百元；吴某支付推广费 2 万余元，实际只赚 200 元，与签约承诺相差甚远。
- **选项核心法律点**：
  - A：订立合同就能规范市场主体的交易行为，维护民事权利 → 错（"就能"绝对化）；
  - B：传媒公司违背诚信经营，因未完全履行合同需承担违约责任 → 正确；
  - C：吴某可申请仲裁，对仲裁不服可提起诉讼解决合同纠纷 → 错（仲裁实行一裁终局，对裁决不服只能在法定情形下申请撤销/不予执行，不能再起诉）；
  - D：传媒公司履行合同义务不符合约定，应向吴某公开道歉 → 错（合同纠纷下不当然产生公开道歉的责任形式）。
- **法律细节**：违约责任与民事责任承担方式；仲裁一裁终局与"或裁或审"原则；合同欺诈与违约的边界。

### 顺义二模 Q11 ｜ 网络匿名差评 + 新闻报道侵权

- **题干情境**：小卫购买 E 公司课程后不满意，在某网站匿名差评；后向某报业公司投稿，该公司经调查后如实报道。E 公司起诉小卫和报业公司构成共同侵权（要求删除内容、公开赔礼道歉、赔偿损失），一审判决驳回。
- **选项核心法律点**：
  - ①小卫匿名差评，积极行使了批评监督权，没有逾越法律红线 → 正确；
  - ②本判决为终审判决，E 公司不得提起上诉 → 错（一审判决可上诉）；
  - ③某报业公司作为新闻媒体，为公共利益如实报道，不承担民事责任 → 正确；
  - ④该案件适用举证责任倒置原则，故小卫需承担举证责任 → 错（一般侵权遵循"谁主张、谁举证"，原告 E 公司举证）。
- **法律细节**：消费者评价权与名誉权边界；新闻报道公共利益抗辩；侵权诉讼举证责任分配；两审终审制中一审判决的可上诉性。

### 西城二模 Q9 ｜ AIGC 技术伦理（弱法律点，关联题）

- **题干情境**：央视网用 AIGC 技术生成各地标志性图片视频，主流媒体推出"智慧+"产品。
- **选项核心法律点**：
  - A：用数字技术主导设计方案及思想表达 → 错（人机关系倒置）；
  - B：以内容创新为行业赋能，提供价值引领 → 正确；
  - C：现代技术是支配事物和人的抽象力量 → 错；
  - D：技术变革带来理念重构、实践重塑、社会重组 → 干扰（强调结果而非"应当重视什么"）。
- **法律细节**：与法律点弱关联，但为后续 AIGC/算法治理类题目埋下情境素材，可关联到 2024 年陆续生效的生成式人工智能管理办法。
