# 选必二《法律与生活》情境版

> 按近三年北京各区一模/二模/期末的真实情境穷尽，每个情境给"看到什么 / 想到什么 / 写什么 / 落到哪"四件套。
> 主观题情境与选择题情境严格分开，不混排。

---

## 第一部分：主观题情境（按七大主线组织，共 36 个情境）

每条情境格式：
- **起因 → 经过 → 结果**（一句话还原案情）
- **材料触发词**（看到这些信号就知道是这一类）
- **必踩关键短语**（写到答案里就给分）
- **落点句**（功能落点段标准句式）
- **典型出处**（参考真实题位）
- **反例提醒**（这样写直接扣分）

### 主线 1 — 合同（共 5 个情境）

#### S1-01 直播带货虚假宣传 / 约定优先于法定
- **起因→经过→结果**：经营者在直播间承诺"假一赔十"等高于法定的赔偿，消费者购后发现产品名实不符，起诉获法院支持。
- **材料触发词**：直播间、承诺"假 X 赔 N"、保真、产品名实不符、约定赔偿。
- **必踩关键短语**：买卖合同 / 约束力 / 意思自治 / 经营者欺诈 / 约定优先 / **消费者权益保护法**全名。
- **落点句**："维护诚信交易秩序，规范新业态经营行为，保障消费者合法权益。"
- **典型出处**：朝阳期末 Q17(1)、海淀一模 Q18 案件 2 同型。
- **反例提醒**：写~~"消法"~~"保护法"等缩写直接扣分；写"商业欺诈罪"是把民事写成刑事。

#### S1-02 合同 + 侵权竞合 / 产品瑕疵致害
- **起因→经过→结果**：买卖合同标的产品有瑕疵造成消费者人身或财产损害，受害人提起诉讼。
- **材料触发词**：买卖合同、产品瑕疵、人身损害 / 财产损失、维权途径选择。
- **必踩关键短语**：合同有效 / 违约责任 / 侵权责任 / 责任竞合 / 受害人择一主张 / 固定证据 → 明确诉求 → 选择争议解决方式。
- **落点句**："维护交易安全，提升消费者依法维权意识，优化营商环境。"
- **典型出处**：东城期末 Q18。
- **反例提醒**：不要把"竞合"理解为"叠加"——是"择一"，不是"全要"。

#### S1-03 合同形式自由 / 口头买卖合同
- **起因→经过→结果**：当事人未采用书面形式订立买卖合同，其后产生履行/价款争议。
- **材料触发词**：口头约定、未签书面合同、收购商、即时合同。
- **必踩关键短语**：形式自由 / 买卖合同口头有效 / 履行行为可证明合同成立。
- **落点句**："尊重当事人意思自治，降低交易成本，激发市场活力。"
- **典型出处**：海淀一模 Q10。
- **反例提醒**：不要写"未签书面合同则未成立"——错。

#### S1-04 合同效力四态 / 重大误解 / 显失公平
- **起因→经过→结果**：缔约一方因对方欺诈/胁迫/重大误解/显失公平 → 请求撤销合同。
- **材料触发词**：误解、被骗、胁迫、价格悬殊、利用对方危困。
- **必踩关键短语**：可撤销合同 / 撤销权 / 撤销自始无效 / 利益返还。
- **落点句**："保护意思表示真实，维护实质公平。"
- **反例提醒**：可撤销 ≠ 无效；可撤销由当事人主张行使。

#### S1-05 违约救济五种 / 定金罚则
- **起因→经过→结果**：一方违约 → 守约方主张多项救济（继续履行 / 修理更换 / 损害赔偿 / 违约金 / 定金）。
- **材料触发词**：违约金、定金、修理重作更换、要求继续履行。
- **必踩关键短语**：继续履行 / 采取补救措施 / 赔偿损失 / 违约金 / 定金罚则（双倍返还）。
- **落点句**："保障合同严肃性，维护交易稳定预期。"
- **反例提醒**：违约金与定金不能并用全额——只能择高。

### 主线 2 — 侵权（共 7 个情境）

#### S2-01 AIGC 数字人 / 未经同意采集人脸声音
- **起因→经过→结果**：企业未经被采集人同意，用 AIGC 生成数字人投入商业使用，且未标注"AI 生成"。
- **材料触发词**：AIGC、AI 生成、数字人、复刻、深度合成、未标注、声音/人脸采集。
- **必踩关键短语**：肖像权 / 声音权益（参照肖像权）/ 个人信息权益 / AI 生成标识合规义务 / 停止侵害 / 消除影响 / 赔礼道歉 / 损害赔偿。
- **落点句**："引导技术向善，明确新业态合规边界，保护人格权益不被技术侵蚀。"
- **典型出处**：东城一模 Q11、房山一模 Q10。
- **反例提醒**：救济用~~"恢复原状"~~是错（人格权侵权用"消除影响"）。

#### S2-02 探店博主 / 拒绝出镜后仍被传播
- **起因→经过→结果**：拍摄者明知被拍者拒绝出镜仍清晰拍摄并实时直播。
- **材料触发词**：直播、拒绝出镜、清晰画面、实时传播、明确反对。
- **必踩关键短语**：肖像权 / 隐私权 / 著作权不能对抗人格权 / 一般侵权"谁主张谁举证"。
- **落点句**："科技应用不能逾越法律红线，公共空间也有人格尊严边界。"
- **典型出处**：房山一模 Q10。
- **反例提醒**：博主~~"对画面享有著作权"~~不能成为抗辩理由。

#### S2-03 建筑物悬挂物脱落致损
- **起因→经过→结果**：建筑物窗户/广告牌/空调外机等脱落，造成行人/车辆损害。
- **材料触发词**：窗户脱落、高空坠物、广告牌掉落、车被砸。
- **必踩关键短语**：建筑物、构筑物或悬挂物侵权 / 过错推定 / 受害人举证损害+因果关系 / 所有人/管理人/使用人证明无过错。
- **落点句**："合理分配证明负担，督促管理人履行注意义务，预防高空坠物风险。"
- **典型出处**：石景山一模 Q18。
- **反例提醒**：不要写"无过错责任"（是过错推定）。

#### S2-04 安全保障义务边界 / 自伤
- **起因→经过→结果**：顾客在经营场所内自身原因摔倒受伤起诉经营者；经营者已尽义务，法院驳回。
- **材料触发词**：餐厅、商场、台阶、摔倒、低头看手机、无积水积雪。
- **必踩关键短语**：安全保障义务（民法典第 1198 条）/ 经营者无过错 / 受害人自身过错 / 过错责任原则 / 义务有边界。
- **落点句**："明确义务边界，引导个人对自身安全负责，鼓励经营者正常营业，防止'和稀泥'裁判。"
- **典型出处**：丰台一模 Q20。
- **反例提醒**：~~"基于公平原则承担部分责任"~~——错；无过错就是无过错。

#### S2-05 知识产权惩罚性赔偿 / 故意+情节严重
- **起因→经过→结果**：一方故意大规模侵害他人知识产权，社会危害大；法院适用惩罚性赔偿。
- **材料触发词**：故意、明知、规模、长期、屡次、产值、销售额。
- **必踩关键短语**：故意 / 情节严重 / 惩罚性赔偿（1-5 倍）/ 增加侵权成本 / 警示规范作用。
- **落点句**："严格保护知识产权，加大侵权打击力度，激发全社会创新活力。"
- **典型出处**：朝阳一模 Q18 案 2、东城一模 Q18 案 2。
- **反例提醒**：写"商业诋毁""不正当竞争""搭便车""消费者知情权"作为案 2 给分点直接扣分（题意是侵权）。

#### S2-06 网络服务提供者 / "通知—删除"规则
- **起因→经过→结果**：网络用户实施侵权行为，平台收到通知后未及时删除，权利人主张连带责任。
- **材料触发词**：网络平台、通知删除、明知应知、推荐算法、链接。
- **必踩关键短语**：网络用户与平台连带责任 / 通知-删除规则 / 平台合理注意义务 / 红旗规则。
- **落点句**："压实平台主体责任，构建网络空间法治秩序。"
- **反例提醒**：不要简单说"平台担全责"——是连带责任的边界。

#### S2-07 人格权请求权 / 救济用词精准
- **起因→经过→结果**：人格权遭侵害（隐私、个人信息、名誉等），权利人主张救济。
- **材料触发词**：公开、传播、删除、停止、致歉。
- **必踩关键短语**：停止侵害 / 排除妨碍 / 消除影响 / 恢复名誉 / 赔礼道歉 / 精神损害赔偿。
- **落点句**："保护人格权益，维护公民人格尊严。"
- **反例提醒**：人格权救济**不写"恢复原状"**（那是物权救济）。

### 主线 3 — 物权（共 4 个情境）

#### S3-01 宅基地使用权处分限制 / 婚后建房归属
- **起因→经过→结果**：一方婚前申请宅基地，婚后夫妻共同建房；一方私下处分或借款用于装修。
- **材料触发词**：宅基地、婚后建房、村委会、集体土地、装修借款。
- **必踩关键短语**：宅基地使用权（用益物权）/ 集体经济组织成员资格限制 / 不得擅自处分 / 建造取得所有权（无需登记）/ 房屋为夫妻共同财产 / 家庭日常生活借款为共同债务。
- **落点句**："维护农村土地集体所有制，保障家庭财产稳定预期，平衡个人意思与家庭团体性。"
- **典型出处**：东城一模 Q10。
- **反例提醒**：~~"宅基地建房需登记取得所有权"~~——错。

#### S3-02 抵押 vs 质押客体辨别
- **起因→经过→结果**：当事人用动产/不动产/股权/票据/知识产权设保，其后处置与受偿引争议。
- **材料触发词**：抵押、质押、股票、汇票、专利、登记、留置。
- **必踩关键短语**：抵押客体（不动产+特定动产）/ 质押客体（动产+权利）/ 登记对抗或登记生效 / 质权实现方式（拍卖变卖折价）/ 质权人不得擅自占有使用。
- **落点句**："保障债权实现，规范担保秩序。"
- **典型出处**：朝阳一模 Q10。
- **反例提醒**：~~"股票抵押"~~——错（应为"股票质押"）。

#### S3-03 物权 + 绿色原则 / 相邻关系
- **起因→经过→结果**：物权行使（不动产 / 用益物权 / 相邻关系）涉及环境保护与多元利益协调。
- **材料触发词**：绿色原则、节约资源、保护环境、相邻关系、生态。
- **必踩关键短语**：绿色原则 / 物权行使全过程 / 节约资源保护环境义务 / 个人权利与公共生态利益平衡 / 良法善治。
- **落点句**："实现私权行使与生态保护协调，推动绿色生产方式转型。"
- **典型出处**：西城一模 Q17。
- **反例提醒**：~~"维护社会主义生态文明"~~（必修三化）。

#### S3-04 善意取得三要件
- **起因→经过→结果**：无权处分人将他人财产转让给善意第三人，第三人取得所有权。
- **材料触发词**：买二手车不知有抵押、不知是赃物、合理价格、已交付/登记。
- **必踩关键短语**：受让人善意（不知）/ 合理对价 / 已登记或交付 / 三要件齐备取得所有权。
- **落点句**："平衡原所有人与善意第三人利益，维护交易安全。"
- **反例提醒**：缺一要件即不构成善意取得。

### 主线 4 — 婚姻继承（共 4 个情境）

#### S4-01 遗赠扶养协议优先于遗嘱
- **起因→经过→结果**：被继承人立遗嘱在前 + 后签遗赠扶养协议 + 受扶养人已履行扶养义务。
- **材料触发词**：遗嘱、遗赠扶养协议、保姆、邻居、长期照顾。
- **必踩关键短语**：遗赠扶养协议（双务有偿）/ 优先于遗嘱继承/遗赠 / 优先于法定继承 / 已履行扶养义务有权主张约定财产。
- **落点句**："鼓励互助养老，承认协议价值，弘扬扶老敬老的社会风气。"
- **典型出处**：通州期末 Q12。
- **反例提醒**：~~"遗嘱在先效力优先"~~——错。

#### S4-02 家务劳动补偿 / 民法典第 1088 条
- **起因→经过→结果**：一方承担较多家务，离婚时主张家务补偿 + 财产分割照顾。
- **材料触发词**：抚育子女、照料老人、协助工作、家务、离婚分割。
- **必踩关键短语**：家务劳动补偿（民法典第 1088 条）/ 夫妻平等 / 承认家务价值 / "谁主张谁举证"（不是举证倒置）。
- **落点句**："彰显夫妻平等理念，承认家务劳动经济价值，引导家庭责任共担。"
- **典型出处**：海淀一模 Q11。
- **反例提醒**：~~"举证责任倒置"~~——错。

#### S4-03 婚后稿酬归属 / 著作权人身权独立
- **起因→经过→结果**：一方婚内创作小说获得稿酬 / 版权 / 后续收益，离婚时引争议。
- **材料触发词**：婚内创作、稿酬、版权费、署名权、发表权。
- **必踩关键短语**：婚内所得财产收益属共同财产 / 著作权人身权（署名/修改/发表/保护作品完整）专属作者 / 财产收益均分。
- **落点句**："分割财产权益的同时尊重作者人格利益。"
- **典型出处**：海淀一模 Q11。
- **反例提醒**：~~"著作权全部归共有"~~——错（人身权部分专属）。

#### S4-04 夫妻共同债务三标准
- **起因→经过→结果**：一方对外举债，另一方主张非共同债务。
- **材料触发词**：借款、未告知、用于装修 / 治病 / 子女教育 / 共同经营。
- **必踩关键短语**：共同意思表示 / 用于家庭日常生活 / 用于共同生产经营 → 三者之一即共同债务。
- **落点句**："平衡债权人保护与未举债方知情权，维护家庭财产安全。"
- **典型出处**：东城一模 Q10。
- **反例提醒**：单纯"夫妻关系"不必然形成共同债务，要看用途。

### 主线 5 — 知识产权 + 新业态（共 5 个情境）

#### S5-01 司法守护"向新力"三案模板
- **起因→经过→结果**：法院在知识产权审判中依次运用调解、惩罚性赔偿、规制滥诉三种工具。
- **材料触发词**：诉讼调解、惩罚性赔偿、恶意诉讼、滥用诉权、上市关键期。
- **必踩关键短语（差异化表达，不重复给分）**：
  - 调解段：诉讼调解 + 实质性化解 + 节约司法资源 / 减少对创新的干扰 / 以和为贵 / 法德结合。
  - 惩罚段：故意+情节严重 + 惩罚性赔偿 + 增加侵权成本 + 警示规范作用 / 维护市场秩序 / 公平竞争。
  - 滥诉段：驳回并谴责恶意诉讼 / 惩治滥用程序权利 + 诚信原则 / 权利义务统一 / 维护诉讼司法秩序。
  - 总说：公正司法 / 以事实为根据，以法律为准绳 / 司法裁判导向作用 + 保护知识产权就是保护创新 / 提升创新活力。
- **落点句**：见上述各段。
- **典型出处**：朝阳一模 Q18、东城一模 Q18。
- **反例提醒（重要）**：
  - 案 2"植物新品种侵权" **绝对不要写**："不正当竞争""搭便车""消费者知情权"。
  - 案 3"恶意诉讼" **绝对不要写**："商业诋毁""不正当竞争"。
  - 三段必须用差异化表达，相同短语跨段不重复给分。

#### S5-02 商业秘密 vs 专利权
- **起因→经过→结果**：非公知技术信息被他人破译/披露/使用，权利人主张救济。
- **材料触发词**：技术秘密、配方、客户名单、源代码、保密措施、破译。
- **必踩关键短语**：商业秘密（非公知+商业价值+保密措施）/ 与专利的差异（公开换保护 vs 保密换保护）/ 不正当竞争 / 停止侵害 / 赔偿损失。
- **落点句**："鼓励企业自主创新，维护公平竞争秩序。"
- **典型出处**：东城期末 Q10B。
- **反例提醒**：把"非公知技术信息"写成"侵犯专利权"——错。

#### S5-03 著作权独立于物权
- **起因→经过→结果**：物的所有人（赠与人/出卖人）转移物的所有权，受让人擅自使用作品权利。
- **材料触发词**：画作赠与、雕塑出售、复制、汇编、展览、出版。
- **必踩关键短语**：物权与著作权独立 / 转移所有权不转移著作权 / 复制权 / 改编权 / 汇编权 / 信息网络传播权。
- **落点句**："明晰物权与著作权边界，保护作者智力劳动成果。"
- **典型出处**：通州期末 Q13、东城期末 Q10D、朝阳一模 Q10。
- **反例提醒**：~~"未满 18 周岁不享有著作权"~~——错（与年龄无关）。

#### S5-04 著作权各项财产权辨别
- **起因→经过→结果**：他人未经许可对作品进行不同方式使用，引发不同侵权类型识别。
- **材料触发词**：未经同意拍成电影、印在宣传页、转载、改编、网络传播。
- **必踩关键短语**：复制权 / 改编权 / 汇编权 / 信息网络传播权 / 法定许可范围有限。
- **落点句**："精细化保护作品多元利用方式，激励创作。"
- **典型出处**：朝阳一模 Q10。
- **反例提醒**：把"小说改编为电影"误为"复制"——错（是改编）。

#### S5-05 平台主体责任 + AIGC 标识合规
- **起因→经过→结果**：平台/经营者使用 AI 生成内容未标注，或未尽合规审核。
- **材料触发词**：未标注 AI 生成、深度合成、平台规则、内容审核。
- **必踩关键短语**：标识合规义务（直播电商监督管理办法第 37 条等）/ 平台主体责任 / 行政违法行为。
- **落点句**："压实新业态合规义务，引导技术向善发展。"
- **典型出处**：东城一模 Q11。
- **反例提醒**：未标注 AI 不是著作权问题——是合规问题。

### 主线 6 — 消费者 + 劳动（共 5 个情境）

#### S6-01 经营者欺诈 / 三倍 / 食品药品十倍
- **起因→经过→结果**：经营者明知不符仍销售/虚假宣传 → 消费者主张退一赔三 / 退一赔十。
- **材料触发词**：欺诈、虚假宣传、明知、隐瞒、退一赔三、十倍赔偿。
- **必踩关键短语**：经营者欺诈成立 / 消费者权益保护法 / 三倍最低 500 元 / 食品安全法十倍 / 惩罚性赔偿。
- **落点句**："强化经营者责任，保障消费者权益，维护诚信交易秩序。"
- **典型出处**：海淀一模 Q18 案件 2。
- **反例提醒**：法律名必须写**消费者权益保护法**全名，写~~"消法"~~"保护法"扣分。

#### S6-02 经营者义务 / 个人信息保护
- **起因→经过→结果**：经营者公开/泄露/不当使用消费者个人信息（监控、聊天、订单等）。
- **材料触发词**：监控录像、聊天记录、消费记录、报复差评、营销骚扰。
- **必踩关键短语**：经营者保护个人信息法定义务 / 隐私权 / 个人信息权益 / 停止侵害 / 赔礼道歉 / 损害赔偿（含精神损害）。
- **落点句**："构筑数字时代消费者人格权保护防线。"
- **典型出处**：海淀一模 Q18 案件 1。
- **反例提醒**：不要把消费者主张写成"客观题答案/题肢判断"。

#### S6-03 消费者其他权利 / 知情权 / 自主选择权
- **起因→经过→结果**：经营者搭售、捆绑、强制升级、设默认勾选等侵犯自主选择/知情权。
- **材料触发词**：捆绑、强制升级、默认勾选、强制接受。
- **必踩关键短语**：消费者知情权 / 自主选择权 / 公平交易权 / 经营者真实告知义务。
- **落点句**："维护消费者主体地位，限制经营者优势滥用。"
- **反例提醒**：知情权 ≠ 受教育权（不能写错权利名）。

#### S6-04 违法解除劳动合同 / 赔偿金 vs 经济补偿
- **起因→经过→结果**：用人单位以"旷工""不胜任"等单方解雇，事实/程序违法。
- **材料触发词**：未签书面合同、口头承诺、单方解雇、旷工、补偿。
- **必踩关键短语**：事实劳动关系受保护 / 违法解除 / 赔偿金（=经济补偿×2）/ 仲裁前置 / 不服仲裁 15 日内起诉。
- **落点句**："保护劳动者合法权益，规范用人单位用工行为，构建和谐劳动关系。"
- **典型出处**：东城期末 Q11。
- **反例提醒**：~~"行政复议"~~——劳动争议不走行政复议；~~"经济补偿"~~——违法解除给的是赔偿金。

#### S6-05 劳动合同基础 / 试用期 / 二倍工资
- **起因→经过→结果**：未签书面合同满月 / 试用期超法定上限 / 试用期任意解雇。
- **材料触发词**：未签合同、试用期、转正、按月支付。
- **必踩关键短语**：建立劳动关系一个月内书面 / 二倍工资差额 / 试用期上限（1月/2月/6月）/ 试用期解雇有限定情形。
- **落点句**："严格用工书面化要求，规范试用期管理。"
- **反例提醒**：把"试用期"等同于"任意解雇"——错。

### 主线 7 — 程序救济（共 6 个情境）

#### S7-01 行政诉讼举证倒置 vs 民事一般规则
- **起因→经过→结果**：原告对行政机关具体行政行为不服 vs 民事侵权当事人争议。
- **材料触发词**：行政机关、具体行政行为、规范性文件、被告举证。
- **必踩关键短语**：行政诉讼举证责任倒置（被告即行政机关举证作出行为的证据和依据）/ 民事侵权"谁主张谁举证"/ 督促依法行政 / 平衡当事人地位。
- **落点句**："合理分配诉讼风险，督促依法行政，平衡公私权益。"
- **典型出处**：石景山一模 Q18。
- **反例提醒**：民事一般侵权写"举证责任倒置"是错。

#### S7-02 民事过错推定 / 建筑物悬挂物脱落
- **起因→经过→结果**：建筑物或附属物脱落致他人受损。
- **材料触发词**：窗户、广告牌、空调外机、坠落、砸中。
- **必踩关键短语**：过错推定 / 受害人举证损害+因果关系 / 所有人/管理人/使用人证明无过错 / 一般侵权与特殊侵权的区别。
- **落点句**："督促管理人履行注意义务，预防危险源造成损害。"
- **典型出处**：石景山一模 Q18 案三。
- **反例提醒**：不要写"无过错"——是过错推定。

#### S7-03 多元解纷 / 诉讼调解
- **起因→经过→结果**：法院在审理过程中促成当事人和解，纠纷以调解结案。
- **材料触发词**：调解、和解、合议庭斡旋、第三方调停。
- **必踩关键短语（必须用四字"诉讼调解"）**：诉讼调解 / 推动纠纷实质性化解 / 节约司法资源 / 减少诉讼对创新的干扰 / 以和为贵 / 和谐价值观。
- **落点句**："构建多元化纠纷解决体系，实现案结事了。"
- **典型出处**：朝阳一模 Q18 案 1、东城一模 Q18 案 1。
- **反例提醒**：写"调解"不写"诉讼调解"少 1 分。

#### S7-04 行政公益诉讼 + 检察建议
- **起因→经过→结果**：检察院发现行政机关不依法履职 → 立案 + 发检察建议 → 行政机关整改。
- **材料触发词**：检察院、立案、检察建议、督促整改、行政机关、不作为。
- **必踩关键短语**：行政公益诉讼 / 检察建议（非诉化解）/ 法律监督职能 / 维护国家利益和社会公共利益 / 弥补"原告缺位"。
- **落点句**："以公益视角监督行政，推动法律实施落地，提升治理效能。"
- **典型出处**：丰台期末 Q18。
- **反例提醒**：写~~"维护社会主义法治体系"~~（必修三化）。

#### S7-05 仲裁 vs 诉讼 / 仲裁协议优先
- **起因→经过→结果**：当事人达成仲裁协议后一方仍诉至法院；法院依协议驳回起诉。
- **材料触发词**：仲裁协议、协议管辖、约定争议解决方式。
- **必踩关键短语**：仲裁协议 / 一裁终局 / 协议优先 / 法院尊重当事人意思自治。
- **落点句**："保障商事自主，分流司法资源。"
- **反例提醒**：仲裁裁决一般不能上诉（一裁终局）。

#### S7-06 个人债务集中清偿 / 失信被执行人
- **起因→经过→结果**：法院将债务人列入失信名单 → 试点个人债务集中清偿 → 保留生活必需品 → 履行后解除惩戒并出具修复证明。
- **材料触发词**：失信被执行人、个人破产、信用修复、保留生活必需品。
- **必踩关键短语**：失信名单（执行措施）/ 个人债务集中清偿试点 / 保留生活必需品（执行豁免）/ 信用修复机制 / 平衡债权人和债务人利益。
- **落点句**："创新司法实践，平衡债权人保护与债务人生存发展。"
- **典型出处**：房山一模 Q11。
- **反例提醒**：~~"失信名单是违约责任承担方式"~~——错。

## 第二部分：选择题情境（按 17 类陷阱模式组织）

每条情境格式：
- **题干情境**（一句话还原）
- **正确判断方向**（怎么选对）
- **典型错项陷阱**（怎么排除）
- **法律边界**（不要超纲也不要必修三化）
- **典型出处**

---

### C1 抵押 vs 质押客体辨别
- **题干情境**：把动产、不动产、股票、债权、专利等罗列，问能否抵押或质押。
- **正确方向**：动产抵押或权利质押才对（如股票应质押）；动产质押需交付；不动产抵押需登记。
- **错项陷阱**：把"股票质押"写成"股票抵押"；把房屋写成"质押"。
- **法律边界**：不纠缠"流质条款效力"等法考细节。
- **典型出处**：朝阳一模 Q10、东城期末 Q10A。

### C2 著作权 vs 物权独立
- **题干情境**：物的赠与/出售后，受赠/受让人擅自展览、复制、汇编。
- **正确方向**：物权与著作权独立；著作权基于作品独创性自动产生，与作者年龄无关。
- **错项陷阱**：~~"未满 18 周岁不享有著作权"~~；~~"赠与同时转让著作权"~~。
- **法律边界**：不纠缠"职务作品"细节。
- **典型出处**：通州期末 Q13、东城期末 Q10D。

### C3 商业秘密 vs 专利权 vs 不正当竞争
- **题干情境**：非公知技术信息被破译并销售。
- **正确方向**：非公知技术信息属商业秘密；破译并销售属不正当竞争行为。
- **错项陷阱**：~~"侵犯专利权"~~（专利已公开，不存在"破译"问题）。
- **法律边界**：商业秘密三要件（非公知+商业价值+保密措施）。
- **典型出处**：东城期末 Q10B。

### C4 失信被执行人 / 个人破产试点
- **题干情境**：法院列失信名单 + 保留生活必需 + 履行后解除惩戒。
- **正确方向**：法院创新司法实践，平衡债权人和债务人利益。
- **错项陷阱**：~~"失信名单是违约责任"~~（错——是执行措施）；~~"保留面包车是用益物权"~~（错——是执行豁免）。
- **法律边界**：个人破产仍属试点，不展开法考细节。
- **典型出处**：房山一模 Q11。

### C5 一般侵权举证 / 过错推定 / 无过错
- **题干情境**：肖像权侵权 / 建筑物悬挂物脱落 / 产品责任。
- **正确方向**：一般侵权"谁主张谁举证"；建筑物悬挂物为过错推定；产品责任为无过错。
- **错项陷阱**：把一般侵权说成"举证责任倒置"。
- **法律边界**：不纠缠医疗侵权细分。
- **典型出处**：房山一模 Q10、海淀一模 Q11。

### C6 网络平台"通知—删除"+ 平台责任
- **题干情境**：直播主播 + 平台共同传播侵权内容。
- **正确方向**：平台收通知后未删→连带责任；著作权不能对抗肖像权/隐私权。
- **错项陷阱**：~~"主播享有著作权可公开发表"~~。
- **法律边界**：不纠缠"红旗规则"完整内容。
- **典型出处**：房山一模 Q10。

### C7 婚姻家庭财产 + 家务补偿
- **题干情境**：婚后稿酬归属 + 家务承担差异 + 离婚分割。
- **正确方向**：婚后所得财产收益共有；著作权人身权专属；家务补偿"谁主张谁举证"。
- **错项陷阱**：~~"家务补偿是举证责任倒置"~~。
- **法律边界**：第 1088 条家务劳动补偿条文。
- **典型出处**：海淀一模 Q11。

### C8 宅基地 + 共同债务 + 用益物权处分
- **题干情境**：婚前宅基地 + 婚后建房 + 装修借款。
- **正确方向**：建造取得不需登记；家庭日常生活借款共同债务；宅基地不得擅自处分。
- **错项陷阱**：~~"宅基地建房需登记"~~；~~"用于装修是个人债务"~~。
- **法律边界**：不纠缠"宅基地能否流转"政策细节。
- **典型出处**：东城一模 Q10。

### C9 AIGC + 数字人格权
- **题干情境**：未经同意采集人脸/声音生成数字人 + 未标注 AI。
- **正确方向**：声音权益参照肖像权；标识违规属合规义务；救济用"停止侵害+消除影响+赔礼道歉"。
- **错项陷阱**：~~"恢复原状"~~（人格权救济用词不当）；~~"是著作权问题"~~（实为合规义务）。
- **法律边界**：参考《直播电商监督管理办法》第 37 条标识义务。
- **典型出处**：东城一模 Q11、房山一模 Q10。

### C10 遗赠扶养协议 vs 遗嘱
- **题干情境**：遗嘱在前 + 遗赠扶养协议在后 + 受扶养人已履行。
- **正确方向**：遗赠扶养协议 > 遗嘱继承/遗赠 > 法定继承。
- **错项陷阱**：~~"遗嘱在先效力优先"~~。
- **法律边界**：协议双务有偿性。
- **典型出处**：通州期末 Q12。

### C11 合同形式自由 / 不可抗力 / 商标驳回救济
- **题干情境**：买卖合同口头约定 / 误伤致损 / 商标驳回。
- **正确方向**：买卖合同口头有效；过失致损不是不可抗力；商标驳回应申请商标评审复审。
- **错项陷阱**：~~"未签书面无效"~~；~~"误伤是不可抗力"~~；~~"商标驳回可行政复议"~~。
- **法律边界**：行政复议与商标复审是不同程序。
- **典型出处**：海淀一模 Q10。

### C12 劳动关系 / 合同 / 争议处理
- **题干情境**：未签书面合同 + 口头承诺 + 单方解雇。
- **正确方向**：事实劳动关系受保护；违法解除给赔偿金；劳动争议仲裁前置。
- **错项陷阱**：~~"未签合同则无劳动关系"~~；~~"行政复议"~~。
- **法律边界**：劳动法与劳动合同法各有侧重。
- **典型出处**：东城期末 Q11。

### C13 隐私权 / 个人信息 / 经营者义务
- **题干情境**：商家公开消费者监控/聊天信息报复差评。
- **正确方向**：经营者侵权成立；救济含精神损害赔偿；经营者负保护个人信息法定义务。
- **错项陷阱**：~~"消费者无名誉权"~~；~~"商家行使表达自由"~~。
- **法律边界**：隐私权与个人信息权益相邻但不同。

### C14 安全保障义务边界
- **题干情境**：经营场所内顾客自伤起诉。
- **正确方向**：经营者无过错则不担责；个人责任意识。
- **错项陷阱**：~~"基于公平原则承担部分责任"~~。
- **法律边界**：民法典第 1198 条界定义务范围。
- **典型出处**：丰台一模 Q20。

### C15 行政诉讼举证倒置 vs 民事举证一般规则
- **题干情境**：被告是行政机关 vs 被告是私主体。
- **正确方向**：行政诉讼倒置；民事一般"谁主张谁举证"；过错推定为民事特殊。
