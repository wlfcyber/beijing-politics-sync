# EXTRACTION_TRIAL — Round 2 小样本试跑

时间：2026-05-04
样本：3 套关键试卷的代表性文件（2024 东城二模、2025 朝阳一模、2026 东城一模）

## 解析栈最终选型

| 文件类型 | 工具 | 状态 |
|---|---|---|
| `.doc`（老 Word） | macOS `textutil -convert txt` | ✓ |
| `.docx` | `python-docx`（含表格遍历，需 dedup 合并单元格） | ✓ |
| `.pptx` | `python-pptx`（含 text_frame + table） | ✓ |
| 数字版 PDF | `pdfplumber.extract_text` + `extract_tables` | ✓ |
| 扫描版 PDF | `pdfplumber.to_image(resolution=200)` → `rapidocr-onnxruntime` | ✓ |

无需 brew、无需 LibreOffice、无需 tesseract 二进制。Python 库（python-docx 1.2.0 / python-pptx 1.0.2 / pdfplumber 0.11.8 / rapidocr-onnxruntime）均已安装到用户 site-packages。

## 6 个测试结果

### Test A — 2025 朝阳一模 阅卷总结19题 合同欺诈.doc
- 工具：`textutil -convert txt`
- 字符数：~7591
- 提取内容：题面 + 阅卷前参考答案 + **评分细则与答案变通说明**（逐分给分点 + "材料3个条件必须写全才得分"硬卡口）+ 阅卷反馈（病象/病因/药方）
- 证据等级：**L1**（阅卷总结，逐题给分硬规则）
- 关键发现：2025 朝阳一模 19 题"合同欺诈"是选必二主观题（合同/可撤销/消费者权益保护法）。阅卷总结质量超预期，含"必须有'消费者法律'字样、'保护法'不给分"等极硬性卡口。

### Test B — 2024 东城二模 17(1).docx
- 工具：`python-docx`
- 字符数：~2726
- 提取内容：题号、参考答案、试题分析、阅卷细则（"选3个给3分"等）、学生问题、教学建议、典型错误
- 证据等级：**L1**（阅卷总结）
- ⚠️ **题号陷阱**：17(1) 是《经济与社会》盐碱荒滩 — **不是选必二**。识题阶段必须按题面内容过滤，不能靠题号认题。
- 副发现：python-docx 表格遍历会把合并单元格内容重复输出，处理时需按 cell._tc 去重。

### Test C — 2026 东城一模 评标 19（1）.pptx
- 工具：`python-pptx`
- 字符数：204；4 张幻灯片
- 提取内容：方法、替代知识、给分点
- 证据等级：**L1 单题级**（评标细则勿传，最高密度）
- ⚠️ **题号陷阱**：2026 东城一模 19(1) 是选必三调查研究法 — **不是选必二**。

### Test D — 2026 东城一模 试卷.pdf

### Test G — 2026 东城一模 补充材料/26东城一模细则.pdf
- 工具：`pdfplumber`
- 字符数：5285（16 页，数字版）
- 内容：跨模块汇总细则（必修四文化/哲学 + 经济与社会 + 选必二 + 选必三）。**选必二专属内容需按页定位**。

### Test F — 2026 东城一模 评标 18+选择7、13.pptx
- 工具：`python-pptx`
- 字符数：1512（16 张幻灯片）
- 提取内容：18 题三案例完整细则（诉讼调解/惩罚性赔偿/恶意诉讼/驳回滥用程序权利）+ 替代知识 + **"不重复给分"边界规则** + 4 档样卷（满分/7分/6分/3分/2分）+ 教学建议
- 证据等级：**L1 单题级**（顶级证据）
- ⚠️ **文件名陷阱**：18 是选必二，但 7/13 不是（7 是选必三逻辑题；13 是必修二货币政策）。必须按题号 + 题面双重判断后才能拆开吸收。
- 关键发现：18 题是 2026 东城一模选必二主观题，案例一/二/三结构涵盖 调解 + 知识产权 + 不正当竞争 + 诚信原则 + 司法秩序，是高频框架主干样本。

### Test H — OCR 兜底（rapidocr）
- 输入：Test E 失败的 16 页扫描细则首页
- 渲染：pdfplumber `to_image(resolution=200)` → 2339×1653 PNG
- OCR：rapidocr-onnxruntime 0.3 秒解出 7 文本块，置信度 0.99-1.00
- 中文识别准确，含水印（扫描全能王）也识到

## 关键陷阱清单（写进解析脚本必防）

1. **题号不能信文件名**：`17(1).docx`（实为经济与社会）、`19(1).pptx`（实为选必三逻辑）、`18+选择7、13.pptx`（混 3 个模块）— 必须按题面判断。
2. **细则.pdf 主目录版常为扫描件**，补充材料版常为数字版。同套要双查。
3. **docx 表格合并单元格会重复**，需 dedup。
4. **PPTX 评标"勿传"目录**是 L1 单题级最高证据，不可漏。
5. **2025 阅卷总结 .doc 文件**含极硬性给分卡口（"必须有'消费者法律'字样"等），等级 L1。
6. **跨模块细则 PDF**（如 26东城一模细则.pdf 补充材料版）：选必二段落需按页定位，不能整文照单全收。

## Round 2 结论

解析栈跑通，没有阻塞。Round 3 可以全量铺开。

- 加表格 dedup、错误处理、ledger 写入：~10 分钟总用时
