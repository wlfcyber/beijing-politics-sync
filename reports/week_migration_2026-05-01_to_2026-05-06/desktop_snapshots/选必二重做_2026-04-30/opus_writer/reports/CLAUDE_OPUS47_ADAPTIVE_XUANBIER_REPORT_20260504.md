## 1. 总判定

**结论：CONDITIONAL PASS — 主干骨架可作本地候选继续推进，但有三处必须修后才能进学生版。**

- **可进**：一核 + 三问 + 四步 是真正能教、能迁移的部分，建议固化为主干。
- **必修**：
  1. 二线中"德治价值"线条过虚，存在宏观法治口号化风险，需改为可操作表达。
  2. 八域作为前台主干会让学生记忆负担过重，必须压回五域作背诵版、八域降为后台索引。

## 3. revise（必须修改）

**3.1 二线表达需要改写**
- 当前："权利保护与权利边界；法治规则与德治价值"
- 问题：第二条线"法治规则 / 德治价值"过虚，会引诱学生写"促进社会和谐""彰显法治精神"等空话，正是宏观法治口号化的入口。
- 建议改为：
  - 线 1：**权利保护 ↔ 权利边界**（民事维度，覆盖合同、侵权、人格权、物权、家庭、劳动、IP）
  - 线 2：**规则适用 ↔ 程序救济**（程序维度，覆盖起诉、举证、判决、执行、公益诉讼）
- "德治"不单列；当意义题需要写到德治时，作为线 1 的边界来源（公序良俗 / 诚信 / 公平）出现，不作独立结构线。

**3.2 四步的第四步需要拆 visible**
- 当前："落责任 / 效力 / 归属 / 程序 / 价值"五并列，学生看不出该选哪个。
- 建议改为**按题型选落点**（嵌进命题路径表，详见 §7）：
  - 合同题落"效力 + 责任"
  - 侵权题落"责任承担方式"
  - 物权 / 继承题落"归属"
  - 程序题落"程序节点 + 救济效果"
  - 评析 / 意义题落"本案结果 → 制度意图"

**3.3 五域 / 八域结构必须做学生版 vs 索引版分层**
- 学生版只背 5 域；八域只在 Codex 后台和教师备课档里出现。
- 详见 §6。

**3.4 命题路径草案必须分型**
- 当前所有题的命题路径都写成"事实→规则→结果"模板，等于没分型。详见 §7。

## 5. reject（应删除或降级）

**5.1 法考化倾向（必须挡在主干外）**
- ❌ 系统讲解 用益物权 / 担保物权 / 所有权 三分体系 — 高中题不需要分类背诵，按"归属 + 排除妨害"两个动作即可。
- ❌ 要约 / 承诺 / 合同成立 / 效力 / 履行 五阶段全套 — 仅保留"是否构成意思表示 + 是否守信履行"两个动作（足以解 2024 石景山 17、2026 丰台 11、2026 海淀 18 等高频合同题）。
- ❌ 侵权责任的"一般侵权 / 过错推定 / 无过错"三分体系作为前台知识 — 学生版只需识别"是否有过错 + 由谁举证"。
- ❌ 上诉期、举证期等具体天数 — 2026 朝阳期末 11 选项 D 涉及上诉期 10 日，是命题人设的法考陷阱，学生只需知道"上诉有时限、不能默认 10 日"，不应反向背全套天数。

**5.2 宏观法治口号化倾向（必须挡在意义题外）**
- ❌ "法治与德治相结合"作为意义题的独立结论 — 必须先回本案权利义务关系，再带出。
- ❌ "构建全链条""完善治理体系""以法治力量护航高质量发展" — 这些是题面里命题人埋的诱饵选项词（2024 丰台 6、2024 丰台 5、2026 丰台 11 都有类似干扰），学生版应明确**不要在答题中复用**。
- ❌ "公正司法的生动实践""彰显司法温度""法治精神薪火相传" — 写则视为没回到本案。

**5.3 结构过细**
- ❌ 八域作为前台学生记忆单元 — 降为后台索引。
- ❌ "生态公益、新业态与法治价值"作为独立第八域 — 仅 2 道题，不足支撑独立域，降为开放容器。
- ❌ 第四步的"五并列落点"作为统一动作 — 改为按题型选落点。

## 6. 五域 / 八域建议

### 6.1 学生可背版（5 域 — 进学生手册）

按"民事关系类型"切，每域配一个抓手动作和一个常错点。学生只背这 5 个：

| 域 | 抓手动作 | 高频常错点 |
|---|---|---|
| ① **合同与消费** (25) | 看意思表示是否成立、看履行是否守信、看消费者是否处弱势 | "扩大消费规模""全链条治理"诱饵 |
| ② **侵权与人格权** (17) | 看过错 / 看损害 / 看因果 / 看免责 | 把场地出租者写成无过错责任 |
| ③ **婚姻家庭与财产继承**（合并旧 6+7，21 题） | 看身份关系、看共同财产、看监护与扶养、看物权归属 | 让未成年人"自行决定" |
| ④ **劳动与职业** (13) | 看劳动关系是否建立、看用人单位义务、看赔偿与解除条件 | 把"录用通知撤回"写成"侵犯平等就业权" |
| ⑤ **知识产权与竞争秩序** (15) | 看作品是否独创、看是否混淆、看是否擅自使用 | 把虚拟数字人套商标法 |

**程序救济**（20 题）不作第六域，**作横切方法论**：所有 5 域题在"对规则—落结果"步都可能要走起诉/举证/判决/执行/公益诉讼路径。这样 20 道程序题既不被丢掉，也不在学生记忆里再加一域。

**生态公益、新业态**（2 题）作开放容器：用 5 域里最近的（多数为程序救济横切 + IP/侵权）解决。

### 6.2 后台索引版（8 域 — 留 Codex 索引、教师备课档用）

保留当前 8 域不动，作为 Codex 题库标签和教师题型映射用：
1. 合同交易与消费者保护
2. 纠纷解决与程序救济
3. 人格权与侵权责任
4. 知识产权与竞争秩序
5. 劳动就业与职业边界
6. 婚姻家庭与继承扶养
7. 财产权、物权与相邻关系
8. 生态公益、新业态与法治价值

**学生版 → 索引版映射规则**：
- 学生版 ③ = 索引版 6 + 7
- 学生版"程序救济"横切 = 索引版 2
- 学生版"开放容器" = 索引版 8
- 其余一一对应。

这样Codex 端八域统计不丢精度，学生端只背 5 域不超载。

## 7. 命题路径建议（关键修订）

### 7.1 当前问题
现在所有题的命题路径都写成同一段："命题人以 X 切入 Y 域，设问路径是事实→规则→结果"。这等于没写。学生看不出"为什么这道题是表格题不是评析题""为什么这题给 7 分不是 6 分"。

### 7.2 建议结构：四要件 × 五题型矩阵

每题的"命题路径"按四要件填，按题型选落点动作。

**四要件（每题必填）**：
- **命题目标**：命题人想让学生输出什么动作（不是"考查什么知识点"，而是"做什么动作"）
- **载体选择**：为什么用这个材料（司法案例 / 法条对照 / 表格 / 政策文件 / 生活情境）
- **设问路径**：从材料到答题的最短链
- **细则奖励动作**：阅卷会给分的关键动作（即"踩到才给分"的事）

**五题型 × 落点动作**：

| 题型 | 命题目标 | 设问路径骨 | 细则奖励动作 |
|---|---|---|---|
| **A. 选择题—绝对化排除型** | 识别题肢中的"绝对化 / 偷换主体 / 错位关系" | 题肢比对干扰词 | 找到"全链条""避免投诉""自行决定""无过错责任"等诱饵 |
| **B. 案例分析—责任落点型** | 把事实摆进法律关系，定责任承担 | 事实 → 关系 → 规则 → 责任方式 | 写明"谁对谁、依何规则、承担何种责任方式" |
| **C. 表格补全型**（裁判理由 / 意义双栏） | 模仿示例，把"裁判理由"写成"规则 + 适用"，把"意义"写成"权利保障 + 制度意图"（不是空价值） | 看示例 → 比照案例 → 双栏对齐 | 裁判理由栏出现具体规则名 + 适用要件；意义栏先回本案 |
| **D. 评析型**（"对…的观点 / 做法谈认识"） | 拆观点 → 找规则依据 → 给结论 → 落价值 | 拆 → 对 → 判 → 升 | 必须出现"该观点的合理 / 不合理之处分别是…，因为…" |
| **E. 程序指导型**（"假如你是法官 / 给当事人专业指导"） | 走完一条程序救济路径 | 当事人处境 → 可走的程序节点 → 节点效果 | 至少写出 2 个程序节点 + 各节点的法律效果 |

### 7.3 嵌入位置
每题解释链顶部插入一段**结构化命题路径**（≤ 80 字），紧跟"价值后置硬规则"提醒：

```
[命题路径]
目标：A 选择题—绝对化排除
载体：消费者权益保护条例 + 题肢
路径：题肢 ① ② ③ ④ → 比对条例关键词 → 排除"避免投诉""确保扩大"
奖励动作：识别绝对化干扰词
[价值后置]
答此题不需上升到法治价值。
```

这样学生第二次见同型题能复用动作，而不是再被"事实→规则→结果"统一模板淹没。

## 8. Confucius 学会性风险（零基础学生卡点）

| 卡点 | 现象 | 改法 |
|---|---|---|
| **K1 看不出"具体法律关系"** | 学生写"这是法律问题"但说不出主体—客体—内容三要素 | 一核操作化为口诀："谁 vs 谁，关于什么，凭什么" |
| **K2 不会"对规则条件"** | 跳过条件直接写结论"违法 / 合法" | 在四步第三步插入"逐条件比对清单"模板 |
| **K3 意义题滑回口号** | "彰显法治精神""推进国家治理" | 价值后置硬规则 + 模板"在本案中→在制度上→在社会上"三阶递进 |
| **K4 程序题不知道往哪走** | 看到"假如你是法官 / 律师"就空写公平正义 | 把程序救济做成横切方法论，给 5 个程序节点卡片（起诉 / 举证 / 调解 / 判决 / 执行 / 公益诉讼）让学生选 2~3 个 |
| **K5 表格题对不齐栏目语气** | "裁判理由"栏写成大段叙事 | 给一个示例栏 + 字数上限（≤ 50 字）+ 关键词卡（必须含"依据…规定 + 因…构成…"） |
| **K6 评析题只会一边倒** | 全部说对或全部说错 | 模板"合理之处：__；不合理之处：__；综合判断：__" |
| **K7 IP 题混淆法律部门** | 把著作权题套商标法（2026 朝阳期末 11 干扰项 A） | 5 域 ⑤ 加"先定法律部门"前置一步：作品 → 著作权；标识 → 商标；不正当竞争 → 反不正当竞争法 |
| **K8 八域记不全** | 学生背完忘域名 | 5 域上墙、8 域不上墙；教师备课档 / Codex 后台才出现 8 域 |
