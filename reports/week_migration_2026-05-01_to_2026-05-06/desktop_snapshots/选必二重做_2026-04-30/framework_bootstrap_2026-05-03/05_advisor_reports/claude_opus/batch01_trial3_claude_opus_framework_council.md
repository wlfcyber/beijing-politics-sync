# Claude Opus 4.7 Adaptive Framework Council Report

## 1. Learnability Diagnosis

当前 V0_SCAFFOLD 对零基础学生**不可直接使用**，三个具体梗阻：

- **节点过密**：一核/二线/三问/四步/五域 ≈ 15 个一级节点。学生在 35-40 分钟答题压力下无法同时调用，必然退化为"背五域名称"。
- **概念交叠**：三问（判什么/凭什么/有什么意义）与四步（定主体→争点→要件→责任）部分同义重复，学生分不清"答题时先用三问还是先用四步"。
- **缺动作锚点**：五域是"题在哪一片"，但三道题（B01-Q1/Q2/Q3）的 rubric_rewarded_action 全是"动作"——匹配要件、归纳关系、分层意义——五域并不直接对应任何一个 rubric 奖励点。
- **价值层无落点**："以法律为准绳/以德治价值"高度抽象，B01-Q3 提示"价值必须锚定具体法律关系"，但当前框架没把这条规则讲出来。

**核心诊断**：当前框架是"教师视野的知识地图"，不是"学生答题的操作手册"。

## 2. Student-Usable Trunk

基于三题 formal/reference 共同呈现的高频动作，建议主干高频层为四步动作链（与 Codex C1 一致）：

1. **定关系**：谁与谁、什么法律关系（劳动/著作/扶养继承/侵权/合同……）。
2. **找争点**：争议点是"是否成立""是否有效""是否侵权""谁承担"中的哪一种。
3. **套要件**：把材料事实切片对照法定要件（B01-Q1 的"三个从属性"是该步教学最佳样板）。
4. **落结果**：责任/效力/程序的具体落点（赔偿、消除影响、确认有效、仲裁支持……）。

证据强度：B01-Q1（formal）、B01-Q3（formal）两题 rubric_rewarded_action 均整段落在这条链上，B01-Q2（reference）路径相符但证据弱。**主干可立，但 N=2 formal，仍需更多批次锁定。**

## 3. Proposition Path For Students

学生应学会"看题先拆三段"，但**不背命题路径名称**，只学一个判别动作——**找题面的"载体—问点—收束"**：

- **载体**：题给的是"典型判例""新现象案例"还是"扶养/家庭场景"？（决定题域容器）
- **问点**：题在问"成立/侵权/有效/谁承担"中的哪个？（决定争点）
- **收束**：题最后一问问"理由—意义—价值"中的哪种？（决定要不要进价值层）

三题的 question_path 表面不同，但都可还原成这三段。**这一层是"看题层"，不是"答题层"**——只用来帮学生把题切成可处理块，不进考点。

## 4. Open Containers

明确归入开放容器（不进主干背诵）：

- **AI/虚拟数字人/新技术知识产权**：仅 B01-Q2（reference_only），证据等级不足，但命题路径重要。容器内只列"未经许可使用 + 不正当竞争 + 创新与秩序意义"三个挂点，等 formal 同类题出现再讨论是否升主干。
- **五域分类本身**：从主干降为"题在哪片"的容器索引，学生**不需要默写五域**，只需在定关系时知道"这是劳动 / 著作 / 继承"即可。
- **新就业形态劳动关系（B01-Q1）/ 遗赠扶养（B01-Q3）**：单点 formal，建议作为"主干样板题"放在主干旁的示范容器，而非另立子主干。

## 5. Risks Of Abstraction Or Misuse

| 节点 | 风险 | 建议 |
|---|---|---|
| "以事实为根据，以法律为准绳" | 学生易写成万能开头空话 | 主干不出现该口号；只在第3步"套要件"时作为方法论提示出现 |
| "权利保护与权利边界"二线 | 与"套要件"重复，学生易把侵权题写成"边界论" | 降为价值层注释，不进答题动作 |
| "法治与德治"二线 | B01-Q3 出现德治价值，但若学生记成口号会答成政治课空话 | 改写为"价值收束必须先点出本案具体法律关系再上升" |
| 三问 vs 四步 | 同时背两套学生会混 | 删三问，保留四步动作 |
| "价值收束必须锚定具体法律关系" | "锚定"二字过书面 | 学生版改为"先说本案是什么法律关系判了什么，再说意义" |

## 6. Proposed Student-Facing Structure

建议 V1_CANDIDATE 学生版三层（背诵层 ≤ 4 项）：

```
【看题三段】（不背，只用来切题）
载体 → 问点 → 收束

【答题四步】（背，且每步配一句口令）
1. 定关系 —— 这是谁和谁、什么法律关系
2. 找争点 —— 是否成立 / 是否有效 / 是否侵权 / 谁担责
3. 套要件 —— 材料事实逐条对法定要件
4. 落结果 —— 责任 / 效力 / 程序的具体落点

【价值收束规则】（背一句即可）
先点本案法律关系和判决，再上升到制度/秩序/价值。
不许跳过法律直接喊价值。

【题域容器】（开放，不背）
合同消费者劳动 | 物权相邻继承家庭 | 人格权侵权 | 知识产权不正当竞争 | 纠纷解决/生态公益/新业态
```

学生需主动记忆的只有 **4 步 + 1 收束规则 = 5 项**，比 V0 的 15 项节点轻一半以上，且每项都对应可见的 rubric 动作。

## 7. Evidence/Placement/Proposition-Path Risks

- **样本量风险**：N=3 题、formal=2，对"主干降五域升四步"这一结构性动作偏单薄。建议在 BATCH02-03 至少补 6 题 formal 才正式发布 V1。
- **题型偏差**：当前三题全为主观大题，**未覆盖选择题/小题**。命题路径在小题上是否仍然"载体—问点—收束"三段，需本地核验。
- **B01-Q2 污染风险**：reference_answer 不能进主干，但"新技术 + 不正当竞争"作为命题方向高度真实。若仅放开放容器，可能在课堂上被低估。**建议在容器中标注"高真实度待补强"**，与一般容器项区分。
- **价值层证据**：B01-Q1（治理意义）+ B01-Q3（养老司法/核心价值观）支持"价值锚定法律关系"，但**两题价值收束方向不同**（治理 vs. 德治），单一规则是否能覆盖未来意义类题，需本地核验。
- **意义类题独立性**：当前未见"纯意义题"（只问意义不问判什么），此类题命题路径是否仍然要"先点法律关系"无证据，需后续批次验证。

## 8. Version Upgrade Criteria

V0 → V1 升级的硬门槛建议：

1. **主干节点门槛**：单节点须 ≥ 3 题 formal_or_scoring_source 命中，且至少跨 2 个题域容器（防止单一题域偏置）。
2. **开放容器升主干门槛**：≥ 2 题 formal 同向命中，且 rubric_rewarded_action 中能拆出独立动作。
3. **下沉门槛**：当前主干节点若连续 2 个批次未被 formal 命中，降级为容器。
4. **命题路径层冻结门槛**：跨批次累计 ≥ 10 题路径可被"载体—问点—收束"还原，方可定为框架公开层；否则保持内部分析层。
5. **学生背诵层硬上限**：永远 ≤ 6 项一级节点。超出即触发再压缩。
