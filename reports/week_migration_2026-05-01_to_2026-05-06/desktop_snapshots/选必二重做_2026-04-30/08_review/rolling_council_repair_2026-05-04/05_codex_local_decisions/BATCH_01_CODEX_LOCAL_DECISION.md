## 2. 本地总判定

Batch01 可以支持三类候选变化：

1. 法律小问入框闸门必须前置。
2. 学生答题主干应保留“一核二线三问四步五域”，但把价值线后置、把题型路径分流。
3. 当前“合同消费者劳动”域存在过宽问题，劳动与职业需要至少作为前台一级子域独立呈现；是否改写最终五域名称，必须等五个批次全部跑完后再定。

Batch01 不能支持三类定稿动作：

1. 不能仅凭本批把全书五域重命名为新五域。
2. 不能把程序救济定死为独立实体域或纯横切方法，只能先保留双身份：横切动作 + 归位容器。
3. 不能把含污染摘录、误入框题、候选答案题计入主干频次。

## 3. 对一核二线三问四步五域的裁决

### 3.1 一核

保留：

> 以事实为根据、以法律为准绳，通过具体法律关系中的规则适用实现定分止争。

在一核之前新增第 0 步：

> 先判是否为《法律与生活》法律小问。

入框三问：

1. 是否存在具体法律主体之间的权利义务关系。
2. 设问是否要求运用《法律与生活》作法律判断。
3. 是否落到责任、效力、归属、救济、权利边界或法律价值。

理由：GPT、Claude、ClaudeCode 均指出 Batch01 中存在关键词误入框风险。本地 pack 中 2025 石景山一模 Q8、2025 昌平二模 Q10、2026 丰台期末 Q11、2026 房山一模 Q11 等至少需要二次核验。

### 3.2 二线

保留二线层，不删除：

1. 权利保护与权利边界。
2. 规则适用与价值校准。

裁决理由：

- 用户明确要求保留“一核二线三问四步五域”。
- GPT 反对 Claude 早期“学生版只讲第一线”的建议，理由成立：价值线不能消失，只能降噪。
- Claude 交叉批判已经撤回“删除第二线”，同意改为“规则适用与价值校准”。

教学限制：

- 价值不作为答案开头。
- 价值只在意义/认识/作用/评析题中启动。
- 必须先回到本案法律关系、处理结果和权利边界，再上升到诚信、公平、公序良俗、核心价值观、市场秩序、劳动保护等。

### 3.3 三问

保留三问：

1. 判什么/怎么处理。
2. 凭什么。
3. 有什么意义。

不采纳“全部改为凭什么 -> 判什么 -> 有什么意义”。

裁决理由：Batch01 显示不同题型顺序不同。劳动关系认定题可先写三从属性再归纳关系；裁判依据题往往先点明支持/不支持再展开依据；意义题必须先回本案处理再升价值；选择题只保留压缩版判断和排错。

### 3.4 四步

保留四步，但修订第四步：

1. 定关系：谁与谁，发生什么法律关系。
2. 抓争点：争的是权利保护、权利边界、效力、责任、归属、救济还是价值认识。
3. 对规则：材料事实对应教材规则、条件、要件或程序要求。
4. 落结果：按题型落到效力、责任、归属、救济路径、支持/驳回、处理方案。

价值作为条件性后置，不再并列塞进第四步。问意义、认识、作用时再“收价值”。

### 3.5 五域

本批不定稿重命名五域。

当前框架仍保留五域层：

1. 合同消费者劳动。
2. 物权相邻继承家庭。
3. 人格权侵权。
4. 知识产权不正当竞争。
5. 纠纷解决生态公益与新业态。

本批临时修订：

- “合同消费者劳动”内部必须拆出两个前台子入口：合同与消费；劳动与职业。
- “纠纷解决生态公益与新业态”必须拆成后台标签：程序救济、公共利益/生态公益、新业态场景。
- 新业态不独立成域，只作为场景标签挂回实体法律关系。
- 程序救济保留双身份：横切动作 + 归位容器，等待全量题核验。

五域划分依据暂记为：

1. 主体关系类型：消费者经营者、劳动者用人单位、家庭身份关系、权利人侵权人、法院/仲裁/调解等。
2. 结果落点类型：责任、效力、归属、人格利益保护、知识成果/竞争秩序、程序救济。
3. 试题频次与学生迁移负担：学生前台必须少而稳，后台索引用于全量归档。
4. 命题路径：命题人从生活案例、法律条文、司法现象、治理场景出发，最终奖励的是事实规则匹配和具体处理，不是法考目录或宏观口号。

## 4. Accepted

以下进入 Batch01 后续生产规则：

1. 增加第 0 步“法律小问入框闸门”。
2. 关键词只能提示，不能决定入框或归域。
3. 主体关系前置识别进入四步第一步。
4. 选择题建立排错动作：绝对化、主体错位、关系错位、责任错位；程序错位先作为后台/域内补充，不在学生通用表里过载。
5. 劳动关系认定题保留“三从属性 + 材料事实匹配 + 劳动关系归纳”识别卡，但“三从属性”只作为劳动关系认定题硬卡，不作为劳动域唯一主干。
6. 价值后置：意义题先案后升。
7. 表格补全题加硬条件：题面必须出现参考示例/完成下表/空格栏目等真实表格结构。
8. 异常评分摘录隔离，不得用于主干升级。
9. reference answer 不推动主干。
10. 新业态作为场景标签，不单独做学生主干。

## 5. Revised

以下建议修订后采纳：

1. “法治规则与德治价值”不删除，改为“规则适用与价值校准”。
2. “合同消费者劳动”不再作为一个不分内核的大块教学入口，至少拆为合同与消费、劳动与职业两个前台子入口。
3. “主观题六环节链”只用于裁判依据题，不套所有主观题。
4. “主观题落点不超过 2 个”作为学生表达约束，但若正式细则分项超过 2 个，以细则为准；学生版必须标清主落点和附带价值，避免五项并列。
5. 程序救济不讲成诉讼法目录；只讲谁能主张、走什么路、要什么证据、产生什么效果。
6. 反不正当竞争暂作跨域识别：题面主导消费者受损时挂合同消费，主导经营竞争秩序或知识成果时挂创新竞争；最终归域待全量核验。
