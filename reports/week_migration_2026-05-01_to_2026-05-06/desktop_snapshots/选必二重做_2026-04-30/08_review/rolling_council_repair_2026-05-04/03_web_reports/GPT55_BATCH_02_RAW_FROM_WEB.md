# Batch 02 框架委员会意见

**总结论：本批次强支持“财产权、物权与相邻关系”和“婚姻家庭与继承扶养”成为稳定覆盖域，但不支持继续把二者合成一个学生前台大域。当前“物权相邻继承家庭”过宽，建议拆入“交易与财产”和“人身与家庭”两个前台域。**

本意见仅基于 Batch02 题卡摘要和可见细则摘录，不声称看过本地源文件、PPTX、DOCX、PDF 或正式阅卷原件。Batch02 共 20 题，主观题 4，选择题 16，证据类型为 official_answer_key 15、formal_or_scoring_source 5；所有题域归属、答案锁定、评分细则采分点和高频判断均为**需本地核验**。

---

## 1. 本批次支持的高频主干

### 1.1 财产权与物权主干：应进入前台候选

**结论：物权归属、物权变动、担保物权、共有财产、相邻关系、财产权边界，应进入学生主干候选。需本地核验。**

Batch02 中财产权、物权与相邻关系题约占一半，且选择题和主观题都有支撑。可提取为学生可操作动作：

1. **定主体关系：** 所有人、占有人、债权人、担保权人、不动产权利人、相邻权利人、业主共同体。
2. **抓争点事实：** 谁享有所有权，谁享有使用利益，登记或交付是否完成，抵押或质押是否成立，相邻方是否应提供便利。
3. **对规则条件：** 所有权、用益、担保、共有、相邻关系、物权变动公示规则。
4. **落结果：** 财产归属、优先受偿、不得对抗善意第三人、停止妨害、赔偿或提供便利。
5. **收价值：** 保护财产权，协调相邻利益，平衡个人利益与公共利益。

本批次强样本包括：房屋物权确认、机动车质押、土地经营权流转、一房二卖与登记、宅基地使用利益、共用暖气管路、机动车交付与抵押登记、老旧小区加装电梯相邻关系等。具体采分点均**需本地核验**。

### 1.2 相邻关系主干：不应只做物权小尾巴

**结论：相邻关系应作为“交易与财产”域中的高频小卡，不宜淹没在抽象物权概念中。需本地核验。**

本批次相邻关系至少出现两类：

1. **共用设施类：** 小区共用暖气管路，三楼业主擅自截断，争点是共有财产和擅自处分。
2. **加装电梯类：** 老旧小区加装电梯，争点是程序合法、相邻便利、是否实际影响采光、是否应停止妨害。

相邻关系的学生动作应是：

> 看是否相邻不动产；看是否必须利用或影响对方；看是否尽量避免损害；看是否按有利生产、方便生活、团结互助、公平合理处理；最后落到提供便利、停止妨害、赔偿或邻里和谐价值。

这里不能讲成完整物权法体系。学生需要的是“相邻权利边界”卡，不需要系统背不动产役权、所有权权能、占有保护等法考细项。

### 1.3 继承与遗赠扶养主干：应进入前台候选

**结论：遗嘱、法定继承、遗赠扶养协议、丧失继承权、继承份额、扶养义务，应进入“人身与家庭”主干候选。需本地核验。**

本批次继承题密度高，且材料样式稳定：

1. 被继承人死亡。
2. 是否有遗嘱或遗赠扶养协议。
3. 谁是法定继承人，谁虽非法定继承人但履行扶养义务。
4. 是否存在丧失继承权事由。
5. 财产最终归谁。

学生动作可压缩为：

> 先看身份关系，再看死亡时间，再看遗嘱或协议，再看法定继承顺序，再看扶养义务和丧失继承权，最后落财产归属。

本批次的遗赠扶养协议题反复出现，支持将其作为继承小卡。但不能扩展为民法继承编体系，只需抓“协议效力、扶养履行、财产归属、养老助老价值”。具体规则细节和答案均**需本地核验**。

### 1.4 婚姻家庭主干：应与继承同域，但不能与物权合成一个大域

**结论：夫妻共同财产、夫妻平等、家务补偿、未成年人监护、家庭义务，应进入“人身与家庭”主干候选。需本地核验。**

本批次家庭题不只是继承，还涉及：

1. 未成年人生活稳定和父母平等监护。
2. 夫妻共同财产与版权费收益。
3. 家务劳动补偿。
4. 夫妻债务或家庭财产清偿。
5. 婚姻法与民法典规定变迁的观点评析。

这些题共同体现：家庭关系题必须先定身份关系，再定财产或监护争点，最后回到平等、保护弱者、家庭责任和社会价值。

### 1.5 选择题排错主干：本批次进一步支持扩展为“六排错”

**结论：Batch02 的选择题支持选择题独立使用排错动作。建议在前次五排错基础上加入“权能价值错位”和“效力顺位错位”。需本地核验。**

此前可保留的五排错：

1. 绝对化。
2. 主体错位。
3. 关系错位。
4. 责任错位。
5. 程序错位。

Batch02 还反复出现两类新错项：

1. **权能价值错位。**
   例如把使用价值、交换价值、所有权、抵押权、质权、宅基地权益混写。需本地核验。

2. **效力顺位错位。**
   例如遗嘱、遗赠扶养协议、法定继承、登记、交付、判决生效之间的效力关系混写。需本地核验。

因此，Batch02 选择题学生动作建议为：

> 定主体，定关系，排绝对化，排主体错位，排关系错位，排责任错位，排程序错位，排权能价值错位，排效力顺位错位。

学生版可以压缩成一句：**选择题先看主体和关系，再排错位。**

### 1.6 价值后置：本批次强支持保留第二线

**结论：Batch02 强烈支持价值后置，也支持保留二线中的价值结构，但要改写为“规则适用与价值校准”。需本地核验。**

本批次主观题中，价值不是附属废话，而是有明确位置：

1. 遗赠扶养协议案：先确认协议效力和扶养履行，再谈养老助老、尊老敬老、社会主义核心价值观。
2. 物权编绿色发展题：先写绿色原则、权利界限、法定义务、责任救济，再谈个人利益与生态公共利益平衡。
3. 加装电梯相邻关系题：先写相邻关系规则和事实分析，再谈业主权益、邻里和谐、友善价值。

这说明“德治价值”不能删除，但必须受控。建议继续使用：

> 线一：权利保护与权利边界。
> 线二：规则适用与价值校准。

价值只在意义题、认识题、公共利益题、家庭伦理题中启动，不能替代法律判断。

## 2. 本批次暴露的框架缺口

### 2.1 当前“物权相邻继承家庭”过宽

**结论：该域不适合作为学生前台域名。需本地核验。**

本批次中，财产权物权相邻与婚姻家庭继承大致各占一半。二者虽然都可能涉及财产归属，但法律关系根基不同：

1. 物权相邻题：核心是物、权能、登记、交付、使用利益、优先受偿、相邻便利。
2. 婚姻继承题：核心是身份、家庭义务、监护、扶养、继承顺序、遗赠扶养协议。

若继续合成“物权相邻继承家庭”，学生会误以为所有财产题都按同一套规则处理，容易把继承问题做成物权归属题，把物权登记题写成家庭身份题。

### 2.2 关键词归域风险仍然存在

**结论：Batch02 继续支持第 0 步入框闸门。需本地核验。**

高风险样本：

1. **2026 海淀一模第 9 题。**
   题面是 12348 公共法律服务热线平台成绩单，选项更偏政府公共服务、善政、法治社会建设。它被归入婚姻家庭与继承扶养，疑似因出现合同、婚姻、继承等咨询类别而触发。是否属于《法律与生活》法律小问，**需本地核验**。

2. **2026 东城期末第 10 题。**
   题面为多案例判断，涉及商业秘密、消费者知情权、著作权、物权返还等多个部门，被归入财产权物权相邻。它更像综合民法选择题或跨域综合题，不能只因出现物权就归物权域。**需本地核验**。

3. **2025 东城期末第 12 题。**
   选项同时涉及合同、遗赠扶养、著作权、抵押质押担保，标为财产权物权相邻可能过窄。**需本地核验**。

4. **2025 海淀期中第 21 题。**
   评分摘录同时出现法律评析和外交内容，明显是混合模块。法律小问边界必须回本地细则核验。

### 2.3 综合民法选择题需要单独处理

**结论：不能把所有多部门民法选择题硬塞进某一个域。需本地核验。**

Batch02 至少出现两类综合选择题：

1. 以民法典为总背景，四个选项分别考合同、继承、知识产权、担保。
2. 以多个案例为背景，四个选项分别考知识产权、消费者保护、著作权、物权返还。

这类题的命题目标不是让学生归某一个域，而是让学生识别具体选项对应的法律关系。建议 Codex 增加后台标签：

> 综合民法判断题，逐项定关系，不按整题单域归位。

学生版只需掌握：

> 一题多域时，不先抢域名；逐项看主体、关系、规则、结果。

### 2.4 价值题存在宏观化风险

**结论：绿色发展、尊老敬老、核心价值观、邻里和谐可以写，但必须先回法律规则和本案事实。需本地核验。**

Batch02 的价值题并不空泛，细则摘录中往往先给法律依据、作用路径、事实分析，再给意义价值。风险在于学生容易直接写：

1. 促进绿色发展。
2. 弘扬传统美德。
3. 践行社会主义核心价值观。
4. 促进邻里和谐。
5. 实现公平正义。

这些表达只能放在末尾。学生主干应明确：

> 没有规则和事实匹配，价值句不给主干分。

### 2.5 程序救济以横切方式出现

**结论：程序救济在 Batch02 不是主域，但持续作为排错和结果落点出现。需本地核验。**

本批次中的法院、诉讼、判决、仲裁、法律援助、热线平台等元素不少，但大多服务于实体关系：

1. 物权确认之诉。
2. 继承诉讼。
3. 相邻关系停止妨害。
4. 家庭纠纷裁判。
5. 公共法律服务平台。

这支持此前判断：程序救济既要在五域中有“救济与公共利益”归位空间，也要作为横切动作贯穿各域。不能讲成诉讼法目录。

## 3. 五域归位是否合理，是否需要拆并改名

### 3.1 当前五域的问题

当前五域为：

1. 合同消费者劳动。
2. 物权相邻继承家庭。
3. 人格权侵权。
4. 知识产权不正当竞争。
5. 纠纷解决生态公益与新业态。

Batch02 显示，第二域内部至少有两个稳定子群：

1. **财产物权相邻群。**
2. **婚姻家庭继承群。**

这两个群都可高频，但不宜合成一个学生域。尤其继承题虽涉及财产归属，解题入口仍是身份关系和扶养义务；物权题虽可能涉及家庭房产，解题入口仍是物权规则和权利边界。

### 3.2 建议学生前台五域继续改为以下版本

建议 Codex 继续采用更稳的前台五域：

1. **交易与财产**
   覆盖合同、消费、债务、物权、担保、相邻关系、共有、财产归属。
   Batch02 支持将物权、担保、相邻关系放入此域。

2. **人身与家庭**
   覆盖人格权、侵权、婚姻家庭、监护、夫妻财产、家务补偿、继承扶养、未成年人保护。
   Batch02 支持将家庭、继承、遗赠扶养协议放入此域。

3. **劳动与职业**
   覆盖劳动关系、劳动合同、试用期、解除、补偿、新就业形态。
   Batch02 本身不主要验证此域，需结合 Batch01 和全量核验。

4. **创新与竞争**
   覆盖知识产权、不正当竞争、商业秘密、平台竞争、作品权利。
   Batch02 中的综合民法题和著作权选项提醒：此域不能被物权域吸收。

5. **救济与公共利益**
   覆盖诉讼、仲裁、调解、执行、公共法律服务、生态公益、绿色发展、公共利益保护。
   Batch02 中绿色发展、法律服务热线、法院判决、相邻纠纷救济均可回流至此域或作为横切动作。

### 3.3 后台八域继续保留

后台八域建议不删，用于 Codex 索引和教师备课：

| 学生前台五域  | 后台索引映射                          |
| ------- | ------------------------------- |
| 交易与财产   | 合同交易与消费者保护；财产权、物权与相邻关系          |
| 人身与家庭   | 人格权与侵权责任；婚姻家庭与继承扶养              |
| 劳动与职业   | 劳动就业与职业边界                       |
| 创新与竞争   | 知识产权与竞争秩序                       |
| 救济与公共利益 | 纠纷解决与程序救济；生态公益、新业态与法治价值中的公共利益部分 |

Batch02 的直接贡献是：它证明“财产权物权相邻”和“婚姻家庭继承”都稳定，但学生前台不能把它们绑成一个大篮子。

## 4. 一核二线三问四步是否需要修订

### 4.1 一核：保留，加入第 0 步入框闸门

一核继续保留：

> 以事实为根据、以法律为准绳，通过具体法律关系中的规则适用实现定分止争。

但 Batch02 继续证明需要前置闸门：

> 第 0 步：先判是否为《法律与生活》法律小问。

执行规则：

1. 是否有具体主体之间的权利义务关系。
2. 是否要求作法律判断，而非只评价政府服务、政策效果或宏观治理。
3. 是否有责任、效力、归属、救济、权利边界、价值意义等法律结果。
4. 混合模块题只抽法律小问，非法律小问不入框。

### 4.2 二线：保留，但改写第二线

原二线：

1. 权利保护与权利边界。
2. 法治规则与德治价值。

Batch02 支持二线保留，但第二线应改写：

1. **权利保护与权利边界。**
   财产权边界、相邻关系边界、监护权边界、继承权边界、家庭财产边界都适用。

2. **规则适用与价值校准。**
   绿色原则、尊老敬老、夫妻平等、友善邻里、公平合理、社会主义核心价值观都要放在规则适用之后。

这样能同时防止两种偏差：

1. 防止删除价值线后无法解释家庭伦理、养老助老、绿色发展、邻里和谐题。
2. 防止价值线失控后写成宏观法治口号。

### 4.3 三问：保留，顺序由题型决定

三问继续保留：

1. 判什么或怎么处理。
2. 凭什么。
3. 有什么意义。

Batch02 中不同题型的顺序不同：

1. **选择题：** 判哪项，凭什么排错，通常不写意义。
2. **物权归属题：** 先判归属或权利效力，再说明登记、交付、判决、抵押、质押等规则。
3. **继承题：** 先判谁有权取得，再说明身份、遗嘱、协议、法定继承、扶养事实。
4. **意义认识题：** 先说本案处理，再说法律依据和作用路径，最后写价值。
5. **观点评析题：** 先拆观点，再判合理与片面，再给法律依据。

不建议把三问固定为单一顺序。

### 4.4 四步：保留，但第四步按题型分流

四步继续保留：

1. 定主体关系。
2. 找争点事实。
3. 对规则条件。
4. 落法律结果。

Batch02 要求第四步更具体：

| 题型      | 第四步落点                     |
| ------- | ------------------------- |
| 物权归属题   | 所有权、使用权、担保权、共有权、不得对抗、优先受偿 |
| 相邻关系题   | 提供便利、停止妨害、赔偿损失、相邻利益平衡     |
| 继承题     | 遗产归属、继承份额、协议效力、丧失继承权      |
| 婚姻家庭题   | 共同财产、家务补偿、监护权、未成年人利益      |
| 价值意义题   | 本案处理、制度作用、社会价值            |
| 综合民法选择题 | 逐项定关系，逐项排错                |

“价值”不能与责任、效力、归属并列为每题必写项，只能在设问触发时后置。

## 5. 每个建议的证据等级和需本地核验项

证据等级说明：

A：本批次多题支持，可进入候选主干，但仍需本地核验。
