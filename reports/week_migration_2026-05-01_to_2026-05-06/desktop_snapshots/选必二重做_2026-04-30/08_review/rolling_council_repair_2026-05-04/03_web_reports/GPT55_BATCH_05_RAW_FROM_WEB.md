# Batch05 纠纷解决、生态公益与新业态框架委员会意见

## 1. 本批次支持的高频主干

**总判定：Batch05 强支持“纠纷解决与程序救济”作为学生前台可见主干，但不支持把“生态公益”和“新业态”与程序救济混成一个学生前台大域。**

Batch05 共 22 题，主观题 12，选择题 10，证据结构为 formal_or_scoring_source=14、official_answer_key=8。该批材料集中出现诉讼时效、民事诉讼、调解、司法确认、公益诉讼、举证责任、诉讼调解、劳动仲裁、知识产权司法保护、生态修复、未成年人平台打赏、交通事故一体化处理等内容。该批可以推动“救济与公共利益”域成型，但所有题域归属、采分细则、题型标签和高频判断均**需本地核验**。

### 1.1 程序救济主干：应进入学生前台

**候选主干：程序救济不只是横切方法，至少应在前台五域中保留归位空间。需本地核验。**

本批 22 题中，大量题的采分点不在实体权利本身，而在“如何解决纠纷”：

1. 诉讼时效及不适用情形。
2. 民事诉讼、行政诉讼、刑事诉讼的主体与举证责任。
3. 调解、和解、诉讼调解、司法确认、民事调解书的强制执行效力。
4. 公益诉讼及生态公共利益保护。
5. 劳动仲裁、普通仲裁、诉讼的区分。
6. 诉前化解、一站式调解、道交纠纷线上平台。
7. 审判监督、上诉、回避、诉讼代理人、辩护人等程序排错。

学生版应形成一张程序救济卡：

> 谁有权主张，走什么程序，解决什么纠纷，产生什么法律效果。

这张卡可以横切合同、劳动、侵权、家庭、知识产权、生态公益，但也必须在前台五域中有明确入口，否则学生遇到“调解、公益诉讼、举证责任、司法确认”会无处归位。需本地核验。

### 1.2 多元纠纷解决主干：调解、和解、诉讼、仲裁的功能区分

**候选主干：多元纠纷解决机制应进入学生小卡，但不能讲成程序法目录。需本地核验。**

Batch05 多题直接考调解与非诉机制：

1. “枫桥经验”题考基层社会治理和人民调解机制。
2. 诉讼调解题考“已进入诉讼程序但以调解书解决”的机制识别。
3. 环境公益诉讼题考和解、司法调解、诉讼共同作用。
4. 道交纠纷一体化线上平台题考多元力量融合、诉前化解效率。
5. 劳动争议调解中心题考一站式服务、劳动关系和谐。

学生版可压缩为：

> 和解靠当事人协商，调解靠第三方促成，仲裁依协议或法定路径裁断，诉讼由人民法院依法裁判，司法确认使调解协议具备强制执行效力。需本地核验。

这不能扩展为完整民事诉讼法、仲裁法、行政诉讼法知识体系。

### 1.3 举证责任主干：本批支持建立“程序证据卡”

**候选主干：举证责任应作为程序救济域的小卡，重点是“谁主张谁举证”和特定情形的责任分配。需本地核验。**

2026 石景山一模第18题明确要求分析举证责任分配方式及理由，案例涉及公诉、行政行为、建筑物坠落侵权。评分摘录区分：

1. 公诉案件由人民检察院证明被告人有罪。
2. 行政诉讼中行政机关提供行政行为证据和依据。
3. 民事诉讼一般“谁主张谁举证”。
4. 建筑物坠落损害中，被告可能对无过错承担举证责任。

这说明举证责任不是单一错项，而是本批重要程序能力。学生版应保留：

> 先判诉讼类型，再判谁提出主张，再看法律是否特别分配举证责任。

但不应把举证责任讲成法考化证据法体系。需本地核验。

### 1.4 公益诉讼与生态公共利益主干：可做“救济与公共利益”小卡

**候选主干：生态公益不宜独立成学生前台域，但应作为“救济与公共利益”下的稳定小卡。需本地核验。**

本批生态公益题至少包括：

1. 民法典绿色原则和污染破坏环境侵权责任。
2. 北京法院“提升生态颜值”。
3. 环境民事公益诉讼中社会组织起诉、和解、司法调解、调解书强制执行。
4. 生态环境公共利益、生态修复、劳务代偿、闭环治理。

这些题的入口通常不是“生态”口号，而是：

> 主体适格，损害风险，公益诉讼或审判执行，法律效力，生态公共利益修复。

因此，“生态公益”应放在“救济与公共利益”下，不宜单列为前台第六域。需本地核验。

### 1.5 程序选择题排错主干：主体、程序、效果三错位

**候选主干：选择题应补充程序类排错表。需本地核验。**

本批选择题反复出现程序错项：

1. 辩护人与诉讼代理人混淆。
2. 自诉、民事诉讼、刑事责任混淆。
3. 二审终审后还能否上诉混淆。
4. 调解主持主体混淆。
5. 审判监督程序与上诉混淆。
6. 仲裁、调解、法律援助、行政裁决混淆。
7. 法庭辩论、宣告判决、举证责任、上诉期限等程序细节混入。

学生版应扩展选择题排错：

> 排主体错位，排程序错位，排效果错位，排绝对化，排法律关系错位。

本批不支持学生背大量程序期限，但支持学生警惕“具体期限、程序名称、主体身份”的错项。需本地核验。

## 2. 本批次暴露的框架缺口

### 2.1 当前“纠纷解决生态公益与新业态”域名过杂

该域把三类性质不同的内容混在一起：

1. **纠纷解决**是程序路径。
2. **生态公益**是公共利益对象。
3. **新业态**是生活场景或技术载体。

这会导致学生以为“生态”“平台”“法院”“调解”都属于同一个答题模板。实际上，小刘直播打赏、AI 知识产权保护、生态公益诉讼、诉讼时效、举证责任、枫桥经验的答题动作差异明显。需本地核验。

建议改名为：

**救济与公共利益。**

其中：

1. 程序救济是主干。
2. 公益诉讼和生态保护是小卡。
3. 新业态只做场景标签，回流到合同消费、人格侵权、劳动职业、创新竞争或程序救济。需本地核验。

### 2.3 formal 摘录污染进入系统性风险

Batch05 中污染尤其明显。多个 formal_or_scoring_source 摘录混入哲学、科学思维、政府经济、文化表达、充电桩方案等非法律内容。若不隔离，会把“创新思维”“宏观调控”“政府职能”等非法律动作误并入《法律与生活》主干。需本地核验。

Codex 应继续执行流程：

1. 题号对齐。
2. 设问对齐。
3. 分值对齐。
4. 材料主题对齐。
5. 法律小问对齐。
6. 非法律小问隔离。
7. 污染摘录不给主干权重。

### 2.4 题型标签仍然大量误贴

本批题型标签问题突出：

1. 诉讼时效原因题被标为表格补全题。
2. 生态作用题被标为观点评析题。
3. 法官专业指导题被标为表格补全题。
4. 督促履行义务告知书认识题被标为表格补全题。
5. 生态颜值题被标为表格补全题。
6. 知识产权护航新质生产力题被标为程序救济题，但更像措施说明加意义题。
7. 环境公益诉讼题被标为观点评析题，实际是一问法律分析、二问思维模块。

题型应按设问动词和栏目结构重判，不能按关键词或题卡模板自动贴。需本地核验。

## 3. 五域归位是否合理，是否需要拆、并、改名

### 3.1 不建议保留“纠纷解决生态公益与新业态”作为学生域名

该名称太长、太杂、不可背。学生无法从域名判断答题动作。

建议改为：

**救济与公共利益。**

其下设四张二级卡：

1. **程序救济卡**：诉讼、仲裁、调解、和解、司法确认、执行。
2. **举证责任卡**：谁主张谁举证、举证责任倒置、诉讼类型对应。
3. **公益诉讼卡**：主体适格、公共利益、法院审查、调解或判决效力。
4. **生态修复卡**：损害风险、责任承担、修复方案、公共利益价值。

“新业态”不作卡名，改为场景标签。小刘直播打赏回到合同效力、未成年人保护和平台责任；AI 或知识产权案件回到创新与竞争；道交平台和一站式调解回到程序救济。需本地核验。

### 3.2 前台五域建议继续固定为“大域 + 二级卡”

经过 Batch01 至 Batch05，较稳的学生前台五域仍建议为：

| 前台五域    | 二级卡                     |
| ------- | ----------------------- |
| 交易与财产   | 合同消费卡、物权相邻卡             |
| 人身与家庭   | 人格侵权卡、家庭继承卡、未成年人保护卡     |
| 劳动与职业   | 劳动关系卡、解除补偿卡、竞业限制卡       |
| 创新与竞争   | 著作权卡、不正当竞争卡、数据与AI卡      |
| 救济与公共利益 | 程序救济卡、举证责任卡、公益诉讼卡、生态修复卡 |

Batch05 的贡献是证明第五域必须可见，但应以“救济与公共利益”命名，不能把“新业态”作为并列域。需本地核验。

### 3.3 后台八域继续保留

后台八域仍有必要，因为 Batch05 中交叉题很多。比如诉讼时效加赡养、生态公益加知识产权、调解加劳动、知识产权加司法保护、平台打赏加未成年人监护。这些都需要后台双标签或多标签。需本地核验。

学生不背八域，Codex 用八域保真。

## 4. 一核二线三问四步是否需要修订

### 4.1 一核：保留，并强化第0步

一核继续保留：

> 以事实为根据、以法律为准绳，通过具体法律关系中的规则适用实现定分止争。

Batch05 再次证明需要第0步：

> 先判是否为《法律与生活》法律小问，再判具体法律关系。

特别是 2024 丰台二模Q17、2025东城期末Q17、2025顺义一模Q19、2026门头沟一模Q18 的摘录均有非法律模块混入风险。需本地核验。

### 4.2 二线：保留，第二线继续改为“规则适用与价值校准”

本批既支持权利保护，也支持价值校准：

1. 诉讼时效保护权利和司法效率。
2. 赡养费不适用诉讼时效体现老年人生存权、公序良俗和人文关怀。
3. 环境公益诉讼维护生态公共利益。
4. 调解机制体现和谐、效率、司法温度。
5. 知识产权司法保护服务创新和市场秩序。

但价值必须后置。不能开头写“司法温度”“生态颜值”“护航新质生产力”“社会治理现代化”。应先写本案主体、程序、效力，再写价值。需本地核验。

建议二线：

1. 权利保护与权利边界。
2. 规则适用与价值校准。

### 4.3 三问：保留，程序题按“怎么处理”优先

三问继续保留：

1. 判什么或怎么处理。
2. 凭什么。
3. 有什么意义。

Batch05 说明程序救济题常以“怎么处理”为第一问：

1. 通过何种纠纷解决方式。
2. 谁可以起诉或申请。
3. 何种协议或裁判具有效力。
4. 举证责任如何分配。
5. 该处理有何价值。

程序题顺序建议：

> 怎么处理，凭什么，有什么法律效果和意义。

选择题不强求三问全套，只做主体、关系、程序、效果排错。

### 4.4 四步：保留，但第四步必须程序化分流

四步继续保留：

1. 定主体关系。
2. 找争点事实。
3. 对规则条件或程序条件。
4. 落法律结果。

Batch05 的第四步不能再写“责任、效力、归属、救济路径、价值”五并列，应改为：

| 题型    | 第四步落点                  |
| ----- | ---------------------- |
| 诉讼时效题 | 是否适用时效，权利保护与司法效率       |
| 调解题   | 调解、诉讼调解、司法确认、强制执行效力    |
| 仲裁题   | 仲裁适用条件、裁决或争议解决效果       |
| 举证责任题 | 谁举证，举什么，为什么这样分配        |
| 公益诉讼题 | 主体适格，公共利益，调解或判决效力，生态修复 |
| 程序选择题 | 程序主体、程序名称、法律效果排错       |
| 司法意义题 | 本案处理，制度功能，权利保护，价值后置    |
