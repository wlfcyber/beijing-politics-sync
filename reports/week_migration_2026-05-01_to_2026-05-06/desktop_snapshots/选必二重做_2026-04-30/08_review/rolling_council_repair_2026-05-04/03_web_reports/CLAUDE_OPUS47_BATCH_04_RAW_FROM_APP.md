## 1. 本批次最适合学生掌握的主干动作

从 15 道题（9 选择 + 6 主观）中能稳定提取的可教动作有 **5 条**：

**M1. 法律部门前置识别（作品 / 标识 / 行为）**
本批次 IP 域真正的硬识别动作不是侵权三要件，而是**先确定属于哪一部法律**：
- **作品类争议** → 著作权法（题 2、7、8、10、11、14 等）
- **标识类争议** → 商标法（题 1、6、14 选项 A 错项陷阱）
- **行为类争议** → 反不正当竞争法（题 1、3、6、9、12、15）

题 14（2026 朝阳期末 11）选项 A"虚拟数字人甲作为独创性智力成果构成美术作品，受商标法保护"——是错项陷阱，故意把"作品"配"商标法"。**学生必须先做法律部门识别才能解题。**

这与 Batch01-03 各域不同——前几域学生可以从主体关系入手，IP 域必须**先识别法律部门**，错配法律部门题就做不出。`需本地核验：本地 IP 题是否多题以法律部门错配作为采分陷阱`

**M2. 著作权"独创性 + 智力投入"识别**
本批次至少 4 道题（题 7、8、10、13）考著作权作品认定的**独创性 + 智力投入**双要件：
- 题 7 细则原文："具备'智力成果'和'独创性'要件，受到著作权法保护"
- 题 8 细则反向印证："凝结了王某的智力劳动成果"
- 题 13 细则原文："AI 不具备民事主体资格"——AI 工具不是著作权主体

**这是本批次最稳定的采分骨架**，应固化为著作权识别口诀：
> 看是否有人的**智力投入** + 看是否具有**独创性**（个性化表达）→ 是 → 受著作权法保护
> AI 工具不是著作权人；使用 AI 创作的人才可能是著作权人

**M3. 反不正当竞争行为识别**
本批次至少 5 道题考反不正当竞争行为类型识别：
- 混淆行为（题 1、9：仿冒包装 / 抢注唤醒词）
- 虚假宣传（题 2、9：虚假和引人误解的商业宣传）
- 搭便车（题 6：未经允许使用知名品牌名称作企业名称）
- 商业秘密（已在 Batch03 错归题 6 涉及）
- 数据保护（题 15：成像效果高度相似的 AI 模型仿造）

题 15（2026 门头沟一模 11）题面给出"甲公司对该模型享有创新优势和市场利益，乙公司的行为应受反不正当竞争法规制"——**这是反不正当竞争法保护范围的扩张样本**（保护未达著作权 / 专利标准但有市场利益的创新成果）。

**M4. AI 时代新规则识别**
本批次至少 6 道题（题 2、7、8、10、11、12、13、14、15）涉及 AI / 虚拟数字人 / 生成内容 / 模型仿造，构成 **Batch04 的命题主线**。AI 题的稳定采分动作：
- AI 不是民事主体（题 13 细则明确）
- AI 生成内容是否构成作品看人的智力投入（题 7、8）
- AI 直播必须标识（题 11 题面、题 12）
- AI 生成内容侵权应停止侵害 / 赔偿损失 / 赔礼道歉（题 12 细则）
- AI 平台已尽提示义务可不担责（题 13 细则）

**这是本批次最具时代特色的命题方向**。学生版应固化为"AI 案件识别四问"：
> 谁是真正的主体？AI 工具的使用人 / AI 平台 / 被 AI 侵害的对象
> 涉及什么权利？著作权 / 肖像权 / 姓名权 / 商业声誉
> 平台是否尽到义务？提示 / 标识 / 审核
> 如何承担责任？停止侵害 / 删除 / 赔偿

**M5. 价值后置 + IP 域专属价值表达**
本批次 IP 域价值题（题 2、4、9、12、13、15）尾段表达高度集中：
- "激发创新活力 / 维护市场秩序"（题 2）
- "鼓励创新 / 营造公平市场环境"（题 4）
- "促进新业态发展 / 数字经济发展"（题 9）
- "引导公众理性使用 AI / 规范 AI 行业发展"（题 13）

题 4（2025 丰台一模 19）细则**反向印证**：
- "维护社会公平正义"**不得分**
- "优化营商环境"**是错误表达**
- "维护公民的合法权益""维护双方的合法权益""维护被告方的权益"**都是错误表达**
- "没有铺垫，直接表达法治中国建设"**不得分**

**这是本批次发现的最严厉的价值表达扣分规则。** IP 域价值表达必须精准到"权利人"或"原告方"，且必须有"权利保护 + 创新激励 + 市场秩序"三段铺垫。`需本地核验：是否多题 IP 细则统一此扣分规则`

## 2. 本批次最容易让学生误学/跑偏的地方

### 2.1 题卡"为什么能想到"段再次延续关键词触发问题

本批次仍有错归疑点，但比 Batch03 轻：

**题 5（2025 丰台二模 12 科技成果先使用后付费）疑似错归**
题面是"科技成果先使用后付费"，4 个题肢全是政治学习模块表达（演绎推理 / 逆向思维 / 辩证思维动态性 / 科技成果与现实生产力转化）。这是**政治模块伪装题**，与 Batch01 题 22（入党誓词）、Batch02 题 14（12348 热线）同型。仅因关键词"专利"被归入 IP 域。

### 2.6 IP 域价值表达扣分点严苛

### 2.7 "信息网络传播权 + 署名权"等具体权能要求

题 7（2025 昌平二模 20）细则明确"侵害了李某的**信息网络传播权和署名权**（二选一，1分）"。这要求学生识别**著作权具体权能**，而不是笼统说"著作权"。

这与 Batch01-03 cross-critique 我们部分 reject 的"复制权 / 信息网络传播权等具体权能系统背诵"产生冲突——**至少在 IP 域，具体权能识别是采分必备**。

## 3. 五域归位的学会性评价

### 3.1 本批次"知识产权与竞争秩序"域真实题量

按本批次题卡 15 题全部归 IP 域。剔除疑似错归（题 5、10）后真实题量约 **13 道**。这与上一轮全量统计的"知识产权与竞争秩序 15 题"基本吻合。`需本地核验：题 5、10 在本地是否归 IP 域还是政治/哲学模块`

### 3.2 IP 域采分动作与其他域差异分析

本批次再次验证 IP 域采分动作与其他域**完全不同**：

| 域 | 第一识别动作 | 域特殊性 |
|---|---|---|
| 合同与消费 | 谁 vs 谁（消费者 vs 经营者）| 主体识别先行 |
| 物权与相邻 | 物权三要件（设立 / 变动 / 对抗）| 物权规则识别 |
| 家庭与继承 | 身份关系 + 财产位序 | 身份识别先行 |
| 人身人格与侵权 | 侵权三要件（过错 / 损害 / 因果）| 要件识别先行 |
| **IP 与竞争** | **法律部门识别（作品/标识/行为）** | **部门识别先行** |

这强烈支持 IP 与竞争**独立成域**，不与其他域合并。GPT Batch01-02 主张的"创新与竞争"域名 + 独立成域是**正确判断**——本批次再次验证。

### 3.3 IP 域内部是否需要分子卡

本批次 13 道真 IP 题分布：
- 著作权类：约 6 道（题 2、7、8、10、11、13）
- 商标类：约 2 道（题 1、14）
- 反不正当竞争类：约 5 道（题 3、6、9、12、15）

**比例较均衡**。建议 IP 域内分 **3 张子卡**（不超 GPT Batch03 提的 3 张二级卡上限）：
- 著作权识别卡（独创性 + 智力投入 + 具体权能）
- 商标 / 标识识别卡（注册商标专用权 + 商业混淆）
- 反不正当竞争识别卡（混淆 / 虚假宣传 / 搭便车 / 商业秘密 / 数据权益）

**与 GPT Batch03 报告 §4.2 主张的 IP 域 3 张子卡（著作权卡 / 不正当竞争卡 / 技术秘密数据卡）方向一致**，但子卡内容略调：商标作为单独一类不应并入著作权或不正当竞争；技术秘密 / 数据权益归入反不正当竞争（因为本批次反不正当竞争法保护范围扩展至此，题 15 验证）。`需本地核验：本地是否将技术秘密、数据权益归反不正当竞争法`

### 3.4 IP 域第四步落点

本批次 IP 题第四步落点高度集中：
- "停止侵害 + 赔偿损失 + 消除影响 + 赔礼道歉"（典型 IP 侵权责任）
- 部分题加"承担诉讼费用"（题 12）

**与 Batch03 侵权域第四步"侵权责任承担方式（四种典型）"**完全一致。这是因为 IP 侵权本质上也是侵权法子集。

但 IP 域有独特的责任方式：
- **惩罚性赔偿**（题 4 涉及，已在民法典 + 反不正当竞争法 + 著作权法 + 商标法 + 专利法五法明确）
- **法定赔偿**（举证困难时由法院酌定）

**修订**：IP 域第四步落点为"侵权责任承担方式（四种典型）+ 惩罚性赔偿（特殊）"。

### 3.5 IP 域与人格权侵权域的边界

本批次题 11（2026 东城一模 11 林某 AI 肖像）和题 12（2026 延庆一模 18 李某 AI 主持人）都涉及**AI 侵害肖像权 / 声音权益**。这是 IP 域与人身人格与侵权域的**边界题**。

GPT Batch03 已主张题 8（朝阳二模 20 摄像头隐私）做"双标签处理"。本批次题 11、12 应同样做双标签：
- 主标签：人身人格与侵权（肖像权 / 声音权益核心）
- 辅标签：IP 与竞争（AI 生成技术背景）

`需本地核验：题 11、12 在本地是否做双标签采分`

## 4. 一核二线三问四步的课堂表达是否要修

### 4.1 一核（保留）
本批次再次验证一核可用。题 4 细则明确反对"维护社会公平正义"作为价值开头——这反向印证**一核中的"具体法律关系中的规则适用"**比抽象法治价值更优先。**保留**。

### 4.2 二线第二线（保留 GPT Batch03 修订）
本批次价值题（题 2、4、9、12、13、15）再次验证"价值校准"线保留。

题 4 细则给出 **IP 域价值表达分层结构**：
- 企业层面（权利与义务角度）：权益保护 + 严惩侵权
- 对科技创新意义：竞争秩序 + 公平市场 + 鼓励创新
- 法治意义：法治信仰（意识）+ 依法层面（行动）+ **铺垫后**才能上升到法治中国建设

**这与 Batch01-03 cross-critique 的"民法基本原则归规则侧、核心价值观归价值侧"分层方案部分契合，但 IP 域更细化为三层**：
1. 权利保护层（民法基本原则）
2. 制度功能层（市场秩序 / 创新激励）
3. 法治价值层（核心价值观 / 法治建设，必须有铺垫）

**修订**：IP 域意义题答题模板增加"三层递进"结构。

### 4.3 三问（保留按题型触发）
本批次 IP 选择题（9 道）只触发"判什么 + 凭什么"两问。
本批次 IP 主观题（6 道）按题型触发：
- 裁判依据题（题 7、9）：判什么 + 凭什么
- 意义认识题（题 2、4、13）：完整三问 + 三层递进
- 观点评析题（题 12）：拆观点 + 凭什么 + 综合判断

**这与 Batch01-03 一致再次验证。**

### 4.4 四步（保留 + IP 域第四步特殊化）
本批次再次验证四步可用。**IP 域第四步落点**为：
- 侵权责任承担方式（四种典型）
- + 惩罚性赔偿（特殊）

**这是第四批次连续验证"第四步按域分流落点"。**

### 4.5 IP 域必须新增"法律部门前置识别"作为第 0.5 步（本批次新增）

**重要修订**：IP 域学生主干必须在四步之前增加"**法律部门前置识别**"动作：
- 看材料：是作品？标识？还是行为？
- 决定法律部门：著作权法 / 商标法 / 反不正当竞争法
- 决定后续四步内的具体规则

**这是 IP 域独有的前置动作**，其他域不需要。原因是 IP 域的"法律部门错配"是命题人最常用的陷阱（题 14 选项 A 验证）。

### 4.6 五域具体表达更新（结合 Batch01-04 四批次证据）

| 学生五域 | 包含子域 | 验证批次 |
|---|---|---|
| ① 合同与消费 | 合同、消费者保护 | Batch01 |
| ② 物权与相邻 | 物权、相邻关系 | Batch02 |
| ③ 家庭与继承 | 婚姻家庭、监护、继承、遗赠扶养 | Batch02 |
| ④ 人身人格与侵权 | 侵权三要件、归责原则、人格权类型、安全保障义务、见义勇为补偿 | Batch03（采用 GPT Batch03 命名）|
| ⑤ **IP 与竞争** | **著作权、商标、反不正当竞争（含商业秘密、数据权益、AI 模型保护）** | **Batch04（本批次正式确立）** |

**Batch04 关键决断**：
- 第五域确立为 **IP 与竞争**，不是 GPT Batch01 主张的"救济与公共利益"
- 程序救济不独立成第五域，作为横切方法论
- 劳动与职业域待 Batch05 验证后定（如果有 Batch05 的话）
- 生态公益、新业态作为开放容器，不独立成域

`需本地核验：本地是否将"劳动与职业"作为独立域，还是与本批次 IP 域并列`

### 4.7 选择题排错表更新（含 IP 域内嵌排错）

结合 Batch01-04 四批次：

**通用前 4 排错**（学生版必背）：
1. 绝对化排错
2. 主体错位排错
