## 1. 总判定

**NEEDS_FIX**

主干术语大多能形成材料触发到答案句的闭环，但目前不适合直接给高三学生单独使用。主要问题是：设问与材料概述混写，评分依据权威性未界定，政治多极化与中国外交桶边界混乱，个别术语触发条件过宽，学生容易学成堆术语模式。

## 3. 可优化清单

1. **出现1次要改口径。**
   现在每个术语都写出现1次，学生可能误读为全书或高考频次。建议改成：在本五题样例中出现1次。

2. **统一教材规范表述。**
   和平和发展建议统一为和平与发展。经济全球化方向建议尽量写完整：更加开放、包容、普惠、平衡、共赢。国家间共同利益是国家合作的基础这一句保留，属于高迁移度表述。

3. **每条补禁用条件。**
   现在只有材料触发，缺少何时不用。比如《联合国宪章》只在材料出现联合国、宪章、国际法治、主权平等、宗旨原则时稳妥；经济全球化方向只在材料出现开放共享、普惠发展、发展中国家受益、共同发展时稳妥。

4. **把长答案句拆成采分句。**
   个别答案句堆了太多术语，例如中国发展和世界发展的关系一条，把本国利益、公共产品、正确义利观、中国方案、大国责任、人类命运共同体全塞进一句。建议拆成2到3个采分句，避免学生学成术语拼盘。

5. **材料触发要更像抓关键词。**
   现在很多触发说明是二次解释。建议增加材料原词或关键词，例如治理赤字、逆全球化、主权平等、多边主义、人才流商品流资金流、全球南方、农业增产、培养本土人才。学生迁移靠这些词识别知识点。

6. **朝阳一模Q20需要整题闭环图。**
   这道题承担了太多术语，应单独做一张结构图：科技创新能力，中国担当，开放合作，全球治理方向。否则学生只看到零散术语，看不到为什么这些点能共同回答中国为什么能。

## 4. 六桶结构是否有错放

| 六桶    | 判定          | 具体问题                                                            | 调整建议                                                    |
| ----- | ----------- | --------------------------------------------------------------- | ------------------------------------------------------- |
| 时代背景  | 基本可用，有轻微混桶  | 时代主题放这里正确。但经济全球化被放进时代背景的复合条目，容易和经济全球化桶重复。                       | 保留时代主题、和平与发展、正逢其时、世界大势。经济全球化相关内容加交叉标签。                  |
| 理论    | 有混桶         | 共同利益、国际竞争实质适合放理论。三大关系更像关系型设问模板或中国发展方法论。                         | 理论桶拆成国际关系基础理论和关系型设问模板。                                  |
| 经济全球化 | 基本可用，个别触发过宽 | 两个市场两种资源、高水平开放、贸易投资自由化便利化、要素流动放这里合理。贸易摩擦条目需要区分知识产权纠纷、合规风险和贸易摩擦。 | 贸易摩擦条目标注经济全球化 × 国际组织机制，不能泛化成所有海外纠纷。                     |
| 政治多极化 | 错放最多        | 人类命运共同体、外交思想、正确义利观、新型国际关系直接放入政治多极化，会压扁中国外交属性。                   | 保留多边主义、国际关系民主化、公正合理国际新秩序、共商共建共享全球治理观。其余中国外交术语移入中国桶或标交叉。 |
| 中国    | 当前过窄        | 只放中国智慧中国方案、大国担当，承接不足。                                           | 应吸收人类命运共同体、习近平外交思想核心理念、正确义利观、新型国际关系、中国智慧中国方案、大国担当、全球倡议。 |
| 联合国   | 基本正确        | 《联合国宪章》放这里正确。但不能把所有国际组织都归到联合国，WTO、国际经济组织、全球经济治理机制要另行处理。         | 联合国桶只管联合国、联合国宪章、宗旨原则、主权平等、国际法治。                         |

**六桶最大问题：政治多极化桶和中国桶边界不清。**
这会直接影响学生迁移，因为北京政治题经常通过设问限定来区分世界形势、国际关系、中国担当、联合国框架。

## 5. 高三学生只看文档能不能迁移

**不能稳定迁移；学生能背出术语和样句，但遇到新题容易混淆设问、材料、评分来源和六桶边界，进而出现堆词、错桶、触发过宽的问题。**
