# 学生文档审稿意见

## 1. 总判定

**目前还不能直接交给高三学生试读。** 它现在更像“工作流的中间产物”，而不是“面向考生的成品”。学生通读时会遇到三个直接的卡点：同一道题的完整设问被原样重复七八次、看不到任何一道题的“整题汇总答案”、`细则位置` 一行充满 PPTX/DOCX 段号这种工作流元数据。先过两道修剪和拼装，再给学生。

---

## 2. 必改问题

**问题 A：完整设问被冗余复制,信息被淹没**
- 表现:朝阳一模 Q20 在五个桶里出现,`完整设问` 那段约 150 字的题干被原文粘了 7 次;通州期末 Q20 被粘了 5 次;朝阳期中 Q17 被粘了 3 次。
- 学生为什么卡住:学生第二次看到一模一样的设问就会跳读,跳读之后就看不到该条目真正要他抓的差异点(只有 `材料触发` 那一段是新的)。
- 改法:每道题在文档里只完整呈现一次设问,放在该题第一次出现的位置;后续条目把 `完整设问` 字段换成一行题号引用(如"设问见上文 2026朝阳一模 Q20"),把腾出来的空间留给材料触发的具体差异。

**问题 B:缺一份"整题汇总答案"视图,学生没法看到目标长什么样**
- 表现:朝阳一模 Q20 的得分点散在五个桶里(时代背景 1 分、理论 2 分、经济全球化 5 分、政治多极化 2 分、中国 3 分),通州 Q20 散在四个桶里。文档现在只给单条术语的"答案句",从来没拼出过任何一道题的完整答案卡。
- 学生为什么卡住:高三学生平时练答题卡,练的是"对着一道题写出 200–400 字的结构化回答",不是"对着一个术语写一句话"。看不到拼装后的样子,他既不知道这道题的答案应该多长、有几层,也不知道这些散点之间的连接词长什么样。
- 改法:每道题在所有术语条目结束后,补一个 `整题汇总卷面答案` 块,把这道题在六个桶里所有"答案句"按一定顺序串起来,给出学生最终要交上去的那一版。

**问题 C:`细则位置` 字段是工作流痕迹,不是学生需要的信息**
- 表现:几乎每条都有 "细则 PPTX 第 9 页"、"细则 DOCX 第 98 段"、"主观题阅卷细则第 17 题第二层" 这种引用。
- 学生为什么卡住:学生不会去查 PPT 第 9 页,他只关心"这点几分、是必答还是任答"。这些段号反而让条目看起来像"内部流程文件",降低信任。
- 改法:把 `细则位置` 压缩成 `分值/必答标识`(例:"2 分,必答"或"任答 1 分"),把页码段号挪到文档末尾的"出处索引"里,愿意核对的老师可以查,学生看不到。

**问题 D:有的"术语"其实是把多个并列得分点拼成一条**
- 典型:政治多极化桶里的 "推动构建国际新秩序、倡导国际关系民主化、践行多边主义、坚持正确义利观、兼顾利益" 被当成一个术语条目。
- 学生为什么卡住:这五个并列点在通州 Q20 细则里是"任答其一 1 分,共 2 分"的关系,学生需要知道答两个就够了;现在被合成一条术语,学生背的时候要么全背要么漏背,把握不准。
- 改法:把这种并列结构在术语行里就拆开,标注"任答两点,每点 1 分",`答案句` 也给出"任选两点"的写法示范,而不是把五点全堆进一个长句。

**问题 E:`出现1次` 标签对学生没价值**
- 表现:每个术语标题后都跟着 `(出现1次)`。
- 学生为什么卡住:他不会去用这个频次决定背不背——标"出现1次"反而让他怀疑"那是不是不重要"。这是给工作流统计用的,不是给学生看的。
- 改法:学生版直接删掉这个括号;如果要保留考频信息,改成"高频/中频/低频"或"近三年出现 X 次"的可读标签。

**问题 F:`材料触发` 段有时候在做元分析,而不是替学生想**
- 典型句式:"题目让解释'正逢其时',要回答倡议为什么在当前世界形势下有现实必要。题干同时呈现两条线:一是…二是…前者触发…后者触发…"
- 学生为什么卡住:这是审题分析的"作者旁白",学生看了知道老师怎么想的,但他自己拿到新题时,没有这种"前者触发/后者触发"的句式可以套。
- 改法:把 `材料触发` 改成两行就够——第一行"材料中的哪句话/哪个细节",第二行"对应触发哪个术语"。把分析味重的连接词("触发""对应""即可看出")收回来,留给学生自己串。

## 3. 文字成品化建议(让答案句更像真实高分答题卡)

现状的 `答案句` 是"术语 + 材料压缩"组成的一句长句,读起来像 AI 缩写,不像考生写的。具体可以动手的几处:

- **朝阳一模 Q20,经济全球化桶**(两个市场两种资源 / 推动贸易投资自由化便利化 / 高水平对外开放 / 要素流动 / 经济全球化方向)——这里五条 `答案句` 句式高度同构,都是"中国通过 X,Y,因此为全球发展注入稳定性和正能量"。学生抄下来五句话连读,会发现尾巴一模一样。真实答题卡里这五点会用 `第一,…;第二,…;第三,…` 串成一段,而且只在最后一句回扣题目"注入稳定性和正能量"。建议在"整题汇总答案"里改成单段三层式,而不是五个并列长句。

- **朝阳期中 Q17 三层结构**(自力更生与对外开放 / 发展与安全 / 中国与世界发展)——这道题的 `答案句` 已经接近成品,但每条都用"一方面…另一方面…"开头,三条连写时三个"一方面"会撞车。建议第一层用"一方面/另一方面",第二层用"既要…又要…",第三层用"在…的同时…",让句式有差异。

- **顺义一模 Q20**(共同利益 / 经济全球化方向:普惠平衡共赢 / 新型国际关系:合作共赢 / 外交思想:人类命运共同体)——四条 `答案句` 都以"科技小院…"开头做主语,真实答题卡里学生会写"国家间存在共同利益…(理论先行)。材料中科技小院通过…(材料对接),这表明…(回扣设问)"。现在文档里把理论句和材料句压成一句,丢了"理论—材料—结论"的三段式。建议把每条 `答案句` 拆成两小句呈现:理论句一句、材料对接一句。

- **海淀期中 Q16(2)**——只孤零零地出现在"经济全球化"桶里一个术语,全题只示范了贸易摩擦 2 分这一点。学生看完会问"这道题就这 2 分?"。如果该题还有别的得分点不在五题术语命中范围内,需要在条目里写明"本文档只示范本题中的 X 分,其余得分点见原卷";否则学生会以为这就是全部答案。

## 4. 六桶结构是否影响学生理解

**结论:六桶适合"背术语",不适合"答整题",必须叠加一个"整题视图"才能用。**

六桶(时代背景 / 理论 / 经济全球化 / 政治多极化 / 中国 / 联合国)作为术语归类是清楚的,学生背术语时可以按桶过。问题在于,学生真正在考场上面对的是一道题,不是一个桶。朝阳一模 Q20 的得分点横跨五个桶,通州 Q20 横跨四个桶——这意味着学生背完六桶之后,他还需要在脑子里反向重组:"这道题该从哪几个桶里取点"。文档目前没有任何辅助这一步重组的结构。

具体两个隐患:
- 学生只读六桶,会形成"一题=一桶"的错觉,看到新题先在某一个桶里找答案,漏掉其他桶。
- "时代背景"和"中国"两个桶在不同题里互相重叠(都会出现"和平与发展时代主题""大国担当""人类命运共同体"),学生读到第三遍会觉得"怎么又是这一句",分不清这两个桶到底差在哪。

修法不是推翻六桶,而是在六桶之外加一层"按题索引":每道题一页,列出"本题命中桶"+"本题完整答案卡"。六桶用来温习术语,题索引用来练答题,两个视图并存。

---

## 5. 最终一句话

**这份五题样例离"成品版"还差三件事:把重复的设问合并到每题一次、为每道题补一份拼装好的整题答案卡、把工作流字段(细则段号、出现频次、元分析口吻)从学生视野里清掉。**
