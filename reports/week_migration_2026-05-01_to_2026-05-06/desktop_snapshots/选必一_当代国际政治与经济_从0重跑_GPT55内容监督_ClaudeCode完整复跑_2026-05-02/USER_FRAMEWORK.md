# User Framework

## Top-Level Structure

Use the user's six buckets as the main organizing logic:

1. 时代背景
2. 理论
3. 经济全球化
4. 政治多极化
5. 中国
6. 联合国

## Placement Rules

- 时代背景: broad world conditions,时代主题, globalization/multipolarity as background, governance challenges, unilateralism, protectionism,逆全球化,霸权主义/强权政治 when rubric uses them as situation.
- 理论: 国家利益, 国家安全, 共同利益, 国际竞争实质, 综合国力, 科技/经济实力, cooperation logic.
- 经济全球化: free/facilitated trade and investment, factor flow, two markets/two resources, international division of labor, globalization direction.
- 政治多极化: new type international relations, global governance view, international relations democratization, multilateralism, human community when emphasis is order/governance.
- 中国: policy, wisdom, responsibility, and reluctant `地位` when source scoring point is China identity/status.
- 联合国: UN status/role, UN Charter, UN-centered international system, China-UN relation.

## Entry Thinking Order

Each student-facing unit must teach:

`材料触发关系 -> 为什么想到这个评分术语 -> 可直接写出的答案句`

The artifact is a scoring-term accumulation and transfer teaching document, not a general answer bank.
