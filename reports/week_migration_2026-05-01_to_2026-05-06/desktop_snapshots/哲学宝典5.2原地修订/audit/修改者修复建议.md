#### 二、辩证法 / 矛盾就是对立统一

- 材料触发点：设问把“人塑造人工智能”与“人工智能塑造人”并列提出；材料写人类受神经系统研究启发开发人工神经网络，也写人工智能“重塑人类对世界的认识”“深刻改变制造业生产模式和经济形态”。

- 答案落点：未来人与人工智能不是单方面塑造关系，而是既相互区别又相互依存、相互作用的对立统一关系；应以矛盾观点把握二者在未来发展中的统一。

#### 三、认识论 / 实践是认识的基础

- 材料触发点：材料写研究人员观察神经元联系并开发人工神经网络，又写人工智能应用会重塑人类对世界的认识。
- 为什么能想到：人工智能产生于人的科研和应用实践；人工智能进入各行业后，又成为新的实践工具，推动人的认识变化。
- 答案落点：人与人工智能的相互塑造建立在实践基础上；人在科研和应用实践中认识、设计、改造人工智能，人工智能又在新的实践中推动人形成新认识。

#### 二、辩证法 / 联系的普遍性与多样性

- 材料触发点：算力向交通、金融、教育、医疗、科技、能源等行业渗透，人工智能与制造业深度融合，并通过数据连接人的生活。
- 为什么能想到：材料呈现人工智能与行业、生产、生活、认识之间的多种联系，说明人与人工智能不是孤立存在。
- 答案落点：未来人与人工智能在生产、生活、认识等多种联系中相互制约、相互塑造，应以联系观点处理二者关系。

#### 二、辩证法 / 发展的观点

- 材料触发点：材料把人工智能写成“产生—突破—未来”的演进过程，从人工神经网络、深度学习到大模型应用。
- 为什么能想到：人工智能及人与人工智能的关系不是静止的，而是在技术突破和实践扩展中不断变化发展。
- 答案落点：人与人工智能的相互塑造是动态发展的过程，应以发展的眼光看待并合理引导这一过程。

#### 一、唯物论 / 尊重客观规律与发挥主观能动性相结合

- 材料触发点：科学家受脑科学研究成果启发开发人工神经网络，同时人工智能初期发展受计算机内存和处理速度限制。
- 为什么能想到：人能主动研发人工智能，但研发和应用必须受脑科学规律、计算技术条件和社会规律制约。
- 答案落点：人塑造人工智能必须把尊重客观规律和发挥主观能动性结合起来，在规律和条件允许的基础上推进技术突破、政策引导和伦理治理。

#### 四、历史唯物主义 / 社会存在与社会意识

- 材料触发点：人工智能深刻改变制造业生产模式和经济形态，也重塑人类对世界的认识。
- 为什么能想到：生产方式、经济形态属于社会存在；认识方式、思想观念属于社会意识。社会存在变化会引起社会意识变化。
- 答案落点：人工智能通过改变社会生产生活方式塑造人的认识方式；同时要用先进社会意识引导人工智能更好服务社会。

#### 五、价值观 / 人生观 / 价值观的导向作用

- 材料触发点：人工智能让每个人都可能被追踪、收集、分析、输出，既带来便利也潜藏风险。
- 为什么能想到：如何处理人与人工智能关系，实质上要求用正确价值取向引导技术研发和应用边界。
- 答案落点：处理人与人工智能关系必须坚持正确价值观，以增进人类福祉、维护公共利益和科技伦理为导向，让人工智能服务人的全面发展。

#### 四、历史唯物主义 / 人民群众

- 材料触发点：材料反复以“我们”“每个人”“人类”为人工智能便利与风险的承受主体。
- 为什么能想到：人工智能发展的根本目的不是技术本身，而是服务最广大人民群众。
- 答案落点：未来人工智能发展应坚持人民立场，把是否增进人民福祉、保障人民根本利益作为判断标准。

- 为什么能想到：材料不是要求放弃传统产业、另起炉灶发展未来产业，而是要求在保留传统产业基础和优势的同时，用未来技术克服低端、薄弱、落后的方面，实现扬弃式升级。这正是辩证否定的“扬弃”和守正创新。
- 答案落点：发展新质生产力不能忽视或放弃传统产业，而应守住传统产业的制造基础、配套能力和市场支撑，同时用未来技术改造低端产能、补强创新能力，推动传统产业向高端化、智能化、绿色化升级。

标记：必须回源核对原题材料、答案和细则；若原细则完全属于选必三逻辑与思维，学生稿只可作为“模块边界案例”，不得当作必修四稳定采分点。

### 3. 2026海淀一模第16题：两条必须保留，其他条谨慎

特别结论：

- “物质决定意识”应保留。
- “主观能动性 / 意识的能动作用”应保留。
- “物质决定意识”条不能写意识反作用。
- 设问不得抄材料；当前两条设问均是完整真实设问，可保留。

物质决定意识建议压缩为：

- 材料触发点：材料先说“社会大发展、大变革的时代，也是人文学科推陈出新的时代”，并举春秋战国百家争鸣、西方文艺复兴为例；随后把人工智能时代也写成技术进步带来的社会巨变。
- 为什么能想到：社会大发展、大变革和人工智能技术进步属于客观时代条件，这些现实变化提出价值冲突、情感理解、历史记忆等新问题，决定人们需要重新认识人文学科的时代价值。
- 答案落点：人工智能时代的客观技术变革和社会巨变，决定人们需要人文学科辨析价值冲突、理解情感和历史记忆，所以人文学科在人工智能时代具有不可替代的必要性。

主观能动性建议压缩为：

- 材料触发点：材料把“辨析价值冲突”“人何以为人”“历史意识、伦理意识和文化理解”交给人来完成。
- 为什么能想到：主观能动性强调人能够有目的、有意识地认识世界、作出判断并选择方向；人工智能能处理信息，但不能替人完成价值判断、伦理反思和文化理解。
- 答案落点：面对人工智能，人不能把价值判断和伦理选择完全交给技术，而要主动辨析价值冲突、理解技术边界和“人何以为人”的问题，这正体现人文学科不可替代的价值。

其他同题条目：

- “社会存在与社会意识”可保留为历史唯物主义条，但答案落点要写自然答案句，不要把“社会意识反作用”塞进物质决定意识。
- “价值观导向”可保留，但只写价值判断和伦理方向，不写文化功能。
- “发展观”若没有明确评分支持，建议标记回源；不要因为材料出现“推陈出新”“时代变化”就机械加发展观。
- 文化功能条直接删除。

## 四、全局写法模板与反例

### 1. 固定顺序

每条必须按以下顺序写：

1. 材料触发点。
2. 设问。
3. 为什么能想到。
4. 答案落点。

不得把设问放进材料触发点；不得把答案或细则放进材料触发点；不得把“评分细则提到”写进为什么能想到。

### 2. 材料触发点合格标准

合格触发点必须是真实材料句、例子、关系、主体行动、矛盾、条件或过程。

合格例：

- “社会大发展、大变革的时代，也是人文学科推陈出新的时代”，并举春秋战国百家争鸣、西方文艺复兴。
- 人类受神经系统研究启发开发人工神经网络，人工智能又重塑人类对世界的认识。
- 传统产业是现代化产业体系的基底，未来产业渗透传统产业链上下游，引领高端化、智能化、绿色化转型。

不合格例：

- “题干信号：符合题意的说法为……”
- “答案含生态价值观、规律与主观能动性。”
- “评分细则要求写某原理。”
- “材料强调本题可从多个角度作答。”
- “文化：中华优秀传统文化、民族精神、文化作用。”

### 3. 为什么能想到合格标准

“为什么能想到”要解释材料为什么触发该原理，而不是说细则提到所以能放。

合格句式：

- “材料中的 A 是客观条件 / 现实变化，它决定了主体必须形成 B 认识，所以触发物质决定意识。”
- “材料同时呈现 A 和 B 两个相反方面，二者不是互相排斥，而是在同一关系中相互依存、相互作用，所以触发对立统一。”
- “材料不是全盘否定旧事物，也不是原样复制旧形式，而是在保留合理内核的同时克服旧局限，所以触发辩证否定 / 守正创新。”
- “材料给出特定对象、群体、场景和问题，说明不能套用统一方案，所以触发具体问题具体分析。”

不合格句式：

- “能想到这个原理，是因为材料给出了直接信号。”
- “材料强调……所以能想到联系观点。”
- “可从某某角度作答。”
- “答案可以落在……”
- “原有框架已记录本题。”

### 4. 答案落点合格标准

答案落点必须是自然答案句：材料真实触发点 + 原理方法论 + 回应设问。

合格例：

- “人工智能时代的客观技术变革和社会巨变，决定人们需要人文学科辨析价值冲突、理解情感和历史记忆，所以人文学科具有不可替代的必要性。”
- “未来人与人工智能不是单方面塑造关系，而是既相互区别又相互依存、相互作用的对立统一关系，应以矛盾观点把握二者在未来发展中的统一。”
- “发展新质生产力不能放弃传统产业，而应守住传统产业的制造基础和市场支撑，同时用未来技术补强创新能力，推动传统产业扬弃式升级。”

不合格例：

- “答案可以落在：材料中的……说明……”
- “写坚持一切从实际出发，尊重规律和发挥主观能动性相结合；用发展观点和辩证否定观说明……”
- “材料体现该原理，应回应设问。”
- “该题可从联系、发展、文化等角度作答。”

### 5. 节点边界

必须按用户框架节点修：

- 物质决定意识：只写客观条件 / 现实变化决定认识或必要性，不写意识反作用。
- 主观能动性 / 意识能动作用：只写人主动认识、判断、设计、选择、引导，不混成物质决定意识。
- 尊重客观规律与发挥主观能动性相结合：必须同时有客观规律 / 条件限制和人的主动作为。
- 联系观点：必须有真实要素之间的相互影响、相互制约、相互作用。
- 发展观：必须有新旧变化、阶段演进、前进上升或动态生成。
- 辩证否定 / 守正创新：必须有“保留合理内核 + 克服旧局限 + 创新发展”，不能只因出现传统文化就套。
- 实践是认识的基础：必须有调查、实验、使用、治理、科研、社会实践推动认识或检验认识。
- 社会存在与社会意识：必须区分社会生产生活条件与思想观念 / 价值意识，不能混进唯物论。
- 价值观导向：必须有价值取向如何引导认识、选择或行动，不能泛写文化自信。
- 人民群众：必须有群众需要、群众参与、群众评价、群众利益或人民主体地位。
