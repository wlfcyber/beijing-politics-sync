# Codex 连续任务步骤文档

## 任务名称

- 选必一《国际政治与经济》三年考题穷尽、六模块归框与零基础满分课稿打磨

## 任务目标

- 围绕 `2024-2026` 北京各区模拟题、期中、期末中所有能稳定落到选必一《国际政治与经济》的题目，继续做穷尽式回收，不漏整套卷、不漏第二小问、不漏跨书综合题里稳定可落到国政经的部分。
- 所有稳定题都要压成 `材料 -> 细则/答题点` 的形式，优先依据细则、评标、阅卷讲评，不把普通参考答案冒充细则。
- 把所有稳定答题点放回用户既定的六模块框架：`时代背景 / 理论 / 经济全球化 / 政治多极化 / 联合国 / 中国`。
- 在穷尽题源和答题点之后，继续把内容打磨成“面对几乎零基础高中生讲一节课也能立刻上手新题”的课堂版本，让学生能从材料直接触发答题点、按模块稳定冲击满分。
- 最终至少同步更新以下主稿：
  - `选必一_国际政治与经济_题目肢解版_材料-细则.md`
  - `选必一_国际政治与经济_课堂压缩版_高频细则点.md`
  - `选必一_国际政治与经济_备课总稿_六模块穷尽版.md`
  - `选必一_国际政治与经济_零基础满分课稿.md`
- 每轮结束同步更新 `reports/governor_board.md`。

## 任务约束

- 只处理选必一《国际政治与经济》，不把纯经济、纯哲学、纯法律题硬并入。
- 时间边界锁定为 `2024-2026`；截至 `2026-04-23`，`2026二模`尚未开考，不纳入。
- 证据优先级严格遵守：`细则 / 评标 / 阅卷讲评明确给分口径 > 用户确认可用的讲评文件 > 其他`。
- 不得把普通参考答案、北京题库主观答案、网传解析静默升级为细则。
- 题号、题面、材料边界、模块归属如果没有锁净，必须诚实写成“排除/暂挂原因”，不能硬并。
- 每一道正式并入题都必须保留来源套卷与题号。
- 所有“材料 -> 答题点”关系都必须是真触发，不许写空洞大词堆砌。
- 课堂化改写必须服务零基础学生提分，重点不是解释教材定义，而是讲清：
  - 什么材料先触发什么模块
  - 什么材料形态先写什么答题点
  - 怎样从一道新题迅速定位到六模块中的答题链
- 每完成一个步骤，都要同步更新本文件和 `codex_连续任务_任务进度文档.md`。
