# Role Contracts

- 决策者: choose next bottleneck; write directives.
- 资料组织者: inventory sources; update source ledger.
- 劳动者: process assigned suites/questions; write findings only unless assigned a write scope.
- 补丁者: check missed multi-point and cross-framework placements after merge.
- 监管者: veto weak evidence, hidden blockers, and fake completion.
- 自动化检测者: verify plan/progress/artifacts/render outputs agree; main thread owns this if no subagent slot.

Each role report ends with `Decision: pass/fail/blocked/needs-merge`.
