# Development Plan

## Completed Strategy

1. Replace textbook-directory ordering with a route-first answering model:
   `设问任务 -> 主体关系 -> 权利客体 -> 法律关系 -> 关键事实/证据 -> 法律规则 -> 责任/救济 -> 价值意义`.
2. Upgrade the coverage matrix before rewriting prose so every case carries a source boundary.
3. Build high-frequency legal-route tables for contract, tort, consumer rights, IP, and procedure.
4. Add lower-frequency but required pathways for property, family/inheritance, and labor.
5. Convert generic meaning statements into material-bound value statements.
6. Preserve gaps instead of fabricating coverage.

## Acceptance Targets

- Each core node has trigger words, legal relationship, knowledge points, student-ready answer sentences, scoring boundary, loss points, and verified sources.
- Each matrix row can return to an original file path and source boundary.
- High-frequency nodes have at least three usable student sentences.
- Consumer/IP/contract/tort/procedure questions are routed by task and facts, not by textbook chapter labels.
