# Phase13 框架版复核报告（候选）

本报告为 Phase13 候选稿的内部复核记录，不作终稿、不作 PASS。

## 1. 结构合规性

### 1.1 主体结构
- 主观题主体按"思维类型 → 思维特点/小方法 → 固定分析流程 → 对应模拟题"组织。
- 推理部分按"题型 → 规则口令 → 常见陷阱 → 解题动作 → 对应模拟题"组织。
- 未采用"主观题/选择题 → 地区 → 时间 → 大题"的旧轴。
- 选择题不再单独成段，而是按其考查的方法或题型挂到对应节点的"易混选择题"小节。

### 1.2 节点划分
思维方法部分共 5 个一级节点：
- 一、科学思维（含客观性、预见性、可检验性、探索性、整体安排）
- 二、创新思维（含联想、逆向、超前、发散与聚合、迁移、改变创造条件、三新）
- 三、辩证思维（含整体性、动态性、分析与综合、质量互变、辩证否定、矛盾分析）
- 四、系统观念
- 五、认识发展历程

推理题型部分共 12 个一级节点：
三段论 / 充分条件假言推理 / 必要条件假言推理 / 联言判断 / 选言判断 / 归纳推理 / 类比推理 / 概念外延关系 / 定义与划分 / 性质判断—换质换位—关系判断 / 逻辑三律 / 复合判断真假与综合题。

第三部分为"边界陷阱与辅助检索"，挂载硬锁标记的非正例题。
