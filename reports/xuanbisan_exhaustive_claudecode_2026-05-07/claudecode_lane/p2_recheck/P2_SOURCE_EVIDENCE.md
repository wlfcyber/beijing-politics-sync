## 2. Q-2025东城期末-13（推理>演绎推理>三段论·中项不周延）

### 2.1 题面（stem & options）— paper.txt 行 110-117

```
13．2024年11月，国资委开设的"国资小新"网上店铺一口气上架了十多款"大国重器"和数款技术过硬
的央企民用产品，让大家感受到央企既"顶天立地"又能保障和改善民生的责任担当。下列三段论推理
违反相同逻辑规则的是（  ）
①"祥云"AS700是大国重器，有的大国重器是载人飞艇，有的载人飞艇是"祥云"AS700
②歼－20飞行员手表有三盘六针，这款手表不是歼－20飞行员手表，这款手表没有三盘六针
③碳纤维自行车适合山地骑行，山地车适合山地骑行，碳纤维自行车是山地车
④石墨烯围巾有智能温控系统，智能温控系统能自动调节温度，石墨烯围巾能自动调节温度

A．①② B．①③ C．②④ D．③④
```

### 2.2 答案与逐项解析 — paper.txt 行 490-509

- **【分析】**：本题考查三段论推理。三段论保真四规则：
  1. 一个形式结构正确的三段论只能有三个不同的项（违反则犯"四概念"错误）；
  2. 中项在前提中至少周延一次（违反则犯"中项不周延"错误）；
  3. 前提中不周延的项在结论中不得周延（违反则大项犯"大项不当扩大"、小项犯"小项不当扩大"）；
  4. 两个否定前提不能必然推出结论（结论为否定，当且仅当前提中有一否定）。
- **【解答】**：
  - ①③符合题意：①中"大国重器"作中项不周延（大前提作肯定判断谓项不周延、小前提"有的大国重器"作主项不周延）；③中"适合山地骑行"作中项不周延（大小前提均作肯定判断谓项不周延），两题违反**相同**逻辑规则——"中项不周延"。
  - ②④不合题意：②"三盘六针"在前提作肯定判断谓项不周延、在结论作否定判断谓项周延，犯"**大项不当扩大**"错误（与中项不周延规则不同）；④大前提"智能温控系统"指作为属性的整体（功能名）、小前提中指作为个体的实物（围巾内零部件），同名异义已是两概念，加上大小项一共四个不同的项，犯"**四概念**"错误（与中项不周延规则不同）。
- **故选：B**。

- manifest framework_node：推理>演绎推理>三段论·中项不周延
- 源支持：①③共同违反"中项不周延"规则；framework_node 精确锁定共同违规规则。②④分别违反"大项不当扩大"与"四概念"规则——属于诱饵识别，但本题考查的核心规则正是 framework_node 标定的"中项不周延"。

### 3.1 题面（stem & options）— paper.txt 行 118-124

```
14．北京中轴线位于北京老城中心，是统领整个老城规划格局的建筑与遗址的组合体，由古代皇家宫苑建
筑、古代皇家祭祀建筑等遗址组成，是中华文明"中""和"文化传统的特殊见证。下列选项正确的是

```

### 3.2 答案与逐项解析 — paper.txt 行 510-542

- **【分析】**：本题考查性质判断。
  - 判断含义：对认识对象情况有所断定的思维形式。
  - 判断分类：①依据"判断本身是否包含其他判断"分简单判断（含**性质判断（直言判断）**+关系判断）和复合判断；②依据判断的质划分，直言判断分肯定判断和否定判断；依据判断的量划分，直言判断分全称、特称、单称三种；从质和量结合上划分，**直言判断有六种基本判断形式**：全称肯定、全称否定、特称肯定、特称否定、**单称肯定**、单称否定。
  - 关系判断：依据对称性分对称/反对称/非对称；依据传递性分传递/反传递/非传递。
- **【解答】**：
  - A错误：属种关系体现共性与个性的关系，"北京中轴线"和"古代皇家宫苑建筑"是**整体与部分**的关系；
  - B错误："位于"关系为**反对称**关系（不是传递关系）；
  - C错误："北京中轴线"的"中"指地理位置，"中华文明"的"中"指中华民族，**外延不一致**；
  - D**正确**：北京中轴线是统领整个老城规划格局的建筑与遗址的组合体，这是一个**简单性质判断**。"统领整个老城规划格局的建筑与遗址的组合体"是**谓项**，**联项是肯定的**，因此**谓项是不周延的**。

## 4. Q-2025东城期末-15（推理>演绎推理>充分条件假言推理）

### 4.1 题面（stem & options）— paper.txt 行 125-131

```
15．为了传承红色基因，赓续红色血脉，某班开展红色之旅研学活动，可选活动方式有：参观北大红楼、
陈独秀旧居、李大钊故居。赵同学和李同学约定，如果赵同学参观北大红楼，那么李同学参观陈独秀旧
居或者参观李大钊故居。如果上述约定为真，以下判断必定为假的是（  ）
A．只有赵同学去北大红楼，李同学才去陈独秀旧居或者去李大钊故居
B．虽然赵同学去北大红楼，但李同学既不去陈独秀旧居也不去李大钊故居
C．即使赵同学不去北大红楼，李同学也会去陈独秀旧居
D．如果李同学去李大钊故居，那么赵同学去北大红楼
```

### 4.2 答案与逐项解析 — paper.txt 行 543-564

- **【分析】**：本题考查假言推理及其方法。三种假言推理规则：
  1. **充分条件**假言推理：肯前→肯后；否前→不能否后；否后→否前；肯后→不能肯前。
  2. **必要条件**假言推理：否前→否后；肯前→不能肯后；肯后→肯前；否后→不能否前。
  3. **充分必要条件**假言推理：肯前→肯后；肯后→肯前；否前→否后；否后→否前。
- **【解答】**："如果赵同学参观北大红楼，那么李同学参观陈独秀旧居或者参观李大钊故居"属于**充分条件假言判断**P→Q。
  - A不合题意："只有赵去……李才去……"为真（必要条件改写真）；
  - B**符合题意**："虽然赵去（肯前）……李既不去陈也不去李大钊（否后）"——属于**肯前+否后**，违反充分条件假言推理"肯前必肯后"规则，因此**该判断必定为假**；
  - C不合题意："即使赵不去……李也会去……"——否前不能否后，C为真；
  - D不合题意："如果李去……那么赵去……"——肯后不能肯前但表述为可能真，D为真。
- **故选：B**。

### 4.3 框架节点对照

- manifest framework_node：推理>演绎推理>充分条件假言推理
- 源支持：B 项违反充分条件假言推理"肯前必肯后"规则，精确锁定 framework_node。诱饵 A（必要条件改写）、C（否前不能否后）、D（肯后不能肯前）——属于充分条件假言推理四规则的逐项诱饵展开。
- evidence_level：B-choice-signal（manifest 原值，未上调）
