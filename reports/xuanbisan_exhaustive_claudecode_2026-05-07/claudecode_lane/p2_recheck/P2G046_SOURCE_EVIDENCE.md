### 4.3 官方符号化推理（046_*support__26东城一模细则.pdf 第9页）

```
太阳风带电粒子：A     星际中性粒子：B            外日球层衍生例子：C
7. ……
实验一：逃逸粒子除非是太阳风带电粒子，否则就是星际中性粒子。  A←-B   假：-A-B
实验二：逃逸粒子或者不是星际中性粒子，或者不是外日球层衍生粒子。  -B∪-C  假：B C
实验三：逃逸粒子如果不是外日球层衍生粒子，就不是星际中性粒子。  -C→-B  假：-C B
```

### 4.4 真值表（046_*support__18+选择7、13.pptx 第5-6 张幻灯片）

```
A ← -B          真：A-B  -A B  A B
-B ∪ -C         真：-B-C  -B C  B-C
-C → -B         真：-B-C  -B C  B C
联合得 -B 真（B 假），A 真，C/-C 自由（√标 A 真 / C 与 -C 均可）
```

### 4.5 推理重现

- 实验一'除非 A 否则 B'≡ ¬A→B ≡ A∨B
- 实验二'¬B∨¬C'≡ ¬(B∧C) ≡ B→¬C
- 实验三'¬C→¬B'≡ B→C（逆否）

实验二 B→¬C 与实验三 B→C 联立：若 B 真则 ¬C 与 C 同真，矛盾；故 B 必假。

B 假代入实验一 A∨B：A 必真。

C 在 B 假下不被三条前提唯一限定（B→¬C 与 B→C 在 B 假时均空真，B 假与 ¬B∨¬C 在 B 假时空真），故 C 真假未定。

→ A、B 选项关于 C 的断言不必然；C 选项'是星际中性'与推论矛盾；D 选项'是太阳风带电'即 A 真，唯一必然为真。

### 4.6 framework 一致性

manifest evidence_level=B-choice-signal；题干'除非……否则……'+'或者不是……或者不是……'+'如果不是……就不是……'三种复合命题串联，正是'复合假言推理'框架节点的标准识别物，evidence_level 维持不变。
