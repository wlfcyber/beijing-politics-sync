# 经济全球化节点合并表 v5

| 节点 | v5 处理 | 不合并边界 |
|---|---|---|
| 两个市场两种资源 | 独立节点 | 不与开放型世界经济、贸易自由化便利化合并。 |
| 贸易和投资自由化便利化 | 独立节点 | 只处理市场准入、零关税、贸易投资便利，不吞并规则制定。 |
| 开放型世界经济 / 普惠包容经济全球化 | 只有同一题同点列为可替代表达时合并 | 不吞并开放型经济水平、全球经济治理规则制定、多边贸易体制。 |
| 全球经济治理和规则制定 | 按规则/标准/话语权功能独立 | 不把企业出海规则环境、国际标准话语权、发展中国家代表性压成一个大筐。 |
| 全球产业链供应链稳定畅通 | 独立节点 | 不与两个市场两种资源或高水平对外开放混合。 |
| 多边贸易体制 | 独立节点 | 不与联合国国际体系混同；它属于经贸秩序。 |
| 数字贸易与数字经济跨境流通 | 独立节点 | 不并入一般贸易便利化。 |
| 绿色贸易 / 绿色经济合作 | 独立或同题组处理 | 只在题面明确绿色贸易、气候经济合作、能源合作时使用。 |
| 高水平对外开放 / 制度型开放 | 作为经济全球化下中国开放机制处理 | 若题目明显属于必修二开放发展理念，则仅在跨模块附录提示。 |
