# 经济全球化节点合并表

| 学生总图类别 | 包含节点 | 不合并边界 |
|---|---|---|
| 市场与资源流动 | 两个市场、两种资源；国内国际双循环；海南封关；全球要素集聚。 | 不并入贸易自由化便利化，除非材料明确是通关、市场准入或贸易成本。 |
| 贸易投资便利 | 贸易自由化便利化；降低贸易成本；扩大市场准入；零关税。 | 不把“开放型经济水平”泛化进来，除非题目问中国开放制度或开放水平。 |
| 开放型世界经济 | 建设开放型世界经济；推进普惠包容经济全球化；开放、包容、普惠、平衡、共赢。 | 不吞并规则制定、供应链、世贸组织等有独立细则原词的节点。 |
| 产业链供应链 | 维护全球产业链供应链稳定畅通；链博会；保税维修；供应链韧性。 | 不等同于一般合作共赢；如果问国际关系结构，转政治多极化。 |
| 规则与标准 | 参与全球经济治理和规则制定；国际标准；数据规则；规则话语权。 | 不并入多边贸易体制，世贸组织题单列。 |
| 数字与绿色贸易 | 数字贸易、数据跨境、绿色贸易、技术共享、创新型开放型世界经济。 | 不把所有科技创新题都归这里；若是综合国力竞争，转理论或中国。 |
| 多边贸易体制 | 维护以世贸组织为核心的多边贸易体制；反对贸易保护主义。 | 不与联合国体系混合；联合国属于政治/安全/发展框架，不是 WTO。 |

## 学生使用规则

- 先看材料关键词归入七类，再选具体卷面句。
- 七类是导航，不是替代答案；真正卷面句仍要接材料事实。
- 同一题可同时命中两类，但要分句写，不要把不同术语压成一串。
