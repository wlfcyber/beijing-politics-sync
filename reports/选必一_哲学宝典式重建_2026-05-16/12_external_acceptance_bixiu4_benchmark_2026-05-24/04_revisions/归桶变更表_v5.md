# 归桶变更表 v5

| 关键答题点 | v5 最终主桶 | 处理规则 |
|---|---|---|
| 合作共赢的新型国际关系 | 政治多极化与国际秩序 | 回答国际关系和国际秩序方向，不能放入理论桶。 |
| 独立自主和平外交政策 | 中国 | 回答中国外交政策立场，不能放入理论桶。 |
| 和平共处五项原则 | 中国 | 作为我国对外关系基本准则进入中国外交政策链。 |
| 国家间共同利益是国家合作的基础 | 理论 | 只在解释合作为什么成立时进入理论桶。 |
| 正确义利观 | 中国为主，理论作限定补充 | 中国行动、发展合作、共享机遇、发展中国家民生语境归中国；只有国家利益理论解释层才进入理论。 |
| 人类命运共同体 | 政治多极化与国际秩序为主，中国为副链 | 问国际关系新思路、全球治理方向时归政治多极化；问中国方案、中国担当时同题组转入中国。 |
| 共商共建共享全球治理观 | 政治多极化与国际秩序为主 | 问治理原则和秩序方向时归政治多极化；涉及联合国宪章时同题组提示联合国链。 |
| 全球发展倡议/全球安全倡议/全球文明倡议/全球治理倡议 | 中国 | 作为中国主张、中国方案、中国公共产品进入中国桶。 |
| 联合国宪章宗旨和原则 | 联合国 | 只处理联合国框架、宪章、安理会、2030议程等信号。 |
| 以联合国为核心的国际体系 | 联合国 | 与政治多极化中的全球治理方向相邻，但主链为联合国框架。 |
| 维护多边贸易体制 | 经济全球化 | 与联合国框架不同，侧重贸易规则和多边经贸秩序。 |
