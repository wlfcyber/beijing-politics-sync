# 经济全球化

## 经济全球化总图

【二级定位表】

| 材料信号 | 优先归类 | 不要混成 |
| --- | --- | --- |
| 经济全球化趋势、国际分工、全球市场联系 | 经济全球化的表现和影响 | 不直接替代具体开放措施 |
| 开放型世界经济、普惠包容、共同繁荣 | 开放型世界经济和经济全球化正确方向 | 不等于我国开放型经济水平 |
| 零关税、市场准入、通关、免签、自贸协定 | 贸易投资自由化便利化 | 不等于全球治理观 |
| WTO、多边贸易体制、贸易保护主义、单边关税 | 全球经济治理和多边贸易规则 | 不等于一般国际关系合作 |
| 产业链、供应链、链博会、保税维修、跨境生产 | 产业链供应链稳定畅通 | 不等于“两个市场两种资源” |

| 两个市场、两种资源、海南封关、国内国际双循环 | 两个市场两种资源和国内国际循环 | 不等于开放型世界经济 |
| 共建“一带一路”、周边自贸、开放平台、展会平台 | 开放平台与共建合作 | 需看材料落在经贸机制还是政治关系 |

【快速判断】经济全球化桶只收“贸易、投资、市场、产业链供应链、规则、资源配置、开放平台”这些经济机制。中国主张、中国担当、联合国宪章、国际秩序方向分别回到中国、联合国和政治多极化桶。
【本模块审题步骤】
- 材料怎么看：先看贸易、投资、市场、产业链供应链、规则、平台、资源流动和开放合作。
- 该写哪个术语：国际经济循环、资源配置、开放型世界经济、贸易投资自由化便利化等放在本模块。
- 为什么触发：经济全球化点要说明中国如何通过开放合作改善世界经济运行。
- 答案句怎么落：全球化机制 + 材料里的开放或合作事实 + 对互利共赢、共同繁荣或稳定供应链的作用。
- 考场动作：先分清市场资源、贸易投资、规则标准、产业链供应链、数字绿色贸易等机制，再写对应卷面句。

【易错边界】
- 本模块负责写开放、市场、贸易、投资、产业链供应链、经济规则和共享发展成果等具体作用链。
- 经济全球化可以作为时代背景出现，但当题目问具体经济合作机制、开放平台或全球经济治理时，应作为主答点展开。
- 本模块同类项合并必须看设问提示表述是否接近、卷面是否可替代；不能把“开放型世界经济”“开放型经济”“全球经济治理和规则制定”“贸易自由化”等只因本质相关就压成一类。
