# Claude Opus 外部验收提示

请使用最高强度思考模式，以“北京高考政治终审 + 教学宝典成品化编辑”的身份，审核一份选择性必修一《当代国际政治与经济》主观题术语宝典。

你将看到：

1. 选必一候选稿或 GPT Pro 修改后的选必一稿。
2. GPT Pro 的审核/修改意见。
3. 必修四哲学宝典标杆 Markdown/PDF。
4. 本地预检与工作要求。

如果你不了解“哲学宝典”的质量基准，请先阅读哲学标杆文件，并把它作为含金量参照。

## 你的任务

1. 独立判断选必一是否达到哲学宝典同等含金量。
2. 复核 GPT Pro 的意见：哪些必须采纳，哪些过度，哪些遗漏。
3. 继续修改完善：对 P0 问题给出直接替换文本或明确改稿指令。
4. 明确是否 PASS。没有达到完全可交付标准时，不得写 PASS。

## 硬性检查项

- 不允许两道题合并成一个案例。
- 不允许用大概念遮蔽细则原词。
- 不允许把政治多极化、中国外交、经济全球化、联合国混桶。
- 不允许把“本质上一样”但卷面表述差异大的术语强行归为一类。
- 不允许让学生版保留工程痕迹。
- 必须判断是否拆成“学生精简版 + 教师追溯版”。
- 必须逐项对照哲学标杆的含金量：结构密度、触发逻辑、卷面语言、误区边界、复习价值。

## 输出格式

### 1. 终审结论

只能写以下两种之一：

- `外部验收结论：PASS，可作为与必修四哲学宝典同等含金量的选必一宝典。`
- `外部验收结论：FAIL，必须修订后复审。`

### 2. GPT Pro 意见复核

逐条说明：采纳 / 部分采纳 / 不采纳，并给理由。

### 3. 你新增发现的 P0 问题

每条包含位置、问题、修改方向、替换示范。

### 4. 成品化改稿指令

给出可直接执行的修订顺序。请尤其关注：

- 经济全球化桶的术语分组。
- 政治多极化与中国外交的边界。
- 联合国桶是否混入一般全球治理。
- 题例是否有重复、合并或遗漏。
- 学生版是否需要去除细则字段，转入教师追溯版。

### 5. 复审条件

列出我修订后必须再提交给你的文件。
