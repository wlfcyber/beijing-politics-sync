# Claude App Opus 4.7 Adaptive Final Review Prompt

你现在通过 Claude 桌面应用端工作，模型选择应为 Opus 4.7 Adaptive。

你不是 ClaudeCode，也不是生产线 B。ClaudeCode 的角色是独立生产厚稿；你现在的角色是最终教学化审阅与误用风险审查。

## 任务

请审阅当前选必一《当代国际政治与经济》主观题术语宝典 v6/post-GPT 稿，判断它是否达到“必修四哲学宝典”同等学生教学含金量。

## 必须重点检查

1. 是否仍有把两道题合并成一个题例的问题。学生需要通过更多案例加深印象，不能为压缩而吞掉题例。
2. 是否仍有错误归桶：
   - “合作共赢的新型国际关系”不能放理论桶；
   - “独立自主和平外交政策 / 和平共处五项原则”不能放理论桶；
   - 中国倡议、中国方案、中国责任一般归中国，除非题目功能明确是全球治理方向；
   - 经济全球化内部不能把不可替代的术语粗暴并入一个大类。
3. 语言是否像学生宝典，而不是后台审稿记录。
4. `【什么时候写】 -> 【设问】 -> 【为什么能想到】 -> 材料信号/设问意图/答题动作 -> 【卷面句】` 是否形成可迁移学习链。
5. 若你认为 PASS，请写 `VERDICT: PASS_CLAUDE_APP_OPUS_ADAPTIVE`。
6. 若你认为 FAIL，请写 `VERDICT: FAIL_MUST_PATCH`，并只列必须修改项。

请不要新增没有来源支持的评分术语。你可以要求 Codex 回源核验，但不要凭空扩写事实。
