# ClaudeCode 原始文件核验日志

本厚稿由 ClaudeCode 独立完成。流程为：先从桌面原始目录递归清点 22 个原始文件，逐一调用对应阅读器抽取文本；只有在完成自有抽取并写入 `02_claudecode_independent/cc_raw_text/<district>/...` 之后，才回过头用 `00_extracted_text` 做交叉核验。Codex 已抽取的 `texts/` 仅作辅助比对，不作为主依据。

## 0. 工具栈与方法

| 类型 | 工具 | 备注 |
| --- | --- | --- |
| `.docx` 阅卷细则/试卷 | `python-docx` 段落 + `[TBL]` 单元格平铺 | 8 个文件全部成功 |
| `.pptx` 阅卷细则 | `python-pptx` 段落 + 表格平铺 | 1 个文件成功（70 张幻灯片） |
| `.pdf`（带文本层） | `PyMuPDF (fitz)` `get_text("text")` | 多数 PDF 都有文本层 |
| `.pdf`（扫描件，文本层 < 100 CJK 字符） | `PyMuPDF` 渲染成 300 DPI PNG → `tesseract --psm 6 -l chi_sim+eng` | 自行 winget 安装 Tesseract 5.4 + 下载 `chi_sim.traineddata` 到 `C:\Users\Administrator\AppData\Local\tessdata` |
| `.doc`（legacy WordDocument 流） | 自写二进制扫描：按 utf-16-le 解码后保留 CJK / ASCII 可打印 / 标点 | 1 个文件成功，正文可读，表格行序混淆但要点完整 |

抽取脚本：`scripts/cc_independent_extract.py`、`scripts/cc_ocr_two.py`、`scripts/cc_ocr_haidian.py`、`scripts/cc_ocr_fangshan.py`。日志：`02_claudecode_independent/cc_raw_text/_extraction_log.txt`。

## 桶一：时代背景

### 1.1【时代主题（和平与发展）】
- 完整设问：海淀 20(2)（7 分）"结合材料，运用《当代国际政治与经济》知识，围绕"全球发展倡议从合作理念到丰富实践"这一主题，撰写一篇短评"。
- 细则位置：`2026海淀二模/细则/26海淀高三政治二模讲评.pdf` 第 82 页 20(2) 评分细则"必要性 1 分：时代主题 / 全球化多极化 / 共同利益"。必答性质——三选一即可得 1 分。
- 来源：2026 海淀二模 Q20(2)。
- 材料触发：题干列举"全球发展倡议提出五周年""80 多个国家加入'全球发展倡议之友小组'""全球共享发展行动论坛""全球发展命运共同体"等中方主导的国际合作实践，要求解释"为什么这些理念能落到实处"，背景必须从"和平与发展是时代主题"切入。
- 答案句：和平与发展是时代主题，求和平、谋发展、促合作、图共赢是不可阻挡的时代潮流；全球发展倡议正是顺应这一时代主题，把"团结合作"作为应对全球发展挑战的总钥匙，从而能从中国主张扩展为国际共识。

### 1.2【时代发展主题】
- 完整设问：石景山 18（8 分）"结合材料，运用《当代国际政治与经济》知识，分析中国与东盟经贸合作能够提质升级的原因"。
- 细则位置：`石景山区高三政治第二次模拟考试答案评分细则(1).doc` Q18 第一段"国家利益和国家实力是影响国际关系的决定性因素 …… 顺应经济全球化趋势，基于双方共同利益，符合时代发展主题（从政治、经济各答出 1 个观点，得 4 分）（依据）"。
- 来源：2026 石景山二模 Q18。
- 材料触发：题干强调"中国连续 16 年保持东盟第一大贸易伙伴地位""东盟连续 5 年成为中国第一大贸易伙伴""中国—东盟经贸合作逆势上扬"，对应宏观背景必须从"和平与发展时代主题"启动。
- 答案句：和平与发展是当今时代主题，中国—东盟连续多年互为对方第一大贸易伙伴，正是双方在和平合作的时代主题下深化经贸联系的体现，符合时代发展潮流。

### 1.3【共同利益 / 国家利益是国际关系的决定性因素】
- 完整设问：石景山 18（同上）。
- 细则位置：`石景山区高三政治第二次模拟考试答案评分细则(1).doc` Q18："国家利益和国家实力是影响国际关系的决定性因素 …… 基于双方共同利益"。
- 来源：2026 石景山二模 Q18。
- 材料触发：中国与东盟在数字经济、绿色发展、规则共建三个方向同时推进，"中国—东盟自贸区不断升级"，材料指向共同利益是合作动力。
- 答案句：国家利益是国际关系的决定性因素，国家间的共同利益是合作的基础；中国与东盟在数字基础设施、绿色低碳、自贸区 3.0 版升级中存在广泛的共同利益，这是双方经贸合作能够稳步前行、提质升级的根本依据。

### 1.4【国家利益和共同利益（数据治理）】
- 完整设问：房山 20（8 分）"结合材料，运用《当代国际政治与经济》知识，分析世界数据组织完善全球数据治理，服务全球数字经济发展的智慧"。
- 细则位置：`26房山评标.docx` Q20"♣1 分 共同利益 / 共同发展"。
- 来源：2026 房山二模 Q20。

- 答案句：世界数据组织基于各国数据发展的共同利益开展合作，推动技术联合研发与数字人才培训，把共同利益做大，是其能完善全球数据治理的根本起点。

## 桶二：理论

### 2.1【国家利益和国家实力是影响国际关系的决定性因素】
- 完整设问：石景山 18（8 分，见 1.2）。
- 细则位置：`石景山区高三政治第二次模拟考试答案评分细则(1).doc` Q18 评分要点首句。该点为"政治角度+经济角度各 1 个观点 = 4 分"中的政治维度核心命题。

# 归类风险复核

> 本节复核与本批补入直接相关的术语归类，逐条说明本厚稿的取舍及理由。

### A.【合作共赢 / 新型国际关系】→ 桶四"政治多极化"，**不**进"理论"桶
- 朝阳 20(2) 角度 3"提供中国方案：可以代替的关键词——国际新秩序 / 新型国际关系等"。该角度评分明确将"新型国际关系"作为推动构建更加公正合理国际治理体系（即新国际秩序方向）的关键词。
- 海淀 20(2) 重要性"推动国际秩序朝着更加公正合理方向发展"。
- 按归类硬规则——"`合作共赢的新型国际关系`、国际秩序、全球治理方向，归入 `政治多极化`，不要放入 `理论`"——这两条朝阳/海淀的"合作共赢"和"新型国际关系"以及房山的"合作共赢 / 互利共赢"全部入桶四 §4.2。理论桶（桶二）仅放石景山 18 的"国家利益和国家实力是影响国际关系的决定性因素"§2.1，这是国际关系基本原理。

### B.【正确义利观】→ 全部归入桶五"中国"，**不**进"理论"
- 海淀 20(2) 与房山 20 是本批仅有的两处明确出现"正确义利观"的题。
- 海淀题的材料重心是中方对发展中国家可持续造血能力的提升、中非三方合作、全球妇女能力建设；房山题的材料重心是 WDO 推动弥合低收入国家数字鸿沟、共享数据红利。两题都属于"题目重心是中国行动、中国担当、发展合作、共享机遇或发展中国家民生"，按归类硬规则归入"中国"桶 §5.3。
- 凡未来若遇到细则把"正确义利观"与"国家利益角度 / 理论逻辑 / 合作基础可选表达"并列（即作为国际关系理论命题）的题目，再考虑放入"理论"桶；本批两题均不符合此例外。

### C.【独立自主和平外交政策 / 和平共处五项原则 / 独立自主的基本立场】→ 桶五"中国"
- 本批 8 区 8 道（含未入库 2 题）选必一主观题均**未出现**此组术语。本厚稿不新增该桶条目。日后若一模/二模出现"独立自主""不结盟""和平共处五项原则""不干涉内政"等词，遵循硬规则全部归桶五。

### D.【全球发展倡议 / 全球安全倡议 / 全球文明倡议 / 全球治理倡议】→ 桶五"中国"
- 海淀 20(2) 的材料与设问完全围绕"全球发展倡议"展开，作为中国主张、中国方案、中国公共产品出现，故"全球发展倡议从合作理念到丰富实践"对应的"中方搭建平台 / 三方合作 / 提升发展中国家造血能力"全部入桶五 §5.2，而其使用的"共商共建共享 / 多边主义 / 国际秩序更加公正合理 / 经济全球化朝着开放包容普惠平衡共赢方向发展"按其本义分别入桶四与桶三。
- 西城 19(1) 类似：中方倡议设立 WDO 这一行为属"中国主张 / 中国方案"，主桶五；但具体说明 WDO 顺应"经济全球化 / 数字经济全球化"与"共商共建共享 / 国际关系民主化"的部分按术语本义分别入桶三与桶四。

### E.【经济全球化细分术语：开放型经济 / 制度型开放 / 全球经济治理和规则制定 / 多边贸易体系 / 普惠包容的经济全球化 / 贸易自由化 / 数据要素跨境流动 等】→ 不硬合并
- 东城 20(3)"高水平对外开放 / 开放型经济 / 制度型开放 / 投资自由化便利化"分列为三条 §3.3、§3.4、§3.5；
- 房山 20"普惠包容的经济全球化 / 数据要素全球流动"分列于 §3.1、§3.6；
- 西城 19(1)/19(2)"数字经济全球化 / 主权平等开放合作包容普惠安全保障"分列于 §3.6、§3.8；
- 石景山 18"两个市场两种资源 / 区域经济合作 / 推动经济全球化朝着开放包容普惠平衡共赢方向发展"分列于 §3.7、§3.2。
- 上述细则在阅卷时**没有任何一处**写明"以下表达可互换得 1 分"——例如房山评标的 8 个 ♣ 各占 1/2 分，是**累加**关系而非互换关系；东城 20(3) 的三层结构（"购在中国对消费的影响→开放红利→对投资中国的影响"）也是逐项叠加。故本厚稿严格按术语本身入桶，不做硬合并。

### F.【缺细则两题（丰台 Q20 自贸区、顺义 Q20 今天的中国明天的世界）】→ 不入桶
- 这两道题在试卷一侧均为典型选必一主观题，归类大方向清晰（丰台 Q20 主体在制度型开放 / 高水平开放型经济 / 中蒙俄经济走廊；顺义 Q20 主体在中国式现代化对世界开放、共享机遇、菌草技术 / 鲁班工坊属"中国"桶），但因**无正式评分细则**支撑"必答 / 必答替换 / 必答可选"的关系，按任务约束"找不到正式细则的题目，不进入主表，写入阻断清单"，故仅记入 §3。日后获取相应教研材料后可按本厚稿同款结构补入。
