# 2026丰台期末_细则 page 73

## text_layer

1.开展核心概念辨析专项训练，通过填空、判断改错强化知识记忆，以
主干知识或时政热点为核心，构建跨模块知识网络，串联多模块知识。
2.培养审题能力，指导学生抓取材料关键词、精准解读题意，进行知识
迁移，行文上逻辑自洽。在构建答案中，有针对性的对回应问题进行练
习。
3.强化逻辑思维训练，紧扣素养导向，针对题型，给出作答思路或维度，
结合试题，进行理解类试题复练，引导学生以知识为基、在试题情境中
解决实际问题。
