# 2026丰台期末_细则 page 15

## text_layer

1.空白卷较多，直接全部或部分抄材料，没有任何分析。因此0分卷比较多，建议
老师们引导学生重视理解材料，要根据关键语句适当结合知识做分析。
2.知识掌握不扎实，哲学世界观和方法论对应不正确。比如：联系具有普遍性，要
用发展的观点看问题，等等。建议老师们再进一步引导帮助学生整合哲学教材知识
体系与核心观点，精准落实重点知识。
3.审题的知识要求不清楚，没有从哲学、文化两部分知识维度作答，还有不少学
生写了《逻辑与思维》的知识中的形象思维与抽象思维、创新思维、认识发展过
程等知识，建议继续巩固知识，通过典型习题训练学生有效分析材料，根据关键
语句精准链接到典型知识，并进行合情合理的分析。
4.罗列知识现象严重，建议老师通过典型题引领学生进行材料分析,努力实现知识
与材料的有机结合。
