# 2026丰台期末_细则 page 40

## text_layer

1.培养学生正确审题能力，学会阅读材料和把握材料的关键词；
2.注重学生的思维逻辑能力训练，引导学生理解当前的考试导向不是考知识而
是考能力和素养，需要学生学会在试题情境中去解决“真问题”，因而学生需要
在把握知识的基础上，运用知识解决经济现象。平时训练可以增加推导类建构。
3.引导学生关注时政热点，能够读时政和思时政，学会分析综合的思维。
