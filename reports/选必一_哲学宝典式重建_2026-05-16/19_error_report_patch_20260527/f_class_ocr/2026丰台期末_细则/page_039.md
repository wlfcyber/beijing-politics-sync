# 2026丰台期末_细则 page 39

## text_layer

1.审题方面，本题需要学生回应两个问题，部分学生忽视“有利条件”这一问题，
没有概括和总结；
2.获取信息能力方面，部分同学获取材料关键信息的能力有待提高，无法结合分
段材料提取出关键信息组织答案；
3.学科语言运用方面，部分学生直抄材料或者无学科用语转化，无法精准对标五
经济术语；
4.逻辑思维能力方面，本题学生体现最明显的不足是经济逻辑推导出现“起点不
明确，终点不落地”的现象。学生作答无法对应设问中指向的“作用于人工智能推
动经济高质量的层次”，前后逻辑不成立，且不指向人工智能；
5.综合素养能力方面，学生在本题中未上升到“市场和政府”结合的层次上，因
而出现高分段失分现象。
