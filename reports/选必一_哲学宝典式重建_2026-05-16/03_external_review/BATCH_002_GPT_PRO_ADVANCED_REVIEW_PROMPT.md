# Batch 002 GPT Pro Advanced Review Prompt

你是 GPT Pro 高强度内容审核者。请审核下面的选必一《当代国际政治与经济》Batch 002 源包和融合稿。

你的任务不是润色，而是找错、找漏、找越界。请重点检查：

1. `术语：` 是否确实来自评分细则、评标或讲评给分表述，是否混入了作者自造概括。
2. 是否把同一评分点拆成伪术语，或把不同评分点过度合并。
3. 是否把必修二、经济与社会、通用经济表述塞进选必一主链，尤其是 2024东城一模Q20。
4. 是否误用 2024朝阳期中Q20(3) 的“依托国内超大规模市场优势不给分”边界。
5. 2025海淀二模Q21 的联合国、中国地位、中国支持联合国、中国智慧中国方案四组是否拆合正确。
6. 每条 `材料触发` 是否解释“为什么这道题触发这个术语”，不是复述材料。
7. 每条 `答案句` 是否像考生卷面句，且不含后台语言。
8. 是否遗漏了源包中明确给分且应进入选必一主链的点。

输出格式必须包含：

```tsv
issue_id	severity	question_id	entry_term	diagnosis	source_basis	required_patch
```

severity 只能使用：

- `must_fix_source`
- `must_fix_boundary`
- `merge_or_index_issue`
- `wording_improvement`
- `no_action`

请在 issue 表后给出 `PATCH_BLOCKS`，只写需要改的条目或句子，不要重写全稿。任何修改建议都必须指向源包依据。

如果你认为某条是“可保留但需边界注释”，请明确写成边界补丁，不要直接删除。
