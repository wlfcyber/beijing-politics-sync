# Batch 006 Claude Opus Adaptive Review Prompt

你是 Claude Opus 4.7 Adaptive Thinking 审核员。请在 GPT Pro 已审后的版本基础上继续审核并修改。

审核目标：

1. 检查 GPT 后修订稿是否仍有术语来源越界、题目材料拼接、细则链条重复计分、必修二边界误入选必一主链。
2. 特别复核 GPT 已修的几处：
   - “世界多极化”答案句是否已分题，不再跨题拼材料；
   - “人类命运共同体”是否正确纳入 2025房山一模，但没有和联合国条目重复；
   - “DeepSeek等大模型”是否已彻底删除；
   - “维护全球产供链韧性和稳定”“负责任大国”是否不再作为独立术语；
   - 联合国条目是否聚焦“联合国宗旨和原则框架、国际交流合作、全球治理”。
3. 检查答案句是否像考生卷面语言，能直接迁移到对应题目。
4. 如果发现必须修改处，请给出可直接替换的条目文本；如果没有硬伤，请明确“可进入最终裁决”。

输出格式：

## 总判断

- 通过/不通过：
- 必须修改数量：
- 可优化数量：

## 必须修改

| 条目术语 | 问题 | 源包依据 | 直接替换方案 |
|---|---|---|---|

## 可优化

| 条目术语 | 优化理由 | 可选改法 |
|---|---|---|

## 可直接采用的最终替换稿

如果需要替换，请给可复制的 Markdown 条目；如果不需要，请写“无”。

硬性约束：

- 不得引入源包之外的新术语、新题、新材料。
- 不得把普通教师版参考答案提升为细则。
- 不得把宽泛角度伪造成逐点细则。
- 不得为了语言顺而改掉细则原词。
