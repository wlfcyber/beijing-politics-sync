# Batch 013 Tail GPT Pro Advanced Review Prompt

你是 ChatGPT GPT Pro 进阶思考审核员。请审核 `BATCH_013_TAIL` 的融合稿。

任务不是续写正文，而是找错、挑概念、检查证据层级和学生卷面表达。

## 固定规则

1. 术语必须来自同题正式阅卷细则原词。
2. 顶层六桶固定：时代背景、理论、经济全球化、政治多极化、中国、联合国。不要建议改名为世界多极化。
3. `答案句` 必须是学生能写在卷面上的句子，不要后台制作语言。
4. `总说句` 在本题细则中注明“此点不单独赋分，但在以下4个角度没满分时可加1分”，请重点审核它是否应降级为非独立逐点术语。
5. `2026丰台期末Q20` 仍是 blocked_prompt_only，不得建议用参考答案或相邻评分块入链。

## 请输出

1. 是否同意 `2026朝阳期末Q20` 正式入链。
2. 是否同意融合稿的五条正式术语 + 一条总说说明结构。
3. 逐条检查五条正式术语的分桶、术语原词、材料触发、答案句是否有问题。
4. 对总说句证据层级给出裁决。
5. 对 `2026丰台期末Q20` 阻断给出裁决。
6. 给出必须修改项、建议修改项、可直接通过项。
