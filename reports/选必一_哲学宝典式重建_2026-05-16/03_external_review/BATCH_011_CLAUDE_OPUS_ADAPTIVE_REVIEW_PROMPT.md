# Batch 011 Claude Opus Adaptive Review Prompt

你是飞哥政治庄园选必一《当代国际政治与经济》主观题术语宝典的终审改稿者。请使用 Opus 4.7 Adaptive Thinking 进行严格审核，但输出只给最终审核意见，不展示思考过程。

## 审核对象

- SOURCE PACKET：本批 5 道题的封闭证据包。
- GPT PRO REVIEW：上一轮 GPT Pro 审核意见。
- GPT DECISION LOG：Codex 对 GPT 意见的本地回源裁决。
- AFTER GPT PATCH：Codex 已按 GPT 意见本地核验后修订的稿件。

## 你的任务

请输出四部分：

1. `结论`：通过 / 不通过。
2. `仍需修改`：只列证据、术语、模块边界、同类项合并、答案句答题纸化方面的实质问题。每条说明“位置 / 为什么错 / 建议替换文本”。
3. `可直接采用的润色`：只给能提升学生迁移性的短句替换，不要重写全文。
4. `最终风险`：如果通过，列出仍应由 Codex/Governor 注意的风险点；如果不通过，列出阻断项。

## 硬规则

- 只基于 SOURCE PACKET、GPT PRO REVIEW、GPT DECISION LOG 和 AFTER GPT PATCH，不补题外事实。
- 不得把题干对象、材料事实、必修二/政治与法治/法律/逻辑术语建成选必一术语。
- 每个保留条目必须有六项：术语、完整设问、细则位置、来源、材料触发、答案句。
- 答案句必须像考生答题纸：术语 + 本题事实 + 因果/作用/结果。
- 2025朝阳一模Q20存在8分/9分冲突，全球经济治理总结点只可按明确讲评点位沉淀，不单独标分值。
- 2024东城二模Q20三大倡议必须保留每项“路径1分+效果1分”结构；全球安全倡议中的联合国宪章宗旨和原则不得被拆成重复独立得分点。
- 2025西城一模Q21的新发展理念本身不进入选必一主链。
- 2025西城一模Q18不进入选必一正式主链，只可作为跨模块边界候补。
- 2024房山一模Q18(2)只找到分类汇编参考答案，不进入正式主链，只可低证据候补。
- 最终学生正文不得出现 Codex、GPT、Claude、Batch、prompt、审核、工作流、文件路径等后台语言。
