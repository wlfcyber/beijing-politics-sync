# Error Taxonomy

旧版不再作为底稿。以下错误类型用于审旧稿、审本批初稿和约束外部模型意见。

| code | 错误类型 | 判定标准 | 处理 |
|---|---|---|---|
| E01 | 细则位置不清 | 找不到套卷、题号、评分点、分值或证据层级 | 不进正文 |
| E02 | 参考答案冒充细则 | 普通参考答案被写成评分术语 | 降级为辅助表述，不作为术语 |
| E03 | 红字/踩分词泛化 | 从教材框架或旧稿推断术语，而非来自本题细则 | 删除或回源 |
| E04 | 材料触发空泛 | 只复述材料，没有说明“设问关系为什么触发术语” | 重写 |
| E05 | 答案句不像卷面作答 | 出现“要写、落到、细则要求、本题需要”等后台语言 | 重写 |
| E06 | 模块边界错 | 必修二、选必二、选必三、必修四术语被硬塞进选必一主链 | 移出主链或加边界说明 |
| E07 | 同类项乱拆乱并 | 同一采分核心被拆成多条，或不同功能被粗暴合并 | 重新做同类项判定 |
| E08 | 旧稿污染 | 旧成品中无源证据的标题、术语、讲法进入新稿 | 删除并回源 |
| E09 | 总说/分说丢失 | 有总说/分说、宏观/细化、替代给分结构却只保留一层 | 补全层次 |
| E10 | 外审未裁决 | Claude/GPT 意见未经 Codex 源证据裁决直接改稿 | 回退到待裁决 |
