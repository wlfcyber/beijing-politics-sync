# 选必二《法律与生活》主观题框架 V5.1：六张动作卡 + 起手铁律

生成时间：2026-05-21 02:10:30

本稿根据 GPTPro 对策、Claude Opus 4.7 复核和 Codex 内部零基础学生压测修订。V5 的七张卡降并为：**六张并列动作卡 + 一条通用起手铁律**。

## 首屏口诀

圈主体 → 抓行为 → 认设问、选卡 → 先表态 → 摆规则、配事实 → 收责任或价值。

## 通用起手铁律

凡设问出现“是否支持 / 能否得到支持 / 是否有效 / 是否构成 / 判决理由 / 评析”，答案第一句必须先表态：

- 某请求应予支持 / 不予支持。
- 某行为构成 / 不构成侵权、违约、欺诈、不正当竞争。
- 某合同成立、有效、无效或可撤销。
- 法院题可写：人民法院以事实为根据、以法律为准绳。不要写反。

表态后，立刻进入六张卡之一写规则、材料、责任或价值。

## 六张动作卡

## LAW_V5_1_N01：一格一答

学生先记：表格题先数空格，看栏目名；一个空只答一件事。

触发词：表格、完成下表、参考示例、补充完整、裁判要点。

动作铁律：每个案例的意义分开写；机制格、理由格、结果格、意义格不能串答案。

满分句骨架：机制格写程序事实；理由格写规则+事实；意义格写保护谁、规范谁、弘扬什么。

支持题号：CC0077_2025_东城_一模_19, CC0084_2025_东城_二模_19, CC0289_2026_朝阳_二模_18, RECOVER_2025_海淀_二模_18, RECOVER_2026_通州_一模_20

## LAW_V5_1_N02：分关系·定责任

学生先记：责任题先判关系：合同、侵权、消费、劳动、相邻。

触发词：法律责任、判决理由、是否赔偿、是否承担责任、合同成立、违约、侵权。

动作铁律：先表态，再写关系；混合题分段写，不把合同和侵权糊成一段；法律名称要写全称。

满分句骨架：某请求应予支持/不支持。甲乙形成某法律关系，某责任成立要满足某条件；材料中某事实符合/不符合条件，所以某方承担/不承担某责任。

支持题号：CC0143_2025_朝阳_一模_19, CC0305_2026_海淀_一模_18_3, CC0063_2024_西城_二模_16, CC0251_2026_丰台_一模_20, CC0244_2026_东城_期末_18, CC0002_2024_丰台_一模_17, CC0200_2025_西城_二模_18

## LAW_V5_1_N03：认产权·抓侵权

学生先记：创新题先认保护对象，再抓侵害行为。

触发词：知识产权、著作权、商标、商业秘密、技术秘密、植物新品种、数据、不正当竞争、保护创新。

动作铁律：创新活力只能放最后，前面必须有具体权利、具体行为、具体司法手段。

满分句骨架：某成果受法律保护；材料中某主体未经许可/不诚信实施某行为，侵害权利人权益或竞争秩序；法院通过停止侵害、赔偿、惩罚性赔偿、调解或规制恶意诉讼保护创新。

支持题号：CC0229_2026_东城_一模_18, CC0283_2026_朝阳_一模_18, CC0103_2025_丰台_一模_19, CC0137_2025_昌平_二模_20, CC0206_2025_西城_期末_19

## LAW_V5_1_N04：排维权步骤

学生先记：维权题排路径、证据、请求、效力。

触发词：如何维权、调解、仲裁、诉讼、起诉状、举证、公益诉讼、司法确认。

动作铁律：先判程序阶段；再判主体；最后写证据、请求和法律效力。公益诉讼不能写成个人维权。

满分句骨架：当事人/公益组织可通过某路径解决纠纷，准备某类证据和诉讼材料；经法院确认的调解或和解协议具有强制执行力。

支持题号：RECOVER_2026_门头沟_一模_18_1, CC0245_2026_东城_期末_18_2, CC0125_2025_延庆_一模_19, CC0150_2025_朝阳_二模_20, CC0223_2025_顺义_一模_19

## LAW_V5_1_N05：划风险边界

学生先记：风险题一项风险配一条规则和一项措施。

触发词：法律风险、法律边界、应对措施、合规、涉嫌违法。

动作铁律：只看设问触发，不看题材触发；AI 宏观产业影响进开放容器，不进本卡。

满分句骨架：材料中的某行为存在某法律风险，触碰某法律边界；相关主体应采取某措施，依法划清权利义务边界。

支持题号：CC0277_2026_房山_二模_18, RECOVER_2025_丰台_二模_19_2, RECOVER_2026_延庆_一模_18_1, RECOVER_2026_房山_一模_17_1

## LAW_V5_1_N06：推价值

学生先记：价值从本案推出，不是最后补口号。

触发词：意义、价值、作用、认识、如何推动、如何护航。

动作铁律：每个价值词前面都要有“因为本案……”。消费案不乱写法治国家，摔倒案不乱写市场秩序。

满分句骨架：因为本案【具体处理】保护了【具体主体】，规范了【具体行为/行业】，所以弘扬了【与本案直接相关的价值】。
