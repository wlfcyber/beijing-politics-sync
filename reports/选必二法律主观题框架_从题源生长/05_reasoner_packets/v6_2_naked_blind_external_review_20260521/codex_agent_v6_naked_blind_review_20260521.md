# Codex Agent V6 裸题盲测严判

结论：V6 已能让学生在多数 formal 题上接近高分，但还不能证明“看完 V6 就稳定接近满分”。主要卡点是 C 的“无过错责任”遗漏，以及 H/I 的非闭环身份必须更硬地压住。

| 题 | 判定 | 理由 | 最可能扣分的词句 |
|---|---|---|---|
| A | PASS | 技术秘密、惩罚性赔偿、公平市场、创新、同类案件示范、司法公信力均踩中。 | “创新成果依法应受保护”略泛，但后文补回技术秘密和惩罚性赔偿。 |
| B | PASS | 有一定影响、抢注、混淆/误认、不正当竞争、诚实信用、品牌商誉、法治化营商环境完整。 | 无明显硬伤。 |
| C | PARTIAL | 两问都答了，合同/违约/侵权/证据/路径基本到位；但关键硬词“生产者产品责任属于无过错责任”没有显性写出。 | “刘某作为销售者……应依法承担相应侵权责任”替代了产品责任无过错规则，可能丢硬点。 |
| D | PASS | 既写保护商业秘密/知识产权，也写劳动权、择业自由、普通技能不能包装成商业秘密、利益平衡。 | 无明显硬伤。 |
| E | PASS | 自愿参加、安全保障义务、无故意或重大过失、自身原因、因果关系、公平和社区互助均到位。 | “如果表格要求……”这种条件式表达，真实表格题里可能显得没直接填表。 |
| F | PASS | AI 非民事主体、意思表示无效、A 公司提示、用户注意义务、无实际损害都到位。 | “企业应尽提示、管理和合规义务”略展开，但不伤主线。 |
| G | PASS | 两案分别落到相邻关系/企业名称纠纷，写出权利边界、调解/行政裁决/诉讼、多元解纷。 | “维护公平正义”若单独出现会泛化，但此答案有案件锚点。 |
| H | PASS（source_check 保分，不作 formal 闭环） | AI 图片著作权、抹水印侵权、信用卡合同违约都保住。 | 没有正面写“著作权主体限于自然人、法人或非法人组织，AI 不能成为著作权主体”。 |
| I | PASS（reference_only 练笔，不作 formal 闭环） | 著作权侵权 + 虚假/引人误解宣传 + 市场秩序方向正确。 | 因 reference_only，不能把这份答案当正式满分闭环证据。 |
| J | PASS | 绿色原则、物权权利边界、义务、责任、救济、生态公共利益都写出。 | 无明显硬伤。 |
| K | PASS | 责任边界、稳定预期、行业规范、用户权益与创新平衡都覆盖。 | “统一裁判标准”最好显性写出，当前主要用“明确规则/规则化认定”替代。 |
| L | PASS | 诉讼时效功能、及时主张、司法资源、赡养费例外、老年人保护、公序良俗完整。 | 无明显硬伤。 |

## 必修项

P0：

1. 把 C 的“生产者产品责任无过错责任”前移到三句保底或硬核检查清单，否则学生会像这次一样写成普通侵权责任。
2. reference_only / source_check 必须在学生使用层更刺眼：reference_only 只能练表达，source_check 只能保方向，任何盲测 PASS 都不得升级为 formal 满分闭环。

P1：

1. H 的保分句第一句必须改成显性主体排除：“AI 不能成为著作权主体；人使用 AI 创作且具独创性，作品才可能受保护。”
2. 表格题训练要禁止“如果表格要求……”式答法，改成直接分格：责任判断 / 法律依据 / 材料事实 / 意义。
3. K 补一个硬句：“判决统一裁判标准，规范行业行为。”

P2：

1. 考场输出压缩“我避免了什么错法”，保留给自检即可。
2. 低频区 J/L 继续保留，但要反复提示“只在原设问命中时使用，不上升为母题”。

最终判断：能证明 V6 对 A/B/D/E/F/G 这类 formal 核心题已有高分能力，对 J/K/L 低频/边界题也能保到位；但 C 的硬点遗漏说明还没达到稳定接近满分。H/I 更不能计入 formal 闭环：H 只是 source_check 保分，I 只是 reference_only 表达练习。
