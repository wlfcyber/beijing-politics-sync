# 选必一《国际政治与经济》零基础满分课稿

## 这节课要达成的结果

- 不是让学生把书背一遍。
- 是让学生听完之后，遇到没见过的新题，也能先判模块，再抓材料，再把答案写成细则语言。
- 这本书三年题目拆完以后，本质上只反复考六件事：
  - `时代背景`
  - `理论`
  - `经济全球化`
  - `政治多极化`
  - `联合国`
  - `中国`

## 第一段先这样讲

- 同学们，国政经最怕的不是不会背教材，而是一看到新材料就慌，不知道该从哪里下手。
- 这本书其实没有那么散。三年北京各区题拆完以后，所有稳定题都能放进六个模块。
- 你们以后做题，先别急着写，先问自己一句话：
  - 这道题到底是在问 `世界怎么了`，还是在问 `国家为什么合作`，还是在问 `开放怎么做`，还是在问 `国际秩序怎么变`，还是在问 `联合国和全球治理`，还是在问 `中国做了什么`。

## 一张判题地图

- 如果材料先写世界动荡、冲突、风险、治理赤字、全球挑战，就先判 `时代背景`。
- 如果材料先写共同利益、脱钩断链、开放与安全、国际竞争、国家实力，就先判 `理论`。
- 如果材料先写外贸、外资、平台、展会、签证便利、自贸港、企业出海、规则对接，就先判 `经济全球化`。
- 如果材料先写全球治理、气候治理、亚太合作、中非合作、霸权主义、小圈子、多边主义，就先判 `政治多极化`。
- 如果材料先写联合国、宪章宗旨原则、安理会、广泛认同、全球治理倡议、四大全球倡议，就先判 `联合国`。
- 如果题目主语很明确就是中国，问中国为什么有作为、中国这样做对世界有什么意义，就先判 `中国`。

## 做题总流程

1. 先判模块。
2. 再圈材料里的关键信息。
3. 再决定先写哪一个主点。
4. 最后用一句总收束把答案拉起来。

## 做题最稳的总收束

- `构建人类命运共同体`
- `高水平对外开放`
- `共同利益是国家间合作的基础`
- `共商共建共享的全球治理观`
- `坚持多边主义`
- `贡献中国智慧和中国方案`

## 第一模块 时代背景

### 你就这样讲

- 这个模块不负责展开写很多具体措施。
- 它最主要的任务，是告诉阅卷人：你知道当今世界的总背景是什么。
- 所以一看到世界乱局、全球风险、治理难题，先别急着写中国怎么做，先交代背景。

### 板书

- `和平与发展`
- `全球挑战增多`
- `世界多极化`
- `命运共同体`

### 看到这些材料就进这个模块

- 世界动荡不安
- 冲突频发
- 风险并存
- 全球治理赤字
- 全球南方
- 外交“变与不变”

### 必背句

- 当今世界虽然面临动荡和风险，但 `和平与发展仍然是时代主题`。
- 世界多极化深入发展，全球性问题增多，任何国家都不能独自解决共同挑战。
- 各国利益深度交融、命运紧密相连，解决全球问题需要推动 `构建人类命运共同体`。

### 新题怎么写

- 材料先交代世界乱象：
  - 先写 `和平与发展仍然是时代主题`
  - 再写 `国际形势复杂严峻、全球挑战增多`
  - 最后顺着材料中的“共同面对”收束到 `命运共同体`

### 这里最容易错

- 不能一看到“世界动荡”就直接冲中国担当。
- 先把背景垫稳，后面的中国方案才有落脚点。

## 第二模块 理论

### 你就这样讲

- 这个模块最重要的是让学生明白：国家之间为什么会合作，为什么不能简单脱钩断链，为什么安全和开放要一起写。
- 这是很多题的底层逻辑。

### 板书

- `共同利益`
- `开放和安全`
- `两个市场两种资源`
- `双循环`

### 看到这些材料就进这个模块

- 脱钩断链
- 共赢链
- 开源合作
- 人工智能+
- 外部风险
- 国际竞争
- 科技革命

### 必背句

- `共同利益是国家间合作的基础`，共同利益越广泛，合作空间越大。
- 面对外部不确定性，要 `统筹发展和安全`，增强产业链供应链韧性。
- 中国发展不能关起门来，要坚持 `高水平对外开放`，充分利用 `两个市场、两种资源`，加快构建 `双循环新发展格局`。

### 新题怎么写

- 如果题目问“为什么要合作”：
  - 第一层写 `共同利益`
  - 第二层写合作能够共同应对挑战
  - 第三层可以升到 `新型国际关系` 或 `命运共同体`

- 如果题目问“怎么处理开放和安全”：
  - 第一层写 `统筹发展和安全`
  - 第二层写 `产业链供应链韧性和安全`
  - 第三层写 `高水平对外开放`
  - 第四层写 `两个市场两种资源` 和 `双循环`

### 这里最容易错

- 不要一看到安全就不写开放。
- 也不要一看到开放就不写风险。
- 北京题的稳定口径是：`安全底线写前面，开放布局跟上去。`

## 第三模块 经济全球化

### 你就这样讲

- 这是最值得学生拿分的模块。
- 因为它的材料长得很像，答题链也最稳定。
- 只要看到“开放平台、便利化、自贸区、企业出海、产业链、规则”，大概率就是这一块。

### 板书

- `自由便利`
- `制度型开放`
- `高水平开放`
- `双循环`
- `国际竞争新优势`

### 看到这些材料就进这个模块

- 免签、口岸签、通关
- 数贸会、链博会、开放平台
- 自贸区、自贸港、东盟合作
- 税目零关税
- 外资、营商环境、规则对接
- 企业出海、海外设厂、标准走出去
- 供应链重构、未来产业

### 必背句

- 中国坚持 `高水平对外开放`，推动形成更大范围、更宽领域、更深层次的对外开放格局。
- 通过完善制度规则、优化营商环境，推动 `制度型开放`。
- 促进 `贸易投资自由化便利化`，推动要素跨境流动和资源优化配置。
- 统筹开放和安全，增强 `产业链供应链韧性和安全`。
- 充分利用 `两个市场、两种资源`，培育 `国际竞争新优势`，服务 `双循环新发展格局`。

### 新题怎么写

- 材料写签证、通关、平台、降成本：
  - 先写 `贸易投资自由化便利化`
  - 再写 `高水平对外开放`

- 材料写条例、规则、标准互认、营商环境：
  - 先写 `制度型开放`
  - 再写 `更高层次开放型经济`

- 材料写出海、设厂、扎根、标准：
  - 先写 `高水平对外开放`
  - 再写 `两个市场两种资源`
  - 再写 `国际竞争新优势`

- 材料写断链、冲击、风险、重构：
  - 先写 `产业链供应链韧性和安全`
  - 再写 `双循环`

### 这里最容易错

- 不能什么开放题都只写一句“高水平对外开放”。
- 要分清：
  - 讲便利，就写 `自由化便利化`
  - 讲规则，就写 `制度型开放`
  - 讲风险，就写 `韧性和安全`
  - 讲出海，就写 `竞争新优势`

## 第四模块 政治多极化

### 你就这样讲

- 这个模块在讲“世界政治关系应该变成什么样”。
- 它不是先讲中国做了什么，而是先讲国际秩序应该怎样更公正合理。

### 板书

- `多边主义`
- `全球治理观`
- `国际关系民主化`
- `新型国际关系`

### 看到这些材料就进这个模块

- 全球治理
- 气候治理
- 亚太合作
- 中非合作
- 霸权主义
- 强权政治
- 小圈子
- 零和博弈

### 必背句

- 全球性挑战需要各国坚持 `多边主义`，通过协商合作共同应对。
- 要推动国际关系 `民主化`，反对霸权主义和强权政治。
- 要推动建设 `相互尊重、公平正义、合作共赢的新型国际关系`。
- 要坚持 `共商共建共享的全球治理观`，推动构建 `人类命运共同体`。

### 新题怎么写

- 材料批判霸权、小圈子、零和博弈：
  - 先写 `反对霸权主义和强权政治`
  - 再写 `国际关系民主化`
  - 再写 `新型国际关系`

- 材料写全球治理、气候治理、区域合作：
  - 先写 `多边主义`
  - 再写 `全球治理观`
  - 最后收束到 `命运共同体`

### 这里最容易错

- 别把这个模块写成“中国贡献了什么”。
- 先讲秩序和关系，再讲中国主张为什么有价值。

## 第五模块 联合国

### 你就这样讲

- 这个模块最稳的抓手是：联合国是平台，宪章是原则，多边主义是方法，中国倡议是方案。
- 只要学生把这四个东西排出来，遇到联合国题基本不会散。

### 板书

- `联合国平台`
- `宪章宗旨原则`
- `多边主义`
- `全球倡议`
- `命运共同体`

### 看到这些材料就进这个模块

- 联合国
- 安理会
- 常任理事国
- 广泛认同
- 全球治理倡议
- 四大全球倡议
- 主权平等
- 国际法治

### 必背句

- 联合国是当今世界最具普遍性、代表性和权威性的政府间国际组织。
- 应当遵循 `联合国宪章宗旨和原则`，坚持真正的 `多边主义`。
- 中国积极参与国际组织和全球治理，提出全球治理倡议和四大全球倡议，为完善全球治理贡献 `中国智慧和中国方案`。
- 这些倡议顺应时代主题，回应共同挑战，契合各国共同利益，推动构建 `人类命运共同体`。

### 新题怎么写

- 一看到联合国：
  - 先写 `联合国宪章宗旨和原则`
  - 再写 `多边主义`

- 一看到“为什么获得广泛认同”：
  - 先写 `契合共同利益`
  - 再写 `回应共同挑战`
  - 再写 `提供中国方案`

- 一看到“四大全球倡议”：
  - 不要分散乱写
  - 要把它们写成共同服务 `全球治理` 和 `命运共同体` 的系统方案

### 这里最容易错

- 很多同学一看到联合国就只会写“和平与发展”。
- 这个点当然可以写，但真正的出分抓手是 `宪章原则 + 多边主义 + 中国倡议`。

## 第六模块 中国

### 你就这样讲

- 这一模块最重要的是让学生敢写“中国做了什么”，而不是只会写“中国很重要”。
- 所以一定要把中国的行动、理念、意义拆开讲。

### 板书

- `中国方案`
- `大国担当`
- `正确义利观`
- `共同发展`
- `中国特色大国外交`

### 看到这些材料就进这个模块

- 元首外交
- 周边外交
- 中国特色大国外交
- 赋能型大国
- 小而美
- 科技小院
- 南南合作
- 中国稳定性和正能量
- 欧盟征税

### 必背句

- 中国坚持 `习近平外交思想`，推进中国特色大国外交。
- 中国坚持走和平发展道路，践行真正的多边主义，推动构建 `人类命运共同体`。
- 中国积极为世界提供 `中国智慧和中国方案`，展现负责任大国的 `责任与担当`。
- 中国开展国际合作坚持 `正确义利观`，兼顾本国利益和他国合理关切，推动共同发展。

### 新题怎么写

- 材料写元首外交、周边工作、外交布局：
  - 先写 `中国特色大国外交`
  - 再写 `习近平外交思想`
  - 再写为世界提供稳定性和正能量

- 材料写小而美、科技小院、援助合作：
  - 先写 `改善民生、促进共同发展`
  - 再写 `正确义利观`
  - 再写 `大国责任与担当`

- 材料写中国理念、中国经验、中国倡议：
  - 先写 `中国智慧和中国方案`
  - 再写 `全球治理观` 或 `命运共同体`

- 材料写征税、保护主义、打压：
  - 先写 `维护国家利益`
  - 再写 `反对贸易保护主义、维护市场秩序`
  - 再写 `扩大开放、合作共赢`

### 这里最容易错

- 不能一看到中国就只会写“担当”。
- 要分清到底是在考：
  - 中国的外交布局
  - 中国的合作理念
  - 中国的世界意义
  - 中国应对外部打压的做法

## 最后十分钟要收成一个模板

- 模板第一句：
  - 先交代这道题属于哪个模块，对应什么时代背景或现实问题。

- 模板第二句：
  - 再把材料里的核心信息翻译成细则语言。

- 模板第三句：
  - 再写这个模块最稳的 `2-3` 个主点。

- 模板第四句：
  - 最后用 `命运共同体 / 高水平开放 / 多边主义 / 中国方案` 这样的总收束把答案拉起来。

## 你在课堂上要反复盯住的四件事

- 第一，不准学生脱离材料空喊概念。
- 第二，不准学生还没判模块就乱堆知识点。
- 第三，不准学生只写一个大词，不展开到细则层。
- 第四，不准学生只有“中国很重要”这种空句，必须写清中国到底做了什么。

## 这节课结束时你要让学生真的会什么

- 会在六模块里给新题定位。
- 会从材料里圈出触发词。
- 会根据触发词调出稳定答题点。
- 会把答案写成 `材料对应 -> 细则语言 -> 总体收束` 的顺序。

## 最后一句收束

- 这本书不是靠死背拿分，而是靠“先判模块，再抓材料，再调主点”拿分。
- 只要六模块边界清楚，学生遇到新题就不会再觉得这本书很散。
