# 选必二法律主观题 V5.2 学生速用稿

生成时间：2026-05-21

## 20 秒启动

圈主体 -> 抓行为 -> 看设问尾词 -> 选动作卡 -> 先表态 -> 规则配事实 -> 收责任或价值。

判决、理由、评析、是否题，第一句先写结论。法院题优先写：人民法院以事实为根据、以法律为准绳。

## 六张动作卡

1. 一格一答：表格题一个空只答一件事，每个案例意义分开写。
2. 分关系·定责任：先判合同、侵权、消费、劳动、相邻，再写规则、事实、责任。
3. 认产权·抓侵权：先判保护对象，再抓侵害行为，最后写司法保护创新。
4. 排维权步骤：写路径、证据、请求、效力；公益诉讼先写公共利益和适格主体。
5. 划风险边界：一项风险、一条规则、一项措施；AI 宏观影响进开放容器。
6. 推价值：每个价值词前面都要有“因为本案……”，价值从本案推出。

## 必踩硬词句库

### 1. 消费欺诈题

先写合同，再写欺诈，最后写消费者法。

满分链：付款并出票，双方合同成立；合同成立不等于合同有效。经营者擅自搭售、隐瞒费用，使消费者不能清楚知悉、不能拒绝，构成欺诈，使消费者违背真实意思订立合同，消费者可以请求人民法院撤销合同。依据《消费者权益保护法》，经营者提供商品或服务有欺诈行为，应支持退还价款和惩罚性赔偿。

禁止：只写“欺诈三倍赔偿”，不写合同成立、可撤销合同、法律全称。

### 2. 司法守护创新题

先写司法总帽，再逐案写司法手段。

满分链：人民法院公正司法，坚持以事实为根据、以法律为准绳，发挥司法裁判导向作用。调解知识产权纠纷，降低诉讼成本、提高效率；适用惩罚性赔偿，增加侵权成本、保护权利人权益；驳回并谴责恶意诉讼，维护诉讼秩序和公平竞争，保护真正创新。

禁止：只写“保护新质生产力”，不逐案写调解、惩罚、驳回恶意诉讼。

### 3. 权利边界题

支持一部分、限制一部分时，一定写权利有边界。

满分链：经营者应依法诚信经营，保护消费者合法权益。民事主体行使民事权利不能超过正当界限，不得滥用权利损害国家利益、社会公共利益或他人合法权益。法院支持合理维权，限制异常加购、恶意索赔，有利于平衡食品安全与生产经营秩序。

禁止：只写保护消费者，不写权利边界。

### 4. 典型案例意义题

意义题按四层写，别只写创新。

满分链：保护原告方或当事人知识产权，严惩侵权行为；规范市场竞争秩序，营造公平市场环境，以法治力量鼓励创新；为同类案件提供范例，提升审判质量和效率，增强司法公信力；引导公民依法办事，增强法治信仰，推动法治中国建设。

禁止：直接裸写法治中国。前面必须先有本案保护知识产权、严惩侵权和司法范例。

### 5. 公益诉讼与调解效力题

公益诉讼不是个人维权。

满分链：公共利益受损风险出现后，适格公益组织提起环境民事公益诉讼；法院通过和解、司法调解、诉讼等多元方式解决纠纷；经人民法院依法确认有效的和解或调解协议具有强制执行力；法院持续监督履行，维护生态环境公共利益。

禁止：只写生态意义，不写适格主体、多元解纷、强制执行力。

### 6. 仅退款调解题

小标的先调解，退款后要处理返还。

满分链：本案标的较小，本着以和为贵原则适用调解。消费者维权应遵循诚实信用原则，不能损害他人利益和社会经济秩序。杨某收到退款后买卖合同已解除，应依法退还货物；若主张商品质量不合格，应保留证据，丢弃商品体现证据意识不足。经营者也应保证商品质量并回应售后。

禁止：只写双方各退一步，不写合同解除、退还货物、证据意识。

### 7. 隐私相邻题

先定性侵权，再写共有和相邻。

满分链：刘先生侵犯孙女士隐私权，应承担侵权责任。楼道属于双方专有部分以外的共有部分，双方享有共有和共同管理权。摄像头拍到日常出行规律及其他动态信息，属于隐私利益；设备范围和功能超出必要限度，刘先生未尽妥善注意义务。相邻权利人应按照有利生产、方便生活、团结互助、公平合理原则处理关系。建议及时拆除摄像头，并清除已拍摄的视频信息。

禁止：只写调角度、关云存储。细则要求拆除设备和清除已拍摄信息。

## 最后检查

- 法律名称能写全就写全：民法典、《消费者权益保护法》、《反不正当竞争法》。
- 材料事实必须带进去：不能只有规则。
- 价值句必须由本案推出：因为本案处理了什么，所以保护谁、规范谁、推动什么。
- 开放容器题不要硬套：生态保护、法治德治、校园欺凌、绿色发展、诉讼时效、AI 宏观影响，只按“制度/规则是什么 -> 材料怎么体现 -> 具体作用”写。
