## 0. 这次重建的硬裁定

V7 不再沿用 V6.9 的“九个战场”作为学生前台。

原因很简单：学生考场上先看到的不是“这属于合同战场还是创新战场”，而是“这道题要我交一段什么答案”。所以 V7 的第一入口必须从“答案成品”进，再由责任链、归责卡、材料翻译生成满分句。

后台仍保留合同、侵权、劳动、消费者、知识产权、程序等标签，但那是教师检索和证据审计用的，不是学生第一眼看到的框架。

## 1. 学生第一页总模型

一句话：

```text
生活冲突 A  ->  法律判断 B  ->  得分句 C
```

中间只做六件事：

```text
看设问 -> 圈主体 -> 抓行为 -> 定争点 -> 套规则 -> 落成品
```

学生脑内话：

1. 先别急着背法律名词。
2. 先问：这道题到底要我交什么成品？
3. 再问：材料里谁和谁发生了什么冲突？
4. 再把生活话翻译成法律话。
5. 最后把法律话写成细则能给分的句子。

## 2. 第一层：八个答案产品门

这是 V7 的前台主轴。

| 门 | 设问信号 | 第一句怎么写 | 后面接什么 | 证据锚点 |
|---|---|---|---|---|
| 门 1 判断表态 | 能否支持、是否有效、是否构成、判决结果 | `XX 的诉求应/不应得到支持。` | 一争点一段，理由跟上 | `CC0143`, `CC0200`, `CC0373`, `CC0150` |
| 门 2 合同链 | 合同成立、合同效力、履行责任 | `先跑要约 -> 承诺 -> 成立/有效 -> 履行或违约。` | 违约链 | `CC0054`, `CC0244`, `RECOVER_2026_朝阳_期末_18_1` |
| 门 3 多主体多请求 | 两个主体、两个诉求、几个案例 | `对 A 先表态；对 B 另起一段。` | 一人一段，一请求一段 | `CC0373`, `CC0238`, `CC0200` |
| 门 4 证据请求路径 | 怎么办、维权、需要做哪些工作、调解、起诉状 | `当事人应围绕……提出……请求，并……` | 请求 + 证据 + 路径 | `CC0125`, `CC0245`, `RECOVER_2024_东城_二模_19_1` |
| 门 5 表格补全 | 完成表格、补全笔记、划线处 | `先看列名，再看示例，再填一格一句。` | 每格只答该格功能 | `CC0077`, `CC0084`, `RECOVER_2025_海淀_二模_18`, `CC0325` |

| 门 8 非核心保险箱 | 涉外、治理、逻辑、哲学、reference_only、待回源 | `先降级，不把它硬写成核心母题。` | 低频/待回源/边界/转出/reference_only | `CC0276`, `CC0277`, `CC0380`, `CC0364`, `CC0040` |

考场口诀：

```text
先交成品，再找规则。
判断题先表态，理由题先找链，表格题先看格，价值题先处理本案。
```

## 3. 第二层：三大责任逻辑链

学生一旦进入实体判断，就用这三条链，不再散写知识点。

### 3.1 违约链

```text
合同成立 -> 合同有效 -> 未按约履行 -> 违约责任
```

可写句：

```text
本案中，XX 的表示内容具体确定，属于要约；XX 作出承诺后，双方合同成立。合同主体适格、意思表示真实、内容不违反法律法规强制性规定，合同有效。XX 未按合同约定履行义务，构成违约，应承担继续履行、赔偿损失等违约责任。
```

材料触发：

- 报价、投标、中标通知、付款出票、签订协议、服务期、培训费、未按期交付。

证据锚点：

- `R_CC0054_2024_石景山_一模_17_M17_01~03`
- `R_CC0244_2026_东城_期末_18_08`
- `R_RECOVER_2026_朝阳_期末_18_1_01~02`

### 3.2 侵权链

```text
权利受保护 -> 行为越界 -> 造成损害 -> 有因果关系 -> 按归责档承担责任
```

可写句：

```text
XX 享有……权利。材料中，XX 实施了……行为，超出合理限度，损害了……权益，与损害结果之间存在因果关系，应依法承担侵权责任。
```

材料触发：

- 偷拍、摄像头、辱骂诋毁、未成年人文身、产品缺陷、动物致害、窗户坠落、隐私信息、身体健康损害。

证据锚点：

- `R_CC0150_2025_朝阳_二模_20_01~07`
- `R_CC0244_2026_东城_期末_18_05~08`
- `R_CC0325_2026_石景山_一模_18_02~03`

### 3.3 不正当竞争 / 知识产权链

```text
保护对象或竞争关系 -> 不正当行为 -> 损害权利或竞争秩序 -> 责任 + 创新价值
```

可写句：

```text
XX 与 XX 存在竞争关系。材料中，XX 通过……方式实施混淆、虚假宣传、商业诋毁或侵犯商业秘密等行为，扰乱市场竞争秩序，损害经营者合法权益，构成不正当竞争，应承担相应责任。
```

材料触发：

- 相似企业名称、抢注标识、虚假宣传、诋毁同行、挖走技术人员、窃取技术秘密、AI 生成侵权链接。

证据锚点：

- `R_CC0206_2025_西城_期末_19_01`
- `R_CC0238_2026_东城_二模_19_M17_01`
- `R_CC0131_2025_房山_一模_19_05`
- `R_CC0319_2026_海淀_期末_19_01`

## 4. 第三层：三档归责卡

这张卡必须单独放在学生版前 10 页。它是高频失分点。

| 档位 | 先判断什么 | 典型题 | 禁错句 |
|---|---|---|---|
| 合同违约档 | 是否有合同、合同是否有效、是否违约 | `CC0054`, `CC0244`, `RECOVER_2026_朝阳_期末_18_1` | 不要用侵权四要件分析违约 |
| 一般过错侵权档 | 行为、损害、因果、过错 | `CC0150`, `CC0200`, `CC0244` 销售者部分 | 销售者一般不能直接写无过错责任 |
| 过错推定 / 无过错档 | 法律是否特殊配置责任 | `CC0180`, `CC0325`, `CC0244` 生产者部分 | 不要写“无过错推定”；生产者产品责任不是一般过错 |

硬背红线：

```text
销售者不是生产者。
产品缺陷找生产者，重点写无过错责任。
销售者有过错才走销售者责任。
建筑物坠落、个人信息等可能走过错推定。
```

证据锚点：

- `R_CC0244_2026_东城_期末_18_05`
- `R_CC0244_2026_东城_期末_18_06`
- `R_CC0180_2025_海淀_期末_20_M17_06`
- `R_CC0325_2026_石景山_一模_18_02~03`

## 5. 第四层：材料翻译三步

学生得分不是因为会背概念，而是因为能把材料翻译成法律语言。

```text
材料事实 -> 法律术语 -> 得分句
```

| 材料事实 | 法律语言 | 得分句 |
|---|---|---|
| 报价内容具体确定 | 要约 | `甲公司的报价内容具体确定，属于要约。` |
| 中标通知送达 | 承诺、合同成立 | `乙公司中标通知送达，双方合同成立。` |
| 平台派单、奖惩、签到 | 人格/经济/组织从属性 | `平台对劳动者进行管理和奖惩，体现人格从属性。` |
| 擅自搭售、不能清楚知悉、无法拒绝支付 | 欺诈、违背真实意思表示 | `经营者擅自搭售，消费者不能清楚知悉并无法拒绝，构成欺诈。` |
| 拍到日常出行规律及动态信息 | 隐私权侵害 | `摄像头拍摄孙女士日常出行规律及其他动态信息，侵犯隐私权。` |
| 捏造事实贬损对手 | 商业诋毁 | `该行为损害竞争对手商业信誉，构成商业诋毁。` |
| AI 生成赔偿承诺 | AI 不是民事主体，意思表示无效 | `AI 不具备民事主体资格，不能独立作出有效意思表示。` |

三条铁律：

1. 译全：材料给了三个条件，答案不能只写一个。
2. 用原词：细则要“日常出行规律”，就不要改成“个人信息”糊弄。
3. 名称准：`消费者权益保护法` 不要写成“保护法”；`欺诈` 不要写成“诈骗”。

## 6. 第五层：价值四层卡

价值不是开头，是收尾。

```text
先处理本案 -> 再推出价值
```

| 价值层 | 句子模板 | 证据锚点 |
|---|---|---|
| 当事人权利层 | `本案通过……处理，维护了……合法权益。` | `CC0045`, `CC0103`, `CC0206` |
| 行业秩序层 | `有利于规范……秩序，营造公平竞争/交易环境。` | `CC0103`, `CC0206`, `CC0319` |
| 司法审判层 | `为同类案件提供裁判示范，提高司法公信力。` | `R_CC0103_2025_丰台_一模_19_01` |
| 法治社会层 | `引导公民依法办事，增强法治信仰。` | `R_CC0103_2025_丰台_一模_19_01` |

禁写：

```text
维护公平正义。
推动法治中国建设。
弘扬社会主义核心价值观。
```

这些裸句不是绝对不能出现，而是不能裸写。前面必须有本案铺垫。

## 7. 第六层：五类保险箱

不是所有题都能升成核心母题。V7 必须让学生知道哪些题只是低频保分、待回源、边界、转出或练笔。

| 保险箱 | 用途 | 学生纪律 | 题源 |
|---|---|---|---|
| 低频规则容器 | 练少见规则，保中高分 | 写规则名 + 材料路径 + 一句价值，不升母题 | `CC0011`, `CC0254`, `CC0332`, `CC0340`, `RECOVER_2024_东城_一模_19` |
| 待回源保分索引 | 防止未锁题卡污染核心 | 未回源前只作最低句提示，不进核心节点 | `CC0223`, `CC0244`, 19 个空设问题 |
| 综合边界容器 | 只守住法律层 | 法律层写完，其他模块另开段 | `CC0276`, `CC0277`, `CC0380`, `RECOVER_2026_西城_二模_18_2/18_3` |
| 转出容器 | 防止把非法律主任务硬套法律 | 主任务是逻辑/哲学就转出，只答法律小问 | `CC0364` |
| reference_only 练笔容器 | 练表达，不支撑核心 | 只能叫参考练笔句，不能叫满分句 | `CC0040`, `CC0162`, `CC0311`, `CC0353` |
