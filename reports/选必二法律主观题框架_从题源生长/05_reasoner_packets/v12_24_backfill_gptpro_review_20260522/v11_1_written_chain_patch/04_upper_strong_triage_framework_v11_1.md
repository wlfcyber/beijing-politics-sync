# 上篇：选必二法律主观题强分诊框架 v11.1

本版只修 v11 已写题链中的 P0 问题，不回填 24 题，不写最终宝典。

## 1. 先导

法律主观题先分诊。

- 先看设问交什么。
- 再看材料事实触发什么法律动作。
- 最后写规则、材料、结论。

飞哥想说：不要一见法院就写公正司法，不要一见协议就写合同，不要一见价值就写全面依法治国。

## 2. 先按设问分诊
### 问诉求能不能支持 / 法院为什么支持
看到：
- 问诉求能不能支持 / 法院为什么支持
这样写：
- 先表态 -> 法律依据 -> 材料事实 -> 支持/不支持/减轻/承担。
别这样写：
- 别一见法院就写必修三；别一见协议就写合同。

### 问判决结果 / 责任分担
看到：
- 问判决结果 / 责任分担
这样写：
- 先看谁是限制民事行为能力人、谁有审查义务、谁有监护责任。再按各方过错和公平原则分责。
别这样写：
- 别写成合同格式条款题；别把平台等同平台用工。

### 问裁判理由 / 补表 / 完成理由
看到：
- 问裁判理由 / 补表 / 完成理由
这样写：
- 只有题目出现“完成裁判理由/补表/处理结果已给定”，才能进入。一格一链，处理结果倒推法律理由。
别这样写：
- 评析题不能进；两个格子不能糊成一个创新题。

### 问评析 / 谈认识
看到：
- 问评析 / 谈认识
这样写：
- 一主体一链。先评谁，再评行为，最后写后果和应然要求。
别这样写：
- 别一锅炖；别只写企业诚信经营。

### 问意义 / 价值 / 作用
看到：
- 问意义 / 价值 / 作用
这样写：
- 先规则，再材料，最后价值。
别这样写：
- 别裸写全面依法治国；价值必须从规则推出。

### 问调解 / 维权 / 如何解决
看到：
- 问调解 / 维权 / 如何解决
这样写：
- 先给方案，再给法律理由，再贴材料依据。
别这样写：
- 别把调解题写成判决书。

### 问法律责任及依据
看到：
- 问法律责任及依据
这样写：
- 先定关系，再定行为，最后定责任。
别这样写：
- 别只列法律词。

## 3. 再按材料分诊
### 好意同乘 / 交通损害
看到：
- 无偿搭乘 + 交通事故 + 非故意非重大过失。
标准写法：
- 好意同乘减责 + 侵权责任边界 + 公平。
不能乱用：
- 不要写合同成立。

### 未成年人交易 / 打赏 / 多方过错
看到：
- 未成年人高额打赏 + 监护人未追认 + 平台审查不足 + 本人规避监管。
标准写法：
- 限制民事行为能力 -> 行为效力 -> 平台/监护人/本人过错 -> 公平分责。
不能乱用：
- 不要写合同格式条款；不要写平台用工。

### 裁判理由双格：AI 著作权 + 信用卡违约
看到：
- 一个格子是 AI 图片著作权，一个格子是信用卡领用合约。
标准写法：
- 一格一链。AI 格写作品保护与侵权；信用卡格写合同有效与违约责任。
不能乱用：
- 不要只写创新价值。

### 平台用工
看到：
- 合作协议 + 派单 + 签到 + 奖惩 + 结算 + 业务组成。
标准写法：
- 看三从属性，认定事实劳动关系。
不能乱用：
- 只有平台二字不够。

### 合同成立 / 履行 / 违约
看到：
- 报价、中标、承诺、信用卡领用合约、透支不还。
标准写法：
- 要约/承诺/合约有效 -> 全面履行/诚信 -> 违约责任。
不能乱用：
- 平台合作协议、好意同乘不能自动进合同。

### 消费者欺诈 / 惩罚性赔偿
看到：
- 虚假宣传、隐蔽搭售、无法清楚知悉或有效拒绝。
标准写法：
- 知情权/自主选择权/公平交易权 -> 欺诈 -> 退赔/惩罚性赔偿。
不能乱用：
- 三倍赔偿不能乱套。

### 知识产权 / 技术秘密 / 不正当竞争
看到：
- 作品、图片、专利、技术秘密、植物新品种、未经许可、惩罚性赔偿、恶意诉讼。
标准写法：
- 权利对象 -> 侵害方式 -> 责任 -> 创新价值。
不能乱用：
- 只有创新二字不够。

### 多案例意义题
看到：
- 一题多个案例，每个案例对应不同法律手段。
标准写法：
- 一案一手段，一手段一价值。
不能乱用：
- 不要把三案合成一句保护创新。

### 人格权 / 隐私 / 人身权
看到：
- 监控、聊天信息公开、高空抛物坠物、身体健康。
标准写法：
- 权利名 -> 侵害行为 -> 保护路径。
不能乱用：
- 别泛写人格尊严。

### 相邻关系 / 物权边界
看到：
- 通行、通风、采光、业主规约、恢复原状。
标准写法：
- 相邻/物权规则 -> 具体妨碍 -> 停止/恢复/协商。
不能乱用：
- 别写成社区治理。

### 家庭继承 / 养老赡养
看到：
- 遗赠扶养协议、赡养义务、老人照料。
标准写法：
- 协议效力/法定义务 -> 老年权益保护。
不能乱用：
- 别只写传统美德。

### 劳动竞业 / 平等就业
看到：
- 竞业限制、商业秘密、离职入职竞争企业、招聘性别限制。
标准写法：
- 保护商业秘密，也保护劳动者就业权。
不能乱用：
- 不是单向保护企业。

## 4. 禁止误判总表
- 有法院，不一定写公正司法。
- 有企业，不一定写经济生活。
- 有平台，不一定写劳动关系。
- 有合同/协议，不一定只写合同。
- 有消费者，不一定写三倍赔偿。
- 有创新，不一定只写保护创新。
- 有评析，不得进裁判理由。
- 有多案例意义，必须一案一手段。

## 5. 最短记忆版
1. 先看设问交什么。
2. 再看材料触发什么法律动作。
3. 裁判理由题一格一链。
4. 评析题一主体一链。
5. 责任分担题多方过错加公平原则。
6. 意义题先规则后价值。
7. 平台劳动必须看三从属性。
8. 未成年人打赏不是格式条款题。
9. 创新题先权利对象和侵害方式。
10. 没有材料事实，不写法律词。
