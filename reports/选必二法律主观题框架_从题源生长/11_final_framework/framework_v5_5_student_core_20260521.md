# 选必二法律主观题满分训练宝典 V5.5

生成时间：2026-05-21 03:17:37

## 这版到底是什么

这版不是把 65 道题都说成同一种满分套路。它分两层使用：

- 第一层：27 道已清洗核心题，给你逐题学会“先判断什么、怎么把材料变法律话、怎么写考场答案”。
- 第二层：其余题只给保分容器和边界提醒，告诉你遇到低频题、综合题、题面不稳题时怎样不乱套。

你真正要背的是：**先判断、再分层、再套规则、最后收责任或价值。**

## 20 秒启动

1. 圈主体：谁和谁发生关系，是消费者、劳动者、经营者、平台、法院、权利人，还是公益组织。
2. 抓行为：交付、搭售、派单、拍摄、宣传、调解、起诉、判决、侵权、违约、拒绝履行。
3. 看设问：问是否支持、为什么、怎么看、怎么办、有什么意义，答案动作完全不同。
4. 定主卡：一题只选一张主卡，最多加一张辅卡。
5. 写第一句：先给结论，不要铺一堆空话。
6. 每段都按“规则 + 材料事实 + 责任/程序/价值落点”写。

## 六张动作卡

| 动作卡 | 什么时候用 | 第一笔怎么写 | 最容易丢什么 |
|---|---|---|---|
| 一格一答 | 表格、笔记、补全 | 先看每个空要机制、理由、结果还是意义 | 把整段答案塞进一个空 |
| 分关系·定责任 | 是否支持、是否有效、是否构成、判决依据 | 先写“某请求应/不应支持”或“某行为构成……” | 不表态、不写责任类型 |
| 认产权·抓侵权 | 商标、技术秘密、著作权、不正当竞争 | 先认保护对象，再抓侵害行为 | 只写创新，不写权利和侵权 |
| 排维权步骤 | 如何维权、调解、仲裁、诉讼、起诉状、举证 | 写路径、证据、请求、效力 | 漏证据或漏请求 |
| 划风险边界 | 风险、合规、AI、平台、边界 | 一项风险配一条规则和一项措施 | 写成宏观治理口号 |
| 推价值 | 意义、价值、典型案例、作用 | 因为本案怎样处理，所以保护谁、规范谁 | 裸写法治中国、核心价值观 |

## 答案生成公式

**结论句 + 规则句 + 材料句 + 落点句。**

- 结论句：请求是否支持、合同是否成立有效、行为是否构成违约/侵权/欺诈/不正当竞争。
- 规则句：写具体法律规则或制度，不要只写“依法”。
- 材料句：把付款、派单、搭售、拍摄、损害、过错、调解、证据等事实塞进规则。
- 落点句：落到责任、救济、程序效力，或由本案推出价值。

## 保命硬规则

- 问“是否支持”，第一句必须表态。
- 问“理由”，每个理由都要有材料事实。
- 问“意义”，先写本案法律处理，再写保护权益、规范秩序、促进创新、提升司法公信力。
- 问“怎么办”，必须写证据、请求、路径，不能只写“依法维权”。
- 遇到多小问，先把题号拆开；一问漏掉，整题就不可能满分。

## 高频硬词句库

### 消费欺诈：先合同，后欺诈，再消费者法
付款出票或付款下单 -> 合同成立；经营者隐瞒、搭售、虚假宣传 -> 真实意思受影响；消费者可以主张撤销或惩罚性赔偿。
### 平台劳动：三个从属性不能只背名词
人格从属性看派单、签到、奖惩；经济从属性看数据和费用标准；组织从属性看是否纳入平台业务。
### 知识产权：保护对象 + 侵害行为 + 市场秩序
先认技术秘密、商标、著作权等保护对象，再写侵害或混淆行为，最后写公平竞争、鼓励创新和司法范例。
### 程序维权：证据、请求、路径、效力
证据要能证明事实；请求要对应责任；路径要写协商、调解、仲裁或诉讼；调解协议经法院确认后有强制执行力。
### 价值题：价值必须由本案推出
本案保护了谁、规范了谁、惩戒了谁、给同类案件什么示范，然后再写社会价值。
