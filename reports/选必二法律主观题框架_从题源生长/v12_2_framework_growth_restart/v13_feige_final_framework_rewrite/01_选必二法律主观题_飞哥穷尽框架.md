# 01 选必二法律主观题 飞哥穷尽框架

## 一、先导：这类题到底在考什么

- 不是背法条。
- 不是法考。
- 不是必修三。
- 是把生活纠纷翻译成法律得分点。
- 考场四步：定题型、翻材料、写规则事实结论、必要时收价值。

最短母公式：

> 设问定型 → 材料翻法 → 规则事实结论 → 必要价值

学生最短记忆版：

1. 先看设问，不先背知识。
2. 再抓材料：谁、做啥、争啥、损啥。
3. 把生活话翻成法律话。
4. 答案写：规则 + 材料 + 结论。
5. 意义题才加价值，价值必须从规则长出来。
6. 法院≠必修三，企业≠经济，平台≠劳动，消费者≠三倍赔偿，案例≠法考。

## 二、总法门：四步一句话

定题型 → 翻材料 → 写法律 → 收价值

### 1. 定题型

看到什么：设问里的“说明理由、完成下表、评析、意义、调解、维权”。

要干什么：先决定答案长相，是先判、填表、分主体、写价值，还是写路径。

别写什么：不要一上来背知识点，不要见法院就写公正司法。

飞哥想说：题型定错，后面写得再多也是跑偏。

### 2. 翻材料

看到什么：谁、做啥、争啥、损啥、有没有证据。

要干什么：把生活话翻成法律话：合同、侵权、人格权、劳动、消费者、知识产权、纠纷路径。

别写什么：不要抄材料，不要只贴章节名。

飞哥想说：政治法律题的难点不是背，是把材料翻成得分词。

### 3. 写法律

看到什么：材料已经能对应某条规则或某种责任。

要干什么：按“规则 + 材料 + 结论”写。

别写什么：不要只写规则不带材料，也不要只讲故事不落规则。

飞哥想说：一条链就是一分路。

### 4. 收价值

看到什么：设问问意义、价值、作用、推动。

要干什么：从本案规则推出保护谁、规范谁、维护什么秩序。

别写什么：不要裸写全面依法治国、公平正义、法治社会。

飞哥想说：意义题最后才收价值，价值必须从规则长出来。

## 三、设问分诊穷尽

### 1. 问理由/诉求/判决结果：先判，再证

看到：设问里有“判决结果”“诉求能否支持”“说明理由”“法理依据”。

第一步：先写支持、不支持、部分支持，或者法院判得对不对。

答案骨架：结论 → 规则 → 材料事实 → 责任或请求。

反向筛：不要先写意义，不要把法院二字写成公正司法。

来源题：CC0305_2026_海淀_一模_18_3、CC0364_2026_通州_期中_19_1、CC0054_2024_石景山_一模_17、CC0244_2026_东城_期中_18、CC0206_2025_西城_期末_19、CC0002_2024_丰台_一模_17、CC0025_2024_朝阳_二模_17、CC0119_2025_丰台_期末_19、CC0373_2026_顺义_一模_18、CC0200_2025_西城_二模_18、CC0143_2025_朝阳_一模_19

### 2. 问裁判理由/补表：一格一链

看到：设问里有“完成下表”“参考示例”“裁判理由”“补充完整”。

第一步：先看每一格问的是事实、规则、理由还是结论。

答案骨架：一格：一个事实 → 一条规则 → 一个结论。

反向筛：不要跨格合并，不要一段话解决三案。

来源题：CC0084_2025_东城_二模_19、CC0189_2025_石景山_一模_20、CC0137_2025_昌平_二模_20、CC0157_2025_朝阳_期末_20、CC0180_2025_海淀_期末_20、CC0213_2025_门头沟_一模_20、CC0214_2025_门头沟_一模_20_2、CC0077_2025_东城_一模_19、CC0325_2026_石景山_一模_18

### 3. 问评析：一主体一链

看到：设问里有“评析”“认识”“谈谈对某观点/行为的认识”。

第一步：先把主体拆开：谁的行为，谁的权利，谁的责任。

答案骨架：合理处/违法处 → 法律边界 → 本案处理。

反向筛：不要只表态，不要把多主体写成一锅粥。

来源题：CC0150_2025_朝阳_二模_20、CC0332_2026_石景山_二模_19、CC0238_2026_东城_二模_19、CC0254_2026_丰台_二模_18、CC0051_2024_海淀_期中_21_1、CC0061_2024_西城_一模_18

### 4. 问意义/价值：规则生价值

看到：设问里有“意义”“价值”“作用”“保护”“推动”。

第一步：先问本题用哪条规则保护谁、约束谁。

答案骨架：规则 → 保护对象 → 规范行为 → 秩序价值。

反向筛：不要裸写法治价值，不要离开材料喊口号。

来源题：CC0340_2026_西城_一模_17、CC0019_2024_朝阳_一模_19、CC0103_2025_丰台_一模_19、CC0131_2025_房山_一模_19、CC0229_2026_东城_一模_18、CC0283_2026_朝阳_一模_18、CC0011_2024_丰台_二模_17、CC0045_2024_海淀_二模_19、CC0181_2025_海淀_期末_21、CC0195_2025_西城_一模_20、CC0063_2024_西城_二模_16

### 5. 问调解/维权/路径：方案+理由+证据

看到：设问里有“调解”“维权”“纠纷如何解决”“遇到的法律问题”。

第一步：先定争点，再定证据，再定路径。

答案骨架：争点 → 证据 → 协商/调解/仲裁/诉讼 → 请求。

反向筛：不要直接替法院判案，不要只写依法维权。

来源题：CC0092_2025_东城_期末_19_1、CC0245_2026_东城_期中_18_2、CC0289_2026_朝阳_二模_18、CC0125_2025_延庆_一模_19、CC0223_2025_顺义_一模_19

## 四、材料触发穷尽

### 1. 合同成立/履约/违约

看到：要约、承诺、付款、交付、期限、质量、解除、违约金。

翻成法律话：合同是否成立有效；是否全面、诚信履行；违约后承担继续履行、补救、赔偿等责任。

高频句：双方意思表示一致，合同成立并生效；一方未按约履行，应承担违约责任。

别写：不要看到商家就直接写消费者，不要看到损害就跳过合同写侵权。

来源题：CC0189_2025_石景山_一模_20、CC0054_2024_石景山_一模_17、CC0244_2026_东城_期中_18、CC0019_2024_朝阳_一模_19、CC0245_2026_东城_期中_18_2

飞哥想说：合同题先看“有没有约”，再看“有没有照约做”。

### 2. 平台用工/三从属性

看到：平台派单、签到、奖惩、统一结算、业务纳入平台组成。

翻成法律话：看人格从属性、经济从属性、组织从属性，判断是否事实劳动关系。

高频句：不能只看合作协议名称，要看实际管理控制和报酬结算。

别写：没有派单、签到、奖惩、结算，不要硬写劳动关系。

来源题：CC0025_2024_朝阳_二模_17

飞哥想说：平台两个字不值分，管理控制才值分。

### 3. 消费者欺诈/知情权/惩罚性赔偿

看到：虚假宣传、隐瞒信息、格式条款、预扣款、未提示说明。

翻成法律话：消费者知情权、公平交易权、安全权是否受损；欺诈成立才谈惩罚性赔偿。

高频句：经营者虚假宣传或提示说明不足，损害消费者知情权和公平交易权，应承担相应责任。

别写：消费者不等于三倍赔偿，先找欺诈。

来源题：CC0200_2025_西城_二模_18、CC0143_2025_朝阳_一模_19、CC0063_2024_西城_二模_16、CC0125_2025_延庆_一模_19、CC0084_2025_东城_二模_19、CC0305_2026_海淀_一模_18_3

飞哥想说：退钱、赔偿、惩罚性赔偿不是一锅端，要看事实够不够。

### 4. 人格权/隐私/名誉/个人信息

看到：监控、聊天记录、个人信息、无事实贬损、未成年人身心利益。

翻成法律话：先叫准权利名，再写侵害行为、损害后果、责任方式。

高频句：未经同意公开私人信息或无事实依据贬损他人，应承担停止侵害、赔礼道歉、赔偿等责任。

别写：不要把隐私、名誉、个人信息全部塞进消费者权益。

来源题：CC0084_2025_东城_二模_19、CC0305_2026_海淀_一模_18_3、CC0150_2025_朝阳_二模_20、CC0332_2026_石景山_二模_19、CC0289_2026_朝阳_二模_18

飞哥想说：人格权题先喊准权利名，喊错就全乱。

### 5. 知识产权/不正当竞争/创新保护

看到：作品、唤醒词、技术秘密、数据抓取、商业诋毁、混淆、创新成果。

翻成法律话：先看权利对象，再看未经许可、绕过限制、虚假贬损或商业化利用。

高频句：法律保护创新投入和经营成果，制止侵权和不正当竞争，维护公平竞争秩序。

别写：创新不等于只写保护创新，必须写权利对象和侵害方式。

来源题：CC0137_2025_昌平_二模_20、CC0157_2025_朝阳_期末_20、CC0206_2025_西城_期末_19、CC0238_2026_东城_二模_19、CC0103_2025_丰台_一模_19、CC0131_2025_房山_一模_19、CC0229_2026_东城_一模_18、CC0283_2026_朝阳_一模_18

飞哥想说：先把“被保护的东西”说出来，再谈创新。

### 6. 多主体过错/未成年人/公平分责

看到：多个人、多公司、未成年人、受害人也有注意义务、惩教并行。

翻成法律话：一主体一链，分别看行为、过错、损害、责任。

高频句：责任承担要结合各方过错、行为与损害之间的联系，依法合理分担。

别写：不要把两个主体糊成一个结论。

来源题：CC0084_2025_东城_二模_19、CC0332_2026_石景山_二模_19、CC0238_2026_东城_二模_19、CC0180_2025_海淀_期末_20、CC0289_2026_朝阳_二模_18、CC0254_2026_丰台_二模_18

飞哥想说：多主体题先分人，再分责。

### 7. 好意同乘/侵权减责

看到：免费搭乘、善意帮忙、乘车人受伤、驾驶人无故意或重大过失。

翻成法律话：先看侵权责任，再看好意同乘的减责边界。

高频句：好意同乘中驾驶人无故意或重大过失的，可依法减轻责任。

别写：不要只写温情，也不要完全免除安全注意义务。

来源题：CC0002_2024_丰台_一模_17

飞哥想说：这类题是情理和法理一起考，但落笔还是规则加材料。

### 8. 相邻关系/物权边界/绿色原则

看到：采光、通行、安全、共有部分、加装电梯、绿色发展、生态环境。

翻成法律话：看权利归属、使用边界、实际影响和公平合理处理。

高频句：相邻关系应按有利生产、方便生活、团结互助、公平合理处理。

别写：不要见法院就写司法，不要见绿色就写经济绿色发展。

来源题：CC0364_2026_通州_期中_19_1、CC0340_2026_西城_一模_17、CC0092_2025_东城_期末_19_1、CC0011_2024_丰台_二模_17

飞哥想说：物权题不是谁嗓门大谁有理，要看边界和影响。

### 9. 竞业限制/商业秘密/劳动权利

看到：保密协议、竞业限制、离职去竞争单位、商业秘密、就业歧视。

翻成法律话：看企业合法利益和劳动者就业权之间是否平衡。

高频句：竞业限制保护商业秘密，但范围、期限、对象必须合法合理适度。

别写：不要把竞业限制写成普通违约，也不要只保护企业。

来源题：CC0213_2025_门头沟_一模_20、CC0214_2025_门头沟_一模_20_2、CC0025_2024_朝阳_二模_17、CC0119_2025_丰台_期末_19、CC0373_2026_顺义_一模_18、CC0181_2025_海淀_期末_21、CC0195_2025_西城_一模_20、CC0103_2025_丰台_一模_19

飞哥想说：劳动题永远两边看：保护劳动者，也允许单位依法管理。

### 10. 纠纷解决/调解/仲裁/诉讼/举证

看到：调解、仲裁、诉讼、举证责任、司法确认、证据准备。

翻成法律话：先定争点，再保存证据，再选路径，再提请求。

高频句：当事人应围绕争议保存证据，依法选择协商、调解、仲裁或诉讼等路径。

别写：不要只写依法维权，不要直接替法院判案。

来源题：CC0092_2025_东城_期末_19_1、CC0245_2026_东城_期中_18_2、CC0289_2026_朝阳_二模_18、CC0125_2025_延庆_一模_19、CC0077_2025_东城_一模_19、CC0325_2026_石景山_一模_18、CC0223_2025_顺义_一模_19

飞哥想说：路径题写的是怎么走，不是直接写谁赢。

### 11. 家庭继承/赡养监护

看到：夫妻共同债务、赡养、遗赠扶养协议、继承、老人儿童权益。

翻成法律话：先定身份关系，再看法定义务、协议效力和家庭责任。

高频句：家庭成员权利义务要放在身份关系和法定责任中判断，保护老人、未成年人等合法权益。

别写：不要把家庭题写成普通合同。

来源题：CC0051_2024_海淀_期中_21_1、CC0061_2024_西城_一模_18、CC0045_2024_海淀_二模_19

飞哥想说：家庭题先认关系，再讲责任。

### 12. 法律制度意义/规则保护谁规范谁

看到：设问问意义、价值、作用、保护、推动。

翻成法律话：把本案规则推出保护对象、规范对象和秩序效果。

高频句：该规则保护了具体权利，规范了具体行为，维护了稳定公平的法律秩序。

别写：不要裸写公平正义、全面依法治国。

来源题：CC0340_2026_西城_一模_17、CC0019_2024_朝阳_一模_19、CC0103_2025_丰台_一模_19、CC0131_2025_房山_一模_19、CC0229_2026_东城_一模_18、CC0283_2026_朝阳_一模_18、CC0011_2024_丰台_二模_17、CC0045_2024_海淀_二模_19、CC0181_2025_海淀_期末_21、CC0195_2025_西城_一模_20、CC0063_2024_西城_二模_16

飞哥想说：意义题不是多写两句漂亮话，是把规则的后果说清楚。

## 五、反向筛查穷尽

1. 法院≠公正司法。先看本案权利、义务、责任。
2. 法治≠全面依法治国。高中法律题先落具体规则。
3. 企业≠经济生活。企业在材料里可能是经营者、用人单位、权利人或侵权人。
4. 平台≠劳动关系，除非有派单/签到/奖惩/结算/业务组成。
5. 消费者≠三倍赔偿，除非有欺诈。
6. 创新≠只写保护创新，先写权利对象和侵害方式。
7. 意义≠裸价值，价值必须从规则推出。
8. 多主体≠一锅炖，一主体一链。
9. 案例≠法考，高中写规则+材料+结论。
10. 调解/维权≠直接判案，先写路径。

## 六、高频得分语言

### 诉求支持类

最推荐背：先写“支持/不支持/部分支持”，再写权利基础、事实条件和责任范围。

什么时候用：问诉求、请求、赔偿、退费、惩罚性赔偿时用。

什么时候别用：没有事实支撑时别直接全支持。

飞哥想说：结论先行，阅卷人先看到方向。

### 裁判理由类

最推荐背：法院这样判，是因为材料事实符合某规则，所以责任这样落。

什么时候用：问判决结果、裁判理由、法理依据时用。

什么时候别用：别把法院写成必修三。

飞哥想说：法院题不是政治题，先讲案子。

### 评析行为类

最推荐背：对每个主体分别写：行为是什么、违反什么、承担什么。

什么时候用：问评析、认识、观点时用。

什么时候别用：别一段话评完所有人。

飞哥想说：评析先拆人，不拆人就丢链。

### 合同类

最推荐背：合同成立并生效后，当事人应全面、诚信履行；不履行或履行不合约定，应承担违约责任。

什么时候用：有报价、付款、交付、质量、期限、解除时用。

什么时候别用：生活消费欺诈事实明显时别只写合同。

飞哥想说：合同先看约定，再看违约。

### 劳动类

最推荐背：既保护劳动者合法权益，也允许用人单位依法管理；平台用工看实际控制和三从属性。

什么时候用：有招聘、试用、解除、竞业、工资、平台派单时用。

什么时候别用：别把劳动题写成普通合同。

飞哥想说：劳动题要两边平衡。

### 消费者类

最推荐背：经营者应真实、全面提示说明，保障消费者知情权、公平交易权和安全权。

什么时候用：有虚假宣传、格式条款、预扣款、退费时用。

什么时候别用：无欺诈别直接写三倍赔偿。

飞哥想说：消费者题先找哪项权利被伤。

### 人格权类

最推荐背：先叫准隐私、名誉、个人信息等权利，再写侵害行为和责任方式。

什么时候用：有监控、聊天记录、无事实贬损、个人信息时用。

什么时候别用：别被平台、消费背景带跑。

飞哥想说：权利名叫准，后面才有分。

### 知识产权/不正当竞争类

最推荐背：先写受保护对象，再写未经许可、绕过限制、虚假贬损或商业化利用，最后落责任和秩序。

什么时候用：有作品、技术秘密、数据、唤醒词、商业诋毁时用。

什么时候别用：别只写保护创新。

飞哥想说：创新题先讲被侵害的东西。

### 纠纷解决类

最推荐背：争点、证据、路径、请求四件套。

什么时候用：问调解、维权、仲裁、诉讼、举证时用。

什么时候别用：别直接判谁赢。

飞哥想说：路径题写路线图。

### 意义价值类

最推荐背：规则保护谁，规范谁，维护什么秩序。

什么时候用：问意义、价值、作用、推动时用。

什么时候别用：别裸写法治口号。

飞哥想说：价值必须从规则长出来。
