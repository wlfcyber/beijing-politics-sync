# Confucius 零基础学生试读报告：v13_feige_final_framework_rewrite

角色：Confucius，零基础高三学生试读 agent。

范围：只读取本目录现有 v13 文件；不使用课本，不使用外部资料，不假装知道原题细则。判断标准不是“专家看着顺不顺”，而是“聪明但完全不会的学生，只靠这份材料学完后，能不能做题不扣分”。

## 1. 总判定

**不通过，不能说学完后做题不扣分。**

这版比空泛提纲强，已经能帮学生在考场上“启动”：先看设问、翻材料、写规则事实结论，避免把法律题写成必修三或经济生活。42 题也能让学生知道大致应该往哪一类规则靠。

但它还不是能让零基础学生稳定拿满的“法律宝典”。主要原因是：很多地方只有“启动链”和“高频句”，没有把每道题应写的完整得分句、漏点风险、分点顺序、材料证据如何嵌入答案讲清楚。学生看完会知道方向，但遇到卷面时仍会少写条件、少写责任方式、少写边界，或者把多案、多主体、多路径题写成一句总话。

`04_v13_acceptance.md` 写的是 `FRAMEWORK_REWRITE_CONDITIONAL_PASS`，从学生试读角度只能理解为“有条件可继续打磨”，不能理解为最终可交付。

## 2. 最不像框架的地方

最不像“先前那种框架”的地方，是 42 题题链仍然像“老师备课摘要”，不像“学生考场答题模板”。

具体表现：

1. 题链常写“写某某”“说明某某”“最后落到某某”，但没有把学生能直接写进答题纸的句子完整给出。
2. “命中框架”有时过粗或错位。例如合同题、知识产权题、绿色原则题中同时出现多个知识链，但主轴和副轴没有用“先写/再写/可补一句/不要写成”的等级标清。
3. 反向筛查有用，但太短。它告诉我不要写错，却不告诉我“如果材料有 A 但没有 B，应该停在哪个结论”。
4. 开放容器和待补题隔离是对的，但只列清单，没有告诉学生这些题为什么暂时不能学、考场遇到相似题该走哪个临时路径。
5. GPT 外部审稿在本目录记录为未完成或只完成初始反馈，因此这版缺少一个真正闭环后的修订版。

## 3. 学生学不会的具体原因

**原因一：缺少“满分答案颗粒度”。**  
框架给了“规则 + 材料 + 结论”的总法门，但很多题只给 3 条提纲。零基础学生不知道每条提纲要扩成一句还是两句，也不知道哪句话是阅卷得分点。

**原因二：缺少“条件边界”。**  
例如消费者题说“欺诈成立才谈惩罚性赔偿”，但没有把“什么事实够欺诈、什么事实只够退费或赔偿”拆成考场判断表。学生会背口号，但碰到边界题仍会乱写三倍赔偿。

**原因三：多链题没有分层标识。**  
人格权 + 消费者、知识产权 + 侵权、相邻关系 + 纠纷解决这类题，材料里经常不止一条链。v13 常说“先分两条链”，但没有告诉我哪条是主干、哪条是补充、每条最少写几句。

**原因四：缺少“从材料抠词到答案句”的中间桥。**  
文件反复说“把生活话翻成法律话”，但很多题没有示范“材料原词 -> 法律词 -> 得分句”。零基础学生最缺的恰恰是这个翻译步骤。

**原因五：题链叫“极简版”，但学生需要的是“极简可背 + 可写展开”。**  
极简链适合老师讲，不足以让学生独立做题不扣分。至少每类题要有“30 秒定位句、3 句基础答案、加分边界句、扣分警戒句”。

## 4. 抽样题链试读

以下抽样覆盖合同、劳动、消费者、人格权、知识产权、侵权、相邻关系、纠纷解决。判断只基于 v13 文件本身。

### 4.1 合同：CC0244_2026_东城_期中_18

- 只看框架能否作答：能启动，不能保证不扣分。
- 会漏哪句：会漏“合同责任与人身损害赔偿之间的连接句”，也可能漏“哪些损失有证据和因果关系才支持”。
- 为什么扣分：题链写了邮件报价、转账、展示机、坠毁砸伤，但学生容易只写“合同成立、刘某违约、应赔偿”。如果没有写“瑕疵交付与坠毁伤害有因果关系”“合理损失可请求赔偿”，赔偿链会薄。
- 需要改法：给可背句：“先判合同成立；再判交付不合约定构成违约；最后只支持与瑕疵交付有因果关系、且有证据证明的合理损失。”

### 4.2 劳动：CC0025_2024_朝阳_二模_17

- 只看框架能否作答：能作答到大方向，但三从属性容易背成名词。
- 会漏哪句：会漏“不能只看合作协议名称，要看实际管理控制”的主判句，或者漏“业务纳入平台组成”。
- 为什么扣分：v13 给了派单、签到、奖惩、结算，但没有把三从属性压成学生可逐点对材料写的三句。零基础学生可能只写“平台派单所以是劳动关系”，论证不够。
- 需要改法：三句固定化：“人格从属性看派单、签到、奖惩；经济从属性看报酬结算和依赖；组织从属性看业务是否纳入平台经营。三项合起来，再认定事实劳动关系。”

### 4.3 消费者：CC0143_2025_朝阳_一模_19

- 只看框架能否作答：能写出隐蔽搭售和知情权，但惩罚性赔偿边界不稳。
- 会漏哪句：会漏“合同成立不等于真实充分同意”，以及“欺诈事实成立才支持惩罚性赔偿”。
- 为什么扣分：学生看见“平台”“消费者”“搭售”后，可能直接写三倍赔偿。v13 虽提醒不要乱写，但没有给边界判据，容易把所有退费题都写成欺诈题。
- 需要改法：补小表：“未提示/未明示 -> 知情权、自主选择权受损；隐瞒并诱导交易 -> 欺诈；欺诈成立 -> 惩罚性赔偿；欺诈不够 -> 退费/赔偿但不写惩罚性赔偿。”

### 4.4 人格权：CC0305_2026_海淀_一模_18_3

- 只看框架能否作答：能分隐私权和消费者欺诈两条链，但学生会写散。
- 会漏哪句：会漏“包间监控和聊天信息公开未经同意，具有私人性，所以侵害隐私权/个人信息权益”，只写“公开信息违法”。
- 为什么扣分：人格权题最怕权利名不准。v13 说“先叫准权利名”，但本题没有示范如何把“监控、聊天信息公开”分别落到隐私、个人信息、经营者责任。
- 需要改法：拆成两条完整句：第一条“未经同意公开包间监控、聊天信息，侵害隐私权或个人信息权益，应停止侵害、赔礼道歉、赔偿损失”；第二条“虚假宣传治疗功效，损害知情权，构成欺诈时才支持惩罚性赔偿。”

### 4.5 知识产权/不正当竞争：CC0206_2025_西城_期末_19

- 只看框架能否作答：能写不正当竞争，但容易把“唤醒词”写成普通商标或人格权。
- 会漏哪句：会漏“已有影响力的商品名称/标识可受反不正当竞争法保护”。
- 为什么扣分：v13 说“小爱同学”有影响力、抢注造成混淆，但学生不懂为什么不是普通流行词。没有“识别来源、造成混淆、违背诚信”的三步模板，容易写成泛泛保护创新。
- 需要改法：固定为“对象 -> 行为 -> 后果 -> 责任”：有一定影响的标识受保护；抢注并同类宣传；足以混淆或误导；构成不正当竞争，停止侵害并赔偿。

### 4.6 侵权：CC0002_2024_丰台_一模_17

- 只看框架能否作答：能识别好意同乘，但侵权责任分配还不够稳。
- 会漏哪句：会漏“不是免赔，而是可减轻责任”，也可能漏“搭乘人未系安全带属于自身过错”。
- 为什么扣分：零基础学生容易把善意帮忙写成“道德好人不用赔”，或只写“鼓励互助”。v13 有提醒，但答题句还不够硬。
- 需要改法：补必背句：“好意同乘不当然免责；驾驶人无故意或重大过失的，可依法减轻责任；受害人自身过错也影响责任分担。”

### 4.7 相邻关系：CC0364_2026_通州_期中_19_1

- 只看框架能否作答：能作答基本结论，但容易漏程序与事实勘查。
- 会漏哪句：会漏“全体业主同意、公示、审批”与“现场勘查未影响采光”这两个事实支点。
- 为什么扣分：学生可能只背“相邻关系应方便生活、公平合理”，但本题得分不只在原则，还在“程序合法 + 没有实际损害 + 停止妨害”。少了事实勘查，结论像空喊原则。
- 需要改法：改成三格：“程序合法 -> 不是私自施工；影响不存在或较小 -> 主张缺事实依据；相邻原则 -> 停止妨害/合理补偿或另行救济。”

### 4.8 纠纷解决：CC0325_2026_石景山_一模_18

- 只看框架能否作答：能看出是举证规则题，但零基础学生很容易混诉讼类型。
- 会漏哪句：会漏“公诉、行政诉讼、民事侵权诉讼的举证责任不同”。
- 为什么扣分：v13 写了三案对应举证人，但没有给总口诀。学生只背“谁主张谁举证”会把行政诉讼、公诉案件都写错。
- 需要改法：补口诀：“刑事公诉检察院证有罪；行政诉讼行政机关证合法；民事原告证损害和因果，特殊侵权被告证无过错。”

### 4.9 纠纷解决 + 相邻 + 不正当竞争：CC0223_2025_顺义_一模_19

- 只看框架能否作答：能看出两案分开写，但题链编号和案例称呼会让学生混乱。
- 会漏哪句：会漏“案例1是相邻纠纷通过诉讼调解化解；案例2是企业名称混淆，通过行政裁决和诉讼定分止争”。
- 为什么扣分：答案骨架写“案例1”“案例2”，但材料触发列了 3 条，学生可能不知道第三条是第二案的展开，还是第三个案例。多案例题必须更清楚地标主案、副案、收束价值。
- 需要改法：改成“案一：邻里相邻纠纷；案二：企业名称混淆纠纷；共同收束：法律程序把生活冲突和市场冲突转化为可证明、可裁判、可履行的解决方案。”

## 5. 下一版必须怎样改才像先前框架

1. **每类框架改成四层：触发词、判定句、得分句、扣分句。**  
   不能只写“看到什么/翻成什么/高频句”。要让学生知道看到材料后第一句写什么，第二句怎么带材料，第三句怎么落责任。

2. **42 题每题补“最低不扣分答案”。**  
   每题至少给 3-5 句可直接写的答案，不需要长，但必须完整。现在很多题只有“写什么”，没有“怎么写出来”。

3. **多链题必须标主轴和副轴。**  
   格式建议：主轴必写；副轴看设问写；价值句最后写；不能写的误区。这样学生才不会把人格权、消费者、合同、侵权混成一段。

4. **边界题做小表。**  
   消费者三倍赔偿、平台劳动关系、竞业限制、相邻关系补偿、好意同乘减责、举证责任，都要有“有 A 才能写 B；没有 A 只能写 C”的判断表。

5. **开放容器要给临时处理法。**  
   5 道参考题和 6 道缺源题隔离是对的，但下一版应说明：这些题不进核心，不代表学生遇到类似题空着，而是先走“设问定型 -> 材料翻法 -> 争点证据路径”的兜底框架。

6. **修复题链错配和断句问题。**  
   例如 CC0157 的答案骨架出现“侵权。3. 违约责任”这种断裂，学生会直接学坏。所有题链都要检查是否一格一链、编号连续、主语清楚。

7. **外部审稿未闭环前，不要把 v13 宣称为最终版。**  
   本目录模型记录显示 GPT 审稿没有完整完成。学生版可以试读，但下一版必须在完整审稿、修订、再试读后才能说接近交付。

## 结论

v13 的价值在于“能帮学生起步”，不是“能让零基础学生不扣分”。它已经有飞哥课堂的口吻和反向筛查意识，但还缺少先前框架那种“学生拿来就能按题写”的密度。下一版要从“框架摘要”改成“考场可背可写的法律答题卡”：每类有固定判定句，每题有最低不扣分答案，每个边界有条件表，每个多链题有主副轴。否则聪明学生能少跑偏，但不能保证不扣分。
