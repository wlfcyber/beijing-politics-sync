# 01 v13.0 双轴法律主观题框架章

生成时间：2026-05-23 14:35 +08:00

## 一、这次纠偏改了什么

v12.2 的六入口能帮助学生判断题目问法，但它本质上是“设问动作框架”：补表、判责、诉求、评析、意义、路径。它能让学生知道怎么开答，却不能稳定回答更前置的问题：这道题到底属于哪一种法律关系、哪一块法律内容。

Round04 中，正确 GPT Pro 网页输出和 Claude Opus 4.7 Adaptive 网页输出都给出 `UPGRADE_TO_DOUBLE_AXIS`。Codex 本地证据裁决接受该结论：v12.2 冻结为回滚基线；v13.0 以 A 轴为主框架，以 B 轴为答题动作轴。

## 二、总公式

> 先定 A：法律关系/内容；再定 B：设问动作/命题路径；最后写“事实触发 -> 规则要件 -> 责任/效力/路径结论 -> 条件性意义价值”。

意义价值不再做大筐。只有当题目问意义、保护、推动、认识，或者材料本身把制度效果作为评分点时，才在答案末端写意义；意义必须指向具体对象，如消费者、劳动者、创新主体、交易安全、家庭成员、公共秩序或程序正义。

## 三、A 轴：法律关系/内容轴

### A1 民事法律关系总论

- 状态：基础层，不作为独立主干计数；本版 primary 支持数：0。
- 触发词：主体、客体、内容、民事行为能力、意思表示、权利义务基础。
- 答题核心：先确认谁和谁之间发生民事法律关系，再判断主体资格、行为能力、权利义务和行为效力。
- 学生预警：A1 是所有民法题的底盘，不能把它当成万能主干；只有主体能力或行为效力是采分中心时才显性写出。
- 治理说明：A1 是底层检查轴，不单独凑成主干；本版把涉及行为能力、意思表示、主体资格的题放入相关实体主轴，并将 A1 写入副轴或说明。

### A2 人身权与人格权

- 状态：启用主干；本版 primary 支持数：4。
- 触发词：生命健康、姓名、肖像、名誉、隐私、个人信息、未成年人身心利益。
- 答题核心：先判断具体人格利益，再对照侵害行为、损害后果、责任方式和保护边界。
- 学生预警：不要把所有网络、平台、自媒体材料都写成消费者题；先看被伤害的是人格利益还是交易利益。

### A3 物权与相邻关系

- 状态：启用主干；本版 primary 支持数：3。
- 触发词：所有权、共有部分、物业、通行采光通风、加装电梯、相邻不动产利益。
- 答题核心：先定物权归属或相邻利益，再写合理使用、便利生活、团结互助、公平合理和程序边界。
- 学生预警：相邻关系不是单纯道德劝和，必须落到权利边界、合理限度和法定程序。

### A4 合同

- 状态：启用主干；本版 primary 支持数：5。
- 触发词：要约承诺、合同成立生效、履行、违约、格式条款、免责、解除、损害赔偿。
- 答题核心：按成立/效力/履行/违约/责任的顺序拆链，最后再落诚信、公平或交易安全。
- 学生预警：看到协议、订单、平台条款，不要直接喊诚信；先把合同链条写清。

### A5 知识产权与市场竞争接口

- 状态：启用主干；本版 primary 支持数：8。
- 触发词：著作权、专利、商标、商业秘密、技术秘密、混淆、商业诋毁、数据抓取、不正当竞争。
- 答题核心：先识别保护对象，再写未经许可/混淆/泄密/诋毁等侵害方式，最后写停止侵害、赔偿、惩罚性赔偿或秩序保护。
- 学生预警：创新材料不是价值题本身；必须说清用哪一种知识产权或竞争规则保护创新。

### A6 侵权责任

- 状态：启用主干；本版 primary 支持数：5。
- 触发词：损害、过错、因果关系、安全保障义务、产品责任、动物致害、高空抛物、环境侵权、减责。
- 答题核心：按侵权要件、归责原则、责任主体、减免责事由和责任方式展开。
- 学生预警：不要把侵权题写成泛泛保护权利；采分点通常在过错、因果、举证、责任分担。

### A7 婚姻家庭与继承

- 状态：启用主干；本版 primary 支持数：3。
- 触发词：夫妻财产、共同债务、赡养扶养、遗赠扶养协议、继承、遗产管理、老年人权益。
- 答题核心：先定身份关系和财产/扶养/继承规则，再写家庭伦理与法律义务如何统一。
- 学生预警：不能只写孝老爱亲；家庭题同样要写具体法律义务和财产规则。

### A8 劳动关系

- 状态：启用主干；本版 primary 支持数：6。
- 触发词：劳动合同、事实劳动关系、平台用工、劳动纪律、竞业限制、集体协商、就业平等。
- 答题核心：先判断劳动关系或劳动权益，再写从属性、权利义务、解除/限制/协商的合法性和公平效率。
- 学生预警：劳动法不是只保护劳动者，也保护依法用工和有序劳动关系。

### A9 消费者权益与经营秩序

- 状态：启用主干；本版 primary 支持数：5。
- 触发词：消费者知情选择、公平交易、食品安全、惩罚性赔偿、平台扣款、直播打赏、经营者诚信。
- 答题核心：先判断消费者/经营者关系，再写知情权、选择权、公平交易、安全保障、欺诈或惩罚性赔偿。
- 学生预警：消费者题不能只写诚信经营；要把具体消费者权利和经营者义务对应上。

### A10 多元纠纷解决与诉讼程序

- 状态：启用主干；本版 primary 支持数：3。
- 触发词：和解、调解、仲裁、诉讼、行政裁决、举证责任、诉讼代理、救济路径。
- 答题核心：先确定争议类型和程序路径，再写谁主张、谁举证、找谁解决、如何救济。
- 学生预警：程序题不能用一个规则打天下；要先区分民事、行政、仲裁、调解等不同路径。

## 四、B 轴：设问动作/命题路径轴

### B1 表格/裁判要点/补链（继承 v12.2 E1）

- 识别口令：完成表格、补裁判要点、按示例补理由或一格一链。
- 命题路径：形式触发 -> 一格一链 -> 法律关系/要件 -> 格内结论。
- 本版支持数：9。

### B2 判决/裁判/责任理由（继承 v12.2 E2）

- 识别口令：问法院、仲裁或裁判机关为什么这样判。
- 命题路径：裁判对象 -> 法律关系 -> 要件/过错/责任 -> 裁判结论。
- 本版支持数：8。

### B3 诉求/请求能否支持（继承 v12.2 E3）

- 识别口令：问某项诉求、请求、主张能否得到支持。
- 命题路径：诉求对象 -> 权利基础 -> 要件匹配 -> 支持/不支持。
- 本版支持数：3。

### B4 评析/认识/观点（继承 v12.2 E4）

- 识别口令：问谈认识、评析观点、评价做法或法律变化。
- 命题路径：观点/做法 -> 法律关系 -> 边界判断 -> 规范评价。
- 本版支持数：7。

### B5 意义/价值/保护/推动（继承 v12.2 E5）

- 识别口令：问制度、规定、做法的作用、意义、价值或如何保护推动。
- 命题路径：规则/制度 -> 保护对象 -> 行为方式 -> 权益和秩序效果。
- 本版支持数：11。

### B6 调解/维权/纠纷解决路径（继承 v12.2 E6）

- 识别口令：问如何调解、怎样维权、救济路径、举证责任。
- 命题路径：纠纷事实 -> 争议焦点 -> 主体/证据/路径 -> 救济方案。
- 本版支持数：4。

## 五、命题人路径

每道题都按五个问题过筛：

1. 命题人想培养的法律治理能力是什么：识别权利义务、匹配要件、判断责任、选择程序，还是解释制度价值。
2. 命题人为什么选择这个载体：案例、法条、裁判、法律现象、社会治理场景。
3. 材料先触发哪一类法律关系：合同、侵权、人格权、知识产权、家庭继承、劳动、消费、物权相邻或程序救济。
4. 设问要求学生完成什么动作：补链、判责、支持诉求、评析、说意义、给路径。
5. 价值输出是否有真实触发：没有事实和规则支撑时，不写空泛法治口号。

## 六、学生拿题时的操作法

第一步，圈主体和关系：谁和谁发生什么法律关系。第二步，圈权利义务和责任：被侵犯的是何种权利，违反的是何种义务。第三步，看问法动作：让补表、判责、评析、说意义还是给路径。第四步，按 A x B 输出答案骨架。第五步，用材料词收口，不让价值句脱离案件。

## 七、坏框架自检

如果一个框架让 E5 意义价值吞掉劳动、知识产权、家庭、物权、侵权、消费等不同内容，它不是好框架。如果一题只有问法标签，没有法律关系标签，学生会记住题型，却不知道怎么迁移。v13.0 的底线是：42 题每题必须同时有 A 标签和 B 标签。
