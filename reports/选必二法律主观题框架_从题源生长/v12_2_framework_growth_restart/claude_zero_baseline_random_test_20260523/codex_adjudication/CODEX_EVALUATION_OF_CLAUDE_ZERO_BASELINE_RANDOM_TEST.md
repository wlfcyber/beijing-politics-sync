## 对框架质量的反思

这次盲测说明 v13.1 不是“空架子”。一个零基础但聪明的学生，在只看框架和题面压缩信息的情况下，能够：

- 较准确识别 A轴法律入口：5题中 4题主入口基本正确，第1题虽主入口错置，但抓住了关键副轴。
- 准确识别 B轴设问动作：5题全部命中 B2/B5/B4/B3/B1。
- 自动形成“材料信号 -> 法律规则 -> 责任/价值/评价结论”的答题顺序。

但它还不是顶级学生版“法律宝典”的全部。最大的缺口不是框架，而是框架后的操作工具：

- 缺少每个 A轴入口后的高频法条/要件工具箱。学生知道进哪个门，但进门后可调用的条文、构成要件、责任方式还不够。
- 缺少 B1 表格题的常见列结构和补链样式。没有示例时，聪明学生会自造表格，真实评分中风险很高。
- A1 作为“得分脊柱但非场景主入口”的排序规则还要更硬，尤其是未成年人打赏、代理、意思表示、时效类题。
- 双链/双主轴题需要明确先后顺序：先人格权/侵权链，后消费者欺诈链；或先主法益，后交易/责任副链。

## 建议补丁

1. 在 v13.1 后续教学版中增加“入口后法条工具箱”：A2/A5/A6/A7/A9 每个入口列出最常见 5-8 个要件句和责任句。
2. 为 B1 增加三种表格模板：案件事实-法理分析-裁判要点；权利/义务-法律依据-结论；行为-法律评价-责任承担。
3. 为 A1 增加硬规则：行为能力、意思表示、代理、时效若只是裁判脊柱，不得自动压过题目真实交易/人格/消费者/合同场景主入口。
4. 给双链题增加排序语句：先按题干最先要求判断的诉求或最核心受侵害法益写，再写副轴责任。
