# 选必二法律与生活：法律宝典 v14.3 GPT补丁版

# 01 先背这套：选必二法律主观题不扣分框架 v14.3

## 0. 这套框架到底解决什么

法律题不是先背法条，也不是先贴标签。它的固定任务只有一句话：

> 把一个生活冲突，翻译成法律关系、法律规则和法律后果。


## 0A. 必须先背的母公式

v14.3 把所有题先压成一条母公式：

`设问定型 -> 材料翻法 -> 规则事实结论 -> 必要价值`

- 设问定型：先看题目动词，是判理由、评析、意义、表格、诉求还是路径。
- 材料翻法：把生活事实翻成法律主体、权利义务、构成条件、证据和程序。
- 规则事实结论：每一句答案都必须是规则带事实，事实落结论。
- 必要价值：只有题目问意义，或判决明显要求价值收束时才写，不能空喊法治。

## 0B. C 轴：题内开关，不增加大入口

C 轴不是新的知识目录，而是遇到特殊题时先按下的开关：

| C开关 | 看到什么就启动 | 写作硬句 |
|---|---|---|
| C1 行为能力与追认 | 未成年人、老人、金额巨大、监护人不知情 | 先查行为能力、意思表示和追认，再谈返还和过错分担。 |
| C2 三从属性 | 平台派单、考勤、奖惩、结算、组织管理 | 不看协议名称，看实际管理控制、经济依赖和组织纳入。 |
| C3 技术秘密三性 | 图纸、配方、数据、算法、研发资料 | 先写秘密性、价值性、保密措施，再写不正当获取使用。 |
| C4 消费者欺诈三倍 | 隐瞒、误导、虚假宣传、搭售 | 欺诈事实足够才写惩罚性赔偿，不足时只写退费或赔偿。 |
| C5 好意同乘减责 | 免费搭车、帮忙接送、非营运 | 减轻不等于免除；还要看驾驶人过错和受害人自身过错。 |
| C6 竞业限制平衡 | 离职、竞业、补偿、商业秘密 | 同时看劳动者就业权、单位秘密保护和限制范围是否适度。 |
| C7 多主体分责 | 平台、监护人、本人、经营者、第三人 | 一个主体一条链，不能糊成一段。 |
| C8 证据路径 | 维权、调解、诉讼、举证、公证 | 先固定争议焦点，再写证据、路径和请求。 |

## 0C. 混合设问拆分规则

题干有两个动词，就拆两个 B 动作：

- 理由 + 意义：先 B2 判理由，再 B5 收意义。
- 评析 + 建议：先 B4 评边界，再 B6 给路径。
- 补全 + 任选展开：先 B7 补短句，再 B6 展开权利、证据、路径、请求。
- 责任 + 维权：先 B2/B3 定责任或请求，再 B6 写证据、路径、请求。

## 0D. 反向筛查

落笔前用这十条排错：

1. 看见法院，不等于写公正司法；先看实体法律关系。
2. 看见平台，不等于劳动关系；先查三从属性。
3. 看见消费者，不等于三倍赔偿；先查欺诈事实。
4. 看见未成年人消费，先查行为能力和追认。
5. 看见监控门铃，不能只写物权；同步查隐私、个人信息和相邻安宁。
6. 看见技术资料，先查秘密性、价值性、保密措施。
7. 看见好意同乘，减责不等于免责。
8. 看见表格，按原表头；看不见表头就用事实、规则、裁判要点三列。
9. 看见两个动词，先拆小问再落笔。
10. 看见意义，必须从本案具体规则推出，不能空喊价值。

所以所有题都按五步走：

| 步骤 | 问自己 | 产出 |
|---|---|---|
| 1. 生活冲突 | 谁和谁因为哪件事冲突？ | 材料一句话 |
| 2. A 入口 | 这是哪类法律关系或法律内容？ | A 轴主入口，必要时副入口 |
| 3. B 动作 | 题目让我写表格、责任、诉求、评析、意义还是路径？ | B 轴答案形状 |
| 4. 得分链 | 哪些事实压进哪些规则？ | 事实 -> 规则 -> 结论 |
| 5. 停止 | 题问到哪里就写到哪里。 | 问责任落责任，问意义落意义，问路径落路径 |

最短口诀：

> 先定生活冲突，再定设问动作；材料进规则，结论不飘；主链拿分，副链补刀。

## 1. 先会总流程，再背 A/B

拿到新题，按这四句话开工：

1. 用一句话说材料：谁对谁做了什么，造成什么争议。
2. 用 A 轴问：这个争议本质是哪类法律关系。
3. 用 B 轴问：设问到底要我产出什么样的答案。
4. 写成得分链：规则不能离开材料，材料不能离开结论。

A/B 双轴不是开头背诵的废话，而是防扣分工具：A 防止法律关系跑偏，B 防止答案形状跑偏。

## 2. A 轴：九个高频入口和一个效力骨架

本批 42 道锁定核心题的 A 轴分布：A10_多元纠纷解决与诉讼程序 3题, A2_人身权与人格权 4题, A3_物权与相邻关系 3题, A4_合同 5题, A5_知识产权与竞争秩序 8题, A6_侵权责任 5题, A7_婚姻家庭与继承 3题, A8_劳动关系 7题, A9_消费者权益与经营者义务 4题。

| 入口 | 学生记忆句 | 材料触发 | 二级开关 | 别误判成 | 得分链 |
|---|---|---|---|---|---|
| A1_民事法律关系总论（辅助骨架） | 先查效力，但在行为能力、意思表示、追认直接决定诉求时，A1就是得分脊柱。 | 年龄、智力、意思表示、代理、追认、时效、民事法律行为效力。 | 年龄/智力/金额是否相适应；是否同意或追认；行为效力是否直接决定诉求；若是，A1作主脊柱，具体场景只作副入口。 | 不能机械说A1只作辅助。未成年人消费、超龄超额交易、追认、效力直接决定结论时，A1先作主脊柱，再让消费者、合同、劳动等入口说明场景。 | 行为能力/意思表示/追认 -> 效力 -> 返还/责任分担 -> 场景入口补充。 |
| A2_人身权与人格权（4题） | 先叫准权利名，再写侵害和责任。 | 生命健康、姓名、肖像、名誉、商誉、隐私、个人信息、未成年人身心利益。 | 先叫权利名；看是否未经同意或无事实依据；看是否有损害；落停止、删除、道歉、赔偿。 | 消费者、平台、邻里、校园等背景不会吞掉人格权。材料出现隐私、个人信息、名誉、肖像、安宁，先把权利名叫准，再看是否需要A3/A9副入口。 | 具体人格利益 -> 侵害行为 -> 损害后果 -> 停止/删除/道歉/赔偿。 |
| A3_物权与相邻关系（3题） | 先判权利边界，再看实际影响。 | 房屋、共有部分、车位、通行、采光、通风、安全、相邻关系、业主共同决定。 | 专有还是共有；是否共同决定；是否实际影响采光通行安全；审批消防是证据还是路径。 | 审批、公示、消防、法院勘查是事实和程序证据，不自动抢成A10；但监控、门铃、采光、通行类题要同步检查人格权边界。 | 权利归属/共有部分 -> 使用边界/相邻影响 -> 程序和实际影响 -> 公平合理处理。 |
| A4_合同（5题） | 先合同成立履行，再责任。 | 买卖、服务、报价、付款、交付、质量、期限、退费、违约金、研发委托、租赁。 | 合同是否成立生效；履行是否合约；损失是否有证据；商业购买还是生活消费。 | 生活消费中的知情权、欺诈、格式条款可转 A9；商业购买或履约损失通常先 A4。 | 合同成立生效 -> 是否全面诚信履行 -> 违约事实 -> 继续履行/补救/返还/违约金/赔偿。 |
| A5_知识产权与竞争秩序（8题） | 保护创新投入和公平竞争。 | 作品、商标、商业秘密、技术秘密、数据抓取、混淆、虚假宣传、商业诋毁、竞争秩序。 | 对象是否受保护；秘密性/价值性/保密措施是否成立；手段是否不正当；是否商业化使用并损害竞争秩序。 | 劳动者离职、合同关系、平台服务只是取得或使用信息的背景，不抢 A5。 | 受保护对象 -> 不正当获取/使用/传播/混淆 -> 损害创新或竞争利益 -> 停止/消除/赔偿。 |
| A6_侵权责任（5题） | 四件套：损害、过错、因果、责任。 | 人身损害、财产损害、安全保障义务、公共场所、旅游游乐、工作人员执行任务、产品致损。 | 损害是什么；过错或管理义务违反是什么；因果关系是否成立；是否有受害人过错。 | 损害从合同瑕疵履行引出时，先 A4 定违约，再用 A6 补损害赔偿。 | 权益受损 -> 违法/过错或管理义务违反 -> 因果关系 -> 赔偿和减责。 |
| A7_婚姻家庭与继承（3题） | 身份关系优先，不写成普通合同。 | 夫妻、亲子、扶养、赡养、共同债务、遗嘱、继承、遗赠扶养协议、老人儿童权益。 | 身份关系是什么；协议/遗嘱/继承效力如何；谁履行了扶养赡养义务；保护哪类家庭责任。 | 遗赠扶养协议、家庭财产安排不简单写成普通合同履行。 | 身份/继承关系 -> 法定义务或处分效力 -> 履行情况 -> 家庭责任和价值。 |
| A8_劳动关系（7题） | 劳动不是普通合同。 | 招聘、简历、试用期、解除、工资、社保、竞业限制、劳动仲裁、用工管理。 | 是否劳动关系；劳动者义务和用人单位义务分别是什么；解除/竞业/社保是否事实充分、程序合法、限制适度。 | 不要把试用期解除、工资社保、竞业限制写成普通合同欺诈。 | 劳动关系 -> 劳动者/用人单位义务 -> 解除或限制是否合法 -> 支持请求或维护用工秩序。 |
| A9_消费者权益与经营者义务（4题） | 生活消费中的知情、公平、安全、欺诈。 | 未成年人消费、打赏、文身、预扣款、格式条款、虚假宣传、知情权、公平交易权、安全保障。 | 是否生活消费；知情、公平、安全、欺诈哪项受损；经营者是否提示说明；退赔或分担是否有事实。 | 商业经营用途购买通常先 A4；人格权请求仍要独立写 A2。 | 消费关系 -> 经营者义务 -> 消费者权利受损 -> 退赔/惩罚性赔偿/合理分担。 |
| A10_多元纠纷解决与诉讼程序（3题） | 只有问路径、证据、程序时才主导。 | 证据、调解、仲裁、诉讼、起诉状、举证责任、司法确认、公益诉讼、维权准备。 | 争议焦点是什么；谁负举证责任；证据有哪些；走协商、调解、仲裁、诉讼还是司法确认。 | 不要因为材料出现法院、判决、调解就抢掉实体法律关系。 | 争议焦点 -> 证据 -> 协商/调解/仲裁/诉讼 -> 明确请求和程序效果。 |

### A 轴不扣分硬句

- **A1_民事法律关系总论**：限制民事行为能力人实施明显超出其年龄、智力和经济认知范围的行为，未经法定代理人同意或追认，不发生应有效力；已取得财产结合过错返还或分担。
- **A2_人身权与人格权**：未经同意公开他人私密活动、私密信息或可识别个人信息，或无事实依据贬损他人名誉、商誉，应承担停止侵害、消除影响、赔礼道歉、赔偿损失等责任。
- **A3_物权与相邻关系**：相邻权利人应按照有利生产、方便生活、团结互助、公平合理原则处理相邻关系；利用共有部分或改变共有部分用途，应符合法定程序并结合实际影响判断。
- **A4_合同**：合同成立并生效后，当事人应全面履行、诚信履行；未按约交付、迟延履行、履行不符合约定或拒不履行，构成违约并承担相应责任。
- **A5_知识产权与竞争秩序**：行为人以不正当方式获取、使用他人合法投入形成的数据产品、商业秘密或经营成果，并进行商业化利用，损害竞争利益和公平竞争秩序，应承担相应民事责任。
- **A6_侵权责任**：经营者或公共场所管理者负有安全保障义务；工作人员执行工作任务造成他人损害的，通常由用人单位或经营者承担相应侵权责任。
- **A7_婚姻家庭与继承**：家庭成员之间的权利义务应在身份关系和法定责任中判断，既保护婚姻家庭秩序，也维护老人、未成年人等主体合法权益。
- **A8_劳动关系**：劳动关系既保护劳动者合法权益，也允许用人单位依法管理；解除、竞业限制、工资社保等争议要同时看事实、制度、程序和合法合理边界。
- **A9_消费者权益与经营者义务**：经营者应保障消费者知情权、公平交易权和安全保障权益；虚假宣传、提示说明不足、格式条款排除主要权利等，应承担退赔、赔偿或相应责任。
- **A10_多元纠纷解决与诉讼程序**：当事人应围绕争议焦点保存证据，选择协商、调解、仲裁、诉讼或司法确认等合法路径，提出明确、合理、可证明的请求。

## 3. B 轴：设问动作决定答案长相

本批 42 道锁定核心题的 B 轴分布：B1_表格/裁判要点/补链 9题, B2_判决/裁判/责任理由 6题, B2_判决/裁判/责任理由 + B5_意义/价值/保护/推动 2题, B3_诉求/请求能否支持 3题, B4_评析/认识/观点 5题, B4_评析/认识/观点 + B6_调解/维权/纠纷解决路径 1题, B5_意义/价值/保护/推动 11题, B6_调解/维权/纠纷解决路径 3题, B7_法律问题识别/填空 1题, B7_法律问题识别/填空 + B6_调解/维权/纠纷解决路径 1题。

| 动作 | 学生记忆句 | 答案形状 | 第一笔写什么 |
|---|---|---|---|
| B1_表格/裁判要点/补链（9题） | 一格一链。 | 事实 -> 规则 -> 裁判要点。真实表头优先；看不清时用默认三列。 | 先读列名，不写散文；一行只处理一个案例或一个要点。 |
| B2_判决/裁判/责任理由（6题） | 先关系，后规则事实，最后责任。 | 法律关系 -> 规则 -> 材料事实 -> 结论 -> 责任方式。 | 第一句就回应法院为什么这样判或谁担责。 |
| B3_诉求/请求能否支持（3题） | 先结论，再分项。 | 支持/不支持/部分支持 -> 权利基础 -> 要件事实 -> 责任范围。 | 不要绕圈，先写诉求是否支持及范围。 |
| B4_评析/认识/观点（5题） | 合理处、边界、正确立场。 | 观点合理处 -> 片面或边界 -> 正确法律立场 -> 本案处理。 | 不要写成纯意义题，要先评观点或做法。 |
| B5_意义/价值/保护/推动（11题） | 从本案推出价值。 | 保护对象 -> 约束行为 -> 维护秩序/推动治理。 | 不空喊法治；每个意义都要挂回材料事实。 |
| B6_调解/维权/纠纷解决路径（3题） | 证据、路径、请求。 | 固定争议焦点 -> 保存证据 -> 选择路径 -> 提出请求。 | 不要只写“依法维权”，要写清准备什么和怎么走。 |
| B7_法律问题识别/填空（1题） | 短语短句，不写大段。 | 法律问题名词 + 关键边界；每空一句，不扩写判决理由。 | 先看横线问的是问题名、责任名、程序名还是权利名。 |
| B7_法律问题识别/填空 + B6_调解/维权/纠纷解决路径（1题） | 先补全，再展开。 | 短句补全所有法律问题 -> 任选一项写权利、侵害、证据、路径、请求。 | 先把每个问答的法律问题补成短句，再展开任选项。 |
| B2_判决/裁判/责任理由 + B5_意义/价值/保护/推动（2题） | 先判理由，再收意义。 | 裁判结论/责任理由 -> 规则事实 -> 价值意义；意义必须从本案规则推出。 | 先回答为什么这样判，再写该判决保护了谁、规范了什么秩序。 |
| B4_评析/认识/观点 + B6_调解/维权/纠纷解决路径（1题） | 先评边界，再给路径。 | 观点合理处/片面处 -> 法律边界 -> 证据、协商、调解、诉讼或整改方案。 | 先评材料中做法或观点是否越界，再提出可执行处理方案。 |

## 4. 混合题排序

| 混合情况 | 排序 | 不扣分提醒 |
|---|---|---|
| A2+A9 消费纠纷里有人格权 | 先人格权请求，再消费者退赔请求 | 隐私、名誉、个人信息不能被“消费者权益”吃掉。 |
| A4+A6 瑕疵商品或服务致损 | 先合同违约，再侵权或产品损害赔偿 | 不只写合同，也不抛开合同只写侵权。 |
| A3 物权题里有审批、消防、法院勘查 | 先物权/相邻边界，再把程序事实当证据 | 除非题问路径，否则 A10 不抢主入口。 |
| A5 竞争题里有劳动离职 | 先商业秘密/数据/竞争秩序，再把离职当接触信息背景 | 不把技术秘密题写成劳动解除题。 |
| B1 表格题后又问启示 | 先填表格，再用一两句价值收束 | 表格是主分，启示只补尾。 |
| B3 诉求题又问意义 | 先支持/不支持及理由，再收意义 | 诉求通常 80%，意义通常 20%。 |
| 一个题里有两个主体或两个诉求 | 一个诉求一条链，逐项表态 | 不要把两个主体、两个请求、两个规则糊成一段。 |

## 5. 学生盲测后补上的十四条硬规则

下面十四条不是装饰，而是零基础学生试读时暴露的扣分点。遇到对应题型，直接按硬规则写：

1. B1 表格题看不见表头时，默认三列写：材料事实 -> 法律规则 -> 裁判要点/意义。
2. 技术秘密题必须写：秘密性 -> 价值性 -> 保密措施 -> 不正当获取使用 -> 责任。
3. 数据抓取题必须写：合法投入形成数据产品 -> 绕过限制抓取 -> 商业化使用 -> 损害竞争秩序。
4. 在职管理人员违规履职归入 A8 边界：岗位职责 -> 诚信审慎义务 -> 违规行为 -> 单位损失 -> 相应责任。
5. A4+A6 商品致损题固定顺序：合同违约 -> 人身/财产损害 -> 因果关系 -> 赔偿范围。
6. 商业损失只有在用途已告知、有证据、有因果、可预见时才直接支持；证据不足时写需证明。
7. B2+B6 混合题先写责任，再写证据、路径、请求。
8. 劳动关系意义题按：表达诉求 -> 保护权益 -> 规范用工 -> 减少争议 -> 稳定生产。
9. 一个题多主体时，必须写成一个主体一条链，禁止糊成一段。
10. 所有 B5 价值题必须先写材料中的具体制度或行为，再写保护对象和秩序价值。
11. 题干已定性为“技术秘密侵权”时，即使材料压缩没列明，也要先写秘密性、价值性、保密措施这个法律前提。
12. 多主体评析题按主体分链：外部竞争侵权 -> 合同履行责任 -> 内部履职责任；哪个主体对应哪条链就写哪条。
13. “补充完整+任选其一”题先短句补全所有法律问题，再任选一项展开权利、侵害、证据、路径、请求。
14. 法律+经济综合意义题按：法律机制 -> 权益公平 -> 规则预期 -> 降低争议成本 -> 提高效率 -> 平衡。
15. 混合设问先拆动词：理由+意义写B2+B5，评析+建议写B4+B6，补全+任选展开写B7+B6；不能用一个B标签吃掉全题。
16. 题卡迁移必须写三信号、主入口和错入口排除；不能只写一句同类题识别。

## 6. 答案生成器

每道主观题都可套以下骨架：

```text
第一句：本题围绕____发生争议，核心是____。
第二句：从法律关系看，____属于____，应适用____规则。
第三句：材料中____表明____，符合/不符合____条件。
第四句：因此____应承担/不承担____责任，或____请求应支持/不支持。
第五句：如果题目问意义，再写：该处理保护了____，规范了____，维护了____秩序。
```

## 7. 四个停止条件

1. 问责任，落责任；不要硬写宏大价值。
2. 问意义，必须从本案权利和行为推出；不要空喊法治。
3. 问路径，写证据、路径、请求；不要只写“依法维权”。
4. 问问题识别或填空，写短语短句；不要扩成长判决。

## 8. 先学后做的顺序

学生学习时不要先刷 42 题。先按这个顺序：

1. 背总流程：生活冲突 -> A 入口 -> B 动作 -> 得分链 -> 停止。
2. 背 A 轴材料触发和别误判成什么。
3. 背 B 轴答案形状。
4. 看 42 题拆解时，只问：这题为什么这样进 A，为什么这样进 B，材料哪句话进了规则。
5. 遇到新题，先照抄流程，不要背题号。
