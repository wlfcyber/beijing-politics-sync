# Codex Evaluation of Claude Zero-Baseline Iterative Test Round04

时间：2026-05-23 21:20 +08:00

## 结论

Round04 是真实 Claude Opus 4.7 Adaptive 网页输出。Claude 只收到 v13.4 框架和题面压缩信息，未收到隐藏答案钥匙。

v13.4 已经把 Round03 的边界自疑大幅压住：五题的 A 轴主入口、B 轴设问动作和主要责任链均命中，没有出现结构性错位。隐藏答案核验粗估约 43.5/50，已经进入稳定高分区间。

但本轮仍不能关闭为“完善框架”，因为 Claude 在总评中明确指出仍有三类会影响 95+ 稳定性的框架级缺口，且 Codex 隐藏键复核也认可这些缺口需要工具化：

- B1 表格题“整题总入口”是草稿定位还是正式答案必写，边界不够清楚。
- 未成年人打赏、治疗功效虚假宣传等高频边界的方向性术语还不够锁死。
- 各类责任方式缺少“最小命中术语清单”，零基础学生仍要凭感觉取舍加分术语。

因此 Round04 判定为 `high_score_not_final_perfect`，进入 v13.5 高分档锁分补丁。

## 逐题核验

- 题1 CC0200：A9 主入口、A1 副轴、B2 正确；写出限制民事行为能力、未追认、返还基础、各方过错和公平分担。缺点是平台“已尽到相应注意义务”写得略偏强，且“不予全额返还/酌定部分返还/小刘本人较大过错”方向性术语还可更硬。约 8.7/10。
- 题2 CC0157：A5 总入口、数据抓取链、A6/A9 旅游安全链、B1 表格形状正确；“题面未显示游客过错不主动减责”已部分吸收。缺点是仍自疑“相应赔偿”是否偏保守。约 8.7/10。
- 题3 CC0084：A2 总入口、三行局部入口正确；未成年人文身、公序良俗、法人商誉、格式条款返还均命中。缺点是“总入口是否正式写出”仍让学生自疑，且文身行可更明确写“行为不发生效力/退费赔偿”。约 8.8/10。
- 题4 CC0189：A4 合同、A5 混淆竞争、B1 正确；合同目的不能实现、返还服务费、违约金、有一定影响、足以混淆均命中。缺点是价值收束和补救措施清单还可更稳。约 8.9/10。
- 题5 CC0305：A2/A9 双链、B3 正确；隐私、个人信息、欺诈、退一赔三、五百元保底和食品药品边界句均命中。缺点是治疗功效/医疗健康功效是否进入特殊高倍赔偿仍造成自疑。约 8.8/10。

## v13.5 必要补丁

1. B1 总入口写作规则：总入口首先是审题定位；正式答案是否写出取决于题干是否要求总括说明。表格题每一行必须自洽，不能让总入口压过行内入口。
2. 未成年人打赏方向性术语：不猜具体比例，但要写“不支持全额返还/酌定部分返还/按各方过错合理分担”；平台采取限制措施只能减轻或影响责任，不当然等于无过错。
3. 治疗功效边界：治疗疾病、医疗功效虚假宣传固定先写一般消费者欺诈退一赔三；只有商品身份、安全标准不合格、健康损害或题干明示特殊责任时才写高倍食品药品赔偿。
4. 最小命中术语清单：给 B1/B3 常见行一个“3-6 个必须词”清单，降低零基础学生在加分术语上的自由猜测。
5. B3 加分术语层级：精神损害抚慰金、五百元保底、撤销/返还等要标清“何时必写、何时可写边界句”。
