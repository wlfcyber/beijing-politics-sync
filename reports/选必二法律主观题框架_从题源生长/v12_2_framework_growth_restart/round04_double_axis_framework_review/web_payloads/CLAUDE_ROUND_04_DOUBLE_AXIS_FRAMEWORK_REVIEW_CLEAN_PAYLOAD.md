### 2. 旧框架有效维度

1. 总入口抓得准：旧框架把《法律与生活》概括为“法条基础 + 法治意义”，并反复用高考/区模拟材料说明主观题不是单纯背法条。
2. “民事法律关系三要素”是可迁移骨架：主体、客体、内容能帮助学生先定位“谁与谁、围绕什么、有什么权利义务”。
3. 侵权部分的“权利 - 要件 - 归责 - 举证 - 责任方式 - 边界”相对成熟，尤其是过错推定/无过错责任、举证责任倒置、相邻关系和合理使用。
4. 旧框架有大量材料圈点意识：经常圈出主体、行为、损害、因果关系、权利、程序角色、意义词，适合发展为“材料触发链”。
5. 法治意义库有用户既有强项：经济、社会、权利、秩序、平衡、法治、公平正义、核心价值观、传统美德、德治法治统一等维度齐全。
6. 旧课件已经关注疑难杂症：肖像权/名誉权、个人信息原则、合同成立/生效/履行、合理使用/法定许可、辩护人/诉讼代理人、惩罚性赔偿边界等，适合转化为选择题错项库。

### 3. 旧框架主要问题

1. 层级混乱：前半用“法治意义/法条层次”，中段按合同、知识产权、侵权，后段按家庭、企业、多元纠纷。知识领域、答题策略、意义模板、题例批注被放在同一层，学生迁移到新题时不清楚先判断法律关系还是先套意义。
2. “企业”不是稳定法律分类：企业与劳动者、消费者、企业之间分别对应劳动争议、消费者权益、不正当竞争/知识产权，强行归为“企业”会遮蔽法律关系与请求权路径。
3. 主观题训练没有固定材料触发链：很多页有圈点，但没有统一成“材料事实 -> 法律关系 -> 法条要件 -> 责任/救济 -> 意义”的可复用步骤。
4. 法治意义容易变成口号库：旧框架的意义维度很多，但没有规定每个意义必须由哪一类材料事实触发，学生容易写“保护合法权益、维护公平正义、弘扬核心价值观”而缺少材料对应。
5. 选择题错项框架不足：旧课件积累了疑难点，但没有把错项类型系统化，如主体错、客体错、权利名称错、构成要件缺失、归责原则错、程序前置错、救济方式错。
6. 程序法被放在尾部，迁移性不足：诉讼代理人/辩护人、调解/仲裁/诉讼、司法确认、两审终审和上诉期限是跨题工具，不应只作为第六板块结束语。
7. 答案、细则、课堂批注边界不清：PDF 中有答案示例和“细则”样式页面，但当前证据不足以判定其为官方评分细则，不能直接升级为主框架的评分规则。
8. 学生操作性不足：旧框架更像教师板书和讲题轨迹，缺少学生看到新材料时的固定动作清单、判定顺序和输出格式。

## Merge candidates

以下为可直接并入主线程候选稿的新版框架。建议主线程以“候选稿 v0.1，待真题/评分细则验证”身份合并，不标记为最终完成版。

### A. 新版总纲：一核三轴五步

核心问题：

> 谁与谁，基于什么法律关系，围绕什么权利义务，请求什么责任或救济，为什么这种处理有法治意义？

三轴：

1. 法律关系轴：主体 -> 客体 -> 内容。
2. 请求/责任轴：权利 -> 义务 -> 构成要件 -> 归责/举证 -> 责任承担/救济路径。
3. 法治意义轴：个体权利 -> 利益平衡/秩序维护 -> 治理效能/司法公正 -> 德治法治/核心价值。

五步材料触发链：

1. 抓主体：自然人/法人/非法人组织；消费者/经营者；劳动者/用人单位；父母/子女；夫妻；权利人/侵权人；调解组织/仲裁机构/法院。
2. 定关系：合同、侵权、物权/相邻、人格权/个人信息、知识产权、婚姻家庭/继承、劳动、消费者权益、竞争秩序、纠纷解决。
3. 找要件：成立/生效/履行；侵权四要件；归责原则；举证责任；权利边界；程序前置或适用限制。
4. 给后果：是否支持请求；承担何种民事责任；是否适用调解/仲裁/诉讼；是否有司法确认/强制执行；是否可惩罚性赔偿或免责。
5. 升意义：只从材料事实出发写意义，避免空喊。每个意义点必须能回扣一个事实、一个法律规则或一个社会治理目标。

### B. 新版知识框架候选目录

0. 考情总入口
   - 本书主线：法条基础 + 法治意义。
   - 题型能力：选择题辨析、主观题法律关系分析、法治意义题、程序/救济路径题。

1. 民事法律关系总工具
   - 主体：自然人、法人、非法人组织；民事权利能力；民事行为能力。
   - 客体：物、行为、智力成果和商业标记、人身利益。
   - 内容：权利义务关系。
   - 学生动作：先写“谁对谁基于什么事实提出什么请求”。

2. 权利体系与边界
   - 人身权：人格权、身份权；姓名、肖像、名誉、荣誉、隐私、个人信息、生命健康身体。
   - 财产权：物权、债权、知识产权、股权。
   - 权利边界：合理批评/评论、公共利益报道和舆论监督、作品合理使用/法定许可、相邻关系。
   - 选择题高频错项：权利名称错、权利边界绝对化、把免费合理使用和有偿法定许可混同。

3. 合同与债
   - 合同成立：意思表示一致。
   - 合同生效：主体适格、意思表示真实、不违法、不违背公序良俗。
   - 合同履行：全面履行、诚信、协作。
   - 合同变更/解除：原则上协商一致，单方变更/解除可能违约。
   - 违约责任：继续履行、补救、赔偿损失、违约金、定金罚则；不可抗力免责。
   - 主观题输出：合同事实 -> 合同义务 -> 履行/违约判断 -> 责任方式。

4. 知识产权与创新秩序
   - 著作权：人身权/财产权及期限；署名权等永久保护；合理使用/法定许可。
   - 专利权：发明、实用新型、外观设计及期限。
   - 商标权：期限与续展。
   - 与不正当竞争的接口：虚假宣传、混淆、搭便车、引人误解、虚假交易。
   - 法治意义触发：保护创作和创新、激发市场活力、维护公平竞争和市场秩序。

5. 侵权责任
   - 被侵权权益：物权、人格权、身份权、知识产权等。
   - 一般侵权四要件：合法权益受损、行为人有过错、侵权行为、因果关系。
   - 归责原则：一般过错、过错推定、无过错。
   - 证据规则：环境污染等特殊领域可出现举证责任倒置。
   - 责任方式：财产责任与非财产责任分开写。
   - 学生动作：不要先背责任方式，先证明要件成立或不成立。

6. 家庭、继承与养老
   - 纵向：父母对子女的抚养、教育、监护；子女对父母的赡养、照料、精神慰藉和特殊需求照顾。
   - 横向：夫妻共同财产与一方财产。
   - 继承：析产、遗嘱、遗产范围内清偿债务、原则一般均等和协商处理。
   - 意义触发：维护老年人合法权益、家庭和谐、传统美德、德治法治结合、社会公平正义。

7. 劳动、消费与市场经营
   - 劳动者与用人单位：协商和解、调解、劳动仲裁、诉讼；劳动保障监察。
   - 消费者与经营者：安全消费权、知情权、自主选择权；食品安全、惩罚性赔偿边界。
   - 经营者之间：不正当竞争与知识产权交叉。
   - 法治意义触发：保护劳动者/消费者权益、规范经营、维护市场秩序、促进行业健康发展。

8. 多元纠纷解决与诉讼角色
   - 非诉机制：和解、人民调解、行政调解、仲裁调解、司法确认。
   - 仲裁：婚姻、收养、监护、扶养、继承纠纷和行政争议不适用；商事仲裁强调协议、自愿、灵活、保密、或裁或审。
   - 诉讼：民事/行政诉讼代理人，刑事诉讼辩护人；两审终审；民事判决 15 日/裁定 10 日，行政和刑事判决 10 日/裁定 5 日。
   - 基层治理：党建引领、调解、法治方式/法治思维、德治法治、多元共治、纠纷化解机制。

### C. 主观题通用作答模板

1. 法律关系题：
   - 材料事实：摘主体、行为、损害/争议、请求。
   - 法律定位：说明属于哪类法律关系。
   - 规则要件：列必要要件，不堆无关法条。
   - 材料对应：逐项对应材料。
   - 结论：是否支持请求、谁担责、承担什么责任。

2. 法治意义题：
   - 权利层：保护谁的何种合法权益。
   - 秩序层：维护何种交易、生产经营、家庭、社区、市场或社会秩序。
   - 治理层：促进司法公正、规则明确、纠纷化解、基层治理或行政/司法协同。
   - 价值层：公平正义、诚信、公序良俗、社会主义核心价值观、中华传统美德、德治法治结合。
   - 禁忌：没有材料事实支撑的意义不写；不能把所有题都套同一组口号。

3. 选择题错项分类：
   - 主体错：把辩护人写成诉讼代理人，或混淆消费者/经营者、劳动者/用人单位。
   - 权利错：肖像权、名誉权、隐私权、个人信息、著作权、商标权等混淆。
   - 要件错：缺少损害、过错、因果关系、意思表示一致、主体适格等。
   - 归责错：把过错责任、过错推定、无过错责任混用。
   - 程序错：仲裁适用范围、劳动争议路径、司法确认、两审终审、上诉期限混淆。
   - 救济错：违约责任与侵权责任方式混用，财产责任与非财产责任混用。
   - 意义错：意义脱离材料，或把道德倡导当成法律责任。

## Evidence C: Current v12.2 framework chapter

# 01 法律主观题框架章

## 一句话总框架

选必二主观题不是按教材页码出题，而是按设问动作出题。先看题目让你做什么，再判断命题人要学生完成哪一种法律工作：补链、判责、判诉求、评观点、写意义、走路径。

## 命题人出题路径

1. 先给一个生活化法律场景：侵权、合同、劳动、婚姻家庭、继承、知识产权、诉讼或调解。
2. 再把材料中的事实拆成若干法律触发点：主体、行为、权利、义务、过错、程序、证据、损害、效果。
3. 最后用设问动作规定答题方式：让学生补表、判责、判断诉求、评析观点、写作用或给路径。

学生拿到题后反过来走：设问动作定入口，材料触发定知识，评分锚点定落点。

## 六个入口

### E1 表格/裁判要点/补链题（9题）

- 识别口令：设问让学生完成表格、补裁判要点、补合同/权利/责任链。
- 命题路径：形式触发 -> 一格一链 -> 法律关系/法律要件 -> 格内结论。
- 作答骨架：先判断每一格问的是什么，再把材料事实翻译成一个法律关系，最后落到这一格的结论。多格题禁止写成一段总论。
- 学生预警：不要因为材料新，比如 AI、平台、数字经济，就新造主题模板；先看它是不是在补一个法律链条。
- 本版题源：CC0077_2025_东城_一模_19、CC0084_2025_东城_二模_19、CC0137_2025_昌平_二模_20、CC0157_2025_朝阳_期末_20、CC0213_2025_门头沟_一模_20、CC0214_2025_门头沟_一模_20_2、CC0289_2026_朝阳_二模_18、CC0325_2026_石景山_一模_18、CC0189_2025_石景山_一模_20

### E2 判决/裁判/责任理由题（8题）

- 识别口令：设问问法院为什么这样判、责任如何承担、裁判依据或法理依据。
- 命题路径：裁判对象 -> 法律关系 -> 要件/过错/程序/责任 -> 裁判结论与边界。
- 作答骨架：先定案由和法律关系，再逐个对上材料中的责任要件，最后说明法院支持、减轻、否定或限缩责任的理由。
- 学生预警：不要一上来背公平正义。判决题的分数在事实如何满足要件，价值只放在结尾。
- 本版题源：CC0002_2024_丰台_一模_17、CC0025_2024_朝阳_二模_17、CC0054_2024_石景山_一模_17、CC0119_2025_丰台_期末_19、CC0181_2025_海淀_期末_21、CC0200_2025_西城_二模_18、CC0206_2025_西城_期末_19、CC0364_2026_通州_期中_19_1

### E3 诉求/请求能否支持题（3题）

- 识别口令：设问直接问某人的诉求、请求、主张能否得到法院支持。
- 命题路径：诉求对象 -> 权利基础 -> 法律要件 -> 支持/不支持及理由。
- 作答骨架：先把不同诉求拆开，再分别写权利依据、事实匹配和法院是否支持。低样本入口，不能泛化成所有判案题。
- 学生预警：只有设问在问“能否支持/能否成立”时才优先进入 E3；普通判决理由题仍回 E2。
- 本版题源：CC0143_2025_朝阳_一模_19、CC0305_2026_海淀_一模_18_3、CC0373_2026_顺义_一模_18

### E4 评析/认识/观点题（7题）

- 识别口令：设问要求评价观点、谈认识、评析做法、说明某种法律变化的合理性。
- 命题路径：观点/做法 -> 法律关系和制度背景 -> 判断边界 -> 规范评价。
- 作答骨架：先判观点是否成立，再回到具体法律关系解释为什么成立或需要补充，最后给出规范性结论。
- 学生预警：不要把评价题写成空泛价值口号；也不要把低频法律变化题扩成新主干。
- 本版题源：CC0150_2025_朝阳_二模_20、CC0238_2026_东城_二模_19、CC0244_2026_东城_期中_18、CC0254_2026_丰台_二模_18、CC0051_2024_海淀_期中_21_1、CC0061_2024_西城_一模_18、CC0332_2026_石景山_二模_19

### E5 意义/价值/作用/保护推动题（11题）

- 识别口令：设问问规定、制度、做法的作用、意义、价值，或如何保护、推动某类权益。
- 命题路径：法律规则/制度 -> 保护对象或治理对象 -> 行为方式 -> 秩序与权益效果。
- 作答骨架：意义价值题写“保护谁、规范谁、维护什么秩序”；如何保护题写“法律手段、保护对象、实践效果”。
- 学生预警：E5 是最大桶，必须拆句型。尤其劳动法题要落在法律公平与效率，不要写成纯经济分析。
- 本版题源：CC0011_2024_丰台_二模_17、CC0019_2024_朝阳_一模_19、CC0063_2024_西城_二模_16、CC0103_2025_丰台_一模_19、CC0131_2025_房山_一模_19、CC0180_2025_海淀_期末_20、CC0195_2025_西城_一模_20、CC0283_2026_朝阳_一模_18、CC0340_2026_西城_一模_17、CC0045_2024_海淀_二模_19、CC0229_2026_东城_一模_18

### E6 调解/维权/纠纷解决/证据路径题（4题）

- 识别口令：设问问如何调解、怎样维权、纠纷如何解决、举证责任或救济路径。
- 命题路径：纠纷事实 -> 权利受损或争议焦点 -> 路径/主体/证据 -> 救济方案。
- 作答骨架：路径先行：先写采用什么法律途径或调解方案，再写依据、行动、证据和救济效果。
- 学生预警：不要把路径题写成抽象法治意义；任选其一类设问要写一条完整链，不是三条半截链。
- 本版题源：CC0125_2025_延庆_一模_19、CC0223_2025_顺义_一模_19、CC0245_2026_东城_期中_18_2、CC0092_2025_东城_期末_19_1

## 入口判断顺序

1. 看到表格、补空、裁判要点，先判 E1。
2. 看到法院为什么判、责任如何承担，先判 E2；如果设问直接问诉求能否支持，再判 E3。
3. 看到评析、认识、观点，判 E4。
4. 看到意义、作用、价值、保护推动，判 E5，并继续分清是意义句型还是如何保护句型。
5. 看到调解、维权、纠纷解决、举证、救济路径，判 E6。

## Round 03 边界补丁

- E1：AI 题只能停在著作权/合同/违约的格子链条，不能升级成 AI 创新主干。
- E1/E6：任选其一的权利保护题，E1 管前面的问答格，E6 管后面的 basis-action-remedy 完整路径。
- E2：`程序合法` 只在 CC0364 的材料与细则奖励中成立，不能当成所有裁判题的开头。
- E3：样本少，必须透明标注为低频入口，避免把所有判案题都塞进诉求支持。
- E4：CC0051 只证明低频评价覆盖，不证明需要新增法律变化主干。
- E5：11 题中要拆成意义价值句型与如何保护句型；CC0195 必须按劳动法公平/效率写，不写纯经济。
- E6：纠纷解决题必须先写路径，再写依据、行动、证据和救济；CC0223 不能写成泛泛法治价值。
