# v13.2 零基础工具箱补丁说明

本目录是在 v13.1 Round05/Round06 终版基础上生成的窄补丁，不覆盖 v13.1。

补丁来源：真实 Claude Opus 4.7 Adaptive 零基础盲测暴露的四个问题：

- A1 得分脊柱容易误升为主入口。
- A 轴入口后缺少要件句和责任句。
- B1 表格题缺少默认列结构和补链规范。
- 双链题主副轴排序不够明确。

本版目标不是重新发明框架，而是让零基础但聪明的学生拿到框架后能写出更接近评分锚点的答案。

验收标准：

- 新一轮真实 Claude 新聊天盲测，不使用旧 Claude 对话。
- Claude 只收到本补丁框架和题面压缩信息，不收到隐藏答案钥匙。
- 结构性 A/B 误判为 0。
- 5 题粗评目标约 42/50，单题不低于 7。
