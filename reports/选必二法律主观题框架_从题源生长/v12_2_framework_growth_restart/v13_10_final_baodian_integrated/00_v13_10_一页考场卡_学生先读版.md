# v13.10 一页考场卡：学生先读版

## A 入口判定

| 题面冲突 | 主入口 | 边界提醒 |
|---|---|---|
| 未成年人消费、打赏、文身、预扣款、虚假宣传 | A9 消费者 | A1 只补行为能力和追认 |
| 隐私、个人信息、名誉、商誉、肖像、生命健康 | A2 人格权 | 有消费背景也要独立写人格请求 |
| 共有部分、相邻、采光、通行、业主共同决定 | A3 物权相邻 | 审批、消防、法院通常只是支持事实 |
| 约定、交付、质量、迟延、退费、违约金 | A4 合同 | 商业用途购买通常先 A4 |
| 瑕疵商品/服务造成身体或财产损害 | A4 主 + A6 副 | 先违约，后损害赔偿 |
| 商业秘密、技术秘密、数据抓取、混淆、商业诋毁 | A5 竞争秩序 | 劳动离职只是背景 |
| 招聘、试用期、解除、工资、竞业限制、社保 | A8 劳动关系 | 不写成普通合同欺诈 |
| 证据、调解、仲裁、诉讼、维权路径 | A10/B6 | 只在问路径时主导 |

## B 动作模板

| 题干动作 | 写法 |
|---|---|
| 完成下表 | 一行一链：事实 -> 规则 -> 裁判要点 |
| 判决/责任理由 | 法律关系 -> 规则 -> 事实 -> 责任 |
| 诉求能否支持 | 支持/部分支持/不支持 + 理由 |
| 评析观点 | 合理处 -> 片面处 -> 正确立场 |
| 意义价值 | 保护对象 -> 约束行为 -> 秩序价值 |
| 维权路径 | 证据 -> 路径 -> 请求 |
| 问题识别/填空 | 短语或短句，不写长段 |

## 混合题排序

| 混合题 | 顺序 | 比例 |
|---|---|---|
| A2+A9 消费纠纷 | 先人格权请求，再消费者退赔请求 | 按诉求分段 |
| A4+A6+B6 瑕疵商品维权 | 先合同违约，再损害赔偿，最后证据路径 | 责任 70%，路径 30% |
| A3+B7 共有部分填空 | 先共有部分/相邻权，再程序/消防/安全边界 | 每空一句 |
| 表格 + 启示 | 先表格，再启示 | 启示只补一句 |

## 四个停止条件

1. 问责任，落责任；不强写宏大价值。
2. 问意义，扣材料对象和行为；不空喊法治。
3. 问路径，写证据、路径、请求；不只写依法维权。
4. 问问题识别，写短句；不扩成长判决。
