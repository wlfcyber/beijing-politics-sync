## 1. Verdict

v13.9 终于像一个框架了，不再只是“法律模块分类表”。它有三件真正有用的东西：

1. A轴抢入口判定句，让我先看真实生活冲突，而不是被“A1民事总论”“A10程序”骗走。
2. B轴把题目动作变成答案形状，尤其是 B1 表格、B3 诉求能否支持、B6 维权路径、B7 法律问题识别。
3. 混合设问排序规则能救命：先实体后程序、先主入口后副入口、先必得分后意义价值。

但我还是要骂：它还没到可以直接扔给全班学生的程度。主要问题不是覆盖不够，而是部分边界仍靠学生自己“法律直觉”补。比如 A2/A9 混合题、A4+A6+B6 维权题、B1 表格题没有表头时，框架给了方向，但不总能给出最稳的行列结构。高分学生能用，普通零基础学生会用到 60%-75%，但还会在“该写多长、分几段、哪个诉求先答”上浪费时间。

我的结论：v13.9 是真框架，但需要结构修补后才能学生交付。

## 3. One-Page Student Memory Map

我读完一次后能记住的版本如下：

先问三件事：

1. 谁和谁冲突，冲突的生活利益是什么？这决定 A轴。
2. 题目让我干什么？判责任、判诉求、填表、谈意义、说路径、识别问题？这决定 B轴。
3. 材料事实能压进哪些要件？每段都要“事实 -> 规则 -> 结论/责任”。

A轴记忆：

- A1：行为能力、意思表示、代理、时效、效力。只在题目直接问效力/追认/行为能力时当主入口。
- A2：人格权，人身、名誉、商誉、隐私、个人信息、肖像。
- A3：物权、相邻、共有部分、采光通风通行安全、业主共同决定。
- A4：合同，成立生效、履行、违约、退款、违约金、损失。
- A4+A6：瑕疵商品或服务造成身体财产损害，先合同违约，再补侵权/损害赔偿。
- A5：知识产权、商业秘密、技术秘密、商标、作品、竞争秩序、数据抓取。
- A6：侵权责任，人身财产损害、安全保障义务、工作人员执行任务、公共场所。
- A7：婚姻家庭继承、遗赠扶养、共同债务、赡养扶养。
- A8：劳动关系，录用、解除、试用期、竞业限制、社保、平等就业。
- A9：消费者和经营者，知情权、公平交易、安全保障、格式条款、未成年人消费、欺诈。
- A10：程序路径、证据、调解、仲裁、诉讼。只在题目问怎么维权、准备什么、走什么路径时当主入口或B6。

B轴记忆：

- B1：填表，按表头；没表头用事实/规则/责任。
- B2：判决理由，法律关系 -> 规则 -> 事实 -> 结论。
- B3：诉求能否支持，先给支持/不支持/部分支持，再逐项说理由。
- B4：评析观点，先承认合理处，再指出边界。
- B5：意义价值，保护什么、约束什么、形成什么秩序。
- B6：维权路径，争议焦点、权利、证据、协商/调解/仲裁/诉讼、请求。
- B7：法律问题识别/填空，短句，不写成长判决。

最重要的防死规则：

- 不要让 A1、A10 抢走主入口。
- 问责任就写到责任方式停。
- 问意义必须扣材料，不许空喊依法治国。
- 多入口题先主链，副链只补边界。

## 5. Where I Got Stuck

1. B1 表格题仍不够稳。框架说尊重表头，没表头用默认三列；但盲包常压缩掉表头，学生不知道答案应像“裁判结果/理由/意义”还是“事实/规则/责任”。
2. A2+A9 并列诉求题需要一个强制分段规则。现在我能判断隐私和欺诈要分开，但框架没有给“同一消费纠纷中人格权请求优先独立成段”的明确口令。
3. A4+A6+B6 混合题容易超写。框架说先实体后程序，但没有告诉学生各部分字数比例，比如责任70%、路径30%。
4. B7 法律问题识别需要短语库。现在我会写方向，但未必写成阅卷喜欢的短语。
5. 框架对“支持/部分支持/不支持”的结论尺度仍偏保守。它避免乱猜比例是对的，但学生想拿分还需要若干常见结论表达。

## 6. Framework Repairs Required

最高优先级修补：

1. 做一张“无表头 B1 默认表”：
   - 案例/事实
   - 法律问题或法律关系
   - 规则与裁判要点
   - 责任/意义
   并说明三列和四列什么时候选。

2. 给 A2+A9、A4+A6+B6、A3+B7 三类高频混合题加“分段顺序卡”：
   - A2+A9：先人格权请求，再消费者欺诈请求。
   - A4+A6+B6：先合同违约，再损害赔偿，再证据路径。
   - A3+B7：先共有部分/相邻权，再程序/消防安全边界。

3. 给 B7 做“短语答案库”，因为识别题不是让学生写作文：
   - 是否改变共有部分用途并履行业主共同决定程序。
   - 是否符合消防安全要求。
   - 是否影响通行、采光、相邻权益。
   - 谁应承担安全管理或侵权责任。

4. 给 B3 加结论尺度句：
   - 支持、部分支持、不支持分别在什么事实不足时使用。
   - 不要求具体比例时，统一写“结合过错、证据、因果关系合理分担”。

5. 把 v13.9 再压缩成考场卡。现在章节版能学，但不能在考试前快速翻。需要一页 A入口判定表 + 一页 B动作模板 + 一页混合题排序卡。
