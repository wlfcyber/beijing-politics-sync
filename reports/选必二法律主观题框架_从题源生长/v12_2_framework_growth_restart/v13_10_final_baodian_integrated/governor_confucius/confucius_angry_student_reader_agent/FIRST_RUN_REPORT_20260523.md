## One-Page Student Memory Map

我读完后脑子里能留下来的版本如下：

1. 先问：这是谁和谁，争什么利益？不要一上来背法条。
2. A 轴先定真实生活场景：人格/隐私名誉、物权相邻、合同、知识产权竞争、侵权伤害、婚姻继承、劳动、消费者、程序维权。
3. A1 是骨架，不是外衣。只有直接问行为能力、效力、代理、时效，A1 才能当主入口。
4. 再问 B 轴：题目让我填表、说明判决理由、判断诉求、评析观点、谈价值，还是给维权路径。
5. 每段都按“事实 -> 规则/要件 -> 结论/责任”写，不准空喊法治价值。
6. 表格题优先沿用原列名。看不清列名才用“材料事实 | 法律关系和要件 | 裁判要点/责任”。
7. 诉求题先写能不能支持，再写权利基础、构成要件、事实、责任。
8. 未成年人消费/打赏：A9 场景 + A1 行为能力脊柱，返还有基础，但通常要按平台、监护人、未成年人过错合理分担。
9. 隐私/名誉/商誉：先叫准权利名，再写停止侵害、删除/消除影响、道歉、赔偿。
10. 消费者欺诈：虚假宣传影响选择，写退还价款、退一赔三、不足五百按五百；食品药品高倍赔偿要有特殊信号。
11. 合同：成立生效、全面诚信履行、违约、返还/违约金/赔偿，别乱堆。
12. 竞争/商业秘密/数据：非公开、价值、保密措施、不正当获取使用、损害竞争秩序。
13. 侵权伤害：权利受损、过错/违法、因果关系、损害、责任；没有受害人过错别主动减责。
14. 价值题只在题目问价值、意义、启示时展开：保护什么，约束什么，维护什么秩序。

  - 处理方向：依法征求业主意见，符合消防和规划要求，兼顾方便生活与安全、公平合理处理相邻利益。
- 信心分：3.5/5。
- 框架帮到我的地方：A3 的共有部分用途、采光通行安全、公平合理原则很对题。
- 框架卡住我的地方：B 轴没有“法律问题识别填空”这一类，导致我不知道答案应是两个短语、两句话，还是一段分析。

## Where I Got Stuck

1. **框架太长，我需要先把框架变成框架。** 真实考场复习时，我不会从 v13.2 翻到 v13.7 找补丁来源。学生版应该先给最终作战图，再把补丁作为附录。
2. **A8 劳动模块明显偏薄。** 简历不实、试用期解除、劳动者诚信告知、用人单位合法解除这些高频劳动争议，框架没有足够工具句。
3. **产品质量/瑕疵交付致人损害缺桥。** 无人机题我能用 A4+A6 拼，但零基础学生会问：这是合同违约、产品责任、侵权，还是消费者？框架要给一句桥接规则。
4. **B 轴缺“法律问题识别/填空”动作。** 第 8 题不是完整判决、表格、诉求或价值题，现有 B1 只能勉强套。
5. **总入口规则仍会制造心理负担。** 多案例表格题真正写答案时每行入口最重要。总入口应该明确只用于草稿，不应成为新扣分焦虑。
6. **主副链比例有帮助，但还缺“最低字数形状”。** 6:4、7:3 是好规则，可学生还想知道 4 分钟内每链写几句。

## Framework Repairs Required

1. **把正文重排成“最终一页作战图 + A轴工具箱 + B轴模板 + 边界补丁附录”。** 不要让学生先读版本史。
2. **新增 B7：法律问题识别/填空题。** 模板：从材料冲突中抽两个“权利/义务/程序/责任”问题，表达为短语或短句，不展开成长篇。
3. **补 A8 劳动争议工具句。** 至少加入：如实说明与录用直接相关情况、诚实信用、试用期录用条件、解除应有事实和制度依据、违法解除边界。
4. **补 A4+A6 产品/瑕疵商品致损桥。** 规则：先看买卖合同是否成立和交付是否符合约定；瑕疵导致人身或财产损害时，再补侵权/产品质量损害赔偿；商业可得利益要看因果关系和可预见性。
5. **每个 A 入口给一个“最小 3 句版”。** 零基础学生先背最小版，再看高分版。否则工具句太多，临场会卡。
6. **多案例表格题写死一句防焦虑规则。** “正式答案优先行内自洽；总入口只用于草稿，拿不准不写。” 这句应该放在总览，不要埋在 v13.5。
7. **每个 B 轴给一个 30 秒模板。** 例如 B2：关系一句、规则一句、事实两句、责任一句、边界一句。学生需要的是可抄结构。
