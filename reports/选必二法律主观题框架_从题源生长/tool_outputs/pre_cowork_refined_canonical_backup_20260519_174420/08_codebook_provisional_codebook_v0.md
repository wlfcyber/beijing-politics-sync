- 满分句型: 材料中的[生活事实]在法律上表现为[法律定性/权利义务/责任条件]，因此[主体]应承担/享有[具体法律后果]。
- 必写词: 材料事实；法律语言；定性；权利义务；责任条件
- 误用风险: 空泛价值=high；法考化=low；模块边界=中：只转译不回答设问仍不完整；必须服务具体设问。
- 保留理由: 双模型同题同细则发现抄材料/堆教材限分。
