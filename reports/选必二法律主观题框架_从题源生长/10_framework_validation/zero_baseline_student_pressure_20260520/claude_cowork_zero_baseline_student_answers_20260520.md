## T1 CC0077_2025_东城_一模_19（表格 ①②③）

### 1. 框架入口

学生版第 1 入口"表格题：一格一答，先判断格子要什么"。句式库 **FWV1_2_N01「表格一格一答」**：先判格子功能，再填主体+行为+规则+意义。
- ①属于"解决机制"格；②③属于"典型意义"格。各自独立作答，不要把意义抄进机制格。

### 2. 材料翻译

- 案件一：法官"了解需求、释法析理"，公司"主动赔偿"——既不是判决也不是仲裁，事实上是法院组织下的**调解**。提示词"既解法结又解心结" = 调解的典型话术。
- 案件二：仲裁认定合同和协议有效、违约金支付是真实意思表示，柳某要返还违反**诚信原则**——翻译成：意思自治受保护 + 诚信履约义务 + 用人单位合法权益（培训投入）受保护。
- 案件三：三兄弟无法定赡养义务但构成事实扶养，法院支持指定遗产管理人——翻译成：依法保障事实扶养人合法财产权益 + 弘扬尊老敬老/互助友爱社会主义核心价值观 + 完善遗产管理人制度运行。

### 3. 最小判断

逐格判断"这格要的是机制还是意义"，再选词填空，不要串答案。

### 4. 考场答案

①（案件一·解决机制）**调解**。

②（案件二·典型意义）依法确认劳动合同与培训协议的效力，认定柳某支付违约金是真实意思表示，驳回其返还请求，**保护用人单位合法的培训投入权益，维护诚实信用原则，引导劳动者诚信履约**，促进劳动关系稳定有序。

③（案件三·典型意义）依法认定三兄弟构成事实上的扶养关系，支持其作为利害关系人申请指定遗产管理人，**保障扶养人合法财产权益**，**弘扬尊老敬老、互助友爱的社会主义核心价值观**，引导形成良好社会风尚。

### 5. 满分自评

**high**。这是核心节点 N01 直接覆盖的表格题。三格分别按"机制 / 意义 / 意义"作答，意义层我同时给了"权益—秩序—价值"三层。
- 潜在风险：①如果细则要求写"诉讼中调解"或"调解书"等更精准说法，我只写"调解"可能被扣半分。

## T2 CC0119_2025_丰台_期末_19（裁判理由）

### 1. 框架入口

学生版第 4 入口"裁判题：先写以事实为根据、以法律为准绳，再落具体法律规则"。句式库 **FWV1_2_N02「裁判锚句起手」** + **N04「私法案例四链」**：定性→规则→材料事实→责任/结果。

### 2. 材料翻译

- 尹某"违反公司管理规章制度""消极怠工""经常无故旷工"——翻译为**严重违反用人单位规章制度**，违反劳动者"完成劳动任务、遵守劳动纪律"的法定义务。
- 购物中心"基于经营需要+不能胜任"调岗 + 多次提醒后才解除——翻译为**用人单位行使用工自主权，程序正当**。
- 法院驳回赔偿金请求——翻译为**解除合法，无需支付违约赔偿金**。

### 3. 最小判断

第一步判断：购物中心解除劳动合同是否合法 → 合法 → 故不需赔偿。

### 4. 考场答案

人民法院以事实为根据、以法律为准绳。

根据《劳动法》第三条，劳动者享有平等就业、取得劳动报酬等权利，同时**应当完成劳动任务、遵守劳动纪律和职业道德**。尹某在被合理调整岗位后消极怠工、经常无故旷工，**违反了劳动者的法定义务**，并构成对购物中心管理规章制度的严重违反。

根据《劳动合同法》第三十九条，劳动者严重违反用人单位规章制度的，**用人单位可以解除劳动合同**。购物中心基于经营需要调岗合理，并在多次提醒无效后才解除合同，行为符合法律规定，**不构成违法解除，不需要支付赔偿金**。

故法院依据上述法律规定，判决驳回尹某要求支付违约赔偿金的诉讼请求，**依法保障用人单位合法用工管理权**，**引导劳动者依法履行劳动义务**，维护劳动关系的稳定与和谐。

### 5. 满分自评

**high**。裁判锚句 + 双法条 + 双向引述（权利义务都点到） + 材料事实嵌入 + 结论复现 + 一层秩序意义。
- 潜在风险：如果细则强调"用人单位**单方**解除权"或"过失性辞退"这种术语，我没有点到。但学生版禁止"法考式大段构成要件"，我克制是对的。

## T3 CC0244_2026_东城_期末_18（无人机·法律责任及依据）

### 1. 框架入口

学生版第 3 入口"案例理由题：定性 → 规则 → 材料事实 → 责任/结果"。句式库 **FWV1_2_N04「私法案例四链」**。
- 同时学生版材料翻译提示：「付款、出票、承诺」→ 合同成立；「损害、过错、因果」→ 侵权。本题两条线都有，存在**违约责任与侵权责任竞合**。

### 2. 材料翻译

- 陈某邮件"急需购买A型号无人机一台，用于重要商业拍摄"= **要约**；刘某回复"全新原装，15000元"= **承诺**；陈某"立即转账"= 履行 → **买卖合同依法成立、生效**。
- 刘某交付的是"内部结构轻微损伤"的"展示机"，不符合"全新原装"约定 → **违约（标的物质量不符）**。
- 无人机因该损伤"失控坠毁，砸伤陈某手臂" → **产品存在缺陷，因缺陷造成人身和财产损害**，构成侵权。
- 损失：医疗费 8000 元（人身损害） + 商业拍摄损失 5000 元（财产损失）。

### 3. 最小判断

第一步：刘某行为同时构成违约和侵权 → 陈某可择一主张请求权（竞合）。

### 4. 考场答案

**（一）违约责任**

陈某通过邮件发出要约、刘某承诺并由陈某付款，**双方买卖合同依法成立并生效**，合同当事人应按约定全面履行。刘某交付的展示机内部存在损伤，不符合"全新原装"的约定，**构成违约**。

根据《民法典》合同编关于违约责任的规定，刘某应承担**继续履行、采取补救措施或赔偿损失**等违约责任，赔偿范围可包括陈某因违约所受的损失（包括因不能完成商业拍摄而损失的 5000 元收入）。

**（二）侵权责任**

刘某交付的无人机存在缺陷，且该缺陷直接导致无人机失控坠毁、砸伤陈某手臂。根据《民法典》侵权责任编关于因产品缺陷造成他人损害应当承担侵权责任的规定，**刘某应当承担侵权责任**，赔偿陈某医疗费 8000 元等人身损害造成的合理损失。

**（三）责任竞合**

由于刘某的同一行为同时构成违约和侵权，根据《民法典》关于责任竞合的规定，**陈某有权选择**按违约责任或侵权责任主张赔偿，以更有利地维护自身合法权益。

### 5. 满分自评

**medium-high，主动刹车说明：**
- 我吃不准选必二是不是要求把"违约/侵权/竞合"三段都写全，还是只要求二选一展开。教师指南提醒"债权与物权不要混同""不要过度法考化"，我担心"竞合"那一段如果不是高中评分点，会被视为画蛇添足。
- 同时我没有点出"惩罚性赔偿"——本题材料没有写"明知"故意销售缺陷产品，刘某是"误寄"，因此**不主张惩罚性赔偿**是合理判断；但如果细则有特别要求，我可能漏分。
- 学生处理建议：考场上优先把"违约 + 侵权 + 各自损害对应赔偿项"写清楚，竞合一句话带过即可，不要展开请求权方法论。

## T4 CC0143_2025_朝阳_一模_19（机票搭售·法院支持理由）

### 1. 框架入口

学生版第 4 入口"裁判题"（设问是"法院支持……的理由"）+ 第 3 入口"案例理由题"。
- 句式库 **N02「裁判锚句起手」** + **N04「私法案例四链」**。
- 学生版材料翻译提示："搭售 → 违背真实意思、消费者权益受侵害"。

### 2. 材料翻译

- A公司"通过技术手段擅自搭售 10 元外卖红包，王某不能清楚知悉、也无法拒绝" → 翻译为**侵犯消费者知情权、自主选择权、公平交易权**，**构成消费欺诈**。
- 实际机票 300 元、却显示 350 元优惠 40 元收 310 元 → 隐瞒真实价格 → 强化欺诈认定。
- 法条直接给出：《消法》第五十五条欺诈三倍赔偿，不足 500 元按 500 元。
- 王某诉请：退还 310 元 + 三倍赔偿 → 法院**全部支持**。

### 3. 最小判断

第一步：A 公司行为是否构成"经营者提供商品或服务有欺诈行为"——构成 → 触发《消法》55 条三倍赔偿。

### 4. 考场答案

人民法院以事实为根据、以法律为准绳。

A 公司在向王某销售机票过程中，通过技术手段**擅自搭售**10 元外卖红包，使王某**无法清楚知悉**费用支出细节，**也无法拒绝**支付额外费用，**侵犯了消费者的知情权、自主选择权和公平交易权，构成消费欺诈**。

王某基于受欺诈支付的 310 元，依法应当予以退还，故法院支持其退还票款 310 元的诉讼请求。

根据《消费者权益保护法》第五十五条，经营者提供商品或者服务有欺诈行为的，应当按照消费者的要求增加赔偿其受到的损失，**增加赔偿的金额为消费者购买商品的价款或者接受服务的费用的三倍；增加赔偿的金额不足五百元的，为五百元**。本案中王某支付价款 310 元，三倍为 930 元，超过 500 元下限，故法院依法支持王某的三倍赔偿请求。

法院的判决，**依法保护了消费者的合法权益**，**惩戒了经营者的欺诈行为，规范在线文旅消费市场秩序**，引导经营者诚信经营，**营造公平有序、安全放心的消费法治环境**。

### 5. 满分自评

**high**。本题路径很清晰：欺诈定性 → 三项消费者权利 → 退款依据 → 三倍赔偿条文嵌入 → 三层意义（个人权益—市场秩序—法治环境）。
- 潜在风险：意义层我写了三句，若评分细则只给两句意义、第三句被视为重复，会浪费时间但不扣分。
- 我没敢写"惩罚性赔偿"四个字（怕踩"法考化"红线），其实《消法》55 条本身就是惩罚性赔偿，写也无妨——这里是**学生提示不足**导致的犹豫，详见总评。

## T6 CC0276_2026_房山_二模_17（中国如何加强涉外法治建设）

### 1. 框架入口

**先问一句**（学生版第一句）：本题主语是"中国"，处理的是**涉外法治建设/依法治国战略**，**不是私人法律关系、权利义务、责任承担、维权程序**。

→ 触发学生版"先别套选必二"。
→ 句式库 **FWV1_2_GATE「边界先判」**：本工程不把此类题生成选必二核心满分句；教师指南也明确写：CC0276_2026_房山_二模_17 是 **formal，但不进选必二核心**。

**必须明确刹车：本题不能用选必二核心框架满分闭合。**

### 2. 材料翻译

不做"私法翻译"，而是按"中国—怎么做—成效"归纳：
- "出台外商投资法及其实施条例""保护中外当事人合法权益" → **完善涉外法律制度**。
- "调解—仲裁—诉讼一站式线上解纷平台" → **创新涉外纠纷解决方式、提升效率**。
- "活扣押""引入外国船东保赔协会调解" → **创新司法工作机制、积累涉外法治经验**。

### 3. 最小判断

第一步：识别这是必修三/法治建设方向题 → **不套选必二私法四链 / 维权三层 / 意义收束模板**。
- 按材料三段（保护/速度/经验）归纳"中国如何加强涉外法治建设"。

### 4. 考场答案

（按材料归纳，不套选必二核心模板）

**①完善涉外法律制度，加强对中外当事人合法权益的平等保护。** 在制度层面出台《外商投资法》及其实施条例，加强对内外资企业的平等保护；在司法实践层面注重保护中外当事人合法权益，**为高水平对外开放提供制度保障**。

**②创新涉外纠纷化解机制，提升涉外法治的便捷性和效率。** 建设"调解—仲裁—诉讼"相衔接的国际商事一站式线上解纷平台，允许当事人全天候网上立案、在线调解、远程庭审、在线接收材料，**让涉外当事人获得高效便捷的司法服务**。

**③探索符合中国国情、与国际接轨的涉外法治经验。** 海事法院在船舶扣押中创造性实施"活扣押"，允许完成剩余航程、促成当事人和解；引入外国船东保赔协会参与调解，**形成人民法院引导、行业组织共同参与的多元化涉外纠纷化解模式**。

**④通过以上举措，中国在涉外法治建设中坚持依法保护、提高效率、创新机制相结合，提升中国法治的国际公信力**，为以负责任大国身份参与国际事务提供有力法治保障，赢得世界尊重。

### 5. 满分自评

**low/medium，主动刹车（最重要的一题）：**
- 本题被教师指南明确划入 **boundary_non_core / boundary_gate**：不进选必二核心框架。
- 选必二核心节点（私法四链、维权三层、裁判锚句、意义三层）**全部不适用**，强行套会偏题（必修三化）。
- 处理方式：用"先问一句"识别为涉外法治建设题，**改走必修三/总论性话语**，从材料三段（保护制度 / 速度效率 / 经验机制）逐段归纳。
- 提醒：学生考场如果不熟悉"涉外法治建设"专题语，最稳的做法是**紧贴材料，把材料三段动作译成三段总结性论述**，避免空套"全面依法治国"。

# 总评

## (1) 哪些题学完框架后能明显接近满分？

- **T1（表格）**：N01 节点路径太清晰，"机制 / 意义"格子分工 + "权益—秩序—价值"三层意义已经够稳。
- **T2（裁判理由 · 劳动法）**：N02 + N04 双节点，加上题面直接给法条，套路化程度高，学生只要克制、不写法考构成要件就能拿高分。
- **T4（消费欺诈 · 三倍赔偿）**：N04 私法四链 + N02 锚句的最佳示范题，三项消费者权利 + 法条直引 + 三层意义都在学生能力范围。

## (2) 哪些题仍然会写偏？

- **T3（无人机·违约/侵权竞合）**：框架告诉了"四链"，但**没告诉学生"高中阶段是否要写竞合""产品缺陷怎么定性""惩罚性赔偿什么时候不写"**。我作为零基础学生只能"凭感觉克制"，但很可能写多写少都丢点分。
- **T5（民法典对生态环境的作用）**：教师指南只说"开放容器，不当稳定模板"，**没告诉学生考场上替代写法是什么**。学生很容易要么写成必修三的"美丽中国/生态文明"口号，要么过度展开"民事公益诉讼"专业术语。
- **T6（涉外法治建设）**：学生版第一句"先问一句"救了我，但**学生版没有给"必修三/总论题"的替代落点**。一个零基础学生即使识别出"不套选必二"，也可能不知道该怎么写——只能死贴材料。

## (3) 框架最缺的学生提示是什么？

按重要性排序，最缺三条：

1. **「请求权竞合」与「惩罚性赔偿」该不该写、写到什么程度。** 学生版只写"看到欺诈、隐瞒、搭售：翻译成违背真实意思或消费者权益受侵害"，但没说"欺诈题要把《消法》55 条三倍赔偿写到位、并且不展开法考构成要件"。学生在 T3、T4 两题都会犹豫。
2. **开放容器题的「考场替代写法」。** 教师指南说"作用题/AI题/综合治理题"是开放容器、不能套核心模板，但学生不知道**那考场上要怎么写才能不丢分**。建议给学生一条"开放容器题三段法：对应法条 → 对应案例 → 具体作用动词；不喊必修三口号"。
3. **「边界题（涉外法治、依法行政、国家治理）」的识别更明确化。** 学生版"先问一句"已经在做这件事，但学生在压力下仍会被"法治""法律""权利"这些词诱导套核心模板。需要一句更硬的话直接列出**典型边界关键词**。

## (4) 如果只改一页学生版，最该加哪三句话？

放在"先问一句"和"六个入口"之间：

1. **"主语是国家、是中国、是依法行政、是涉外法治建设、是新时代法治体系——直接停手，本框架不适用，改按必修三/总论题答；考场上紧贴材料三段动作归纳即可。"**（直接救 T6 类边界题，并给可执行替代）
2. **"看到欺诈/搭售/隐瞒：先定性侵犯知情权、自主选择权、公平交易权，再写《消法》55 条三倍赔偿、不足五百按五百，不展开'惩罚性赔偿'法考论证。"**（直接救 T3、T4 这类消费欺诈题，把"惩罚性赔偿"是否写、写到哪一层从灰色变成清晰）
3. **"作用题、意义题、典型意义题：法条对法条、案例对案例、动词写具体——禁止只写'维护公平正义、促进社会和谐'。"**（直接救 T1③、T5 这类意义题，逼学生贴材料而不是套必修三口号）
