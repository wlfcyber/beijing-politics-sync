# 上篇：选必二法律主观题强分诊框架

## 1. 先导

选必二法律主观题，不是背法条。

它考的是：
- 把生活纠纷看成法律关系。
- 把材料事实翻译成权利、义务、责任、程序。
- 把法律判断写成阅卷喜欢的短句。

飞哥想说：
- 看到法院，不等于写公正司法。
- 看到企业，不等于写经济生活。
- 看到平台，不等于写劳动关系。
- 看到合同，不等于所有题都走合同。
- 看到意义，不等于裸写全面依法治国。

考场第一步不是背知识。

先分诊。

## 2. 先按设问分诊

### 问诉求能不能支持 / 法院为什么支持
看到：
- 问诉求能不能支持 / 法院为什么支持
这样写：
- 先表态。
- 再写法律依据。
- 再贴材料事实。
- 最后落到支持、不支持、减轻、承担。
别这样写：
- 别一见法院就写必修三；别一见协议就写合同。

### 问裁判理由 / 补表 / 完成理由
看到：
- 问裁判理由 / 补表 / 完成理由
这样写：
- 一格一链。
- 处理结果倒推法律理由。
- 一格只写一个法律链条。
别这样写：
- 别把几个小问串成一个材料；别把所有格都塞进合同。

### 问评析 / 谈认识
看到：
- 问评析 / 谈认识
这样写：
- 先问：评谁？评什么行为？
- 一主体一链。
- 行为定性 -> 法律后果 -> 应然要求。
别这样写：
- 别写成“该行为不诚信”这种空话；多个主体不要一锅炖。

### 问意义 / 价值 / 作用
看到：
- 问意义 / 价值 / 作用
这样写：
- 先规则。
- 再材料。
- 最后价值。
别这样写：
- 别裸写“有利于全面依法治国”；价值必须从法律规则推出。

### 问调解 / 维权 / 如何解决
看到：
- 问调解 / 维权 / 如何解决
这样写：
- 先给方案。
- 再给法律理由。
- 再贴材料依据。
别这样写：
- 别把调解题写成判决书；也别把所有法院题都写成纠纷解决。

### 问法律责任及依据
看到：
- 问法律责任及依据
这样写：
- 先定关系。
- 再定行为。
- 最后定责任。
别这样写：
- 别只写“违法要担责”；责任方式要落地。

### 问规定/制度如何发挥作用
看到：
- 问规定/制度如何发挥作用
这样写：
- 制度不是背目录。
- 要写它怎样约束行为、保护权利、推动秩序。
别这样写：
- 别写成政治制度套话。

## 3. 再按材料分诊

### 好意同乘 / 交通损害
看到：
- 无偿搭乘 + 交通事故 + 驾驶人无故意或重大过失。
标准写法：
- 好意同乘减责 + 侵权责任边界 + 公平。
不能乱用：
- 合同成立、人格权、消费者、平台用工。

### 民法绿色原则 / 生态环境
看到：
- 绿色原则、排污损害、停止侵害、消除危险、赔偿损失、恢复原状。
标准写法：
- 绿色原则约束民事活动；侵权责任修复环境公共利益。
不能乱用：
- 没有具体作品/商标/专利侵权，不要写成知识产权侵权题。

### 饲养动物损害 + 融情调解
看到：
- 动物造成损害 + 饲养人责任 + 法院调解方案。
标准写法：
- 先写饲养动物损害责任，再写调解如何定分止争、兼顾情理。
不能乱用：
- 不要写成经济发展题，不要串页。

### 平台用工
看到：
- 合作协议 + 派单 + 签到 + 奖惩 + 结算 + 业务组成。
标准写法：
- 不看协议名称，看三从属性；认定事实劳动关系。
不能乱用：
- 只有平台二字不够；机票代销平台不是平台用工。

### 合同成立 / 履行 / 违约
看到：
- 报价具体明确、中标通知、承诺到达、拒不履行、格式条款效力。
标准写法：
- 要约 -> 承诺 -> 成立/效力 -> 履行/违约责任。
不能乱用：
- 不要把平台合作协议、好意同乘、遗赠扶养全写成普通合同。

### 消费者欺诈 / 惩罚性赔偿
看到：
- 虚假宣传、隐瞒、隐蔽搭售、无法清楚知悉、无法有效拒绝。
标准写法：
- 知情权/自主选择权/公平交易权 -> 欺诈 -> 撤销/退赔/惩罚性赔偿。
不能乱用：
- 三倍赔偿不能乱套，必须有消费者 + 经营者欺诈。

### 食品安全 / 知假买假边界
看到：
- 不合格食品 + 首单维权 + 后续扩大购买。
标准写法：
- 首单保护；后续牟利性扩大购买要看权利边界。
不能乱用：
- 不要把消费者保护写成赔偿一路到底。

### 知识产权 / 技术秘密 / 不正当竞争
看到：
- 作品、图片、商标、专利、技术秘密、未经许可、混淆、抢注、商业诋毁。
标准写法：
- 先判权利对象，再判侵害方式，最后写责任和创新价值。
不能乱用：
- 只有创新二字，不够写知识产权。

### 人格权 / 隐私 / 名誉 / 人身权
看到：
- 摄像头、聊天记录公开、肖像名誉、身体健康、高空抛物坠物。
标准写法：
- 权利名 -> 侵害行为 -> 责任方式/保护路径。
不能乱用：
- 别泛写人格尊严，先叫准权利名。

### 相邻关系 / 物权边界
看到：
- 通行、通风、采光、加装电梯、业主规约、恢复原状。
标准写法：
- 有利生产、方便生活、团结互助、公平合理 -> 具体妨碍 -> 停止/恢复/协商。
不能乱用：
- 别写成社区治理或邻里道德。

### 家庭继承 / 养老赡养
看到：
- 遗赠扶养协议、子女赡养、老人照料、督促履行。
标准写法：
- 协议效力/法定义务 -> 材料事实 -> 老年权益保护。
不能乱用：
- 别只写传统美德。

### 劳动竞业 / 平等就业
看到：
- 竞业限制、商业秘密、离职入职竞争企业、招聘限性别。
标准写法：
- 保护企业商业秘密，也保护劳动者平等就业和自主择业。
不能乱用：
- 竞业题不是单向保护企业。

### 未成年人校园欺凌
看到：
- 校园欺凌、未成年人违法、惩戒、教育矫治、监护。
标准写法：
- 法律责任 + 教育矫治 + 未成年人保护。
不能乱用：
- 不要只严惩，也不要只教育。

## 4. 禁止误判总表

- 有法院：不一定写公正司法。先筛：是不是要写具体私法规则？
- 有企业：不一定写经济生活。先筛：是在问经营措施，还是法律责任？
- 有平台：不一定写劳动关系。先筛：有没有派单、签到、奖惩、结算、业务组成？
- 有合同/协议：不一定只写合同。先筛：真实关系是不是劳动、扶养、好意同乘？
- 有消费者：不一定写三倍赔偿。先筛：有没有经营者欺诈？
- 有创新：不一定只写保护创新。先筛：有没有权利对象和侵害方式？
- 有意义：不一定写全面依法治国。先筛：价值能不能从具体规则推出？

## 5. 最短记忆版

1. 先看设问交什么，不先背法条。
2. 再看主体、行为、冲突、结果。
3. 法院题先写私法规则，不裸写公正司法。
4. 意义题先规则后价值。
5. 平台题先查三从属性。
6. 合同题先判成立/效力/履行。
7. 消费者题先判权利和欺诈。
8. 知产题先判权利对象和侵害方式。
9. 人格权题先叫准权利名。
10. 多主体题一主体一链。
11. 调解题给方案加理由，不要直接判案。
12. 没有材料事实，不写法律词。

飞哥最后一句：

法律主观题不是比谁会背，是比谁能把材料翻译成法律得分话。
