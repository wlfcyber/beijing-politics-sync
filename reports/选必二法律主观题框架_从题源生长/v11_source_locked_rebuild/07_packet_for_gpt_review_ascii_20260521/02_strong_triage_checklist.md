# 02 强分诊框架清单

本清单不是 v10 那种“先搭框架再自动塞题”。它只回答一个问题：看到某类设问或材料时，什么条件下才准命中，什么情况下必须禁止。

使用口径：核心支撑只取 01 回源审判表中“保留 / 重写”的题；OPEN_OR_REFERENCE 只能作参考；待用户确认题不得支撑核心分诊。

## A. 设问分诊

### A1 诉求能否支持/法院为什么支持
- 看到什么：设问问“诉求能否支持”“法院为什么支持/酌情减轻/判决结果”
- 必须同时满足：必须能锁定一个具体诉求或处理结果，并能找到对应材料事实与法律依据
- 正确答法：先表态；再用法律规则对材料事实作证明；最后落到支持/不支持/责任边界
- 不能进入/禁止误判：不能一见法院就写公正司法；不能一见协议就写合同成立；不能把意义价值当主入口
- 正式支撑题：CC0002_2024_丰台_一模_17;CC0143_2025_朝阳_一模_19
- 恢复支撑题：CC0305_2026_海淀_一模_18_3;CC0373_2026_顺义_一模_18
- 边界提醒：同一题有多个诉求时，一诉求一链，不要合成万能结论

### A2 裁判理由/补表/完成理由
- 看到什么：设问要求“完成裁判理由”“说明裁决理由”“完成表格”
- 必须同时满足：必须有处理结果或表格空格可倒推法律理由
- 正确答法：一格一链：处理结果 -> 法律规则 -> 材料事实 -> 责任/后果
- 不能进入/禁止误判：不能把每格都塞进同一个合同或纠纷解决框架；不能把细则文字当材料
- 正式支撑题：CC0025_2024_朝阳_二模_17;CC0054_2024_石景山_一模_17;CC0137_2025_昌平_二模_20;CC0238_2026_东城_二模_19
- 边界提醒：补表题最怕跨小问串材料，先分格再写

### A3 评析/谈认识
- 看到什么：设问问“评析某主体行为”“谈谈对判决/做法/惩教并行的认识”
- 必须同时满足：必须先确定被评析对象是谁、做了什么、侵害或履行了什么
- 正确答法：一主体一链：行为定性 -> 法律后果 -> 应然要求/价值收束
- 不能进入/禁止误判：不能写成道德评价；不能把多个主体混成一句“不诚信”
- 正式支撑题：CC0238_2026_东城_二模_19;CC0244_2026_东城_期中_18
- 恢复支撑题：CC0332_2026_石景山_二模_19
- 开放参考题：CC0040_2024_海淀_一模_19
- 边界提醒：评析题不是喊口号，是给行为定法律性质

### A4 意义/价值/作用
- 看到什么：设问问“意义”“社会价值”“法治价值”“如何推动绿色发展/保护创新”
- 必须同时满足：必须先有具体法律规则或判决路径，价值必须从规则和材料推出
- 正确答法：规则 -> 材料事实 -> 保护谁/规范谁/救济谁 -> 价值
- 不能进入/禁止误判：不能裸写全面依法治国；不能只写“有利于创新/发展/公平”
- 正式支撑题：CC0011_2024_丰台_二模_17;CC0063_2024_西城_二模_16;CC0103_2025_丰台_一模_19;CC0131_2025_房山_一模_19;CC0340_2026_西城_一模_17
- 恢复支撑题：CC0045_2024_海淀_二模_19
- 待回源题：CC0276_2026_房山_二模_17
- 边界提醒：意义题不是最后加一句价值，是前面规则自然推出价值

### A5 调解/维权/解决纠纷
- 看到什么：设问问“如何调解”“如何依法保护权利”“如何解决/治理风险”
- 必须同时满足：必须出现具体权利或纠纷路径，且材料给出可操作事实
- 正确答法：方案/路径 -> 法律理由 -> 材料依据；需要时写调解而非直接判案
- 不能进入/禁止误判：不能把所有法院题都写成纠纷解决；不能直接判决化
- 正式支撑题：CC0125_2025_延庆_一模_19;CC0254_2026_丰台_二模_18;CC0289_2026_朝阳_二模_18
- 开放参考题：CC0380_2026_顺义_二模_18_2
- 待回源题：CC0283_2026_朝阳_一模_18
- 边界提醒：有“调解”才优先写调解作用；没有路径事实不要硬写程序

### A6 法律责任及法律依据
- 看到什么：设问问“涉及的法律责任及法律依据”“法院判决理由/裁判依据”
- 必须同时满足：必须先锁定具体私法关系和被侵害权利
- 正确答法：权利/义务 -> 违法或违约事实 -> 责任方式
- 不能进入/禁止误判：不能泛化成“违法要担责”；不能只列法条名
- 正式支撑题：CC0244_2026_东城_期中_18;CC0254_2026_丰台_二模_18
- 待回源题：CC0206_2025_西城_期末_19
- 边界提醒：责任题要落到责任方式，不要停在价值

### A7 规定/制度如何发挥作用
- 看到什么：设问问“规定如何推动/保护/规范”“制度作用”
- 必须同时满足：必须能指出具体制度怎样改变行为或保护权利
- 正确答法：制度规则 -> 主体行为边界 -> 责任/激励 -> 价值
- 不能进入/禁止误判：不能把制度作用写成政治制度套话
- 正式支撑题：CC0011_2024_丰台_二模_17;CC0340_2026_西城_一模_17
- 开放参考题：CC0311_2026_海淀_二模_18
- 待回源题：CC0318_2026_海淀_期中_18_2
- 边界提醒：制度题也要有材料动作，不是背目录

## B. 材料分诊

### B1 好意同乘交通损害
- 看到什么：材料出现无偿搭乘、交通事故、搭乘人受损、驾驶人无故意或重大过失
- 必须同时满足：必须同时有无偿帮忙搭乘和交通损害责任边界
- 正确答法：无偿搭乘不是合同主轴；按侵权责任和公平减责写，兼顾安全义务与责任边界
- 不能进入/禁止误判：禁止命中合同成立、人格权主轴、消费者、平台用工
- 正式支撑题：CC0002_2024_丰台_一模_17
- 边界提醒：这是独立触发，不要因为“搭乘约定”误判为合同题

### B2 民法绿色原则/生态环境保护
- 看到什么：材料出现民法典绿色原则、节约资源保护生态、排污破坏环境、法院判令停止侵害/修复/赔偿
- 必须同时满足：必须有生态环境事实或绿色义务，不是单纯经济发展
- 正确答法：写绿色原则如何约束民事活动；侵权责任如何修复环境和公共利益
- 不能进入/禁止误判：无具体作品/商标/专利侵权时，不得命中知识产权侵权主轴
- 正式支撑题：CC0011_2024_丰台_二模_17;CC0340_2026_西城_一模_17
- 边界提醒：有专利技术可写技术支撑，但不能把生态题改成知产侵权题；动物损害调解另入 B3。

### B3 饲养动物损害与融情调解
- 看到什么：材料出现饲养动物造成损害、饲养人/管理人责任、法院调解方案兼顾赔偿和行为约束
- 必须同时满足：必须有动物致害事实和法院以法律责任促成调解
- 正确答法：先写饲养动物损害责任，再写调解如何定分止争、兼顾情理和生态修复
- 不能进入/禁止误判：禁止命中合同、消费者、知识产权、非法律串页
- 正式支撑题：CC0254_2026_丰台_二模_18
- 边界提醒：这是法理+调解题，不是经济发展题

### B4 平台用工事实劳动关系
- 看到什么：材料出现合作协议、平台派单、签到、奖惩、结算、业务组成
- 必须同时满足：必须同时能看出人格从属性、经济从属性、组织从属性中的主要事实
- 正确答法：不看协议名称，按三从属性认定事实劳动关系，再写劳动权益和典型意义
- 不能进入/禁止误判：禁止命中普通合同题、消费者题、知识产权题
- 正式支撑题：CC0025_2024_朝阳_二模_17
- 开放参考题：CC0162_2025_海淀_一模_18
- 待回源题：CC0195_2025_西城_一模_20
- 边界提醒：只有“平台”二字不够，必须有管理控制和报酬结算

### B5 合同成立/履行/违约
- 看到什么：材料出现报价内容具体明确、中标通知、承诺到达、拒不履行或格式条款效力
- 必须同时满足：必须能完整判断合同是否成立/有效/履行，不是所有协议都算合同主轴
- 正确答法：要约/承诺 -> 合同成立或效力 -> 全面履行/诚信原则 -> 违约责任
- 不能进入/禁止误判：禁止把平台用工合作协议、好意同乘、普通诉讼都塞进合同
- 正式支撑题：CC0054_2024_石景山_一模_17;CC0200_2025_西城_二模_18
- 待回源题：CC0317_2026_海淀_期中_18
- 边界提醒：合同题先判成立/效力，再谈责任；别一上来写诚信

### B6 消费者欺诈/惩罚性赔偿
- 看到什么：材料出现消费者、经营者虚假宣传/隐瞒/隐蔽搭售/无法清楚知悉或有效拒绝
- 必须同时满足：必须有消费者关系和欺诈或权利受损事实；三倍赔偿必须有欺诈条件
- 正确答法：知情权/自主选择权/公平交易权 -> 欺诈或虚假宣传 -> 撤销/退赔/惩罚性赔偿
- 不能进入/禁止误判：禁止命中平台用工三从属性；禁止只写诚信经营
- 正式支撑题：CC0143_2025_朝阳_一模_19
- 恢复支撑题：CC0305_2026_海淀_一模_18_3
- 待回源题：CC0019_2024_朝阳_一模_19;CC0180_2025_海淀_期末_20
- 边界提醒：三倍赔偿别乱套，必须是消费者+经营者欺诈

### B7 知假买假/食品安全边界
- 看到什么：材料出现购买不合格食品、首单合理维权、后续扩大购买牟利
- 必须同时满足：必须能区分正常消费维权和滥用权利边界
- 正确答法：首单保护消费者权益；后续超出生活消费需要的知假买假不当然支持
- 不能进入/禁止误判：不能把所有食品安全题都机械写三倍赔偿到底
- 正式支撑题：CC0063_2024_西城_二模_16
- 边界提醒：这是消费者保护和诚信边界题

### B8 知识产权/技术秘密/不正当竞争
- 看到什么：材料出现作品、图片、商标、专利、技术秘密、离职人员带技术、未经许可使用、混淆、抢注、商业诋毁
- 必须同时满足：必须先判权利对象，再判侵害方式；创新价值只能在规则之后写
- 正确答法：权利对象 -> 未经许可/不正当手段/混淆抢注 -> 责任 -> 创新和公平竞争价值
- 不能进入/禁止误判：禁止只有“创新”就写知识产权；禁止把生态技术应用题当知产侵权
- 正式支撑题：CC0103_2025_丰台_一模_19;CC0131_2025_房山_一模_19;CC0137_2025_昌平_二模_20
- 开放参考题：CC0040_2024_海淀_一模_19;CC0353_2026_西城_期中_17
- 待回源题：CC0283_2026_朝阳_一模_18;CC0319_2026_海淀_期中_19
- 边界提醒：知产题不怕写价值，但价值必须从权利保护和责任机制推出

### B9 人格权/隐私/名誉/个人信息/人身权
- 看到什么：材料出现隐私、聊天记录、摄像头、肖像名誉、身体健康、高空抛物坠物
- 必须同时满足：必须能说出具体权利名和侵害方式
- 正确答法：权利名 -> 侵害行为 -> 责任方式/保护路径；高空抛坠物注意举证规则
- 不能进入/禁止误判：禁止泛写人格尊严；禁止把摄像头题写成治安管理
- 正式支撑题：CC0289_2026_朝阳_二模_18
- 恢复支撑题：CC0305_2026_海淀_一模_18_3
- 待回源题：CC0084_2025_东城_二模_19
- 边界提醒：先叫准权利名，再谈保护

### B10 相邻关系/物权边界
- 看到什么：材料出现通行、通风、采光、加装电梯、业主规约、恢复原状
- 必须同时满足：必须是相邻不动产权利或物权使用边界，不是普通邻里道德
- 正确答法：有利生产/方便生活/团结互助/公平合理 -> 材料事实 -> 停止侵害/恢复原状/协商
- 不能进入/禁止误判：禁止写成社区治理套话或必修三基层自治
- 正式支撑题：CC0340_2026_西城_一模_17;CC0289_2026_朝阳_二模_18
- 待回源题：CC0364_2026_通州_期中_19_1
- 边界提醒：相邻题要落在具体妨碍，不要只喊和谐邻里

### B11 家庭继承/养老赡养
- 看到什么：材料出现遗赠扶养协议、老人养老、子女赡养、告知书督促履行
- 必须同时满足：必须有家庭身份关系或扶养/赡养/继承法律事实
- 正确答法：协议效力/赡养义务 -> 材料事实 -> 老年权益保护和家庭文明价值
- 不能进入/禁止误判：禁止只写传统美德；禁止把遗产争议写成普通合同
- 正式支撑题：CC0061_2024_西城_一模_18
- 恢复支撑题：CC0045_2024_海淀_二模_19
- 待回源题：CC0051_2024_海淀_期中_21_1
- 边界提醒：家庭题不是道德作文，先有法定义务或协议效力

### B12 劳动竞业/平等就业
- 看到什么：材料出现竞业限制、商业秘密、高管/员工离职、普通员工就业、招聘限性别
- 必须同时满足：必须区分企业商业秘密保护与劳动者就业权；竞业限制不得过度
- 正确答法：合法适度保护商业秘密/知识产权，同时保障劳动者平等就业和自主择业
- 不能进入/禁止误判：禁止把竞业限制直接写成不正当竞争；禁止忽略劳动者权益
- 正式支撑题：CC0181_2025_海淀_期末_21
- 恢复支撑题：CC0373_2026_顺义_一模_18
- 待回源题：CC0119_2025_丰台_期末_19
- 边界提醒：竞业题核心是平衡，不是单向保护企业

### B13 未成年人校园欺凌惩教并行
- 看到什么：材料出现校园欺凌、未成年人违法、行政处罚、监护教育
- 必须同时满足：必须有未成年人违法和保护教育并重事实
- 正确答法：法律责任 -> 教育矫治 -> 未成年人保护和社会秩序
- 不能进入/禁止误判：禁止只写严惩；禁止只写教育感化
- 恢复支撑题：CC0332_2026_石景山_二模_19
- 边界提醒：惩和教要并行，不是二选一

## C. 禁止误判表

### C1 有合同字样但不是合同题
- 看到什么：材料出现协议、合作、约定
- 必须同时满足：必须能判断合同成立/效力/履行；若实质是劳动管理或无偿帮忙，不走合同主轴
- 正确答法：先看真实关系，再决定是否合同
- 不能进入/禁止误判：把平台用工合作协议、好意同乘、遗赠扶养全部塞进合同
- 正式支撑题：CC0025_2024_朝阳_二模_17;CC0002_2024_丰台_一模_17
- 边界提醒：协议名不是法律关系本身

### C2 有平台字样但不是劳动题
- 看到什么：材料出现平台、网络、App
- 必须同时满足：必须有派单、签到、奖惩、结算、业务组成等劳动从属性事实
- 正确答法：有三从属性才写劳动；否则按消费者/侵权/知产等真实事实分诊
- 不能进入/禁止误判：把机票代销平台消费者欺诈写成平台用工
- 正式支撑题：CC0143_2025_朝阳_一模_19
- 边界提醒：平台不是劳动关系自动按钮

### C3 有法院字样但不是必修三题
- 看到什么：材料出现法院、判决、司法
- 必须同时满足：设问若要求《法律与生活》或具体私法规则，先写具体权利义务责任
- 正确答法：私法规则优先，价值放后面
- 不能进入/禁止误判：一见法院就写公正司法、司法为民
- 正式支撑题：CC0002_2024_丰台_一模_17;CC0143_2025_朝阳_一模_19;CC0254_2026_丰台_二模_18
- 边界提醒：法院只是载体，得分通常在私法规则

### C4 有企业字样但不是经济生活
- 看到什么：材料出现企业、经营、市场
- 必须同时满足：筛是否问经营措施还是法律责任/权利保护
- 正确答法：问法律责任就写权利义务、侵权违约、不正当竞争
- 不能进入/禁止误判：写企业经营战略、市场调节、宏观调控
- 正式支撑题：CC0103_2025_丰台_一模_19;CC0131_2025_房山_一模_19
- 边界提醒：企业材料也可以是法律题

### C5 有创新字样但不能只写保护创新
- 看到什么：材料出现创新、科技、新质生产力
- 必须同时满足：必须有知识产权、技术秘密、惩罚性赔偿、恶意诉讼等法律事实
- 正确答法：先规则后创新价值
- 不能进入/禁止误判：只写激发创新活力，不写权利对象和侵害方式
- 正式支撑题：CC0103_2025_丰台_一模_19;CC0131_2025_房山_一模_19;CC0137_2025_昌平_二模_20
- 边界提醒：创新价值是结尾，不是主干

### C6 有消费者字样但不能乱写三倍赔偿
- 看到什么：材料出现消费者、购买、服务
- 必须同时满足：必须有经营者欺诈或法律规定的惩罚性赔偿条件
- 正确答法：先判断权利和欺诈，再决定退赔/惩罚性赔偿
- 不能进入/禁止误判：所有消费纠纷都写三倍赔偿
- 正式支撑题：CC0143_2025_朝阳_一模_19;CC0063_2024_西城_二模_16
- 恢复支撑题：CC0305_2026_海淀_一模_18_3
- 边界提醒：三倍赔偿是条件句，不是万能句

### C7 有法治字样但不能裸写全面依法治国
- 看到什么：设问有法治价值、法治意义、依法保护
- 必须同时满足：必须有具体规则保护谁、规范谁、救济谁
- 正确答法：规则-材料-价值三段写
- 不能进入/禁止误判：开头结尾全是全面依法治国套话
- 正式支撑题：CC0011_2024_丰台_二模_17;CC0131_2025_房山_一模_19;CC0340_2026_西城_一模_17
- 边界提醒：法治价值要从具体民事规则长出来

## 02 自检

- 没有把一题塞进多个万能篮子；本清单强调进入条件与禁止命中。
- 合同、纠纷解决、意义价值都被设置了触发条件，不能作为默认入口。
- OPEN_OR_REFERENCE 没有单独支撑核心分诊。
- 待回源题只作为待补位提醒，不作为核心支撑。
