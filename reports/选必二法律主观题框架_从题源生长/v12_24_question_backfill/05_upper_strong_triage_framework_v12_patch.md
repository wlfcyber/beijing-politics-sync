# 上篇强分诊框架 v12 补丁

本补丁只吸收 v12 已 source-lock 回填的题。未锁定的 6 题不推动框架新增点。

## 1. 新增/强化设问分诊

### 问法律原则的现实作用
看到：设问问某一民法原则如何促进市场、社会或秩序。
只能进入：规则作用题。
不能进入：道德评价、全面依法治国套话。
代表题：CC0019。

### 问良法/观点评析
看到：材料给旧法与新法，要求评析某同学观点。
只能进入：先表态，再比较时代背景和权利保护。
不能进入：一句话判对错，或只写必修三科学立法。
代表题：CC0051。

### 问维权准备和路径
看到：协商未果，问当事人需要做好哪些工作。
只能进入：路径选择 + 证据准备 + 诉求明确。
不能进入：直接判案，或乱写上诉、行政诉讼。
代表题：CC0245。

### 问举证责任分配
看到：材料要求完成举证责任表。
只能进入：先判诉讼类型，再判谁承担举证责任。
不能进入：一律写“谁主张谁举证”。
代表题：CC0325。

## 2. 新增/强化材料分诊

### 诚信原则进入市场秩序
触发条件：合同履行、司法定分止争、经营者信用公开至少出现一条具体事实。
标准写法：诚信原则 → 具体主体行为 → 权利保护/市场秩序。
代表题：CC0019。

### 住房/社区公共空间和相邻权
触发条件：共有部分、公共区域、相邻采光通行、安全风险。
标准写法：共有权/相邻权/安全义务 → 事实影响 → 合法程序或停止妨害。
代表题：CC0092、CC0364、CC0223。

### 数据产品与技术抓取
触发条件：原始数据经算法形成产品，另一方用技术手段抓取并商业化。
标准写法：反不正当竞争 → 数据权益/商业道德/市场秩序 → 停止侵害赔偿。
代表题：CC0157。

### 唤醒词/字号/企业名称混淆
触发条件：已有影响力标识被抢注、仿冒、相似使用、造成混淆。
标准写法：反不正当竞争 → 混淆/虚假宣传/商誉损害 → 停止侵害赔偿。
代表题：CC0206、CC0223。

### 侵权责任类型先判
触发条件：缺陷产品、禁养危险动物、安全保障义务、建筑物坠落。
标准写法：先判责任类型 → 结合损害事实和因果 → 责任结果。
代表题：CC0180、CC0157、CC0325。

## 3. 禁止误判补丁

- 有合同事实，不等于主轴一定是合同；如果题目问劳动纪律、相邻权或不正当竞争，合同只能作背景。
- 有法院，不等于写必修三公正司法；先看是否要求具体私法规则。
- 有平台，不等于平台用工；消费者预订平台、数据平台、公共AI平台都要重新分诊。
- 有创新，不等于只写保护创新；必须写法律手段：调解、惩罚性赔偿、规制恶意诉讼、合同违约或不正当竞争。
- 有意义题，不等于裸写价值；必须从规则推出价值。
