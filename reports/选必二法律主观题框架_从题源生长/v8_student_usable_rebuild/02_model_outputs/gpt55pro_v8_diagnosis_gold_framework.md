# 1. V7.1 失败诊断

## P0 问题 1：v7.1 的首屏不是学生框架，而是工程验收语言

`01_v7_failure_diagnosis.md` 已判定 v7.1 为 `FAIL_TO_REBUILD`。它的核心失败不是排版，而是成品定位错了：学生打开后先看到的是 `guarded v2 / Framework v2 / Status / Validation Position / evidence` 这类工程词，而不是“看到设问先干什么、看到材料圈什么、第一句怎么写”。

对应文件和字段：

* `01_v7_failure_diagnosis.csv`：`DIAG_0001`，`A_framework`，状态 `FAIL`
* `v71_current_baodian_core.md`：首页仍有工程状态、验证位置、证据口径
* `01_v7_failure_diagnosis.md`：明确指出当前版本“更像 guarded 工程验收包 + 证据侧栏 + 答案汇编”

v8 必须把学生版第一屏改成：

```text
看设问 → 定任务
看材料 → 抓主体、行为、冲突、结果
转法律 → 权利、义务、责任、程序、价值
写答案 → 法律规则 + 材料事实 + 结论/价值
```

不能再让学生先看工程状态。

`full_score_sentence_bank.csv` 只有 12 行，而且更像“节点模板/工程说明”，不是学生可背、可替换、可迁移的满分句库。




# 2. 推荐 8 道金样板题

这 8 道不按教材目录选，而是按“主观题作答机制”选。每一道都应成为 v8 的样板，不只是知识点例题。

## 1. CC0137_2025_昌平_二模_20

* **类型覆盖**：完成裁判理由类；AI 作品著作权；裁判理由生成。
* **证据状态**：PASS，formal。
* **为什么代表一种作答机制**：它不是简单问“AI 作品是否受保护”，而是要求学生完成裁判理由。学生必须按“法律主体资格 → 作品条件 → 侵权行为 → 责任结果”生成裁判语言。
* **能沉淀出的考场动作**：看到“完成裁判理由”，不要先写意义；先写法律规则，再把材料事实塞进规则，最后落结论。
* **不是哪种题**：不是泛泛“科技创新意义题”，也不是法考式复杂著作权论文。
* **不能被它支撑的边界**：不能用它支撑所有 AI 题；它只支撑“AI 辅助生成作品 + 人的智力成果 + 未经许可使用”的裁判理由动作。

---

## 2. CC0143_2025_朝阳_一模_19

* **类型覆盖**：支持诉讼请求理由类；合同成立/可撤销；消费者欺诈；三倍赔偿。
* **证据状态**：PASS，formal。
* **为什么代表一种作答机制**：它要求学生把一个诉讼请求拆成多个法律层次：先说明合同成立，再说明欺诈导致可撤销，再说明消费者权益保护法支持三倍赔偿，最后收束到网络消费秩序。
* **能沉淀出的考场动作**：看到“说明法院支持诉讼请求的理由”，必须先拆请求，再逐项找法律依据。
* **不是哪种题**：不是单纯消费者权益题；也不是只写“平台应诚信经营”的企业经营题。
* **不能被它支撑的边界**：不能把所有消费者题都写成三倍赔偿，只有材料出现欺诈、虚假宣传、隐瞒真实信息等才触发。

---

## 3. CC0025_2024_朝阳_二模_17

* **类型覆盖**：劳动仲裁理由 + 案例意义；新就业形态；平台劳动关系三从属性。
* **证据状态**：PASS，formal。
* **为什么代表一种作答机制**：它清楚展示了“材料事实 → 法律判断”的过程。不是看到平台就写经济发展，而是要判断人格、经济、组织三种从属性，进而认定事实劳动关系。
* **能沉淀出的考场动作**：劳动题先判“是不是劳动关系”，不能被“合作协议”表面名称带跑。
* **不是哪种题**：不是平台经济发展题，也不是企业经营题。
* **不能被它支撑的边界**：不能把所有平台合作都判定为劳动关系，必须有从属性事实支撑。

---

## 4. CC0238_2026_东城_二模_19

* **类型覆盖**：评析行为类；不正当竞争；违约责任；劳动纪律/职业道德；营商环境。
* **证据状态**：PASS，formal。
* **为什么代表一种作答机制**：它要求学生一主体一评析，不能把乙公司和张某混成一个主体。乙公司要评商业诋毁、不正当竞争、违约责任；张某要评违反劳动纪律和职业道德。
* **能沉淀出的考场动作**：看到“评析某主体行为”，先分主体，再逐个写“行为定性 + 法律后果 + 应然要求”。
* **不是哪种题**：不是只写“企业诚信经营”的经济题。
* **不能被它支撑的边界**：不能用它支撑所有营商环境题；只有材料有具体违法/违约/失信行为时才进入此动作。

---

## 5. CC0305_2026_海淀_一模_18_3

* **类型覆盖**：权利保护/侵权责任类；隐私权；消费者知情权；虚假宣传/欺诈；惩罚性赔偿。
* **证据状态**：PASS_RECOVERED，formal。
* **为什么代表一种作答机制**：它非常适合训练“权利名 + 侵害行为 + 责任结果”的作答链。材料中两个侵害点分别触发不同法律语言：公开包间监控/聊天信息触发隐私权，夸大治疗功效诱导消费触发知情权、欺诈、惩罚性赔偿。
* **能沉淀出的考场动作**：看到“是否支持诉求/侵权责任”，先找权利，再找侵害方式，再写责任。
* **不是哪种题**：不是泛泛消费者保护题；也不是只写“保护合法权益”的价值题。
* **不能被它支撑的边界**：不能没有欺诈事实就写惩罚性赔偿；不能没有公开/泄露私密信息就写隐私权。

---

## 6. CC0229_2026_东城_一模_18

* **类型覆盖**：知识产权/创新保护；诉讼调解；惩罚性赔偿；恶意诉讼；司法守护创新。
* **证据状态**：PASS_RECOVERED，formal。
* **为什么代表一种作答机制**：这是“价值从法律手段长出来”的典型题。不能只写“司法促进创新”，而要把三个案例分别转成法律手段：调解、惩罚性赔偿、规制恶意诉讼，再收束到创新保护。
* **能沉淀出的考场动作**：多案例意义题要一案一手段，一手段一价值。
* **不是哪种题**：不是必修三公正司法套话题。
* **不能被它支撑的边界**：不能用它支撑所有创新题；必须有知识产权、诉讼调解、惩罚性赔偿、恶意诉讼等法律事实。

---

## 7. CC0125_2025_延庆_一模_19

* **类型覆盖**：纠纷解决/调解类；网络消费“仅退款”；合同解除；诚信原则；证据意识。
* **证据状态**：PASS，formal。
* **为什么代表一种作答机制**：它不是让学生判谁赢谁输，而是让学生说明“如何调解并说明理由”。这类题必须把答案写成“调解方案 + 法律理由 + 材料依据”。
* **能沉淀出的考场动作**：看到“如何调解”，不要只讲法律结论；要写可执行调解方案，并说明为什么合理。
* **不是哪种题**：不是普通合同责任判决题。
* **不能被它支撑的边界**：不能把所有纠纷都写成调解优先；只有设问要求调解或材料适合协商化解时使用。

---

## 8. CC0340_2026_西城_一模_17

* **类型覆盖**：法律制度意义/社会经济价值类；民法典物权编；绿色原则；权利义务边界；绿色发展。
* **证据状态**：PASS，formal。
* **为什么代表一种作答机制**：它适合训练“法条规定如何推动某种价值”。学生必须先写法律依据，再写作用路径，最后写绿色发展意义。
* **能沉淀出的考场动作**：看到“法律规定如何推动……”，要按“法律依据 → 作用路径 → 价值结果”写。
* **不是哪种题**：不是生态文明哲学题，也不是经济绿色发展措施题。
* **不能被它支撑的边界**：不能脱离物权编法条直接写绿色发展；必须把权利界限、法定义务、责任救济写出来。

# 3. 推荐学生版框架

这个框架不是最终定稿，但我建议 v8 学生版按以下“考场动作”重构。核心是：**先看设问交付物，再把材料转成法律语言，最后把法律语言转成得分句。**

## 节点 1：先定任务——这题要我交什么答案？

**看见：**

* “说明理由”
* “完成裁判理由”
* “评析行为”
* “谈谈认识”
* “分析意义”
* “如何调解”
* “如何依法保护”
* “诉求能否得到支持”

**先判：**

先判断设问要你交的是哪种成品：

```text
裁判理由
行为评析
诉求判断
调解方案
权利保护路径
制度意义
```

**再转：**

把设问翻译成答题顺序：

```text
理由题：规则 + 材料 + 结论
评析题：表态 + 定性 + 后果 + 应然
意义题：规则 + 作用路径 + 价值结果
调解题：方案 + 法律理由 + 材料依据
```

**这样写：**

```text
本题先回答……，再结合材料说明……，最后落到……
```

**别这样写：**

不要一上来背整章法律知识；不要没表态就直接说价值。

**代表题：**

CC0137、CC0143、CC0125、CC0340。

**边界：**

这个节点只是入口，不是答案本身。

---

## 节点 2：先表态——支持不支持，违法不违法，有效无效

**看见：**

* “诉求能否得到支持”
* “评析某行为”
* “法院为什么支持”
* “合同是否成立/有效”
* “是否构成侵权/欺诈/违约”

**先判：**

必须先给结论：

```text
支持 / 不支持
违法 / 不违法
有效 / 无效
构成 / 不构成
应承担 / 不应承担
```

**再转：**

找结论背后的法律理由：

```text
权利是否存在
行为是否侵害
合同是否成立/有效
是否存在欺诈、违约、侵权
责任或救济是什么
```

**这样写：**

```text
法院支持……是有依据的。因为……构成……，依法应……
```

**别这样写：**

不要只说“有利于维护公平正义”；不要把表态题写成意义题。

**代表题：**

CC0143、CC0305、CC0373、CC0238。

**边界：**

没有明确判断任务的意义题，不要硬套“支持/不支持”。

---

## 节点 3：事实转法律——把材料里的普通话翻译成法律话

**看见：**

材料里出现具体行为：

```text
擅自搭售
未经许可使用
抹去水印
公开监控录像
虚假宣传
平台派单
制度奖惩
拒绝面试
签订竞业协议
加购索赔
```

**先判：**

这个行为在法律上叫什么：

```text
欺诈
侵犯隐私权
侵犯著作权
不正当竞争
违约
就业歧视
存在劳动从属性
滥用权利
```

**再转：**

按这个格式转：

```text
材料事实 → 法律定性 → 法律后果
```

**这样写：**

```text
材料中……，说明……，构成/体现……，依法应……
```

**别这样写：**

不要只抄材料；不要只写法律词不带材料事实。

**代表题：**

CC0025、CC0143、CC0305、CC0238、CC0137。

**边界：**

如果材料没有具体行为，不能凭空写侵权、欺诈、违约。

---

## 节点 4：裁判理由——规则、事实、结论、责任四句走完

**看见：**

* “完成裁判理由”
* “说明法院判决理由”
* “法院支持/不支持的理由”
* “分析本案判决”

**先判：**

法院为什么这样判？必须回答：

```text
适用什么规则
材料事实满足什么条件
得出什么法律结论
谁承担什么后果
```

**再转：**

按四句写：

```text
法律规则：
材料事实：
法律结论：
责任/效果：
```

**这样写：**

```text
根据……，……依法受保护。材料中……，说明……。因此……构成……，应承担……
```

**别这样写：**

不要写成法考构成要件长文；高中答案只要把规则、事实、结论、责任写清。

**代表题：**

CC0137、CC0143、CC0054、CC0305。

**边界：**

意义题不能只写裁判理由，还要补价值结果。

---

## 节点 5：多主体一主体一链，多案例一案例一链

**看见：**

材料有多个主体或多个案例：

```text
乙公司和张某
闫某和张某
案例一、案例二、案例三
表格多行
多个诉讼请求
```

**先判：**

不要混写。先拆：

```text
主体 1：行为 → 定性 → 后果
主体 2：行为 → 定性 → 后果
案例 1：法律手段 → 作用
案例 2：法律手段 → 作用
```

**再转：**

每个主体/案例只对应自己的法律语言。

**这样写：**

```text
对甲来说……；对乙来说……
案例一体现……；案例二体现……；案例三体现……
```

**别这样写：**

不要把多个主体混成一句“都应该依法诚信”；不要把三个案例写成一个泛泛价值。

**代表题：**

CC0238、CC0229、CC0373、CC0325。

**边界：**

单一主体题不必强行拆多链。

## 节点 6：程序救济单独写——路径、证据、救济不能丢

**看见：**

* “调解”
* “仲裁”
* “诉讼”
* “举证责任”
* “如何依法保护权利”
* “请求停止侵害、赔偿损失”
* “协商、调解、仲裁、诉讼”

**先判：**

这题是不是在问“怎么解决纠纷”或“如何证明/救济”。

**再转：**

按三步写：

```text
路径：协商/调解/仲裁/诉讼
证据：谁主张谁举证/举证责任倒置
救济：停止侵害、赔礼道歉、赔偿损失、惩罚性赔偿等
```

**这样写：**

```text
当事人可以通过……解决纠纷；主张……的一方应……；权利受到侵害时可请求……
```

**别这样写：**

不要把程序题写成“依法治国”；不要只写结果不写路径和证据。

**代表题：**

CC0125、CC0289、CC0325、CC0380 只可作参考运行。

**边界：**

CC0380 是 `OPEN_OR_REFERENCE`，不能支撑核心节点。

---

## 节点 7：价值必须从规则长出来

**看见：**

* “意义”
* “社会价值”
* “如何推动……”
* “如何守护创新”
* “如何推动绿色发展”
* “案例的意义”

**先判：**

不能直接写价值。先找法律规则保护谁、约束谁、解决什么冲突。

**再转：**

按三层写：

```text
法律规则保护/约束谁
解决材料中的什么问题
形成什么秩序或价值
```

**这样写：**

```text
……规则明确了……的权利义务，规范了……行为，保护了……合法权益，有利于……秩序/发展。
```

**别这样写：**

不要裸写“有利于全面依法治国”；不要把选必二意义题写成必修三法治建设题。

**代表题：**

CC0340、CC0229、CC0063、CC0025。

**边界：**

价值句必须绑定材料事实和法律规则，否则降级为套话。

## 金样板二：CC0143_2025_朝阳_一模_19

### 题源

* **question_id**：CC0143_2025_朝阳_一模_19
* **证据等级**：PASS，formal
* **设问**：结合材料，运用《法律与生活》知识，说明人民法院支持王某的诉讼请求的理由。
* **材料核心**：王某通过 A 机票代销平台购买机票，页面显示总价 350 元，优惠后实付 310 元；事后发现机票实际总价为 300 元；A 公司通过技术手段擅自搭售 10 元外卖红包，王某不能清楚知悉费用支出细节，也无法拒绝支付额外费用；王某起诉要求退还票款并三倍赔偿，法院支持。

---

### 一、框架入口

这题从“诉讼请求支持理由”入口进。

设问触发词：

```text
说明人民法院支持王某的诉讼请求的理由
```

第一步不是写意义，而是拆诉讼请求。

王某的诉求至少有两层：

```text
1. 退还票款 310 元
2. 三倍赔偿
```

所以答案也要拆成两条法律理由：

```text
合同层：合同成立，但因欺诈可撤销
消费者权益层：经营者欺诈，支持三倍赔偿
```

---

### 二、最小必要判断

**第一判断：王某和 A 公司之间有没有合同？**

有。王某通过平台购票并出票，双方合同成立。

不判断这个，会漏掉合同成立 1 分。

---

**第二判断：合同成立是否等于合同有效？**

不等于。材料中 A 公司隐蔽搭售，使王某在不能清楚知悉、不能拒绝的情况下支付额外费用，属于欺诈，合同可撤销。

不判断这个，会把题误写成普通违约责任。

---

**第三判断：三倍赔偿的依据是什么？**

依据消费者权益保护法中经营者提供商品或服务有欺诈行为的三倍赔偿规则。

不判断这个，会写成泛泛“保护消费者权益”，拿不满分。

---

### 三、材料事实到法律语言

**材料事实 1：王某通过 A 公司购买机票，页面显示价格并出票。**

法律语言：

```text
双方经过要约、承诺，合同成立。
```

```text
原告支付机票款，已经出票，合同成立。
```

**材料事实 2：实际机票价格为 300 元，但页面显示优惠后实付 310 元。**

法律语言：

```text

```

对应细则：

```text
王某不能清楚知悉费用支出细节。
```

---

**材料事实 3：A 公司擅自搭售 10 元外卖红包，王某无法拒绝支付。**

法律语言：

```text

```

对应细则：

```text
主观上具有欺诈故意，客观上实施欺诈行为。
```

---

**材料事实 4：王某要求退还票款并三倍赔偿，法院支持。**

本题得分链：

```text
购票出票 → 合同成立

消费者权益保护法 → 三倍赔偿
判决支持 → 保护消费者、规范平台经营、优化网络消费环境
```

---

### 五、满分句

**满分句 1：合同成立**

```text
王某通过 A 公司平台购买机票并支付价款，机票已经出票，说明双方之间的网络服务合同成立。
```

**满分句 2：合同成立不等于合同有效**

```text
合同成立不等于合同有效，若一方以欺诈方式使对方在违背真实意思表示的情况下订立合同，该合同依法可以撤销。
```

**满分句 3：欺诈事实**

```text
A 公司通过技术手段擅自搭售 10 元外卖红包，王某在购买界面不能清楚知悉费用支出细节，也无法拒绝支付额外费用，说明 A 公司主观上具有欺诈故意，客观上实施了欺诈行为。
```

**满分句 4：撤销和退还**

```text
因此，王某请求人民法院撤销相关合同并退还票款，具有民法典依据。
```

**满分句 5：三倍赔偿**


**满分句 6：价值收束**

```text
该判决维护了消费者合法权益，引导平台企业规范经营、诚信经营，有利于规范网络消费市场秩序，促进数字经济健康发展。
```

### 六、完整考场答案

王某通过 A 公司平台购买机票并支付价款，机票已经出票，说明双方之间的网络服务合同成立。但合同成立不等于合同有效。A 公司在销售机票过程中，通过技术手段擅自搭售 10 元外卖红包，王某在购买界面不能清楚知悉费用支出细节，也无法拒绝支付额外费用，说明 A 公司主观上具有欺诈故意，客观上实施了欺诈行为，使王某在违背真实意思表示的情况下订立合同。根据民法典规定，王某请求撤销相关合同并退还票款具有法律依据。

同时，根据消费者权益保护法，经营者提供商品或者服务有欺诈行为的，消费者可以要求增加赔偿。A 公司提供服务时存在欺诈行为，侵犯了消费者的知情权、公平交易权和自主选择权，因此法院支持王某要求三倍赔偿的诉讼请求具有法律依据。该判决维护了消费者合法权益，引导平台企业规范经营、诚信经营，有利于规范网络消费市场秩序，促进数字经济健康发展。

### 七、易错路径

**易错 1：只写消费者权益，不写合同层。**

错误写法：

```text
A 公司侵犯消费者权益，所以应三倍赔偿。
```

为什么错：

```text
细则要求先写合同成立，还要写合同成立不等于生效。
```

框架如何避免：

```text
节点 2“先表态” + 节点 4“裁判理由”：先拆诉求，再逐层写依据。
```

---

**易错 2：把欺诈事实写不全。**

错误写法：

```text
A 公司多收费，所以构成欺诈。
```

为什么错：

```text
细则强调三个材料条件：擅自搭售、不能清楚知悉费用细节、无法拒绝额外费用。
```

框架如何避免：

```text
节点 3“事实转法律”：每个法律词前必须配材料事实。
```

---

**易错 3：法律名称写错。**

错误写法：

```text
根据消费者保护法……
```

为什么错：

```text
细则要求“消费者权益保护法”字样，法律名称不准确可能丢分。
```

框架如何避免：

```text
满分句库必须保留必写关键词：消费者权益保护法、欺诈、三倍赔偿。
```

---

**易错 4：写成不正当竞争或隐私权。**

为什么错：

```text
本题核心不是商家之间竞争，也不是私人信息泄露，而是消费者在交易中被隐蔽搭售、违背真实意思表示。
```

---

### 八、迁移提醒

以后遇到：

```text
平台隐蔽收费
自动勾选增值服务
虚假优惠
强制搭售
夸大宣传诱导消费
消费者要求退赔
```

可以使用这条链：

```text
合同成立 → 是否存在欺诈/误导 → 可撤销或退还 → 消费者权益保护法 → 惩罚性赔偿 → 规范市场秩序
```

不能乱用到：

```text
没有消费者身份的普通商事纠纷
没有欺诈事实的普通违约
只有价格争议但没有隐瞒、误导、强制搭售的题
```

# 5. 给 Codex 的改写警告

## 警告 1：不要把 v8 写成“更漂亮的工程报告”

v8 不是再生成一个：

```text
coverage matrix + evidence map + pass report
```

而是要生成学生能用的：

```text
看见什么
先判什么
再转什么
这样写
别这样写
```

工程证据只能放教师版。

---

## 警告 2：不要先写总框架，必须先写金样板

如果 Codex 一上来写：

```text
选必二法律主观题总框架如下……
```

大概率会再次失败。

正确顺序必须是：

```text
8 道金样板题 → 从样板中抽学生动作 → 学生版框架 → 教师证据版 → 53 题逐题运行
```

---

## 警告 3：不要让 OPEN_OR_REFERENCE 偷偷支撑节点

`OPEN_OR_REFERENCE` 可以出现在：

```text
参考运行
边界说明
附录
```

不能出现在：

```text
核心节点支撑
金样板
主干满分句库
```

---

## 警告 4：不要把 pending 三题写回闭合正文

以下三题仍然 pending：

```text
CC0094_2025_东城_期末_19_3
CC0259_2026_丰台_期中_19
CC0118_2025_丰台_期末_18_2
```

它们只能放：

```text
补证附录
风险说明
扩展计划
```

不能进入 53 题正文运行。

---

## 警告 5：不要把意义题写成必修三法治套话

每条价值句必须先有法律规则和材料事实。

禁止裸写：

```text
有利于全面依法治国
有利于建设法治国家
有利于维护公平正义
```

推荐写法：

```text
该规则明确了……的权利义务，规范了……行为，保护了……合法权益，因此有利于……
```

---

## 警告 6：不要写成法考

高中政治选必二不是法考案例分析。

禁止：

```text
复杂构成要件
过长法理论证
多层责任竞合
细密法律推理
```

推荐：

```text
谁的权利
谁的行为
违反什么规则
承担什么后果
体现什么价值
```

---

## 警告 7：不要让满分句库继续停留在“答案摘抄”

满分句库必须按这个格式重写：

```text
材料信号：
适用设问：
必写关键词：
推荐句式：
禁止乱用：
代表题：
```

没有使用条件的句子，不算满分句库。

---

## 警告 8：逐题运行必须展示“学生脑内路径”

每道题不能只写答案。必须写：

```text
这题从哪里进框架
第一步判断什么
材料事实怎么转成法律语言
法律语言怎么对应细则
满分句怎么生成
完整答案
易错路径
迁移提醒
```

遮住完整答案，只看前面的“框架运行”，学生也应该能推出答案。这个标准不过，v8 不能标完成。

---

## 警告 9：继续扫描 CC0229 旧错配坏词

v8 生成后必须全量扫描，禁止出现：

```text
逃逸粒子
创新资源集聚
空间布局精准
全链条产业生态
```

这些词一旦出现，说明旧错配污染回流。

---

## 警告 10：最终宝典顺序必须重排

推荐顺序：

```text
第一章：这本书怎么用
第二章：选必二法律主观题到底怎么从材料得分
第三章：8 道金样板题
第四章：学生版一页框架
第五章：教师证据版框架
第六章：高频满分句库
第七章：53 题逐题运行
第八章：边界与风险
```

不要再把工程验收、证据统计、版本状态放在学生第一屏。
