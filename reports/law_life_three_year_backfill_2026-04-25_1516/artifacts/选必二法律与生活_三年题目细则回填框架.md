## 二、主观题回填：材料到知识到给分链

### 1. 合同成立与民法原则：碳排放权交易

题源：2024石景山一模17，8分，答案。  
案例：乙公司发布碳排放配额采购比选公告，甲公司报价并承诺违约补差价，乙公司确认中标并送达中标通知。甲公司后拒不履行，法院判令其支付差价款289万余元及利息。

回填到框架位置：合同成立、合同履行、民法基本原则、绿色原则。

答题链：

| 材料抓手 | 知识落点 | 主体/行为 | 给分表达 |
|---|---|---|---|
| 甲公司报价，内容具体确定 | 要约 | 甲公司 | 报价内容具体确定，属于要约 |
| 乙公司确认中标并送达中标通知 | 承诺 | 乙公司 | 中标通知送达，属于承诺，双方合同成立 |
| 甲公司拒不履行，乙公司另购产生差价 | 合同履行、违约责任 | 甲公司 | 合同合法有效，当事人应按约全面履行；拒不履行应承担违约责任 |
| 碳排放配额交易、双碳目标 | 诚信、公平、绿色原则 | 法院判决 | 引导民事主体遵循诚信、公平、绿色原则，维护碳排放权交易市场秩序，推动企业转型升级，落实碳达峰、碳中和 |

课堂处理：这题是合同成立的标准样板，不能只背“合同有约束力”。学生必须能从材料中划出“报价=要约”“中标通知送达=承诺”。意义部分不是泛写法治社会，而是把民法原则和碳交易市场秩序连起来。

### 2. 合同、违约、侵权并行：无人机案

题源：2026东城期末18，11分，细则。  
案例：陈某发邮件急需购买A型号无人机，刘某回复“全新原装，15000元”，陈某转账。刘某误寄内部结构轻微损伤的展示机，首次使用坠毁砸伤陈某，产生医疗费8000元和商业拍摄损失5000元，协商未果。

回填到框架位置：合同成立与有效、违约责任、侵权责任、维权准备。

细则化答题链：

| 层次 | 材料抓手 | 给分点 |
|---|---|---|
| 合同成立与有效 | 邮件购买、报价、转账 | 要约+承诺=合同成立；主体适格、意思表示真实、形式内容合法=合同有效。分析清楚2分；只说成立且有效1分 |
| 违约责任 | 约定全新原装，实际误寄损伤展示机 | 刘某未全面履行合同义务，违反全面履行原则/诚信原则/公平原则，构成违约，应赔偿损失等。该层共4分 |
| 侵权责任 | 无人机因内部损伤坠毁，砸伤手臂 | 对销售者主张一般过错侵权：行为、损害、因果、过错，完整分析2分，不完整1分；结论构成侵权并担责1分 |
| 产品责任边界 | 若主张生产者责任 | 生产者产品责任可按无过错责任分析；销售者一般按过错责任分析。把生产者写成一般过错、销售者写成无过错均为错 |
| 维权准备 | 已“协商未果” | 选调解、仲裁或诉讼并具体说明；准备起诉状、身份信息、三年时效；固定合同/订单/转账/邮件/故障/损害/因果/损失证据；明确违约或侵权诉求 |

课堂处理：这题是近三年最适合训练“合同与侵权不要混同”的题。答违约时不能套侵权四要件；答侵权时不能只写“违约造成损失”。维权部分要抓住材料已经“协商未果”，不能机械写继续协商。

### 3. 直播“假一赔十”：合同条款、欺诈、消费者权益

题源：2026朝阳期末复练题，见2026丰台一模细则复练材料，参考答案。  
案例：直播间将大叶紫檀手串宣称为小叶紫檀，承诺“保真”“假一赔十”。消费者支付1000元购买后发现材质不符，诉请赔偿1万元，法院支持。

回填到框架位置：合同履行、违约金/惩罚性赔偿、消费者知情权和公平交易权。

答题链：

| 材料抓手 | 知识落点 | 答题表达 |
|---|---|---|
| 观看直播并付款购买 | 买卖合同成立有效 | 依法成立的合同具有法律约束力，当事人应全面履行 |
| 主播明确“保真”“假一赔十” | 合同条款 | 直播承诺构成合同内容，对店铺有约束力 |
| 大叶紫檀冒充小叶紫檀 | 违约+欺诈 | 商家交付材质不符，未履行质量义务，构成违约；隐瞒真实材质构成欺诈 |
| 法院判1万元 | 约定违约金/消费者保护 | 自愿约定的“假一赔十”不违反强制性规定，应履行；保护消费者知情权、公平交易权，体现诚信原则 |

课堂处理：不能把“假一赔十”只写成法定三倍或十倍赔偿。这里要突出“合同承诺成为合同条款”，再接欺诈和消费者权益。

### 4. 食品安全惩罚性赔偿与理性维权

题源：2024西城二模16，8分，试题+细则。  
案例：消费者先购30盒黄芪薏米饼干，发现不符合食品安全标准后又大量加购200盒，诉请全部十倍赔偿。法院支持首单惩罚性赔偿，不支持加购部分。

回填到框架位置：消费者权益、惩罚性赔偿、权利行使边界。

细则化答题链：

| 层次 | 给分点 | 材料扣合 |
|---|---|---|
| 经营者责任 | 经营者应依法、诚信经营，保护消费者合法权益，2分 | 首单适用惩罚性赔偿，打击违法生产经营，维护消费者权益，1分 |
| 消费者权利边界 | 民事主体行使民事权利不能超过正当界限，不得滥用权利损害公共利益或他人权益，2分 | 加购超出正常生活消费需要，不支持加购赔偿，引导消费者诚信、理性维权，1分 |
| 判决价值 | 平衡食品安全与生产经营秩序两种价值取向，2分 | 服务保障经济社会高质量发展 |

课堂处理：这题要把“保护消费者”与“消费者不得滥用权利”同时教出来。学生只写“消费者维权、法院保护权益”会漏掉加购部分的权利边界。

### 5. 公共场所安全保障义务的合理限度

题源：2026丰台一模20，8分，细则。  
案例：原告在餐饮/商业场所摔倒，证据显示事发现场不存在影响通行的客观因素，原告是完全民事行为能力人，法院认定经营者和管理者无过错，不承担赔偿责任。

回填到框架位置：侵权责任、安全保障义务、证据与过错。

细则化答题链：

| 材料抓手 | 知识落点 | 答题表达 |
|---|---|---|
| 宾馆、商场等公共场所 | 安全保障义务 | 经营者、管理者因过错造成他人损害才承担侵权责任 |
| 现场无影响通行因素 | 过错判断、证据 | 安全保障义务应保持在合理限度内，证据不足以证明被告有过错 |
| 原告完全民事行为能力人 | 自我注意义务 | 摔倒系自身未尽安全注意义务所致，被告不承担赔偿责任 |
| 判决意义 | 权利义务平衡 | 平衡双方权利义务，明确经营场所安全保障义务边界，倡导自我负责与安全文明出行 |

课堂处理：这题要纠正常见模板病。不能见公共场所就写经营者必担责，也不能泛写市场秩序、营商环境。核心是“合理限度+证据+过错”。

### 6. 好意同乘：侵权责任减轻

题源：2024丰台一模17，8分，答案。  
案例：甲免费搭乘乙车，上车后未按提醒系安全带；乙雨天路滑撞灯杆造成甲受伤并主动垫付部分费用。法院酌情减轻乙赔偿责任。

回填到框架位置：侵权责任、过错程度、公平原则、社会主义核心价值观。

答题链：

| 材料抓手 | 知识落点 | 答题表达 |
|---|---|---|
| 免费搭乘 | 好意同乘关系 | 驾驶人对搭乘人生命财产安全负有保障义务 |
| 交通事故造成受伤 | 生命权、健康权 | 乙造成甲受伤，侵犯生命权、健康权，应承担相应责任 |
| 雨天路滑、非故意或重大过失、甲未系安全带 | 减轻责任 | 乙虽有过错但非故意或重大过失，甲要求全部赔偿有违平等/公平，根据好意同乘规定应减轻赔偿责任 |
| 主动垫付费用 | 价值导向 | 维护双方合法权益，促进社会公平，弘扬友善互助、绿色出行 |

课堂处理：不能把好意同乘讲成“好心就无责”。本题的答案结构是“有责但可减轻”。

### 7. 知识产权与不正当竞争：虚拟数字人 Ada

题源：2024海淀一模19，8分，答案；2024海淀一模细则。  
案例：甲公司创作虚拟数字人 Ada 并制作相关视频。乙公司未经许可发布视频，替换标识、添加注册商标和课程营销信息，并以其他虚拟数字人名称作标题。法院判决消除影响并赔偿。

回填到框架位置：著作权、商业宣传、不正当竞争、创新保护。

细则化答题链：

| 层次 | 给分点 | 材料扣合 |
|---|---|---|
| 权利基础 | 民事主体的著作权受法律保护；甲公司创作 Ada，对其享有著作权 | 写出“著作权”并对应甲公司创作 |
| 侵权行为 | 乙未经许可使用相关视频，替换标识、加注注册商标 | 写出“侵权”并加材料，细则中该层可得2分 |
| 竞争行为 | 乙进行虚假和引人误解的商业宣传，属于不正当竞争 | 不只写“侵犯著作权”，还要补上不正当竞争 |
| 判决价值 | 赔偿损失有利于激发创新活力、维护市场秩序 | 价值指向创新和市场秩序，不泛化为所有法治意义 |

课堂处理：这题是一材多点：著作权+商标标识混淆+课程营销+不正当竞争。学生只答“知识产权受保护”会丢材料点。

### 8. 专利与技术秘密：从“蜜胺”到新能源汽车技术秘密

题源A：2024房山一模17，11分，答案。  
题源B：2025房山一模19，8分，细则；2025丰台一模19，题干/答案。  

回填到框架位置：专利权、技术秘密、不正当竞争、惩罚性赔偿、劳动者保密义务。

关键知识链：

| 问题 | 2024房山一模17“蜜胺” | 2025房山一模19/丰台一模19技术秘密 |
|---|---|---|
| 专利与技术秘密区别 | 发明专利有明确保护期，发明专利在保护期内他人即使独立作出也不得实施；技术秘密只要保护得当可一直保护，但他人独立作出或正当获得则不构成侵权 | 2026东城期末选择题继续考“专利权公开换保护”，不存在“不为公众知悉的专利” |
| 行为定性 | 未经许可使用专利生产产品，非法获取技术秘密，构成不正当竞争 | B集团挖走核心技术人员并窃取技术秘密，属于典型不正当竞争，材料1分+定性1分 |
| 责任方式 | 判令赔偿、停止销售使用、促成和解 | 2倍惩罚性赔偿、6.4亿余元，提高侵权成本 |
| 法治价值 | 保护原始创新，维护发明人和设计人利益，激发创新活力，营造公平竞争环境 | 法院判决打击违法行为，促使企业依法诚信经营，维护公平竞争环境，促进市场经济健康发展；强化知识产权保护，使企业更愿意投入研发；公正司法弘扬核心价值观 |
| 劳动者角度 | 可作为企业合规补充 | 劳动者有保守秘密义务，在就业中坚持权利义务统一、严守职业道德 |

课堂处理：专利和技术秘密不能混为“知识产权”。看到“公开换保护”要想到专利；看到“不为公众知晓、保密措施、商业价值、非法获取”要想到技术秘密和不正当竞争。

### 9. 人民法院保护知识产权护航新质生产力

题源：2026朝阳一模18，细则。  
设问：结合材料，运用《法律与生活》知识，谈谈人民法院如何依法保护知识产权以护航新质生产力发展。

回填到框架位置：知识产权保护、纠纷解决、惩罚性赔偿、恶意诉讼规制、法治化营商环境。

细则化答题链：

| 司法措施 | 关键词给分 | 材料到意义 |
|---|---|---|
| 特邀第三方调解 | 调解1分；提高诉讼效率/降低诉讼成本1分 | 高效化解知识产权纠纷，维护权益，激发创新积极性和效率 |
| 惩罚性赔偿 | 惩罚性赔偿/提高违法成本1分；维护市场环境/秩序1分 | 针对故意侵权，提高违法成本，保护创新主体核心利益 |
| 规制恶意诉讼 | 驳回谴责/遏制不诚信诉讼1分；诚信/公平原则或核心价值观1分 | 避免“假维权”拖垮“真创新”，维护诉讼秩序和经营秩序 |
| 综合法治保障 | 保护知识产权人合法权益1分；兼顾公共利益/法治环境1分 | 为新质生产力发展提供稳定、可预期的法治环境 |

课堂处理：措施题必须写“法院做了什么+怎样保护+产生什么效果”。只写“保护知识产权、推动新质生产力”是空话。

### 10. AI虚拟主持人/直播：人格权、消费者权益、不正当竞争三线并行

题源：2026延庆一模18（1），8分，细则。  
案例：M公司未经李某允许，用AI合成李某形象开展虚假直播营销，不当攫取流量。

回填到框架位置：人格权、消费者知情权、不正当竞争、诉讼请求与事实理由。

细则化答题链：

| 层次 | 给分点 |
|---|---|
| 违法行为3分 | 人格权角度：侵犯肖像权、名誉权；消费者权益角度：侵犯知情权；竞争秩序角度：虚假和引人误解商业宣传，构成不正当竞争 |
| 诉讼请求1分 | 停止侵害、赔偿损失、承担诉讼费用、赔礼道歉 |
| 事实与理由 | 未经许可合成形象用于虚假直播营销，不当攫取流量；形象对应肖像权，虚假宣传影响社会评价对应名誉权；违背诚信原则 |
| 分析质量1分 | 事实与侵权分析要对应，不能只列权利名称 |

课堂处理：AI题不是单纯“新技术要监管”。法律作答要拆成“人格权+消费者+竞争秩序”，并且学会写起诉书式的诉讼请求和事实理由。

### 11. 消费者隐私权与虚假宣传

题源：2026海淀一模相关主观题，细则。  
案例A：商家在社交平台公开消费者包间监控录像片段、网络聊天信息。  
案例B：赵某虚假宣传有“治疗糖尿病并发症”等功效，夸大功效、诱导消费。

回填到框架位置：隐私权、消费者权益、惩罚性赔偿、诚信原则。

细则化答题链：

| 案例 | 给分链 |
|---|---|
| 包间监控隐私 | 支持原告诉讼请求1分；消费者在经营者包间内活动具有私密性；公开监控录像、网络聊天信息构成隐私权侵害，应承担停止侵权、赔礼道歉等责任；同时侵犯消费者合法权益，3分 |
| 虚假宣传功效 | 经营者应保护消费者知情权；信息应真实、全面，不得作虚假或引人误解宣传；夸大治疗功效诱导消费构成欺诈，应退款并承担惩罚性赔偿；不符合诚信等核心价值观，3分 |
| 失分边界 | 写“满足人民幸福感、获得感、安全感”不给分 |

课堂处理：消费者权益题不能总写“安全消费权”。监控、聊天记录公开，落点是隐私权；功效夸大诱导消费，落点是知情权、欺诈、惩罚性赔偿。

### 12. 家庭、赡养、继承：从精神赡养到诉讼时效

题源A：2024西城一模18，8分，答案。  
题源B：2024东城一模19，答案/细则。  
题源C：2026通州期末12，选择题。  

回填到框架位置：亲属关系、赡养义务、遗赠扶养协议、诉讼时效。

答题链：

| 题源 | 材料抓手 | 知识与答点 |
|---|---|---|
| 2024西城一模18 | 老人有退休金，法院未支持赡养费，但向三个儿子发《督促履行义务告知书》 | 成年子女对父母有赡养义务；赡养包括经济供养、生活照料、精神慰藉；告知书保护老人合法权益，引导孝老敬亲，发挥法律对道德建设的促进作用 |
| 2024东城一模19 | 欠款诉讼时效届满败诉；赡养费请求不适用诉讼时效 | 诉讼时效督促当事人积极主张权利、推动纠纷解决、节约司法资源；赡养关系到老年人生存权利，成年子女有赡养义务，不适用诉讼时效有利于保护老人、彰显公序良俗 |
| 2026通州期末12 | 遗嘱后又签遗赠扶养协议，保姆履行扶养义务 | 遗赠扶养协议为双务合同，效力优先于遗嘱；扶养人履行义务后有权主张约定财产 |

课堂处理：家庭继承题不能只靠道德话术。必须给出法律关系：赡养义务的内容、诉讼时效例外、遗赠扶养协议优先效力。

### 13. 土地权属与三权分置

题源：2025海淀期末19，6分，答案/细则；2026东城期末选择题讲评。  

回填到框架位置：财产权、用益物权、土地承包经营权、农村土地“三权分置”。

细则化答题链：

| 角度 | 给分点 |
|---|---|
| 权利角度2分 | 确定财产归属；保护农民合法权益、土地承包经营权、农民财产权或用益物权 |
| 纠纷解决2分 | 定分止争，减少土地纠纷 |
| 具体意义最多2分 | 促进土地经营权有序流转、优化资源配置、提高土地利用效率、增加农民收入、促进农村经济发展、推进乡村振兴、实现共同富裕、落实“三权分置”改革 |
| 错误边界 | 写“农民的人身权”“农民土地所有权”、照抄材料、套党政哲学模块不给分 |
| 2026东城期末延伸 | 土地所有权归集体，不得流转；可流转的是承包经营权或经营权 |

课堂处理：看到农村土地，先写“所有权集体所有、承包权稳定、经营权可流转”。不要把“经营权流转”写成“所有权流转”。

### 14. 劳动争议与竞业限制

题源A：2025丰台期末19，教师版答案。  
题源B：2025海淀期末21，答案。  
题源C：2026东城期末选择题劳动关系。  

回填到框架位置：劳动者权利义务、劳动合同解除、竞业限制、劳动关系成立。

答题链：

| 题源 | 材料抓手 | 答题点 |
|---|---|---|
| 2025丰台期末19 | 尹某岗位调整后消极怠工、无故旷工，购物中心多次提醒后解除劳动合同 | 劳动者享有权利也应完成劳动任务、遵守劳动纪律和职业道德；用人单位依规合法解除不违法；裁判保护合理用工自主权，引导劳动者权利义务统一，构建健康劳动关系 |
| 2025海淀期末21 | 竞业限制保护商业秘密，但普通员工不当然适用 | 竞业限制是对负有保密义务劳动者自由择业的合理限制；保护知识产权、维护公平竞争；同时不能过度侵犯劳动者基本劳动权利；只对负有保密义务人员有效 |
| 2026东城期末选择题 | 未签劳动合同、违法解除 | 劳动关系自实际用工之日起成立；未签书面合同不必然导致劳动关系不成立；违法解除劳动合同应承担经济赔偿 |

课堂处理：劳动题不要只站劳动者一边。北京题近三年反复考权利义务统一、企业合理用工与劳动者权益平衡。

## 三、选择题错项回填

| 错项类型 | 题源 | 错在哪里 | 正确处理 |
|---|---|---|---|
| 债权客体误写成物 | 2024西城一模6 | 债权关系的客体不是机动车本身，而是给付行为 | 质押涉及担保物权，对应交换价值；交付质物不转移所有权，质权人可依法优先受偿 |
| 物权确认生效时点混乱 | 2024东城一模9 | 认为不办登记就影响法院判决导致的物权变动，或把房屋与土地法律关系等同 | 法院生效法律文书导致物权变动，注意不动产登记的一般规则和例外 |
| AI生成作品一概无著作权 | 2024门头沟一模12 | 把AI当作者，或认为未登记就不受保护 | 看是否有人类独创性投入；著作权一般自作品完成自动产生，登记不是保护前提 |
| 公共场所一律无过错担责 | 2024朝阳一模10；2026丰台一模20 | 把经营场所安全保障义务写成无过错责任 | 经营者、管理者通常因过错担责，且安全保障义务有合理限度 |
| 惩罚性赔偿误写成普通违约责任 | 2024石景山一模11；2024西城二模16；2026朝阳期末直播手串 | 看到买卖合同就写违约责任，漏掉消费者欺诈/食品安全的特殊规定 | 按材料区分违约责任、欺诈惩罚性赔偿、约定“假一赔十”合同条款 |
| 侵犯知情权与自主选择权混淆 | 2026东城期末选择题 | 强制捆绑安装写成侵犯知情权 | 捆绑同意重点是侵犯自主选择权；虚假/不全面信息才优先落知情权 |
| 专利与商业秘密混淆 | 2026东城期末选择题；2024房山一模17 | 把专利写成“不为公众知晓”，或把正当破译一律写侵权 | 专利是公开换保护；技术秘密看是否不为公众知悉、是否采取保密措施、取得手段是否合法 |
| 动产所有权与著作财产权混淆 | 2026东城期末选择题 | 受赠画作后以为可复制、汇编、发行 | 交付转移动产所有权，不等于著作财产权转让；公开展览、复制、汇编、发行需授权 |
| 回避申请与调解/和解混淆 | 2025海淀二模18 | 模拟法庭中把“回避申请”写成起诉、撤诉、和解、调解 | 程序性权利要精准：回避针对审判人员/书记员等可能影响公正审理的情形 |
| 商标恶意抢注写成普通侵权 | 2025海淀二模18 | 未对应“老字号、在先权利、恶意抢注” | 证据证明老字号/知名度/在先权利，再写抢注、损害在先权利、违反诚信或公平原则 |
| 赡养只写给钱 | 2024西城一模18；2024东城一模19 | 漏掉精神慰藉和照料义务 | 赡养包括经济供养、生活照料、精神慰藉；请求支付赡养费不适用诉讼时效 |
| 土地所有权可流转 | 2025海淀期末19；2026东城期末选择题 | 把土地经营权流转写成土地所有权流转 | 农村土地所有权归集体；承包权稳定，经营权可依法流转 |
| 劳动关系以书面合同为成立前提 | 2026东城期末选择题 | 未签劳动合同就写劳动关系不成立 | 劳动关系自实际用工之日起成立 |

## 四、框架重写后的课堂顺序

1. 先上“民事问题四步法”：定主体、定权利、定行为、定责任。所有案例都按这四步回到框架。
2. 合同模块用2024石景山一模17和2026东城期末18打底：一个训练要约承诺，一个训练合同与侵权分流。
3. 侵权模块用2026丰台一模20、2024丰台一模17、2026东城期末18组对比：无责、有责减轻、违约侵权并行。
4. 知识产权模块用2024海淀一模19、2024房山一模17、2025房山一模19、2026朝阳一模18串成“权利类型-侵权/竞争-责任方式-创新价值”。
5. 消费者模块用2024西城二模16、2026海淀一模、2026朝阳期末直播手串把“知情权/自主选择权/公平交易权/隐私权/惩罚性赔偿”拆开。
6. 家庭继承模块用2024西城一模18和2024东城一模19补足具体答点：精神赡养、赡养费时效例外、遗赠扶养协议。
7. 劳动模块用2025丰台期末19、2025海淀期末21、2026东城期末选择题讲平衡：劳动者权益、用工自主、保密与竞业限制。
8. 诉讼模块用2025海淀二模18和2026东城期末18维权准备收束：程序、证据、诉求、时效。

## 五、必须保留在学生版框架里的“短句答案库”

| 高频触发词 | 学生应写出的短句 |
|---|---|
| 报价、订单、付款、确认中标 | 要约与承诺达成一致，合同成立；主体适格、意思表示真实、形式内容合法，合同有效 |
| 交付不符合约定商品 | 未全面履行合同义务，构成违约，应承担赔偿损失等违约责任 |
| 造成身体损害 | 生命权、身体权、健康权受法律保护；结合行为、损害、因果、过错或责任类型判断侵权责任 |
| 产品缺陷造成损害 | 生产者产品责任可适用无过错责任；销售者责任一般看过错，不能混用 |
| 公共场所摔倒 | 安全保障义务有合理限度，经营者/管理者通常因过错担责 |
| 虚假宣传、夸大功效、假一赔十 | 侵犯消费者知情权、公平交易权；构成欺诈的承担惩罚性赔偿；自愿承诺可成为合同条款 |
| 公开包间监控、聊天记录 | 以公开、泄露等方式侵害隐私权，应承担停止侵权、赔礼道歉等责任 |
| 未经许可使用视频、替换标识、加注商标 | 构成著作权侵权；虚假和引人误解的商业宣传构成不正当竞争 |
| 专利、技术秘密同时出现 | 专利公开换保护；技术秘密看保密性、商业价值、保密措施和取得手段是否合法 |
| 恶意抢注老字号商标 | 证明老字号/知名度/在先权利，再写抢注损害在先权利，违背诚信或公平原则 |
| 父母赡养 | 成年子女有赡养义务，包括经济供养、生活照料、精神慰藉 |
| 遗赠扶养协议与遗嘱冲突 | 遗赠扶养协议是双务合同，扶养人履行义务后，协议效力优先于遗嘱 |
| 土地权属登记、三权分置 | 确定财产归属、定分止争；土地所有权集体所有不流转，经营权可依法流转 |
| 未签劳动合同 | 劳动关系自实际用工之日起成立 |
| 竞业限制 | 保护企业商业秘密和公平竞争，但仅能合理限制负有保密义务的劳动者 |
| 维权准备 | 选择调解、仲裁或诉讼；固定证据；明确法律依据和诉求；注意诉讼时效和不告不理 |
