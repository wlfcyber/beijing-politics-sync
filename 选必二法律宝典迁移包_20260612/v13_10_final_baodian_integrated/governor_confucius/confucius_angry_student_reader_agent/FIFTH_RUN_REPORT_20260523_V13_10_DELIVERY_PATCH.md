## One-Page Student Memory Map

My memory after one read:

1. First ask: this is what real-life conflict?
2. If it is minor consumption, false advertising, refund, prepayment, go A9; A1 only helps with capacity and ratification.
3. If it is privacy, personal information, reputation, portrait, health, go A2 even if there is a consumer scene.
4. If it is shared area, neighbor, light, passage, owners' decision, go A3; procedure and fire safety are usually support or boundary.
5. If it is promise, delivery, quality, delay, refund, penalty, go A4; if defective goods hurt body/property, add A6.
6. If it is technical secret, trade secret, data grabbing, confusion, commercial disparagement, go A5; labor departure is background unless the labor issue itself is asked.
7. If it is hiring, probation, dismissal, wages, non-compete, social insurance, go A8.
8. If it asks evidence, mediation, arbitration, litigation, path, use A10/B6 only after entity responsibility unless path is the main question.
9. B-axis decides the answer shape: table is fact-rule-conclusion; responsibility is relationship-rule-fact-liability; claim support is support/partial/not plus reasons; value is protected object-behavior-order; path is evidence-path-request; issue identification is short phrases.
10. Stop where the question stops. Do not perform for the teacher by writing huge value slogans when the question asks responsibility.
