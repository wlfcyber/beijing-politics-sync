## 1. Verdict

v13.8 终于像一个框架了，不再只是“我把 42 题归好类你自己悟”的答案手册。它有总流程、A 轴入口、B 轴动作、边界决策、最小答案形状，这些东西是学生考场能调出来的。

但我还是要骂：它能帮我把题放进法律场景，也能逼我按设问写答案形状；可它对“实体规则的细化层级”仍然不够稳。比如消费者欺诈、未成年人交易、商业秘密、劳动试用解除、物权共同决定这些点，框架给了硬句，但如果题目追问更细，我还是要靠常识和猜。它不是失败，但还没有到闭眼套用的程度。

我的总判断：**真实框架，带修补通过**。

## 3. One-Page Student Memory Map

我读完一遍后能记住的是：

1. 先看真实生活冲突：谁和谁争什么利益，不要一看法院、程序、民事行为能力就被 A10 或 A1 抢走。
2. A 轴是法律内容入口：
   - A1：行为能力、意思表示、代理、时效、效力，只在直接问效力时当主入口。
   - A2：人格权，隐私、个人信息、名誉、肖像、生命健康。
   - A3：物权和相邻关系，业主、共有部分、通行、采光、通风、安全。
   - A4：合同，成立、生效、履行、违约、返还、违约金。
   - A5：知识产权和竞争秩序，作品、商标、商业秘密、数据、虚假宣传、商业诋毁。
   - A6：侵权责任，人身财产损害、安全保障义务、产品损害、工作人员、公共场所。
   - A7：婚姻家庭继承，夫妻、亲子、扶养、遗嘱、遗赠扶养协议。
   - A8：劳动关系，劳动者与单位、试用期、简历、录用条件、解除、竞业限制。
   - A9：消费者权益，消费者和经营者、知情权、公平交易、安全保障、格式条款、欺诈。
   - A10：纠纷解决和诉讼程序，只有题目直接问证据、调解、仲裁、诉讼、维权路径才主导。
3. B 轴是设问动作：
   - B1 表格/裁判要点：按列写事实-规则-结论。
   - B2 判决/责任理由：法律关系-规则-事实-结论-责任。
   - B3 诉求能否支持：逐项写支持/不支持。
   - B4 评析观点：合理处、片面处、正确立场。
   - B5 意义价值：保护对象、约束行为、形成秩序、治理价值。
   - B6 维权路径：争议焦点、权利、证据、协商调解仲裁诉讼、请求。
   - B7 法律问题识别：短句化，不展开成长篇理由。
4. 答案要“事实进规则”，不要空喊法治价值。
5. 问责任就停在责任，问意义再写价值，问路径就写主体证据路径请求。

## 6. Framework Repairs Required

最高优先级修补：

1. 增加“混合设问排序规则”：先实体关系与责任，再程序/路径；先主入口完整跑完，再副入口补充；价值题只在题目问价值时收束。
2. 给高频节点加二级小开关：
   - A9 未成年人消费：年龄/金额/监护人知情追认/平台注意义务/过错分担。
   - A4+A6 瑕疵商品：合同责任、产品损害、人身财产损失、可得利益、证据和可预见性。
   - A5 商业秘密：秘密性、价值性、保密措施、不正当获取使用、竞争秩序损害。
   - A8 试用期解除：录用条件、直接相关信息、严重不实、考核事实、解除依据。
   - A3 共有部分：业主共同决定、合法程序、相邻影响事实、安全规范。
3. 每个 A 轴给一个“主副入口抢夺判定句”：这比触发词更重要，因为考场最怕两个入口都像。
4. B1 增加“表头不完整/要求参照示例”的答题形状。
