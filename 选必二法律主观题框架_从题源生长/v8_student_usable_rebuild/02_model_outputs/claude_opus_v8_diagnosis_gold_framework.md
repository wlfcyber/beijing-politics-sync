### 金样板 8

## 3. 推荐学生版框架

设计原则：不按教材目录（不分"民法总则/物权/合同/侵权/婚姻继承"）；不堆证据编号；不写抽象大词；按"设问要的成品"组织。共 1 个分诊节点 + 6 个动作节点。每个节点固定七栏。学生测试时只看本框架，不看逐题答案，应能独立推出答案。

### 节点 1｜设问分诊台（先挡边界，再认交付物）

节点名称：设问分诊台
看见：设问的最后一句、最后一个动词——"谈谈/分析/评析/完成……/说明……理由/能否得到支持/如何调解/意义/作用"；以及材料主语是不是"国家/最高法/涉外/依法行政"。
先判：①边界——这题是不是私人之间的权利义务、责任救济、程序路径？若设问主语是国家治理、涉外法治建设、依法行政，先挡出去，不套本框架。②交付物——设问要的成品是哪一种：表态、补理由、评行为、说判决理由、给路径、说意义。
再转：把设问动词翻成动作节点编号——"能否支持/是否构成/是否有效/支持谁"→节点 2；"完成/补充裁判理由、完成下表"→节点 3；"评析……的行为"→节点 4；"说明……判决/裁决理由、涉及的法律责任及依据、说明……成立的理由"→节点 5；"如何维权/如何调解/作为法官提供指导"→节点 6；"意义/现实意义/社会价值/作用/法理依据"→节点 7。
这样写：动笔前在草稿角写一行"交付物=____，主节点=____"。劳动就业信号（招聘、解除、平台用工、竞业限制、就业歧视、社保）额外标"先判身份"。多设问题（如 5 分 + 3 分）按分值各写一块。
别这样写：别一上来就背"以事实为依据、以法律为准绳"；别看到"法院"就默认写判决书三段；别把多个设问揉成一问；别把《逻辑与思维》《政治与法治》题当本框架题做。
代表题：CC0137（完成裁判理由）、CC0305（能否支持）、CC0002（法理依据+意义）。
边界：OPEN_OR_REFERENCE 五题（CC0040、CC0162、CC0311、CC0353、CC0380）只用于练分诊、不当任何节点的标准范例；CC0364 是《逻辑与思维》题、CC0223 标"综合"，分诊后不进法律动作节点。

### 节点 2｜先表态

节点名称：先表态
看见：设问里有"能否得到支持/是否构成/是否有效/是否侵权/支持谁的请求"。
先判：先在心里给一个明确的"是/否"，并锁定"谁对谁、几个诉求、几条独立侵害"。多主体题逐个表态，结论可以相反；单主体多侵害题按事实分层。
再转：表态写成第一句、独立成行——"×××的诉讼请求应予支持（不予支持）。"每个表态后接"法律规则一句 + 本案事实一句 +（必要时）责任/结果一句"。
这样写：表态 → 规则 → 本案事实 → 责任/结果。劳动题先判：是不是劳动者身份、什么用工关系、权利有没有被合法限制。
别这样写：别先写意义再表态；别用"这件事比较复杂"开头；别只写"支持"不写规则和事实；别把刑事、行政责任硬塞进民事题。
代表题：CC0305（单主体多层：隐私 + 欺诈）、CC0373（多主体分别表态：闫某支持、张某不支持）。
边界：表态题不是意义题，结尾最多一句价值收束，不展开三层意义；OPEN_OR_REFERENCE 题不作本节点代表题。

### 节点 3｜完成裁判理由 · 一格一句

节点名称：一格一句
看见：题干给了表格、给了"示例行"，或设问是"完成/补充裁判理由""阅读材料完成下表"。
先判：每个空格、每条裁判理由是独立赋分单元；先看"示例行"的句子颗粒度照着填；判这格要的是定性、规则还是结论。
再转：一格写满"主体 + 行为定性 + 法律规则 +（结论/责任）"，一句一格，不跨格抄。
这样写：先写法律依据（这类权利/合同受什么法保护）→ 再写本案谁的什么行为构成侵权/违约 → 再写应承担什么责任。
别这样写：别写"维护双方合法权益""保护公民权利"这种不分主体的话（被显式扣分）；别让价值套话挤占关键词；别一格塞三句；别跨格搬模板。
代表题：CC0137（AI 著作权格 + 信用卡违约格）。
边界：表格容量有限，不在格子里写三层意义；非表格的"说明判决理由"走节点 5，不走本节点。

### 节点 4｜评析双主体行为

节点名称：评析双主体
看见：设问是"评析×××和×××的行为"，材料里有两个及以上行为主体，各做了一件关键事。
先判：先把每个主体的关键行为单独拎出来，逐个判合法/违法及性质（违约？侵权？不正当竞争？违反劳动纪律？）。
再转：每个主体写"行为定性（违反什么原则/什么法）+ 应承担什么责任"，主体之间分句分写。
这样写：主体 A 行为 → 定性 → 责任；主体 B 行为 → 定性 → 责任；末句写行业/市场秩序导向（如"市场主体应依法诚信经营，劳动者应依法履行义务"）。
别这样写：别把两个主体合并成"他们都违法了"；别只评违法不写责任；别漏掉某个主体；别把评析写成意义题。
代表题：CC0238（乙公司商业诋毁+违约、张某违反劳动纪律）。
边界：评析题要落到责任，不停在道德评价；价值收束只一句；不以"是/否"开头（那是节点 2）。

### 节点 5｜案件定责四步（含劳动就业先判身份分支）

节点名称：定责四步
看见：设问是"说明……判决/裁决的理由""分析本案涉及的法律责任及法律依据""说明……合同成立的理由"。
先判：先判这是什么法律关系——合同、侵权、相邻、消费欺诈、知识产权/竞争、劳动用工。劳动就业分支先判：劳动者身份、用工关系（劳动关系/劳务/平台从属性）、有没有就业歧视或竞业限制。
再转：走四步——①定性（谁的什么权利/什么行为）→ ②援引规则（受什么法、什么原则保护或约束）→ ③嵌入本案事实（材料具体事实对上规则）→ ④落责任或结果（承担什么责任、判决怎么定）。
这样写：一个争议点走完四步再写下一个；合同题先证"合同成立有效"再说违约责任；侵权题先定"侵害什么权利"再说侵权责任；劳动题先把身份/用工关系坐实再定责。
别这样写：别列法考式构成要件清单（不逐条写"主体/客体/主观/客观"）；别把违约和侵权混在一句里；别只引法条不对材料事实；别把劳动题塞进普通侵权链。
代表题：CC0054（合同成立要件验证 → 民法原则）、CC0238、CC0373（劳动分支）。
边界：高中限度——四步够用，不展开过错推定、无过错责任等法考责任形态；OPEN_OR_REFERENCE 题不进此节点做标准范例。

### 节点 6｜维权与调解 · 路径三件套

节点名称：路径三件套
看见：设问是"如何维权/需要做哪些准备/如何调解/作为法官提供专业指导"。
先判：先判这是"维权准备"还是"调解方案"。维权题判：要主张什么实体请求。调解题判：双方各让什么、依据什么原则、举证责任落到谁。
再转：维权写"实体请求 + 证据准备 + 程序路径（协商→调解→仲裁→诉讼，按材料纠纷类型选）"；调解写"调解结果（双方各承担什么）+ 依据原则（诚信/公平/全面履行）+ 本案事实 + 举证责任归属"。
这样写：路径按材料给的纠纷类型选；调解给双方都安排一个动作，不息事宁人，也不写成判决书。
别这样写：别笼统写"通过法律途径解决";别把上诉、行政诉讼塞进普通民事纠纷；别把调解写成"谁对谁错"的判决。
代表题：CC0125（DIY 商品"仅退款"纠纷调解）。
边界：路径题不是意义题；公益诉讼/司法确认是另一类，不套普通停止侵害+赔偿模板。
边界：OPEN_OR_REFERENCE 题不作本节点代表题。

### 节点 7｜价值从规则长出来

节点名称：价值从规则长出来
看见：设问是"意义/现实意义/社会价值/典型意义/作用/法理依据"。
先判：先判要几层、几分（如"法理依据 5 分 + 现实意义 3 分"就分两块）；价值必须从本案的具体法律规则里推出来。
再转：每写一句价值，先带一条具体法律规则或本案事实，再说它"保护了谁 → 解决了什么冲突 → 形成了什么秩序"。意义可按对个人/对司法/对社会（或对当事人/对行业/对司法/对法治社会）分点。
这样写：法理依据题先讲规则为什么这样定，再讲意义；典型案例意义按角度分点，每点先规则后价值。
别这样写：别裸写"维护社会公平正义、推动法治中国建设"（无规则铺垫不得分）；别写"优化营商环境"等被判错的表达；别整段抽象价值不带材料。
代表题：CC0002（法理依据 + 现实意义）、CC0103（知识产权典型案例意义）。
边界：意义题也要落到本案规则；AI 产业宏观影响类属开放容器，不当稳定满分模板；本节点核心代表题只用 PASS / PASS_RECOVERED 题。

### 4.1 重写 CC0137｜完成裁判理由 · 一格一句

题源：`CC0137_2025_昌平_二模_20`
证据等级：formal / 框架运行 PASS（strict v8 PARTIAL，本次按 v8 学生版重写）
设问：阅读材料，运用《法律与生活》知识，完成裁判理由。（6 分）
材料核心：表格题，两行案件，法院"处理结果"列已给作基线。①李某用 AI 绘图软件制作图片、加"AI 绘画"等标签发在自己社交平台；网友刘某直接拿来作配图、抹去平台署名水印；李某诉至北京互联网法院——法院支持李某诉讼请求。②小王申请信用卡时声明已阅知并愿遵守领用合约及申请表全部条款；后透支消费、经银行多次催告仍不还款；银行诉请还本金 2 万元及利息——法院判小王还本金 2 万元并按约付息。

**一、框架入口**
节点 1 分诊：设问动词"完成裁判理由" → 交付物 = 补理由，主节点 = 节点 3"一格一句"。两格独立赋分（每格 3 分），先看法院已给的"处理结果"列作基线——它告诉你结论方向，你补的是理由。

**二、最小必要判断**
- 格①：法院已"支持李某诉讼请求"。要判：李某对 AI 生成图有没有著作权？刘某行为侵犯什么权利？定性——这是著作权侵权题，理由走"法律依据 → 侵权定性 →（鼓励创新）"。
- 格②：法院已判"小王还本金及利息"。要判：信用卡领用合约是不是有效合同？小王不还款属什么？定性——这是合同违约题，理由走"合同有效 → 违约定性 → 违约责任"。
- 两格各 3 分、互不通用，不能把格①的著作权话搬到格②。

**三、材料事实到法律语言**

格①：

| 材料事实 | 转成法律语言 |
|---|---|
| 李某用 AI 软件制图、加"AI 绘画"标签、发在自己平台 | 著作权人限于自然人、法人或非法人组织，AI 不能成为著作权主体；该图片具备"智力成果"和"独创性"要件，是受著作权法保护的作品，著作权归李某 |
| 刘某直接拿来作配图、抹去平台署名水印 | 未经许可使用 + 去除署名标识，侵害了李某的信息网络传播权和署名权，构成侵权 |

格②：

| 材料事实 | 转成法律语言 |
|---|---|
| 小王申请时声明已阅知并愿遵守全部条款 | 领用合约系双方真实意思表示、内容不违反法律规定，合法有效，对双方均具有约束力 |
| 透支消费、经多次催告仍不还款 | 未按约定履行还款义务，构成违约，违背诚信原则和全面履行原则 |

**四、法律语言到细则点**
（对照该题 rubric_text 评分细则）

格①（3 分）：
1. 著作权法规定著作权人限于自然人/法人/非法人组织，李某使用 AI 制作的图片具备智力成果与独创性要件、受著作权法保护——1 分（法律依据）。
2. 刘某未经许可作配图使用且抹去水印，侵害信息网络传播权和署名权（二者择一）——1 分。
3. 结合材料说出"李某虽用 AI 但作品具独创性、享著作权"，或落到"鼓励创新"——1 分。

格②（3 分）：
1. 合约系真实意思表示、合法有效、对双方有约束力（替代：权利义务相统一/公平原则/协商一致）——1 分。
2. 小王透支经催缴仍不还构成违约（替代：应承担违约责任）——1 分。
3. 违背诚信原则和全面履行原则，应承担违约责任（替代可写诚信核心价值观，但必须出现"诚信"二字）——1 分。

**五、满分句**
- 格①满分句："李某使用人工智能软件制作的图片具备'智力成果'和'独创性'要件，受著作权法保护；著作权人限于自然人、法人或非法人组织，著作权归属于李某而非 AI。刘某未经许可将其用作配图并抹去署名水印，侵害了李某的信息网络传播权和署名权，构成侵权，应承担侵权责任。"
- 格②满分句："小王与银行签订的领用合约系双方真实意思表示、内容不违反法律规定，合法有效，对双方均具有约束力。小王透支消费且经多次催告仍不还款，构成违约，违背了诚信原则和全面履行原则，应承担违约责任。"

**六、完整考场答案**
格①：关于 AI 作品的权利归属，著作权人限于自然人、法人或非法人组织，AI 不能成为著作权主体。李某使用人工智能软件制作的图片具备"智力成果"和"独创性"要件，受著作权法保护，李某享有著作权。刘某未经许可将其作为配图使用且抹去署名水印，侵害了李某的信息网络传播权和署名权，属于侵权行为，应承担侵权责任。法院判决保护著作权、鼓励创新。
格②：双方签订的领用合约系双方真实意思表示、内容不违反法律规定，合法有效，对双方均具有约束力。小王使用信用卡透支消费且经多次催缴仍不还款的行为构成违约，违背了诚信原则和全面履行原则，应承担违约责任。

**七、易错路径**
- 把格①写成"保护双方合法权益"——不分主体、不出关键词，按 CC0084 同类被显式扣分。
- 在格①纠结"AI 能不能当作者"展开法考式主体资格论证——一句点明"AI 非著作权主体、著作权归李某"即可，重心是李某享有著作权。
- 格②只写"小王不还钱不对"，漏掉"合同合法有效"这个前提——丢第一个赋分点。
- 格②写了"违约"却没出现"诚信原则"——细则要求必须出"诚信"。
- 跨格抄套话、把价值收束塞满格子，挤掉关键词。

**八、迁移提醒**
所有"完成裁判理由/完成下表"题都走节点 3：先看示例行颗粒度 → 判这格要定性还是结论 → 一格一句"主体 + 行为定性 + 法律规则 + 责任"。著作权题统一句式："具备智力成果 + 独创性 → 受著作权法保护 → 未经许可使用/抹去标识 → 侵害署名权与信息网络传播权 → 侵权责任"。合同违约题统一句式："真实意思表示 + 合法有效、对双方有约束力 → 未履行义务构成违约 → 违背诚信与全面履行原则 → 违约责任"。

### 4.2 重写 CC0305｜先表态 · 单主体多层侵权

题源：`CC0305_2026_海淀_一模_18_3`
证据等级：formal / core_recovered_split / 框架运行 PASS_RECOVERED（设问经回补；strict v8 FAIL，本次按 v8 重写修复该逐题运行）
设问：判断是否支持原告诉讼请求，并说明理由（细则要求覆盖隐私权、消费者知情权、虚假宣传/欺诈、侵权责任、惩罚性赔偿）。
材料核心：服务消费纠纷。消费者在经营者提供的包间内活动具有私密性；商家公开了包厢内监控录像片段、网络聊天信息。经营者赵某虚假宣传产品"具有治疗糖尿病并发症等功效"，夸大功效诱导消费。

**一、框架入口**
节点 1 分诊：设问动词"判断是否支持原告诉讼请求" → 交付物 = 表态 + 理由，主节点 = 节点 2"先表态"。注意：这是单一原告对商家，但材料含两条彼此独立的侵害事实（公开隐私信息 + 虚假宣传欺诈）——表态之后要按事实分两层。

**二、最小必要判断**
- 先给明确表态：支持原告诉讼请求。
- 锁定两条侵害线：①商家公开包厢监控录像 + 网络聊天信息 → 隐私权；②赵某夸大治疗功效诱导消费 → 消费者知情权 + 虚假宣传/欺诈。
- 定性：这是民事侵权 + 消费欺诈题，责任落在停止侵害、赔礼道歉、退款、惩罚性赔偿，不涉刑事、不涉行政。
- 两层各自独立配规则与责任，不能合写。

**三、材料事实到法律语言**

| 材料事实 | 转成法律语言 |
|---|---|
| 消费者在包间内活动具有私密性；商家公开包厢监控录像片段、网络聊天信息 | 私密空间、私密活动、私密信息受隐私权保护；任何组织或个人不得以刺探、侵扰、泄露、公开等方式侵害他人隐私权；商家擅自公开的行为构成对消费者隐私权的侵害 |
| 赵某虚假宣传产品具有治疗糖尿病并发症功效、夸大功效诱导消费 | 经营者应保障消费者知情权，提供真实、全面的信息，不得作虚假或引人误解的宣传；赵某夸大功效诱导消费，构成欺诈 |

**四、法律语言到细则点**
1. 表态"支持原告诉讼请求"——独立赋分点，必须第一句独立写。
2. 隐私权层：援引"不得以刺探、侵扰、泄露、公开等方式侵害隐私权" + 商家公开行为构成侵害 + 承担停止侵害、赔礼道歉等侵权责任。
3. 欺诈层：援引"经营者保障知情权、提供真实全面信息、不得虚假宣传" + 赵某夸大功效构成欺诈 + 退款并承担惩罚性赔偿责任。

**五、满分句**
- 表态句："原告的诉讼请求应予支持。"
- 隐私层满分句："任何组织或者个人不得以刺探、侵扰、泄露、公开等方式侵害他人的隐私权。消费者在包间内的活动具有私密性，商家擅自公开包厢监控录像和网络聊天信息，构成对消费者隐私权的侵害，应承担停止侵害、赔礼道歉等侵权责任。"
- 欺诈层满分句："经营者应当保障消费者的知情权，提供真实、全面的信息，不得作虚假或者引人误解的宣传。赵某夸大产品治疗功效、诱导消费，构成欺诈，应当退还货款并承担惩罚性赔偿责任。"

**六、完整考场答案**
1. 支持原告诉讼请求。
2. 任何组织或者个人不得以刺探、侵扰、泄露、公开等方式侵害他人的隐私权；消费者在包间内的活动具有私密性，商家公开包厢监控录像和聊天信息，构成对消费者隐私权的侵害，应承担停止侵害、赔礼道歉等侵权责任。
3. 经营者应当保障消费者的知情权，提供真实、全面的信息，不得作虚假或者引人误解的宣传；赵某夸大治疗功效、诱导消费，构成欺诈，应当退还货款并承担惩罚性赔偿责任。

**七、易错路径**
- 不先表态，直接讲隐私权知识——丢独立表态分，阅卷找不到落点。
- 把隐私和欺诈混成一段——两条侵害事实、两套规则、两种责任，必须分层，混写必漏关键词。
- 隐私层只写"侵犯隐私"不写责任形式（停止侵害、赔礼道歉）；欺诈层只写"欺诈"不写"惩罚性赔偿"——细则按责任点给分。
- 把"惩罚性赔偿"写成"三倍赔偿"硬数字——材料未给倍数，写"惩罚性赔偿责任"即可，不擅填数字。
- 收尾拉一段"维护社会公平正义、推动法治中国建设"——表态题不展开三层意义。

**八、迁移提醒**
所有"是否支持/是否构成/是否有效/能否得到支持"题走节点 2：先独立成行表态 → 每条侵害或争议单独一层 → 每层"法律规则 + 本案事实 + 责任"。单主体多事实题（一个原告多条侵害）按事实分层；多主体题（如 CC0373 闫某/张某）按主体分别表态、结论可相反。隐私权题统一句式："私密空间/活动/信息受保护 → 不得刺探、侵扰、泄露、公开 → 停止侵害、赔礼道歉"。消费欺诈题统一句式："经营者保障知情权、禁止虚假宣传 → 夸大或隐瞒构成欺诈 → 退款 + 惩罚性赔偿"。
