# 选必二法律框架重写规格 - 学完先前框架后的版本

## 1. 先判定：上一版为什么烂

上一版最大问题不是证据少，而是教学前台不成立：

- 它像“证据分类表”，不像“学生上手框架”。
- 它给了入口名，但没有把“案件争点”放到第一层。
- 它给了模板句，但没有充分训练“材料事实如何翻译成法律语言”。
- 它让学生看到 65 题逐题演练，却没有形成强烈的“我以后遇到陌生题也这样切”的感觉。
- 它没有学到先前框架最强的“先导页 + 基本模型 + 主体积累 + 情境触发 + 错法改法”。

## 2. 新版必须采用的主结构

学生版新版目录必须改成：

1. 法律主观题到底在考什么。
2. 一张总模型：生活冲突 A -> 法律判断 B。
3. 30 秒读题法：圈主体、抓行为、定争点、找结果。
4. 一二三四五总图。
5. 主体法律动作积累页。
6. 五类材料世界：不是教材目录，而是材料世界。
7. 情境触发链：看到什么材料，翻译成什么法律词。
8. 满分句生成法：关系句、争点句、规则句、材料句、结果句、价值句。
9. 全 65 题逐题演练。
10. 错法改法与禁用表达。

## 3. 法律版“一二三四五”

一件事：

- 用法律把生活冲突处理成一个明确结果。

两条线：

- 事实规则线：谁、做了什么、触发什么规则、产生什么结果。
- 价值收束线：保护谁、规范什么秩序、弘扬什么价值。

三问：

- 判什么：支持吗、有效吗、侵权吗、违约吗、谁举证、走什么程序。
- 凭什么：哪个材料事实对应哪个法律规则。
- 怎么得分：最后落责任、效力、归属、救济、意义。

四步：

- 圈主体；
- 抓行为；
- 定争点；
- 落结果。

五类材料世界：

- 交易与用工；
- 财产与家庭；
- 人身与侵权；
- 创新与竞争；
- 程序与公共/新技术。

## 4. 逐题演练必须换格式

每一道题固定格式：

### 题源

年份、区、阶段、题号。

### 材料一句话

用“起因 -> 经过 -> 结果”压成一个完整小故事。

### 这题争什么

一句话写清争点。

### 第一判断

学生必须先判什么；如果不判，会写偏到哪里。

### 材料翻译表

| 材料事实 | 翻译成法律语言 | 对应得分句 |
| --- | --- | --- |

### 满分句

必须是高三学生能写出来的自然句，不是后台标签。

### 完整答案

按考场语言写完整，不出现证据等级、模型术语、backend 字段。

### 错法改法

错误写法 -> 为什么错 -> 改成什么。

### 迁移

以后看到什么题也这样切；什么题不能乱用。

## 5. 主体积累页必须先于逐题演练

新增“谁出现，先写什么”：

| 主体 | 一出现先想 | 可直接写的句子 |
| --- | --- | --- |
| 法院/仲裁委 | 支持或不支持请求，理由必须是事实+规则 | 人民法院/仲裁机构应以事实为根据、以法律为准绳，结合双方关系、行为性质和法律后果作出处理。 |
| 经营者 | 真实、诚信、提示说明、不得欺诈 | 经营者应依法诚信经营，真实全面提供商品或服务信息，保障消费者知情权、自主选择权和公平交易权。 |
| 消费者 | 依法、诚信、有证据地维权 | 消费者依法享有维权权利，但应遵循诚信原则，不得滥用权利损害经营者合法权益。 |
| 用人单位/平台 | 是否管理、控制、支付报酬，是否依法解除 | 判断是否形成劳动关系，不能只看协议名称，应结合人格、经济和组织从属性。 |
| 劳动者 | 如实说明、遵守纪律、商业秘密边界 | 劳动者应遵守劳动纪律和职业道德，履行与劳动合同直接相关的如实说明义务。 |
| AI/平台企业 | 提示、审核、保密、权利边界 | 平台和企业应对算法、数据、作品和用户权益尽到必要提示、审核、保密和合规义务。 |
| 当事人 | 程序、证据、请求 | 当事人维权应选择适当纠纷解决路径，围绕合同、付款、损害、因果关系和请求收集证据。 |

## 6. 情境触发链必须有“翻译词典”

新版必须把高频材料信号翻译成法律语言：

- 付款、出票、中标通知 -> 要约、承诺、合同成立。
- 材质不符、虚假宣传、费用不清 -> 欺诈、知情权、自主选择权、惩罚性赔偿。
- 派单、签到、奖惩、结算、业务组成 -> 人格从属性、经济从属性、组织从属性。
- 包厢监控、聊天记录、公开发布 -> 隐私权、个人信息、停止侵害、赔礼道歉。
- 名称图标近似、引人误认 -> 混淆、不正当竞争、停止侵害、赔偿。
- AI 承诺、AI 幻觉 -> AI 不具备民事主体资格、提示义务、注意义务、损害和因果。
- 表格空格 -> 看示例、按格子填，不写散文。
- 起诉状 -> 诉讼请求、事实经过、法律理由。

应改成：

> 你先学习 `PRIOR_FRAMEWORK_DEEP_DNA_20260520.md`，再看 65 题 evidence compact。不要先输出框架。先输出“法律框架应该如何模仿先前框架的教学结构”，然后再给候选目录和样章。

也就是说，GPTPro/Claude 的第一步必须是“学习优秀框架结构”，不是直接开始法律归纳。

## 8. 新版验收标准

一个完全不会的聪明学生读完后，应能做到：

- 看到陌生题先说出：这题争什么。
- 把材料至少 3 个事实翻译成法律语言。
- 第一段写出关系、规则、事实、结果。
- 第二段只从本案推出价值，不写空泛必修三。
- 遇到程序题、表格题、AI题不乱套实体责任模板。

如果做不到，继续退回改。
