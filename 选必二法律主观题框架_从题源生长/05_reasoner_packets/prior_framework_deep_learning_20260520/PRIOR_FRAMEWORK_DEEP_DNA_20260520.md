## 2. 真正值得学的不是“名字”，而是“先导结构”

先前框架最强的第一层不是目录，而是先导页。

典型写法：

- 先问“这本书在考什么”。
- 再给“备考建议”。
- 再给“核心方法”。
- 最后才进入框架。

这和我前面 V4 的失败正相反：V4 开头虽然有口诀，但仍然像把证据分类压成学生语言；它没有先把学生的真实恐惧讲清楚。

法律版必须先导成：

- 法律主观题在考什么：不是背法条，而是把生活冲突判成法律结果。
- 学生最容易死在哪里：不敢表态、不会翻译材料、价值空泛、规则和事实两张皮。
- 核心方法：把案件小故事压成“主体关系 -> 争点事实 -> 规则条件 -> 法律后果 -> 价值收束”。
- 框架只服务这件事：让学生从陌生材料启动并写出得分句。

## 3. “基本模型 A -> B”是先前框架的灵魂

经济习题课第一页不是直接列知识点，而是先画：

`A -> B`

并说明：

- 针对 A 应如何；
- 针对过程应如何；
- 针对 B 为什么；
- 学生需要训练的是拆解设问的能力；
- 学生需要积累的是角度与术语。

法律框架也必须有自己的 A -> B：

`生活事实 A -> 法律判断 B`

中间必须有一条可训练链：

`主体 -> 行为 -> 争点 -> 规则 -> 结果`

如果学生不知道 A 是什么、B 是什么，就算背七个节点也不会用。

## 4. “五四三二一”式压缩值得学

必修二框架不是散列知识点，而是把经济学压成一个可背总图：

- 五：五大发展理念；
- 四：生产、分配、交换/流通、消费等环节；
- 三：基本经济制度；
- 二：政府和市场两只手；
- 一：总目标/高质量发展主线。

这类压缩不是为了漂亮，而是为了让学生记住“全书地图”。

法律版不能再写成七条同级路线。更适合压缩成：

- 一：一件事，定分止争，把生活冲突判成法律结果。
- 二：两条线，事实规则线 + 价值收束线。
- 三：三问，判什么、凭什么、怎么落分。
- 四：四步，圈主体、抓行为、定争点、落结果。
- 五：五类材料世界，交易用工、财产家庭、人身侵权、创新竞争、程序公共。

但注意：这只能是学生地图，不能替代 65 题证据。

## 5. “主体积累页”比 V4 句库更像学生可用工具

经济“应如何类考题-主体积累页”按主体列句：

- 政府；
- 市场；
- 企业；
- 国家；
- 消费者/劳动者等。

每条句子不是抽象框架，而是能直接写上卷面的主体责任句，且带频次。

法律版应该照这个做“主体法律动作积累页”：

- 法院/仲裁委：以事实为根据、以法律为准绳，支持/驳回请求，定分止争。
- 经营者：真实全面告知、诚信经营、不得欺诈、不得用不公平格式条款。
- 消费者：依法维权、诚信维权、保留证据，不得滥用惩罚性赔偿。
- 用人单位/平台：从属性管理、劳动保护、依法解除、不得违法限制择业。
- 劳动者：如实说明、遵守劳动纪律、保守商业秘密。
- 平台/AI企业：提示、审核、保密、合规使用数据和作品，划清责任边界。
- 当事人：选择协商、调解、仲裁、诉讼，保存证据，明确请求。

V4 的句库太像分类词表，缺少“主体一出场，学生就知道写哪句”的积累页。

## 6. 先前法律好稿的可取处：每题有“争点”

旧《选必二法律框架踩分逻辑》最值得保留的结构是：

`材料一句话（起因-经过-结果） -> 核心争点 -> 细则怎样解决争点 -> 作答逻辑怎样解决争点`

这比 V4 的“设问、入口、材料抓手、答案骨架”好，因为它先把题目变成一个需要解决的冲突。

法律题真正的启动点不是“属于哪个节点”，而是：

- 这件事到底争什么？
- 谁的请求要不要支持？
- 哪个事实决定法律结果？
- 细则为什么奖励这个句子？

新框架必须把“争点”放到前台。

## 7. 旧法律好稿的错误也必须避开

旧稿有三处不能直接继承：

1. 把选择题、主观题混在同一套统计里，容易污染主观题框架。
2. 把旧的未锁细则/参考答案也做成强结论，和当前证据等级规则冲突。
3. 有些“全书框架”太像大而全目录，学生读起来容易记不住。

因此新法律框架只能学它的结构，不继承它的旧结论。

## 8. 先前框架的语言风格

好框架的语言不是“教师审计语言”，而是“学生脑内操作语言”：

- “看到……先想……”
- “先别写……”
- “这一题不是问……而是在问……”
- “材料里的 X，要翻译成 Y。”
- “第一句必须先判……”
- “最后一句只能从本案推出……”

V4 的问题是用了太多“入口、材料抓手、必须带词、答案骨架”，但句子没有足够像老师在课上把学生脑袋掰正。

## 9. 先前框架的排版顺序

可迁移的顺序应为：

1. 先导：这类题在考什么。
2. 总模型：A -> B / 一句话总口诀。
3. 速通：3 天、7 天、30 天分别怎么学。
4. 核心动作：少数动作，能直接启动。
5. 主干地图：一二三四五压缩。
6. 主体积累页：谁出现，就写什么法律动作。
7. 情境触发页：看到什么材料，翻译成什么法律语言。
8. 逐题演练：每题必须有“起因-经过-结果、争点、规则、满分句”。
9. 易错改法：错误答案 -> 为什么错 -> 改成什么。
10. 教师证据附录：证据等级、题号、细则原子放后面，不污染学生前台。

## 10. 对当前法律重写的硬结论

下一版不能再从“节点”开始，而要从“法律题的基本模型”开始：

`生活冲突 -> 争点 -> 法律翻译 -> 法律结果 -> 价值收束`

并且逐题演练必须改成：

- 材料一句话；
- 争点一句话；
- 学生第一判断；
- 材料翻译表；
- 满分句；
- 完整答案；
- 错法改法；
- 归入主干还是容器。
