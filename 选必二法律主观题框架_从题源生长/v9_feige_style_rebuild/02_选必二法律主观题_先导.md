# 先导：选必二法律主观题在考什么

## 一、题型特点

选必二法律主观题，不是背法条。

也不是法考。

更不是必修三法治建设。

它考的是：

> 把生活纠纷，翻译成法律得分点。

材料说的是人话：

- 买东西被骗了。
- 平台说不是劳动关系。
- 商家用了别人的标识。
- 邻居影响生活。
- 法院支持了谁的诉求。
- 双方都觉得自己有理。

答案要写成法律话：

- 谁享有什么权利。
- 谁负有什么义务。
- 谁违反什么规则。
- 谁承担什么责任。
- 为什么这样处理有价值。

飞哥想说：

- 法律题不是让你骂人。
- 法律题不是让你喊口号。
- 法律题是让你把“这事不对”说成“哪条法律关系不对”。

## 二、主观题核心能力

第一，看设问要你交什么。

- 问理由：你要证明为什么支持。
- 问评析：你要判断行为对不对。
- 问意义：你要说规则保护了谁、规范了谁。
- 问怎么解决：你要给路径，不是直接判案。
- 问诉求能不能支持：先表态，再说法理和事实。

第二，从材料抓四件事。

- 谁？
- 做了什么？
- 争什么？
- 造成什么后果？

第三，把普通话翻译成法律话。

- “商家说得不真实” → 虚假宣传、欺诈、侵害知情权。
- “平台天天派活打卡考核” → 可能存在劳动关系。
- “用了别人的作品” → 可能侵犯知识产权。
- “公开别人隐私” → 侵害人格权。
- “合同签了不履行” → 违约、诚信、全面履行。

第四，把法律话写成阅卷喜欢的话。

不是只写名词。

要写成：

> 法律规则 + 材料事实 + 结论责任。

必要时再加：

> 保护谁、规范谁、维护什么秩序。

## 三、最核心方法

两个词：

正向触发。

反向筛查。

正向触发：

> 看到什么材料，就想到什么法律答法。

反向筛查：

> 看起来像某模块，也要先筛掉错误方向。

例子：

- 看到法院，不等于写公正司法。
- 看到企业，不等于写经济生活。
- 看到法治，不等于写全面依法治国。
- 看到案例，不等于写法考构成要件。
- 看到意义，不等于最后喊一句“有利于依法治国”。

## 四、考场底线

没有材料事实，不写法律词。

没有法律规则，不写价值。

不把法律题写成道德评价。

不把意义题写成全面依法治国套话。

不把高中题写成法考分析。

飞哥想说：

- 选必二最怕裸写价值。
- 价值要从规则里长出来。
- 先处理纠纷，再谈意义。
- 先定权利义务责任，再说社会价值。
