## 二、学生可用性判断

**20 秒启动是否够用？** 够用。六步链条"圈主体 → 抓行为 → 看设问 → 定主卡 → 写第一句 → 规则+材料+落点"是动作链，不是知识链，零基础学生照着能动笔。比 V5 系列早期版本多了"看设问"这一步，正好解决"不知道进哪张卡"。唯一的破绽是第 4 步"定主卡：一题只选一张主卡，最多加一张辅卡"和后面 27 个例子不一致（见 P1-1）。

**六张动作卡是否能让学生选入口？** 能。六卡表给了"什么时候用 / 第一笔怎么写 / 最容易丢什么"三栏，触发词清楚（表格→一格一答；是否支持/判决依据→分关系·定责任；知识产权→认产权·抓侵权；维权调解→排维权步骤；风险边界→划风险边界；意义价值→推价值）。问题不在卡本身，在 27 核心示范没按"一主一辅"落地。

**27 道核心题逐题示范是否像考场答案？** 像。每题结构为"入口 / 这题先判断什么 / 满分前检查 / 考场满分答案 / 可以拆成这些满分句 / 必踩关键词 / 易错刹车"。"考场满分答案"是连贯段落，"可以拆成满分句"把它拆成采分点，"易错刹车"逐题给反例。这是真正可学、可背、可迁移的结构。证据对账证明内容也准（见第三节）。

**38 道非核心保分索引是否能防止学生乱套？** 能。每题先标类型（低频规则题 / 设问需教师再核对题 / 普通答案练习题 / 综合边界题 / 转出本书题），再给"最低可写三句 + 别这样写 + 能不能背"。综合边界题统一写"其他模块另开段落，不能把本卡当全题答案"；reference_only 统一写"不支撑核心框架，只练思路"；转出本书题写"应转到《逻辑与思维》"。学生即使遇到陌生非核心题，也能写出保分三句而不空卷、不滑向必修三大作文、不误把参考题当满分模板。

**"这题先判断什么"是否足够具体？** 核心题足够具体。例如核心题 1"先判断这是好意同乘：乙某是否无偿搭载、是否有故意或重大过失、甲某未系安全带能否支持减责"；核心题 11"先判断未成年人充值是否被监护人追认，再分别看平台审查义务和监护人责任"。都是可执行的判断动作，不是空话。

**是否还能看到后台证据语言或审计语言？** 看不到。27 核心答案无"法理依据：/现实意义：/为必踩点/原答案："等后台标签，无悬空"，。"，无"（，多选一）"留白，无试卷页眉。v5_3 审计 0 命中，我逐题通读确认。这是 V5.7 相对 V5.2 最大的进步。非核心保分卡也是干净的三句话，没有审计残留。
