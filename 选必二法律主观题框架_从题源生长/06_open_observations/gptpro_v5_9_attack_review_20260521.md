**PASS_WITH_MAJOR_REWRITE**

### C. 结构重构建议

#### 1. 学生第一眼应该看到什么

V6 第一眼不应先看“六张动作卡”，应先看：

```text
先看设问尾词，再决定答案动作。
```

建议第一页开头直接放这张诊断表：

| 设问尾词                | 第一动作         | 进入模块   |
| ------------------- | ------------ | ------ |
| 是否支持、能否支持、是否有效、是否构成 | 先表态          | 表态定责   |
| 为什么、理由、依据           | 规则 + 材料事实    | 法理依据   |
| 有什么意义、社会价值、典型意义     | 先写本案处理，再推价值  | 本案到价值  |
| 怎么办、如何维权、应准备什么      | 证据 + 请求 + 路径 | 证据请求路径 |
| 完成表格、补充笔记、补全裁判理由    | 一格一答         | 表格逐格采点 |
| 涉 AI、涉外、国家治理、逻辑推理   | 先降级判断边界      | 边界保险箱  |

#### 2. 第一页怎么重排

V6 第一页建议顺序：

1. 考场 6 秒：看设问尾词。
2. 考场 20 秒：圈主体、抓行为、定争议。
3. 设问尾词诊断树。
4. 答案公式：结论句 + 规则句 + 材料句 + 落点句。
5. 三条禁令：不裸写法治口号，不展开法考构成，不把非核心当核心。
6. 红线题降级规则：SOURCE-CHECK、REFERENCE-ONLY、BOUNDARY、TRANSFER 只保分。

六张动作卡放到第二页，作为诊断树后的执行工具。

#### 3. 六张动作卡是否保留

保留，但建议改名，更贴近学生临场动作。

| ------- | ------ | ----------------- |
| 一格一答    | 表格逐格采点 | 学生一眼知道每个空单独拿分     |

| 认产权·抓侵权 | 产权与竞争  | 覆盖知识产权、不正当竞争、数据权益 |
