# Codex A 初步攻击审查：V5.9 为什么仍不像“学生看完就会满分”

时间：2026-05-21 05:31 CST

状态：等待 GPTPro、Claude Opus 4.7、本地 student-agent 三方攻击报告前的 Codex A 初判。此文件不是最终结论，只作为 V6 重构工作台。

## 总裁定

V5.9 证据守门基本安全，但学生端仍不是满意成稿。它像一份“已经清洗过的逐题示范索引”，不够像一套能让学生在陌生题里自动启动的强框架。

如果现在直接发给学生，学生可能会背 27 道核心题答案，却仍然不能稳定处理陌生题。原因不是答案不连贯，而是“从题目进入框架”的动作还没有被训练成条件反射。

## 主要失败点

1. **开头仍像状态说明，不像战斗入口。**
   V5.9 第一屏先解释 candidate、27+38、双审门禁。证据纪律对，但学生打开第一页最需要的是“我第一眼圈什么、第一句写什么”。

2. **六张动作卡太平，缺少优先级。**
   表格、责任、产权、维权、风险、价值被并列摆放，但真实题常常混合：法院裁判 + 法律责任 + 意义，合同 + 侵权 + 维权，知识产权 + 创新价值。学生不知道谁是主动作、谁是附加层。

3. **“分关系·定责任”吞了 15/27 核心题，已经过宽。**
   这一卡覆盖好意同乘、平台劳动、合同成立、消费欺诈、竞业限制、AI 幻觉、生成式 AI 判决等。它安全，但不像分诊工具，学生会觉得“反正大多数题都选它”。

4. **价值题仍缺防空话机制。**
   V5.9 说“价值必须由本案推出”，但没有把“谁受益、规范谁、弘扬什么”做成逐格填空。学生仍可能写成“维护公平正义、促进社会和谐”的通用句。

5. **27 核心题是好示范，但没有被再提炼成题型训练路径。**
   当前是 27 个连续案例。学生读完会记住例题，但未必形成“同类陌生题怎么迁移”的动作。

6. **38 非核心索引太像保险柜，不像备用路。**
   source-check/reference/boundary 红线是必要的，但“最低三句”还不够教学生在低频题里保分。它告诉学生别乱升核心，却没有足够强的替代打法。

7. **缺少错答改写。**
   先前框架的强处不是只给正确答案，而是把学生会错在哪里讲狠。V5.9 每题有“易错刹车”，但多数只有一条，缺少“错误答案 -> 为什么丢分 -> 改成满分句”的训练。

8. **教师版和学生版没有真正分层。**
   V5.9 想同时服务学生、教师、审计。结果学生看到红线和索引，教师又缺少完整课堂脚本。

9. **满分句库没有变成可检索动作库。**
   句子很多，但学生不一定知道何时调用。V6 应按“设问触发词 + 材料信号 + 必写关键词 + 禁用场景”组织。

10. **没有清晰的学习路线。**
    学生应该按“先学总入口 -> 练 6 个母题 -> 做错答改写 -> 再刷 27 核心 -> 最后看 38 防误区”学习。V5.9 直接进入 27 题，节奏太硬。

## V6 必须改成什么

### 第一层：学生战斗版首屏

只保留一条总口令：

> 先刹车，抓设问，判关系，把材料塞进规则，最后落责任或价值。

首屏只回答五个问题：

1. 这题是不是选必二法律核心题。
2. 设问让我判断、说明、维权、评析、意义，还是完成表格。
3. 主法律关系是什么。
4. 第一段第一句怎么写。
5. 满分前检查哪几个硬词。

### 第二层：强主干，不再只并列六卡

建议 V6 改为“1 条总路 + 5 个主战场 + 2 个刹车区”：

1. 总路：判、规、料、果/值。
2. 表格战场：一格一答。
3. 裁判/评析战场：先判谁赢，再写依据。
4. 私法责任战场：合同、侵权、消费、劳动、相邻分链。
5. 创新竞争战场：保护对象、侵害行为、市场秩序。
6. 程序救济战场：路径、证据、请求、效力。
7. 意义价值收束：保护谁、规范谁、弘扬什么。
8. 边界/低频刹车区：只按题内材料保分，不升核心。

### 第三层：每个主战场都必须有四件东西

1. 题目信号：看到什么设问和材料就进来。
2. 第一判断：学生第一秒必须判什么。
3. 满分句模板：可直接落笔。
4. 错答改写：给一条学生烂答案，现场改成满分句。

### 第四层：27 核心题重排

不要按年份连续排。改为按主战场分组：

- 表格题组。
- 裁判评析题组。
- 合同/侵权/消费/劳动责任题组。
- 知识产权与不正当竞争题组。
- 程序救济题组。
- 意义价值题组。
- AI/风险边界题组。

每组先给“母题拆解”，再给同组题快速运行。

### 第五层：38 非核心改为“保分附录”

保留红线，但每类要给统一备用打法：

- source-check：按题源最低三句写，不背成模板。
- reference-only：只练思路，不当正式细则。
- boundary：先停核心私法模板，按制度、机制、程序、效果写。
- low-frequency：法条/制度 + 材料事实 + 具体功能。
- transfer：明确转出本书。

## V6 验收标准

V6 不看“有没有 65 题都出现”，先看学生能不能做到：

1. 30 秒内选出主战场。
2. 第一笔能写表态句。
3. 能把材料事实塞进规则，不在后面贴故事。
4. 能区分合同/侵权/消费/劳动/知识产权/程序。
5. 意义题不空喊价值。
6. 边界题不必修三化、不法考化。
7. 27 核心题可接近满分；38 非核心不误升核心但能保分。
