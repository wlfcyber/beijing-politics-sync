# Framework v2 Student One Page

## 先问一句

这题是不是在处理私人法律关系、权利义务、责任承担或维权程序？如果主语是国家治理、涉外法治建设、依法行政，先别套选必二。

如果已经判断为国家治理、涉外法治、依法行政、生态治理、AI责任等综合题，但考场必须作答，就按材料中的制度、机制、程序、效果分层归纳；不要冒充选必二核心满分模板。

## 六个入口

1. 表格题：一格一答，先判断格子要什么。
2. 评析题：先表态，再写法律依据和材料事实。
3. 案例理由题：定性 -> 规则 -> 材料事实 -> 责任/结果。
4. 裁判题：先写以事实为根据、以法律为准绳，再落具体法律规则。
5. 维权建议题：程序路径 + 准备事项 + 实体请求。
6. 意义题：个人合法权益 -> 司法/行业秩序 -> 公平正义或核心价值观。

## 材料翻译

- 看到付款、出票、承诺：翻译成合同成立。
- 看到合同事实和人身/财产损害同时出现：先分开写合同成立/有效/违约，再写侵权事实/损害/因果/责任，不要糊成一段。
- 看到欺诈、隐瞒、搭售、无法清楚知悉、无法拒绝支付：先写合同成立但欺诈导致违背真实意思、合同无效或可撤销，再写消费者知情权/自主选择权/公平交易权和《消费者权益保护法》第五十五条三倍赔偿。
- 看到损害、过错、因果：翻译成侵权责任判断。
- 看到协商未果、起诉、证据：翻译成程序路径和准备。
- 看到劳动合同解除是否合法：先写劳动者义务和解除依据，结尾补用工自主权、劳动者权利义务相统一、劳动关系稳定。
- 看到典型案例/意义：先问受益主体是谁，别空写法治口号。
- 看到作用题、意义题、开放容器题：法条对法条，案例对案例，动词写具体；禁止只写维护公平正义、促进社会和谐。

## 禁止

- 不把必修三法治建设话术当选必二答案。
- 不写法考式大段构成要件。
- 不用参考答案题支撑核心模板。
- 不把开放容器题当已经稳定的满分套路。
