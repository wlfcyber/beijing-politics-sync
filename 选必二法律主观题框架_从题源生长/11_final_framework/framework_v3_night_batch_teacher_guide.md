# 选必二法律主观题框架 v3 教师讲义

## 课型安排

第一课：先刹车、圈主体、表格与评判裁判。
第二课：私法责任链、消费欺诈、混合责任。
第三课：知识产权竞争、程序救济、AI风险合规、意义价值。

## 板书总线

主体 -> 行为 -> 冲突 -> 规则 -> 材料事实 -> 责任/程序/价值。

## 每个节点的课堂问法

### 格子题：一格一答

1. 先问学生：有表格先服从格子
2. 追问材料：表格、填入、表中、机制/理由/措施/意义分栏。
3. 让学生套句：这一格要求说明【格子任务】。材料中【事实】对应【法律规则/程序】，因此应写【最短结论】。
4. 批改错法：跨格写一大段“维护合法权益、促进公平正义”，格子问机制却写意义。

### 评判裁判题：先判后证

1. 先问学生：问是否、判决、认识，先表态
2. 追问材料：是否、支持、不支持、有效、无效、构成、判决、法院、谈谈认识、法理依据。
3. 让学生套句：我认为【结论】。人民法院以事实为根据、以法律为准绳。根据【法律规则】，材料中【事实】符合/不符合【要件】，所以【责任/判决/请求】成立。
4. 批改错法：没有首句表态；只写法条不落到本案；把裁判锚句当成整题答案。

### 私法责任题：切责成链

1. 先问学生：合同、侵权、消费、劳动、相邻都走责任链
2. 追问材料：合同、要约、承诺、履行、违约、侵权、赔偿、消费者、欺诈、劳动、相邻、人格权、生命健康权。
3. 让学生套句：【主体A】与【主体B】之间形成【法律关系】。【行为】符合【规则要件】。材料中【事实】说明【要件成立/不成立】，因此【责任、效力或诉求】应当【支持/不支持】。
4. 批改错法：只背权利名称，不判断法律关系；材料事实贴在后面当故事，没有塞进规则要件。

### 创新竞争题：定行为、护秩序

1. 先问学生：知识产权和竞争题先定行为
2. 追问材料：知识产权、著作权、商标、专利、商业秘密、商业诋毁、混淆、不正当竞争、AI作品、核心代码、创新。
3. 让学生套句：材料中【行为】属于【侵权/不正当竞争类型】，损害了【权利/竞争秩序】。依法应【停止侵害/赔偿/承担责任】，这有利于保护创新成果、规范市场竞争秩序。
4. 批改错法：一看到创新就空写新质生产力；著作权、商标、商业秘密混用；漏法院具体责任。

### 程序救济题：路径、证据、请求

1. 先问学生：问怎么办、如何维权，先走路径
2. 追问材料：调解、仲裁、诉讼、起诉状、证据、举证、诉讼请求、司法确认、强制执行、公益诉讼、和解。
3. 让学生套句：当事人可以通过【路径】解决纠纷，并围绕【争议事实】提交【证据】，提出【请求】。若经法院依法确认，【协议/裁判】具有相应法律效力。
4. 批改错法：把所有题都写成起诉；调解题写成判决；公益诉讼写成个人维权。

### 风险合规题：一险一界一措施

1. 先问学生：AI、数据、平台、合规题先分风险
2. 追问材料：风险、合规、AI、人工智能、数字员工、数据、算法、平台、开源智能体、治理、边界。
3. 让学生套句：材料中的【风险】触及【法律边界/权利义务】。应当通过【制度/合同/告知/授权/审核/责任机制】加以规范，做到【具体效果】。
4. 批改错法：一看到 AI 就写科技伦理；一看到风险就写诉讼赔偿；不把风险和措施一一对应。

### 意义价值题：从本案推出三层价值

1. 先问学生：问意义、作用、价值，最后补三层
2. 追问材料：意义、作用、价值、有利于、弘扬、促进、维护、规范、秩序、公平、正义、诚信、核心价值观。
3. 让学生套句：该处理有利于保护【具体主体】的合法权益，有利于规范【具体司法/行业/市场秩序】，有利于弘扬【诚信/公平/友善/法治】等具体价值。
4. 批改错法：上来空喊公平正义；三层主语重复；价值句没有从本案法律规则推出。
