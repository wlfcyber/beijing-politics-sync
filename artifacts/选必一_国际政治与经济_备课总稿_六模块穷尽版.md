## 备课总抓手

- 先让学生会判模块：
  - 世界怎么了，看 `时代背景`
  - 国家为什么合作，看 `理论`
  - 货、钱、技术、规则怎么流，看 `经济全球化`
  - 国际秩序往哪走，看 `政治多极化`
  - 全球治理靠什么平台和原则，看 `联合国`
  - 中国怎么做、为什么能做，看 `中国`

- 全书跨模块最高频主点：
  - `构建人类命运共同体（17）`
  - `高水平对外开放（14）`
  - `推动经济全球化朝更加开放、包容、普惠、平衡、共赢方向发展（13）`
  - `共同利益是国家间合作的基础（9）`
  - `贡献中国智慧和中国方案（9）`
  - `促进贸易投资自由化便利化（9）`
  - `共商共建共享的全球治理观（8）`
  - `时代主题：和平与发展（8）`
  - `坚持多边主义（7）`
  - `遵循联合国宪章宗旨和原则（7）`

- 课堂组织的主顺序：
  - 先讲 `时代背景`，把“为什么必须合作、治理、回应共同挑战”讲清楚。
  - 再讲 `理论`，把“国家为什么合作、为什么不能脱钩断链、为什么开放和安全要一起写”讲清楚。
  - 然后讲 `经济全球化`，这是最稳的得分大头。
  - 再讲 `政治多极化` 和 `联合国`，把国际秩序和全球治理讲成一条线。
  - 最后讲 `中国`，把前面所有模块收束到中国特色大国外交、中国方案和中国担当。

### 模块定位

- 这个模块主要负责交代国际大势和设问必要性。
- 学生如果只会背“和平与发展”，但不会从材料里判断为什么现在要合作、要治理、要回应共同挑战，就拿不到分。

### 最值得先讲熟的句子

- 这个模块专门处理“中国如何行动、为什么这样行动、对世界有什么意义”。

- `2025东城期末 Q20` 欧盟对中国新能源汽车征税

- `2025西城一模 Q21` 发展理念成为国际公共产品

### 高频出分链

- `中国行动/外交布局 -> 共同利益 -> 中国方案/大国担当 -> 开放合作或多边主义 -> 命运共同体`
- `援助合作/小而美 -> 民生改善 -> 共同发展 -> 正确义利观 -> 全球南方共同利益`
- `经贸摩擦/外部打压 -> 维护国家利益 -> 维护市场秩序 -> 扩大开放 -> 合作共赢`

### 最值得先讲熟的句子

- 一看到中国主语，就先写 `中国做了什么`，不要一上来堆大词。
- 一看到“不附加条件、惠民生、共同发展”，优先写 `正确义利观`。
- 一看到“周边、元首外交、外交布局”，优先写 `中国特色大国外交` 和 `习近平外交思想`。
- 一看到“对外打压、征税、壁垒”，要把 `维护国家利益` 和 `扩大开放` 一起写。

## 备课使用法

- 第一轮备课：
  - 先只讲六模块边界和高频点，不求一次讲全题。
  - 重点让学生会用材料判断模块。

- 第二轮备课：
  - 每个模块挑 `2-3` 道代表题，带学生练“材料 -> 点”。
  - 尤其反复练 `经济全球化`、`联合国`、`中国` 三个高频模块。

- 第三轮备课：
  - 让学生拿一套没见过的新题，先自己判模块，再自己搭出主链。
  - 教师只纠正三件事：模块判错、主点顺序错、材料和术语没对上。

- 最终目标：
  - 学生看到新题，先能在六模块里定位。
  - 定位后能快速调出 `3-5` 个稳定主点。
  - 写答案时能做到 `材料对应 -> 细则语言 -> 总体收束`，而不是空喊口号。
