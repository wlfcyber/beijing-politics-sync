## 使用口径

- 只收`思维部分`，纯`推理规则题`不纳入；若一道题前半考推理、后半考思维，只收后半。

  - `A类`主观题/子问 `33` 道
  - `B类`选择题稳定思维信号 `29` 道
- 证据层级：
  - `A类`：已有细则/评标/讲评，能锁定题号与思维方法。
  - `B类`：题面与答案/讲评足以锁定稳定思维信号。
  - `C类`：当前本地不正式并入的条目，分为`缺源待补`和`边界已排除`两类。

## 零基础怎么用这份逐题库

- 先学`A类`，因为这些题有细则或评标支撑，最适合拿来练“材料 -> 思维”的稳定路径。
- `B类`不要拿来背答案，只拿来练识别：看到什么材料形态，最先排除什么错法。
- 每道题只做四步：
  - 先圈材料里的动作词：`调查、规划、转型、融合、协同、反转`
  - 再判入口：它更像`科学`、`辩证`还是`创新`
  - 再补小方法：`分析与综合/超前/联想/逆向/辩证否定`
  - 最后写成一句完整的话，不要只报名词
- 零基础最低配答题模板：
  - `材料里……`
  - `这说明它是在……地想问题`
  - `所以这里应写……思维`
- 零基础最常见三种错法：
  - 只写方法名，不解释材料为什么能触发它
  - 看见“创新”两个字就乱写`创新思维`
  - 把`科学思维`和`辩证思维`、`创新思维`并列乱报，不知道前者常常是总帽子

## 一、A类：主观题/子问逐题积累

### 2024

### `2024丰台一模 Q19(2)`
- 材料信息：围绕垃圾分类做模拟政协研究，具体落点是`商品包装垃圾分类标识率低`，需要查法条、看样本、补研究方法。
- 触发思维：`科学思维`是总帽子，稳定下沉到`超前思维`、`分析与综合`，也可带出`发散到聚合`。
- 路径积累：一看到`调查商品样本 -> 查标准 -> 补方法 -> 形成提案`，就先写获取客观材料，再写对资料做分析综合，最后写据此判断预测并形成可执行方案。

### `2024海淀一模 Q17(2)`
- 材料信息：京津冀三地各有功能定位和特色优势，但目标是协同发展、优势互补、写成整体新图景。
- 触发思维：`分析与综合`。
- 路径积累：`分别分析三地定位`是分析，`把三地放回区域协同大局`是综合。凡材料是`多主体分工 + 一盘棋收束`，优先写分析基础上的综合。

### `2024东城一模 Q18(3)`
- 材料信息：题眼不是二选一，而是`传统产业不能丢`、`未来产业要提前布局`同时成立。
- 触发思维：`辩证否定 + 超前思维`。
- 路径积累：传统产业部分写`不是简单抛弃，而是扬弃、改造、升级`；未来产业部分写`面向未来需求、技术赛道、产业布局提前落子`。

### `2024石景山一模 Q19(3)`
- 材料信息：企业“走出去”时同时遇到法律、标准、市场、文化等多重摩擦，不是单点问题。
- 触发思维：主轴是`分析与综合`，可辅写`辩证否定`。
- 路径积累：先拆不同冲突类型，再综合成总体应对方案；如果材料强调`既适应当地又保持自身优势`，就补上扬弃和转换。

### `2024西城一模 Q19(5)`
- 材料信息：不是背未来产业名称，而是根据现有数据、国际趋势和产业条件去判断未来方向。
- 触发思维：`超前思维`，并稳定带出`调查研究 + 矛盾分析 + 推理想象`。
- 路径积累：写法顺序最稳的是`立足现实调查 -> 分析内外条件与矛盾 -> 用推理想象把现在延伸到未来`。

### `2024朝阳期中 Q19`
- 材料信息：首发经济“北京朝外样本”不是单一创新，而是业态、品牌、场景、价值链条一起改。
- 触发思维：`超前思维 + 逆向思维 + 联想思维 + 发散与聚合`。
- 路径积累：`业态规划`写超前，`品牌引进与首店逻辑`写逆向，`策展型零售和跨界组合`写联想，`多元素最后收束成新消费场景`写发散到聚合。

### `2024丰台二模 Q18(2)`
- 材料信息：评析“只要准确预测冰雪经济前景，就能推动其高质量发展”。
- 触发思维：先肯定`超前思维`重要，再补`辩证思维`边界。
- 路径积累：这类评析题固定写法是`预测必要但不充分`，再落到`从实际出发、尊重规律、具体问题具体分析、统筹优势与短板`。

### `2024海淀二模 Q17(1)`
- 材料信息：全国时间利用调查强调`全面、真实、准确`，而且调查范围、对象、工具、阶段安排都升级。
- 触发思维：`科学思维`，可拆成`客观性`、`创新性`、`整体性/分析与综合`。
- 路径积累：看到`调查更全、更细、更真`先写客观性；看到`新方法、新工具、新设计`写创新；看到`分阶段实施、任务明确`写整体统筹。

### `2024顺义二模 Q16(2)`
- 材料信息：推进“无废城市”建设，既有现实废物问题，也有资源化、清洁生产、长效治理目标。
- 触发思维：`超前思维`，并带出`矛盾分析法`。
- 路径积累：这类题稳定写`尊重城市建设规律 -> 预判未来废物压力与治理方向 -> 把减量化、资源化、经济与社会效益统一起来`。

### `2024朝阳二模 Q19(1)`
- 材料信息：“生生之宇宙观”强调四时流转、生生不息、变化运行。
- 触发思维：`辩证思维`的`动态性`。
- 路径积累：凡材料重点在`变、流转、生成、发展`，就优先往动态性落，而不是空写“联系”或“全面”。

### 2025

### `2025东城一模 Q18(1)`
- 材料信息：“两重”实施里同时出现`硬投资与软建设`、`当前与长远`、`重点领域与整体布局`。
- 触发思维：`辩证思维`，重点是`整体性 + 动态性 + 重点论`。
- 路径积累：碰到`既要……又要……`而且层次多、周期长的材料，先写统筹当前和长远，再写抓重点带整体。

### `2025丰台一模 Q18(1)`
- 材料信息：国家推动人工智能发展，强调从实际出发、预判趋势、提前部署、分段规划、实践检验。
- 触发思维：`科学思维`，稳定拆成`客观性 + 预见性 + 可检验性`。
- 路径积累：这类题最稳写法是三分法，不要只写“超前布局”；必须把`从实际出发`和`实践检验调整`补全。

### `2025延庆一模 Q18`
- 材料信息：低空经济既有巨大潜力，也有基础薄弱、场景单一、治理配套不足等挑战。
- 触发思维：`辩证思维`，并能带出`质量互变`。
- 路径积累：先写一分为二看潜力与问题，再写由小到大、由散到优的渐进突破，这就是量变到质变的展开方式。

### `2025房山一模 Q16(3)`
- 材料信息：琉璃河遗址如何“热”起来，本质是让沉睡文物、考古资源、历史叙事进入新场景。
- 触发思维：`联想思维 + 发散思维`。
- 路径积累：凡是`遗址/博物馆/文物`怎么出圈，都先想跨界嫁接、沉浸体验、数字化表达，再想多方案生成。

### `2025石景山一模 Q19`
- 材料信息：科学建议奖材料同时出现问题意识、多元方法、实验数据、生活灵感。
- 触发思维：主轴是`科学思维`，可向`创新思维`外溢。
- 路径积累：一看到`建议来自观察、实验、验证、改进`，就写科学思维三特征；若材料强调生活灵感和新办法，再补创新意识。

### `2025门头沟一模 Q21(1)`
- 材料信息：门头沟从“千年煤城”转向“京西智谷”，不是单改产业，而是生态、产业、文旅、科技整体重构。
- 触发思维：`辩证否定 + 整体性/动态性 + 联想/发散 + 超前思维 + 科学思维`。
- 路径积累：这类复合题可按`破旧路径依赖 -> 统筹多系统 -> 跨界嫁接新资源 -> 前瞻布局新赛道 -> 立足现实检验`五步写。

### `2025昌平二模 Q19`
- 材料信息：沉浸式演艺把观众从“看戏的人”变成“入戏的人”，并从经典文本中开发多元场景。
- 触发思维：`逆向思维 + 发散与聚合 + 迁移想象`。
- 路径积累：`观众由被动到主动`写逆向，`从多个角度开发经典`写发散聚合，`营造身临其境新场景`写迁移与想象。

### `2025东城期末 Q18(2)`
- 材料信息：登月服设计把功能需求、月面环境、火箭形象、传统文化元素放在一起统筹。
- 触发思维：`联想/迁移想象 + 发散与聚合 + 超前思维`。
- 路径积累：看到`造型元素跨界联结`写联想，看到`多需求统筹`写发散与聚合，看到`提前预判月面环境`写超前。

### `2025海淀期末 Q18`
- 材料信息：北京城市图书馆从`人找书`转成`书找人`，并把“赤印”等文化意象迁移进公共文化空间设计。
- 触发思维：`逆向思维 + 联想思维`。
- 路径积累：凡材料出现`服务关系倒过来`，先写逆向；凡出现`文化意象转成建筑/空间/场景`，再写联想迁移。

### `2025海淀二模 Q20`
- 材料信息：题面直接围绕`共享发展理念推进共同富裕`展开，材料把`全民共享、全面共享、共建共享、渐进共享`四层一起展开。
- 触发思维：`辩证思维`，稳定落点是`分析与综合 + 整体性 + 动态性/质量互变`，并可补`辩证否定`。
- 路径积累：看到`全民、全面、共建、渐进`同时出现，就先写从各维度拆分共享内涵，再综合到共同富裕整体推进；`渐进共享、从低级到高级、从不均衡到均衡`稳定触发动态发展和量变到质变；若材料强调继承传统共享思想并回应新时代现实问题，还可补扬弃和发展。

### 2026

### `2026东城一模 Q19(1)`

- 触发思维：`辩证思维 + 创新思维`。
- 路径积累：遇到这种“总帽子是科学思维”的题，不要空写定义，要把它拆到`全面联系看问题`和`通过发散聚合改造方案`上。

- 路径积累：先拆机遇和挑战，再统摄成整体判断；再写积累条件、实现突破；最后补“顺势而动、因时而变”。

- 触发思维：`超前思维`。
- 路径积累：超前思维应用题的稳定结构是`调查研究需求 -> 分析现实矛盾 -> 用推理和想象做中长期规划`。

- 触发思维：总帽子是`科学思维`，并可下沉到`辩证思维`、`创新思维`。
- 路径积累：`建立资源库、掌握生存策略、为治理提供依据`先写科学思维追求认识客观性；`没有万能模式、各地因地制宜、构建立体防护体系`再写分析与综合、整体性、具体问题具体分析；若从多路径精准防治继续展开，还可补创新思维或发散与聚合。

## 二、B类：选择题稳定思维信号

### 2024

### `2024朝阳一模 Q4`

## 四、现阶段最好用的课堂落点

- 真正要背的不是方法名，而是`什么材料形态先触发什么方法`。
- `调查/方案/验证`最先调`科学思维`，再细拆到`客观性、预见性、可检验性`。
- `旧资源翻新、旧产业升级、旧空间出新`高频触发`辩证否定`。
- `协同治理、一盘棋、系统工程`高频触发`分析与综合 + 整体性 + 动态性`。
- `文旅融合、文创设计、沉浸式体验`高频触发`联想 + 发散聚合 + 逆向`。
