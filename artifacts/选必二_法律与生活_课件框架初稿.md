## 一、全书总抓手

- 选必二在考什么：`法条基础知识 + 法治意义`
- 这本书的学习主线，课件压成了两句话：
  - `法条知识扎实`
  - `意义角度全面`
- 课件给出的备考建议：
  - 这是一册需要认真阅读教材的书
  - 要熟练掌握法律学科语言
  - 要积累一些“疑难杂症”型案例
  - 要刷选择题
  - 要总结常见的`法治意义`模板

## 二、两条总主轴

### 1. 法条主轴

- 先判断是什么法律关系
- 再判断适用哪一块具体制度
- 重点落点集中在：
  - 民事法律关系与民事权利
  - 合同
  - 知识产权
  - 侵权责任
  - 家庭、婚姻与继承
  - 企业与劳动者、消费者、其他经营者的关系
  - 多元纠纷解决机制与诉讼

一句话定位：
- 选必二的法条线，不是零散背法条，而是形成“`法律关系判断 -> 具体规则套用`”的意识。

### 2. 意义主轴

- 课件反复强调：法律题不能只停在“会背法条”，还要补上`法治意义`
- 这条意义线主要从四个方向展开：
  - 经济意义
  - 社会意义
  - 权利保护与利益平衡
  - 法治价值与德法结合

一句话定位：
- 选必二很多题最后都要从“`为什么这样判/这样做有何价值`”收束，而不是只停在结论。

## 三、六块高频知识板块

### 1. 民事法律关系与民事权利体系

- 民事法律关系三要素：
  - 主体
  - 客体
  - 内容
- 主体部分重点：
  - 自然人
  - 法人
  - 非法人组织
  - 民事权利能力
  - 民事行为能力
- 客体部分重点：
  - 物
  - 行为
  - 智力成果和商业标记
- 内容部分重点：
  - 人身关系
  - 财产关系
- 民事权利体系重点：
  - 人格权
  - 身份权
  - 物权
  - 债权
  - 知识产权
  - 股权等财产性权利
- 课件特别提醒的易混点：
  - `肖像权`与`名誉权`区分
  - 个人信息保护原则：`合法、正当、必要、诚信`

一句话定位：
- 这部分是全书底座，很多案例题第一步都要先回到“谁是主体、争的是什么权利、法律关系是什么”。

细则回填（已核准）：

- `2026西城一模第17题`
  - 这题把`物权`、`绿色原则`和`公共利益`正式接上了
  - 法律依据层要写：`贯彻民法绿色原则` + `为绿色发展提供法律依据/法治保障`
  - 作用路径层要写：`合理划定权利界限`、`强化法定义务`、`明确法律责任与权利救济`
  - 意义层要落到：`平衡个人利益与生态公共利益`、`推动资源开发利用绿色转型`、`促进可持续发展`
  - 这说明物权题后面不能只背权利类型，还要预备“`绿色发展/公共利益`”这一条意义线

### 2. 合同与知识产权

- 合同部分主线：
  - 合同成立：意思表示一致
  - 合同生效：主体适格、意思表示真实、不违法、不违背公序良俗
  - 合同履行：全面履行原则、诚信原则、协作原则
  - 合同变更或解除：原则上要协商一致，不能随意单方变更
  - 违约责任承担方式：
    - 继续履行
    - 采取补救措施
    - 赔偿损失
    - 支付违约金
    - 符合条件时适用定金罚则
  - 免责情由：法定免责重点是`不可抗力`
- 知识产权部分主线：
  - 著作权
  - 专利权
  - 商标权
- 课件在知识产权里强调了：
  - 著作人身权和著作财产权的区分
  - 不同权利的保护期限
  - 专利权、商标权的基本期限记忆

一句话定位：
- 合同题重点看“意思表示、履行、违约”；知识产权题重点看“权利归属、是否未经许可使用、有没有破坏市场秩序”。

细则回填（已核准）：

- `2024海淀一模第19题`
  - 这题把数字知产题的给分层次压得很清楚：
  - 第一层先定`著作权`
  - 第二层再定`侵权`
  - 第三层补`不正当竞争`
  - 第四层再写`激发创新`、`保护创作`、`维护市场秩序`或`公平竞争`
  - 细则特别强调：
  - 只笼统写`知识产权`、`财产权`会掉分
  - 如果把这道题写成`专利权`，直接不得分
  - 所以后面遇到 AI、虚拟数字人、平台传播等材料，必须先把权利种类写准

- `2025丰台一模第19题`
  - 这题把`知识产权保护 -> 创新保护 -> 法治信仰`这条链压得很直
  - 企业层面要先写准：
  - `维护当事人合法权益`
  - `保护知识产权`
  - `严惩侵权行为`
  - 对创新和市场层面要写到：
  - `规范市场竞争秩序`
  - `营造公平市场环境`
  - `以法治力量鼓励创新`
  - 对司法和法治层面还要补：
  - `为类案提供范例`
  - `提升审判效率`
  - `增强司法公信力`
  - `引导公民依法办事`
  - `增强法治信仰`
  - 这说明知产题后面不能只停在“侵权没侵权”，还要会把个案裁判写成`创新生态 + 法治中国`的支撑

- `2025房山一模第19题`
  - 这题把`技术秘密`、`劳动者保密义务`和`惩罚性赔偿`连成了一条完整链
  - 行为定性层要写到：
  - `挖人并窃取技术秘密`
  - `构成不正当竞争`
  - `破坏公平竞争环境`
  - 责任层要写到：
  - `惩罚性赔偿`
  - `提高侵权成本`
  - `强化知识产权保护`
  - `激发创新活力和科技进步`
  - 关系层还要补：
  - 劳动者负有`保密义务`
  - 要体现`权利义务相统一`
  - 意义层可收束到：
  - `公正司法`
  - `维护社会公平正义`
  - 这题提醒我：知产题越来越常和`劳动者义务`、`商业秘密`、`惩罚性赔偿`并题

- `2025石景山一模第20题`
  - 这题是典型的`案件一违约 + 案件二不正当竞争`复合题
  - 案件一要按`法理依据 + 事实依据`写：
  - 被告`超过履行期限`
  - `未提交符合合同约定功能的技术成果`
  - 因而构成违约，应承担`返还研发费用`、`支付违约金`等责任
  - 案件二要按`保障作用`写：
  - `依法打击不正当竞争`
  - `促进公平竞争`
  - `维护市场秩序`
  - `增强市场活力`
  - `营造创新环境`
  - `护航软件产业高质量发展`
  - `培育新质生产力`
  - 这题说明 2025 一模已经出现很稳定的`合同责任 + 创新保护`双案拼接结构

- `2025海淀期末第21题`
  - 这题把`竞业限制`的给分口径拆得很清楚：
  - 规范端要写到：`保护企业商业秘密`、`保护知识产权`、`推动社会创新`
  - 限制端要写到：`保护劳动者基本劳动权利`、`有利于人才资源合理配置`
  - 总结端必须回到：`平衡劳动者合法权益与企业商业秘密保护`
  - 案例端的区分也很关键：负有保密义务的高级管理人员李某违反竞业限制，应承担违约责任；普通厨师刘某不属于适格对象，协议应认定无效

- `2026海淀期末第18题（1）（2）`
  - 先定条款性质：`预先拟定、重复使用、订立时未协商`，所以属于`格式条款`
  - 再定无效原因：该条款限制了承租方`自由订立合同的权利`，违背格式条款中的`公平原则`
  - 意义层面不能只写“保护权益”，细则要求进一步落到：
  - `规范合同内容`
  - `防止格式条款滥用`
  - `保护租赁双方合法权益`
  - `减少合同纠纷`
  - `弘扬诚信的社会主义价值观`
  - `规范住房租赁市场秩序`
  - `推动严格执法、良法善治`

- `2025丰台二模第19题（2）`
  - 这题虽然是 AI 场景，但本质上已经把合同/权利边界题推到了“法律平衡功能”层面
  - 细则要求先总写：法律在`技术创新与社会秩序之间`发挥平衡作用
  - 再写“划红线”侧：
  - `明确 AI 应用边界`
  - `防止技术滥用`
  - `保护创作者合法权益`
  - `维护社会公共利益`
  - 再写“留空间”侧：
  - `为技术创新提供合法路径`
  - `贯彻公平、诚信、守法和公序良俗原则`

- `2025东城二模第19题`
  - 这题是很典型的“一道表格题塞三种民法关系”
  - 案件一“给 13 周岁未成年人文身”：
  - 合同效力线要写：`限制民事行为能力人` + `法定代理人未追认` + `合同无效`
  - 侵权线要写：经营者`未尽核实义务`、`存在过错`，从而侵犯未成年人的`身体权/健康权`
  - 案件三“到店支付却被提前扣款”：
  - 要抓住`格式条款`
  - 公司负有`提示、说明义务`
  - 只在冗长服务条款中约定，不足以发生有效提示效果
  - 这题提醒我：合同板块后面要常备两种高频边界
  - `限制民事行为能力人的合同效力`
  - `格式条款中的提示说明义务`

### 3. 侵权责任

- 侵权可能涉及的权利：
  - 物权
  - 人格权
  - 身份权
  - 知识产权
- 一般侵权责任的核心构成：
  - 合法权益受到损害
  - 行为人主观上有过错
  - 行为与损害结果之间有因果关系
- 归责原则分类：
  - 过错责任
  - 过错推定责任
  - 无过错责任
- 课件点到的高频特殊情形：
  - 饲养动物致人损害
  - 产品责任
  - 环境污染责任
  - 机动车交通事故责任
  - 高度危险责任
  - 建筑物相关责任
  - 医疗产品、器械损害责任
- 侵权责任承担方式：
  - 停止侵害
  - 排除妨碍
  - 消除危险
  - 返还财产
  - 恢复原状
  - 赔偿损失
  - 消除影响
  - 恢复名誉
  - 赔礼道歉
- 侵权也有界限，课件特别提到：
  - 作品的合理使用、法定许可使用
  - 相邻关系

一句话定位：
- 侵权题的标准走法是：`先看有没有损害和因果，再看归责原则，再看责任怎么承担，最后补上公平正义和法治价值`。

细则回填（已核准）：

- `2024丰台一模第17题`
  - 好意同乘题的细则要求先定：驾驶人对搭乘人的生命健康安全负有保障义务，原告受损的是`生命权、健康权`
  - 再定：被告乙某有过错，但并非`故意或重大过失`，因此不支持让其承担全部赔偿责任
  - 最后再进意义层：
  - `维护双方合法权益`
  - `促进社会公平`
  - `维持人际关系和谐`
  - `弘扬社会主义核心价值观`
  - 这说明侵权题不能只停在“构成侵权”，还要写清`为什么减责`

- `2025朝阳二模第20题（1）`
  - 这题把`隐私权侵权`和`相邻关系`绑在一起考
  - 定性层要直接写：`侵犯隐私权` + `应承担侵权责任`
  - 确权层要写：楼道属于双方`共有部分`，双方享有`共有和共同管理的权利`
  - 材料层要写：摄像头拍到日常进出信息，与财产安全、私人生活习惯高度关联，属于`具有隐私性质的人格利益`
  - 行为评价层要写：安装摄像头不能超出合理范围，安装前应尽到妥善注意义务
  - 最后补`相邻关系`原则：`有利生产、方便生活、团结互助、公平合理`
  - 这说明邻里纠纷题不能只写相邻关系，也可能要从人格权侵权切入

- `2026海淀一模第18题（3）`
  - 这题把消费者场景里的两类侵权放在一起了
  - 第一类是`隐私权侵权`
  - 经营者公开包厢监控录像、聊天信息，属于`刺探、侵扰、泄露、公开`他人隐私
  - 可落责任：`停止侵权`、`赔礼道歉`
  - 第二类是`虚假宣传 + 欺诈`
  - 经营者夸大功效，诱导消费，侵犯消费者`知情权`
  - 应承担：`退款` + `惩罚性赔偿`
  - 价值判断上细则还明确要求回到：`诚信等社会主义核心价值观`

### 4. 家庭、婚姻与继承

- 家庭纵向关系：
  - 父母对未成年子女有抚养、教育、监护职责
  - 子女对父母有赡养义务
  - 赡养不仅是经济供养，也包括生活照料和精神慰藉
- 继承部分重点：
  - 继承开始前要先析产，先区分个人合法财产
  - 继承方式包括法定继承和遗嘱继承
  - 继承债务原则上在遗产范围内清偿
  - 分配原则通常先讲一般均等，再结合协商、照顾、贡献等因素
- 婚姻中的夫妻财产制度：
  - 共同财产
  - 个人财产
- 课件列出的高频共同财产：
  - 工资、奖金、劳务报酬
  - 生产经营、投资收益
  - 知识产权收益等
- 高频个人财产：
  - 婚前财产
  - 因人身损害获得的赔偿或补偿
  - 一方专用生活用品
  - 明确只归一方的遗嘱或赠与财产

一句话定位：
- 家庭类题经常是“`权利义务 + 伦理价值 + 法律保障`”一起考，尤其容易连到敬老爱老、德法结合这些价值角度。

细则回填（已核准）：

- `2024海淀二模第19题`
  - 这题不是单纯问协议效力，而是问`判决的社会价值`
  - 细则核心口径是：
  - `确认遗赠扶养协议`
  - `认定居委会已经尽到扶养义务`
  - `充分肯定社区助老养老送终的作用`
  - 意义层要能写到：
  - `实现个案公平正义`
  - `倡导全社会积极助力养老`
  - `促进“不尽孝者少分或者不分遗产”的司法理念深入人心`
  - `弘扬尊老、敬老、爱老、助老的中华传统美德`
  - `推动社会主义核心价值观成为社会风尚`
  - 这题对家庭与继承板块很重要，因为它说明养老、扶养、继承并不只是家庭内部关系，也可能进入社区治理和社会价值层

- `2025东城一模第19题`
  - 这题第三案把`遗产管理制度`、`扶养贡献`和`适当分得遗产`放到了一起
  - 作答时要先确认：
  - 对被继承人尽了较多扶养义务的人，可以`适当分得遗产`
  - 还要写到`遗产管理人`对遗产清理、保管和分割的制度作用
  - 意义层不能只写“照顾好人”，细则要求进一步落到：
  - `倡导尊老敬老、守望相助`
  - `弘扬社会主义核心价值观和中华传统美德`
  - `促进社会和谐`
  - 这说明家庭继承题后面要常备一条线：`法律规则 + 伦理价值 + 社会示范`

### 5. 企业运行中的法律关系

- 课件把企业部分分成三组关系：
  - 企业与劳动者
  - 企业与消费者
  - 企业与企业
- 企业与劳动者：
  - 劳动者维权路径是高频点
  - 基本顺序包括：
    - 协商和解
    - 申请调解
    - 劳动仲裁
    - 诉讼
  - 还会连到劳动保障监察等行政渠道
- 企业与消费者：
  - 企业要依法经营、守法经营
  - 消费者的高频权利包括：
    - 安全消费权
    - 知情权
    - 自主选择权
- 企业与企业：
  - 高概率落到`不正当竞争`
  - 课件点到的典型情形：
    - 混淆行为、搭便车
    - 虚假或者引人误解的商业宣传
    - 虚假交易等

一句话定位：
- 企业专题最常考的是“三组关系分别适用什么规则”，再往上提一层就是“保护谁的权益、维护什么秩序”。

细则回填（已核准）：

- `2025朝阳一模第19题`
  - 平台搭售题的细则要求按`双轨`作答，不能只写一条线
  - 第一轨是消费者权益/侵权线：
  - 抓手材料必须写到：A 公司`擅自搭售 10 元外卖红包`，消费者`不能清楚知悉费用支出细节`且`无法拒绝支付`
  - 结合消保法，可落到：`欺诈`、`知情权`、`公平交易权`、`自主选择权`、`惩罚性赔偿`
  - 第二轨是合同线：
  - 要从`要约、承诺、合同成立`写起
  - 再审查是否因为`欺诈导致意思表示不真实`
  - 再判断应否承担返还、赔偿等责任
  - 这类题以后默认先做“消费者权益线 + 合同效力线”双轨扫描

- `2025西城一模第20题`
  - 这题不是抽象谈`公平与效率`，而是要求先把`工会 + 集体协商 + 集体合同`写出来
  - 公平侧要写：
  - `发挥工会作用`
  - `借助集体力量`
  - `确保双方协商一致、平等自愿`
  - `更好保护劳动者合法权益和劳动报酬`
  - `促进体面劳动`
  - 效率侧要写：
  - `促进企业依法规范用工`
  - `激发劳动者积极性`
  - `构建和谐劳动关系`
  - `提高生产效率`
  - `培育良好商誉`
  - 这说明劳动题后面要默认准备两层答案：`公平保护`和`效率提升`

- `2025东城一模第19题`
  - 这题第二案把劳动争议题的给分口径也压实了：
  - 要写`劳动者权利义务相统一`
  - 既要看到劳动者依法流动，也要看到`维护用人单位合法权益`
  - 最后要落到`构建和谐劳动关系`
  - 这提醒我：服务期、培训、离职类题，不能只站在劳动者一边写“维权”

- `2024西城二模第16题`
  - 这题把`食品安全惩罚性赔偿`和`权利不得滥用`放在了一起
  - 支持首单赔偿这一侧，要写：
  - 经营者`依法、诚信经营`
  - `保护消费者合法权益`
  - 适用食品安全法的`惩罚性赔偿`来打击违法生产经营行为
  - 不支持大量加购赔偿这一侧，要写：
  - `民事主体行使权利不能超过正当界限`
  - `不得滥用民事权利损害他人、社会公共利益`
  - 甲大量加购已经`超出正常生活消费需要`
  - 因而法院不支持该部分惩罚性赔偿请求
  - 总结层要回到：
  - `保证食品安全`
  - `维护生产经营秩序`
  - `平衡两种价值取向`
  - `引导消费者诚信、理性维权`

- `2026海淀期末第19题`
  - 这题把`企业与企业`、`企业与消费者`两条线并到了一起
  - 首先要用`反不正当竞争法`给行为定性：扰乱市场竞争秩序，损害其他经营者或者消费者合法权益
  - 材料落点要写具体：
  - `利用技术手段为用户提供不正当抢票优势`
  - `破坏平台购票规则`
  - `干扰、妨碍平台售票业务正常开展`
  - `损害甲公司信誉及其他合法权益`
  - 所以构成`不正当竞争`，并应承担侵权责任
  - 同时还要补消费者和价值层：
  - `损害消费者合法权益和长远利益`
  - `违背经营者公认的商业道德`
  - `不利于社会整体福祉的增进`
  - `扰乱市场竞争秩序`

- `2025东城二模第19题`
  - 案件二的细则也值得固定下来：
  - 自媒体在没有事实依据的情况下侮辱、诽谤企业，造成社会评价降低，属于`侵害名誉权`
  - 典型意义不能只写“赔礼道歉”，还要写到：
  - `引导自媒体经营者树立法治意识`
  - `规范其行为`
  - `提升媒体公信力`
  - `维护良性市场竞争秩序`

- `2025西城期末第19题`
  - 这题用“小爱同学”把`混淆`、`虚假宣传`和`数字经济法治环境`连起来了
  - 评分参考要求先写行为定性：
  - 明知他人已经使用并有一定影响，仍`抢先注册`
  - 并发布`引人误解的商业宣传信息`
  - 再写法律评价：
  - 构成`混淆`和`虚假宣传`
  - 属于`不正当竞争`
  - 违背`诚实信用原则`
  - 最后写判决效果：
  - `合理确定各方利益`
  - `规制滥用权利行为`
  - `保护科技创新型企业的品牌商誉`
  - 意义收束要能落到：
  - `促进新业态发展`
  - `为数字经济发展营造良好的法治化营商环境`

### 6. 多元纠纷解决机制与诉讼

- 课件强调：纠纷解决不是只有打官司
- 多元纠纷解决机制包括：
  - 和解
  - 调解
  - 仲裁
  - 诉讼
- 调解部分重点：
  - 人民调解
  - 行政调解
  - 仲裁调解
  - 诉讼调解
  - 司法确认使调解协议获得强制执行力
- 仲裁部分重点：
  - 商事仲裁
  - 或裁或审
  - 协议仲裁
  - 自愿、灵活、保密
  - 劳动争议仲裁也要单列掌握
- 课件特别提醒仲裁不适用的情形：
  - 婚姻、收养、监护、扶养、继承等家庭纠纷
  - 行政争议
- 诉讼部分重点：
  - 民事诉讼
  - 行政诉讼
  - 刑事诉讼
  - 诉讼代理人和辩护人的区分
  - 两审终审制

一句话定位：
- 这一块的答题关键，是先判断“这类纠纷最适合走什么渠道”，再写各种机制的功能、边界和程序特点。

细则回填（已核准）：

- `2025东城一模第19题`
  - 这题是一道典型的`一题三案`程序题，不能脱离具体纠纷类型去背程序
  - 第一案要优先想到：
  - `调解`
  - `诉讼调解`
  - `诉讼`
  - 第二案是劳动争议，要看到`劳动仲裁前置`
  - 第三案是继承纠纷，要看到其不适用一般商事仲裁
  - 这题说明多元解纷板块后面一定要和`纠纷类型`一起记

- `2025延庆一模第19题`
  - 这题把`调解`、`诚信维权`、`合同解除`和`证据意识`压到了一道小额网购纠纷题里
  - 程序选择层先写：
  - 本案`标的较小`
  - 可本着`以和为贵`原则适用`调解`
  - 法律评价层要写：
  - 消费者维护合法权益也要遵循`诚信原则`
  - `退款后买卖合同已经解除`
  - 因而应依法退货
  - 诉讼结果层还要写到：
  - 当事人主张商品有问题却又称已将商品扔掉，体现出`证据意识不足`
  - 这部分请求依法难获支持
  - 这题把`诚信维权 + 证据准备`正式推进了多元解纷母题

- `2026东城一模第18题`
  - 这题的细则把司法护创新拆成了三种动作：
  - `诉讼调解`
  - `惩罚性赔偿/严惩知产侵权`
  - `驳回并谴责恶意诉讼`
  - 对应的给分口径分别是：
  - 调解：`推动纠纷实质性化解`、`以和为贵`、`效率高`、`节约司法资源`
  - 惩罚：`增加侵权成本`、`警示规范`、`保护当事人合法权益`、`维护市场秩序和公平竞争`
  - 规制恶意诉讼：`贯彻诚信原则`、`明确权利行使界限`、`维护诉讼秩序`
  - 总结点要能回到：`公正司法`、`以事实为依据、以法律为准绳`、`保护知识产权就是保护创新`

- `2026朝阳一模第18题`
  - 这题的细则也是“措施类”，要求直接写`人民法院做了什么 + 做得怎么样`
  - 第一层：`特邀第三方调解`，并写出`高效化解纠纷`、`提高诉讼效率`或`降低诉讼成本`
  - 第二层：`惩罚性赔偿规定`，并写出`提高侵权违法成本`、`维护创新生态`、`维护公平竞争环境`
  - 第三层：`谴责不诚信诉讼行为`，并写出`诚信/公平原则`，避免“假维权”拖垮“真创新”
  - 总括层：`保护知识产权人的合法权益` + `兼顾社会公共利益` + `提供稳定、可预期的法治环境`

- `2026门头沟一模第18题（1）`
  - 这题不是单纯让你列举解纷方式，而是把`公益诉讼主体合法性`、`多元解纷路径`和`司法确认后的法律效力`压进同一道题
  - 第一层先抓主体：
  - 材料里是`公益社会组织`对涉案企业提起`环境民事公益诉讼`
  - 所以要先写主体合法，并点出其起诉目的指向`公益/公共利益`与`社会责任`
  - 第二层再抓程序：
  - 本案不是单一路径，而是`和解 + 司法调解 + 诉讼`并用
  - 这一步不能只写“可以调解”，而要写出多元纠纷解决机制`功能各有侧重`
  - 第三层必须写效力升级：
  - 法院依法审查和解协议，并出具`民事调解书`
  - 经法院依法确认后，和解或调解协议就具有`强制执行效力`
  - 这题把多元解纷板块又往前推了一步：以后遇到生态公益诉讼、群体性纠纷或公共利益保护题，优先按`主体是否合法 -> 方式是否多元 -> 法院确认后是否刚性执行`这条链组织答案

## 四、这套课件最看重的法治意义角度

### 1. 经济意义

- 激发创新
- 保护创作
- 优化资源配置
- 维护市场秩序、生产经营秩序
- 促进相关行业规范健康发展
- 促进经济高质量发展
- 提高生产积极性、增加收入

### 2. 社会意义

- 保障食品安全
- 减少社会风险
- 化解矛盾纠纷
- 倡导文明生活方式
- 营造敬老爱老等良好社会氛围

### 3. 权利保护与利益平衡

- 维护当事人的合法权益
- 体现权利义务相统一
- 定分止争
- 平衡不同主体利益
- 在保护个体权利的同时维护公共秩序、生产秩序、交易秩序

### 4. 法治价值与德法结合

- 坚持平等、自愿、公平、诚信、守法和公序良俗
- 维护社会公平正义
- 弘扬社会主义核心价值观
- 弘扬中华传统美德
- 坚持德治与法治相结合

## 五、课件中的高频案例落点

### 1. 合同纠纷类

- 重点落在：
  - 合同是否成立、生效
  - 是否全面履行
  - 是否构成违约
  - 违约责任如何承担
  - 如何通过明确合同内容、协商一致来预防纠纷

### 2. 知识产权与不正当竞争类

- 例如“虚拟数字人”案例：
  - 先看是否侵犯著作权
  - 再看是否构成不正当竞争
  - 最后补上创新保护和市场秩序的法治意义

### 3. 侵权责任类

- 例如好意同乘、环境污染、食品安全等案例：
  - 先看侵权责任构成要件
  - 再看是过错责任、过错推定还是无过错责任
  - 特殊情形要注意举证责任分配
  - 最后再落到公平、公序良俗、消费者权益、市场秩序等意义层面

### 4. 家庭与养老类

- 重点落在：
  - 赡养义务
  - 精神慰藉义务
  - 老年人合法权益保护
  - 德法结合
  - 敬老爱老的社会价值

### 5. 基层治理与矛盾化解类

- 例如预付式消费纠纷、基层矛盾化解等材料：
  - 能连到消费者权益保护
  - 能连到行政监管和行业规范
  - 能连到调解、司法确认、诉讼等多元纠纷解决机制
  - 还会连到法治方式、法治思维和基层治理效能

## 六、一版压缩成总逻辑

- 第一步：先判断法律关系，抓住`主体、客体、权利义务`
- 第二步：迅速定位知识板块，看是合同、侵权、婚姻继承、劳动、消费者还是纠纷解决
- 第三步：用规范语言作答，把法条知识写准
- 第四步：补上意义角度，常见收束方向是：
  - 保护合法权益
  - 维护市场秩序
  - 维护社会公平正义
  - 弘扬核心价值观
  - 推动德法结合

## 七、我目前对这份课件的整体判断

- 这套课件的重心非常明确，不是让学生背散点，而是把全书压成“`法条判断 + 意义提升`”两步走。
- 其中最关键的能力不是单个法条记忆，而是把案例还原成具体法律关系，再把答案从`规则`自然推到`价值`。
- 如果后面这个线程专门负责选必二，我会默认沿用这份结构：
  - 先按知识板块定位
  - 再补法治意义
  - 遇到案例时优先判断它属于合同、侵权、家庭、企业还是纠纷解决哪一条线

## 八、三年模拟题补出的高频设问

这部分是 2024、2025、2026 三年模拟题把框架进一步“逼实”的地方。

### 1. 判决认识类

- 高频问法：
  - 谈谈对本案判决的认识
  - 分析法院判决的结果并说明理由
  - 说明人民法院支持或驳回某项诉讼请求的理由
- 三年里反复出现的典型来源：
  - 2024 丰台一模：`好意同乘`，问`法理依据和现实意义`
  - 2024 海淀一模：`虚拟数字人`，问`对判决的认识`
  - 2025 朝阳一模：问`支持王某诉讼请求的理由`
- 这类题的默认顺序：
  - 先定法律关系
  - 再定侵权/违约/合同效力等法律性质
  - 再写应承担的责任或法院支持的请求
  - 最后补判决的法治意义

### 2. 法律规定并限制类

- 高频问法：
  - 为什么法律要“规定某项权利/制度”，又要“对其加以限制”
  - 说明法律这样规定的原因或意义
- 三年里已经出现的典型来源：
  - 2025 海淀期末：`竞业限制`
  - 2025 海淀期末：`著作权并对其加以限制`
- 这类题不能只写“保护权利”，还要补上：
  - 为什么不能把某项权利绝对化
  - 法律到底在平衡哪两端甚至哪三端利益
  - 这种平衡如何服务创新、就业、流通、文化传播、社会公平

### 3. 法院如何保护类

- 高频问法：
  - 人民法院是如何依法保护某类权益的
  - 法院如何以司法之力守护某类发展目标
- 典型来源：
  - 2026 朝阳一模：`依法保护知识产权以护航新质生产力发展`
- 这类题比普通“意义题”更强调：
  - 法院做了什么
  - 做得怎么样
  - 这些司法举措如何转化成制度效果和发展效果

### 4. 调解建议与维权策略类

- 高频问法：
  - 如果你是人民调解员，你会如何调解并说明理由
  - 维权成功需要做好哪些准备和策略选择
- 典型来源：
  - 2024 东城一模：调解员视角给建议
  - 2025 朝阳二模：提出解决纠纷的建议
  - 2026 东城期末：`法律责任及法律依据 + 维权准备`
- 这说明选必二的最后一单元不能当“收尾知识”处理，而是已经进入真题高频区。

## 九、三年模拟题补出的高频情境

### 1. 数字技术、AI 与知识产权

- 高频材料：
  - 虚拟数字人
  - AI 相关创作或平台传播
  - 专利、商标、著作权、不正当竞争
  - 知识产权护航新质生产力
- 框架修正：
  - 不能把知识产权只当作传统背诵点
  - 要把它和`创新激励`、`市场秩序`、`公平竞争`、`新质生产力`并联起来

### 2. 平台消费、格式条款与消费者保护

- 高频材料：
  - 预付式消费
  - 食品安全
  - 生鲜灯
  - 先用后付
  - 格式合同、格式条款
- 框架修正：
  - 企业与消费者这条线要重点补`知情权、安全消费权、自主选择权`
  - 还要补`经营者依法诚信经营`、`惩罚性赔偿`、`维护市场秩序`

### 3. 劳动关系的新边界

- 高频材料：
  - 劳动合同变更和解除
  - 事实劳动关系
  - 竞业限制
  - 新就业形态
- 框架修正：
  - 劳动这条线不只考维权路径
  - 还会考`合同边界`、`劳动权保障`、`商业秘密保护`、`人才合理流动`

### 4. 家庭、继承与德法结合

- 高频材料：
  - 遗嘱与遗赠扶养协议
  - 赡养义务
  - 精神慰藉
  - 养老保障与老年人权益保护
- 框架修正：
  - 家庭题不能只写法条
  - 要主动补`敬老爱老`、`中华传统美德`、`德治与法治相结合`

### 5. 侵权责任与公共风险

- 高频材料：
  - 好意同乘
  - 经营场所安全保障义务
  - 环境污染
  - 无人机等新场景损害
- 框架修正：
  - 侵权题必须形成固定顺序：
    - 受损权利是什么
    - 责任构成是否成立
    - 归责原则是什么
    - 责任方式怎么落
    - 公平正义和秩序意义如何收束

### 6. 多元纠纷解决机制

- 高频材料：
  - 人民调解
  - 仲裁
  - 诉讼
  - 特邀第三方调解
  - 协商未果后的维权路径选择
- 框架修正：
  - 纠纷解决机制已经不是单纯程序知识
  - 现在要和`效率`、`成本`、`实质解纷`、`社会治理效能`一起写

## 十、三年模拟题补出的答题顺序

### 1. 判决认识题

- 第一步：判断案件属于合同、侵权、家庭继承、劳动还是知识产权
- 第二步：锁定核心争点，例如合同是否有效、是否构成侵权、应否承担惩罚性赔偿
- 第三步：写清责任承担或裁判结论的法律依据
- 第四步：补上保护合法权益、维护市场秩序、彰显公平正义等法治意义

### 2. 法律规定并限制题

- 第一步：先写法律为什么要设立这项制度
- 第二步：再写如果不限制会损害什么权利或秩序
- 第三步：写法律在平衡哪些主体利益
- 第四步：最后落到促进创新、保护劳动者、推动公平竞争、促进文化传播等现实意义

### 3. 法院如何保护题

- 第一步：先概括法院采取了哪些司法措施
- 第二步：写这些措施分别保护了谁
- 第三步：写这些措施如何抑制侵权、降低成本、提高效率、稳定预期
- 第四步：收束到创新生态、市场秩序、法治环境

### 4. 调解建议与维权策略题

- 第一步：识别纠纷类型和相关主体
- 第二步：判断优先适用协商、调解、仲裁还是诉讼
- 第三步：写出当事人应准备的证据、请求和程序路径
- 第四步：补出选择该路径的原因，如成本更低、效率更高、便于实质解纷

## 十一、我对这份框架的最新修正

- 课件原来的总抓手`法条知识扎实 + 意义角度全面`是对的，但经过三年模拟题验证后，要把它再具体化成：
  - `法律关系判断`
  - `争点识别`
  - `责任/效力/程序落点`
  - `法治意义收束`
- 原来我把“多元纠纷解决机制”放得偏后；现在看三年模拟题，它应该上升为一条独立主线，因为`调解、诉讼、维权准备、程序选择`已经成为真题高频。
- 原来“知识产权”更像合同后的一个知识专题；现在看 2025、2026 的真题趋势，它其实已经和`AI、数字技术、新质生产力、不正当竞争`连成一条热点主线。
- 后面这个线程如果继续学题，我会默认按“`题型模板 + 情境类型 + 法治意义`”三层同步补框架，而不是只往知识板块里堆点。

## 十二、第一轮真题研究补出的稳定母题

### 1. 调解建议题

- 2024 东城一模已经说明，这类题往往不是单纯程序题，而是：
  - 合同或债务关系
  - 家庭权利义务
  - 调解建议
- 默认模板：
  - 先指出各方做法哪里不当
  - 再明确各方权利义务
  - 最后给出协商、调解、履约、维系家庭关系等建议

### 2. 判决认识题

- 2024 丰台一模、2024 海淀一模都说明，这类题至少要走四步：
  - 定权利或法律关系
  - 定行为性质
  - 定责任或裁判结果
  - 定法治意义

### 3. 双轨判断题

- 2025 朝阳一模已经把这类题做得很典型：
  - 一轨看侵权/消费者权益
  - 一轨看合同/违约/欺诈
- 以后遇到平台消费、搭售、格式合同、预付式消费，默认先做双轨扫描

### 4. 规定并限制题

- 2025 海淀期末之后，这类题已经可以视作稳定母题
- 默认模板：
  - 保护什么
  - 为什么不能无限扩张
  - 法律在平衡什么
  - 这种平衡带来什么社会效果

### 5. 司法护创新题

- 2025 丰台二模、2026 朝阳一模、2026 东城一模都在往这个方向走
- 默认模板：
  - 划定边界
  - 保护权利
  - 惩治侵权或规制滥诉
  - 维护市场秩序
  - 护航创新生态或新质生产力

### 6. 维权策略题

- 2026 东城期末已经把它直接放进题面
- 以后这类题默认要准备：
  - 留存证据
  - 明确请求
  - 选择路径
  - 说明为什么这样选

## 十三、第二轮细则回填后的直接结论

- 现在这份框架里，已经不只是“知识板块框架”，而是开始带有`细则驱动的母题模板`
- 已经回填进去的条目，能稳定支持这些默认作答动作：
  - 遇到平台消费题，先做`消费者权益线 + 合同线`双轨判断
  - 遇到竞业限制、著作权保护期限、格式条款等题，优先按`规定 + 限制 + 平衡`组织答案
  - 遇到知识产权护创新题，优先按`调解 + 惩罚 + 规制滥诉 + 创新环境`组织答案
  - 遇到侵权减责题，不能只写构成，还要写`为什么减责`
- 暂时没有核准题号或来源不够稳的细则，我这轮没有并入主框架，后面核准后再补

## 十四、继续学习后的题型分类与共性

### 1. 目前可以稳定归出的十类题

- `判决认识/支持或驳回诉请理由题`
  - 代表题源：`2024丰台一模第17题`、`2024海淀一模第19题`、`2025朝阳一模第19题`、`2025西城期末第19题`
- `规定并限制/利益平衡题`
  - 代表题源：`2025海淀期末第21题`、`2025西城一模第20题`、`2025丰台二模第19题（2）`
- `司法护创新/法院如何保护题`
  - 代表题源：`2025丰台一模第19题`、`2025房山一模第19题`、`2025石景山一模第20题`、`2026东城一模第18题`、`2026朝阳一模第18题`
- `调解建议/维权策略题`
  - 代表题源：`2024东城一模`、`2025东城一模第19题`、`2025延庆一模第19题`、`2026东城期末`
- `格式条款/合同效力/诚信边界题`
  - 代表题源：`2025东城二模第19题`、`2026海淀期末第18题（1）（2）`
- `消费者保护/惩罚性赔偿/理性维权题`
  - 代表题源：`2024西城二模第16题`、`2026海淀一模第18题（3）`
- `知识产权/不正当竞争/数字经济题`
  - 代表题源：`2024海淀一模第19题`、`2025丰台一模第19题`、`2025房山一模第19题`、`2025西城期末第19题`、`2026海淀期末第19题`
- `人格权侵权题`
  - 代表题源：`2025朝阳二模第20题（1）`、`2025东城二模第19题`、`2026海淀一模第18题（3）`
- `家庭继承/扶养社会价值题`
  - 代表题源：`2024海淀二模第19题`、`2025东城一模第19题`、`2024东城一模`
- `物权/绿色原则/生态公共利益题`
  - 代表题源：`2026西城一模第17题`

### 2. 这些题的四个稳定共性

- 第一，越来越多的题不是单线知识题，而是`一题多关系`、`一题多规则`
- 第二，细则几乎都要求先`定法律关系/权利种类`，再`写行为定性与责任`，最后`补价值和治理效果`
- 第三，光写抽象结论很难拿高分，必须把材料里的关键事实写进去，例如：
  - `擅自搭售`
  - `抢先注册`
  - `引人误解的宣传`
  - `超出正常生活消费需要`
  - `公开监控录像和聊天信息`
- 第四，意义层已经越来越稳定，不再是泛泛写“保护权益”，而是反复收束到：
  - `公平竞争`
  - `市场秩序`
  - `诚信原则`
  - `创新生态`
  - `公共利益`
  - `核心价值观`

### 3. 三年的变化脉络

- `2024`
  - 重点是把基础规则做成复合争点题，开始逼学生从“会背法条”走向“会解释为什么这样判”
- `2025`
  - 重点明显转向`边界`、`平衡`、`支持或驳回诉请的理由`
- `2026`
  - 进一步转向`实体法 + 程序法 + 司法治理 + 公共利益`联动，创新保护和绿色发展都更明显

### 4. 当前还在跟踪但暂不按细则回填的信号

- `2025海淀一模第18题`
  - 当前证据层级是`试卷题面 + 教师版参考答案`
  - 已能确认的题型信号是：
  - `格式合同有效`
  - `条款符合公平原则`
  - `违背诚信原则`
  - `法院驳回诉讼请求`
  - 这类题后面可以和`格式条款/合同效力/诚信边界题`并类继续跟踪

## 十五、细则反复出现的高频词与共性提醒

- 高频复现词现在已经比较稳定：
  - `合法权益`
  - `市场秩序`
  - `公平竞争`
  - `诚信原则`
  - `公共利益`
  - `创新`
  - `平衡`
- 这说明后面的题，不能只停在“法条结论”，而要默认往这些意义词上收束
- 高频复现逻辑也越来越稳定：
  - `定关系`
  - `贴材料`
  - `定责任/效力/程序`
  - `升法治意义`
- 只要题干里出现：
  - `边界`
  - `平衡`
  - `保护并限制`
  - `支持或驳回理由`
- 就优先往老母题上套，而不是把它当成完全陌生的新题

## 十六、零基础满分课讲授顺序与套写模板

- 这一步不再继续加新题，而是把前面已经稳定下来的 `十一类题型`、`高频共词`、`母题链` 压成能直接上课的顺序。

### 1. 先讲全书总模板，再讲具体板块

- 第一讲只做一件事：让学生把所有选必二主观题先统一成四步
  - `先定关系`
  - `再抓争点`
  - `再落责任/效力/程序`
  - `最后收法治意义`
- 第二讲先讲最容易直接提分的基础题：
  - `判决认识/支持或驳回理由题`
  - `双轨判断题`
  - `格式条款/合同效力题`
- 第三讲专讲近三年反复出现的 `保护 + 限制 + 平衡`：
  - `竞业限制`
  - `著作权限制`
  - `AI 应用边界`
  - `物权与绿色原则`
- 第四讲专讲程序与维权：
  - `调解建议题`
  - `维权策略题`
  - `多元解纷/公益诉讼/司法确认效力题`
- 第五讲专讲创新与治理：
  - `司法护创新题`
  - `知识产权/不正当竞争/数字经济题`
- 第六讲再做情境迁移：
  - `平台消费`
  - `人格权侵权`
  - `家庭继承`
  - `劳动关系边界`
- 这样排课的原因是：
  - 先把所有题统一成同一套答题动作
  - 再从最常见、最稳定、最容易拿分的母题入手
  - 最后才把新情境和复合题压进同一模板

### 2. 零基础学生先背这一套“四句总口令”

- 第一句：`先看谁和谁发生了什么法律关系，争的到底是什么权利、义务或程序。`
- 第二句：`再把材料里的关键事实贴进去，说明为什么能这样定性。`
- 第三句：`再写责任、效力、法院态度或者程序结果，不能只喊价值口号。`
- 第四句：`最后一定收束到保护合法权益、维护市场秩序、公平竞争、诚信原则、公共利益、创新环境这些高频意义词。`

### 3. 六条最值得课堂反复操练的母题模板

#### 1. 判决认识题

- 套写顺序：
  - `先定法律关系/权利种类`
  - `再定行为性质`
  - `再写法院为什么支持或驳回`
  - `最后写法治意义`
- 学生口令：
  - `先说争什么，再说错在哪，再说怎么判，最后说这样判保护了谁、维护了什么秩序。`

#### 2. 规定并限制题

- 套写顺序：
  - `先写为什么要设这项制度`
  - `再写为什么不能无限扩张`
  - `再写法律在平衡哪几方利益`
  - `最后写这种平衡的现实效果`
- 学生口令：
  - `先写为什么要保护，再写为什么要限住，最后写法律在做平衡。`

#### 3. 格式条款/合同效力题

- 套写顺序：
  - `先定是不是格式条款或是否构成有效合同`
  - `再审提示说明义务、诚信原则、公平原则`
  - `再写条款有效、无效或责任承担`
  - `最后写减少纠纷、规范市场秩序`
- 学生口令：
  - `先定条款性质，再看程序和原则，最后才下结论。`

#### 4. 调解建议与维权策略题

- 套写顺序：
  - `先拆争点和主体`
  - `再选协商、调解、仲裁或诉讼路径`
  - `再写证据准备、请求设计、程序安排`
  - `最后写为什么这条路径更能实质解纷`
- 学生口令：
  - `先看纠纷，再选路径，再备证据，最后说明为什么这样最有效。`

#### 5. 多元解纷/公益诉讼/司法确认效力题

- 套写顺序：
  - `先定谁能启动程序、为什么能启动`
  - `再写为什么不是只靠单一路径，而是和解、调解、诉讼并用`
  - `再写法院审查、司法确认、民事调解书和强制执行力`
  - `最后收束到公共利益、执行保障、实质解纷`
- 学生口令：
  - `先写谁能告，再写为什么要分流，最后写法院确认后效力升级。`

#### 6. 司法护创新题

- 套写顺序：
  - `先写法院或司法机关做了什么`
  - `再写这些动作保护了谁、打击了什么`
  - `再写如何提高侵权成本、节约司法资源、规制恶意诉讼`
  - `最后收束到创新生态、市场秩序、新质生产力`
- 学生口令：
  - `先写司法动作，再写直接效果，最后写治理效果。`

### 4. 课堂收尾时要反复提醒学生的失分边界

- 不能只写 `知识产权`、`人格权`、`消费者权益` 这种大词，必须尽量写准到具体权利或具体制度。
- 不能只背法条，不贴材料事实；细则真正给分的是 `定性 + 材料 + 结果` 的连贯动作。
- 不能一看到调解就停在“化解矛盾”，要继续看有没有 `司法确认`、`强制执行力`。
- 不能一看到“保护”就忘了“限制”和“平衡”；近三年很多题失分就失在这一层。
- 不能一看到新技术、新平台、新场景就慌，先回老母题：
  - `判决认识`
  - `规定并限制`
  - `程序维权`
  - `司法护创新`
