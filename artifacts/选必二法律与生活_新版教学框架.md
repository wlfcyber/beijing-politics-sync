# 选必二《法律与生活》做题路由型深度框架

> 核心总式：**设问任务 -> 主体关系 -> 权利客体 -> 法律关系 -> 关键事实/证据 -> 法律规则 -> 责任/救济 -> 价值意义**。
>
> 学生口令：**审问法，定主体；找客体，辨关系；抓事实，套规则；下结论，升价值。**

本框架不再按教材目录顺排，而是按北京模拟题的答题路径组织。教材四大单元仍作为知识库，但课堂训练的入口必须是材料和设问。

## 0. 证据等级和使用边界

| 证据等级 | 可怎么用 | 不可怎么用 | 本框架标注 |
|---|---|---|---|
| 评分细则 | 可作为给分边界、失分点和主例 | 不得脱离题目事实扩大为通用答案 | `评分细则` |
| 参考答案 | 可抽取答题链和材料化短句 | 不得称为评分细则 | `参考答案` |
| 教师版答案 | 可作讲评边界和训练素材 | 不得当作官方细则 | `教师版答案` |
| 教师讲评/PPT | 可互证教师讲评口径 | 不得冒充正式答案或细则 | `讲评边界` |
| 复练题 | 可作迁移训练 | 不得作为区级正式题源 | `复练题边界` |
| 只核到试题 | 可保留触发词和设问 | 不得写入主采分链 | `待核答案` |

所有题源见 `选必二法律与生活_三年题源覆盖矩阵.csv`。本版已把 `2026海淀一模` 核为 `18(3)`，把 `2025海淀二模18` 拆为“实体商标”和“程序证据”两条。

## 1. 30秒总分流图

这一页只解决一件事：学生拿到题后先看设问任务，不要一看到材料就冲进知识点。

| 设问第一眼 | 进入哪里 | 答题核心 |
|---|---|---|
| 问“是否支持、是否构成、是否有效” | 实体判断 | 结论 + 规则 + 材料事实 + 责任 |
| 问“为什么这样判” | 裁判理由 | 法律依据 + 事实对应 + 责任边界 |
| 问“如何维权、应准备什么” | 程序证据 | 路径 + 证据 + 时效/仲裁/管辖 |
| 问“如何保护、如何护航” | 措施价值 | 措施 + 效果 + 秩序/权益/创新 |
| 问“评析某观点” | 权利边界 | 合理处 + 片面处 + 权利义务统一 |
| 问“意义、价值、启示” | 材料化价值 | 保护谁 + 维护什么秩序 + 平衡什么利益 |

总规则：**材料可以多点触发，但答案必须由设问裁剪。问实体，不硬塞程序；问程序，不偷答实体；问价值，不脱离具体规则；问维权，必须写证据和路径。**

## 2. 设问任务总入口

| 设问任务 | 先答什么 | 材料读取重点 | 常用结尾 | 常见失分 |
|---|---|---|---|---|
| 是否支持、是否构成、是否有效 | 明确结论 | 主体身份、权利客体、法律关系、规则触发事实 | 谁承担何种责任或请求能否支持 | 只背规则，不下结论 |
| 法院为什么这样判 | 裁判结论+法律依据 | 法院认定的事实、证据、责任类型 | 维护当事人合法权益和规则秩序 | 写成道德评价或泛泛意义 |
| 如何维权、应做哪些准备 | 维权路径 | 证据、诉讼时效、仲裁前置、管辖、代理 | 依法理性维权、降低纠纷解决成本 | 只写起诉，不写证据和程序条件 |
| 如何保护、如何护航 | 措施链 | 调解、惩罚性赔偿、权利滥用规制、司法建议 | 形成稳定可预期的法治环境 | 空写“法治中国”“公平正义” |
| 评析某观点 | 先判观点片面或错误 | 哪个权利被绝对化、哪个义务被忽略 | 权利行使有边界，权利义务相统一 | 只说“正确/错误”，不拆理由 |
| 谈意义、谈价值 | 先扣具体案件 | 保护什么主体、维护什么秩序、平衡什么利益 | 材料化价值句 | 套话堆砌，不扣题源事实 |

## 4. 宏观四域与训练八节点

对学生讲课时先用教材四域建立知识归属，再用训练八节点建立做题路径。

| 教材四域 | 对应训练节点 | 用途 |
|---|---|---|
| 民事权利与义务 | 民事法律关系总入口；合同与违约；侵权与人格/个人信息；物权、相邻关系与财产权；知识产权与公平竞争 | 判断权利、义务、责任和边界 |
| 家庭与婚姻 | 婚姻家庭与继承 | 判断身份关系、赡养、扶养、继承和遗赠扶养协议 |
| 就业与创业 | 劳动、消费、经营 | 判断劳动关系、消费者权益、诚信经营和竞争秩序 |
| 社会争议解决 | 纠纷解决与诉讼证据 | 判断路径、证据、举证责任、回避、代理、调解、仲裁、诉讼 |

每题最终落到八步法：**审设问 -> 定主体 -> 找客体 -> 辨关系 -> 抓事实/证据 -> 套规则 -> 下结论/定责任 -> 升价值**。

## 5. 共同前置：主体、客体、关系、边界

| 检查点 | 课堂落点 | 可背短句 |
|---|---|---|
| 主体身份 | 人名要改写成法律身份 | “材料中的张某不能只写成张某，要定性为消费者、劳动者、权利人、侵权人或诉讼当事人。” |
| 权利客体 | 先看争的是什么 | “争的是作品看著作权，争的是标识看商标，争的是非公知技术信息看商业秘密，争的是人身安宁看人格权。” |
| 法律关系 | 同一材料可能多关系 | “同一材料可以同时触发合同、侵权、消费者、知识产权或程序规则，但答题范围由设问决定。” |
| 关键事实 | 每条规则都要回扣材料 | “法律规则不能悬空，必须找到材料中的承诺、交付、损害、公开、未经许可、保密措施或证据事实。” |
| 权利边界 | 权利不是任性行使 | “民事主体依法享有权利，也应依法、诚信、合理行使权利，不得损害他人合法权益和公共利益。” |

共同失分点：把“保护权益、维护公平、建设法治中国”写成独立段落。意义句只能压回材料，例如“依法制止老字号商标抢注和搭便车行为，有利于维护企业商誉和公平竞争秩序。”

## 6. 合同节点

### 6.1 合同成立/有效/履行/违约/侵权分流表

| 分流问题 | 材料触发词 | 法律判断 | 答题短句 | 主例/对照 | 常见误写 |
|---|---|---|---|---|---|
| 合同是否成立 | 报价、确认中标、立即转账、承诺保真 | 要约和承诺一致 | “双方通过要约和承诺达成一致意思表示，合同成立。” | 2026东城期末18；2024石景山一模17 | 只写“买卖关系”不写要约承诺 |
| 合同是否有效 | 主体适格、意思表示真实、内容形式合法 | 成立不等于有效，还要审有效要件 | “双方主体适格、意思表示真实，合同内容和形式不违反法律强制性规定，合同有效。” | 2026东城期末18 | 把合同成立和合同有效混为一分 |
| 是否全面履行 | 全新原装、展示机、未按约交付、欠付货款 | 按约全面履行、诚信履行 | “一方未按约履行或履行不符合约定，违反全面履行和诚信原则，应承担违约责任。” | 2026东城期末18；2024朝阳一模19 | 只写诚信原则，不写具体违约事实 |
| 条款是否进入合同 | 购物须知、退换货须知、直播承诺、假一赔十 | 明示且同意的条款可能成为合同内容 | “经营者作出的明确承诺若成为合同内容，应受合同约束。” | 2025西城二模10；2026朝阳期末复练 | 把所有宣传都直接写成合同条款 |
| 是否要审格式条款 | 页面勾选、免责说明、退换货限制 | 不得不合理免除经营者责任、加重对方责任、排除主要权利 | “格式条款不得不合理免除或减轻经营者责任、加重消费者责任、排除消费者主要权利。” | 2025西城二模10 | 只写“自愿点击同意所以都有效” |
| 是否转入侵权 | 砸伤、医疗费、人格权受损、产品缺陷 | 有损害再审侵权，责任竞合不等于重复赔偿 | “有约定先看合同义务，造成他人人身、财产损害的，再判断是否构成侵权或产品责任。” | 2026东城期末18 | 违约损失和侵权损害重复赔 |

### 6.2 合同题学生短句

1. “有交易约定时，先看合同是否成立并有效；合同有效后，当事人应按约全面履行。”
2. “材料中一方承诺‘全新原装’却交付受损展示机，属于履行不符合约定，应承担违约责任。”
3. “合同责任保护的是交易期待利益，侵权责任保护的是人身、财产等民事权益，二者不能混写成双重赔偿。”
4. “直播间‘保真’‘假一赔十’等明确承诺进入合同内容的，经营者应受该承诺约束。”

### 6.3 给分边界

| 给分点 | 写法 |
|---|---|
| 合同成立 | 要约+承诺，而不是“双方有交易” |
| 合同有效 | 主体适格、意思表示真实、内容形式合法 |
| 违约 | 具体指出未按约交付、欠款、未履行承诺 |
| 责任 | 继续履行、采取补救措施、赔偿损失、支付违约金等，按设问写 |
| 价值 | 维护交易安全、诚信秩序、绿色交易秩序，必须扣材料 |

### 6.4 必答点 / 可加点 / 禁写点

| 层级 | 功能 | 合同题处理 |
|---|---|---|
| 必答点 | 不写就丢主分 | 合同成立；合同有效；约定义务；未按约履行；违约责任 |
| 可加点 | 有材料才加 | 诚信原则、全面履行原则、格式条款、直播承诺、消费者安全权 |
| 禁写点 | 写了容易错 | 不区分违约和侵权；同一损害重复赔偿；只写诚信不写违约事实；把所有宣传都写成合同条款 |

已核查题源：`2026东城期末18` 评分细则；`2024石景山一模17` 参考答案；`2024朝阳一模19` 参考答案；`2025西城二模10` 教师版试题；`2026朝阳期末复练题` 复练边界。

## 7. 侵权责任节点

### 7.1 侵权责任四要件+责任类型+举证边界表

| 判断层 | 触发事实 | 规则 | 答题句 | 题源 | 失分边界 |
|---|---|---|---|---|---|
| 受保护权益 | 生命健康、隐私、财产、著作权、商标权 | 民事权益受法律保护 | “陈某手臂受伤、产生医疗费，涉及生命健康权和财产损失。” | 2026东城期末18 | 不写权利客体 |
| 侵权行为 | 交付缺陷产品、公开监控、未经许可使用 | 行为侵害他人权益 | “商家公开包间监控和聊天信息，属于以公开方式侵害隐私权。” | 2026海淀一模18(3) | 只写“违法”不写行为 |
| 损害结果 | 医疗费、名誉受损、隐私暴露、商誉损害 | 有损害才谈赔偿 | “材料中的医疗费、商业拍摄损失要分清侵权损害和违约损失。” | 2026东城期末18 | 把所有损失都算侵权 |
| 因果关系 | 因展示机损伤坠毁、因摔倒受伤 | 行为与损害要有因果 | “无人机因上述损伤失控坠毁，与陈某受伤之间存在因果关系。” | 2026东城期末18 | 只列损害不连事实 |
| 一般过错 | 销售者明知或应知风险、经营管理疏忽 | 过错责任 | “销售者主张侵权时，应分析行为、损害、因果关系和过错。” | 2026东城期末18 | 销售者一律写无过错 |
| 无过错责任 | 产品缺陷造成损害、法律特别规定 | 无需证明生产者过错 | “对生产者主张产品责任时，可按产品缺陷造成损害判断无过错责任。” | 2026东城期末18 | 生产者写一般过错 |
| 过错推定 | 个人信息处理者侵权、动物园动物致害等 | 法律推定过错，可反证 | “过错推定要有法定情形，不能把所有侵权都写成过错推定。” | 2026东城期末10细则延伸 | 随意套过错推定 |
| 安全保障义务 | 宾馆、商场、公共场所摔倒 | 合理必要措施，不是绝对保险 | “公共场所经营者的安全保障义务限于合理必要范围，尽到义务且无过错的，不当然承担全部责任。” | 2026丰台一模20 | 出了事就写经营者必赔 |
| 受害人过错 | 未尽注意义务、风险自担 | 可减轻责任 | “受害人自身过错会影响责任承担，不能只看损害结果。” | 2024丰台一模17；2026丰台一模20 | 忽略减责 |
| 举证责任 | 谁主张谁举证，证据不足承担不利后果 | 证据链决定程序分 | “主张侵权的一方应提交损害、行为和因果关系等证据，证据不足将承担不利后果。” | 2025海淀二模18(程序) | 只写实体，不写证据 |

### 7.2 侵权题学生短句

1. “判断侵权责任，要看民事权益受损、侵权行为、损害结果、因果关系以及过错或法定归责类型。”
2. “安全保障义务要求经营者采取合理必要措施，不等于对一切损害承担绝对赔偿责任。”
3. “同一损害可能涉及违约和侵权，但责任竞合不意味着权利人可以就同一损害重复获赔。”
4. “主张侵权的一方要用证据证明材料事实，证据不足的，由负有举证责任的一方承担不利后果。”

### 7.3 必答点 / 可加点 / 禁写点

| 层级 | 功能 | 侵权题处理 |
|---|---|---|
| 必答点 | 不写就丢主分 | 受保护权益；侵权行为；损害结果；因果关系；过错或法定归责；责任方式 |
| 可加点 | 有材料才加 | 安全保障义务、受害人过错、产品责任、举证责任、停止侵害/赔礼道歉等具体责任 |
| 禁写点 | 写了容易错 | 出事即赔；安全保障义务绝对化；销售者和生产者归责错配；侵权与违约重复赔 |

## 8. 消费者权益节点

### 8.1 消费者权益触发词表

| 触发词 | 具体权利/义务 | 答题短句 | 题源 | 不能写成 |

| 食品、标签、功效、成分、保质期 | 安全消费权、知情权 | “经营者应保障消费者人身财产安全，并真实、全面提供商品信息。” | 2024西城二模16 | “保护消费者权益”一句话 |
| 夸大治疗功效、虚假宣传 | 知情权、欺诈、惩罚性赔偿 | “经营者作虚假或引人误解宣传，诱导消费，构成欺诈的，应退款并承担惩罚性赔偿责任。” | 2026海淀一模18(3) | 消费者受损就惩罚性赔偿 |

### 8.2 消费者题学生短句

1. “消费者权益题不能只写合法权益，要准确落到安全权、知情权、自主选择权、公平交易权、隐私和个人信息等具体权利。”
2. “经营者应真实、全面提供商品或服务信息，不得作虚假或者引人误解的宣传。”
3. “惩罚性赔偿须以欺诈或法律特别规定为前提，不能写成消费者一受损就当然适用。”
4. “消费者维权要扣证据和路径，既维护权利，也要依法理性。”

### 8.3 给分边界

| 题型 | 优先写什么 | 边界 |
|---|---|---|
| 食品/商品质量 | 安全权、知情权、经营者真实全面告知义务 | 食品安全、标签瑕疵、欺诈要按材料区分 |
| 虚假宣传 | 宣传内容、消费者误信、欺诈或误导 | 行政违法不必然等同民事惩罚性赔偿 |
| 直播营销 | 主播承诺、合同条款、欺诈、消费者权利 | 看设问取合同或消费，不全盘堆砌 |
| 隐私/个人信息 | 私密空间、公开泄露、处理必要性 | 不要用知情权替代隐私权 |

| 必答点 | 不写就丢主分 | 具体消费者权利；经营者义务；材料中的虚假、隐瞒、强制或泄露事实；责任或维权路径 |
| 可加点 | 有材料才加 | 惩罚性赔偿、合同条款、个人信息保护、直播平台或主播身份、依法理性维权 |
| 禁写点 | 写了容易错 | 只写“侵犯消费者合法权益”；消费者受损即惩罚性赔偿；隐私权和知情权混写；把复练题当正式细则 |

## 16. 主观题采分链模板

### 16.1 合同/侵权类

适用题源：`2026东城期末18无人机案`。

模板：

1. 定主体：买方陈某、卖方刘某，必要时区分生产者。
2. 定法律关系：陈某与刘某之间存在买卖合同关系；无人机坠毁砸伤另涉侵权或产品责任。
3. 定权利义务：刘某应按“全新原装”约定全面履行，经营者还应保障消费者人身财产安全。
4. 扣材料：刘某误发内部结构轻微损伤的展示机，首次使用即失控坠毁并造成医疗费等损害。
5. 定责任/救济：违约侧承担违约责任；侵权侧按一般过错或产品责任判断，不能重复赔偿同一损失。
6. 材料化意义：该处理维护交易诚信和消费者安全，明确合同责任与侵权责任边界。

不能写：只写“刘某侵权所以赔偿全部损失”；把销售者无条件写成无过错责任；不区分商业拍摄损失和人身损害。

### 16.2 消费者/IP类

适用题源：`2026延庆一模18(1)`、`2025海淀二模18(实体)`、`2024西城二模16`。

模板：

1. 审设问：是判断违法行为、支持诉求，还是分析保护创新和市场秩序。
2. 定主体：权利人、经营者、消费者、竞争者分别是谁。
3. 找客体：肖像/名誉、消费者知情利益、商标、作品、技术秘密或商品安全。
4. 抓事实：未经许可AI合成形象、虚假直播、恶意抢注、食品功效宣传、保密措施等。
5. 套规则：人格权、消费者权益保护、商标权、商业秘密、不正当竞争分别适用。
6. 下结论：停止侵害、赔礼道歉、赔偿损失、惩罚性赔偿或规制不正当竞争。
7. 升价值：维护权利人权益、消费者信赖、公平竞争和创新生态。

不能写：所有知识产权题都写“著作权”；所有消费题都写“惩罚性赔偿”；意义句脱离案件。

### 16.3 程序证据类

适用题源：`2025海淀二模18(程序)`、`2024西城一模18`、`2024东城一模19`。

模板：

1. 定纠纷类型：民事、劳动、商事仲裁还是调解。
2. 定程序角色：当事人、诉讼代理人、书记员、法院、仲裁机构。
3. 定程序权利：回避、委托代理、举证、申请调解或仲裁。
4. 抓证据事实：书证、物证、证人证言、电子数据等能证明什么。
5. 套程序规则：谁主张谁举证；证据不足承担不利后果；有利害关系应回避。
6. 下结论：程序是否合法、证据能否支持主张、维权路径如何选择。
7. 材料化意义：保障程序公正、提高纠纷解决效率、维护当事人合法权益。

不能写：程序题只答实体权利；把诉讼代理人写成辩护人；把调解和仲裁混为一谈。

## 19. 最小闭环验收

任取一个节点，应能看到：材料触发词 -> 主体身份 -> 权利客体 -> 法律关系 -> 关键事实/证据 -> 法律规则 -> 责任/救济 -> 材料化价值。

任取一个题源，应能在覆盖矩阵中回到：`source_id`、题号/小问、原文件路径、页码/位置、答案依据类型、是否评分细则、核查状态和风险备注。
