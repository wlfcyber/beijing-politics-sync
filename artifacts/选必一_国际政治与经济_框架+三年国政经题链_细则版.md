# 选必一《国际政治与经济》框架+三年国政经题链（细则版）

## 使用口径

- 只看试卷原题与细则、评标、阅卷讲评。
- 不把参考答案当作细则。
- 每道题统一压成：`材料场景 -> 细则抓手 -> 回填框架`。
- 本轮已按细则稳定回填 `35` 题，暂挂起 `2` 题。

## 一、六模块总框架

### 1. 时代背景

- 和平与发展仍然是时代主题，但世界进入百年变局加速演进期。
- 经济全球化、政治多极化深入发展，同时国际局势变乱交织、局部冲突频发、治理赤字加重。
- 常见背景词：
  - 国际形势复杂多变
  - 单边主义、保护主义、霸权主义、强权政治
  - 全球性挑战增多
  - 风险挑战与发展机遇并存

### 2. 理论

- 国家利益是国际关系的决定性因素。
- 共同利益是国家间合作的基础。
- 维护本国利益是主权国家对外活动的出发点，同时要兼顾他国合理关切。
- 国家安全是最高利益，要统筹发展和安全。
- 高频关系型表达：
  - 自力更生与对外开放
  - 发展与安全
  - 中国发展与世界发展

### 3. 经济全球化

- 高频主线：
  - 贸易投资自由化便利化
  - 商品、服务、技术、资金、劳务等要素流动
  - 优化资源配置
  - 营商环境
  - 两个市场、两种资源
  - 高水平对外开放
  - 更高层次开放型经济
  - 双循环
  - 产业链供应链韧性与安全
  - 比较优势、国际竞争新优势

### 4. 政治多极化

- 高频主线：
  - 相互尊重、公平正义、合作共赢的新型国际关系
  - 共商共建共享的全球治理观
  - 国际关系民主化
  - 真正的多边主义
  - 全球治理体系改革和建设
  - 全球南方
  - 人类命运共同体

### 5. 联合国

- 联合国是当今世界最具普遍性、代表性、权威性的政府间国际组织。
- 联合国是践行多边主义的重要平台。
- 要抓：
  - 遵循联合国宪章宗旨和原则
  - 主权平等
  - 国际法治
  - 多边主义
  - 以联合国为核心的国际体系

### 6. 中国

- 我国奉行独立自主的和平外交政策。
- 宗旨是维护世界和平、促进共同发展。
- 基本立场是独立自主。
- 基本准则是和平共处五项原则。
- 新时代总抓手：
  - 习近平外交思想
  - 中国智慧、中国方案
  - 正确义利观
  - 大国责任与担当
  - 元首外交、周边工作、南南合作、全球倡议

## 二、三年题目回填

### 1. 时代背景

- `2024海淀二模 Q18(1)`：材料是“民主与治理现代化”“人工智能与民主的未来”“多极化世界中的民主与全球治理”“全球南方”等议题时评。细则抓手是先交代时代主题、世界多极化，再落到人类命运共同体、国际组织与全球治理。回填时放在“时代背景 -> 多极化世界中的治理议题”。

- `2025海淀期中 Q21(2)`：材料是“新中国外交的变与不变”。细则抓手分两层：`变` 写政治多极化、经济全球化、和平与发展、中国综合国力和国际地位提升、习近平外交思想形成；`不变` 写国家性质、独立自主、维护世界和平促进共同发展、主权安全发展利益、和平共处五项原则。回填时放在“时代背景 -> 大势变化”，并兼跨“中国”“理论”。

- `2025东城二模 Q20`：材料是“同球共济”小论文。细则抓手分三层：先写国际局势变乱交织和全球挑战，再写“同球共济”符合共同利益、丰富命运共同体内涵，最后写中国以实际行动提供方案和行动。回填时放在“时代背景 -> 变乱交织中的世界”。

### 2. 理论

- `2024朝阳一模 Q21`：材料是“如何推动中国经济在变局中开新局”。细则抓手分两步：先概括经济长期向好但外部环境严峻、风险挑战增多，再从总体国家安全观、推动新型国际关系、构建新发展格局、推进高水平对外开放等角度展开。回填时放在“理论 -> 统筹发展和安全”，并兼跨“经济全球化”“中国”。

- `2025延庆一模 Q20(2)`：材料是外交部回应“脱钩断链”与“共赢链”。细则抓手有两部分：对内写创新发展理念、政府作用、双循环、提升产业链供应链韧性和安全水平；对外写共同利益、和平与发展、反对脱钩断链、兼顾他国合理关切、共筑人类命运共同体。回填时放在“理论 -> 共同利益 + 发展安全关系”。

- `2025房山一模 Q20(2)`：材料是“中国开源战略不仅是技术革新，更是全球合作新模式的探索和实践”。细则抓手分两组：一组写国际竞争实质、高水平对外开放、科技发展推动建设创新型开放型世界经济；一组写在联合国框架下构建人类命运共同体、加强国际交流合作、积极参与全球治理。回填时放在“理论 -> 中国发展与世界发展关系”，并兼跨“联合国”“中国”。

- `2026延庆一模 Q20(2)`：材料是中国推动重塑全球能源治理格局。细则抓手先写共同利益是合作基础、顺应和平发展合作共赢潮流，再写依托“一带一路”和绿色能源合作践行共商共建共享、推动新型国际关系、提供中国方案、推动命运共同体。回填时放在“理论 -> 共同利益 + 时代潮流”，并兼跨“政治多极化”“中国”。

- `2026朝阳期中 Q17`：材料是推进“人工智能+”要处理哪些重要关系。细则抓手明确三组关系：`自力更生和对外开放`、`发展和安全`、`中国发展和世界发展`。每组都要求“总说关系 + 分说两边”。回填时直接做成“理论模块”的三大模板题。

### 3. 经济全球化

- `2024海淀一模 Q18(1)`：材料是免签、口岸签证、延期换发等便利中外人员往来举措。细则抓手是三条：降低来华成本、促进贸易投资自由化便利化；吸引商品服务技术资金劳务等要素流动，联动两个市场两种资源；推进高水平对外开放，发展更高层次开放型经济。回填时放在“经济全球化 -> 人员往来便利化”。

- `2024朝阳期中 Q20(3)`：材料是数贸会、数字产品、数字投资、数字商贸走廊。细则抓手是数字贸易促进技术应用和世界经济动力、促进投资贸易自由化便利化和资源配置、体现中国平等开放合作共享理念、助力建设高层次开放型经济。回填时放在“经济全球化 -> 数字贸易”。

- `2024丰台一模 Q20`：材料是基础设施、供应链金融、链博会、数字化和绿色技术推动“共赢链”。细则给分强调按几个方面展开分析，核心落点是：促进产品服务和生产要素流动、缓解供应链资金约束、搭建国际合作平台、以技术降本增效提质，最后落到贸易投资便利化、全球供应链高质量发展、合作共赢。回填时放在“经济全球化 -> 供应链合作”。

- `2024石景山一模 Q19(2)`：材料是新能源汽车从“走出去”到“扎下根”。细则抓手是顺应经济全球化、依靠技术创新和优质供给满足国际市场需求，利用当地比较优势优化资源配置、降低成本、形成国际竞争新优势，同时通过技术交流、就业带动和绿色生态圈实现互利共赢。回填时放在“经济全球化 -> 比较优势 + 走出去”。

- `2025丰台一模 Q20`：材料是北京“两区”建设，通过便利投资、便利贸易和科技创新推动开放升级。细则抓手很清：便利投资要写法治化营商环境和吸引外资；便利贸易要写通关效率、数据流动、服务贸易；科技创新要写人才和政策优惠；最后可上升到两个市场两种资源和双循环。回填时放在“经济全球化 -> 营商环境 + 制度型开放”。

- `2025朝阳一模 Q20`：材料是绿色低碳转型、数字技术融合、全球产业链供应链新态势。细则抓手分四束：多元化布局、区域化合作、绿色化转型、数字化加速，最后归到高水平对外开放、全球治理和双循环。回填时放在“经济全球化 -> 产业链供应链韧性”。

- `2025海淀一模 Q21(2)`：材料是综合保税区、市场潜力、外贸连上两个万亿级台阶。细则稳定抓手目前集中在两点：`贸易投资自由化便利化` 和 `产业结构优化调整`。回填时放在“经济全球化 -> 外贸增长的制度与结构原因”，后面讲题时可继续往“开放平台 + 市场空间”补。

- `2025西城二模 Q19(2)`：材料是对最不发达国家实行零关税。细则抓手是开放广阔中国市场、与发展中国家共享发展机遇、惠民生增福祉、互利共赢，世界意义是推动普惠包容的经济全球化、建设开放型世界经济、促进共同发展与持久和平。回填时放在“经济全球化 -> 市场开放与中国担当”。

- `2026东城一模 Q19(3)`：材料是北京发展未来产业时统筹高水平对外开放与产业链韧性。细则抓手分两面：`开放促进韧性`，写引进来走出去、国际高标准规则、国际科创合作、制度型开放、国际化营商环境；`韧性支撑开放`，写关键核心技术、自主创新、政策支持、强链补链。回填时放在“经济全球化 -> 开放与韧性双向支撑”。

- `2026西城一模 Q20(2)`：材料是中国—东盟自贸区3.0版。细则抓手分三层：标准规则降低制度性交易成本、推动贸易投资便利化；产业链供应链层面写比较优势、国际分工、产业协同与就业；综合层面写区域转型升级、发展中国家话语权、多边贸易和合作共赢。回填时放在“经济全球化 -> 区域经贸规则与产业链”。

- `2026门头沟一模 Q20`：材料是海南自贸港全岛封关。细则抓手对中国侧写零关税、加工增值、口岸通关、营商环境、双循环；对世界侧写更大市场空间、制度型开放范例、全球产业链供应链稳定畅通、开放型世界经济。回填时放在“经济全球化 -> 制度型开放 + 自贸港”。

### 4. 政治多极化

- `2024朝阳二模 Q20`：材料是完善全球气候治理。细则抓手先写原则：`平等`、`开放`；再写气候变化是全球挑战，要建立公平合理、合作共赢的全球气候治理体系，不同国家特别是发达国家要落实责任，最后上升到命运共同体。回填时放在“政治多极化 -> 全球治理与气候议题”。

- `2025石景山一模 Q17(2)`：材料是《和平共处五项原则发表70周年纪念大会北京宣言》。细则抓手是概述中国主张：共商共建共享、国家一律平等、国际上的事由各国平等协商、推进国际关系民主化、推进普惠包容的经济全球化、践行真正的多边主义。回填时放在“政治多极化 -> 完善全球治理的中国主张”。

- `2025丰台期末 Q20`：材料是中非在发展、安全、文明、全球治理等方面达成一系列战略性共识。细则抓手分为理论逻辑和价值意蕴：理论逻辑写和平与发展、共同利益、中非命运共同体、平等互利、共商共建共享；价值意蕴写新型国际关系、中非携手推进现代化、命运共同体。回填时放在“政治多极化 -> 南南合作与中非命运共同体”。

- `2026石景山一模 Q20`：材料是从“共同”“开放”“包容”三个关键词中任选两个说明中国倡议如何助推亚太合作。细则抓手要求“一个关键词 + 一个学科术语 + 合理分析”。高频落点是：共同对应共同利益和共商共建共享；开放对应多边主义、开放型区域经济环境；包容对应经济全球化更加包容可持续、维护发展中国家正当权益。回填时放在“政治多极化 -> 亚太合作”。

### 5. 联合国

- `2025丰台二模 Q20`：材料是“为完善全球治理注入中国力量”。细则抓手分四角度：作为最大发展中国家推动全球南方现代化；积极参与联合国、金砖、G20、博鳌等国际组织；作为联合国安理会常任理事国遵循联合国宪章宗旨原则、践行多边主义；推动命运共同体，按平等开放合作共享原则推动全球治理更公正合理。回填时放在“联合国 -> 中国在多边平台中的作用”。

- `2026通州期末 Q20`：材料是“全球治理倡议正逢其时、指引方向、彰显担当”。细则抓手三层最清楚：`正逢其时` 写百年变局和治理赤字；`指引方向` 写主权平等、国际法治、多边主义、以人为本、行动导向五大原则，并强调与联合国宪章一脉相承；`彰显担当` 写这是中国提供的重要公共产品，为世界注入稳定性和确定性。回填时放在“联合国 -> 全球治理倡议”。

- `2026东城期末 Q20`：材料是四大全球倡议如何系统推动构建人类命运共同体。细则抓手是：发展倡议对应繁荣，安全倡议对应稳定，文明倡议对应互信，治理倡议对应公正；四大倡议相互促进、有机统一于命运共同体“五个世界”；材料层面还要点出“从中国主张到国际共识、从合作理念到共同行动”。回填时放在“联合国 -> 全球倡议的系统性”。

- `2026丰台期末 Q21`：材料是以“四大全球倡议为构建人类命运共同体凝聚磅礴合力”为主题写短评。细则抓手强调三维作答：`是什么`，从四大倡议整体看；`为什么`，结合当前世界形势写必要性；`有什么作用`，写其对命运共同体建设的意义。知识落点明确指向时代主题、多边主义、共商共建共享、新型国际关系。回填时放在“联合国 -> 四大全球倡议的整体论证”。

### 6. 中国

- `2025东城一模 Q20`：材料是元首外交“精彩”之处。细则抓手分三层：共同利益是合作基础；推动高水平对外开放和经济全球化朝开放包容普惠平衡共赢方向发展；维护发展中国家利益、推进全球治理向更公正合理方向发展；最后收束到中国特色大国外交新境界和命运共同体新实践。回填时放在“中国 -> 元首外交”。

- `2025门头沟一模 Q19`：材料是中国做“赋能型大国”的世界意义。细则抓手可以压成四条：释放超大规模市场红利、共享发展机遇；以科技帮助发展中国家提升自主发展能力、改善民生；提供国际公共产品和中国方案；倡导文明平等互鉴和全人类共同价值，推动国际秩序更公正合理。回填时放在“中国 -> 中国智慧与中国方案”。

- `2025顺义一模 Q20`：材料是“小而美”项目为何成为中国开展国际发展合作的亮点和名片。细则抓手分四面：项目本身聚焦紧迫需求、灵活可推广；对当地改善民生、促进就业和发展；对中国体现正确义利观、文明互鉴、大国担当、和平外交政策；对世界推动命运共同体、国际关系民主化和开放包容普惠的全球化。回填时放在“中国 -> 国际发展合作”。

- `2025朝阳二模 Q21`：材料是我国为什么要努力开创周边工作新局面。细则抓手分三个主体：对中国来说是发展繁荣基础、国家安全重点、外交全局首要、命运共同体关键；对地区来说有共同利益、区域经济一体化、和平稳定与周边命运共同体；对世界来说能推动新型国际关系、国际关系民主化和真正的多边主义。回填时放在“中国 -> 周边工作”。

- `2025东城期末 Q20`：材料是欧盟对中国新能源汽车征收反补贴税。细则抓手分三层：先写中国新能源汽车产业对世界经济与绿色转型的贡献；再批判欧盟行为属于贸易保护主义、破坏市场竞争与秩序；最后写中国要坚定维护本国利益，坚持互利共赢、通过磋商维护贸易自由便利和市场秩序。回填时放在“中国 -> 坚定维护发展利益”。

- `2025朝阳期末 Q21`：材料是如何通过中国特色大国外交为推进中国式现代化营造良好外部环境。细则抓手要求总分清楚：总体上写坚持党的领导、习近平外交思想、总体国家安全观、命运共同体；政治环境写平等有序多极化、全球南方、新型国际关系、国际关系民主化、全球治理体系改革；经济环境写普惠包容的经济全球化、高水平对外开放、新发展格局；文化舆论环境写文化自信、正确义利观、文明多样性、中国故事。回填时放在“中国 -> 特色大国外交”。

- `2026朝阳一模 Q20`：材料是中国为什么能为全球发展注入稳定性和正能量。细则抓手有三束：一是创新驱动和综合国力，为全球产业链供应链稳定提供支撑；二是分享减贫、生态治理、职业教育经验，体现义利观、大国责任、中国智慧，改善发展中国家民生并推动命运共同体；三是扩大自贸朋友圈、释放超大规模市场潜力、促进要素流动、推动经济全球化朝开放包容普惠平衡共赢方向发展。回填时放在“中国 -> 中国贡献与中国机遇”。

- `2026顺义一模 Q20`：材料是“科技小院”成为南南合作典范的深层逻辑。细则抓手是必答共同利益，然后从国际政治和国际经济两边展开：国际政治写和平与发展、命运共同体、合作共赢、义利观；国际经济写农业技术提升生产率、降低成本、培养人才、增强内生动力、推动可持续发展和普惠平衡共赢的经济全球化。回填时放在“中国 -> 南南合作 + 中国方案”。

## 三、高频重复点回灌

### 1. 经济全球化高频

- `贸易投资自由化便利化` 是出现频率最高的一条，海淀一模、丰台一模、西城二模、西城一模、门头沟一模都在反复考。
- `两个市场、两种资源` 已经不是点缀句，而是很多开放题的中层逻辑。
- `高水平对外开放 -> 更高层次开放型经济 -> 双循环` 已经形成固定链条。
- `产业链供应链韧性` 是 2025、2026 的明显高频词，常和 `安全`、`多元稳定`、`关键核心技术` 连用。
- `比较优势 + 资源配置 + 国际竞争新优势` 适合写“走出去”“区域合作”“自贸区”“外贸升级”。

### 2. 政治多极化高频

- `共商共建共享`、`真正的多边主义`、`国际关系民主化` 已经是全球治理题的硬核词组。
- `命运共同体` 越来越不是结尾空话，而是要和具体治理议题、地区合作、南南合作连在一起写。
- `全球南方` 的位置明显抬升，常与“发展中国家利益”“代表性与发言权”配套出现。

### 3. 联合国高频

- 遇到全球治理倡议、四大全球倡议、全球秩序题，要优先想 `联合国宪章宗旨和原则`。
- `主权平等`、`国际法治`、`多边主义` 已成为联合国模块的稳定答题抓手。

### 4. 中国模块高频

- `习近平外交思想`、`正确义利观`、`中国智慧中国方案`、`大国责任担当` 是高频收束词。
- `元首外交`、`周边工作`、`南南合作`、`国际发展合作` 是近两年中国模块最常见的材料入口。
- 中国题常见收束方式不是单写“中国很重要”，而是写 `既维护本国利益，又兼顾他国合理关切`。

### 5. 理论模块高频

- 关系型设问越来越多，直接记成三组：
  - 自力更生和对外开放
  - 发展和安全
  - 中国发展和世界发展
- `共同利益是合作基础` 是很多国际合作题的起笔句。
- `总体国家安全观` 不只出现在安全题，也经常嵌入产业链、开放、科技、周边工作题。

## 四、当前可直接反复用的答题链

- 开放类：`便利往来/规则对接/平台建设 -> 要素流动与资源配置 -> 贸易投资自由化便利化 -> 高水平对外开放 -> 开放型经济`
- 供应链类：`多元布局/区域合作/绿色转型/数字赋能 -> 产业链供应链韧性 -> 多元稳定经贸关系 -> 双循环`
- 全球治理类：`背景挑战 -> 共同利益/多边主义 -> 共商共建共享 -> 国际关系民主化 -> 命运共同体`
- 中国担当类：`中国经验/中国市场/中国倡议 -> 惠及发展中国家 -> 中国智慧中国方案 -> 大国责任 -> 命运共同体`
