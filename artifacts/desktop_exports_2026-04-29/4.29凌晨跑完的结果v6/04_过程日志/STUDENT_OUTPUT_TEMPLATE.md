# Student Output Template v6

本轮最终学生版必须同时覆盖：

- 主观题材料-知识触发链
- 哲学/文化选择题正确项触发链
- 哲学/文化选择题典型错肢识别链

学生版不放路径、行号、文件号、OCR、debug、source id。

## 主观题条目

```markdown
## 知识点名称

**来源题目**：年份区卷 阶段 第X题

**完整设问**：……

**材料触发点**：……

**为什么能想到这个原理**：……

**答案落点**：……
```

## 选择题正确项条目

```markdown
## 知识点名称

**来源题目**：年份区卷 阶段 第X题

**完整题干**：题干 + 必要选项，至少保留正确项。

**正确项**：……

**材料触发点**：……

**为什么能想到这个原理**：……

**答案落点**：……
```

## 选择题错肢条目

```markdown
## 错肢类型名称

**来源题目**：年份区卷 阶段 第X题

**完整题干**：题干 + 被分析错项。

**错肢触发点**：……

**为什么不能这样想**：……

**答案落点**：……
```

边界、证据等级、路径、阻塞项写入审计版和 CSV，不进学生版正文。
