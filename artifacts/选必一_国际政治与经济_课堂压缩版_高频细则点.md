# 选必一《国际政治与经济》课堂压缩版（高频细则点）

## 说明

- 本稿以 `选必一_国际政治与经济_2024-2026题源穷尽清单.md` 锁定的 `49` 道正式并入题为母表。
- 全稿共回收 `210` 条 `材料 -> 细则`，归出 `54` 个稳定答题点。
- 括号内数字表示该答题点在当前母表中的重复次数。
- 证据口径只用 `细则 / 评标 / 阅卷讲评中的明确给分口径`，不把普通参考答案静默升级为细则。
- 本稿不是逐题展开，而是把最适合上课直接讲的 `模块入口 -> 高频点 -> 高频链条 -> 代表题型` 压成一页页可讲内容。

## 总入口

- 先判模块：
  - 世界乱局、治理赤字、外交“变与不变”先看 `时代背景`
  - 国家为什么合作、怎么处理开放和安全、怎么理解共同利益先看 `理论`
  - 外贸、外资、平台、展会、自贸区、自贸港、企业出海、产业链、规则对接先看 `经济全球化`
  - 全球治理、气候治理、亚太合作、中非合作、国际秩序、公平正义、多边主义先看 `政治多极化`
  - 联合国、全球治理倡议、四大全球倡议、广泛认同、常任理事国、宪章宗旨原则先看 `联合国`
  - 元首外交、周边外交、大国担当、“小而美”、南南合作、中国为什么有作为先看 `中国`

- 新题三步：
  - 第一步先锁模块，不要一上来乱堆教材话。
  - 第二步从材料里圈关键词，判断先写 `背景 / 合作基础 / 开放举措 / 治理理念 / 中国角色` 哪一层。
  - 第三步用“先点题，再对应材料，最后收束”的顺序组织答案。

- 全书总收束：
  - `构建人类命运共同体（17）`
  - `高水平对外开放（14）`
  - `推动经济全球化朝更加开放、包容、普惠、平衡、共赢方向发展（13）`
  - `共同利益是国家间合作的基础（9）`
  - `贡献中国智慧和中国方案（9）`
  - `促进贸易投资自由化便利化（9）`
  - `共商共建共享的全球治理观（8）`
  - `时代主题：和平与发展（8）`
  - `坚持多边主义（7）`
  - `遵循联合国宪章宗旨和原则（7）`

## 时代背景

- 模块作用：
  - 这部分不是展开写措施，而是先交代“世界为什么到了必须合作、必须治理、必须回应共同挑战的阶段”。

- 常见材料入口：
  - 世界动荡、冲突、风险、治理赤字
  - 民主与全球治理、全球南方、人工智能与全球治理
  - 新中国外交“变”与“不变”

- 必写点：
  - `时代主题：和平与发展（8）`
  - `国际形势复杂严峻、全球挑战增多（3）`
  - `世界多极化深入发展（3）`
  - `构建人类命运共同体（17）`
  - `贡献中国智慧和中国方案（9）`

- 高频链条：
  - `世界乱象/共同挑战 -> 时代主题 -> 多极化/挑战增多 -> 各国共同利益 -> 命运共同体`
  - `外交“变” -> 国际格局变化 -> 外交思想升级 -> 外交“不变”中的独立自主与和平共处`

- 代表题型：
  - 世界大势/治理赤字类：`2024海淀二模 Q18(1)`、`2025东城二模 Q20`
  - 外交“变与不变”类：`2024海淀期中 Q21(2)`

- 学生先背熟：
  - 材料先写世界动荡、治理难题，就先把 `和平与发展` 和 `全球挑战增多` 写出来。
  - 材料强调“共同面对”“命运与共”，最后大概率要收束到 `构建人类命运共同体`。
  - 材料写新中国外交延续性，就不能只写“变”，还要落到 `独立自主的和平外交政策`。

## 理论

- 模块作用：
  - 这部分解决“国家为什么合作”“为什么不能脱钩断链”“开放和安全怎么同时抓”“中国为什么既自立自强又扩大开放”。

- 常见材料入口：
  - `脱钩断链`、`共赢链`
  - `开源合作`、`人工智能+`
  - 国家竞争、科技革命、产业变革
  - 外部环境严峻、风险挑战增多

- 必写点：
  - `共同利益是国家间合作的基础（9）`
  - `统筹发展和安全（3）`
  - `产业链供应链韧性和安全（7）`
  - `高水平对外开放（14）`
  - `充分利用两个市场、两种资源（5）`
  - `双循环新发展格局（5）`
  - `推动经济全球化朝更加开放、包容、普惠、平衡、共赢方向发展（13）`

- 高频链条：
  - `共同利益 -> 国家合作 -> 治理合作/新型国际关系 -> 命运共同体`
  - `安全底线 -> 产业链韧性 -> 高水平开放 -> 两个市场两种资源 -> 双循环`
  - `科技革命/国际竞争 -> 综合国力较量 -> 维护国家利益 -> 提升国际话语权`

- 代表题型：
  - 合作逻辑类：`2025延庆一模 Q20(2)`、`2026延庆一模 Q20(2)`
  - 开放与安全关系类：`2024朝阳一模 Q21`、`2026朝阳期中 Q17`
  - 外交理论与总遵循类：`2024海淀期中 Q21(2)`、`2025房山一模 Q20(2)`
  - 国际竞争与国家实力类：`2024顺义思政二模 Q19(2)`

- 学生先背熟：
  - 题目一旦问“为什么合作”，最稳的起笔就是 `共同利益是国家间合作的基础`。
  - 题目一旦出现“风险、冲击、断链、卡链”，不要只写开放，先写 `安全` 和 `韧性`。
  - 题目一旦出现“不能关起门来”，就要把 `开放` 和 `双循环` 连起来写。

## 经济全球化

- 模块作用：
  - 这部分是整本书最容易出题、也最容易得分的板块，核心是“货、钱、技术、规则、平台、市场、产业链如何在全球流动和配置”。

- 常见材料入口：
  - 免签、口岸签、通关便利、零关税
  - 展会、数贸会、链博会、开放平台
  - 自贸区、自贸港、东盟合作、区域合作
  - 外贸、外资、营商环境、制度型开放
  - 企业出海、海外设厂、中国标准走出去
  - 供应链重构、未来产业、规则对接

- 必写点：
  - `高水平对外开放（14）`
  - `推动经济全球化朝更加开放、包容、普惠、平衡、共赢方向发展（13）`
  - `促进贸易投资自由化便利化（9）`
  - `产业链供应链韧性和安全（7）`
  - `制度型开放、优化营商环境（6）`
  - `充分利用两个市场、两种资源（5）`
  - `双循环新发展格局（5）`
  - `形成国际竞争新优势（5）`
  - `发展更高层次开放型经济（3）`
  - `促进商品、服务、技术、资金、劳务等要素流动（3）`

- 高频链条：
  - `平台/政策便利 -> 贸易投资自由化便利化 -> 制度型开放 -> 高水平对外开放 -> 更高层次开放型经济`
  - `供应链风险 -> 韧性和安全 -> 双循环 -> 两个市场两种资源 -> 国际竞争新优势`
  - `企业出海/标准出海 -> 规则对接 -> 拓展国际市场 -> 提升竞争力 -> 合作共赢`

- 代表题型：
  - 平台与便利化类：`2024海淀一模 Q18(1)`、`2024朝阳期中 Q20(3)`、`2025丰台一模 Q20`
  - 产业链/未来产业/标准类：`2024丰台一模 Q20`、`2025朝阳一模 Q20`、`2026海淀一模 Q20`
  - 企业“走出去”类：`2024石景山一模 Q19(2)`、`2025西城期末 Q20(2)`
  - 区域合作/自贸区自贸港类：`2026西城一模 Q20(2)`、`2026门头沟一模 Q20`、`2026房山一模 Q19`

- 学生先背熟：
  - 材料一出现“签证便利、通关便利、降成本”，优先写 `贸易投资自由化便利化`。
  - 材料一出现“条例、规则、标准互认、营商环境”，优先写 `制度型开放`。
  - 材料一出现“断链、重构、替代、供应风险”，优先写 `产业链供应链韧性和安全`。
  - 材料一出现“出海、扎根、设厂、标准输出”，优先写 `高水平对外开放 + 国际竞争新优势`。

## 政治多极化

- 模块作用：
  - 这部分不是写中国单方作为，而是写世界政治格局和国际关系应该往什么方向走。

- 常见材料入口：
  - 全球治理、气候治理、亚太合作、中非合作
  - 霸权主义、强权政治、零和博弈、小圈子
  - 更公正合理的国际秩序
  - 平等协商、共商共建共享、真正的多边主义

- 必写点：
  - `构建人类命运共同体（17）`
  - `共商共建共享的全球治理观（8）`
  - `坚持多边主义（7）`
  - `相互尊重、公平正义、合作共赢的新型国际关系（5）`
  - `国际关系民主化（4）`
  - `反对霸权主义和强权政治（1）`

- 高频链条：
  - `全球挑战 -> 任何国家都不能单打独斗 -> 多边主义 -> 全球治理观 -> 命运共同体`
  - `批判霸权/小圈子 -> 国际关系民主化 -> 新型国际关系 -> 更公正合理的国际秩序`

- 代表题型：
  - 全球治理类：`2024朝阳二模 Q20`、`2025石景山一模 Q17(2)`
  - 中国答案与国际关系新思路类：`2024西城二模 Q19`
  - 区域合作类：`2025丰台期末 Q20`、`2026石景山一模 Q20`

- 学生先背熟：
  - 题目写“全球治理”，不是直接冲中国担当，而是先写 `多边主义` 和 `全球治理观`。
  - 材料批判霸权、小圈子、零和博弈，就顺着写 `国际关系民主化` 和 `新型国际关系`。
  - 这一模块最后往往要收束到 `命运共同体`，但前面一定要有秩序逻辑和治理逻辑。

## 联合国

- 模块作用：
  - 这部分核心不是泛泛写国际组织，而是写联合国框架下的全球治理、中国倡议和国际认同。

- 常见材料入口：
  - 联合国、安理会、常任理事国
  - 全球治理倡议、四大全球倡议
  - 为什么获得国际认同
  - 主权平等、国际法治、联合国宪章、多边主义

- 必写点：
  - `遵循联合国宪章宗旨和原则（7）`
  - `坚持多边主义（7）`
  - `共商共建共享的全球治理观（8）`
  - `构建人类命运共同体（17）`
  - `全球治理倡议/四大全球倡议（3）`
  - `维护发展中国家和全球南方共同利益（3）`
  - `积极参与国际组织和全球治理（2）`

- 高频链条：
  - `世界乱局/治理赤字 -> 联合国仍是核心平台 -> 宪章宗旨原则 + 多边主义 -> 全球治理观/全球倡议 -> 更公正合理治理体系 -> 命运共同体`
  - `国际广泛认同 -> 契合共同利益 -> 回应共同挑战 -> 提供中国方案 -> 形成国际共识和行动`

- 代表题型：
  - 联合国与中国作用类：`2025丰台二模 Q20`、`2025海淀二模 Q21`
  - 全球治理倡议认同类：`2026通州期末 Q20`、`2026海淀期中 Q22(1)`
  - 四大全球倡议类：`2026东城期末 Q20`、`2026丰台期末 Q21`

- 学生先背熟：
  - 一看到联合国，先想 `宪章宗旨原则`，再想 `多边主义`。
  - 一看到“为什么广泛认同”，先写 `契合共同利益、回应共同挑战`。
  - 一看到“四大全球倡议”，不能拆成口号，要写它们共同服务 `全球治理` 和 `命运共同体`。

## 中国

- 模块作用：
  - 这部分专门回答“中国为什么有作为”“中国做了什么”“中国这样做对世界有什么意义”。

- 常见材料入口：
  - 元首外交、周边外交、中国特色大国外交
  - 赋能型大国、小而美、南南合作、科技小院
  - 中国稳定性、正能量、周边命运共同体
  - 欧盟征税、中国市场开放、中国如何回应外部打压

- 必写点：
  - `贡献中国智慧和中国方案（9）`
  - `大国责任与担当（7）`
  - `正确义利观（5）`
  - `维护发展中国家和全球南方共同利益（3）`
  - `改善民生、促进共同发展（3）`
  - `习近平外交思想（2）`
  - `开放广阔中国市场、共享发展机遇（2）`
  - `维护本国利益，兼顾他国合理关切（2）`

- 高频链条：
  - `中国行动/外交布局 -> 共同利益 -> 中国方案/大国担当 -> 多边主义或开放合作 -> 命运共同体`
  - `援助合作/小而美 -> 民生改善 -> 共同发展 -> 正确义利观 -> 全球南方共同利益`
  - `应对打压/征税 -> 维护国家利益 -> 维护市场秩序 -> 扩大开放 -> 合作共赢`

- 代表题型：
  - 外交作为类：`2025东城一模 Q20`、`2025朝阳期末 Q21`、`2026朝阳期末 Q20`
  - 大国担当类：`2025门头沟一模 Q19`、`2024丰台二模 Q19`、`2026丰台一模 Q19`
  - 发展合作类：`2025顺义一模 Q20`、`2026顺义一模 Q20`
  - 对外经贸摩擦类：`2025东城期末 Q20`

- 学生先背熟：
  - 题目主语只要明确是中国，就别只写世界秩序，先写 `中国做了什么`。
  - 材料一出现“不附加条件、惠民生、共发展”，优先写 `正确义利观`。
  - 材料一出现“周边、元首外交、外交布局”，优先写 `中国特色大国外交` 和 `习近平外交思想`。

## 最后收束

- 备课时最值得反复讲的不是教材定义，而是这三句话：
  - 先判模块，再落主点。
  - 先对材料，再写术语。
  - 先把主链写稳，再用中国方案或命运共同体收束。

- 学生训练时最值得反复操练的是三种能力：
  - 看到材料先分清这是 `背景 / 理论 / 全球化 / 多极化 / 联合国 / 中国` 哪一类。
  - 看到关键词就能条件反射地调出 `3-5` 个稳定答题点。
  - 能把答案写成 `材料对应 -> 细则语言 -> 总体收束` 的顺序，而不是堆口号。
