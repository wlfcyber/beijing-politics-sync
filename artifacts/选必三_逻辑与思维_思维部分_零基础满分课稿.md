# 选必三《逻辑与思维》思维部分零基础满分课稿

## 这节课要达成的结果

- 不是把`科学思维`、`辩证思维`、`创新思维`的定义各背一遍。
- 是让学生听完之后，遇到没见过的新题，也能自己完成四步：
  - `独立审题`
  - `独立判模块`
  - `独立搭框架`
  - `独立把答案写成北京题细则喜欢的样子`
- 截至`2026-04-24`，本课稿建立在本机已穷尽并正式纳入的三年题源之上：
  - `A类主观题/子问 33道`
  - `B类选择题稳定思维信号 29道`
- `2026二模`截至`2026-04-24`尚未开考，不在本稿口径内。

## 第一段先这样讲

- 同学们，选必三思维部分和哲学题的处理方法是一样的，本质都不是死背概念，本质都是`触发`。
- 命题人不是在问你“你会不会背一个方法名”，而是在问你：`你看到这段材料，能不能立刻判断它逼你用什么方式想问题`。
- 所以后面做题，一律不要先想教材定义，要先看材料到底在干什么。
- 它如果在`调查、研判、测试、规划`，大概率先触发`科学思维`。
- 它如果在`统筹、协同、既要又要、由低到高、由旧到新`，大概率先触发`辩证思维`。
- 它如果在`跨界组合、文创设计、场景转换、旧资源新玩法、把方向反过来`，大概率先触发`创新思维`。
- 真正的分数，不是从“概念解释”里来，而是从`材料动作 -> 思维方法 -> 细则化表达`这条链里来。

## 一张判题地图

- 材料先写`调查研究、数据样本、实验验证、趋势研判、方案设计`
  - 先判`科学思维`
- 材料先写`多主体协同、多环节统筹、一盘棋、整体推进、动态变化`
  - 先判`辩证思维`
- 材料先写`文旅融合、跨界联结、场景创新、老资源新用法、反转原有关系`
  - 先判`创新思维`
- 材料同时出现`研究`和`协同`
  - 先把`科学思维`当总帽子，再往下拆到`分析与综合`或`辩证思维`
- 材料同时出现`未来布局`和`现实调查`
  - 常见写法是`科学思维 -> 超前思维`
- 材料同时出现`旧产业改造`和`新赛道布局`
  - 常见写法是`辩证否定 + 超前思维`
- 材料同时出现`跨界拼接`和`方案收束`
  - 常见写法是`联想思维 + 发散思维 + 聚合思维`
- 材料先给很多具体形态，再把它们提炼成一个核心概念，最后又落回一个更完整的整体图景
  - 优先判`思维抽象 -> 思维具体`

## 先给学生的总口令

- `先看材料动作，不先背概念。`
- `先定总帽子，再找小方法。`
- `先写材料在干什么，再写它为什么对应这种思维。`
- `科学思维常是总帽子，真正得分常靠往下拆。`
- `辩证思维常是统筹框架，创新思维常是解决方案。`
- `看到很多具体现象被提炼成一个更高层概念，再展开成完整图景，先想思维抽象和思维具体。`

## 做题总流程

1. 先圈材料里的动作词。
2. 再判它更像`科学`、`辩证`还是`创新`。
3. 再补小方法，比如`分析与综合`、`超前思维`、`辩证否定`、`联想思维`、`逆向思维`。
4. 最后把答案写成一句完整的话，而不是只报名词。

## 零基础最低配答题句式

- `材料里……，说明它不是凭主观想象处理问题，而是在……地想问题。`
- `这体现出……思维。`
- `具体看，材料先……，再……，所以又触发了……。`

## 第一讲 科学思维

### 你就这样讲

- 科学思维最常见的入口，不是“科学”两个字，而是`调查`、`数据`、`方案`、`验证`、`预测`。
- 只要材料里出现“先去摸清真实情况，再作判断，再形成方案”，你就先往科学思维走。
- 科学思维最稳不是空写大帽子，而是拆成三条：
  - `追求认识的客观性`
  - `对未来有预见性`
  - `能在实践中检验和修正`

### 板书

- `客观性`
- `预见性`
- `可检验性`
- `分析与综合`
- `超前思维`

### 看到这些材料就进这个模块

- 问卷
- 访谈
- 调研
- 数据
- 样本
- 实验
- 反复测试
- 迭代
- 提前布局
- 趋势研判

### 必背句

- 材料立足客观实际，通过调查研究、掌握一手材料来认识问题，体现了`科学思维追求认识的客观性`。
- 材料不是停留在对现状的描述上，而是根据趋势和条件作前瞻判断，体现了`预见性`和`超前思维`。
- 材料在实践中反复测试、修正、优化方案，体现了`科学思维的可检验性`。
- 材料先分析各要素，再综合形成方案，体现了`分析与综合统一`。

### 新题怎么写

- 材料写`调研 + 数据 + 方案`
  - 第一层写`客观性`
  - 第二层写`分析与综合`
  - 如果有“未来布局”，第三层写`超前思维`

- 材料写`现实问题 + 趋势判断 + 长远规划`
  - 第一层写`立足现实`
  - 第二层写`预见未来`
  - 第三层写`前瞻布局`

- 材料写`试点 + 反馈 + 改进`
  - 第一层写`实践检验`
  - 第二层写`动态优化`

### 三年最典型题链

- `2024丰台一模 Q19(2)`
  - 垃圾分类研究不是直接喊口号，而是`查法条、看样本、补方法、形成提案`
  - 这道题教学生：凡是`研究型`主观题，先写科学思维

- `2025丰台一模 Q18(1)`
  - 人工智能发展题最稳的写法就是`客观性 + 预见性 + 可检验性`
  - 这道题适合当科学思维的标准模板题

- `2026顺义一模 Q19(2)`
  - 具身机器人研发先去养老场景蹲点，再反复测试迭代
  - 这道题把`真实需求`、`趋势跟踪`、`测试修正`三条全补齐了

- `2026海淀期末 Q20(2)`
  - 城市书房题不是拍脑袋提建议，而是`先看需求，再看矛盾，再做中长期规划`
  - 这道题最适合训练`超前思维应用题`

- `2026丰台期末 Q20`
  - 青海治沙题先建立资源库、掌握物种生存策略，再因地制宜构建体系
  - 这道题提醒学生：科学思维常是总帽子，但后面往往要继续拆到辩证思维

### 这里最容易错

- 只写“体现了科学思维”，不往下拆。
- 只看见“未来”就乱写超前思维，不写现实调查和客观依据。
- 把“有创新”直接等于科学思维，忽视它到底是`研究`还是`跨界设计`。

## 第二讲 辩证思维

### 你就这样讲

- 辩证思维最稳的判断标准，不是抽象定义，而是材料里有没有`关系`。
- 只要题目在处理`整体和部分`、`当前和长远`、`机遇和挑战`、`传统和未来`、`量变和质变`，就往辩证思维走。
- 北京题近三年最爱考的，不是空泛“联系发展”，而是四个稳定落点：
  - `分析与综合`
  - `整体性`
  - `动态性`
  - `辩证否定`

### 板书

- `一分为二`
- `分析与综合`
- `整体性`
- `动态性`
- `质量互变`
- `辩证否定`

### 看到这些材料就进这个模块

- 既要……又要……
- 当前和长远
- 机遇和挑战
- 转型升级
- 系统工程
- 协同治理
- 渐进推进
- 从低级到高级
- 传统和未来一起写

### 必背句

- 材料不是孤立地看问题，而是把多个要素放在联系和发展中统一把握，体现了`辩证思维`。
- 材料先拆各部分、各层次、各矛盾，再统摄成整体方案，体现了`分析与综合统一`。
- 材料统筹当前和长远、局部和整体，体现了`整体性和动态性`。
- 材料不是简单抛弃旧事物，而是在保留合理因素的基础上推动更新发展，体现了`辩证否定`。
- 材料强调由积累到突破、由不均衡走向更均衡，体现了`质量互变规律`。

### 新题怎么写

- 材料写`多主体协同`
  - 第一层写`分析与综合`
  - 第二层写`整体性`

- 材料写`当前问题 + 长远布局`
  - 第一层写`动态性`
  - 第二层写`整体推进`

- 材料写`旧产业升级 + 新产业布局`
  - 第一层写`辩证否定`
  - 第二层写`超前思维`

- 材料写`机遇和挑战并存`
  - 第一层写`矛盾分析法`
  - 第二层写`分析与综合`
  - 如果有“积累后突破”，第三层写`质量互变`

### 三年最典型题链

- `2024海淀一模 Q17(2)`
  - 京津冀协同发展题就是`先分析三地功能，再综合成整体格局`
  - 这是`分析与综合`的标准题

- `2024东城一模 Q18(3)`
  - 传统产业和未来产业不是二选一，而是`旧的升级 + 新的布局`
  - 这是`辩证否定 + 超前思维`的标准组合

- `2025海淀二模 Q20`
  - 共同富裕题把`全民共享、全面共享、共建共享、渐进共享`一起铺开
  - 这道题最适合训练`先分后合`和`渐进共享`的写法

- `2025延庆一模 Q18`
  - 低空经济既有潜力也有短板，属于最典型的`一分为二看问题`
  - 如果材料还写到由小到大推进，就顺带补`质量互变`

- `2026朝阳期中 Q20`
  - 机遇和挑战题是近三年辩证思维最完整的一道
  - `矛盾分析法 + 分析与综合 + 质量互变 + 动态性`都能在这题里练出来

- `2026房山一模 Q18(1)`
  - 空气治理题里`整体推进`、`日积月累`、`分区分类`三条分别对应三个辩证落点
  - 这道题特别适合教学生怎么从材料动作反推方法名

### 这里最容易错

- 一看到“发展”就只写发展观点，不写具体是哪种辩证方法。
- 把`辩证思维`和`科学思维`并列乱报，不知道前者常常是后者往下拆出的主落点。
- 只写“全面”“系统”这类空话，不落到`分析与综合`、`整体性`、`动态性`这些细则能识别的词上。

## 第三讲 创新思维

### 你就这样讲

- 创新思维最怕学生看见“创新”两个字就乱写。
- 北京题里的创新思维，真正稳定的不是空喊创新，而是三类动作：
  - `把不相连的东西连起来`
  - `把可能性先铺开，再收束成方案`
  - `把原来的方向、关系、状态反过来`
- 所以创新思维最常见的落点是：
  - `联想思维`
  - `发散思维`
  - `聚合思维`
  - `逆向思维`
  - `迁移和想象`

### 板书

- `联想`
- `发散`
- `聚合`
- `逆向`
- `迁移`
- `想象`

### 看到这些材料就进这个模块

- 文旅融合
- 文创设计
- 沉浸式体验
- 跨界组合
- 老资源新场景
- 冷资源变热经济
- 人找书变书找人
- 观众变参与者

### 必背句

- 材料把不同领域、不同资源、不同元素重新联结起来，体现了`联想思维`。
- 材料围绕同一问题从多个角度生成可能方案，体现了`发散思维`。
- 材料再把多个创意收束成可落地方案，体现了`聚合思维`。
- 材料把原有方向、关系或状态反过来处理问题，体现了`逆向思维`。
- 材料把已有经验、文化元素、历史符号迁移到新情境中，并通过想象形成新场景，体现了`迁移和想象`。

### 新题怎么写

- 材料写`文化资源 + 新消费场景`
  - 第一层写`联想思维`
  - 第二层写`迁移和想象`

- 材料写`提出多种方案，最后形成一个重点方案`
  - 第一层写`发散思维`
  - 第二层写`聚合思维`

- 材料写`服务关系倒过来`、`冷资源变热经济`
  - 第一层写`逆向思维`

- 材料写`文创、展陈、沉浸式体验`
  - 第一层写`联想`
  - 第二层写`想象`
  - 如果方案很多，再补`发散聚合`

### 三年最典型题链

- `2024朝阳期中 Q19`
  - 首发经济题一题带出`超前、逆向、联想、发散聚合`
  - 这道题告诉学生：创新思维往往不是单点，而是组合动作

- `2025房山一模 Q16(3)`
  - 遗址“热起来”这类题，核心不在背文化意义，而在`怎么把老资源接进新场景`
  - 这是`联想 + 发散`的代表题

- `2025昌平二模 Q19`
  - 沉浸式演艺把观众从“看戏的人”变成“入戏的人”
  - 这道题是`逆向思维`最标准的练习题

- `2025东城期末 Q18(2)`
  - 登月服设计把功能、环境、科技形象、传统文化元素一起统筹
  - 这道题适合练`联想/迁移想象 + 发散聚合 + 超前`

- `2025海淀期末 Q18`
  - 北京城市图书馆从`人找书`到`书找人`
  - 这是最典型的`逆向思维`材料

- `2026朝阳一模 Q17(1)`（后半）
  - 铁路博物馆文创设计题，本质就是把馆藏元素迁移到消费场景中
  - 这是`联想 + 想象`的标准题

- `2026朝阳期中 Q21(2)`
  - 冰雪资源、音乐、人文叙事、消费期待、场景转化一起出现
  - 这道题是创新思维复合写法的代表题：`超前 + 联想 + 逆向 + 发散聚合`

### 这里最容易错

- 只要题里有“创新”二字就报创新思维，不看材料动作。
- 只写“联想思维”四个字，不写它到底联了什么。
- 把`发散思维`和`聚合思维`割裂开，不知道北京题常常是先发散再聚合。

## 第四讲 思维抽象与思维具体

### 你就这样讲

- 这一块是选择题里最容易丢分、但又最值得单独讲透的点。
- 学生最容易被`联想`、`系统整合`、`分析综合`这些词吸走，但题目真正要考的，可能是`思维抽象与思维具体`。
- 零基础先记一句最短的话：
  - `看见一堆具体东西 -> 提炼出一个核心概念 -> 再把这个概念组织成更完整的整体图景`
  - 这就是`感性具体 -> 思维抽象 -> 思维具体`

### 板书

- `感性具体`
- `思维抽象`
- `思维具体`
- `先分离本质`
- `再回到整体`

### 最小定义

- `感性具体`
  - 就是材料先给你看到的那些丰富、杂多、直观的对象、现象、形态和关系。
- `思维抽象`
  - 就是从这些杂多现象里抽出关键属性、本质规定、核心概念。
- `思维具体`
  - 不是回到原来那种零散直观，而是把前面抽出的本质重新组织起来，形成更深刻、更完整的整体认识。

### 给学生的审题抓手

- 第一步，先看题干是不是先铺开了很多具体对象。
- 第二步，再看这些对象是不是被压成了一个更高层的概念、理念、主题或结构。
- 第三步，再看题干是不是又把这个概念展开成一个更完整、更系统、更丰富的整体图景。
- 如果这三步都在，优先考虑`感性具体 -> 思维抽象 -> 思维具体`。

### 和相近选项怎么区分

- 它和`分析与综合`的区别
  - `分析与综合`重在`拆部分 -> 合整体`
  - `思维抽象与思维具体`重在`从杂多现象里抽本质 -> 再按本质重建整体`
  - 看到题目在强调`概念提炼、本质把握、理念生成`，优先后者

- 它和`系统整合`的区别
  - `系统整合`重在`把多个要素协同起来形成机制`
  - `思维抽象与思维具体`重在`认识过程`
  - 如果题目问的是“怎样形成更深刻认识”，不是“怎样搭一个治理机制”，就不要被系统整合带跑

- 它和`联想/类比推理`的区别
  - `联想/类比`要看有没有`由此及彼`、`借甲说乙`、`把甲经验迁到乙`
  - `思维抽象与思维具体`不要求找另一个参照物，它是对同一对象认识不断深化
  - 没有明显“拿一个东西去比另一个东西”，就不要轻易选类比推理

- 它和`发散与聚合`的区别
  - `发散与聚合`重在`方案生成`
  - `思维抽象与思维具体`重在`认识提升`
  - 题目如果不是在出方案，而是在提升对一个对象的理解，优先排除发散聚合

### 必背句

- 材料先呈现丰富的具体形态，再从中提炼出核心理念，并进一步形成对对象更完整的整体认识，这体现了`从感性具体到思维抽象再到思维具体`。
- 这里不是简单把资源拼在一起，也不是只做系统整合，而是在认识过程中由现象上升到本质、再由本质回到更深刻的整体。

### 必讲例题：`2026通州期末 Q11`

- 题干关键信息：
  - 大运河通州段的“河—合—和”理念
  - 先给出`水脉、文脉、人脉`
  - 再上升为一个`河—合—和`的总理念
  - 再展开成`文化之河、生态之河、发展之河、民生之河`

- 正确识别顺序：
  - `水脉、文脉、人脉`是`感性具体`
  - `河—合—和`是从杂多具体现象中提炼出的`思维抽象`
  - `文化之河、生态之河、发展之河、民生之河`是按这个理念重新组织出来的`思维具体`

- 为什么不能选 `D`
  - 题干里虽然有“融于一身”，容易让人联想到系统整合
  - 但题目核心不是“多个部门怎样协同搭机制”，而是“对大运河这一对象的认识怎样从杂多现象上升为一个更深刻的整体”
  - 题干也没有明显“拿甲比乙”的推理动作，所以`类比推理`不成立

- 为什么 `B` 也不够准
  - `分析到综合`只说对了一半
  - 它没把`先抽出本质概念，再回到更深刻整体`这一层说出来
  - 这道题的关键不只是综合，而是`抽象链`

### 选择题陷阱口令

- `看到很多元素合在一起`，先别急着选系统整合
- `看到不同资源联结`，先别急着选联想思维
- `看到整体图景`，先别急着选分析综合
- 先追问一句：
  - `题干有没有把杂多对象先压成一个概念，再用这个概念展开更完整认识`
- 如果有，优先选`思维抽象与思维具体`

### 专门训练

- 题干若长成：
  - `先列很多具体面向/要素/脉络`
  - `再提出一个总理念、总命题、总结构`
  - `最后把它展开为多重价值或完整图景`
- 标准优先项：
  - `感性具体 -> 思维抽象 -> 思维具体`

- 题干若长成：
  - `多个主体、多个部门、多个环节协同工作`
  - `形成一套运行机制或治理体系`
- 标准优先项：
  - `系统整合`或`分析与综合`

- 题干若长成：
  - `把甲的经验、形态、特征迁到乙`
  - `由一个对象想到另一个对象`
- 标准优先项：
  - `联想思维`或`类比推理`

### 这里最容易错

- 一看到“和”“合”“融于一身”就直接冲系统整合。
- 只看到题目在“合”，没看到它先做了“抽”。
- 只会认`分析综合`，不会继续追问有没有`从现象到本质再回到整体`。

## 第五讲 复合题怎么拆

### 先告诉学生一个原则

- 近三年尤其是`2026`，命题越来越不爱出单方法题。
- 所以做复合题时，不要把所有方法平铺并列。
- 最稳的写法是三层：
  - `总帽子`
  - `主落点`
  - `补充点`

### 复合题固定写法

- 第一句：先给总帽子。
  - `材料体现了科学思维/辩证思维/创新思维。`
- 第二句：写主落点。
  - `具体看，材料先……，体现了……。`
- 第三句：再补充次落点。
  - `同时，材料还……，因此又体现了……。`

### 三道必须反复讲的复合题

- `2025门头沟一模 Q21(1)`
  - 最稳拆法：
    - `破路径依赖`写`辩证否定`
    - `产业生态社会一体重构`写`整体性/动态性`
    - `文旅农林商跨界`写`联想/发散`
    - `布局人工智能`写`超前思维`
    - `立足现实资源约束`再回到`科学思维`

- `2026海淀一模 Q17(2)`
  - 最稳拆法：
    - 研究设计是否真实可靠，写`科学思维`
    - 多方法是否匹配问题，写`分析与综合`
    - 信息是否充分展开再收束成方案，写`发散与聚合`

- `2026丰台期末 Q20`
  - 最稳拆法：
    - `种质资源库和生存策略研究`写`科学思维客观性`
    - `没有万能模式、因地制宜分类治理`写`辩证思维`
    - `多层防护体系和精准防治`可以继续补`创新思维`

## 第六讲 满分答案怎么组织

### 一般题三段式

- 第一段：`先点总帽子`
- 第二段：`按材料顺序拆主落点`
- 第三段：`收束到题目要求`

### 可直接套写的总模板

```text
材料中，……，说明其不是孤立、主观地处理问题，而是在尊重实际、把握联系、统筹发展的基础上思考和解决问题，这体现了……思维。
具体看，材料先……，体现了……；又……，体现了……。
因此，材料表明……，有助于……。
```

### 科学思维模板

```text
材料立足客观实际，通过……获取真实信息，并在此基础上分析问题、形成方案，体现了科学思维。
其中，……体现了追求认识的客观性；……体现了预见性/超前思维；……体现了在实践中检验和优化方案。
```

### 辩证思维模板

```text
材料不是片面地处理问题，而是把……放在联系、发展、整体中统一把握，体现了辩证思维。
其中，……体现了分析与综合/整体性；……体现了动态性/质量互变；……体现了辩证否定或矛盾分析法。
```

### 创新思维模板

```text
材料通过重新联结不同资源、拓展多种可能并形成新方案，体现了创新思维。
其中，……体现了联想思维；……体现了发散与聚合；……体现了逆向思维或迁移想象。
```

### 思维抽象与思维具体模板

```text
材料先呈现……等丰富具体的现象或要素，再从中提炼出……这一核心理念/本质规定，并进一步形成对……更完整、更深刻的整体认识，这体现了从感性具体到思维抽象再到思维具体。
这里既不是简单拼接要素，也不只是一般的分析综合，而是由现象上升到本质、再由本质回到整体。
```

## 第七讲 新题迁移训练稿

## 训练一 研究设计题

### 看到这种材料先怎么审

- 如果材料里有`调研、样本、问卷、实验、验证、测试、规划`，先不要慌。
- 先问自己三个问题：
  - 它有没有从真实情况出发
  - 它有没有根据现状推未来
  - 它有没有在实践中修正方案

### 先搭这个框架

- `客观性`
- `分析与综合`
- `预见性/可检验性`

### 直接这样写

- `材料通过……掌握真实情况，体现了追求认识的客观性。`
- `材料在分析各方面信息的基础上形成方案，体现了分析与综合。`
- `材料还根据……作出前瞻布局/通过……反复修正方案，体现了预见性或可检验性。`

## 训练二 协同治理题

### 看到这种材料先怎么审

- 只要材料铺开`多个主体`、`多个环节`、`多个层次`，又要求`统筹推进`、`系统治理`、`一盘棋`，就先往辩证思维走。

### 先搭这个框架

- `分析与综合`
- `整体性`
- `动态性`

### 直接这样写

- `材料先分析……的不同功能和作用，再把各部分放回整体布局中统筹推进，体现了分析与综合和整体性。`
- `材料不是一次性解决问题，而是在发展变化中持续推进，体现了动态性。`

## 训练三 转型升级题

### 看到这种材料先怎么审

- 只要材料出现`老产业、旧模式、传统资源`，又出现`升级、转型、新赛道、未来布局`，就不要写成简单淘汰旧事物。

### 先搭这个框架

- `辩证否定`
- `超前思维`
- `整体推进`

### 直接这样写

- `材料不是简单否定原有基础，而是在保留合理因素的基础上推动升级改造，体现了辩证否定。`
- `材料又围绕未来趋势和新需求提前布局，体现了超前思维。`

## 训练四 文旅文创题

### 看到这种材料先怎么审

- 只要题目问`怎么让文化资源火起来`、`怎么做文创`、`怎么把人文底蕴转成发展优势`，先不要背文化意义。
- 这类题首先问的是：`你有没有把老资源接进新场景。`

### 先搭这个框架

- `联想思维`
- `迁移和想象`
- `发散与聚合`
- `逆向思维`

### 直接这样写

- `材料把……与……跨界联结，体现了联想思维。`
- `材料把原本静态的文化资源转化为沉浸式、体验式、消费式场景，体现了迁移和想象。`
- `材料从多个角度开发方案并形成重点路径，体现了发散与聚合。`
- `若材料出现人找服务变服务找人、观众变参与者、冷资源变热经济，则补写逆向思维。`

## 训练五 未来布局题

### 看到这种材料先怎么审

- 只要材料里有`未来产业`、`新赛道`、`抢占先机`、`趋势变化`、`中长期规划`，就稳定考虑`超前思维`。

### 先搭这个框架

- `立足现实`
- `分析条件`
- `推理和想象`
- `前瞻布局`

### 直接这样写

- `材料不是脱离实际空想未来，而是在掌握现实条件的基础上分析趋势、预判方向，体现了超前思维。`
- `材料通过……提前布局……，有助于把握发展主动权。`

## 训练六 评析边界题

### 看到这种材料先怎么审

- 如果设问长成`“只要……就能……”`，基本都不是让你单边肯定。
- 这类题第一步不是直接反驳，而是先判断前半句有没有道理，再补它为什么不充分。

### 先搭这个框架

- `先肯定局部合理性`
- `再指出条件和边界`
- `最后补完整思路`

### 直接这样写

- `该观点看到了……的重要性，这有其合理性。`
- `但把……绝对化了，因为……还必须……。`
- `因此，应当把……与……统一起来，才能真正……。`

## 训练七 思维抽象与思维具体选择题

### 看到这种材料先怎么审

- 先看材料是不是给了很多零散而丰富的具体对象。
- 再看这些对象有没有被提炼成一个总理念、总概念、总命题。
- 最后看题干是不是又用这个总概念去重组出更完整的整体图景。

### 先搭这个框架

- `感性具体`
- `思维抽象`
- `思维具体`

### 先排这些高频干扰

- 只有“多要素协同”，没有“抽出核心概念”
  - 排除`思维抽象与思维具体`
- 只有“借甲说乙”或“迁移到别的对象”
  - 优先考虑`类比推理/联想`
- 只有“拆成几个部分再合起来”
  - 优先考虑`分析与综合`

### 直接这样判断

- `如果题干先列举丰富具体现象，再把这些现象提炼成一个总理念，并进一步展开成完整图景，就优先选“感性具体 -> 思维抽象 -> 思维具体”。`

### 反误选提醒

- `合`不等于`系统整合`
- `整体`不等于`分析综合`
- `联结`不等于`联想/类比`
- 先问是不是在讲`认识深化链`

## 第八讲 课堂上反复纠偏的错法

- 只写方法名，不解释材料为什么能触发这个方法。
- 看见“创新”就写创新思维，看见“发展”就写发展观点。
- 把`科学思维`、`辩证思维`、`创新思维`平排罗列，不分总帽子和主落点。
- 答案全是教材定义，没有顺着材料写。
- 只写一种方法，不会处理复合题。
- 评析题只会表态，不会写边界和补充条件。
- 选择题里只盯`合、融、系统、联结`这些表面词，不会继续判断是不是`先抽象再具体`。

## 最后一段这样收

- 选必三思维部分，真正决定分数的不是你记住了多少定义，而是你能不能在十几秒内完成一句判断：
  - `这段材料到底在逼我用什么方式想问题。`
- 以后所有题都按同一顺序做：
  - `先看动作`
  - `再判总帽子`
  - `再拆小方法`
  - `最后顺着材料组织句子`
- 你只要把这套顺序练熟，碰到新题就不会乱。
- 因为三年北京题反复考的，其实就是这些稳定触发链：
  - `研究验证 -> 科学思维`
  - `统筹协同 -> 辩证思维`
  - `跨界重组 -> 创新思维`
  - `旧中出新 -> 辩证否定`
  - `面向未来 -> 超前思维`
  - `先铺开再收束 -> 发散与聚合`
  - `把方向反过来 -> 逆向思维`
  - `先给杂多现象 -> 抽出核心概念 -> 回到完整图景 -> 思维抽象与思维具体`

## 课后自己复盘时只看这张口令表

- `调查、数据、测试、规划` -> `科学思维`
- `协同、整体、既要又要、渐进推进` -> `辩证思维`
- `跨界、文创、沉浸、反转关系` -> `创新思维`
- `未来布局` -> `超前思维`
- `旧资源升级` -> `辩证否定`
- `多方案生成再收束` -> `发散与聚合`
- `人找书变书找人` -> `逆向思维`
- `很多具体面向 -> 一个总理念 -> 更完整整体图景` -> `思维抽象与思维具体`
