# 必修四哲学材料-知识触发总框架（持续更新版）
**最后更新**：2026-04-24
**用途**：用于持续累计北京各区政治试卷中，必修四哲学部分的“材料—知识”触发链，防止后续增量分析时遗失已沉淀内容。
**框架来源**：依据《哲学与文化 2026班课》提炼的哲学框架整理。

## 使用规则
- 统一记录四项：来源、材料信息、触发知识（原理/方法论）、逻辑链。
- 纯文化项暂不并入本表，另行整理。
- 同一题若能稳定触发多个哲学点，分条归入对应模块。
- 暂时没有稳定挂点的知识条目保留空位，后续补入。
- 遇到扫描版细则或评标材料，必须先渲染/OCR/读图核定；不能因文本层无内容而悬置或跳过。
- 选择题中的哲学相关正确项，也必须按“材料信息—原理/方法论—逻辑链—来源题号”并入本表，不能只停留在错肢库。

## 课件总框架（便于后续增量更新时对齐）
### 一、唯物论
- 物质决定意识，意识对物质具有能动作用
- 一切从实际出发 / 实事求是
- 主观能动性 / 意识的能动作用
- 尊重客观规律与发挥主观能动性相结合
- 规律的客观性

### 二、辩证法
- 联系：联系的普遍性、联系的客观性、联系的多样性、整体与部分
- 发展：发展的观点 / 发展的普遍性、量变与质变、前进性与曲折性、辩证否定
- 矛盾：对立统一、矛盾的普遍性、矛盾的特殊性、矛盾普遍性和特殊性、主次矛、矛盾主次方面、两点论重点论

### 三、认识论
- 实践与认识
- 认识发展原理
- 真理观

### 四、历史唯物主义
- 社会存在与社会意识
- 社会发展的两大基本规律和基本矛盾
- 改革
- 人民群众

### 五、价值观 / 人生观
- 价值观的导向作用
- 价值判断与价值选择
- 实现人生价值

#### 物质决定意识，意识对物质具有能动作用
2. **来源**：2026北京西城高三（上）期末 第16题第（2）问（用户补充评分页截图）
   - 材料信息：“真实”被界定为客观事物及其规律的本来面目，只有如实反映客观事实和规律，作品与表达才能真正打动人心。
   - 触发知识：物质决定意识，意识是对客观存在的反映
   - 逻辑链：材料先把“真实”落在客观事物及其规律上，再谈其何以打动人心，说明作品的感染力首先来自对客观世界的如实把握，而不是主观臆造。只有主观认识与客观实际相符合，表达才有根基，因此这里稳定触发“物质决定意识，意识是对客观存在的反映”。

#### 一切从实际出发 / 实事求是
1. **来源**：2025北京丰台高三（上）期末 第16题（`丰台期末细则.pptx`）
   - 材料信息：“胸中有‘数’”要求在工作生活中做计划、定方案，在国家治理中看大数、看长远；评分标准把“数从哪里来”对应到实践、实际、规律、发挥主观能动性。
   - 触发知识：一切从实际出发 / 实事求是
   - 逻辑链：“数”不是主观臆想出来的，而是对客观情况、统计数据、百分比和数量界限的把握。材料要求从实际情况中取得“数”，再依据实际确定方案，因此触发“一切从实际出发、实事求是”。
2. **来源**：2025北京朝阳高三（上）期末 第16题（`朝阳期末评标.pdf`扫描页读图核定）

   - 触发知识：一切从实际出发 / 实事求是
   - 逻辑链：用文化同世界对话不能只按自我表达逻辑输出，而要同时看中国文化自身实际、传播对象的接受特点和国际交流语境。材料把“当地策展人、普通观众、现代生活契合点”放进来，说明对外文化交流要从具体对象和具体场景出发。
3. **来源**：2026北京朝阳高三一模 第1题
