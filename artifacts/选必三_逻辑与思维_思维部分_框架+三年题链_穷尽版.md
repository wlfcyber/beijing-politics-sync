# 选必三《逻辑与思维》思维部分框架+三年题链（穷尽版）

## 使用口径

- 本稿只处理`思维部分`，不展开纯`推理部分`。
- 时间范围锁定为`2024-2026`。其中`2026二模`截至`2026-04-23`尚未开考，不纳入本轮。
- 本轮以本地试卷、细则、讲评中`题号能锁定`的主观题或子问为准，共正式纳入`33`道。
- 写法统一压成：`材料场景 -> 触发的思维方法 -> 为什么是它`。
- 按你的课上口径，这一部分和哲学题一样，本质不是背方法名，而是看见材料就知道它在逼你调哪种思维方式。

## 零基础起手

- 第一步，不先背定义，先看材料到底在干什么：是`调查研究`、`提前布局`、`旧中出新`、`统筹协同`、`跨界组合`，还是`把原来方向反过来`。
- 第二步，先定`总帽子`，再定`小方法`。例如先看出是`科学思维`，再继续拆到`客观性`、`分析与综合`、`超前思维`。
- 第三步，答题不要只报方法名，要写成一句完整的话：`材料里……，说明它是在……，所以这里触发的是……`。
- 零基础最低配句式只有三段：
  - `材料先做了什么`
  - `这说明它不是拍脑袋，而是在用什么方式想问题`
  - `所以对应哪种思维方法`
- 真正容易丢分的不是不会背概念，而是把`科学思维、辩证思维、创新思维`乱并列，不知道哪一个是总帽子、哪一个是材料里的真落点。

## 一、总判断

- 看到`调查、研究、样本、方案、验证、研判`，先想`科学思维`，再往下细分到`分析与综合`、`超前思维`、`发散与聚合`。
- 看到`统筹、协同、一盘棋、多环节、多主体、系统工程`，优先触发`辩证思维`里的`整体性、动态性、分析与综合`。
- 看到`旧产业/旧模式/旧资源`遇到`转型、升级、破立、扬弃`，优先触发`辩证否定`。
- 看到`未来产业、趋势判断、提前布局、抢占先机、前瞻规划`，稳定触发`超前思维`。
- 看到`跨界融合、文旅+、科技+文化、老资源新场景、冷资源变热经济`，优先触发`联想思维`，再看是否还要补`发散与聚合`或`逆向思维`。
- 看到`从多路信息里先铺开，再收束成一个可执行方案`，就是`发散思维 -> 聚合思维`的标准链。
- 看到`把原来默认的方向、位置、状态、流程反过来想`，比如`人找书 -> 书找人`、`冷资源 -> 热经济`、`被动看 -> 主动沉浸`，往往就是`逆向思维`或`存在状态转换`。
- 科学思维经常作为总帽子出现，但真正在材料里拿分，往往还是要把它拆成`客观性`、`预见性`、`可检验性`，并落实到具体方法。

## 二、三年题链穷尽

### 2024

- `2024丰台一模 Q19(2)`：围绕垃圾分类做主题研究。材料一给的是`研究怎么做`，所以先触发`科学思维`；一旦出现问卷、访谈、资料搜集和判断预测，就自然连到`超前思维`与`分析、综合、比较、归纳`。

- `2024朝阳二模 Q19(1)`：“生生之宇宙观”体现了辩证思维的什么特征。虽然只是小问，但触发非常干净：只要材料强调`生生不息、变化流转、四时推移`，就稳定落在`辩证思维`的`动态性`。

- `2024海淀一模 Q17(2)`：京津冀协同发展如何书写新篇章。只要材料是`三地各有定位、最后写成整体新图景`，就不是单纯背协同口号，而是`先分析各自功能，再综合成整体格局`，稳定落在`分析与综合`。

- `2024东城一模 Q18(3)`：传统产业和未来产业是什么关系。凡是材料里同时出现`不能丢传统产业`和`又要布局未来产业`，就是典型的`辩证否定 + 超前思维`双触发，核心不是二选一，而是`改造提升旧的，同时前瞻布局新的`。

- `2024石景山一模 Q19(3)`：企业“走出去”过程中如何应对法律、标准、市场、文化等冲突。材料不是单点困难，而是多重摩擦并列，所以更适合触发`分析与综合`；如果从“既适应当地又保持自身优势”的角度切，还可连到`辩证否定`。

- `2024西城一模 Q19(5)`：怎样研判我国未来产业发展的方向。题面不是让你背产业名称，而是让你`根据现在推未来`，所以稳定触发`超前思维`，并要求补上`调查研究 + 矛盾分析 + 推理与想象`。

- `2024朝阳期中 Q19`：首发经济“北京朝外样本”如何体现创新思维。细则里已经把材料和方法对得很直：`业态规划`触发`超前思维`，`品牌引进`触发`逆向思维`，`策展型零售`触发`联想思维`，`价值耦合`触发`发散与聚合`。

- `2024丰台二模 Q18(2)`：评析“只要准确预测冰雪经济前景，就能推动其高质量发展”。这类题不是只写“要预测”，而是要先指出`超前思维重要但不够`，再补上`辩证思维、具体问题具体分析、尊重规律、从实际出发`，本质是给`超前思维`加边界。

- `2024海淀二模 Q17(1)`：时间利用调查为什么体现科学思维。细则已经很稳：`为了全面真实准确`触发`客观性`，`思路新、方法新、用新工具`触发`创新思维三新`，`分阶段实施、任务明确`触发`整体性/分析与综合`。

- `2024顺义二模 Q16(2)`：如何运用超前思维推进“无废城市”建设。材料一边是现实问题，一边是系统化、科学化、长效化目标，标准写法就是`在尊重城市建设规律基础上作预判，把资源化、清洁生产、经济与社会效益统一起来`，本质是`超前思维`。

### 2025

- `2025东城一模 Q18(1)`：“两重”实施如何体现辩证思维。材料里出现`硬投资和软建设`、`当前和长远`、`重点领域和整体布局`，就是典型的`整体性 + 动态性 + 抓主要矛盾`。

- `2025丰台一模 Q18(1)`：人工智能发展为什么体现科学思维。细则给得很直：`从实际出发`对应客观性，`准确预判趋势、提前部署`对应预见性，`在实践中检验并不断完善`对应可检验性；同时还能向`超前思维`和`整体/动态`顺带延展。

- `2025延庆一模 Q18`：如何运用辩证思维应对低空经济挑战。只要材料同时写了`潜力大`和`基础薄弱、场景单一`，就会稳定触发`辩证思维`；再加上`稳步扩大规模、由小散走向大优`，自然连到`质量互变`。

- `2025房山一模 Q16(3)`：如何让琉璃河遗址“热”起来。只要是文化遗址、博物馆、传统资源“如何出圈”，基本就往`联想思维 + 发散思维`走，因为关键不在判断对错，而在跨界嫁接和多方案生成。

- `2025石景山一模 Q19`：科学建议奖获奖同学的建议如何体现科学思维。材料入口很典型：`问题意识 + 多元化方法 + 实验数据 + 从生活中捕捉灵感`，所以主轴仍是`科学思维`，但后半已经开始向`创新思维`外溢。

- `2025门头沟一模 Q21(1)`：门头沟从“千年煤城”转向“京西智谷”的智慧。这个题是近三年最典型的复合题之一：`破煤炭路径依赖`触发`辩证否定`，`产业—生态—社会协同`触发`整体性和动态性`，`文旅农林商跨界`触发`联想和发散`，`布局人工智能`触发`超前思维`，`立足资源约束现实果断转型`又回到`科学思维`。

- `2025昌平二模 Q19`：沉浸式演艺如何体现创新思维。题眼非常稳定：`观众由被动变主动`触发`逆向思维`，`从多角度开发传统经典`触发`发散与聚合`，`身临其境的新场景`又把`迁移与想象`带出来。

- `2025东城期末 Q18(2)`：登月服设计如何体现创新思维。材料里既有`飞天飘带+火箭尾焰`的文化元素重组，也有`材料、视效、功能`的技术整合，所以是`联想/迁移想象 + 创新思维三新`的标准场景。

- `2025海淀期末 Q18`：北京城市图书馆如何运用创新思维建设新型公共文化空间。细则已经锁得很清：`从“人找书”到“书找人”`触发`逆向思维`，`把“赤印”意象迁移进建筑设计`触发`联想思维`。

- `2025海淀二模 Q20`：如何运用辩证思维更好坚持共享发展理念、推进共同富裕。材料把`全民共享、全面共享、共建共享、渐进共享`四层一起铺开，所以首先稳定触发`分析与综合`与`整体性`；其中`渐进共享、从低级到高级、从不均衡到均衡`又明显对应`动态性/质量互变`；如果从“继承传统共享思想、回应新时代现实问题”的角度切，还可补`辩证否定`。

### 2026

- `2026东城一模 Q19(1)`：中关村特色产业园研究采用什么方法。材料明确是项目研究型设问，一看到`文献法之外还要补方法并说明理由`，就优先触发`调查研究法`；细则把它往`多种研究方法综合运用`和`分析与综合统一`上收。

- `2026东城一模 Q19(4)`：“把‘1’拉长、推进”怎样体现系统观念与创新思维。材料本身就给出了`从单点突破到链式升级`，所以触发非常稳定：`系统集成`是总逻辑，`发散拓展应用场景 + 聚合围绕成果转化整合资源 + 超前思维健全机制`是落点。

- `2026延庆一模 Q18(2)`：怎样看待虚拟数字人直播。细则显示这题不是简单支持或反对，而是`既看积极影响也看挑战`，稳定触发`辩证思维`；`支持发展同时适度监管`触发`适度原则`；`鼓励创新同时完善规则`触发`创新思维`，甚至可进一步连到`辩证否定`。

- `2026房山一模 Q18(1)`：北京空气治理路径如何体现思维方法。细则给出三条非常标准的治理路径：`整体推进`对应`整体性/分析与综合`，`日积月累实现提升`对应`质量互变/动态性`，`分区分类、一企一策`对应`具体问题具体分析`。

- `2026朝阳一模 Q17(1)`：识别 AI 任务指令中的推理类型，并用联想思维设计铁路博物馆文创产品。这道题前半是推理，后半已经进入思维部分；真正稳定可复用的是后半，凡是`文创设计`题都容易触发`联想思维`中的`迁移和想象`。

- `2026海淀一模 Q17(2)`：围绕调研活动提出改进建议。虽然这题前面有逻辑错误辨析，但第二问已经明显进入思维部分：可以从`科学思维`、`辩证思维`、`发散与聚合`、`反向推导因果关系`几个方向落笔，本质是`研究设计题`。

- `2026石景山一模 Q17(2)`：为传承中医药文化提出两条建议。材料入口是`传统文化+健康中国+传播创新`，所以稳定触发`联想思维`、`发散思维`、`超前思维`三件套。

- `2026门头沟一模 Q18(2)`：法院解决环境纠纷过程中运用了哪些科学思维方法。虽然题面写的是`科学思维`，但细则实际要求学生把方法落到`辩证思维 + 创新思维`上去：既要`联系、发展、全面看问题`，又要`通过发散与聚合改造调解方案`。

- `2026顺义一模 Q19(2)`：具身机器人研发如何体现科学思维特征。材料从`走进社区和养老院蹲点观察`、`围绕真实需求设计功能`、`多轮测试迭代`、`追踪老龄化与具身智能趋势`展开，所以可稳定拆成`客观性 + 预见性 + 可检验性`三条。

- `2026朝阳期中 Q20`：如何面对新征程上的“机遇”与“挑战”。这是近三年最标准的辩证思维大题之一：`机遇和挑战对立统一`触发`矛盾分析法`，`先拆促机因素和风险因素再统摄全局`触发`分析与综合`，`脚踏实地积累再实现突破`触发`质量互变`，`顺应科技革命和国际变化`触发`动态性`。

- `2026朝阳期中 Q21(2)`：如何运用创新思维将人文底蕴转化为经济发展优势。细则已经把触发链写明：`预测消费趋势和旅游期待`触发`超前思维`，`冰雪资源+音乐+文化叙事`触发`联想思维`，`静态建筑 -> 动态文化场景`或`冷资源 -> 热经济`触发`逆向/状态转换`，如果再往上提炼，还能补`发散与聚合`。

- `2026海淀期末 Q20(2)`：围绕城市书房建设与运营提出建议。细则明说“符合超前思维的方法”，所以这题就是`超前思维`的标准应用题：`调查研究需求 -> 分析建设运营中的矛盾 -> 用推理和想象做前瞻规划`。

- `2026丰台期末 Q20`：青海防沙治沙之路体现了什么科学思维。材料先写`种质资源库、原生物种生存策略、为治沙提供科学依据`，所以总帽子一定是`科学思维`，核心落在`追求认识的客观性`；再看到`没有万能模式、各地因地制宜、构建多层防护体系`，就自然下沉到`辩证思维`里的`分析与综合、整体性、具体问题具体分析`；如果再往深处讲，`精准防治`与`多路径治沙`还可以继续连出`创新思维/发散与聚合/超前思维`。

## 三、三年重复点回灌

- `未来产业/新赛道/抢先机`已经成为思维部分最稳的入口，基本都往`超前思维`收。
- `转型升级`类材料越来越多，不再是简单写“发展新事物”，而是写`旧中有新、新中带旧`，所以高频落点是`辩证否定`。
- `区域协同、城市治理、生态治理、系统工程`类材料反复出现，核心不是背“系统观念”四个字，而是要会写`先分析各部分，再从整体上综合，最后说明如何统筹推进`。
- `文旅融合、文创设计、沉浸式体验`已经成为创新思维最稳定的场景库，几乎总会连出`联想、发散与聚合、逆向`中的两个以上。
- `科学思维`在近两年越来越像一个总帽子，经常和`超前思维`、`辩证思维`、`创新思维`套着考，而不是孤立考。
- `2026`相比`2024`、`2025`更明显的变化是：命题越来越爱把几种思维方法压在同一道材料里，不让学生只写单点概念。

## 四、可直接复用的触发链

- `现实问题很多 -> 做调查、看数据、查资料、做验证 -> 形成方案`
对应：`科学思维 -> 分析与综合 -> 可检验`

- `旧产业/旧资源/旧空间还在 -> 不能简单丢掉 -> 要升级成新业态/新动能`
对应：`辩证否定`

- `未来需求不确定 -> 技术路线在变 -> 必须提前布局`
对应：`超前思维`

- `材料里有很多主体、环节、层次 -> 题目又问协同、统筹、系统、整体`
对应：`分析与综合 + 整体性 + 动态性`

- `传统文化/地方资源/老场景要出新`
对应：`联想思维`

- `先尽量铺开可能性 -> 再收束成一个重点方案`
对应：`发散思维 -> 聚合思维`

- `把原来的顺序、方向、状态调过来`
对应：`逆向思维`

## 五、三年走势

- `2024`还比较像“单方法练熟”：`超前思维`、`分析与综合`、`发散与聚合`各自单独立题的情况较多。
- `2025`开始明显复合化：一道题里经常出现`科学思维统摄 + 辩证/创新分层展开`。
- `2026`则更进一步，不只问“是什么方法”，而是更爱问`为什么这样统筹、如何把几种方法一起调出来`。

- `2026西城一模`：本地已看到明显的思维部分选择题信号，但本轮未锁定到稳定的思维部分主观设问，因此没有硬并进来。

## 七、现阶段可下的总结

- 选必三思维部分最怕的不是不会背概念，而是`不会把材料翻译成方法`。
- 近三年北京题已经很清楚：命题人越来越爱把`科学思维`当总帽子，把`辩证思维`当统筹框架，把`创新思维`当解决方案。
- 所以课上真正要讲透的，不是一个个方法孤立定义，而是`什么材料形态先触发哪种方法，为什么是它，不是别的`。
