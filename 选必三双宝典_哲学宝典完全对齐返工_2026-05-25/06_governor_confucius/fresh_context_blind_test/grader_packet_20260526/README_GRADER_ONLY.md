# README_GRADER_ONLY

purpose: `fresh_context_blind_test_grading`

本包只给评分人使用，不得给学生模拟器使用。评分前必须先取得学生包生成的原始答卷，不能让学生看到本包内容。

## 评分任务

1. 打开学生原始答卷。
2. 使用 `GRADER_ANSWER_KEY_AND_RUBRIC_20260526.md` 逐题评分。
3. 输出 `PASS`、`CONDITIONAL_PASS` 或 `FAIL`。
4. 如果不是 `PASS`，必须写清错因对应到两本宝典的哪个节点需要补。

## 通过标准

- A 卷 4 题均能从材料动作进入思维方法，并写出卷面答案。
- B 卷 4 题均能判定推理形式或概念关系，并说明有效/无效理由或错项错因。
- 不要求逐字一致；等值表达可以通过。
- 只给结论、没有材料信号或理由，不通过。
