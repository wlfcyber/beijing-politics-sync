## 二、思维宝典差距

verdict: `REWRITE_REQUIRED`

旧思维宝典已经按“思维类型/小方法”分了一层，但仍不像哲学宝典，主要问题是：

1. 开头缺哲学宝典式封面、前言和页码目录。
2. 正文数量只有 44 题，无法形成哲学宝典那种“节点下密集挂同类题”的厚度。
3. 存在 `使用口令`、`触发口令`，更像讲义卡片，不像哲学宝典正文。

5. 部分表述仍在解释“主线不能夺走”“推理册再处理”等后台分册逻辑，破坏学生版纯净度。
6. `为什么能想到` 有时已经接近哲学宝典，但不少条目仍混入模块裁决话术，而不是纯材料到方法的自然触发。
7. `答案落点` 有些句子可写，但还没有形成哲学宝典那种“每个节点下反复训练同一方法”的稳定复现。

- `## 使用口令`
- `触发口令：`
- `科学总帽下的复合触发`

这些可以保留到教师备课说明或审计包，但不能进入最终学生正文。

## 三、推理宝典差距

verdict: `STRUCTURE_REWRITE_REQUIRED`

旧推理宝典覆盖了一批主观题和选择题，且补了完整选项、正确理由、诱人错项和错因。但它仍然不等于哲学宝典式宝典，主要问题是：

1. 开头是 `题型速查`，不是哲学宝典式封面、前言、页码目录。
2. 正文先讲 `规则口令`、`有效式/无效式`，再挂题；读感像形式逻辑规则索引。
3. 大量 `p -> q`、`therefore` 等公式写法直接暴露在学生正文前台，削弱飞哥式中文讲解感。
4. 主观题字段是 `推理形式 / 规则口令 / 有效式/无效式 / 材料或结构 / 答案落点`，缺少哲学宝典的 `题干触发点 / 设问 / 为什么能想到`。
5. 选择题虽然有 `完整题干 / 完整选项 / 答案 / 正确理由 / 诱人错项和错因`，但没有被统一改造成“看题干信号怎样想到该推理形式”的链条。
6. 规则说明压过题目讲解，导致学生容易背规则，不能像哲学宝典那样从材料或题干自然触发。

规则和有效式只作为 `【为什么能想到】` 内部的中文解释，不能单独把整本书做成公式规则表。

### 推理样章

至少覆盖：

- 充分条件假言推理；
- 必要条件假言推理；
- 三段论周延；
- 类比推理；
- 归纳推理；
- 概念外延/定义；
- 矛盾律、排中律或同一律。

样章通过标准不是“标题齐”，而是读起来像哲学宝典：学生能从题干信号自然想到推理形式，再写出卷面判断。
