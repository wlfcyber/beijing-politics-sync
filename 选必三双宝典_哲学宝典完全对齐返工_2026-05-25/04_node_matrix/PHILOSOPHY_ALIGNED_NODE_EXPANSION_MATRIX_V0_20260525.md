# 一、入正文硬门禁

## 思维宝典

入正文条件：

1. 必须是主观题；思维选择题不进正文，只可留边界或转入推理宝典。
2. 必须锁定题面、材料、设问、答案或细则。
3. 必须能写成哲学宝典四件套：【材料触发点】、【设问】、【为什么能想到】、【答案落点】。
4. 目录按 思维类型 -> 小方法/触发链 -> 主观题 展开。
5. 同一题可在不同小节点重复出现，但每次必须围绕当前节点重写触发链，不能复制粘贴。

不进正文条件：

- 选择题中的思维抽象、形象思维、边界辨析，只能进审计边界或推理宝典选择题区。
- 只锁到细则引用、未找到原卷设问的题，先写 prompt_pending，不能装作最终正文。

## 推理宝典

入正文条件：

1. 主观题和选择题都收。
2. 目录按 推理形式 -> 规则触发 -> 有效式/无效式 -> 题目 展开。
3. 主观题统一改成哲学式四件套：【题干触发点】、【设问】、【为什么能想到】、【答案落点】。
4. 选择题必须有：【完整题干】、【完整选项】、【答案】、【题干触发点】、【为什么能想到】、【正确理由】、【诱人错项和错因】。
5. 不使用前台污染词：使用口令、触发口令、p -> q、therefore、外审/后台编号。

# 四、正式扩写顺序

1. 先扩写思维宝典三大主节点：科学思维、辩证思维、创新思维。每个节点先放已锁主观题，不放选择题。
2. 再扩写推理宝典八大推理形式。每个推理形式至少先放一题主观题和一题选择题；如果某形式没有主观题，要在审计中说明。
3. 每写入一题，就在正文旁边保留后台审计映射：题源、设问锁定状态、答案/细则来源、证据等级；学生版不显示这些后台字段。
4. 全书 Markdown 成型后，再按哲学宝典页面参数重建 DOCX/PDF：A4、目录页码、页码、标题层级、正文密度、完整选项。
