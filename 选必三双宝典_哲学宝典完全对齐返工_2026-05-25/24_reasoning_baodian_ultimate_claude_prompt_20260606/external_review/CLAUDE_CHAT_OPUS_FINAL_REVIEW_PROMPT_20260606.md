# Claude Chat Opus Final Review Prompt

请作为独立外审，审核附件中的《选必三逻辑与思维_推理宝典_终极版_20260606》及 QA 证据。只给验收结论和必须返修的问题，不要改写全文。

## 输入文件

- `选必三逻辑与思维_推理宝典_终极版_20260606.md`：最终学生用正文。
- `QA_REPORT_V24_20260606.md`、`QA_GATES_V24_20260606.json`：脚本门禁。
- `PDF_RENDER_QA_V24_20260606.json`：PDF 渲染 QA。
- `DEFECT_LEDGER_V24_20260606.csv`：缺源/错位台账。
- `FINAL_ACCEPTANCE_REPORT.md`：本地验收边界。
- `CHATGPT_PRO_RECHECK7_RESULT_20260606.md`：GPT Pro 第七轮外审结论。

## 必审门禁

1. 83 题是否完整，无删题蒙混。
2. 每题 6 块是否齐全：这道题考什么 / 题目材料 / 设问 / 怎么想到的 / 满分作答示范 / 采分点对照。
3. `满分作答示范` 是否是真正可写上卷面的答案，而不是 `本题应围绕...`、`本题考查...`、方法层说明、或照搬 `怎么想到的`。
4. `满分作答示范` 与 `采分点对照` 是否避免整段复制，同时覆盖采分点。
5. 是否还有后台语、工程痕迹、OCR 残片、页眉页脚、破碎标点、旧标题、错字、裸挖空、重复表头、断尾分值。
6. 两条 `needs_manual_source` 是否保持中性缺源处理，没有伪造题干或确定采分点。
7. PDF/Word 交付边界是否清楚：本地 QA 与 GPT Pro 已过，最终只待本次 Claude Chat Opus 应用端复核。

## 输出格式

请按以下格式输出：

```
## VERDICT
ACCEPT / REVISE / BLOCK

## Critical Findings
如无，写 None。

## Required Patches
只列必须返修项。每项写清题号、层级、问题、建议修法。

## Sampled Evidence
列出你抽查的题号和证据点，至少覆盖：答案层、题目材料层、缺源台账、PDF/QA 证据。

## Boundary
说明本次结论不能替代原卷/官方细则/人工补源的边界。
```

判定规则：若存在可导致学生抄错、答案层仍是假答案、题干材料明显残缺但未入台账、或 83 题/六层结构不成立，给 `BLOCK`。若无阻断但有局部必须返修，给 `REVISE`。若只剩两条已入台账的真实缺源边界，且正文未伪造答案，给 `ACCEPT`。
